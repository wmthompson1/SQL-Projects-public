Creating Projects from Other Projects

You can create new projects by copying another project. The copy project function is robust: the header card, legs, tasks, and work orders associated with the project are copied, along with the information in the Project Maintenance and Warehouse Maintenance windows. You also have the option of copying budget information and document references.

  1. From the File menu, point to New Project then From Project, and select the project you want to copy.

The New Project from Project dialog box appears.

  1. Click the Project ID button and select the project from which you want to create a project.

  2. Click in the New Project ID field and enter the identification number for your new project.

You can enter the new Project ID and Version ID, or allow VISUAL to auto number your new project.

  1. A Lot ID of zero automatically appears in the Lot ID field. If you do not specify a new ID for a project, VISUAL automatically numbers the new project with the next sequential number. For more information on automatic numbering, refer to your VISUAL documentation.

  2. Using the calendar buttons, select the Release and Want dates.

If you do not select Release or Want Dates VISUAL automatically enters the current date.

  1. If you want to enter a quantity of more than one, click in the Quantity field and enter the quantity you want.

If you want VISUAL to forward schedule this project from the Release date, select the Forward schedule from release date check box.

If you want VISUAL to schedule this project to begin on its release date, select the Treat release date as hard check box.

  1. If you want to copy the budget information from the item you are copying, select the Copy budget information check box.

  2. If you want to copy all document references that have been attached to the item you are copying, select the Copy All Document References check box.

  3. Click the Save button on the toolbar.

VISUAL creates the project with a status of Unreleased.

Because VISUAL also copies Description field information for the Project and its warehouse, you may want to immediately change the descriptions to better suit the needs of the new project.
