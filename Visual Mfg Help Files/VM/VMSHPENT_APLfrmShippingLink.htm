Linking Cartons

To create an electronic link between the cartons you use on an order to the order's Bill of Lading:

  1. With the appropriate Order ID selected in the Shipping Entry window, select Link Cartons from the Edit menu.

The Link Cartons for … dialog box ,appears populated with the linked and unlinked cartons for your order.

  1. In the Unlinked Cartons table, select the cartons you want to link.

  2. Click the Add button.

  3. When you have finished adding cartons, click the Create Links button.

Because VISUAL does not create the links until you click the Create Links button, you can remove the cartons from the Linked table.
