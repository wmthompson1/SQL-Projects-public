Printing IBT Receipt Acknowledgements

Send this report to the recipient of the consigned material so that they can return it to you when they have received the material. This allows you to create an IBT receiver so that you can move the material into the consignment warehouse.

To print IBT Receipt Acknowledgement reports:

  1. Select the Inter Branch Transfer Shipping Entry option from VISUAL’s Inventory menu.

The Inter Branch Shipping Entry window appears.

  1. Click the Shipper ID browse button and select the shipper for the Receipt Acknowledgement Report you want to print.

  2. Select the Print Receipt Acknowledgement option from the File menu.

The Print IBT Receipt Acknowledgement dialog box appears.

  1. If you are licensed to use multiple sites, click the Site ID(s) arrow and select the sites to view in the report. The site selection applies only if you select the Print All Shipped IBTs Through option. If you are licensed to use a single site, this field is unavailable.

  2. Select the option for what you want to print:

Print Current IBT Shipper – If you only want to print the current inter branch transfer information, select the Print Current IBT Shipper option.

Print All Shipped IBTs Through – If you want to print information for a range of inter branch transfers, select the Print All Shipped  IBTs Through option.

Click the calendar buttons and select From and Through dates for your range.

  1. Select what you want to appear on the report:

Do Not Print Previously Printed Acknowledgements – Select this option if you do not want acknowledgements you have previously printed to appear on the report.

Consignment Transfers Only – Because the primary use of this report is to print acknowledgements you can send to consignment customers or vendors, the Consignment Transfers Only check box appears selected. If you want to print reports containing all of your IBT shipments regardless of whether they are consignments, clear the Consignment Transfers Only check box.

  1. Click the drop-down arrow and select the destination for the report. You can select one of the following:

Print – Select this option to output the file to a printer. VISUAL sends the file to the selected default printer.

View – To see what the printed output will look like before you print it, select View. When the information appears on the screen, you can scroll through and print it.

File – To save the information to a text file, select File. You can then edit the file outside of VISUAL.

E-Mail – Select this option if you want VISUAL to prepare the report and attach it to an e-mail. VISUAL prepares the report as a RTF file and attaches it to a Microsoft Outlook e-mail. For more information on addressing and sending the e-mail, refer to your Microsoft Outlook user documentation. If you want to convert this document to PDF (Portable Document Format), select the PDF Format check box. If you want to send this e-mail to all of the contacts associated with this document, select the Send to Document Contacts check box.

  1. In the Format section, select the output format you want to use for the report.

You can select the Standard, Print Form, or any of three user defined formats. The User-Defined option buttons are active only if you have created user defined report formats.

Select the Print Form option button to print the report with a preprinted form using a laser printer. The resulting report is of better quality than the report you can print using the Standard print format option.

  1. Click the Ok button.

VISUAL prints the report using the selections you made.

Notice the Quantity Used area for the vendor or customer to use for their inventory usage and the Received By signature area.
