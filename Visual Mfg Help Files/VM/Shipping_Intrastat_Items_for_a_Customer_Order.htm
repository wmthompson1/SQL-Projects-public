Shipping Intrastat Items for a Customer Order

Shipping orders that contain Intrastat-related line items is similar to normal shipments, but VISUAL gives you the opportunity to view and edit Intrastat data for the shipment.

  1. Follow the normal procedure to ship the order.

When you click the Save or Ship All buttons, a dialog box appears, asking you if you want to view or edit the Intrastat data.

  1. Click Yes to view and edit the information.

The Intrastat Dispatch dialog box appears. The dialog box displays the details for the packlist.

  1. In the Options section, select from the following options:

Show Current C/O Packlist - Click this radio button to view the packlist for the current customer order.

Print Orders Shipped Between - Click this radio button to view line items by shipping dates. Enter the date range the Starting Date and Ending Date fields.

Incorrect Lines Only - This check box is checked by default. When this check box is checked, only those lines with missing data that is required for reporting appear in the line item table.

  1. Make any changes to the information in the line item table.

If this line is for a consignment, the check box in the Consignment column appears selected.

  1. Click the Save button to save the information.

You can also view Intrastat information by clicking on the Intrastat button in the Customer Shipments window.
