Setting a Customer's Allocation Fence

With the appropriate Customer ID in the Customer ID field, click the Order Management tab.

Allocation Fence - The number of days within which orders from a customer of this type the Allocation Utility considers. If an order’s desired ship date is outside the allocation fence, the Allocation Utility ignores it during allocation.

A message box appear after you select a Customer Type asking you if you want to use the default allocation fence for this Customer Type.

Click the Yes button to accept this default setting; click the No button to enter an independent Allocation Fence value.
