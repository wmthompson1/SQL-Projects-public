Copying Purchase Requisitions

If you are licensed to use multiple sites, you can copy a purchase requisition within the same site only. You cannot copy a purchase requisition created in one site to create a purchase requisition in a second site.

To copy a purchase requisition:

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site in which you created the requisition. If you are licensed to use a single site, this field is unavailable.

  2. With the purchase requisition you want to copy open, select Copy Purchase Requisition... from the Edit menu.

The Copy Purchase Requisition dialog box appears.

The ID of the requisition you are copying appears in the Copy from PR ID field.

  1. Enter an ID for the new requisition into the Copy To PR ID field. If you use auto-numbering, leave this field blank.

  2. Select Copy options:

Select the Copy PR Line Items box if you want VISUAL to copy requisition lines.

Select the Copy Delivery Schedule box if you want VISUAL to copy delivery schedules. This check box is only active if the copy-from requisition contains delivery schedules and the Copy PR Line Items check box is selected.

Select the Copy Approval Teams box if you want VISUAL to copy approval teams.

  1. Click the Copy button.

VISUAL copies the requisition and a dialog box appears asking you if you want to view the new requisition.

  1. Click the Yes button to load the new requisition.
