Sending Shipments

You can send shipments using the Delivery Schedule Shipment dialog box. Use the Delivery Schedule Shipment dialog box for all order lines that have delivery schedules.

  1. Select the customer order line.

  2. Select the Del Sched tab.

The Delivery Schedule Shipment dialog box appears.

  1. Select the delivery schedule line you want to use.

  2. Click Save to send the shipment.
