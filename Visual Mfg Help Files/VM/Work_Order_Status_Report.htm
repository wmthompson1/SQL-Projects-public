Work Order Status Report

The Work Order Status report shows schedule status for a selected group of work orders. For each work order, the following information appears: Part/Description, Customer (if any), Want Date, Scheduled Finish Date (if scheduled), Quantity required, Status (R for Released, F for Firmed), and a scheduling message.

If the schedule finish date for the work order is greater than its want date, the work order is marked Early. If the scheduled finish date is after the want date, it is marked Late. However, if the want date is previous to the current date, then the work order is marked Late! (with exclamation point) to indicate that the want date has already been passed, rather than just being scheduled to be late. If the work order has not been scheduled into the production schedule, it is marked N/S (Not Scheduled).

  1. From the Print Production Schedule Reports window, click the Work Order Status Report button.

  2. If you are licensed to use multiple sites, click the Site ID(s) arrow and select the sites to include in the report. If you are licensed to use a single site, this field is unavailable.

  3. Enter the range of work orders to print.

  4. Select the status of the work orders you want to print.

You can print report for Firmed work orders, Released work orders, or both, by checking the appropriate boxes in the Status section.

  1. Select and set the appropriate work order schedule filter.

You can choose to print only work orders whose Scheduled Finish Date is earlier or later than the Want Date by a specified number of days. To use these options, check either box and enter the number of days late and/or early. If you select both, VISUAL considers all work orders earlier and later than the days you specify.

  1. Select the sequence for the report.

You can sequence the report by Want Date, Scheduled Finish Date, or Days Late.

  1. Select the output for the report.

Select Print to output the report to a printer.

Select View to output the report to your screen for viewing.

Select File to output your report as a text file. You can then edit the report using a text editing application.

  1. Click the Ok button.

If you selected Print as the output, a standard Windows dialog box appears.

  1. Make the appropriate selections then click Ok.

VISUAL prints the report.
