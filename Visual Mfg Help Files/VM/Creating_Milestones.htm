Creating Milestones

A milestone is associated with an operation in a work order. When the operation is complete, the milestone is achieved. The billing event for the milestone can then be triggered.

You are not required to associate every operation in a work order with a milestone

To create a milestone:

  1. Select Sales, Progress Billing Entry.

  2. If you are licensed to use multiple sites, click the Site ID arrow and select the site in which you created the customer order. If you are licensed to use a single site, this field is unavailable.

  3. In the Order ID field, click the browse button and select the order to progress bill. The order must be linked to a work order. See Associating Line Items To Work Orders.

  4. Highlight the appropriate line item in the Progress Billing Entry window.

  5. Select Edit, Milestones.

  6. Specify an ID in the Milestone ID field.

  7. Specify a description for the milestone in the Description field. This is an optional field.

  8. Specify or select the appropriate cost accounts for material, labor, burden, and service in the appropriate account fields.

  9. Select the operation to associate with the milestone, and then click Set Operation to Milestone.

  10. Click Save.

The operation is associated with the milestone. The ID of the milestone is inserted after the resource ID used on the operation.
