Adding Drill-to Buttons to Query Results

To add drill-to buttons, you must clear the No Auto Joining check box when you write the query.

The drill-to buttons are added automatically provided that your query includes the required primary or foreign keys and columns. For example, you can drill into Site Maintenance if SITE.ID or CUSTOMER_ORDER.SITE_ID is included in your query.

You must also specifically select the database column in your query. For example, if you specified SELECT * FROM RECEIVER, drill-to links would not be added to the query result. If you specified SELECT r.ID, r.PURC_ORDER_ID, r.SITE_ID FROM RECEIVER r, then the query result would include drill-to buttons for the receiver ID, purchase order ID, and site ID.
