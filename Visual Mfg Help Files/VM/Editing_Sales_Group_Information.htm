Editing Sales Group Information

You can change any of the fields in the Sales Tax Maintenance window with the exception of the Tax Group ID. Changing the Tax Group ID creates a new tax group.

To edit any of the fields in the Sales Tax Maintenance window:

  1. Select the Tax Group ID you want to edit.

  2. Make the appropriate changes.

To change the Sales Tax ID percentage, select the Edit Percents button to call up the Tax Percents table. Make the desired changes to this table and click OK to save the changes and return to the Sales Tax Group Maintenance window.

  1. Click the Save button to save the changes.
