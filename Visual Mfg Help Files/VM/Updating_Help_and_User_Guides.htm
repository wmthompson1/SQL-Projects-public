Updating Help and User Guides

You can find the most up-to-date versions of the online help and user guides on Infor Support Portal. To download up-to-date files:

  1. Sign into Infor Support Portal.

  2. Select Search, Browse Documentation. If VISUAL is your main product, then the VISUAL documentation categories are shown in the left pane. If VISUAL is not your main product, then click Browse All Products to view the VISUAL documentation categories.

  3. To view the documentation for this release, expand Infor VISUAL, then expand Infor VISUAL 11.0.0.

  4. To save the documentation to your computer, right-click the documentation link and:

  5. If you use Internet Explorer, select Save Target As.

  6. If you use Google Chrome or Mozilla Firefox, select Save Link As.

If you use PDFs can be downloaded directly into your VISUAL executables directory. Online help files must be unzipped into your VISUAL executables directory.
