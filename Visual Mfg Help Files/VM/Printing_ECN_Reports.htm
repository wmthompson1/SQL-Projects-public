Printing ECN Reports

Use ECN Entry to print reports containing comprehensive ECN implementation and completion detail information. Print these reports by ECN ID, Creator, Assigned To, Origin, date, or according to status.

  1. Select Eng/Mfg, ECN Entry.

  2. In the main ECN window, click the Site ID drop-down arrow and select the site that contains the ECNs that you want to print. If you are licensed to use a single site, then this field is unavailable.

  3. Optionally, click the ECN ID browse button and select the record to print. Skip this step if you are printing ECN information for more than one ECN.  

  4. Select File, Print ECNs or click the Print button on the main toolbar.

If you selected an ECN in ECN Entry, the ID that you selected is inserted in the Starting ECN ID and Ending ECN ID fields. You can specify whether to print a detail or summary report, whether to include cancelled line items in the report, and the report format. All other fields are inactive. To change the selected ECN and reactivate the fields in the dialog, clear the Current ECN check box.

  1. Specify the filters to apply to the report. To include all ECNs in the report, leave all fields blank. Specify these filters:

Starting ECN ID and Ending ECN ID – To include a range of ECNs by ID, specify the starting ID and ending ID in the Starting ECN ID and Ending ECN ID fields. The list of ECNs is limited to the site that you chose in the main ECN Entry window. To print information for a particular ECN ID, specify the same ID in the Starting ECN ID and Ending ECN ID fields.

Creator – Click the Creator button to choose a Creator. Only ECNs with this Creator will appear on the report. To include all Creators, leave this field blank.

Assigned To – Click the Assigned To button to choose an Assigned To user for the report. To include all Assigned To users, leave this field blank.

Origin – To include ECNs with a specific originator, first click the type of originator. Then, click the browse button to select the originator.

  1. Use the Date Range section to filter the report by date. Click the type of date to use, and then specify the date range in the Starting Date and Ending Date fields. If you do not want to limit the report by date, leave the date fields blank.

  2. Use the Status section to filter the report by ECN status. Select one or more of these check boxes:

Undefined – Select the Undefined check box to include all applicable ECNs with an undefined status.

Pending – Select the Pending check box to include all applicable ECNs with a pending status.

In Process – Select the In Process check box to include all applicable ECNs with an in process status.

On Hold – Select the On Hold check box to include all applicable ECNs with an on hold status.

Completed – Select the Completed check box to include all applicable ECNs with a completed status.

Cancelled - Select the Cancelled check box to include all applicable ECNs with a cancelled status.

  1. Use the ECN Type section to filter the report by type. To select multiple, non-contiguous codes, hold down the Ctrl key while selecting the appropriate entries.

  2. Use the ECN Reason section to filter the report by reason code. To select multiple, non-contiguous codes, hold down the Ctrl key while selecting the appropriate entries.

  3. Select a report type, either Detail or Summary.

Summary reports provide access to high-level ECN information and are useful if you are printing many ECNs. Detail reports include more comprehensive data. If the report is for only the current ECN or a small range, use the Detail option.

 

  1. Choose a sequence option for the report:

ECN – Choose the ECN option button to sequence the report by ECN ID.

Date Range – Choose the Date Range option button to sequence the report by date.

Status - Choose the ECN option button to sequence the report by ECN status.

  1. Select an output for the report:

Print - Select Print to output the file to a printer.

View - Select View to view the report before printing.

File - Select File to save the information to a text file.

E-Mail - Select E-Mail to send the report in an email. To send the report as a PDF, select the PDF Format check box. To send the report as Rich Text Format (.RTF) file, clear the PDF Format check box.

  1. Click the Ok button to output the report to the specified destination.
