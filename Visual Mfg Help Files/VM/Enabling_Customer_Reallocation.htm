Enabling Customer Reallocation

Before creating new allocations, existing allocations may be removed. If you select this check box, the Allocation Utility removes and recalculates allocations for this customer. If this customer type has Reallocation as a default setting, VISUAL selects this check box when you accept the Customer Type default information. See Setting a Customer's Allocation Fence for more information on accepting Customer Type default values.
