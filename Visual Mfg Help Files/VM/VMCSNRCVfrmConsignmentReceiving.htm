What is Consignment Receipt Entry?

If you are receiving a consignment of parts or materials from a vendor or customer, use the Consignment Receiving window to record the receipt.

To receive consigned inventory:

  1. Select Purchasing, Consignment Receipt Entry of the VISUAL main menu.

The Consignment Receiving window appears.

  1. If you are entering Consignment Receiver from vendors, click the Vendor tab; for customers, click the Customer tab.

  2. If you are licensed to use multiple sites, click the Site ID arrow and select the site that is receiving the materials. If you are licensed to use a single site, this field is unavailable.

  3. For Consignment Receivers from vendors, click the browse button and select the Order ID; for customers, select the Customer ID.

For vendors, the following information appears:

  • Vendor ID

  • Contact Information

  • Buyer

  • FOB

  • Ship Via

  • Order Status

  • Intrastat

  1. If you want to enter intrastat for this consignment receipt, click the Intrastat browse button and enter the information you want to use for this shipment.

VISUAL tracks the intrastat for this consignment receipt. When you take ownership of this consignment by entering a purchase order, VISUAL does not allow you to enter intrastat information because you have already done so at the consignment phase.

  1. If you want to change the Receive Date, click the calendar button and select the appropriate dater.

  2. For vendors, click the drop-down arrows in the field and select the FOB and Ship Via information for this consignment receiver.

  3. If you are receiving a different quantity that specified on the order, enter the quantity you are receiving in the Quantity Received column for each line item.

  4. If you are receiving a complete order, click the Receive All toolbar button.

If you are receiving a partial order, click the Save toolbar button.
