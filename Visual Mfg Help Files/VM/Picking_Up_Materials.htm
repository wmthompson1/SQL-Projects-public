Picking Up Materials

Use this procedure to pick up the materials in a move request. You can also pick up materials in the Move Request window. See Using the Move Request Window to Pick Up Move Requests.

  1. Select Inventory, Material Handler Queue.

  2. In the Move Request table, select the check box for the row that you are picking up. The move request must have a status of Ready for Pickup. An employee must be listed in the Pickup Employee column, or you must specify an employee ID in the Assign to Emp ID field. You can select more than one request.

  3. Click the Pick Up button. The status of the move request is updated to Picked Up.
