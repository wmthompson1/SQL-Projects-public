Printing the Audit History Report

  1. Select Admin, Audit Maintenance.

  2. Optionally, filter the results. See Filtering Audit Results.

  3. In the Reporting Options section, specify how to output the report:

Print - Select this option to send the report to a printer.

View - Select this option to view the report on the screen.

File - Select this option to save the report to a file.

E-Mail - Select this option to attach the report to an email. To attach the report as a PDF, select the PDF Format check box. To attach the report as an RTF, clear the PDF Format check box.
