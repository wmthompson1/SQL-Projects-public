Using the APS Export Utility

To use the APS Export Utility:

  1. Use Windows Explorer or My Computer to navigate to the directory where your executables are installed, and double-click the VMAPSEXP.EXE program name to start the APS Export Utility.

  1. Site ID - Click the Site ID arrow and select a site to use.

  2. Click the Type of Data arrow and select the type of scheduling data you want to export.

You can select:

PLN

Frozen Operations

Calendar Exceptions

  1. Click in the Outbound Rule section and select what you want VISUAL to do with the information.

You can select:

Replace File - Select the Replace File option if you want to overwrite the original file.

Append File - Select Append File if you want VISUAL to add the new information to the end of the file and not delete any of the information the file currently contains.

  1. If you have multiple Schedule IDs, the names appear in the Schedule IDs list. Select the schedules you want to export.

  2. If you do not want VISUAL to include masters with obsolete parts, select the Skip masters with obsolete parts check box.

  3. In the Output File section, click the browse button and select the file to which you want VISUAL to save your information.

  4. Click the Extract toolbar button.

VISUAL exports the data to the file you specified.
