Returning Materials with Demand Links to Vendors

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site in which you created the receiver. If you are licensed to use a single site, this field is unavailable.

  2. If a purchase order line that you receive is linked to demand, before you return the whole quantity or a portion of the quantity, you can view the line’s demand links before you save the return line.

  3. With the received purchase order line that you want to commit a return against highlighted in the Purchase Receipt Returns dialog box line item table, click the Assign Demand button.

The Assign Return to Demand dialog box appears.

The line item table contains the following columns:

Receive Quantity - The quantity of the Order ID that you received.

Return Quantity - The quantity of the Order ID that you are returning.

Remaining Quantity to Assign - The Return Quantity minus the Allocated Quantity. If the Return Quantity is greater than the Allocated Quantity, you have this many parts still available to you to assign to demand.

  1. Click the Ok button when you are finished viewing the return demand link information.

In the Quantity Returned column, enter what portion of the received quantity you want to return.

  1. Double-click <Location ID> to select a location in a warehouse into which you want to return the parts.

  2. Click the Save button to return the linked purchase order receipt.
