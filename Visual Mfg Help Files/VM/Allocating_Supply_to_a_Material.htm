Allocating Supply to a Material

With the work order line that contains the material requirement to which you wan to allocate supply highlighted, from the Edit menu select Allocations and then Allocate Supply to a Material. The Supply Links dialog box appears.

You can allocate supply to a work order material requirement from these sources:

  • Co-product supply

  • Inventory

  • Purchase Order Delivery Schedule supply

  • Purchase Order supply

  • Interbranch Transfer supply

  • Work Order Supply
