Project Summary

This dashboard analytic is available only if you are licensed to use projects/A&D functionality. This analytic shows a variety of project summary information. You must specify a project ID to display data in this chart.

Type: Line Graph

Y-axis: Amount (expressed in accounting entity’s functional currency)

Available lines: Budget Amount, Budget Hours, Actual Amount, Actual Hours, EAC Amount, EAC Hours, Earned Amount, Earned Hours, Budget Variance, Cost Variance, EAC Budget Variance, Schedule Variance.

Series groupings (X-axis): No groupings available for selection. The series grouping is month and year.

Date filter applies to: Project performance duration. For example, if you began a project in January 2014 and ended the project in December 2014, you could view data for March 2014 through June 2014.

Filters: Project ID. You must select a project ID to view data in this chart.

Detail grid: n/a
