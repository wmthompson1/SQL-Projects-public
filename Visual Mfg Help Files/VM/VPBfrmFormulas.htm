Maintaining Formulas

Formulas are what determine and calculate the various prices within VISUAL Price Book. VISUAL Price Book uses the Microsoft Active Script Object to evaluate formulas, so you must use Visual Basic script code when creating formulas.

Use the Formulas function to set up price calculations within the VPB database. Through the Formulas function, you can read your external ERP system’s database.

To access the Formulas window:

  1. With VISUAL Price Book open, from the navigation pane, double-click the Formulas icon. The Formulas window appears. Depending on the settings you used on the Tools menu, your default values may vary.

  2. Enter or select an Identification Code for the Formula. If you have existing formulas you want to edit, click the Formula ID Browse button to access the list of existing formulas. VISUAL populates the appropriate fields. If you want to create a new formula, enter the ID manually. VISUAL prompts you to enter the necessary data to set up a new formula.

  3. Enter a Description for the Formula. Descriptions allow you define the formula for easy identification in the future.

  4. Enter the necessary information on the General, Formula, and Variables tabs as defined below.

  5. On the VISUAL Price Book main toolbar, click the Save button to save your changes to the Formulas window.
