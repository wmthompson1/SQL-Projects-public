Project Summary Report

Use the Project Summary Report to output the information currently displayed in the Project Summary window. You can also use this report to review costs for all periods.

To run the report:

  1. In the Project Summary window, enter the project you would like to view in the Project ID field. You can select a project by clicking the Project ID button and selecting the project ID from the table.

  2. If you want to output specific information from the Project Summary window, select the information you would like to view. For example, select Budget Variance in the Summary Type(s) section to view budget variance information. The information appears in the Project Summary window.

  3. From the File menu, select Print Project Summary.... The Print Project Summary dialog box appears.

  4. Select the information you want to view in the report. To view the information that currently appears in the Project Summary window, select Print data currently showing on screen. To print information for all periods, select Print detail for all periods. If you select Print detail for all periods, the report displays the summary type(s) you selected in the Project Summary window with cumulative totals for the entire project, not just the current period.

  5. If you select Print detail for all periods, the Cost Group Type section becomes available. Select the cost groups you would like to view in the report:

All – Select All to view all cost groups in the report.

Labor – Select Labor to view only labor-related costs in the report.

Material – Select Material to view only material costs in the report.

ODC – Select ODC to view other direct costs in the report.

  1. If you select Print detail for all periods, the Group By section becomes available. Use the Group By section to select a sort order for the report. Your choices are:

  1. Select the Print grid lines check box to print grid lines in the report. The grid lines separate each line.

  2. If you select Print detail for all periods, the Print Subtotals check box becomes available. Select Print Subtotals to view the sub-total by group.

  3. Click the View arrow to select the output method for the report. Your choices are view, print, email, and file.

  4. Click Ok. The report outputs.
