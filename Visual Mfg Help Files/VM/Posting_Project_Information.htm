Posting Project Information

After you run the costing utilities and prepare the journals, post the journals to the General Ledger.

  1. From the Ledger menu, select Post Manufacturing Journals. The Post Manufacturing Journals window appears.

Leave the Batch ID and Description fields blank. VISUAL populates these fields.

  1. Enter the date you would like to use for posting the journals in the Batch Date field. If you are preparing the month-end closing, enter the end date of the month you are closing. VISUAL populates the Accounting Period.

  2. Clear the Current User Only check box to process journal entries for all users.

  3. Click the Post All button to post all journal types.
