Entering Basic Information into ECNs

  1. From the Eng/Mfg menu, select ECN Entry.

The ECN Entry window appears.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site in which you are creating the ECN. If you are licensed to use a single site, this field is unavailable.

  2. If you want to manually number this ECN, enter a unique ID into the ECN ID field.

If you want VISUAL to assign the next available ECN ID, leave the ECN ID field empty.

  1. VISUAL automatically enters the ID of the current user into the Creator and Assigned To fields.

To select a different Creator, click the Creator button to browse a list of all users in the database and choose the appropriate ID.

  1. Click the Assigned To button and select the user to which you want to assign this ECN.

To select a different Assigned To user, click the Assigned To button to browse a list of all users in the database and choose the appropriate ID.

  1. Specify a Request Date and a Desired Finish Date:

Request Date – The date on which the ECN was entered into the database. The current system date is the default.

Desired Finish Date - The date by which you want the ECN completed. The current system date is the default.

  1. In the Teams/Completion Status section, specify the following:

    If you have assigned any groups as Default groups, those groups automatically appear in the Teams/Completion section.

Authorization - Click the Authorization button to select the user or group responsible for ECN authorization.

Implementation - Click the Implementation button to select the user or group responsible for ECN implementation.

Approval - Click the Approval button and select the user or group responsible for ECN approval.

Distribution - Click the Distribution button to select the user or group responsible for ECN distribution.

Update After – If the Update After check box is not already selected (you can specify a default Update After preference), select the point from the four ECN stages at which revision update should occur.

After assigning a user or group to a particular status, VISUAL places an asterisk at the end of the button label.  

  1. Add lines to the ECN. See Entering and Editing ECN Tasks.

  2. Enter any additional information about this ECN on the appropriate tab. See Entering Information on ECN Tabs.

  3. Click the Save button on the toolbar or select Save from the File menu.

If you have not entered an ID for this ECN, VISUAL automatically assigns the next available ID.
