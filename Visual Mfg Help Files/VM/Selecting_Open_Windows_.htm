Selecting Open Windows

With multiple windows open and arranged, or with multiple windows open but hidden except for the current displayed window, from the Window menu select the Base ID/Lot ID of the work order/master you want to have the default position.

VISUAL places a check mark (3) next to the default window.

With arranged windows, you can click anywhere in a window to make it the default.

Upon selection, the top portion of the default window becomes blue.

Selecting Windows if more than Nine Windows are Open:

VISUAL can list a maximum of 9 windows in the Window menu. If you have more than nine windows open, select More Windows from the Window menu to view a complete list of all open windows.

The Selected Windows dialog box appears.

Select the appropriate window and click the Ok button.

VISUAL moves the window to the front and makes it the default

Click the Cancel button to exit the dialog box and return to the main display.
