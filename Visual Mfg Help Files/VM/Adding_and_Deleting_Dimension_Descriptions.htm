Adding and Deleting Dimension Descriptions

Use Quality Maintenance to insert and delete dimension descriptions for tests.

Adding Dimension Descriptions

  1. Select Dimension Descriptions from the Quality Maintenance Edit menu.

The Dimension Description dialog box appears.

  1. Click the Insert button.

A new line appears in the list of dimensions.

  1. Enter the new dimension description.

  2. Click the Save button to save the new description.

Deleting Dimension Descriptions

You can use Quality Maintenance to delete dimension descriptions.

  1. Select Dimension Descriptions from the Quality Maintenance Edit menu.

The Dimension Description table appears.

  1. Select the description you want to delete.

  2. Click the Delete button.

An X appears to the left of the description.

  1. Click the Save button to complete the deletion.
