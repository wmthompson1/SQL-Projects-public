In-Context View Panels in Purchase Order Entry

You can view the Purchase View Panel, Vendor View Panel, and Part View Panel in Purchase Requisition Entry.

To access the Purchase View Panel, select View, Purchase View Panel. The Purchase view panel shows general information about your purchasing practices. It shows information at the tenant level. This information is displayed:

  • Open orders

  • Last order date

  • Aged payables amount

  • Oldest open payable

  • Average days to pay

  • Last receipt date

To access the Vendor View Panel, select View, Vendor View Panel. Use the Vendor View panel to view information about the selected vendor. You can view summary information, detail information, and a chart. For information about setting up the summary and detail views, see In-context View Panels. For information about the chart, see Vendor View Panel Chart.

To access the Part View Panel, select View, Part View Panel. For information about setting up the summary and detail views, see In-context View Panels. For information about the Part view panel chart, see Part View Panel Chart.
