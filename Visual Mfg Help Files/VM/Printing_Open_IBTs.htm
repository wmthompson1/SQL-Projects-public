Printing Open IBTs

You can print a list of open IBTs by sending warehouse, receiving warehouse, or all warehouses.

  1. From the File menu, select Print Open IBTs. The Print Open Interbranch Transfer dialog box appears.

  2. In the Dated Through field, enter the date through which you want to print open IBTs

  3. Select how you want VISUAL to output the report:

Print - Select this to view a print preferences dialog box after clicking Ok.

View - Select this to view the report on your screen after clicking Ok.

File - Select this to save the report to a file after clicking Ok.

  1. From the Warehouse section, select what information you want the report to contain:

Print Current From Whose ID - Select this box to print only information regarding the current IBTs From Whose Warehouse ID.

Print Current To Whose ID - Select this box to print only information regarding the current IBTs To Whose Warehouse ID.

You can select these two boxes at the same time.

Print All Warehouses - Select this box to print information regarding open IBTs in all warehouses. Selecting this box makes the previous print warehouse options unavailable.

  1. Select how you want VISUAL to sequence the report: by IBT ID, Supplying Whse ID, Receiving Whse ID, or Part ID. You can select only one of these options.

  2. Click the Ok button. Depending on your selection above, VISUAL now presents a print preferences dialog box (Print), displays the report on your screen (View), or displays a dialog box that lets you specify where you want to save this report (File).
