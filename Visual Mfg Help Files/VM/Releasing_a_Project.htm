Releasing a Project

Before you can link a project to a customer order, you must change the project’s status to Released. To release a project:

  1. Select the project’s Header Card.

  2. From the Status Change menu, select Release.... The Change Status dialog box appears.

  3. To release the entire project, select the Change status of subordinate materials and operations check box.

  4. Click Ok. VISUAL releases the project.
