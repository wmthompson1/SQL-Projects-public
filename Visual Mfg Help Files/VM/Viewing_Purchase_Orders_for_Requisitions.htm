Viewing Purchase Orders for Requisitions

After creating a purchase order from a requisition, you can view details of the requisition/purchase order relationship using this feature.

  1. Select the requisition you want to view purchase order information for and select Show Purchase Orders for Requisitions.

The Show PO Requisition dialog box appears, displaying the specified requisition in both the Starting and Ending fields.

This dialog box allows you to specify a range of requisitions, a starting ID and an Ending ID.

  1. Click the Starting Req ID button to choose the starting requisition.

Click the Ending Req ID button to choose the ending requisition.

  1. Review the table for details of the requisition/purchase order relationship. The table includes the following fields:

Requisition ID - The ID of the requisition. If there is more than one line in the requisition, the Requisition ID appears in multiple rows.

Requisition Line # - The line within the requisition.

Purchase Order ID - The ID of the purchase order created from the requisition. It is possible to create more than one purchase order per requisition, providing you do not try to reuse the same line in more than one order.

Purchase Order Line #- The purchase order line the requisition line is linked to.

Requisition Quantity - The quantity of the requisition line.

Purchase Order Quantity - The quantity of the purchase order line.

Requisition Status - The current status of the requisition line. This is not necessarily the overall status of the requisition, which supports a header status as well as a status per line.

Purchase Order Status - The current status of the purchase order.

Requisition Buyer - The Buyer specified on the requisition. The Requisition Buyer and Purchase Order Buyer must be the same to create a purchase order from a requisition.

Purchase Order Buyer - The Buyer specified on the purchase order. To create a purchase order from a requisition, the requisition and the purchase order must share a common Buyer.

  1. To view a purchase order in Purchase Order Entry, select an entry and click the Purchase Order button or double-click the appropriate line.
