Deleting Vendors from the Table

To delete a vendor from the Vendors Supplying Part table:

  1. Select Inventory, Part Maintenance.

  2. If you are licensed to use multiple sites, click the Site ID arrow and select the Site ID for which you are adding vendors. If you are licensed to use a single site, this field is unavailable.

  3. In the Part ID field, click the browse button and select the Part ID. You must select a purchased part to activate the Vendors Supplying This Part dialog box.

  4. Select Maintain, Vendors Supplying This Part. The dialog box shows the vendors that are currently assigned to the part.

  5. Select the vendor to be deleted.

  6. Click the Delete button.

VISUAL deletes the vendor’s information. This does not remove the vendor from the list of available vendors, just from this part.
