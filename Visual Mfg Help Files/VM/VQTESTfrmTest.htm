Entering Quality Test Results

  1. In the Test ID field, enter the test ID for the test you used for the operation.

  2. In the Base ID field, enter the base ID for the work order you tested.

You can also click the Work Order button and select a work order from the Work Orders table.

  1. In the Seq # field, enter the operation sequence number that you tested.

You can also click the Operation button and select an operation from the Work Order Operations table.

  1. Highlight the test line item for the test.

  2. Click the Input button.

The Test Input dialog box appears for that operation.

  1. In the Enter Date/Time and Tester area, select the appropriate radio button:

Once for all tests - All tests have the same date, time, and tester.

For each subgroup - Each subgroup has a different date, time, and tester.

For each test - Each test has a different date, time, and tester.

For each reading - Each reading has a different date, time, and tester.

  1. In the Enter Date/Time and Tester area, enter the number of subgroups that were tested.

  2. In the Subgroup Information area, enter the date and time of the readings and the tester.

You can click the Date/Time Config button to modify date and time information.

The system defaults to the current time and date.

  1. In the Input area line item table, enter the results of the test for each of the pieces tested.

  2. Click the Save button to save the test information.

A dialog box appears, asking you if you want to run decision tests.

If you click No, the Test Input dialog box closes.

If you click Yes, the Check Lack of Statistical Control dialog box appears for that operation.

  1. Enter the appropriate information then click Run.

VISUAL checks for parts that are outside of statistical control, based on the values you entered. The results appear in the Current Count column in the dialog box.

  1. Click the Close button when you are finished.

Click the Save Decision Tests button to save the decision tests that you just ran.
