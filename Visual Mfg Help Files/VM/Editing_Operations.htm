Editing Operations

  1. Select the operation you want to edit, then select Edit from the Edit menu.

If your current viewing mode has operations hidden, you need to select Show/Hide Operations from the View menu so that operations are shown.

  1. Edit the appropriate information.

You can edit all of the fields of an operation.

  1. Click the Save button to save the changes. Select Close from the File menu to close the dialog box without saving the changes.
