Viewing Requisition Tasks

Each requisition is comprised of tasks, or individual sequence numbers assigned to requisition approval team members. Use this feature to review task status for each approval team member.

  1. Select the requisition you want to view task information for and select Tasks... from the Info menu.

The Tasks for (PR-) dialog box appears, displaying the individual sequence numbers for the requisition.

  1. The table displays one task per line, in descending sequence. For more information on each line, review the information in each column:

Task No - The number of the task. Typically, VISUAL uses the Requisition ID as the task number. For example, 14 if the ID is 0000014.

Seq No - The sequence number of the task.

User ID - The ID of the user assigned the task.

Type - The type of task. REQ is the default task type for a requisition.

Sub Type - The level of approval the sequence is linked to. For example, if you employ a 4-level approval tier, VISUAL uses A1, A2, A3, and A4 to describe the various levels.

The user who initiated the requisition (the user who was signed into VISUAL at the time) receives a different Sub Type: Assigned To.

EC ID - Engineering Change ID - The engineering change ID of the task, which is typically the Requisition ID.

Status - The current status of the task. Completed tasks prompt VISUAL to update the completion meters on the Approvals tab. See later for more information.

Date Completed - The date on which the task was reported as completed.

Specifications - Any text regarding the task completion. After the user reports the task as completed, VISUAL generates text detailing the task change, including a date and time stamp. See Step 3 for more information.

Reject Code - If for some reason the task was rejected, VISUAL places the text supplied by the user who rejected the task.

Rejected Task Number - The task number of the rejected task.

Rejected Seq Number - The sequence number of the rejected task.

Create Date - The date and time on which the task was created.

Site ID – If you are licensed to use multiple sites, the site in which the task was created.

  1. To view or modify a sequence’s specifications, click the Task Specifications button, double-click the line, or select Specifications from the Edit menu.

  2. Insert or modify information as necessary.

  3. If appropriate, print the task list. Click here for more information.
