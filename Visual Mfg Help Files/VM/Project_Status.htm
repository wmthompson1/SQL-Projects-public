Project Status

Use the Project Status window to review current costs and any related labor and inventory transactions. To view the Project Status window:

  1. In the Project Summary window, select the project you would like to view in the Project ID field.

  2. From the Info menu, select Show Project Status.... The Project Status window appears.

  3. The window displays costs by WBS code/Sub ID/Seq No combination.

  4. To view additional information about work orders, double-click a line with a work order as a cost source, or select the line and click the Open Work Order button. The work order appears.

  5. When you are finished reviewing project status information, click the Close button.
