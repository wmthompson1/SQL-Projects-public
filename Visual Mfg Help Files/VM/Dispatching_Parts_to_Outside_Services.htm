Dispatching Parts to Outside Services

When sending parts to an outside service, you may want to dispatch the parts to the outside service in portions, rather than all at once. Service Dispatch Entry allows you to send the parts from the purchase order in portions.

  1. Select Service Dispatch Entry from the Tools menu.

The Service Dispatch window appears.

  1. Enter the Dispatch ID in the Dispatch ID field.

  2. Enter the Order ID in the Order ID field.

  3. Enter the Lot ID/Split in the Lot ID/Split ID field.

  4. If you want to dispatch from a purchase order, select the Purchase Order radio button.

If you want to dispatch from a work order, select the Work Order radio button.

  1. Enter the free on board code in the FOB field. FOB (Free On Board) codes determine at what point in the shipping process the customer must take responsibility for shipping fees for the goods your company is shipping.

  2. Enter the shipping type in the Ship Via field.

  3. Enter the actual dispatch date in the Actual Dispatch Date field.

  4. To dispatch services from an existing order, select the line with the parts you want to dispatch.

  5. Enter the dispatch part amount in the Dispatch Quantity column.

  6. Click Save to save any changes to the Service Dispatch window.
