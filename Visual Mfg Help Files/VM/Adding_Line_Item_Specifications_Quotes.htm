Adding Line Item Specifications - Quotes

Line Item specifications are specific to an individual line item in a quote and there can only be one specification per line item.

  1. With the appropriate quotation open, highlight the line item for which you want to add the specification.

  2. Click the Line Spec button.

The Line Spec dialog box appears.

  1. Enter the information for the line item.

You can enter an unlimited amount of information.

Use the Paste button to paste text into the dialog box that you copied from another location.

Use the Copy button to copy text in this dialog box to the clipboard.

  1. Click Close to close the Line Spec dialog box and return to the Estimating Window.

Changes to specifications are saved when you click the Save button.

If you are pasting information into the dialog box the entire contents of the dialog is replaced with the incoming text.
