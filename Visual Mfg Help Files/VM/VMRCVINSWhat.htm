What is Receiving Inspection?

You may inspect received parts for various reasons.

Before you can use parts and materials you have marked for inspection, you must:

  • Inspect them and release those you have passed to your inventory for use. If the parts you receive into inspection are linked to work orders, VISUAL defers the issue until you release the parts.

  • Return those you have failed to the appropriate vendor.

  • If there are multiple order lines, you can return or release individual lines.

  • If you fail only part of a shipment you can

You can mark parts and materials for inspection at:

  • The line level in the Receiving window. It is necessary to mention the location of the warehouse where the parts are actually received for each line item.

  • The Part level in Part Maintenance.

Starting Receiving Inspection

Open the Receiving Inspection window by selecting the Receiving Inspection option from the Purchasing menu of the VISUAL main window.

VISUAL populates the Receiving Inspection window with all of the orders you have received and marked for inspection.
