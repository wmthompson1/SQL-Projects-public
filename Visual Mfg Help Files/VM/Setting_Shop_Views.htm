Setting Shop Views

Use the Set Shop View option to limit the resources displayed to a subset of all possible resources. You can define shop views in Shop Resource Maintenance.

  1. Select Set Shop View from the View menu.

The Shop View dialog box appears.

  1. Choose a shop view from the drop down list box to display the list of resources belonging to that shop view.

Select ***Full View*** to display load and capacity for all defined resources that were processed in the current schedule ID. When you first open the Scheduling window, it defaults to showing the last shop view that appeared on that computer.

  1. Click Ok to select that view, or Cancel to close the dialog box without selecting a view.
