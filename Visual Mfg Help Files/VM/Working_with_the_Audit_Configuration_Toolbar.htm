Working with the Audit Configuration Toolbar

The Audit Configuration toolbar is unique to the Audit Maintenance function. While some buttons may be familiar to you, there are some buttons you have not encountered elsewhere within the VISUAL application.

Button

Label

Description

Save

Click this button to save changes to your Audit Configuration for the current table only. When you click this button, the current table’s red icon in the list pane turns blue.

Save All

Click this button to save changes to all the database tables that you have configured for Audit tracking. When you click this button, all red database table icons in the list pane turn blue.

Exit

Click this button to exit the Audit Configuration window and return to Audit Maintenance.

Quick Filter

If you want to enable the Quick Filter functionality for the Audit Configuration window.

Clear Quick Filter

If you have applied a filter, and you want to remove it, click the Clear Quick Filter button.

Write Trigger

Click this button to write all the triggers for the current database table. Remember that this applies only to the current table.

Write Triggers

If you want to write all the triggers for all selected tables, click this toolbar button. This affects all selected Tracked database columns, regardless of the database table.

Select All

To enable tracking for all rows on the Audit Configuration table, click this toolbar button. If you want to track most of the table’s information, it may be easier for you to select all, then clear the rows you do not want to track.

When you select this function, VISUAL also selects the Insert, Update, and Delete options for all database table columns.

Unselect All

To remove tracking from all rows on the Audit Configuration table, click Unselect All.

Restore Selections

If you want to return to the most recent tracking selections, click this toolbar button. Keep in mind that this only reverts to the settings from the most recent Save.
