Copying Shipping Addresses

You can copy an address to create a new address.

To copy information:

  1. Select Sales, Customer Maintenance.

  2. In the Customer ID field, specify the ID of the customer for whom you are adding addresses.

  3. In the table, select the address to copy.

  4. Click Copy.

  5. To create a new line, click Insert.

  6. Click in the new line and click Paste.
