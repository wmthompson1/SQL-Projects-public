Maintaining Goods Under Inspection

Marking to Return or Release Inspected Receivers

When you have finished inspecting your parts and materials, you must either return them or release them.

To mark a receiver to return to vendor or release to manufacturing:

  1. In the Receiving Inspection window, select the appropriate line by clicking on the row header.

The row appears highlighted and the toolbar buttons appear.

  1. If you need to return the product on the line to the vendor, click the Mark to Return toolbar button.

If you want to mark the line for release, click the Mark to Release toolbar button.

Return or Release appears in the Action column for the toolbar you clicked. A check mark appears indicating you have not saved the action yet.

  1. Click the Save toolbar button.

VISUAL returns the receiver or releases the parts into inventory.

Performing Partial Returns/Releases

Often you will find that you pass or fail only part of an order or part of individual lines on an order. If this occurs, you must perform a Partial Return or Partial Release.

To perform partial returns/releases:

  1. In the Receiving Inspection window, select the appropriate line by clicking on the row header.

The row appears highlighted and the toolbar buttons appear.

  1. Click the Partial Return/Release toolbar button.

The lines for the order you selected appear in the lower table.

The following information appears for each line:

  • Part Identifier

  • Order Quantity

  • Received Quantity

  • Quantity of parts currently in inspection

  • Release Quantity

  • Return Quantity

If you are working with Traceability or Dimensional Inventory, additional information appears.

  1. Enter the return and release quantities for each line in the # Pcs to Return and # Pcs to Release columns.

  2. If you want to specify a different warehouse into which you want VISUAL to transfer your passed inventory, enter the warehouse and location in the New Warehouse ID and New Location ID columns.

  3. Click the Save button.

VISUAL transfers the quantity you passed to the part’s default location, or the location you specified.

For more information on returning material to vendors, click here.

Clearing Selections

If you have made selections but not saved them, you can clear your selections by clicking the Clear Selections toolbar button.
