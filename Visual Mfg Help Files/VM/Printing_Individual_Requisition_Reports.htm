Printing Individual Requisition Reports

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site in which you created the requisition. If you are licensed to use as single site, this field is unavailable.

  2. Click the Requisition ID arrow and select the requisition to print.

  3. Select Print Requisition... from the File menu or click the Print Requisition button on the main toolbar.

The Print dialog box appears.

  1. From the list-box, select an output for the report:

Print - Select Print to output the file to a printer.

View - Select View to view the report before printing.

File - Select File to save the information to a text file.

E-Mail - Select E-Mail to send the report in a Rich Text Format (.RTF) through electronic mail.

  1. Click the Ok button to output the report to the specified destination.

If you selected E-Mail, when you generate the report the system attaches the file to a Microsoft Outlook e-mail. For more information on addressing and sending the e-mail, refer to your Microsoft Outlook user documentation.
