Editing Trace IDs

From within the Part Trace Maintenance window, you can edit the Trace ID, Quantity Assigned, Comment, and Expiration Date fields of the Trace IDs you have previously assigned and to which you have not reported a labor or material trace transaction.

To edit a Trace ID:

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site that owns the part trace profile. If you are licensed to use a single site, this field is unavailable.

  2. Click the Profile ID button and select the profile that contains the trace you want to edit.

The individual Trace IDs for the profile you selected appears in the table.

  1. Click the information you want to edit.

The row appears highlighted and the cursor appears in the information you select.

  1. Make the changes you want.

  2. Click the Save icon on the toolbar.

The Trace History Comments dialog box appears.

  1. Enter any comments you want regarding this trace edit.

  2. Click the OK button.

VISUAL tracks the changes you have made in the Trace History table.
