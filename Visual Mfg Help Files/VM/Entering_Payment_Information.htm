Entering Payment Information

Use the Payment tab of the Vendor Maintenance window to enter electronic payment information for vendors.

Enter the following information on the Payment tab:

Enter information in the following fields:

Bank Account ID - Select a Bank Account ID for the vendor.

Payment Method - Select the appropriate payment method. You can select File, BACS, or Check.

Payee Reference Code - This is for use with on-line bank payments. If you enter a number here, you usually do not have to send any other vendor bank information when making a payment.

Note: These three fields only appear if you have activated Withholding in Accounting Entity Maintenance. See Setting Electronic Payment Information.

Withholding Code - Enter the vendor’s withholding code.

Vendor Type - Select the appropriate type of vendor type, either Company (C) or Person (P).

Payment Priority - Enter the priority you want to use when making payments to this vendor.

Social Security No. - Enter the vendor’s social security number.
