Using the Order Management Window Substitute Parts Window

Substitute parts are inventory parts that you designate as reasonable replacements to other inventory parts. Use substitute parts to meet order demands when the part you originally intended to use is unavailable.

Define substitutes for parts in Part Maintenance. See Adding Parts to the Database.

From the Tools menu of the Order Management Window, select Substitute Parts.

The Substitute Parts for Part window appears.

The following fields in the header section summarize the current customer order:

Part ID - The ID of the primary part in the current customer order. This is the part you are replacing with the substitute.

Warehouse ID - The ID of the warehouse for the customer order line.

ShipTo ID - The customer’s default ShipTo ID. Enter a default ShipTo ID in Customer Maintenance under the General tab.

The following columns appear in the right pane of the window:

Substitute Part ID - The ID of the substitute part for the primary part.

Part Description - A description of the substitute part.

Unit Price - The price per unit of the substitute part.

Unallocated Demand - The available quantity of the part that you have not yet allocated to demand.
