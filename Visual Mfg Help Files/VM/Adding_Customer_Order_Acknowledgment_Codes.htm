Adding Customer Order Acknowledgment Codes

When customers send orders to their suppliers electronically, they often require a return document verifying the receipt and acceptance of the order. In Electronic Data Interchange (EDI), this document is called the Order Acknowledgement, typically referred to as the 855 X12 Transaction or the Purchase Order Response Message in EDIFACT.

  1. Select Admin, Application Global Maintenance.

  2. Select Maintain, CO Acknowledgement Codes. X12 codes are displayed by default. You can delete any of these codes.

  3. Click Insert.

  4. In the Code field, enter a unique alphanumeric identifier for the code.

  5. In the Description field, enter a description of the code.

  6. Click Save.
