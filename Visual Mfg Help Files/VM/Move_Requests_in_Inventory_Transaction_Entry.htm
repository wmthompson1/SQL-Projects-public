Move Requests in Inventory Transaction Entry

If you are licensed to use WIP Inventory functionality, you can create a move request directly in Inventory Transaction Entry. You can create move requests for these transaction types:

  • Receipt by WO

  • Issue

  • Receipt Return

  • Issue Return
