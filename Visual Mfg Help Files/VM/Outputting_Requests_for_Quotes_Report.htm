Outputting Requests for Quotes Report

You can send printed requested quotations to prospects/vendors. You can use the standard format provided with the system, or up to 3 user-defined formats.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site in which you created the vendor RFQ.

  2. To print a particular RFQ, click the Request ID button and select the RFQ to print.

  3. Select Print Request for Quote from the File menu.

The Print dialog box appears.

  1. Select the appropriate option:

Print Current Request for Quote - Prints the Request for Quote currently in the Vendor RFQ Entry for all vendors referenced by the RFQ.

Print Current Request for Selected Vendor - Prints the Request for Quote for the selected vendor currently in the Vendor RFQ Entry window.

Print All Open Requests - Prints the request for quote from following date. The request for quotes are ready from the following date, after the request for quotes print, the status changes to Printed.

Set Followup Date to - Sets the followup date for the quotes you have selected to print.

  1. If you select the Request for Quote following Date option, enter the Starting date.

  2. Click the Select Some button to select vendors and parts to print.

The Select Vendors and Parts to Print dialog box appears.

  1. Select the rows to print then click Ok.

  2. Select the format you want to use.

User-defined formats are available only if the forms have been created.

  1. If you want to send this report to all of the contacts in this customer’s E-mail Documents list, select the Send To Document Contacts option.

  2. Click the drop-down arrow and select how you want VISUAL to output the report. You can select:

Print – If you want to send the report to your printer, select the Print option.

View – If you want to view the report using VISUAL’s report viewer, select the View option.

File – If you want to send the report to text file, select the File option. VISUAL prepares your report as a RTF file and a dialog box appears prompting you to enter the location and file name for the file to be saved.

E-mail – If you want VISUAL to prepare the report and attach it to an e-mail, select the E-mail option. VISUAL prepares the report as a RTF file and attaches it to a Microsoft Outlook e-mail. For more information on addressing and sending the e-mail, refer to your Microsoft Outlook user documentation. Click the Send button when you are ready to send the message.

  1. If you want to attach a PDF (Portable Document Format) file to your e-mail instead of a RTF file, select the PDF Format check box in the Type section.

  2. Select OK.

A standard Windows Print dialog box appears, which allows you to select the number of copies to print.

  1. Make the appropriate selections and click Ok to print the report.
