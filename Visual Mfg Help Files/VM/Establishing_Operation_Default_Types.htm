Establishing Operation Default Types

This feature gives you the ability to predefine all of the attributes of an operation so that information about the operation can be easily copied during operation add or edit in the Manufacturing Window. You can have up-to 99 different operation Types, each with different attributes. When you change the Type of an operation, in Manufacturing Window, you will be prompted whether or not to have the attributes copied.

Only non-null data fields will be copied (i.e., those fields which were defined in the operation type).

To add operation Types do the following:

  1. Select Operation Types from the Edit menu of Shop Resource Maintenance. It is NOT necessary to be viewing a Resource ID in order to add Operation Types.

  2. Choose the Insert button. A new line will be marked with an arrow.

  3. Enter an up-to 15 character Type ID and a Description.

  4. Select the Save button.

  5. Highlight the Type you want to define and select the Edit button.

  6. Enter all data fields you want to pre-define for that type. Remember, when you enter this type in the Operation window during Manufacturing Window entry, data from these fields are copied over to that operation from this window.

  7. Select Ok when you are done.

  8. If you want to, highlight another Type line and add operation default information for it. Continue until done and return to the previous Shop Resource Window.
