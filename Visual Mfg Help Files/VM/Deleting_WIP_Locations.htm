Deleting WIP Inventory Locations

You can delete a WIP inventory location only if it is not specified on a move request or WIP transactions.

  1. Select Inventory, Warehouse Maintenance.

  2. If you are licensed to use multiple sites, click the Site ID arrow and select the site. If you are licensed to use a single site, this field is unavailable.

  3. Click the Warehouse ID browse button and select the warehouse from which you are deleting WIP inventory locations.

  4. Select Edit, Add WIP Locations.

  5. Select the row that contains the location that you are deleting.

  6. Click Delete. The location is marked for deletion.

  7. Click Save. The location is deleted.
