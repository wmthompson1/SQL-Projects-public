Adding Variable Information

Variables are placeholders within your database, and may change during the time that the Formula is executed. This allows you to set a fixed name to information that changes during the execution of a formula. For example, if you have a counter within your formula, you do not need to know what value the counter is set to, but that the counter is referenced each time VPB runs the formula.

To add Variable tab information:

  1. On the Formulas window, click the Variables tab. Variable information appears in the field. If you are working with a new formula, the Variables table is empty.

  2. On the VISUAL Price Book main toolbar, click the Insert Row button to activate the next line on the Variables table.

  3. Enter an Identification Code for the current Variable line. This helps you identify the variable within the Formulas tab.

  4. From the drop-down list, select the Data Type for the current Variable within the Formula. Make your selection from: Boolean, Date, Number, or String.

  5. Based on the selection made in the Data Type column, enter the Value for the variable. You may have to enter Boolean Value, Date Value, Number Value, or String Value.

  6. On the VISUAL Price Book main toolbar, click the Save button to save changes to the current formula.
