Searching on Initial Entry

To display the Query by Example dialog box when you manually access an advanced browse dialog:

  1. For the field where you want to use automatic browsing, click the browse button.

  2. Select Options, Search on initial entry.

If you use auto browsing, then the Query by Example is not displayed if you open the browse dialog by typing in a field and then pressing TAB. The Query by Example dialog is only displayed if you click the browse button to open the advanced browse dialog.
