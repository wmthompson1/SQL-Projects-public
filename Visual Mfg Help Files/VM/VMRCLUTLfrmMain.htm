Using the Reclass Utility

To use VISUAL's reclass utility:

  1. Select Automatic Reclass Utility from the Inventory menu.

The Automatic Reclass Utility window opens.

  1. Select the schedule you want to set up.

You can set up:

Automatic at this frequency - If you want to run a reclass at specified intervals, select the Automatic at this frequency option button and enter the interval time in the hours, minutes, and seconds boxes.

Automatic daily at this time - If you want to run a reclass at a specific time of the day, select the Automatic daily at this time and enter the time of day you want VISUAL to perform the reclass.

Manual - If you want VISUAL to perform a reclass only when you want, select the manual option.

  1. When you are ready to begin the setup you have entered or if you have selected Manual, click the Run toolbar button.

If you have selected Manual, VISUAL runs a reclass of you inventory.

If you have selected either Automatic option, VISUAL begins the countdown to the next reclass.

To stop a the timer or a reclass, click the Stop toolbar button.
