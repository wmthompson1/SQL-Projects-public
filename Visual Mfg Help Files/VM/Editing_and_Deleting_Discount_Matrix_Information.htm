Editing and Deleting Discount Matrix Information

Often, a Discount Matrix changes or expires, perhaps based on seasonal items. If the discount matrix information changes, you can edit the table. After a discount price break has expired, you may want to remove the Discount Matrix information from the table.

Editing Discount Matrix Lines

  1. On the Discounts window, select the Context and ID for the Discount Matrix you want to edit. The discount Matrix table information appears.

  2. Click the field on the table that you want to modify. VISUAL highlights the line.

  3. Make the appropriate change to the table information. VISUAL flags the row as changed by inserting a check mark in the row headers that you have altered.

  4. On the VISUAL Price Book main toolbar, click the Save button to save the changes to the discount matrix.

Deleting Discount Matrix Lines

  1. On the Discounts window, select the Context and ID for the Discount Matrix you want to delete. The Discount Matrix table information appears.

  2. Click the row headers of the lines you want to delete. VISUAL highlights the line. You can select multiple rows on the table by holding down the CTRL key on your keyboard when clicking the row header.

  3. On the VISUAL Price Book main toolbar, click the Delete Row button. VISUAL flags the selected rows for deletion with a black "X" in the row header. To clear a deletion flag, click the row marked for deletion then click the Delete Row button again.

  4. On the VISUAL Price Book main toolbar, click the Save button to delete the flagged rows from the current Discount Matrix.

    You can delete an entire Matrix by clicking the Delete button on the VISUAL Price Book main toolbar. This affects only the currently opened Matrix
