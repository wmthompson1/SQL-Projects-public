Setting Up VISUAL to Store Macros in the Database

To set up VISUAL to store macros in the database:

  1. Select Admin, Application Global Maintenance.

  2. Click the Defaults tab.

  3. In the Macros section, select the Store Macros in Database check box. When you select this check box, any macros you create are stored in the database. When you run macros from a VISUAL executable, the macros stored in the database are run. Any macros stored on the workstation are ignored.

  4. Click Save.
