Printing Invoices

To generate and print invoice:

  1. If the Invoice Forms window is not open, select Invoice Forms from the Sales menu.

The Print Invoices window appears.

  1. From the File menu, select the printing option you want. You can select:

Print – If you want to print all of your invoices in one run, select Print.

Print One Invoice – If you want to print a single invoice, select Print One Invoice. If you are licensed to use multiple sites, click the Site ID arrow and select the site whose invoices you are generating. If you are licensed to use a single site, this field is unavailable.

Print Duplicate Invoices – If you want to print duplicates of invoices you have already printed, select Print Duplicate Invoices. If you are licensed to use multiple sites, click the Site ID arrow and select the site whose invoices you are generating. If you are licensed to use a single site, this field is unavailable.

If you selected to combine packlists, the Packlists - Eligible for Invoice dialog box appears. 

  1. Select the packlists you want to combine on this invoice and click the OK button.

The Customer Orders-Eligible for Invoice dialog box appears. 

  1. Select the customer order for which you want to print an invoice and click the Select and Close buttons.

The Print dialog box appears.

  1. Click the Ok button to output the invoice to your default printer.
