Copying Tests from Engineering Masters

You can copy tests from existing engineering masters from the Work Order and Operation Input window.

  1. Click the Copy Tests from Master button.

The Copy Tests from Eng Master dialog box appears.

There are two sections in this dialog box: the From section and the To section. The current work order ID appears in the To section.

  1. Enter the part ID for the engineering master from which you want to copy the tests. You can click the Part ID button and select a part from the Parts table.

  2. Select the part engineering ID and the sub ID from the Part Eng ID and Sub ID drop down boxes.

  3. Click the Save button.

VISUAL copies the tests from the existing engineering master to the current engineering master.
