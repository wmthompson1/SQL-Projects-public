What is Interbranch Transfer Shipping Entry?

After you have entered and saved an IBT, use the Interbranch Transfer Shipping Entry window to ship the IBT. This window closely resembles the Shipping Entry window used to ship customer orders. Interbranch Transfer Shipping Entry supports many of the same features as Shipping Entry. However, IBT Shipping Entry does not support:

  • Editable Packlists.

  • Print Trace Labels.

  • Print Packlist Report.

  • Pallet Details and Bills of Lading.

  • Copy Estimated Freight.

  • RMAs
