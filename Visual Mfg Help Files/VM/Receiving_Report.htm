Receiving Report

Use the Receiving Report to review purchase receipts.

  1. From the Purchasing menu, select Receiving Report. The Receiving Report window appears.

  2. In the Receiving Report window, click the Site ID(s) arrow and select the sites to use.

  3. In the Starting Date field, enter the first day of the period you would like to close. In the Ending Date field, enter the last day of the period you would like to close.

  4. In the PO Receipt Status, select Scheduled.

  5. Leave the fields in the Selection Criteria blank to return all scheduled receipts for the month.

  6. Click the Print button. The Purchased Part Receiving Report – Scheduled appears.

  7. Run the report again, this time selecting Actual in the PO Receipt Status box.

  8. Click Print. The Purchased Part Receiving Report – Actual appears.

  9. Compare the Purchased Part Receiving Report – Actual to the Purchased Part Receiving Report – Scheduled.

If there are scheduled deliveries that do not appear on your actuals report, you may want to investigate the discrepancy.
