Entering New Master Production Schedules

The entry of Master Production Schedules is a simple process. Enter schedules with varying horizons and settings so you forecast demand and maintain planning information for a part in several different ways.

  1. From the Edit Menu, select Master Production Schedules. The Master Production Schedules dialog box appears. All current master production schedules appear in the line item table

  2. Click Insert to begin adding line item information for the new master production schedule.

  3. Enter information into the following columns:

Schedule ID - A unique identifier for the schedule.

Description - A description of the schedule.

Rank - An assignment of importance to the schedule. VISUAL assigns a rank of 50 automatically to every new master production line item that you enter. You can enter any numerical value up to four digits.

  1. Click the Save button to save the new master production schedules.
