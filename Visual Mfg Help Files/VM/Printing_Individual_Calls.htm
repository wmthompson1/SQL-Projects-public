Printing Individual Calls

You can print UPM calls one at a time.

  1. From the File menu, select Print Call Detail or click the Print button on the main toolbar.

The Print Call Detail dialog box appears.

  1. If you know the ID of the call you want to print, enter the Call ID in the Call ID data field.

If you want to search for the Call ID, click the Call ID button to view a table containing all Call IDs in the database.

  1. Select how you want to output the report:

Print - Select Print if you want VISUAL to print the report. After clicking Ok, you can specify print preferences and begin printing.

View - Select View if you want to view the report. After clicking the Ok button, VISUAL opens Centura Report Builder and displays the report.

File - Select File if you want to save the report to file. After clicking Ok, VISUAL allows you to name the file; VISUAL then saves the file in your VISUAL directory.

E-mail - Select this option to send the document through electronic mail. When you generate the report the system attaches the file to a Microsoft Outlook e-mail. For more information on addressing and sending the e-mail, refer to your Microsoft Outlook user documentation. If you would like to send a PDF, select the PDF Format check box. VISUAL starts the distiller, converts the document to PDF, and attaches it to the e-mail.

  1. Click the Print Setup button to specify which printer you want VISUAL to use when printing Unplanned Maintenance reports.

  2. Click the Ok button to begin the printing process.

  1. Click the Cancel button to return to the Unplanned Maintenance window.
