Starting Work Order/Operation Input

Work Order/Operation Test Input is available from the Manufacturing Window.

With the appropriate work order in the Manufacturing Window, select Work Order/Operation Input from the Eng/Mfg menu.

The Work Order and Operation Input window appears.

The window displays the part ID and description, has a Work Order button for selecting a work order, data fields for manually entering a work order, data fields showing who the engineer of the work order, the engineered date, drawing ID and drawing revision number.

There are two push buttons for copying tests from the part’s engineering master or from another work order for the same part.

The line item table lists the operations of the work order, including subassemblies, with the following columns:

Sub ID - The subassembly number of the operation.

Sequence No - The sequence number of the operation.

Resource ID - The resource ID for the operation. This information comes from the database.

Test ID - The test ID assigned to the operation.

Test Description - The test description assigned to the operation.

Test Status - The status of the test assigned to the operation.

Start Quantity - The calculated start quantity of the operation.

Completed Quantity - The quantity completed for this operation.

Reported Quantity - The quantity reported to Quality.

Quantity to Report - The quantity to currently report to Quality. You can edit this field, but it is usually set to the difference between the Completed Quantity and the Reported Quantity (or zero if this difference is negative).

Start Hours - The calculated run duration of the operation in hours.

Completed Hours - An estimated time in hours spent on this operation determined from the run duration and quantity completed.

Reported Hours - The time for this operation in hours reported to Quality.

Hours To Report - The time in hours to currently report to Quality. You can edit this field, but it is usually set to the difference between the Completed Hours and the Reported Hours (or zero if this difference is negative).

Service Resource - Operation information from the database.

Drawing ID - Operation information from the database.

Drawing Rev No - Operation information from the database.
