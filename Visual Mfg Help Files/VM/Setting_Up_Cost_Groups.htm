Setting Up Cost Groups

Cost groups are groups into which you place various types of costs.

Typically you will want to set up three types of cost groups for a department within a project—Material, Labor, and Other Direct Charges (ODC).

To set up cost groups:

  1. From the Project menu, select G/L Burden Mapping Maintenance.

  2. From the Maintain menu, select Cost Groups....

The Cost Group Maintenance table appears. By default, the three typical cost groups appear automatically.

  1. To enter a new cost group, click the Insert button.

A new line appears highlighted in the table and the cursor appears in the ID column.

  1. Enter the identification you want to use for this cost group.

  2. Click in the Description column and enter a description for this cost group.

  3. Click in the Type column and click the arrow that appears. Select the cost group type you want to use for this cost group.

You can select Labor, Material, or ODC.

  1. Click the Save button.

  2. When you have finished entering cost groups, click the Close button.

The Cost Group Maintenance window closes.
