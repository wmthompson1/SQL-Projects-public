What is Message Maintenance?

Message Maintenance creates and maintains records of specific events and sends notification that the event has occurred.

Message Maintenance can send notification in the following ways:

  • Pop up messages that appear on your computer screen

  • Email to specified people

  • Printed notification to your printer

  • SQL and Oracle DML statements and queries

This feature is helpful if you want to:

  • Track certain events over a period of time

  • Need immediate notification that an event has occurred.
