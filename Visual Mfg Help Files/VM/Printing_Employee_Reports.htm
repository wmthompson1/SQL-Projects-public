Printing Employee Reports

There are a number of options available for outputting employee information. By using the Print command, you can output a complete listing of employee information.

  1. Select Print from the File menu.

The Print dialog box appears.

If necessary, you can alter the printer setup from the print dialog box by selecting the Print Setup button.

  1. Enter the Employee ID for the first employee you want to print in the Beginning Part ID field. Enter the last employee to print in the Ending Employee ID field.

Leave these fields blank to print all employees.

  1. Choose the output destination by selecting the arrow and selecting one of the following:

Print - To output the file to a printer. The file prints on the default printer.

View - To see what the printed output will look like before it is printed. When the information appears in the window, you can scroll through it and print it if you need a hard copy.

File - To save the employee information to a comma-separated text file, one part to a line.

You can use this file for transferring data to another database, spreadsheet, or other product.

Click OK to bring up a dialog box for the entry of a file location and name.

Enter a location for the file and whether or not the field names are to be included in the text file. You can now edit the file outside of VISUAL using any text editor.

  1. Select the Bar Codes check box to print bar codes for the employee information.

  2. Select the Print Inactive check box to print both active and inactive employees.

  3. Click Ok to output the information.

If you selected the Print option, clicking on Ok brings up a standard Windows printer dialog box allowing you to select the number of copies to be printed.

  1. Make the desired selections and click Ok.

Windows prints your information.
