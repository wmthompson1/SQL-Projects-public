Editing Existing Warehouses

You can edit warehouse information using Warehouse Maintenance.

  1. Enter the Warehouse ID in the Warehouse ID field, or select the warehouse from the Warehouses Table.

  2. Make any necessary changes to the warehouse information.

You can change any field in the Warehouse Maintenance window except the Warehouse and Location IDs. Changing the Warehouse ID creates a new warehouse.

  1. Click the Save button to save your changes.
