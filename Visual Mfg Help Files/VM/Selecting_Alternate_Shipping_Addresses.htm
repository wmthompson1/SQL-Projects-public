Selecting Alternate Shipping Addresses

You can select an alternate shipping address that you set up in Customer Order Entry.

  1. Select Shipping Address from the Edit menu.

The Shipping Addresses dialog box appears.

  1. Select the alternate shipping address and click Ok.

The alternate shipping address appears in the Sold To field below the Sold To address. You may have to scroll down to view the alternate shipping address.

  1. Click Save to save any changes to the Customer Shipments window.
