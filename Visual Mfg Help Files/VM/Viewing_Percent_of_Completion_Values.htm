Viewing Percent of Completion Values

Percent of Completion or POC is a method of recognizing revenue and profit, based on the percentage of completion against the job, prior to transferring the title of the goods to your customer.

For more information on the methods of calculating POC, refer to the American Institute of Certified Public Accountants (AICPA) SOP 81-1.

  1. In the Project Summary window, select the project whose percent of completion values you want to view.

  2. From the Info menu, select the Percent of Completion Values option.

  3. If you want to view a different Version ID of this project, click the Version ID drop-down arrow and select the version you want to view.

  4. If you want to specify a date limit for your completion values, use the Actual Costs as of calendar button.

  5. When you have finished viewing percent of completion values, click the Close button.
