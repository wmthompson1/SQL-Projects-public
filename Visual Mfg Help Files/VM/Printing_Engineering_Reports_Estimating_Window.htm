Printing Engineering Reports

The Engineering Report provides you with a screen view or print out of a selected Quote Master or Engineering Master. Information on the report includes operations, materials and services.

  1. Select the Quotation Master or Engineering Master you want to output.

  2. Select Print Engineering Report from the File menu.

The Engineering Report dialog box appears.

  1. Choose the output destination by selecting the arrow next to the output field. Select one of the following:

Print - To output the file to a printer, select this option. VISUAL sends the file to the selected default printer.

View - To see what the printed output will look like before it is printed, select View. When the information appears on the screen, you can scroll through it and print it if you need a hard copy.

File - To save the information to a text file, select File from the pull down menu.

  1. Select the Short From check box to print an abbreviated version of the report.

  2. Click Ok to output the report.

You can also print this report from the Estimating Window by selecting a quote line item (highlight) and clicking on the Eng Report push button.
