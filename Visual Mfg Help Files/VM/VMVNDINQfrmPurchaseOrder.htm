Viewing Purchase Order Information

From the Search results dialog box, double-click the record you want to view. You can also click the View Order button.

The selected item information appears in the Vendor Inquiry window.

You can view the following information from the Vendor Inquiry window:

  • Receipts

  • Back Orders

  • Work Order Status

  • Vendor Month-to-date and Year-to-date statuses

  • Purchase Order Requisitions

  • Purchase Order Request for Quotes
