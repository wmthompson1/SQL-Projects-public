Using the Order Management Window WIP/VAS Window

Before you begin using the WIP/VAS child window:

1. Activate support for VISUAL WIP/VAS in Application Global Maintenance under the Order Management tab, and then create WIP/VAS types.

2. Use Warehouse Maintenance to establish WIP/VAS support in the warehouses of your choice.

3. Then specify which parts within the warehouse are eligible for WIP/VAS. See Warehouse Maintenance for more information.

WIP/VAS (Work In Process/Value Added Services) are customization services performed on components or inventory. While not completely manufacturing in nature, WIP/VAS requires the management of inventory moving in and out of service sections in a warehouse. WIP/VAS starts during order entry, when a customer requests one of these specialized services. WIP/VAS is only for warehouses/DCs with the capability to manage the services. WIP/VAS specifications are also available at the customer order line level.

To access this window, in the Order Entry window, open the order for which you want to order WIP/VAS .

From the Tools menu, select WIP/VAS.

The WIP/VAS window appears.

If the part in the customer order is eligible to receive WIP/VAS in the warehouse, the available WIP/VAS appear in the right pane. If the right pane is empty, then the part in the order is not set up as a WIP/VAS part in the warehouse. You can easily do this by opening Warehouse Maintenance, and entering the part as a WIP/VAS part. See Warehouse Maintenance for more information.

Select the Customer WIP/VAS check box to view WIP/VAS for the customer.

If VISUAL finds any WIP/VAS, the following columns appear in the right pane:

WIP/VAS - The WIP/VAS ID of the work in process or value added service in the warehouse for the focus part. Enter WIP/VAS IDs in Application Global Maintenance.

Description - A description of the WIP/VAS. Enter WIP/VAS Descriptions in Application Global Maintenance.

Unit Price - The price per unit that you charge the customer for this customization service. Enter WIP/VAS Unit Price information in Application Global Maintenance.

Prompt:

1.Confirm

2. Do not ask

3. Order

A customer’s WIP/VAS order instruction. If the prompt is Confirm, you must confirm the ordering of WIP/VAS with the customer; if the prompt is Do not ask, do not ask the customer and do not order the WIP/VAS; if the status is Order, you may order the WIP/VAS without asking the customer.
