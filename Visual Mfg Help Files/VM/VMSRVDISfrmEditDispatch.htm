Editing Dispatch Packlists

After VISUAL creates a dispatch pack lists for service parts, you can edit its contents, specifically the quantity dispatched.

  1. Either by Order ID or Dispatch ID, select the service dispatch pack list you want to edit.

  2. After VISUAL generates the pack list and dispatches the parts or service, click the Edit Pack list button on the main toolbar.

The Vendor Service Dispatches dialog box appears. For more information on dispatching parts for service, click here.

The Order ID and Pack List ID appear in read-only reference fields at the top of the dialog box.

  1. If appropriate, adjust any of the header fields.

Actual Dispatch Date - If appropriate, adjust the date of the dispatch pack list.

FOB - If appropriate, select an FOB code for the dispatch.

Ship Via - If appropriate, select a Ship Via code for the dispatch.

Carrier - If appropriate, select a carrier code for the dispatch. Use Application Global Maintenance to add carriers to the database.

If applicable, in the line item table below, adjust the quantity dispatched in the Quantity Dispatched column.

An identical "Quantity Dispatched" field appears next to the first Quantity Dispatched field (see above). The second field is read-only that displays the dispatch quantity of the pack list line. Use the first field (see step 4.) to adjust the dispatched quantity.

  1. If applicable, modify the Weight U/M, Promise Date, and Desired Receive Date.

  2. After making all adjustments, click the Save button to save the modified dispatch pack list.

If the "Ask for Pallets Details on Save" option is active, the Dispatch Pallet Details dialog box appears, allowing you to modify pallet lines before you save the dispatch pack list.
