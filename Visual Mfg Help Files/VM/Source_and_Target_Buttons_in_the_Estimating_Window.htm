Source and Target Buttons in the Estimating Window

When you are viewing a transaction, you can use the source and target toolbar buttons to view related transactions. Use the source button to view the previous transaction in the lifecycle. Use the target button to view the next transaction in the lifecycle.

In the Estimating Window, the Source button is not displayed. The Target button is labeled Customer Order. It is available if you generated a sales order from the estimate. Click the Target button to open the sales order.
