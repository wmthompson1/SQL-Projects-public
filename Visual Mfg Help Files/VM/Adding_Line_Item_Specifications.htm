Adding Line Item Specifications

Line item specifications are specific to an individual line item in an order. There can only be one specification per line item.

  1. With the appropriate purchase order open in the Purchase Receipt Entry window, highlight the line item for which you want to add the specification.

  2. Select P/O Line Item Specifications from the Notes menu.

The Line Spec dialog box appears.

  1. Enter the information for the line item.

You can enter an unlimited amount of information.

Use the Paste button to paste text that you copied from another location into the box.

Use the Copy button to copy text in the dialog box to the clipboard.

The entire contents of the dialog box is copied to the clipboard. If you are pasting information into the dialog box, the current text of the dialog box is replaced with the incoming text. If you wish to paste without replacing the entire text, use the Shift-Insert key combination.

  1. Click the Close button to close the dialog box and return to the Purchase Receipt Entry window.

  2. Click the Save button to save the specifications.
