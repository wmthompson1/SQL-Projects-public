Viewing and Adding Specifications

You can view and add Line Specs, Invoice Specs, and Billing Specs in Progress Billing Entry.

To add or view specifications for customer order line items in Progress Billing, select the line then click the Line Specs button.

To add or view invoice specifications for customer order line numbers in Progress Billing, select the line then click the Invoice Specs button.

To add or view billing specifications for customer order line numbers in Progress Billing, select the line then click the Billing Specs button.
