Source and Target Buttons in Purchase Requisition Entry

When you are viewing a transaction, you can use the source and target toolbar buttons to view related transactions. Use the source button to view the previous transaction in the lifecycle. Use the target button to view the next transaction in the lifecycle.

In Purchase Requisition Entry, the Source button is not displayed. The Target button is labeled Purchase Order. It is available if you generated a purchase order from the requisition. Click the Target button to open the purchase order.
