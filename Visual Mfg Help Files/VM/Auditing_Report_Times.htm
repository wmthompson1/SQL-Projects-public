Auditing Report Times

To track the duration of reports, select the Audit Report Times check box in the Audit Configuration window. The start time and end time of any report run is recorded. If you select this check box, all reports are tracked. You cannot choose which reports to track.

  1. Select Admin, Audit Maintenance.

  2. Select File, Configure Auditing.

  3. Select the Audit Report Times check box.

  4. Click Save.
