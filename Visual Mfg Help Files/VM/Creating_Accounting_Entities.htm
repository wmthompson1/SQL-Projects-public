Creating Accounting Entities

If you upgraded from a previous version, you defined one accounting entity ID during the upgrade process.

If you have not exceeded the maximum number of accounting entities allowed by your license, you can create additional accounting entities.

To create an accounting entity ID:

  1. Select Admin, Accounting Entity Maintenance.

  2. Click the New button.

  3. In the Entity ID field, specify the accounting entity ID. We recommend that you do not use single quotations (‘), forward slashes (/), or commas (,) in your entity ID. Using these characters may cause issues with integrations to other products through Infor ION.

  4. Click the Save button.
