Shipping the Repaired Materials

To ship the repaired materials:

  1. Select Sales, Shipping.

  2. Click the Order ID browse button and select the order to ship. If you selected the Ship from Original Order check box, select the original customer order. If you cleared the Ship from Original Order check box, select the new order created for the RMA.

  3. In the RMA ID field, select the ID of the RMA.

  4. Specify the quantity of part to ship. Perform one of these steps:

  5. If you are shipping a standard part, specify the quantity in the Ship Qty column.

  6. If you are shipping a piece-tracked part, specify the quantity in the Ship Pieces column.

  1. Click Save.

In the RMA Entry window, the Shipped Date is inserted. The RMA status is closed.
