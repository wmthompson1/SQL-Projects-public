Specifying Cost Categories for Departments

Departments and Cost Categories determine where VISUAL charges your costs in your projects and general ledger.

Default Cost Categories for Transfers:

For each department you use, establish three default cost categories: material, labor, and other direct charges (ODC). VISUAL maintains a default department in the WBS maintenance window for each project. When project transfers are processed VISUAL obtains the default department from the associated WBS of the inventory being transferred or issued and obtains the applicable transfer cost categories from the Department table.

To specify Cost Categories for departments:

  1. From the Maintain menu of the G/L Burden Mapping Maintenance window, select  Departments....

The Department table appears.

  1. Select the department whose Cost Categories you want to specify and adjust the table using the scroll bar until the Cost Category columns appear.

  2. For each of the following, double-click the row header and select the Cost Category you want to use:

Material Cost Category

Labor Cost Category

Service Cost Category

Only the appropriate categories for the column you click appear.

  1. When transactions have been recorded using the department ID, a check mark appears in the Has History column. If a check mark appears in the column, you cannot change the department ID’s cost categories. You also cannot delete the department.

If no check mark appears in the Has History column, you can change the cost categories associated with the Department ID. You can also delete the department ID if necessary.

  1. Click the Save button.

  2. When you have finished specifying Cost Categories, click the Close button.
