Activating the User Toolbar Function

Before you can display user toolbars, you must set up your preferences. To set up your preferences to display user toolbars:

  1. Select Admin, Preferences.

  2. Click Insert.

  3. Specify this information:

Section – Specify Visual Mfg.

Entry – Specify UserToolbar.

Value – Specify Y.

  1. Click Save.
