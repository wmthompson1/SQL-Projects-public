Setting the Vendor Service Calendar Exceptions

You can set the outside service vendor’s calendar exceptions using the Vendor Service Calendar Exceptions dialog box.

  1. Click the Vendor Calendar button in the Scheduling tab.

The Vendor Service Calendar Exceptions dialog box appears.

  1. Enter the dates the outside service vendor’s schedule varies in the Start Date and End Date columns.

  2. Enter the percent of the day worked in the Percent of Day Worked column. The percent in based on the outside service vendor’s operation run rate.

  3. Click Save to save any changes to the Vendor Service Calendar Exceptions dialog box.
