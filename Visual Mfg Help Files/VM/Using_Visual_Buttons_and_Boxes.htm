Using VISUAL Buttons and Boxes

There are a number of buttons and boxes throughout the Infor VISUAL software suite. It is important to know how these buttons and boxes function.

Using Option Buttons

Option buttons are small round buttons that you can click to select various options. You can only select ONE option button at a time; clicking on another radio button automatically deselects the one originally selected. VISUAL uses option buttons when only one selection at a time can be made. This contrasts with check boxes, where you can select a number of options at one time.

Using Check Boxes

Check boxes are small square boxes that you can click to select various options. Unlike option buttons, you can select any number of check boxes.
