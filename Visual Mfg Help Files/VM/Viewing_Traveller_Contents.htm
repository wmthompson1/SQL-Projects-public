Viewing Traveller Contents

All specified travellers are printed as one print job. Pages break for each leg and each work order to allow separate distribution.

A traveller has the following basic sections:

Work Order/leg Information - Printed at the start of each work order or leg, includes Work Order Base ID-Sub ID/Lot ID, Release and Want Dates, Drawing/Revision, and Quantity.

Customer Order Information - If selected, as described above, prints at the beginning of each work order.

Operation Information - For each operation, includes Resource ID, Specifications, Start/End Quantities, Setup and Run Hours, and Scheduled Start and Finish Date. Operation write-ins and barcodes print here, if selected.

Material Requirements - For each material requirement on each operation, Part ID, Unit of Measure, Quantity Required, Quantity Due, and a write-in area to indicate any issued material are printed beneath the operation.
