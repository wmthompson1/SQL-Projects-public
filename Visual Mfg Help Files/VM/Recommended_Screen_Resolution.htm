Recommended Display Settings

You must use the Smaller - 100% option in Windows Display settings. If you use any other Windows Display setting, the VISUAL interface will not display properly.

To set the Windows Display Settings:

  1. Click the Windows Start button and select Control Panel.

  2. Select Display.

  3. Click the Smaller - 100% option.

  4. Click Apply.

  5. Log off of Windows and then log back on.
