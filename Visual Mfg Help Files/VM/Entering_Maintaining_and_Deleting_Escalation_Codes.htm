Entering, Maintaining, and Deleting Escalation Codes

Use the Escalation Codes dialog box to enter and maintain escalation codes. Escalation Codes consist of a code (ID) and a description.

  1. From the Maintain menu, select Escalation Codes.

The Escalation Code dialog box appears.

  1. To enter a new code, click the Insert button.

  2. Enter information in the following fields:

Code - An identifier for the escalation code. You can enter a maximum of 15 alphanumeric characters.

Description - A description of the escalation code. Be detailed in your descriptions. For example, enter text that helps to clarify what the escalation indicates. You can enter a maximum of 40 alphanumeric characters.

  1. Click the Save button to commit the information.

  2. To modify information, place the mouse pointer in the appropriate column in the appropriate line and manually edit the information. When finished, click the Save button.

  3. To delete an escalation code, highlight the appropriate line and click the Delete button.

VISUAL places an X at the beginning of the line.

Click the Save button to complete the deletion.

  1. Click the Close button to return to the Unplanned Maintenance window.
