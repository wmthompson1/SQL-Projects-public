Printing Customer Bookings Report

The Customer Bookings report prints information for all Firm, Released, and Closed customer orders in a specified date range. Order amounts are totaled for each group in the sorting sequence. For example, if sequencing by Customer ID, the report displays total order amounts for each customer.

Back orders are not included in this report.

  1. Select Sales, Customer Bookings Report.

  2. If you are licensed to use multiple sites, click the Site ID(s) arrow and select the sites to include in the report. If you are licensed to use a single site, this field is unavailable.

  3. To generate a detail report, clear the Print Month to date/Year to date summary and Print Month to date summary check boxes. In a detail report, individual orders are listed based on the sequence that you select.

To generate a summary report, select one of these check boxes:

Print Month to date/Year to date summary

Select this check box to print month to date and year to date totals. If you select this option, then you can sequence the report by customer ID, customer name, or none. If you click None, then the report shows summary totals that are not broken down by customer.

If you select this check box, January 1 of the current year is inserted in the Starting Date field, and the current date is inserted in the Ending Date field.

Print Month to date summary

Select this check box to print month to date totals. If you select this option, then you can sequence the report by customer ID, customer name, or none. If you click None, then the report shows summary totals that are not broken down by customer.

  1. In the Sequence section, specify the sort sequence for the report. For example, if you click Part ID, the report shows customer orders for each part ID. If you selected the Print Month to date/Year to date summary or Print Month to date summary check boxes, you can sort the report by Customer ID, Customer Name, or None.

  2. If you are generating a detail report or a month to date summary report, specify the date range for the report in the Starting Date and Ending Date fields.

  3. In the Starting ID and Ending ID fields, specify the range of IDs or records to include in the report. To include all IDs in the report, leave the fields blank. To print the report for one ID, specify the same ID in the Starting ID and Ending ID fields. The label for these fields changes based on the sequence option that you select.

  4. In the Report Currency field, specify the currency for the report. Select ** Native ** to view amounts in each order's currency. If you select ** Native **, grand totals are not provided. If you select sites in more than one accounting entity, then the currency list is filtered to show the currencies that the entities share.

  5. From the output list, select the output format for the report. If you select email, select the PDF Format check box to attach a PDF to the email instead of an RTF file. See Report Output Options.

  6. Click Print/View.
