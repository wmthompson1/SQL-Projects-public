Allocating Supply Quantities to Outgoing Interbranch Transfers and Work Order Material Requirements

  1. With the Advanced Material Planning Window open and a Part ID selected, left-click anywhere on the Dependent Demand line. The Dependent Demand Details for Part dialog box appears.

  2. Select the Outgoing Interbranch Transfer or Work Order Material Requirement to which you want to allocate supply.

In the Order ID column, Interbranch Transfer Order IDs are preceded by WH.

  1. From the Edit menu, select Allocate. The Supply Links dialog box appears.

Any allocations of supply, or "supply links" that you have made to the interbranch transfer or work order material requirement appear in the line item table.
