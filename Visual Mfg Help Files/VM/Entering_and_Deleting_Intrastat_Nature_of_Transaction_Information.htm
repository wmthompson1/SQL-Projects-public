Entering and Deleting Intrastat Nature of Transaction Information

  1. Click the Nature of Transaction button.

The Nature of Transaction dialog box appears.

  1. Click the Insert button.

A new line appears in the line item table.

  1. In the Code column, enter a code for the nature of transaction.

  2. In the Description column, enter a description for the new nature of transaction code.

  3. Click the Save button to save the new information.

Deleting Nature of Transaction Codes

  1. Click the Nature of Transaction button.

The Nature of Transaction dialog box appears.

  1. Select the nature of transaction code you want to delete, and then click the Delete button.

An X appears in the row header beside the code you are deleting.

  1. Click the Save button to save the new information.
