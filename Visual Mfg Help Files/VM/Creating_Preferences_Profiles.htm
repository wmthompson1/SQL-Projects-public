Creating Preferences Profiles

Only a user with system administration privileges can perform this procedure.

In Preference Maintenance, a User ID does not necessarily correspond to a database user. The system administrator can create user templates that can be applied to actual database users. For example, if you want all sales representatives to have the same preference settings, the system administrator can set up a preference user ID named SALES REP, then apply the SALES REP preferences to the appropriate database users.

To create a preference template:

  1. Select Admin, Preferences Maintenance.

  2. Select Edit, Copy Preferences.

  3. Click the Copy From arrow and select the user whose preferences you would like to copy.

  4. In the Copy to User ID field, enter the name you want to use for the template.

  5. Click Copy. The system creates the new user.

You can edit the preferences for the new user profile as necessary.
