Viewing Order Information

The Material Planning Window allows you to directly access full viewing and editing for an order, without having to go to the VISUAL Main menu, and without having to explicitly type the order ID.

To view a demand or supply order in its native editor, select its line from the table, and select Display Demand (Peg) Order or Display Supply (due) Order from the Info Menu.

If the order is a customer order, Customer Order Entry opens. If the order is a work order, the Manufacturing Window opens. If the order is a purchase order, Purchase Order Entry application will be called up.

In all cases, VISUAL automatically calls the order into the window. Your viewing and editing capabilities in these windows depend on settings for your User ID in Program Security and Application User Permissions.

If you wish to make changes to the order and then observe the effects on netting, remember that you must save the order, and then select Refresh from the File menu. In the Manufacturing Window, make sure you recalculate quantities if you have changed quantities or added material requirements.
