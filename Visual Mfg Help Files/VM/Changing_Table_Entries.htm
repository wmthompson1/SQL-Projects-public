Changing Table Entries

  1. Select the unit of measure you need to edit in the To table.

The line selected becomes highlighted, indicating that you can edit it.

  1. Move the cursor to the appropriate entry and make the change.

A check mark appears to the left of the unit of measure.

The From table does not immediately reflect the changes you make in the To table. First, you must save your changes.

  1. Click the Save button.

    You cannot change the Unit Of Measure column. If you need to change a Unit Of Measure, you must add it again.
