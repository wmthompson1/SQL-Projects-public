Implementing Leadtime Processes

Complete these procedures to implement leadtime processes:

  1. Plan how to measure leadtime steps. You can use built-in functionality, SaveProcess macros, or a combination of the two.

  2. After you decide how to measure leadtime steps, specify the UpdateProcessActivity preference in Preferences Maintenance.

  3. If you are using SaveProcess macros, create the macros.

  4. In Part Maintenance, create leadtime processes.

  5. In Part Maintenance, assign leadtime processes to parts.
