Using Preference Maintenance

You can use the Preference Maintenance window to:

  • Load any existing preference values from the VISUAL.ini file

  • Set your own preferences

  • Print a list of your preferences

In addition to the above tasks, the system administrator can:

  • Set default preferences

  • Set preferences for individual users

  • Set up preference profiles and apply them to individual users

  • Set up scheduling preferences

  • Print a list of global and individual user preferences

To access Preference Maintenance, select Admin, Preference Maintenance.
