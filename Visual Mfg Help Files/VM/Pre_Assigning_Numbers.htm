Pre Assigning Numbers

Preassigning numbers allows you to create trace records for the selected part ahead of time. This is useful if you want to control numbering and/or size of lots for fabricated materials. If you have set Numbers Must Be Pre-Assigned in the part’s traceability profile, then this is the only method by which a new lot number can be created. You are not allowed to enter new records.

To use this option:

  1. Select Pre-Assign Numbers from the Edit menu.

The Pre-Assign Numbers dialog appears, showing the Part ID and Description of the current part.

  1. Enter the Pre-Assign Quantity.

Enter the total quantity of the part for which you want to preassign lot numbers.

Click Ok.

The standard Traceability dialog box appears.

  1. Create and/or Enter Lot Numbers and Assign Quantities.

The system uses any maximum lot size specified in the part’s traceability profile to create one or more new lines for the quantity you specify. As usual, Total Quantity shows the quantity you entered to assign, and Numbered Quantity shows how much has currently been assigned to new lot numbers.

Using this window at this point is very similar to its use in Purchase Receipt Entry. You can think of this function as a pre-receipt function; you are essentially creating receipt trace records for materials that have not yet been received. The quantities specified are shown as Quantity Assigned in the Part Trace Maintenance window.

  1. Click the Save button. 

The Traceability window automatically closes.

The new numbers are created, and are listed when receiving quantities of finished goods using Inventory Transaction Entry, or when receiving quantities of purchased goods, using Purchase Receipt Entry.
