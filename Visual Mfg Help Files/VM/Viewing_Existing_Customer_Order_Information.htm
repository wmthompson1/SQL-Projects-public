Viewing Existing Customer Order Information

  1. With an order open in the VISUAL DDP Customer Order Entry window with the appropriate Customer ID, double-click the icon in the Customer Order History window.

All existing orders for that customer appear.

  1. Highlight the order for which you want to view order information.

The following columns appears in the right pane:

Line Number - The order line as it appears in relation to the other lines, if any, in the order.

Part ID - The Part ID of the order line.

Part Description - The description of the Part ID.

Status - The status of the order: A (active) or C (closed).

Order Quantity - The quantity of the part on the order line.

Shipped Quantity - The quantity to the current date that you have shipped to the customer

Last Shipped Date - The date on which you last shipped an order to the customer

Amount Ordered - The dollar amount of the order line.

Amount Shipped - The dollar amount of what you have shipped from the order line.

  1. Click the plus box next to an order or double-click the order to expand the tree to view the PO ID, Part ID and quantity for each of the order lines.
