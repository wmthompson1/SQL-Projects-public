Engineering Report

The Engineering Report summarizes the legs and tasks in your project. To output an engineering report:

  1. From the Project Window, select File, Print Engineering Report. The Engineering Report dialog box appears.

The Base ID, Lot ID/Eng ID and Split ID of the project that you are currently viewing in the Project Window appear in the dialog box.

  1. Select the information you would like to view in the report.

Short Form – Select the Short Form check box to view only the quantity, total hours, and hours per piece information. Clear the check box to view quantity, task type, scrap %, start quantity, end quantity, resource ID, set up time, fixed scrap amount, run time, minimum move, service ID, transit time, and capacity minimum information for each task.

Sort by Op Seq No – Select the Sort by Op Seq No check box to view the report by operation sequence number. VISUAL indents operations and tasks in the same manner that they are indented in the Project Window. If you do not select the check box, VISUAL displays operations and tasks at the same indent level.

Skip masters with obsolete parts – Select the Skip masters with obsolete parts check box to omit operations and tasks that use obsolete parts. Clear the check box to view all operations and tasks.

  1. Click the View arrow to select how you would like to output the report. The choices are View, Print, File, and Email. Refer to your VISUAL documentation for more information on outputting reports.

  2. Click Ok. The report outputs.
