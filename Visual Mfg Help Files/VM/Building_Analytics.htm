Building Analytics

After you create a data source, you can use the data source as the basis for an analytic.

To build an analytic, follow these basic steps:

  1. Specify a configuration ID, description, and data source. See Creating Value-based Analytics, Creating Period-based Analytics, or Creating Top N Analytics.

  2. Select the analytic type. The analytic type that you select determines the other information that you specify in the dialog. See Creating Value-based Analytics, Creating Period-based Analytics, or Creating Top N Analytics.

  3. Select the chart type. The chart type that you select determines the other information that you specify in the dialog. See Creating Value-based Analytics, Creating Period-based Analytics, or Creating Top N Analytics.

  4. Specify the measurement basis of the graph. The measurement basis is the value that you are measuring. You can specify a column in your data source that has the data type of number, or you can count the number of times pieces of data are used in a column in your data source. See Creating Value-based Analytics, Creating Period-based Analytics, or Creating Top N Analytics.

  5. Specify label and chart axis formats. See Formatting Analytics.

  6. Test the analytic. See Testing Analytics

  7. Activate the analytic. See Activating Analytics.
