Linking Purchase Requisitions to Projects

VISUAL allows you to generate purchase requisitions by project.

  1. Open the Purchase Requisition window and from the Edit menu select Add Purchase Parts from EM/WO/Project.

The Add Purchase Parts from EM/EO/Project dialog box appears.

  1. Click the Project option.

  2. Click the Project ID button.

The Select Project browse dialog box appears.

  1. Double-click the project to which you want to link the current purchase requisition.

The dialog box closes and the appropriate information appears in the Add Purchase Parts from EM/EO/Project dialog box.

  1. If you want to select what to include in the requisition, select one of the following options in the Include section:

Parts

Services

Both

  1. Click OK.

The Add Purchase Parts from EM/EO/Project dialog box closes.

All purchasable material and service requirements on work orders linked to the project appear on the purchase requisition.

Requisition lines also include the following project related information:

  • Warehouse ID

  • WBS Code, Proj Ref Sub ID, Proj Ref Seq No, and Department ID combination.

If you want to override the WBS Code, Proj Ref Seq No, or Department ID, double-click the browse button and select the code or ID you want to use.

You cannot save project related purchase requisitions without a valid WBS, Proj Ref Seq No, Proj Ref Sub ID, and Dept ID combination.

  • Task Number

  • Cost Category

For more information on Purchase Requisition, refer to your Infor VISUAL documentation.
