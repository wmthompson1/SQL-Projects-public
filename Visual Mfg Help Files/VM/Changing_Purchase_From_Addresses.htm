Changing Purchase From Addresses

  1. Click the Purc Addr button.

The Purchase Addresses dialog box for the vendor appears.

The table at the top of the dialog lists the alternate addresses, if any, for the vendor.

  1. To use an alternate address for this purchase order, select the line of the appropriate address and click the Use Selected Address button.

  2. If you want to enter Intrastat information for the purchase address, click the Intrastat tab and enter the information in the following fields:

Port of Transshipment - This is place from which the goods are shipped.

Country ID - Enter the Country ID for the purchase address

  1. Click the Save button to save the new information.

To change back to the vendor’s main address, click the Use Vendor Address button.

The dialog box automatically closes. The selected address now appears in the Purchase From window at the top of the purchase order entry window.

  1. Click the Save button to save new and changed addresses.

    Saving does not set the selected address in the purchase order.
