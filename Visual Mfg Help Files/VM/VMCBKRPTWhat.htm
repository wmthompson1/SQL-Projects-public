What is the Customer Bookings Report?

Customer Booking Report prints information for all Firm, Released, and Closed customer orders in a specified date range. Order amounts are totaled for each group in the sorting sequence. For example, if sequencing by Customer ID, total order amounts for each customer are shown.

Backorders are not included in this report. Use the Customer Backorder Report.

Printing/Viewing Customer Booking Report

Starting/Ending Order Date Range

When not printing for Month to Date/Year to Date, you can specify the range of the Order Date (note that this is NOT the Ship Date, but the Order Date.)

Month to Date/Year to Date Summary

Month to Date and Year to Date totals for the current month and year are shown for each Customer ID or Customer Name. By choosing None for the sort sequence, you can get a single MTD/YTD summary. This is a summary report; individual order and line item details do not appear.

When this option is selected, you can only sort by Customer ID, Customer Name, or choose to use no sort sequence. The Starting/Ending Date Range does not apply to this report, since it is always month and year to date.

Sequencing

Nine different sort sequences are supported for the standard report: Order ID, Customer ID, Customer Name, Part ID, Product Code, Commodity Code, Sales Rep, Territory, and G/L Account ID. You can specify a range for all of these sequences.

Order line item detail is shown for Customer ID, and Customer Name sequencing.
