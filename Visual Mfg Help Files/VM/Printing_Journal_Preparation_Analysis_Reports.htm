Printing Journal Preparation Analysis Reports

The Journal Analysis report examines each purchase order, work order, sales order, adjustment and indirect labor ticket, and attempts to determine if the item’s current value, based on its inventory transactions, labor tickets, service receipts, direct invoices, etc., is equal to the value posted to the general ledger.

This report can be run for a range of orders or transactions for the selected journal type, or for all orders or transactions for that journal. It can also be run for exceptions only. The exceptions parameters are user definable in the Plus/Minus % and or the Plus/Minus Amounts fields. When running this report for exceptions, it is recommended that you also select the Mark as Posting Candidate check box. This tells the system that any transaction that is not equal should be reevaluated with the next run of the Costing Utilities Receipt Transaction and Inventory Transaction costing functions.

  1. Select Print Journal Preparation Analysis Report from the File menu.

The Analyze Journal Preparation dialog box appears.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site to view in the report. If you are licensed to use a single site, this field is unavailable.

  2. Enter the Starting and Ending purchase orders for the report if you request information on the purchase journal.

  3. Select the appropriate print option. Select Print, View, or File from the drop down menu.

  4. Select the type of journal from the Type section.

  5. Select the Exceptions Only check box to print exceptions. Apply a sensitivity to it with a Plus/Minus % and or a Plus/Minus Amount deviation.

  6. Select the Mark as Posting Candidate check box to reevaluate any transaction that is not equal with the next run of the Costing Utilities Receipt Transaction and Inventory Transaction costing functions.

  7. Click the Print button to output the report.
