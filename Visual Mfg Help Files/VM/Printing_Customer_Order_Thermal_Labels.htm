Printing Customer Order Thermal Labels

You can print thermal labels for customer orders. What a label contains is entirely up to you. On the same label, you can can place barcodes and fields that correspond to any valid VISUAL database fields.

  1. From the File menu, select Print Customer Orders.

The Print Customer Orders dialog box appears.

  1. Click the Thermal Labels button.

The Print Labels dialog box appears

  1. Click the Site ID arrow and select a site to use.

  2. Click the Label ID button to select the label format you want to use for the customer order thermal label.

  3. In the # of Labels fields, enter the number of labels you want to print.

  4. Using the Thermal Printers list box, select the appropriate thermal printer. This is the thermal printer in your network neighborhood to which you want to send the print order.

  5. In the Spool Into section, click the browse button to select where you want your thermal print order to spool before printing.

  6. Click the Ok button when you are finished setting customer order thermal label print information.
