Setting Exchange Rates for Individual Customer Orders

When you enter a customer order that uses a foreign currency, Customer Order Entry automatically obtains the currency from the vendor and exchange rate from Currency Exchange Rate Maintenance. VISUAL uses the exchange rate based on the date of the transaction. You can, however, fix the exchange rates for individual orders manually. This allows you to set the exchange rates on a per order basis. Use Customer Order Entry to set exchange rates for individual orders.

  1. With the appropriate order open in the Customer Order Entry window, select Set/Reset Exchange Rates from the Edit menu.

The Edit Exchange Rates for Current Order dialog box appears.

The dialog box displays the customer’s currency and the current sell and buy rates for the customer’s currency.

  1. Make any necessary modifications to the line item table.

The table is arranged according to how much of one foreign currency it takes to equal one unit (dollar) of the entity currency.

  1. Click Close when you are done editing or viewing exchange rates.

To view exchange rates for a period of time:

You can view what the exchange rate was for a currency for a period of time by manually entering a date or clicking on the calendar icon to select a date and then clicking Lookup Rates.

VISUAL displays in the line item table what the exchange rate was for that currency during the time you specified.

You can use this value or reselect the current date to use up-to-date exchange rates for your customer order.

Select the Fix Rates check box to apply this exchange rate to the customer receipt as well as the A/R invoice.
