Deleting Export Formats

To delete Export Formats:

  1. Select Admin, Application Global Maintenance.

  2. Select Maintain, Export Formats.

  3. Click the Type arrow and select the type to which the format you want to delete belongs.

  4. Click the Export ID arrow and select the format you want to delete.

  5. If you created contexts for the export ID, click the Context arrow and select the context you want to delete. If you created multiple contexts, you must delete each context one-by-one to delete the entire Export ID.

  6. Click the Delete button.

  7. The system displays a confirmation dialog box. Click the Yes button.

VISUAL deletes the Export Format.

To delete lines from an Export Format:

  1. Click the Type arrow and select the type to which the format you want to modify belongs.

  2. Click the Export ID arrow and select the format you want to delete.

  3. Click the row header to select the line you want to delete.

  4. Click the Delete button.

The system displays an X in the row header indicating you have marked it for deletion.

  1. Click the Save button.

VISUAL removes the line you selected from the Export Format.
