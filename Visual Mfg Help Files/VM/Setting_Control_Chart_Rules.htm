Setting Control Chart Rules

You can edit the control chart interpretation rules using Quality Maintenance.

  1. Select Control Chart Interpretation from the Edit menu.

The Control Chart Rules for Interpreting Lack of Statistical Control dialog box appears.

The dialog box contains eight tests to determine if the process you are testing is outside of statistical control.

  1. Enter the appropriate values for any of the tests.

For example, in the dialog box shown above, the process is outside of statistical control if nine points in a row are either less than or greater than the mean, or if two out of three points in a row are on the same side of the mean.

  1. Click Ok to save the new information and close the dialog box.
