Setting Up Your Test Scenario

If you want to follow the example, set up the following:

  1. Set up the VMDI layout main information.

  2. Specify the columns you want to use in your layout. Set up columns in the header, line item, and sub line item tabs.

  3. Set up a customer order, named CUMMTEST, that specifies the delivery schedules you want to use for your test.

  4. Set up the customer order EDI Line Accumulator info for each line.

  5. An example of how the marking of the Subtract Intransit checkbox affects the delivery schedules.
