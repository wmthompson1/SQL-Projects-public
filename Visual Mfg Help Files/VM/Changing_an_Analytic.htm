Changing an Analytic

After you add an analytic, you can change the filters for the analytic or select a different analytic to display. To change an analytic:

  1. Select File, Dashboard.

  2. In the dashboard, select File, Open and select the dashboard to use.

  3. Right-click the analytic and select Chart Settings.

  4. Change the chart information and filters as needed

  5. Click Ok.
