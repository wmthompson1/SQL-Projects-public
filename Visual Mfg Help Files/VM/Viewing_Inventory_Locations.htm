Viewing Inventory Locations

To view all the valid warehouse locations for the current part:

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site ID to use. If you are licensed to use a single site, this field is unavailable.

  2. Click the Part ID button and select the part to view.

  3. Select Warehouse Locations from the Info menu.

The Warehouse Locations for the part dialog box appears.
