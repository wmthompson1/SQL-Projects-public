Check Availability

Use the Check Availability tool to assess potential bottlenecks to the completion of your project. To use the tool:

  1. From the File menu, select Check Availability. The Check Availability dialog box appears.

VISUAL populates the schedule ID, Work Order/Master to schedule, Release Date, Want Date, and Desired Qty fields based on the project you are currently viewing in the Project Window.

  1. Select the information you would like to analyze.

Schedule ID – Select the schedule for the project. You can set up schedule IDs in the Concurrent Scheduler window. Refer to your VISUAL documentation for more information.

Check Material Availability – Select the Check Material Availability check box to assess the material inventory for the project. Leave this check box blank if you do not want to consider material availability in scheduling the project.

Check Service Availability – Select the Check Service Availability to assess the availability of services you need to procure for the project. Leave this check box blank if you do not want to consider service capacity in scheduling the project.

Force Forward Schedule – Select the Force Forward Schedule check box to schedule the project for completion as soon as possible after the release date.

Treat Release Date as Hard – Select the Treat Release Date as Hard check box if you know the exact release date you would like to use. Clear the check box if the release date can be changed. VISUAL may have selected this check box automatically based on your selections in the Schedule Current Project... window.

  1. Click Schedule. VISUAL schedules the project and identifies potential delays.

  2. To view the work orders associated with the project, click the Work Order Audit button.

  3. Click Close to exit the window.
