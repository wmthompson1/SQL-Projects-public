Entering Equipment Profiles

Now that you’ve entered maintenance profiles, you can attach them to pieces of equipment. Equipment profiles provide a means of specifying maintenance specifics, such as the day of the week you want to perform the maintenance on, and any date exceptions. Exceptions allow you to insert days of inactivity into an equipment profile. Saved equipment profiles appear in the Equipment Maintenance window in the Profiles tab.

  1. From the Maintenance menu, select Equipment Profiles.

The Equipment Profiles dialog box appears.

  1. From the Profile ID lost box, select a Profile ID. You have to create maintenance profiles before anything appears in the list box. Instructions for creating maintenance profiles precede this section

VISUAL auto-fills the read-only fields and the line table with profile information.

  1. Click the Insert button to add a new line to the table.

  2. Double-click the <Equip ID> column header to view the EQUIPMENT table and select a piece of equipment.

  3. Enter a day for the profile. This is the day of the week on which you want the maintenance to begin. Using the above Frequency (two weeks for example), VISUAL schedules maintenance for this piece of equipment every two weeks - or every other Monday, for example.

  4. If applicable, enter a ship date for the profile.

If you associated a part/engineering master to the Equipment ID in Equipment Maintenance, the Part ID/engineering master appears in the Engineering Master column. If there is no engineering master associated to it, "No Part ID on Equipment" appears. To view an engineering master in the Manufacturing Window, select a line and click the Eng Master button.

  1. Click the Save button to save the new line.

  2. To make exceptions to the profile line, highlight a line and click the Exceptions button.

The Exceptions dialog box appears. The Equipment ID and Profile ID appear in read-only fields at the top of the dialog box.

  1. Click the Insert button to add a new exception.

  2. Enter the month in which the exception will take place. Then, within the month, enter the dates on which there will be an exception.

  3. Click the Save button to save the exception.

When you return to the Add Equipment Profiles dialog box, you’ll find that VISUAL has selected the Exception check box in the line table.
