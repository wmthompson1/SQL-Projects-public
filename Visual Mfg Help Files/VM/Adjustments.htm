Adjustments

Adjustments relate to inventory corrections resulting from inconsistencies between the perpetual inventory records and actual quantities held in inventory. Each adjustment either adds an inventory cost layer (FIFO) to inventory or subsequently removes costs from one or more layers.

To check the costing records:

Enter the Starting and Ending Transaction IDs.

Click the Check button.

This populates the table with transactions. As with other Cost to Check options, a note in the Remarks column flags any inconsistencies.

To correct cost inconsistencies:

  1. Select the appropriate line items to be corrected. Click in the corresponding row headers.

This places a check mark (3) in the row headers, indicating the lines to be changed.

  1. Click the Set Posting Candidate button.

This changes the value in the P/C column from N to Y.

  1. Click the Post button.

The Costing Tools make any necessary adjustments to the costs.
