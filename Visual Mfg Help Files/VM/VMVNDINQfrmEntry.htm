Starting Vendor Inquiry

Vendor Inquiry is available from the Infor VISUAL Main window.

To access the Vendor Inquiry window:

  1. Select Vendor Inquiry from the Purchasing menu.

The Vendor Inquiry Search window appears with no information in it.

The Vendor Inquiry Search window contains the following fields.

Purchase Order ID - Use the unique Purchase order identifier for the desired order as it was entered during Purchase Order Entry (or assigned by the system).

Vendor ID - Use the unique identifier for the vendor, as it was entered in Vendor Maintenance.

Vendor Name - Use the full name of the vendor, as it was entered in Vendor Maintenance.

Part ID - Use the unique part identifier associated with the purchase order as entered in Purchase Order Entry.

Service ID - Use the unique service identifier associated with the purchase order as entered in Purchase Order Entry.

Vendor Part/Service ID - Use the unique identifier for the vendor part or service associated with the purchase order entered in Purchase Order Entry.

Work Order Base ID - Use the work order identifier associated with the purchase order as entered in Purchase Order Entry.

Receiver ID - Use the unique identifier associated with the receiving transactions.

Dispatch ID - Use the unique dispatch identifier associated with the purchase order as entered in Purchase Order Entry.

Invoice ID - Use the Invoice ID associated with the Purchase Order called up by the vendor as entered in the Accounts Payable module.

Expense Account ID - Use the Expense Account ID associated with the purchase order.

Site ID - Use the Site ID associated with the purchase order.

Group ID – Use the parent group ID of the vendor on the purchase order.  If you specify criteria in the Group ID field, purchase orders for all vendors belonging to the parent are returned.

  1. Enter the information in the field or fields you want to search.

You can enter search data in as many of the fields as you want. You can also use the following search criteria to further enhance the ability to find the information in the databases:

Wildcards - Wildcards can make your searches even more powerful. Use a wildcard character in place of a character or group of characters in the search string. The two allowable wild card characters are the percent (%) and the underscore (_).

Percent (%) - Use the percent symbol to replace a group of characters. For example, if the part being searched for begins with "AC", enter "AC%" to return every part that begins with AC such as ACORN NUT, ACE BANDAGE, etc.

Underscore (_) - Use the underscore to replace a single character. For example, if you are searching for a part beginning with "SCRE," use the underscore (such as "SCRE_") to return the part SCREW.

Prefixed Operators - These are symbols that enhance the search capability of a field. You can only use these prefixes in front of the desired search value. When using these prefixes, do not put a space between the prefix and the information. The allowable prefixes are:

> (Greater Than)

< (Less Than)

>= (Greater Than or Equal To)

<= (Less Than or Equal To)

You CANNOT use wildcards and prefixed operators together in one data field, but you CAN use them in the same window in different fields.

  1. Select the appropriate Status check boxes for the search information. You can choose from Firm, Released, Closed, and Cancelled.

The search only returns the information that meets the status requirement and the search criteria you enter in the fields.

  1. If you also want to include back ordered items in the search, select the Include Separate Back Orders in the Search check box.

If this box is not checked, back orders do not appear.

  1. Click the Search button.

A table containing the information appears.

  1. Choose the appropriate information to view by highlighting the row and clicking on the View Order button.

The selected item information appears in the Vendor Inquiry window.

  1. After viewing the information, click the Close button to close the window, or the Prev button to view the previous purchase order from the Search results table. Click the Next button to view the next purchase order from the Search results table.
