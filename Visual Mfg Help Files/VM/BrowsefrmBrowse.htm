Using Advanced Browsing

For certain fields in VISUAL, an advanced browse table is available. You can use the advanced browse table to search for specific information, to create result sets that you can scroll through in the main window, and to output information to Excel or to a printer. In certain cases you can edit information directly in the advanced browse table and save it to your database.
