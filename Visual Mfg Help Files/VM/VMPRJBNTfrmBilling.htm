Creating Billing Events

To create Billing Events for your project customer orders:

  1. Select Project Billing Entry from VISUAL main window.

  2. Click the Site ID drop-down arrow and select a site to use.

  3. Click the Order ID button and select the order whose bills you want to set up.

The individual lines of the order appear in the table along with their identifying CLINs.

  1. Select the line for which you want to create billing events.

    Single lines of your Customer Orders can have multiple Billing Events, each contained in one Billing Event table.

  1. Click the Edit Billing Events icon on the toolbar.

  2. Click the Insert button.

  3. Enter information in the following fields:

Seq # - VISUAL automatically enters the next available number in increments of ten.

Description - Enter a description for this billing event.

Billing Basis - Click the arrow and select the basis on which you are configuring this billing event.

Billing Amount - Enter the amount for which you are billing. If you are billing on a Cost or Cost Plus Earned Fee, you can also enter a Fee Amount.

The total amount of billing events cannot be greater that the total of the order line to which you have attached the events.

If you are billing on a Cost or Cost Plus Earned Fee, you can also enter a Fee Amount.

Fee Amount - Enter the amount of the fee for this billing event.

Revenue Recognition - Select the Revenue Recognition check box if you want the system to recognize the billing event as revenue. If you have set up revenue recognition to be based on shipment or percentage of completion, a warning message appears when you select the check box. Make sure you want to associate the billing event with revenue recognition before proceeding.

  1. Click the Calendar button to enter the date on which the event will take place.

  2. Click the Save button.

  3. When you have finished creating Billing Events for this line, click the Close button.

    The total amount of billing events cannot be greater that the total of the order line to which you have attached the events.
