About the Examine Triggers Window

Use the Examine Triggers window to compare the triggers found in your database with a list of required triggers. The Examine Triggers utility identifies missing, additional, disabled, and audit triggers.

Starting the Examine Triggers Window

Select Admin, Examine Triggers.

Task

This table shows the task that you can complete in the Examine Triggers window:

Task

Description

Examining Triggers

Use this procedure to identify inconsistences between the triggers in your database and a master list of triggers.

Fields

System Version

The current version of VISUAL.

Database Brand and Patch Level

The database engine that you use and the current patch level.

Database Version

The version of the database

Total Triggers Expected

The total number of required triggers.

Total Triggers Found

The total number of triggers in your database.

Missing Triggers

Valid triggers that were not found in your database.

Total Missing Triggers

 The number of missing triggers.

Additional Triggers

Triggers that were found in your database but are not in the list of valid triggers.

Total Additional Triggers

The number of additional triggers.

Disabled Triggers

Disabled triggers that were found in your database.

Total Disabled Triggers

The number of disabled indexes.

Audit Triggers

Audit triggers that have been added to the database. Audit triggers can be added by system administrators and are not considered to be incorrect.

Total Audit Triggers

The number of audit triggers.
