Manufacturing Window

After you have set up an Engineering Master or work order, you can assign different WIP accounts.

  1. Double click the master header card.

The Engineering Master/Work Order dialog box appears with information on the master.

  1. Click the G/L Accts button.

The Work Order WIP Accounts dialog box appears.

  1. Assign accounts for Material, Labor, Burden, and Service.

Accounts must be set up before you attempt to assign them in the Manufacturing Window.

  1. Click the Ok button to save the account information.

You can also set up an Engineering Master or work order for internal jobs such as Tool Repair, Building Maintenance, R&D, etc. Once set up you can then assign specific expense accounts for the job. This allows you to schedule and track the job cost, but not post it as a Work In Process inventory. Because costs in the system are transferred when inventory transactions occur, you should cancel "Expense" jobs manually in the Manufacturing Window.
