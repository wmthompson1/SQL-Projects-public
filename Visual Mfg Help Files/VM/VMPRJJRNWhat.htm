What is Project Adjustment Entry

Use Project Adjustment Entry to create journals that post directly to accounts in your General Ledger. Although Infor VISUAL Financials creates the majority of your G/L transactions through subsystems (Accounts Receivable or Accounts Payable) or as a result from running Costing Utilities in Infor VISUAL, you may need to manually create account balance adjustment transactions using Project Adjustment Entry.

You can manipulate single currency balances or all tracking currency balances as well as view postings from all subsidiary ledgers. You can create reversing entries or transpose an entry made improperly.
