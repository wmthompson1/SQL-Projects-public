Setting the System Effective Exchange Rate Date

In the Effective Exchange Rate area of the General tab section, select either the system date or transaction date to determine exchange rates as applied to currency transactions. This selection is a onetime only selection. By default, the option is set to "Transaction Date." If you prefer to use the current system date for conversion, you must make this change before you enter a single transaction in the system. The two options related to currency conversion are: Transaction Date (Typical) and Current System Date.
