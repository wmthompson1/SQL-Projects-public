Starting Inventory Transaction Entry

Inventory Transaction Entry is available from the Infor VISUAL Main window.

To start Inventory Transaction Entry:

Select Inventory Transaction Entry from the Inventory menu. The Inventory Transaction Entry window appears with no information in it.

Depending on the kind of transaction in progress, different fields are enabled and/or required. For instance, for an adjustment in or out, you must specify the part number; VISUAL disables the fields for Work Order and piece number. Conversely, when an issue is in progress, you cannot specify the part ID unless you are processing a new requirement. These rules ensure that you cannot create invalid transactions.
