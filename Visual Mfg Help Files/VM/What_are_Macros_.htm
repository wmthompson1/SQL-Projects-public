What are Macros?

A macro is a small, script-based program usually developed by the application user in response to a specific need. For example, if you are a part planner that works with parts that have a particular product code, you can create a macro in Part Maintenance that inserts your name as the Planner ID, the product code you work with in the Product Code field, and other standard information. You can also use macros to check for particular conditions, such as the weight of a shipment, and trigger certain actions as a result of the condition. Use macros to automate and control business processes.

When you write a macro, you can interact with the database fields used in the windows. Not all database fields are available. Use the Available Fields field in the Edit Macro dialog to view a list of the fields you can work with.
