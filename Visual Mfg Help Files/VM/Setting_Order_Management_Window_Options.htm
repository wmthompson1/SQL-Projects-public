Setting Order Management Window Options

You can only set options for the Order Management window if you have at least one of the seven child windows open. From the Tools menu, select Options, or click the Options icon (see above left) located under the toolbar in every child window. The Options dialog box appears.

Setting General Options

  1. From the Tools menu, select Options, or click the Options icon.

The Options dialog box appears. The General tab has the default position.

  1. Select the Refresh Automatically check box if you want VISUAL to refresh the display in the Order Management Window automatically.

Leave this check box cleared if you want to manually refresh the current display using the Refresh button. See Order Management Window Buttons and their Functions for more information.

  1. Determine the frequency of the refresh.

For example, if you want VISUAL to refresh the display every ten seconds, enter 10 in the third - or seconds - field; if you want VISUAL to refresh the display every hour, enter 1 in the first - or hour - field

  1. Click the Apply button to commit the refresh settings.

As the time that you have specified lapses, VISUAL refreshes the display. The screen may flicker when this happens.

Setting Part Options

If you’re not currently viewing customer order part information in the Part Information child window, you can use the window to view general information for parts in your database.

Begin this procedure with the Part Information child window open and on top.

  1. From the Tools menu, select Options, or click the Options icon.

The Options dialog box appears. If it is not already displaying, click the Parts tab.

  1. Click the Part ID button to select a part from the PART table. A searchable table dialog box appears; select the appropriate part. If you know the name of the part, manually enter it in the Part ID field.

  2. Click the Apply button to commit the part selection.

Part information appears in the Part Information child window.

Specifying Customer Options

If you’re not currently viewing a specific order in the Order Management Window, you can use the Customer Order History Window to view order information for any valid VISUAL customer. In the options dialog box, under the Customer tab, you can select a customer, specify a date range, and then view all activity occurring within or between those dates.

Begin this procedure with the Customer Order History child window open and on top.

  1. From the Tools menu, select Options, or click the Options icon.

The Options dialog box appears. If it is not already displaying, click the Customer tab.

  1. Click the Customer ID button to select the customer for which you want to view order history. A searchable table dialog box appears; select the appropriate customer. If you know the name of the customer, manually enter it into the Customer ID field.

  2. If you want VISUAL to display orders by date, constrain the scope of order information by choosing the appropriate date options.

Select the Dated Within option button if you want to view orders before (by a specified number of days) or after (by a specified number of days) the current system date.

In the Days Before field, enter the number of days before the current system date you want VISUAL to use when searching for current customer order information.

In the Days After field, enter the number of days after the current system date you want VISUAL to use when searching for current customer order information. Use this field if you want to view orders with Order Dates in the future.

Select the Dated Between option button if you want to specify a date range between which VISUAL displays order activity for the specified customer.

Click the Calendar icon to select a Starting Date and an Ending Date. Only orders with dates between the range you specify will appear in the Order Management Window.

  1. Click the Apply button to commit the settings.
