Deleting a Warehouse

Deleting a Warehouse will permanently remove the information from the database. Deleted warehouse information is not recoverable.

Deleting Location Rows

Before you can delete a warehouse, you must delete the warehouses's location rows.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site where you created the warehouse. If you are licensed to use a single site, this field is unavailable.

  2. Call the warehouse you want to delete into the Warehouse Maintenance window.

  3. Select the location row you want to delete, then click the Delete Row button.

An x appears to the left of the row.

  1. Select and delete all rows in the warehouse.

  2. Click the Save button.

VISUAL deletes the location rows from the warehouse. You can now delete the warehouse.

Deleting Warehouses

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site where you created the warehouse. If you are licensed to use a single site, this field is unavailable.

  2. With the warehouse you want to delete in the Warehouse Maintenance window, click the Delete button.

A dialog box appears, asking you to confirm the deletion.

  1. Click Yes to continue, or No to cancel the deletion.

If you click Yes, VISUAL removes the warehouse information from the database, though the Warehouse Maintenance window still contains the warehouse information.

You cannot delete the Warehouse ID if Part Maintenance or Inventory Transactions are using it.

  1. Click the Clear button to clear the warehouse information from the window.
