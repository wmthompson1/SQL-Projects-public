Choosing a Commissions Basis

Accounting Entity Maintenance allows you to specify a global retro payment standard to use when calculating commissions.

  1. From the VISUAL main menu, select Accounting Entity Maintenance from the Admin menu.

  1. Click the Payment tab.

  2. From the Pay Commissions by section, select the appropriate option:

Cash Receipt - Select the Cash Receipt check box if you want sales representatives to receive commissions based on the date you receive moneys owed.

Invoice/Invoice Date - Select the Invoice/Invoice Date check box if you want sales representatives to receive commissions based on invoice dates.

  1. Click the Save button on the main toolbar or select Save from the File menu.
