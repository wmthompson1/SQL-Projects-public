Accessing Purchase Order Specifications and Notations

Purchase Receipt Entry allows viewing and modification of Order Specifications and Line Specifications associated with the selected purchase order.

To access Order Specs for the current PO, select Purchase Order Specifications/Notes from the Notes menu, or press F6.

To access Line Specs for a PO line, highlight the line and select Line Specifications/Notes from the Notes menu, or press F7.

These dialog boxes can remain open for the whole session, and their contents change as the current PO and selected line item change.

You can also make changes and additions to these specifications.
