Using the Customer Order Repair Line for Repair Costs

If you did not create a repair work order or you charge a particular fee for repair costs, use the customer order line created for the repair to charge customers for repair costs. If you created a new customer order to process the repair, perhaps with a New Customer ID, then the repair line is on the new customer order. If you are using the original order to process the repair, then the repair line is on the original customer order.

To price the repair:

  1. Perform one of these steps:

  2. If you created a new customer order for the repair, open the new customer order in Customer Order Entry.

  3. If you created a new customer order line on the original order for the repair, open the existing customer order.

  1. On the repair line, specify the unit cost for the repair.

  2. Click Save.

  3. When you ship the repaired parts, specify Approved in the Ship Status field. An approved packlist is eligible for invoicing. See Shipping the Repaired Materials.
