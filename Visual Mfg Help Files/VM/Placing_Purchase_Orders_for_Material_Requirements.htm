Placing Purchase Orders for Material Requirements

You can place a purchase order for one or more material requirements directly from the Material Availability dialog box.

You can purchase materials for a single line in the Material Availability dialog box or for multiple lines.

Placing a Purchase Order for a Single Line

To place a purchase order for a single line:

  1. Select Info, Material Availability.

  2. Select the line.

  3. Select Edit, Purchase One Material.

The dialog shows information about the quantity of part needed to meet the material requirement. To view information about purchase orders that have already been linked to the material requirement, select a row in the table at the top of the dialog box.

  1. To create a formal allocation link between the purchase order and the material requirement, select the Link to each requirement check box. If you create a formal link, then the materials are issued to the work order when you receive them in Purchase Receipt Entry. If you do not want to create a formal link, clear the Link to each requirement check box. You can manually issue the materials to the work order after you receive the purchase order, or you can issue parts already residing in your inventory.

  2. The quantity needed to meet the requirement is inserted in the Reqd Qty field. You cannot edit this field. To order an additional quantity of part, specify the quantity in the Add'l Qty field. If you order an additional quantity and you selected the Link to each requirement check box, then two purchase order lines are created for the material. One line contains the full quantity for the material requirement linked to the work order. The second line contains the additional quantity. The additional quantity is not linked to the work order. If you cleared the Link to each requirement check box, then a single purchase order line is created for the quantity required for the work order and the additional quantity.

  3. Click Order. The purchase order entry window is opened.

  4. Complete the information in the purchase order, and then click the Save button.

Placing a Purchase Order for Multiple Lines

To place a purchase order for multiple lines:

  1. Select Info, Material Availability.

  2. Select the lines. To select individual lines, press and hold CTRL and then click the line. To select a range of lines, select the first line in the range. Press and hold Shift and select the last line in the range.  

  3. Select Edit, Purchase All Selected Lines. The purchase order entry window is opened. One line is inserted for each material requirement you selected in the Material Availability dialog box. A link to the work order requirement is inserted on each line.

  4. Complete the information in the purchase order, and then click the Save button.
