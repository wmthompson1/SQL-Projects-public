Modifying a Traceability Profile

You can modify Part Trace Profiles.

  1. Enter the Part ID for the traceable part whose profile you wish to change. You can do this in three ways:

  1. Enter the Part ID using the keyboard.

  2. Click Part ID to select from the part browse of all parts in the system.

  3. Click Profile ID to select from a browse of all traceable parts in the system.

  1. Make any change to the information.

You should be cautious about changing a profile that has been in use. Observe the notes that apply when changing the following information:

Label - Changing the label will change the column title as it appears in all the traceability windows.

When to Apply - You should carefully consider changing the information, especially clearing boxes. Although all changes are supported, you may be requiring certain users to change their method of using an application.

For example, if you clear the Receiving box, the users of Shipping Enter may suddenly find that they have to create new trace records, where they were already created in the past.

Numbering - You can change Numbering at any time. You should be careful about changing numbering profiles that may cause reuse of the same number (the system does not allow reuse).

Properties - You can add new properties at any time, by entering their labels. Changing existing labels changes the column titles for the properties. Removing a label causes the existing property column to no longer be shown; the information is still in the database, and if another property is later added in its place, the information for the old property reappears with the new label.

In general, you should set up properties once during the implementation of the traceability profile for the part.

  1. Click the Save button.

Changes take effect on the next transaction using the part.

You can view or modify profiles for multiple parts by using the Next button to automatically call in the profile for the next traceable part.
