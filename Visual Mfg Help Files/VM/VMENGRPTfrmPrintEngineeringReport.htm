Printing/Viewing Engineering Report

The Engineering Report provides a complete printout of an engineering master, work order, or quote.

  1. Select Engineering Report from the Eng/Mfg menu.

The Print Engineering Report window appears.

The window shows all engineering masters, work orders, and quotes. This window is similar to the Search window in the Manufacturing Window. To perform a search, select Search from the File menu.

  1. Make the following selections for your report:

Site ID(s) - If you are licensed to use multiple sites, click the Site ID(s) arrow and select the site(s) to include in the report. If you are licensed to use a single site, this field is unavailable

Short Form - If you want to print the short form of the report, select the Short Form check box.

The long form shows essentially all information defined in the Manufacturing Window for the header, each operation, and each material requirement.

The short form shows Quantity, Setup Hours, and Run Time for each operation, Quantity for each material, Drawing IDs, and Revs.

Sort by Op Seq No - If you want VISUAL to sort the report by Sequence Number of each operation, select the Sort by Op Seq No check box.

Skip Masters With Obsolete Parts - If you want VISUAL to exclude masters with obsolete parts, select this check box.

Include Active ECNs - If you want VISUAL to include active ECNs in your report, select the Include Active ECNs check box.

  1. Select the sequence for the report.

You can sequence the table window list by Work Order ID, Customer ID, Create Date, Want Date, or Release Date. Additionally, you can check the Descending check box to reverse the order of the listing.

The masters and orders you select print in this sequence.

  1. Select the output for the report.

Select Print to output the report to a printer.

Select View to output the report to your screen for viewing.

Select File to output your report as a text file. You can then edit the report using a text editing application.

  1. If you want to run the report for a selected line or lines, make the selections, then select Print/View Selections from the File menu.

To print all lines, choose Print/View All Lines from the File menu. This option is useful when you have used a search to limit the number of lines in the list.

A standard Windows dialog box appears.

  1. Make the appropriate selections for how you want the information to appear:

Descending - If you want the information to appear in descending order.

No Customer - If you do not want customer information to appear.

  1. Click the Print toolbar button.

VISUAL prints the report.
