Printing Part Comparison Reports

The Part Comparison report is similar to Tag Comparison, only it shows quantities and values for a Part ID, rather than individual tags. For example, if a Part ID is found in 5 locations, there are 5 tags for it. The Tag Comparison report shows all 5 tags. The Part Comparison report shows one summarization line for all 5 tags of the part.

To use this report:

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site where you conducted the count. If you are licensed to use a single site, this field is unavailable.

  2. In the table, select the physical inventory count.

  3. Select Print Part Comparison from the File menu.

The Print Part Count Comparison Report dialog box appears.

  1. Enter the Physical Count ID.

You can use this report with any existing physical count. If a count is in progress, the Physical Count ID defaults to the current count. If you want to compare a different count, enter the Count ID.

  1. Select the Part Range.

You can print this report for a single part, a range of parts, or all parts.

To print for a single part, enter the Part ID in both Starting and Ending Part ID fields.

To print for a range, enter the Starting and Ending Part ID. You can print an open-ended range by only entering one of these values.

To print for all parts, leave these fields blank.

To print for a single part, enter the Part ID in both Starting and Ending Part ID fields.

  1. Select the Restrictions to filter parts that appear in this report.

Print Original value on Uncounted Tags - Select this check box if you want to apply the current dollar value of inventory on the uncounted tags. An ‘N’ will be printed to the right of the value indicating this value is an uncounted value.

  1. Select the output.

Select Print, View, or File.

  1. Click Ok.

  2. A standard Windows Print dialog box appears.

  3. Enter the appropriate information and click Ok.

For each part listed, the following data is shown: Part ID, Part Description, Unit of Measure, On-Hand Quantity (total of all locations), Previous Value (value based on the On-Hand Quantity), Count (for all tags of the part), and New Value (value based on the Count). Previous Value and New Value are totaled.
