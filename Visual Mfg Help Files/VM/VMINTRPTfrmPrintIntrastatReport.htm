Printing Intrastat and ESL Reports

You can generate Intrastat and ESL reports from the Intrastat Arrival dialog box. To do this:

  1. Clicking on the Print button.

The Intrastat/ESL Reports dialog box appears.

  1. Enter the appropriate information in the following fields:

Site ID (range) - Click the Site ID (range) arrow and select the site(s) to use.

Country ID - This option exists to view data by country. For dispatches, the country selected is the to county. For arrivals, the country selected is the from country.

Year - Enter the reporting year.

Period - Enter the reporting period.

Report # - Enter the next sequential report number you want to use.

From/To - Enter the date range of the reporting period.

Period Type - Select the appropriate period type: Monthly, Quarterly, or Annually.

The period and year default to the date range of the current date. You cannot edit the From/To date range. You can only change the Period Type in draft mode.

  1. Select the appropriate output format from the Print combo box. You can select Print, View, or Export.

  2. If you do NOT want to commit any reports to the database, click the Draft Mode check box.

In draft mode, you can see what the line item quantity would be if the reporting frequency were changed. This mode also allows the user to print a report and make sure all the necessary data is there.

  1. Select the following information from the Intrastat section:

Report Type - Select either Arrivals or Dispatches.

Draft Mode - If you want the report to print in draft mode, select the Draft Mode check box.

Show Reported Lines - If you want the report to show reported lines, select this check box.

Sort By - You can sort by Destination Country, Originating Country, Region ID, Receiver/Shipper ID Shipped Date, or Invoice ID.

Sub Sort By - You can further sort by Nature of Transaction, Mode of Transport, or Tariff Code.

Format - Reporting supports the following user defined formats:

Intrastat Arrival: VFISTAUx.QRP where x is 1, 2, or 3.

Intrastat Arrival: VFISTDUx.QRP where x is 1, 2, or 3.

  1. Click the ESL tab to enter the following information:

Sort By - Sort by Customer ID, Country ID, or Date.

Format - Reporting supports the following user defined formats:

Intrastat Arrival: VFESLUDx.QRP where x is 1, 2, or 3.

  1. Click the Print button to print the report.
