Displaying Toolbars

When you open the Order Management window, all toolbars and status bars are active. If you want to turn some or all of these off, go to the View menu and clear the check boxes next to each item as appropriate.

Management Toolbar - The Management Toolbar occupies the uppermost part of the Order Management window. It contains the View menu and customer summary information, Customer ID, Warehouse ID, Ship To ID, and Customer Name.

Main Toolbar - The Main toolbar is the first floating toolbar beneath and to the left of the Management toolbar. It contains the Refresh button and the Options button. See the earlier section on specifying options for more information. You can drag this toolbar and position it - or "dock" it - in other parts of the window. This toolbar is always available.

Personal Toolbar - The Personal toolbar is the large toolbar occupying the left of the Order Management window. It is the same as the VISUAL Personal Manu toolbar, available from the VE main File menu. To change the appearance of the Personal menu in the Order Management window, change the Personal Menu toolbar available from the VE main File menu. Changes made there appear in each application.

Customer Order Main Toolbar - The Customer Order Main Toolbar is next to the Main toolbar and contains buttons that allow you to perform some of the more basic functions: Save, Print, Delete, Refresh, and view Customer Notations, for example. This toolbar is available only when the Customer Order Entry child window has the focus.

Customer Order Table Toolbar - The Customer Order Table Toolbar is next to the Customer Order Main Toolbar and contains buttons that allow you to control line items in the line item table. This toolbar is available only when the Customer Order Entry child window has the focus.

Status Bar - The Status Bar runs along the bottom of the window.

You can display or hide the tool bar and status bar. By default, they both appear when you open the Order Management Window. From the View menu, either select or clear the Show Toolbar and Show Statusbar options as appropriate.
