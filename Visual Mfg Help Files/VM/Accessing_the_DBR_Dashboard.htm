Accessing the DBR Dashboard

The DBR Dashboard is not available as a menu selection. You must launch the executable directly from the VISUAL executables directory.

To open the DBR Dashboard:

  1. Navigate to the directory where your VISUAL executables are stored.

  2. Locate VMDBRDBD.EXE.

  3. To open the file, double-click it.

  4. Provide your log-in credentials. Specify this information:

User ID – Specify your user ID.

Password – Specify the password associated with the user ID.

Database – Specify the database to sign into.

  1. Click Sign In.
