Printing Engineering Master Simulation costs

  1. Select Inventory, Cost Simulation.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site to use for the simulation. If you are licensed to use a single site, this field is unavailable.  

  2. Click the Simulation ID browse button and select the simulation.

  3. Select Maintain, Eng Master Costs. The Engineering Master Simulation Costs table is displayed.

  4. Select an output option.

  5. Select File, Print.
