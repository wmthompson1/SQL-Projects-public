Viewing Trace Histories

Every time you make a change to a trace record, VISUAL tracks those changes so you will have a permanent record.

To view trace history:

  1. If you are licensed to use multiple sites, click the Site ID browse button and select the site where you traced the part. If you are licensed to use a single site, this field is unavailable.

  2. Open the appropriate trace.

  3. Click the row header for the trace whose history you want to view.

The row appears highlighted.

  1. From the Info menu, select Show Trace History.

The Trace History window appears populated with the trace history for the trace you selected.

  1. When you have finished viewing or printing the trace history, close the Trace History window by clicking the Close button.
