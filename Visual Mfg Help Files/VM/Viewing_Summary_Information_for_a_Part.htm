Viewing Summary Information for a Part

After you select a part, click the Summary tab, if it is not already displaying. The Summary tab contains the following read-only information:

The Summary tab contains the following read-only information:

Primary Warehouse - The default primary warehouse of the part. See Part Maintenance for more information.

Primary Location - The primary location of the part. The primary location must be a location in the Primary Warehouse. See Part Maintenance for more information.

Total On Hand - The number of parts that you have on hand but not necessarily available.

Total On Hold - The number of parts that you currently have placed on hold.

Netable - This quantity is always equal to the On Hand quantity. The quantity of parts for which you can determine netting information - your entire stock at any one time.
