Using the Popup Menu

This menu item is active by default. When this selection is NOT active and you right-click an operation within the display, a small window appears that displays the scheduled start and finished dates of the operation as well as the want date of the work order to which that operation belongs.

When this selection is active and you right click an operation, a menu pops up that allows you to choose from some of the selections that would otherwise be available only from the Edit or Info menus.

This feature can save a great deal of mouse movement when you want to use one of the selections available.
