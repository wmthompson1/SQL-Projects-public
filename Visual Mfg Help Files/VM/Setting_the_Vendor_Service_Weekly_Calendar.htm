Setting the Vendor Service Weekly Calendar

You can set the outside service vendor’s weekly calendar using the Vendor Service Weekly Calendar dialog box.

  1. Select the Vendor Week button in the Scheduling tab.

The Vendor Service Weekly Calendar dialog box appears.

  1. For each weekday enter the percent of the day worked in the Percent of Day Worked column. The workday percentages are based on the outside service vendor’s operation run rate.

  2. Click the Save button to save any changes to the Vendor Service Weekly Calendar.
