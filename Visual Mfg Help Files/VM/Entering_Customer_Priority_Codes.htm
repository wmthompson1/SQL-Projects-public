Entering Customer Priority Codes

  1. Select Admin, Accounting Entity Maintenance.

  2. In the Entity ID field, select the accounting entity for which you are setting up customer types.

  3. Click the Order Mgt tab.

  4. Click the Priority Codes button.

  5. Click Insert to start adding line item information to the table.

Priority Code - An alphanumeric expression that serves as a unique identifier of the type

Description - A description of the priority code.

SKU Rate - A number between 0 and 100 that indicates the percentage of the available stock that you can allocate to the customer.

  1. Click Save when you are finished adding priority code information.
