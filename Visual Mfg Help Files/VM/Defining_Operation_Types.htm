Defining Operation Types

Use the Operation Types feature to pre-define all of the attributes of an operation so that you can easily copy information about the operation during operation add or edit in the Manufacturing Window. When you change the Type of an operation in the Manufacturing Window, the systems asks you whether or not to have the attributes copied. Most fields available for entry for an operation in the Manufacturing Window are available here for entry.

  1. Select Eng/Mfg, Shop Resource Maintenance.

  2. Select Edit, Operation Types.

Creating Operation Types that reference multiple group type resources is permitted.

  1. Click the Insert button.

  2. Enter a Type ID, up to 15 characters, and a Description.

  3. Click the Save button.

  4. Highlight the Type you want to define and click the Edit button.

The Edit Operation Type window looks the same as the Operation window in the Manufacturing Window. Any fields that you specify are copied in to the corresponding fields when you use the operation type.

  1. Specify all data fields that you want to pre-define for operations of that type.

  2. Click Ok.

  3. Click the Save button to save the type.
