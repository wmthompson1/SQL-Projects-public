Starting Part Trace Maintenance

Part Trace Maintenance is available from the VISUAL Main menu.

To start Traceable Part Maintenance:

Select Part Trace Maintenance from the Inventory menu.

The Part Trace Maintenance window appears.

Information for a single part appears in the Part Trace Maintenance window, which you specify by entering its Part ID. When working with more than one part, you can use the Next button to quickly move to the next Part ID.

The Traceable check box at the upper right of the window indicates whether the current part is traceable, i.e., whether a traceability profile has been defined for the Part ID.

The table is similar to the Traceability tables used throughout the other applications. However, it is a union of all those tables; all trace records in existence for the part appear, with all quantity and property information for each. The following columns appear in the table:

Site ID – If you are licensed to use multiple sites, click the Site ID arrow to select the site that owns the part. If you are license to use a single site, this field is unavailable.

Part ID – Click the Part ID browse button to select the part to trace.

Trace ID - The identifier for the grouping of parts. This column is titled with the label defined in the part’s part trace profile, ex: Lot Number, Serial Number, etc. This field cannot be modified.

Quantity Received - Total quantity for this lot number that was received into inventory through Purchase Receipt Entry or an inventory Receipt transaction. This field cannot be modified.

Quantity Issued - Total quantity for this lot number that was issued to a work order through an inventory Issue transaction, or to a customer order using Shipping Entry. This field cannot be modified.

Quantity Reported - Total quantity for this lot number that was reported as completed using Labor Ticket Entry. This field cannot be modified.

Quantity Assigned - Quantity that was originally assigned using the Pre-Assign Numbers function.

Properties - One column is shown for each alpha and numeric property defined in the part’s traceability profile.

Comments - Any comments entered for the lot number throughout the traceability cycle.

Expiration Date - Expiration date for parts with a finite shelf life.
