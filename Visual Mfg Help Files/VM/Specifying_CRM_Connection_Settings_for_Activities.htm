Specifying CRM Connection Settings for Activities

To drill back from Activity Maintenance to CRM, you must specify the location of the CRM executable and the name of the database server.

To specify this information:

  1. Access Activity Maintenance. For example, select Admin, Activity Maintenance. See Starting Activity Maintenance.

  2. Select Options, Preferences.

  3. In the CRM Drillback section, specify this information:

CRM Exe Directory – Specify the directory where the main CRM executable is located.

DB Server Name – Specify the name of the server where your VISUAL database resides. The CRM database tables are located in the VISUAL database. When you drill back to CRM, this information is used to sign in.

  1. Click Ok.
