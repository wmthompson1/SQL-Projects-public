Entering Warehouse WIP/VAS Information

You must set WIP/VAS information for each warehouse.

  1. With the appropriate warehouse in the Warehouse ID field, select the WIP/VAS supported box.

  2. Click the Save button.

You cannot enter any WIP/VAS information for a warehouse until you do this.

The WIP/VAS button is now available.

  1. Click the WIP/VAS button.

The WIP/VAS dialog box appears with no information in it.

  1. Enter a Warehouse ID or click the Warehouse ID button to select a Warehouse ID from the line item table.

The Warehouse ID and description appear along with any WIP/VAS line items that you may have already entered.

  1. Click Insert to start adding line item information.

  2. Enter information in the following fields:

Part ID - The part in the warehouse that you are making eligible for the performance of the value added service.

Either enter a Part ID or click the column header to select one from the table.

WIP/VAS ID - The ID of the value added service or work in process that you are assigning to the part.

Either enter a WIP/VAS ID or click the column header and select one from the table. Enter WIP/VAS IDs in Accounting Entity Maintenance.

Unit Price - The price per part of the value added service.

The default Unit Price that you established for this WIP/VAS ID in Accounting Entity Maintenance is displayed. You can change the unit price.

  1. Click the Save button to commit the new information.

  2. Click the Close button to return to the Warehouse Maintenance window
