Tasks

This dashboard analytic shows tasks created by Purchase Requisition Entry, ECN Entry, and Workflow. Workflow tasks for all accounting entity IDs are included in the chart.

Type: Pie chart

Y-axis: n/a

Series categories (wedges): Type, User ID, Status

Series groupings (X-axis): n/a

Date filter applies to: Create Date

Filters: Site ID

Detail grid: Click a wedge to view the Tasks Detail grid filtered to the tasks that contribute to the wedge.
