Spreading Project Budgets

Dividing your costs for the tasks within a project over the life of each task is commonly known as "Spreading Budgets." Because it is very important to be able to set budgets for your projects and bill for your costs over very specific time periods, VISUAL's Project Summary window allows you to set different budget spreads using different rates.

To set up budget spreads:

  1. Open the Project Summary window and select the project whose budget you want to spread.

  2. From the Maintain menu of the Project Summary window, select Budget.

The Budget Maintenance window appears.

The Project ID automatically appears.

  1. Click the Save/Spread Budget button.

VISUAL calculates the budget spread for your project based on the budget figures you entered and the Distribution ID you selected.
