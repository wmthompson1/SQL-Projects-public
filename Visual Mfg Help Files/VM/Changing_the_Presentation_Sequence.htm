Changing the Presentation Sequence

Use the Presentation Sequence feature to change the order in which the shop resources appear in the Scheduling Window.

  1. Select Presentation Sequence from the Edit menu.

The Presentation Sequence of Resource in Schedule dialog box appears.

  1. Click in the Seq # column to the left of the resource you are changing and change the sequence number.

  2. Click Ok to save the changes.

The next time you open the presentation sequence dialog box, the resources appear in the order defined by the position number.
