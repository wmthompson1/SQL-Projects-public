Creating an RMA for a Piece Tracked Part

When you set up an RMA for a dimensional part, you have the option of changing the dimensions of the part that the customer is returning to you. The dimensions specified on the shipper are displayed on the RMA for reference.

To create an RMA for a piece tracked part:

  1. Select Sales, Return Material Authorization.

  2. Complete the header as you would for a standard part.

  3. In the lines, the number of pieces and the dimensions from the customer order are displayed in the C.O. Pieces, C.O Length, C.O Width, and C.O Height fields. Specify this information:

Rtn Pieces Authorized – Specify the number of returned pieces.

Length, Width, and Height – Specify the dimensions of the pieces that the customer is returning to you.

After you specify the number of pieces and the dimensions, the Rtn Qty Authorized is calculated.

  1. Complete the RMA line as you would for a standard part.

  2. Click Save.
