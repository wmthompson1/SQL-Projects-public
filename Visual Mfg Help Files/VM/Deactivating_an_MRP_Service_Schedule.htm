Deactivating an MRP Service Schedule

You can deactivate your scheduled MRP run for a site. If you set up MRP schedules for more than one site, then you must deactivate each schedule separately.

To deactivate an MRP schedule:

  1. Select Inventory, Material Planning Window.

  2. In the site ID field, select the site whose schedule you want to deactivate.

  3. Select File, Material Requirements Planning Service Schedule. The ID of the site you selected is displayed in the title bar of the dialog.

  4. Clear the Enabled check box.

  5. Click Save.
