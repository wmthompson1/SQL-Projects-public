Editing Notification Settings

Use this dialog to edit the list of internal recipients that receive email notifications about this order:

  1. Select Sales, Customer Order Entry.

  2. Open the order record.

  3. Select Edit, Order Notifications. The selections that you made on the Notifications tab are displayed.

  4. To select the employees that receive notifications, click the Internal Employee browse button. One tab for each notification event is displayed. The tab is active only if you selected the corresponding check box in the Send Internal Notifications section.

  5. Click the tab for the business event that you are setting up. A list of available employees is displayed. To be displayed in the list, the employee’s record in Employee Maintenance must meet these requirements:

  6. An email address must be specified for the employee

  7. The employee must be active.

  1. Select the employees to notify, then click the Add to Selected List button.

  2. Click Ok.
