Source and Target Buttons in Purchase Order Entry

When you are viewing a transaction, you can use the source and target toolbar buttons to view related transactions. Use the source button to view the previous transaction in the lifecycle. Use the target button to view the next transaction in the lifecycle.

In Purchase Management Window, the Source button is labeled Purchase Requisition. It is available if you generated the purchase order from a requisition. Click the button to open the requisition. The Target button is labeled Purchase Receipt. It is available if you have received any portion of the purchase order. Click the Target button to open the purchase receipt.

To activate the buttons, the Purchase Order Entry window must be the active window.
