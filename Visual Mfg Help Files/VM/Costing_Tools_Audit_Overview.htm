Costing Tools/Audit Overview

VISUAL provides several recalculate functions to change transaction data within the database. The following Recalculate options are available:

  • Recalculate Distributions

  • Recalculate Standard Costs

  • Recalculate WIP Balances

  • Recalculate Inventory Balances

The analysis reports provide a means to track the manipulated transaction data. The following reports are available:

  • Cost of Goods Analysis

  • Cost Distribution Analysis

  • P/O Accrual Analysis

  • Journal Prep Analysis

  • FIFO Analysis
