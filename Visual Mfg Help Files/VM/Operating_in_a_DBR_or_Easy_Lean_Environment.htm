Operating in a DBR or Easy Lean Environment

When you are licensed to use DBR, the DBR scheduler creates work orders and planned purchase orders for you. If you use DBR, you should not create work orders manually. You should let the DBR scheduler create work order when it finds demand.

Similarly, you do not have to enter purchase orders manually. The DBR scheduler creates planned purchase orders if it finds demand for purchased parts. You can use the Material Planning WIndow to release these planned orders, and then process the purchase order as you would any other purchase order.

If you are licensed to use DBR, you should NOT use the Concurrent Scheduler. You should only use the DBR scheduler.
