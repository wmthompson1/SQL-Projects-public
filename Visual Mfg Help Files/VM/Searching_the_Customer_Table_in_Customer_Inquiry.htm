Searching the Customer Table in Customer Inquiry

If your database is large, listing the entire database may be impractical. The system has been designed to allow you to do focused searches in order to speed up this process. You can use either wildcards or prefixed operators to make the search process easier and faster. Use any of the following guidelines to help you in this search process:

Wildcards and prefixed operators cannot be used together in one data field, but can be used on the same window in different data fields.

If the resulting search list is not too long, use the vertical scroll bar to navigate up or down the table window list.

The resulting table window will be in ascending order.

When you find the data element you want, select it with the mouse or keyboard.

If you cannot find the data element because you cannot remember the correct spelling or the exact value, usually there is an alternate search field on the window. For example, if you want to search on Customer ID, but are not sure of the correct information, try the Customer Name data field instead.
