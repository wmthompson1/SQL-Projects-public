What is the APS Import Utility?

Whenever you want to run VISUAL APS using your company's manufacturing data, you must import that information using the VISUAL APS Import Utility.

This simple import utility reads ONLY the information fields you specify and converts it for use in VISUAL APS.

You can import several different types of data, using the APS Import Utility. Because each separate type of data is stored in its own ASCII text file, it is viewable by many different text-editing programs.

The types of information you may import are:

  • Parts

  • Shop Resources

  • Frozen Operations

  • Customer Orders

  • Inventory Transactions

  • Master Schedule

  • Work Orders

  • Purchase Orders

  • Calendar Exceptions

  • Work Order Schedule Priority

By supporting different types of delimiter characters the VISUAL APS Import Utility is compatible with many ERP export programs.

Each time you run the Import program, VISUAL APS records the import transactions to a log file. This file is appended each time the utility is run, giving the user a record of all transactions.

When the import process is complete APS copies and renames the imported text files to backup files. The APS program also deletes the old import files, thus preparing the system for the next time you import files using the same file names.

If you want to add a shortcut icon to your Windows Desktop to start the VISUAL APS Import program, please consult your Windows documentation for further instructions.
