In-context Vendor View Panel

To access the Vendor view panel:

  1. Select Purchasing, Vendor Maintenance.

  2. Open the vendor record.

  3. Select View, Vendor View Panel.

  4. In the Entity ID field, specify the ID of the entity whose information you want to view.

Use the Vendor View panel to view information about the selected vendor. You can view summary information, detail information, and a chart. For information about how to set up summary information and detail information, see In-context View Panels. For information about the Vendor view panel chart, see Vendor View Panel Chart.
