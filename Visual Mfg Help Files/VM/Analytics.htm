Analytics

Analytics provide you with a snapshot view of data in your database. You can access more information directly from the pie chart and bar graph. You can:

  • View the values of the data that make up the chart or graph

  • View the transactions that make up each data point

  • Print the graph

All analytics are displayed as a pie chart, vertical bar graph, horizontal bar graph, or line graph. The analytic you select determines what type of chart or graph is used. You cannot select the type of graph. You also cannot customize the colors used in the graphs.
