Entering Customer Order Types

  1. From the Maintenance section, click the Customer Order Types button. The Customer Order Types dialog box appears.

There are 7 Order types already entered in the table.

  1. Click the Insert button to start adding line item information to the table.

Type - A unique identifier of the type.

Description - A description of the type.

  1. Click Save when you are finished entering Customer Order Type information.
