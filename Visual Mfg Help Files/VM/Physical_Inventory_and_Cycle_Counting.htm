Physical Inventory and Cycle Counting

Physical Inventory can be greatly facilitated through the use of barcodes. The standard Infor VISUAL windows provide an excellent method for locking inventory and printing the necessary paperwork for a Physical Inventory and/or Cycle Count.

The chain of events is similar to the following:

  • Physical Inventory / Cycle Count is created within standard VISUAL Window

  • Tags/Sheets are printed within standard VISUAL Window with Barcodes

  • Material counts are performed using Barcode Hardware

  • Counts are verified within standard VISUAL Window

  • Material Recounts are performed using Barcode Hardware

Physical Inventory

Physical Inventory counts, recounts are entered through the barcode scanners. This provides for more accurate counts and part match-ups during a physical inventory.

Cycle Counting

Cycle Counts, and recounts are entered through the barcode scanners. This provides for more accurate counts and part match-ups during cycle counting.
