Using Drag n' Drop to Copy/Append an Existing Order to a Current Order

You can copy/append an entire existing order or specific lines from existing orders to an order by dragging the whole order or an order line from the Customer Order History child window into the Order Entry window.

  1. In the Order Entry window, open the order to which you want to copy or append a line.

  2. If not already open, open the Customer Order History child window. Tile or cascade the two windows so that you can easily perform a drag n’ drop operation.

  3. In the Customer Order History window, double-click the Customer icon.

All existing orders for the customer appear.

  1. Either highlight the order you want to copy (this will copy/append the whole order), or click the plus box next to the order and highlight the line you want to copy. Hold down the left mouse button until the drag n’ drop box appears.

  2. Drag the box into the open Order Entry window and release it in the line item table.

VISUAL adds the new line to the order.

Maximize the Order Entry child window so you can easily view the new copied/appended lines.

  1. Modify the information as necessary and click the Save toolbar button to save the order.
