Importing Data from Microsoft Project

Use the Infor VISUAL - Microsoft Project Interface window to import information you have modified from Microsoft Project.

To import the changes you made in Microsoft Project back into VISUAL:

  1. In either the Project window or the Project Summary window, open the project you want to send to Microsoft Project.

  2. From the File menu, select Send to Microsoft Project.

The Infor VISUAL - Microsoft Project Interface appears populated with the data from the project you previously opened.

  1. Click the Get from MS Project button.

VISUAL starts Microsoft Project, opens the appropriate project file, and imports your project information. VISUAL populates the project data on the right side of the Microsoft Project Inter face window.

  1. Click the Save button.

The Infor VISUAL Updated Data Report dialog box appears.

  1. Select the method you want to use for viewing the changes to your project by clicking on the arrow and making your selection.

If you want to print the VE Project Update Report, select Print.

If you want to view the VE Project Update Report, select View.

If you want to save the VE Project Update Report to a file, select File. VISUAL prompts you for where you want to save the report.

If you want to e-mail the VE Project Update Report, select E-mail. VISUAL opens a Microsoft Outlook e-mail message and attaches the report.

  1. Click the Ok button.

VISUAL updates your project with the changes you have made in Microsoft Project.

  1. When you have finished importing data, click the Close button.

VISUAL prompts you to select an option to handle the active Microsoft Project file.

  1. To save the Microsoft Project file, with or without a baseline, click Yes.

  2. To close Microsoft Project without saving any changes, click No.

  3. To keep Microsoft Project open, click Cancel.

To view your imported data, refresh the Project Window or Project Summary Window.
