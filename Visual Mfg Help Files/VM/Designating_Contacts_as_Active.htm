Designating Contacts as Active

When you create a contact, it is designated as active by default. An active contact can be associated with customers or vendors and is selectable for transactions. By default, only active contacts are displayed in the contacts table.

To activate one contact at a time after their initial creation:

  1. Select View, Inactive Contacts to view the inactive contacts.

  2. Select the contact.

  3. On the General tab select the Active check box.

  4. Click the Save toolbar button to update the contact status as active.

To activate one or more contacts simultaneously:

  1. Select View, Inactive Contacts to view the inactive contacts.

  2. Select one or more rows in the contact table.

  3. Click the ActivateContact(s) toolbar button.

  4. Click the Save toolbar button to update the contact status as active.
