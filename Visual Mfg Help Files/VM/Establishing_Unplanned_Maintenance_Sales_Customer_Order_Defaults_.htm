Establishing Unplanned Maintenance Sales (Customer) Order Defaults

Use the Sales Order dialog box to specify a default status for sales orders you create in the Unplanned Maintenance window.

  1. From the Options menu, select Preferences, Customer Order Defaults.

The Customer Order Defaults dialog box appears.

  1. Select a default status for customer orders:

Release - Released customer orders are orders on which you are ready to begin work. They have been, quite literally, "released" to the shop floor.

Firm - Firm customer orders are orders existing more for planning purposes. Though they are "real" customer orders and exist in the database, the VISUAL Concurrent Scheduler does not schedule them as it does released work orders. Schedulers and planners use the Firm designation as a kind of place holder.

  1. Click the Save button to commit the default status.
