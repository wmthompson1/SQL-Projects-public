Sending Infor VISUAL/Financials Information

Applications that support the Send To function include a Send To option on the File menu. Applications that support toolbars also have a Send To toolbar button.

  1. Open the record you would like to send. For example, if you would like to send customer order 0001, open Customer Order Entry and select order 0001 from the Our Order ID browse table.

  2. From the File menu of the application in which you are currently working, select Send To or click the Send To toolbar button.

The system opens a Microsoft Outlook e-mail message with the vmx file attached. The system inserts the name of the window from which you are sending the e-mail in the subject line. For example, if you are sending a customer order the system inserts Customer Order Entry in the subject line.

  1. Choose recipients, carbon copy recipients, and blind carbon copy recipients, and enter text in the body of the e-mail. You can also change the subject line if needed.

  2. Click Send to send the e-mail.

For more information on using Microsoft Outlook, consult your Microsoft Outlook user documentation.
