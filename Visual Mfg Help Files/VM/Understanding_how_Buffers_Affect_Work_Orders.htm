Understanding how Buffers Affect Work Orders

simple work order in VISUAL with a DBR Buffer table with buffers generated for a CCR of AS

Line 1 - Line 1 in the table is the Shipping buffer and is the length of time from the last CCR operation to the work order completion.

Line 2 - Line 2 in the table is the first CCR buffer.

Line 3 - Line 3 in the table is the Rod buffer separating the two CCRs by a minimum amount of time.

Line 4 - Line 4 in the table is the second CCR buffer.

Line 5 - Line 5 is the Assembly buffer.
