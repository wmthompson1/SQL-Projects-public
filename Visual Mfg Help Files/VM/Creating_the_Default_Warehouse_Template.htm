Creating the Default Warehouse Template

To create the warehouse you would like to use as the default warehouse template:

  1. From the Inventory menu, select Warehouse Maintenance. The Warehouse Maintenance window appears.

  2. Enter the new Warehouse ID.

  3. Enter a description for the new warehouse in the description field.

  4. If the warehouse is a VISUAL Distribution warehouse, select the VDW Installed check box. VDW warehouses can only be used in conjunction with VISUAL Distribution.

  5. Select the Independently Planned Warehouse check box.

  6. If you would like to exclude the warehouse from consideration during material resource planning, select the MRP Exempt check box. If you would like to consider the warehouse during material resource planning, clear the MRP Exempt check box.

After you close a project, consider designated the closed project warehouse as MRP Exempt. This will prevent MRP from considering the closed warehouse.

  1. If the warehouse supports WIP/VAS, select the WIP/VAS Supported check box. WIP/VAS must be globally activated in Application Global Maintenance.

  2. Click the Inventory Not Shared arrow to define whether or not the warehouse shares its inventory. Select Y if the warehouse does not share inventory. Select N if the warehouse does share inventory.

  3. Click the Insert Row button. A new entry appears in the location table.

  4. Enter the Location ID for the location you are creating within the warehouse.

  5. Select the type for the location from the Type drop down box.

The five types are: Regular, Floor Stock, In Transit, From Customer, and From Vendor. The From Customer and From Vendor types are used with consigned inventory. For more information on consigned inventory, see the "Consigned Inventory" chapter in the Infor VISUAL Inventory User's Guide.

  1. Enter a description for the location in the Location Description column.

  2. You can continue to enter additional locations by clicking the Insert Row button and repeating steps 9 through 11.

  3. Click the Save button to save the warehouse information.
