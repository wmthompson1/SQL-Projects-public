Setting Up Budgets

Project budgets express how you want the various costs involved in your project distributed within Infor VISUAL — Projects/A&D. By selecting the appropriate WBS Code you can specify how many hours (or a financial amount) required by that department and how you want VISUAL to distribute the amount for the life of the project.

After you have added all the required tasks to your project structure, you can set the project budget at the resource level. To set the project budget, define what resources are required to accomplish each task by selecting:

  • A Department

  • A Cost category

  • An estimate of the labor hours required or an estimate of the monetary cost for either material or other direct charges such as travel expenses.

  • A Distribution ID

Use the Budget Maintenance window to set up and maintain budgets for each task within your breakdown structure.

To set up budgets:

  1. From the Maintain menu of the Project Summary window, select Budget.

The Budget Maintenance window appears.

  1. If you selected a project in the Project Summary window, the appropriate WBS information appears in the Budget Maintenance window.

You can limit the WBS codes that appear by selecting Starting and Ending WBS codes.

If you did not select a project in the Project Summary window, click the Project ID button and select the project whose budget you want to set up.

  1. When you select a project, the appropriate work breakdown structure information appears in the table.

  2. Click the Rate ID button to select the rate to apply to the budget.

  3. Click the row header for the WBS Code whose budget you want to set up.

The row appears highlighted.

  1. Click the Insert button.

The first free row in the lower table appears selected ready for you to set up its budget.

  1. Double-click Dept ID and select the department you want to use for this budget setup.

  2. Double-click Cost Cat ID and select the cost category you want to use for this budget setup.

  3. Double-click Dist ID and select the active Distribution ID you want to use for this budget setup. The Distribution ID determines how VISUAL spreads the budgeted hours/costs across the duration of the task.

  4. Depending on the type of cost category you selected, click in either the Hours or Amount column and enter the appropriate figure for this budget.

  5. Click the Save button.

If you want to enter additional budget lines for the WBS, click the Insert button again and enter the appropriate information for the new budget set up.

Before you can spread your project’s budget, you must first schedule that project. For more information, refer to Scheduling Projects.
