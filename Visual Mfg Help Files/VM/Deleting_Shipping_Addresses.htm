Deleting Shipping Addresses

You can delete a customer's shipping address only if it is not used on any transaction.

If you no longer want to use an address that has been previously used on a transaction, inactivate the address.

To delete a shipping address:

  1. Select Sales, Customer Maintenance.

  2. In Customer Maintenance, specify the ID of the customer.

  3. Select Maintain, Shipping Addresses.

  4. In the table, select the shipping address to delete.

  5. Click Delete.

  6. Click Save.
