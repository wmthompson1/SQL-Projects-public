Resetting Service Operation Costs

You can reset operation costs used in quote masters, work orders, and engineering masters. When using standard costs, this feature corrects and updates costs for services in existing quotes, work orders, and engineering masters.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site to use. If you are licensed to use a single site, this field is unavailable.

  2. Click the Service ID browse button and select the service for which to reset operation costs.

  3. Select Reset Operation Costs from the Edit menu.

The Reset Service Operation Costs dialog box appears.

  1. If you want to update for the current Service ID only, select the Current Service Only radio button.

To update a range of services, select All Services. You can control the range through the remaining steps.

  1. Check the types of master/work orders whose operations you wish to update.

You can choose any combination of engineering master, quote master, and/or work order.

  1. If you are resetting work order costs, you can reset based on the status of each operation.

Check the According to Operation Status box, and then check each status of operation that you want to update. For example, you may choose to only update Unreleased and Firmed operations - check these two boxes.

  1. You can also update by work order/master creation date. To do this, check this option and enter the after and before dates to define a range.

  2. Click the Ok button.

You automatically return to the Service Maintenance window when the update is complete.
