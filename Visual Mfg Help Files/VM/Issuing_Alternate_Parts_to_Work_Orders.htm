Issuing Alternate Parts to Work Orders

If the material requirement allows for the use of alternate parts, you can issue the alternate part to the work order instead of the part on the material requirement card.

When you select a material requirement that supports the use of alternate parts, the system activates the Part ID button. Click the Part ID button to select the alternate part to issue to the work order. The system only displays the alternate parts you specified on the material card.

You must choose an alternate part that has the same unit of measure as the part it is replacing. If you choose an alternate part that has a different unit of measure, the system prevents you from issuing the alternate part.

You can use the Alternate Part Information dialog box to decide which alternate part you would like to use.

To use the alternate part information dialog box:

Select View, Alternate Part Information.

When you issue an alternate part to a work order, the system creates a new material card for the alternate part. The new material card is identified as an alternate part and references the original material card. On the original material card, the system inserts the quantity of alternate part issued for the requirement on the Alternate Parts tab.
