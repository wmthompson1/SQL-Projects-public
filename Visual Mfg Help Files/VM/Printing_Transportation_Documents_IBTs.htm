Printing Transportation Documents - IBT

This option is only available if you have enabled Shipment Tracking in Site Maintenance.

This option is not available until you ship the Inter/Branch Transfer.

A Transportation Document is a status report that details the location and value of inventory and why you are shipping it.

  1. From the File menu, select Print Transportation Document.

The current IBT ID appears in the Order ID field.

  1. From the Ship Reason list box, select the reason for this shipment. Only the ship reasons applying to Inter/Branch Transfers appear in this list box. Enter IBT Ship Reasons in Site Maintenance.

In Site Maintenance, under the Shipment Tracking tab, you must specify whether or not Ship Reason Codes are required for the printing of transportation documents. If Optional is the default, you do not have to select a Ship Reason Code. If Required is the default, you must select a Ship Reason code before VISUAL prints the document.

  1. From the Carrier ID list box, select a Carrier ID or click the Carrier ID button to select one from the line item table. Enter carriers in Application Global Maintenance.

  2. Select Freight Terms: Prepaid, Billed, or Collect.

  3. Click Ok.

VISUAL displays the report on you screen.
