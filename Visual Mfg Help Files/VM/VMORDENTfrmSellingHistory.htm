Viewing a Selling History

The Part Selling History feature allows interactive viewing of previous sales performance for a given part.

You can access this option in two ways:

  • Select Selling History from the Info menu.

Enter the Part ID into the Part ID field, or click the Part ID button and select from the Parts table.

  • With a line item containing a Part ID selected, click the History button.

The Part Selling History dialog box appears with the part automatically selected.

The table contains one line for each existing customer order line item that is for the specified Part ID. This includes customer orders of all status values.

If you are licensed to use multiple sites, click the Site ID(s) arrow and select the sites to view. If you are licensed to use a single site, this field is unavailable.

You can view the following information in the dialog box:

Shipping Performance - The Order Date, Desired Ship Date, and Last Actual Ship Date are shown for each order.

Price History - For each line, the Order Quantity, Shipped Quantity, Unit Price, Discount Percent, and Standard Unit Price are shown.

You can sort the Part Selling History table by Order Date, Desired Ship Date, or Customer Order. You can sequence in ascending or descending order.

The View option allows you to inspect the order containing a line in the table. To do this:

Select the appropriate line and click the View button, or double-click the line.

The Customer Order Entry window appears with the selected order. This allows viewing of any other details of the order not shown in the table, including notations and specifications.

If you make any changes to an order, VISUAL asks you to confirm any changes before it resets the Customer Order Entry window. This is a quick way to create a new order from an existing one by blanking the Order ID and resaving to copy the existing order.
