Maintaining Part Groups

VISUAL Price Book uses Part Groups to classify internal parts and merchandise within your operation. Use Part Groups to create bundles of goods that are frequently sold together, or that complement each other. For example, if you build computers, you may want to group together the outer case, a hard drive, and motherboard. This allows you to define prices, discounts, commissions, etc. for parts sold as a group that differ from the discounts, prices, commissions for the parts when sold individually.

Maintain Part Groups on the Part Groups window.

To access the Part Groups matrix:

With VISUAL Price Book open, from the navigation pane, double-click the Price Groups icon. The Price Groups Matrix window appears. Depending on the settings you used on the Tools menu, your default values may vary.

If you have previously configured Part Groups, you can access the matrix information by clicking on the Part Group ID Browse button. See Editing Part Group Matrix information for further details.

Adding Part Groups Matrix Information

  1. If you are working with an existing Part Group, enter or select the Part Group ID. Any existing Part Group matrix information appears on the table.

If you are working with a new Part Group, enter a value in the Part Group ID field. VISUAL Price Book uses Part Group IDs to classify internal parts. This allows you to group components together and apply discounts to the end product as a whole. Following the computer example mentioned previously, you could easily group together all the components necessary to build a computer and apply a discount to every component at once, rather than applying the same discount to a number of records.

  1. On the VISUAL Price Book main toolbar, click the Insert Row button to add a new line to the Part Groups matrix. VISUAL activates the next line on the table and automatically inserts the Part Group ID from the Part Group ID field on the header portion of the window.

  2. Enter or select a Part ID to add to the Part Group. You can enter the Part ID manually, or select it from a Browse window by double clicking the row header on the Part Groups matrix table.

  3. Click the Save button to save your changes to the Part Groups window.
