Editing a Request For Quote

Individual line items of an existing request for quote can be edited. These edits can be adding a line item, changing an existing item, adding line item specifications, or deleting a line item. This section discusses each of these operations.

You can edit all fields and line items of a request for quote at any time.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site in which you created the RFQ. If you are licensed to use a single site, this field is unavailable.

  2. Call up the request for quote you want to edit.

  3. Make any necessary changes to any field or line item.

In particular, you may wish to periodically update the following fields:

Follow-up - Update this date to the next date you want to follow up this quote for some reason.

Expected Update this date as necessary.

  1. Click the Save button to save the changes.
