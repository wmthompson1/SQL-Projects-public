Printing Transportation Documents - Shipping Entry

A Transportation Document is a status report that details the location and value of inventory and why you are shipping it.

  1. From the File menu, select Print Transportation Document.

The Print Transportation Document dialog appears.

The current Packlist ID appears in the Packlist ID field.

  1. Depending on whether or not you require ship reasons for the printing of transportation documents, from the Ship Reason list box, select a reason for the printing of this document.

The requirement of ship reasons is a setting in Site Maintenance under the Shipment Track tab.

  1. Select an output for the document. See Report Output Options.

  1. Click Ok.

If you selected Print, VISUAL prints the document to your printer.

If you selected View, VISUAL displays the document.

If you selected File, the Print to File dialog box appears. Name the file.

If you selected E-mail, a Send dialog box appears in which you can specify to whom you want to send the document.
