Analyzing Rates

Analyzing the rates you use in calculations and comparing them against each other is important in helping you manage your projects and contracts.

To analyze rates:

  1. From the Project menu of the Infor VISUAL main window, select Rate Management.

  2. From the Info menu, select Rate Analysis.

  3. Click the Site ID drop-down arrow and select a site to use.

  4. Click the Year drop-down arrow and select the year for which you want to run a rate analysis.

  5. Use the Period drop-down arrows to select the Starting and Ending periods for this Rate Analysis run.

  6. Click the Run Rate Analysis button.

VISUAL populates the three Year to Date columns with the following information:

Applied Rate – The current applied rate percentage by Burden Category ID appears in the Applied Rate column.

Actual Rate – VISUAL calculates the Actual Run Rate Percent

as:

Burden Category Pool

Burden Category Base

= Actual Rate 

Variance – VISUAL calculates the variance as the Actual Rate percent subtracted from Current applied rate percentage.
