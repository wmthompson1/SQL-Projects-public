Viewing Information about Piece Tracked Parts

Use the Inventory Pieces dialog box to view the pieces currently in your inventory.

To view inventory pieces:

  1. Select Info, Material Availability.

  2. Select the line.

  3. Select View, Inventory Pieces.

The line item table contains columns for Site ID, Part ID, Warehouse ID, Location ID, Number of Pieces, Length, Width, Height, and Dimension of Unit Measure.

You can also sort the order in which parts appear by selecting the appropriate option button in the Dimension Sort sections: Length, Width, or Height.
