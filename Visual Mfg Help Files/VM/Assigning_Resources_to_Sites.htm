Assigning Resources to Sites

To add existing shop resources to a site:

  1. In Site Maintenance, select Maintain, Site Resources.

  2. Click the Site ID arrow and select the site to which you are adding shop resources.

  3. Select the Add to Site check box for the shop resources you are adding to the site. If you choose to add a shop resource group ID, you are asked if you want to add the resources that are members of the group to the site. Click Yes to add the group members.

To add all shop resources to the site, click Select All for Add. To clear all selections in the Add to Site column, click Unselect All for Add.

  1. Click Save to add the shop resources to the site. After you click save, the Exists in Site check box is selected for the shop resources you added.

After you add shop resources to the site, you can modify certain site-specific resource information in Shop Resource Maintenance.
