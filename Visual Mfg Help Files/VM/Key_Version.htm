Key/Version

The eight layout keys represent each of the four VISUAL targets intended for importing or exporting. They include:

CPO - Select this layout key for inbound customer orders.

PLN - Select this layout key for inbound customer part forecasts.

ASN - Select this layout key for outbound Advanced Ship Notices.

INV - Select this layout key for outbound invoices.

VPO - Select this layout key for outbound vendor purchase orders.

ACK - Select this layout key for outbound customer invoices.

CSH – Select this layout key for inbound cash receipts.

RCA – Select this layout key for outbound material receipts (receiving advices).

WSA – Select this layout key for Warehouse Shipping Advance layouts
