Viewing a Quote History

The Quoting History feature allows interactive viewing of previous quoting performance for a given part.

If you are licensed to use multiple sites, you can view quotations for multiple sites at the same time.

You can access this option in two ways:

  • Select Quoting history from the Info menu.

  • You can select the part of interest by entering the Part ID into the Part ID field, or by pressing the Part ID button and selecting from the Parts table.

  • Press the History button while a line item containing a Part ID is selected.

The Part Quoting History window appears with the part automatically selected. This window appears even if you have not specified a Part ID. If the quote line item is selected, the History tab functions the same as pulling up the window from the Info menu.

The table appears with one line for each existing quote line and quantity that is for the specified Part ID. This includes quotes of all status values.

You can view the following information:

Sales Performance - The initial Quote Date, the Won/Lost date, and the Expected Date appear for each quote, along with the customer and Sales Rep.

Price History - For each quote, Quoted Quantity, Quoted Unit Price, and Discount % are shown, along with the standard unit price for the part.

You can sort the Part Quoting History table by Quote Date, Customer ID, or Customer Order, any of which you can sequence in ascending or descending order.

The View option allows inspection of the quote that contains the information of that particular line in the table. You can select a line and press the View button, or double-click the line.

The Estimating Window appears with the selected quote. This allows viewing of any other details of the quote not shown in the table, including notations and specifications. If you are currently editing or creating a quote, VISUAL asks you to confirm before resetting the Estimating Window. This is a quick way to create a new quote from an existing one. Simply blank the Quote ID and resave to copy the existing quote.
