About the Material Handler Queue Window

Use the Material Handler Queue to process move requests.

Starting the Move Requests Window

Select Inventory, Material Handler Queue.

Tasks

This table shows the tasks that you can complete in the Move Requests window:

Task

Description

Sorting and Filtering the Material Handler Queue Table

Use this task to specify the move requests to view in the material handler queue table.

Assigning Move Requests to an Employee

Use this task to assign a move request to an employee.

Picking Up Materials

Use this task to pick up materials for a move request.

Delivering a Move Request

Use this task to deliver the materials for a move request.

Delivering Materials to an Alternate Location

Use this task to deliver the materials to a different location than the one specified on the move request.

Fields

Site ID

The site where the parts are located.

Assign to Emp ID

The ID of the employee to whom you are assigning the move request.

Hide Details

Select the check box to view only header information. Clear the check box to view header information, the General tab, and the Details tab.

Move Request Table

The list of move requests.

General tab

Move Request ID

The ID of the move request.  

Status

The status of the move request. These statuses are used:

Pending

The move request is not complete. You can also use this status to indicate that the parts are being prepared for pickup.

Loading

The parts are being prepared for pickup.

Ready for Pick Up

The parts are ready to be moved.

Picked Up

The parts are in transit to the new location.

Delivered

The parts have been delivered.

Cancelled

The request has been cancelled.

WIP Delivery Type

The type of delivery you are requesting.

If the parts in the move requests are raw materials or sub-assemblies that will be used in a work order, or otherwise will not be permanently stored, select Drop Only.

If the parts will become part of your WIP inventory, select WIP Inventory.

Request Type

Indicates how the move request was created.

Priority

The request takes priority over other requests.

Request by Emp ID

The ID of the employee who submitted the move request.

Equip Type

The piece of equipment that the material handler should use to move the parts. Set up equipment types in the Move Equipment Type dialog box. See Move Equipment Types.

Weight

The total weight of the parts that are being moved, including packaging.

Weight U/M

The unit used to measure the weight.

Container ID

The ID of the container that contains the parts.

Comments

Additional information about the move request.

Pickup Location section

Resource

If the parts are located at a shop resource, the ID of the resource.

Department

If the parts are located in a department, the ID of the department.

Warehouse

If the parts are located in a warehouse, the ID of the warehouse.

WIP Location

If the parts are located in a WIP location, the ID of the location.

Directions

Directions to the pick up location.

Requested Pickup

The date that the parts should be picked up.

Time

The time that the parts should be picked up.

Employee ID

The ID of the employee assigned to the move request.

Picked Up

The date that the move request was picked up.

Time

The time that the move request was picked up.

Deliver To Location section

Resource

If parts should be moved to a shop resource, the ID of the resource.

Department

If parts should be moved to a department, the ID of the department.

Warehouse

If parts should be moved to a warehouse, the ID of the warehouse.

WIP Location

If parts should be moved to a WIP location, the ID of the location.

Directions

Directions to the drop-off location.

Actual Deliver To Location section

Resource

If parts were delivered to a shop resource, the ID of the resource.

Department

If parts were delivered to a department, the ID of the department.

Warehouse

If parts were delivered to a warehouse, the ID of the warehouse.

WIP Location

If parts were delivered to a WIP location, the ID of the location.

Directions

Directions to the drop-off location.

Date

The date that the parts were delivered.

Time

The time that the parts were delivered.

Detail Tab

The list of parts to move.
