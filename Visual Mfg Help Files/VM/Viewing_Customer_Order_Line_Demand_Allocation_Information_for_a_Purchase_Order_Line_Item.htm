Viewing Customer Order Line Demand Allocation Information for a Purchase Order Line Item

After you have either fully or partially allocated demand from a purchase order line to a customer order line using the preceding procedures, you can view allocation information on the purchase order line level.

For each purchase order line item from which you have attached links to demand, the following appears:

  • In the Base ID column, if there is only one link of customer order delivery schedule demand, the Customer Order ID appears.

  • In the Base ID column, if there are multiple links of customer order demand, Multiple Links appears.

  • In the Link Part ID column, if there is only one link of customer order line demand, the Part ID of the purchase order line that you have allocated supply from appears.

  • In the Link Part ID column, if there are multiple links of customer order line demand, nothing appears.

  • In the Link Warehouse ID column, if there is only one link of customer order line demand, the Warehouse ID of the customer order line appears.

  • In the Link Warehouse ID column, if there are multiple links of customer order line demand, nothing appears.

  • In the Link Quantity Unallocated column, if there is only one link of customer order line demand, the quantity of the customer order line to which you have yet to assign demand appears.

  • In the Link Quantity Unallocated column, if there are multiple links of customer order line demand, nothing appears.

  • In the Total Qty Allocated column, the quantity thus far of the purchase order line that you have allocated to sources of demand appears.
