Exporting Data from the Change Country Results Table

You can export the date in the Change Country search results table to Microsoft Excel.

  1. In the VISUAL executables directory, double-click VMCHGCNT.EXE.

  2. Sign in as the SYSADM user.

  3. Run the query. See Using the Change Country Utility.

  4. After you execute your search, select File, Send to Microsoft Excel.
