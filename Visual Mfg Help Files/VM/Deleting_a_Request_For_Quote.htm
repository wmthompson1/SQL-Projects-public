Deleting a Request For Quote

This operation removes the request for quote from the system including all of its line items and all Request for Quote Masters.

A warning message appears informing you that the entire request for quote will be deleted including its line items.

  1. With the appropriate request for quote open in the Vendor RFQ Entry window, click the Delete button.

A warning message appears, informing you that the entire request for quote, including its line items, will be deleted if you continue.

  1. Click Yes to permanently remove the request for quote or No to leave the request for quote in the system.

  2. Click the Save button to complete the operation.
