Printing Material Planning Reports from the Browse Dialog Box

You can print Material Planning Reports from the Browse dialog box, which appears after you search for a part. Select all parts for which you want to print the report from the Browse dialog box.

If you want to print for all parts, you need not make any selections.

  1. Click the Print Entire List button to print reports for all of the parts.

Click the Print Selections button to print reports for parts you have selected in the list.

The Print Material Planning Report dialog box appears.

  1. Choose the appropriate netting options, including: Net Available Quantity and Deduct Safety Stock

  2. Choose the appropriate Netting Type, either Net By Required Date, Net Using MRP Rules or Net By Release Date.

These options are the same as those previously discussed under Material Netting Options.

  1. Choose the appropriate printing options: Print, View, or File.

  2. Click Ok to print the report.

VISUAL prints the report to either a printer, the screen, or a file.
