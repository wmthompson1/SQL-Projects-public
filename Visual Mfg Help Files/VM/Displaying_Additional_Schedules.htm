Displaying Additional Schedules

You may want to have more than one schedule being displayed at the same time. This is done using the Load Schedule ID function.

If you are licensed to use multiple sites, you can load schedules from different sites.

  1. Select Load Schedule ID from the File menu.

The Schedule ID dialog box appears.

  1. Select the new schedule you want to display.

  2. Select one of the Action radio buttons:

Select Replace Current Window to replace the current window by creating a new window

Select Create New Window to create a new window while leaving the old window open).

Select Add to Current Window to add the new schedule to the existing window.

You can add up to three schedules to the same window. After that, you must create a new window to add another schedule, or replace the current window with the new schedule.

  1. After you have made your selections choose Ok.

Each schedule that appears in the same window is stacked vertically for each resource. The background color indicates to which schedule the bars belong. This allows you to compare schedules for each resource separately.

The Synchronize Windows option in the Window menu controls how scrolling behaves when you are showing multiple schedule windows. If this option is set (a check mark appears in the menu), the vertical and horizontal scroll bar for either window causes both windows to scroll. This keeps the windows in synch by resource and by date/time. If this option is not active, then each window scrolls independently.
