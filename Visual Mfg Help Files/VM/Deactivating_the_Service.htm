Deactivating the Service

If you use the Scheduling Service with a site that has multiple schedules, you can deactivate the service for a single schedule.

To deactivate the service for a particular schedule:

  1. Select Scheduling, Global Scheduler.

  2. In the table, select the schedule for which you no longer want to use the service.

  3. Select File, Set as Scheduled. The ID of the schedule you selected is displayed.

  4. Clear the Enabled check box.

  5. Click Save.
