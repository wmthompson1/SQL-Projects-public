Entering Notations

  1. With the appropriate purchase order open in the Purchase Receipt Entry window, select the appropriate Notations selection from the Notes menu.

The Notations dialog box appears.

  1. Click the New button to make a new entry.

A line appears with the current date and time for the entry.

  1. Enter the appropriate information.

Use the Paste button to paste text that you copied from another location into the box.

Use the Copy button to copy text in the dialog box to the clipboard.

  1. Click the Save button to save the Notes.

VISUAL adds the user ID to the end of the notation.
