Starting Customer Order Entry

Customer Order Entry is available from the VISUAL Main window.

To start Customer Order Entry:

Select Customer Order Entry from the Sales menu.

The Customer Order Entry window appears.

The Customer Order Entry window contains two sections: Customer Order Header information and the Customer Order Line Item table.

The Customer Order Header information section, located at the top of the Customer Order Entry window, contains basic information about customer and shipping details.

The Customer Order Line Item table contains one line for each item on the customer order.
