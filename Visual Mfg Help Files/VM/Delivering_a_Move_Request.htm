Delivering a Move Request

  1. Select Inventory, Material Handler Queue.

  2. In the Move Request table, select the check box for the row that you are delivering, or highlight the row. The move request must have a status of Picked Up. You can select more than one row.

  3. To complete a drop only delivery, click the Deliver (Drop Only) button. The materials in a drop-only delivery will be used immediately and not stored. To complete a WIP inventory delivery, click the Deliver to WIP Inventory button. The materials in a WIP delivery are not immediately used. The selection that you make does not change WIP delivery type that was originally specified on the move request.
