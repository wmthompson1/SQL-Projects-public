Changing Sort Settings

Within the Project Summary window you can change the criteria by which VISUAL sorts your information. VISUAL automatically sorts by Project, WBS Code, Sequence number, Department, and Cost Category.

To change the sort settings:

  1. From the Options menu, select Sort Preferences.

The Sort Preferences dialog box appears.

  1. Select the criteria on which you want VISUAL to sort your information.

  2. Click the Ok button.

The Sort Preferences dialog box closes and VISUAL sorts the information in the Project Summary window table on the criteria you selected.
