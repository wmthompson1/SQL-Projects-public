Entering General VAT Information

You can enter the following VAT information in the VAT tab of the Accounting Entity Maintenance window:

Value Added Tax is Discounted by Terms Discount - Check this box to have VISUAL compute VAT from the price after terms discount. This does not include other discounts

Use VAT Books to generate invoice numbers - Check this box if you need the enhanced features of VAT book numbering.

Calculate VAT on Freight - If you want VISUAL to calculate the VAT on the freight portion of the charge, select the Calculate VAT on Freight check box.

VAT Categories Required - If you want your VAT codes to belong to categories, select the VAT Categories Required check box.

VAT Registration - Enter the VAT number of your company.

Next VAT Sequence # - Enter the next number to use for VAT reporting.

VAT Receivables Export ID - Enter the Receivables Export ID if you are generating reports to file.

VAT Payables Export ID - Enter the Payables Export ID if you are generating reports to file.
