Configuring the Material Planning Window

You can show and hide various columns in the Material Planning Window.

  1. Select Configure Table from the Options menu.

The Configure Table Window dialog box appears, listing all of the columns for the Material Planning Window and their current status (visible or hidden).

  1. Select the column you would like to show, then select the Include check box. To hide a column, clear the Include check box.

You can click the Select All button to select all columns and the Deselect All button to deselect all columns.

  1. Click Ok to save the changes.

The dialog box closes. The Material Planning Window appears with the modifications you just made.

Deducting Safety Stock Figures

Because MPS takes into consideration your safety stock levels, you can delete this safety stock from the calculation appearing in the Material Planning Window.

To delete safety stock figures, select Deduct Safety Stock from the View menu.

VISUAL deducts the safety stock figures and re-populates the Material Planning window.
