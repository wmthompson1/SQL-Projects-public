Specifying Email Lists for Documents

To specify the recipients of emailed reports:

  1. To set up customer recipients, select Sales, Customer Maintenance.

To set up vendor recipients, select Purchasing, Vendor Maintenance.

  1. Select the ID of the customer or vendor.

  2. Select Maintain, Email Documents.

  3. In the dialog, a row for each contact and site combination is displayed. For each contact, select which documents to send.

  4. The email specified for the contact on the contact’s record is displayed. To send the email to a different address, specify the address in the Send Docs to: Email Addr column.

  5. Click Save.
