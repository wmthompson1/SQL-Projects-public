Project Summary Window Tools

The Show Source for Details and Show Source for Actuals options in the Project Summary window allow you to examine individual transactions that make up your project summary figures. Double-clicking on a line in the Project Summary window table often brings up additional information.

To view source information:

  1. In the Project Summary window, click the Site ID arrow and select the site for which to view source information.

  2. Click the Project ID button to select the project you would like to view. The project information appears in the window.

  3. In the Summary Type(s) section, select the type of information you would like to view. The table is populated with the appropriate information.

  4. Double-click a line (including the total line). The Direct Costs Detail window appears.

The window breaks down costs by cost category.

  1. Click the X button to close the window and return to the Project Summary window.

  2. Select a line in the table, then select Show Detail for Actuals from the Info menu. The Detail for Actuals window appears.

  3. Click the X button to close the window and return to the Project Summary window.

  4. Select a line in the table, then select Show Source for Actuals from the Info menu. The Source for Actuals window appears.

  5. Double-click a line in the table to view more detailed information.

  6. Double-click a line to view the original document that created the actual cost. For materials, you can also click the Open Cost Source button. For labor, you can click the Open Work Order button.
