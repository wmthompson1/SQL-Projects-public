Receiving Shipments

You can receive shipments using the Delivery Schedule Shipment dialog box. You must use the Delivery Schedule Shipment dialog box with purchase order lines that contain delivery schedules.

  1. Select the purchase order line from the Shipping Entry window line item table.

  2. Click the Del Sched button.

The Delivery Schedule Shipment dialog box appears.

  1. Select the delivery schedule line you want to use.

  2. Click Save to ship the shipment.
