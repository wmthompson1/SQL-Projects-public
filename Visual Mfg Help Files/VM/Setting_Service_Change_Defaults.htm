Setting Service Charge Defaults

Use Site Maintenance to set any service charges your company may charge.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site for which you are setting up service charge defaults. If you are licensed to use a single site, this field is unavailable.

  2. Select Service Charge Defaults from the Maintain menu.

The Service Charge Maintenance dialog box appears.

  1. In the ID field, enter an ID for the service charge.

  2. In the Description field, enter an description of the service charge.

  3. In the Unit Price field, enter the unit price of the service charge.

  4. In the Trade Discount % field, enter any discounts.

  5. In the Commission % field, enter commissions.

  6. From the Product Code drop down box, select the product code for which you are setting this service charge.

  7. From the Commodity Code drop down box, select the commodity code for which you are setting this service charge.

  8. In the Sales Tax Grp ID field, enter a sales tax group ID for the service charge.

  9. In the Revenue Acc’t ID field, enter the account into which the service charge will be received.

  10. Click the Save button to save the service charge information.
