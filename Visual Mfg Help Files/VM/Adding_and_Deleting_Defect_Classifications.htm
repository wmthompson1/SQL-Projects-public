Adding and Deleting Defect Classifications

Use Quality Maintenance to add and delete defect classifications.

Adding Defect Classifications

  1. Select Defect Classification from the Edit menu.

The Defect Classification Coding dialog box appears.

  1. Click the Insert button.

A line appears in the list of the defect classifications.

  1. In the Name column, enter a name for the new defect classification.

  2. Click the button to save the new defect classification.

Deleting Defect Classifications

  1. Select Defect Classification from the Edit menu.

The Defect Classification Coding dialog box appears.

  1. Select the defect classification you want to delete.

  2. Click the Delete button.

An x appears to the left of the defect classification you are deleting.

  1. Click the Save button to complete the deletion.
