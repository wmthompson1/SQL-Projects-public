In-Context View Panels in the Order Management Window

You can view the Customer View Panel, Part View Panel, and Order View Panel in the Order Management Window.

To select these panels from the View menu, the Customer Order Entry window must be the active window. You can attach the panels to the left, right, or bottom edge of the Order Management Window. You cannot attach the panels to the Customer Order Entry child window.

To access the Customer View Panel, select View, Customer View Panel. Use the Customer View panel to view information about the selected customer. You can view summary information, detail information, and a chart. For information about setting up the summary and detail views, see In-context View Panels. For information about the chart, see Customer View Panel Chart.

To access the Part View Panel, select View, Part View Panel. For information about setting up the summary and detail views, see In-context View Panels. For information about the Part view panel chart, see Part View Panel Chart.

To access the Order Line View Panel, select View, Order Line View Panel. The Order Line View Panel contains a chart only. The chart shows information about costs and anticipated profit for the selected line. For information about the Order Line View Panel chart, see Order Line View Panel Chart.

To access the Production Cost and Gross Profit Panel, select View, Production Cost and Gross Profit. The chart shows information about costs and anticipated profit for the order. For more information about the Production Cost and Gross Profit Panel chart, see Production Cost and Gross Profit View Panel Chart.

To access the Process Activity View Panel, select View, Process Activity View Panel. The charts show information about the performance of your sales process compared to a standard that you set up. For information about the charts, see Viewing the Leadtime View Panel Charts.
