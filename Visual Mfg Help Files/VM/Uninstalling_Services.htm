Uninstalling Services

To uninstall an instance of a service for a site:

  1. In your VISUAL executables directory, locate the executable for the service.

  2. Right-click the executable and select Run as Administrator. The Sign In dialog is displayed.

  3. Specify this information:

User ID – Specify the user ID that the service uses to sign into the VISUAL database. This can be any valid VISUAL user ID.

Password – Specify the password associated with the user ID.

Database – Specify the database on which to run the service.

  1. Click Sign In.

  2. If the service is installed by site, in the Site ID field specify the ID of the site where you no longer want to run the service.

  3. Click Remove Service.
