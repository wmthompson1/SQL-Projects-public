Printing Warehouse Information

You can print warehouse information for one warehouse or all warehouses. Also, you can print barcodes.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site to use for the report. If you are licensed to use a single site, this field is not available.

  2. From the File menu of the Warehouse Maintenance window, select Print.

The Print dialog box appears.

  1. Enter the ID of the warehouse with which you want to begin the report in the Starting Warehouse ID field.

  2. Enter the ID of the warehouse with which you want to end the report in the Ending Warehouse ID field.

  3. Select the print output:

Print - The warehouse report is printed after you select standard print options.

View - The report is displayed on your screen before printing.

File - You are presented with a Print To File dialog box in which you can specify where you want VISUAL to place the report.

E-mail - Select this option to send the report in a Rich Text Format (.RTF) through electronic mail. On saving, a Send - Print dialog box appears that you can use to specify to whom you want to send the QRP report information.

  1. Select the Print Barcodes check box and select the type of barcode to include in the printed report.

Code39 – This barcode, also known as Code 3 of 9, contains variable length, discrete symbology. You must have a Code 39 barcode font installed to view the barcode. If you do not have the Code 39 font installed, then the alphanumeric ID is displayed instead with a prefix and suffix. This pattern is used: *%ID%*.

QR Code – This is a two-dimensional or matrix barcode. QR stands for quick response.

  1. Click Ok when you are finished setting print preferences.

If you selected Print, a standard print dialog box appears in which you can specify print preferences for the report.

If you selected View, the report appears on your screen for you to view before printing 

If you selected File, a dialog box appears in which you can specify where you want VISUAL to place the report.

If you selected E-mail, a Send - Print box appears in which you can specify to whom you want to send the report.
