Editing EDI Accumulator Fields

The customer information fields are editable. However, this information is usually received via EDI from the customer. The values that appear in the adjustment fields in the Actual section are the only fields that you should make a practice of editing. 

After modifying information, click the Save button to save changes.

The dialog box closes and you return to the Customer Order Entry window.
