Dispatching to Outside Services

You can dispatch parts to outside services using the Service Dispatch Entry window. You can dispatch parts from a work order or a purchase order.

To dispatch to an outside service from a purchase order:

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site in which the purchase order was created.

  2. In the Order ID field, enter the Outside Service Purchase Order ID.

The outside service purchase order information appears in the window.

  1. Enter the quantity you want to dispatch in the Dispatch Quantity column.

  2. Enter the appropriate quantity in the Actual Freight column.

  3. Enter the shipping weight in the Shipping Weight column.

  4. Enter the weight unit measure on the Weight U/M column.

  5. Enter any links in the Links column.

  6. Click the Disp All button to dispatch the specified quantity to the outside service vendor.

Dispatching to an Outside Service from a Purchase Order Using an Existing Dispatch:

  1. Site ID - Click the Site ID arrow and select a site to use.

  2. Enter the Dispatch ID in the Dispatch ID field.

The table populates with the outside service purchase order information.

  1. Enter the quantity you want to dispatch in the Dispatch Quantity column.

  2. Enter the appropriate quantity in the Actual Freight column.

  3. Enter the shipping weight in the Shipping Weight column.

  4. Enter the weight unit measure on the Weight U/M column.

  5. Enter any links in the Links column.

  6. Click the Disp All button to dispatch the specified quantity to the outside service vendor.

VISUAL asks you if you want to create a new dispatch.

  1. Click Yes to create a new dispatch.

The window populates with a new Dispatch ID and information.
