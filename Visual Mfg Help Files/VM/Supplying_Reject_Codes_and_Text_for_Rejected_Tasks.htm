Supplying Reject Codes and Text for Rejected Tasks

When changing the status of a task from Pending to Rejected, VISUAL prompts you to choose a reject ion code (a code classifying the reason for the rejection) and enter explanatory text. You don’t have to use a separate feature to do this, as VISUAL displays a dialog box automatically when you try to save the change to the database.

  1. Identify the task you want to reject, and then select Rejected from the Status list-box.

  2. Click the Save button or select Save from the File menu.

The On Hold Information dialog box appears.

The Task Type/Sequence #/ and Reference ID of the task appear in the title bar of the dialog box.

  1. Click the Code button to select a rejection code for the task.

The Task Rejection Codes dialog box appears.
