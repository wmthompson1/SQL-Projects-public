Displaying the Picture in the Part Maintenance Window

If you add a picture to a Part Maintenance record, you can display the picture in the Part Maintenance window. To display the picture, set up the ShowPicture preference setting.

To set up the preference:

  1. Select Admin, Preferences Maintenance.

  2. Click Insert.

  3. Specify this information:

Section – Specify PartMaintenance.

Entry – Specify ShowPicture.

Value – Specify Y.

  1. Click Save.
