Prompting for Part Revision Comments

If you manually control the revision numbering of the parts in your database (that is, you do not use the VE ECN module), VISUAL prompts you to enter details for a part revision when you save a record.

To activate this feature:

Select Prompt for Revision Comments from the Options menu.

VISUAL places a check mark next to the option.

Each time you assign a part a new revision number, VISUAL displays a comment dialog box into which you can enter comments regarding the revision.
