Using the Tree Display

If the Graphical Display is displayed when you open the Document Lifecycle Viewer, select View, Show Graphical Display to clear the check box next to Show Graphical Display. The Tree Display is displayed.

In the Tree View, you can perform these actions:

  • View document details

  • Configure document details information

  • View document details in separate tables

  • Open a related document

  • Print the Document Lifecycle Report
