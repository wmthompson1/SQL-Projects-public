Receiving Returned Materials

To receive returned materials from your customer:

  1. To open the Customer Order Shipment Return dialog box, perform one of these steps:

  2. From the RMA Entry window, select Edit, Customer Order Shipment Return.

  3. From the Shipping Entry window, select Edit, Return.

  4. From the Purchase Receipt Entry window, select Edit, Customer Order Shipment Return.

  1. Specify this information:

Customer Order ID – Select the ID of the order that contains the materials the customer is returning.

RMA ID – If an RMA is required for the return, specify the RMA ID. When you specify an RMA ID, the RMA record is updated with the date that the materials were returned after you click the Save button in this dialog box.

Return Date – Specify the date that the customer returned the items. By default, the current date is inserted.

  1. In the table, the lines from the customer order are inserted. On each line, specify the quantity of part that the customer is returning. If you specified an RMA ID, the returned quantity cannot exceed the quantity authorized on the RMA record. To specify the returned quantity, perform one of these steps:

  2. If the customer is returning a standard part, specify the quantity in the Qty Returned field.

  3. If the customer is returning a piece-tracked part, specify the quantity in the Pieces Returned field.

  1. In the Description column, optionally specify a description of the transaction. You can use the description to specify the reason for the return. If you view the transaction in Inventory Transaction Entry, the description that you specify is displayed in the Description field.

  2. Click Save.
