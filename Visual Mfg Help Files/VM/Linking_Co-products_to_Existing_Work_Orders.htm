Linking Co-products to Existing Work Orders

You can link co-products to existing work orders directly from Customer Order Entry .

  1. From the Customer Order Entry window, enter the customer order ID in the Our Order ID field.

The information for the order appears in the Customer Order Entry window.

  1. Click the Insert button to add a new line item.

A blank line appears in the table.

  1. Enter the quantity of the part created in this work order in the Quantity column.

  2. In the Job ID column, enter the work order ID to which you want to link the co-product.

  3. In the Part ID column and enter the co-product part ID.

If you double click the Part ID column header, a dialog box appears, asking if you want to browse the Parts table by Part ID, Part Description, or Co-products.

  1. Enter the unit price for the part in the Unit Price column.

  2. Click the Save button to save the new line item.
