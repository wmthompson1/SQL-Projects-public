Accessing Online Help

To access the online Help system from the main window, select Help, Contents or press the F1 key.

To access the online Help system from all other windows, select Help, Help Topics or press the F1 key.

When you press the F1 key from within a VISUAL window, the Help system opens to the appropriate topic in the help file. For example, if you are working in Part Maintenance and want to view the help for Part IDs, click in the Part ID field and press the F1 key. The help system opens at the Part Maintenance book and Part ID topic.

You can also use the What’s This? toolbar button to open context-sensitive information. To use this feature, click the What's This? button, and then click a field in the window.

The left pane shows the table of contents, index, and search function. Use the tabs to navigate between functions. When you select a topic, the contents of the topic are displayed in the right pane.
