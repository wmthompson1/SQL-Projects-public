Setting Customer Shipment Thermal Label, Packlist & BOL Print Options

You can configure print options so that thermal labels and packlists, and bills of lading print automatically after VBTS processes a transaction and assigns it a status of Completed.

With the VISUAL BTS Customer Shipments window open, select Print Options from the File menu.

The BTS Print Options dialog box appears.

Select the Auto Print Thermal labels check box if you want VBTS to print thermal labels for all customer shipment transactions when they reach a Completed status.

Select the Auto Print Packlist check box if you want VBTS to print packlists for all customer shipment transactions when they reach a Completed status.

Select the Auto Print BOL check box if you want VBTS to print bills of lading for all customer shipment transactions when they reach a Completed status.

Click the Save button to commit these settings.
