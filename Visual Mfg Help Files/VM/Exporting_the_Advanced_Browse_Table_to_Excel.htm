Exporting the Advanced Browse Table to Excel

Use the Export to Excel option to export data from the browse table to your database.

To export data:

  1. Access the appropriate advanced browse dialog.

  2. Filter the browse table as needed.

  3. Click Export to Excel. The data currently present in the browse table is exported. If you have hidden columns, information in the hidden columns is not exported.

If you make any changes to your Excel file after you export the information, the information is not saved in your database unless you use the Import to Excel function.
