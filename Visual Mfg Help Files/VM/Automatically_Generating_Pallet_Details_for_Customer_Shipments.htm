Automatically Generating Pallet Details for Customer Shipments

When processing shipper lines, pallet details are automatically generated for the shipment lines according to container settings in the part master.

The Shipping Entry window features an option that allows you to view pallet details as the order is saved. With this option selected, the Pallet Details dialog box appears before the order is shipped and an inventory transaction is created, so you can verify all details and modify them if necessary.

To set this option, select Ask for Pallet Details on Save from the Options menu or press the F9 key.
