Print Open Purchase Orders

You can print all Purchase Orders with a status of Firm or Release that were opened on or before a specified date.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site to use. If you are licensed to use a single site, this field is unavailable.

  2. Select Print Open Purchase Orders from the File menu.

The Print Open Order Report dialog box appears.

  1. Enter the Dated Through date.

This causes the system to print all open purchase orders that have a desired receive date up to and including this date. Any open purchase orders that have a later date do not print.

  1. Select the appropriate Vendors radio button.

Click the Print Current Vendor radio button if you want only purchase orders for the selected vendor. Click the Print All Vendors radio button to print all vendors.

  1. Select the sequence for the purchase orders.

You can choose to sequence by Order ID, Vendor ID, Part ID, or Service ID.

If printing by Order ID or Vendor ID, one line prints for each PO, showing vendor, date, and shipping information. Additionally, one line prints for each PO line item, showing Part/Service ID, Description, Quantity, and Unit of Measure. If sequencing by Vendor ID, the Order ID lines are grouped by Vendor.

If printing by Part ID or Service ID, a section prints for each Part ID or Service ID, showing ID and description. Under each section, one line is shown for each purchase order line item for the Part/Service. For each line, Order ID, Vendor ID/Name, Order Date, Recv Date, Quantity, and U/M are shown. These lines are sequenced by Order ID.

  1. Choose the output destination by clicking on the arrow next to the output field. Select one of the following:

Print - Select this option to output the file to a printer.

View - Select this option to preview what the printed output will look like when printed.

File - Select this option to save the information to a text file.

  1. If you selected the Print in Step 5, click Ok to output the information.

A standard Windows printer dialog box appears, allowing you to select the number of copies to print.

  1. Select the Delivery Detail check box if you want to print delivery schedule information.

  2. Make the appropriate selection and click Ok.
