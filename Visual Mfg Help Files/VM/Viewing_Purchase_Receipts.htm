Viewing Purchase Receipts

You can view purchase receipts for a specified order using Vendor Inquiry.

To view purchase receipts:

  1. Call up the order for which you want to check purchase receipts into the Vendor Inquiry item window.

  2. Select List Receipts/Invoices from the Info menu.

The Receipts dialog box appears, listing all of the receipts for the current order. You can view the received date, the quantity received, reference information, the invoice ID, and the invoice date.

  1. Click the Close button when you are finished.
