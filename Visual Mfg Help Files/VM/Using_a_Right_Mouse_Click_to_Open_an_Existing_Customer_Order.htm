Using a Right Mouse Click to Open an Existing Customer Order

You can open an order by right-clicking on it in the Customer Order History child window.

  1. If not already open, open the Customer Order History child window by selecting Customer Order History from the Tools menu.

  2. In the Customer Order History window, double-click the Customer icon.

All existing orders for the customer appear.

  1. Highlight the order you want to open and right-mouse click.

An option box appears.

  1. Select Open.

VISUAL opens the order and loads the order data into the Order Entry window. Maximize the window to view the order information.

  1. Edit the customer order information as necessary and click the Save button on the Customer Order main toolbar.
