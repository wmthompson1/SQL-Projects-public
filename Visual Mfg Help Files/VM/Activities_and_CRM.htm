Activities and CRM

If you use VISUAL CRM 8.0, you can view CRM tasks in VISUAL and VISUAL activities in CRM. Earlier versions of VISUAL CRM are not compatible with VISUAL 8.0.0. If you specify CRM connection settings in Activity Maintenance, you can open CRM directly from Activity Maintenance.

CRM tasks and VISUAL activities are stored in the same database table. If you use VISUAL CRM, you can view any CRM task in Activity Maintenance. You can also edit information for tasks you create and tasks assigned to you directly in Activity Maintenance. If you set up CRM connection settings, you can click the drill-down button next to a CRM Task ID to open the task in CRM Task Maintenance.

In CRM, you can use CRM Task Maintenance to edit information for VISUAL activities you create or are assigned to you.

CRM and VISUAL also share the Activity Category and Activity Type tables. Any category or type you create in one application is available in the other application.
