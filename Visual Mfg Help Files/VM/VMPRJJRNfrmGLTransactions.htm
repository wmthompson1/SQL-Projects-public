Making Project Adjustments

To make general ledger adjustments for your projects:

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site to use for this adjustment. If you are licensed to use a single site, this field is unavailable.

  2. If you want to change the Entry Date, click the calendar button and select the date you want.

  3. Click the Currency ID arrow and select the currency you want to use for this adjustment.

  4. If you want to manually number this adjustment, click in the Transaction ID field and enter the ID you want to use for this adjustment. Click in the Description field and enter a description for this adjustment.

  5. In the Post Based On section, select on what you want this adjustment to be based.

You can select:

All Tracking Currencies

Document Currencies Only

  1. Click the Insert Row toolbar button.

A new row appears in the table.

  1. The Site ID column automatically populates with the site currently selected.

  2. Click the G/L Account ID browse button and select the account you want to adjust.

You must select a Posting Level Account.

The account description appears.

  1. Click in the appropriate Credit or Debit column and enter the amount of adjustment you want to make.

  2. Enter the following information for each line of the adjustment:

Reference - Click in the Reference column and enter the reference you want to use for this adjustment.

Project ID - Double-click the browse button and select the project you want to reference for this adjustment.

WBS Code - Double-click the browse button and select the WBS Code you want to reference for this adjustment. The Proj Ref Sub ID and Proj Ref Seq No for the WBS Code appear.

Dept ID - Double-click the browse button and select the department you want to reference for this adjustment.

Cost Cat ID - Double-click the browse button and select the Cost Category you want to reference for this adjustment.

Account Year - The system inserts the current account year.

Hours - Specify the number of hours to adjust.

  1. Because adjustments should always be made in pairs—Debit from one account and Credit another account the same amount, click the Insert Row toolbar button and complete the adjustment.

  2. Click the Save toolbar button.
