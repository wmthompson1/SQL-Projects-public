Applying Change Notices to Work Orders

This function provides a means of executing the ECN changes done against the master on each eligible work orders of the same part if the work order is unreleased, released, or firmed.

  1. Select Engineering Change Notice from the File menu, then Apply Change Notice to Work Orders from the pop out menu.

The Apply ECC to Work Orders dialog box appears.

  1. Enter or select and engineering change notice.

A list of eligible work orders appears in the dialog box.

  1. If you want to skip certain work orders, click in the Skip check box.

  2. Click the Save button.

The start and end quantities are re-calculated for the work orders.
