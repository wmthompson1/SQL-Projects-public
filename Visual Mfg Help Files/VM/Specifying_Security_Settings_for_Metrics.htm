Specifying Security Settings for Metrics

Use Security Maintenance to restrict access to system and custom metrics. You can secure metrics for individual users and for security profiles.

If a user does not have access to a metric, the user cannot add the metric to a dashboard or view the metric on a shared dashboard.

  1. Select Security, Security Maintenance.

  2. Perform one of these steps:

  3. To specify security settings for profiles, click Profile. Click the Profile ID browse button and select the profile.

  4. To specify security settings for an individual user, click User. Click the User ID browse button and select the user ID.

  1. Click the Dashboard Item Security tab. By default, all analytics and detail grids are listed. Use the Custom Analytic, System Analytic, Custom Detail Grid, and System Detail Grid check boxes to filter the information in the table. For example, to view only custom analytics, click the System Analytic, Custom Detail Grid, and System Detail Grid check boxes.

  2. To prevent the use of an analytic or grid, clear the Access check box.

  3. Click Save.
