Activities in Service Dispatch Entry

Activities are available in Service Dispatch Entry. Use activities to manually create reminders about follow-up items or to record information about services.

To add a new activity, select Edit, Activity Entry. See Creating Activities.

To view existing activities, select View, Activities. To view all activities related to all outside services, select View, Activities from a blank Service Dispatch Entry window. To view all activities related to a particular service dispatch, select the service dispatch in Service Dispatch Entry before selecting View, Activities. See Reviewing Activities.
