What is the Order Management Window?

The VISUAL Order Management Window is a standard part of VISUAL designed to work with Infor VISUAL DCMS. Aside from providing all of the functions that the basic Customer Order Entry window provides, the VISUAL Order Management Window features an advanced GUI ideally suited for high-volume order entry. See later in this section for more information.

The Order Management Window features 5 child windows that allow you total, real-time access to pertinent order information.
