Pricing Repair Work Orders for Repair RMAs

You can use these options to charge customers for repair costs:

Use quote pricing from the evaluation RMA – This option is available only if you created the repair RMA by converting from an evaluation RMA, and you created a quote for repairs as part of the evaluation process. To use the pricing you specified on the quote for the repair costs, select the Use Quote Pricing check box before you convert the evaluation RMA to a repair RMA. See Converting to a Repair RMA.

Use the customer order repair line – When you create a repair RMA, either a new customer order is created for the repair or a new line for the repair is inserted on the original customer order. You can use this new line to charge customers for the cost or repair. Use this option if you did not create a work order for the repair RMA, or if you charge a fixed cost for repairs.

Use the costs from the repair work order – If you created a work order for the repair, you can use the costs incurred for the work order as the basis for the repair costs. You can specify an hourly labor rate, and add mark-ups to labor, material, burden, and service costs. Use the Price Repair Orders dialog box in the RMA Entry window to add the mark ups. You must first ship the repaired items before you can use the Price Repair Orders dialog box.
