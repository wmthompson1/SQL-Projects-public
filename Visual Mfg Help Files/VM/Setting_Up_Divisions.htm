Setting Up Divisions

Division codes represent the actual divisions of your company you want to use for reporting purposes.

To set up Division codes:

  1. In Project Maintenance, click the Maintain menu and select Division Table.

The Division table appears.

  1. Click the Insert button.

The next available row in the table appears highlighted.

  1. Enter the Code and Description you want to use.

  2. Click the Save button.

  3. When you have finished entering Division Codes, click the Close button.
