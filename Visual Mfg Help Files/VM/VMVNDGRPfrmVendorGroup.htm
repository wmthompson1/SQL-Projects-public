Adding Vendor Groups

Each vendor ID can belong to only one vendor group.

To add a vendor group:

  1. Select Sales, Vendor Group Maintenance.

  2. In the header, specify this information:

Vendor Group ID – Specify an ID for this vendor group.

Group Name – Specify a description of the vendor group ID.

Bill to Address – Specify the bill to address for the vendor group.

  1. To add vendor records to the vendor group, click the Insert button.

  2. Double-click the Vendor ID browse button and select the vendor to add to the group. Each vendor ID can belong to only one group.

  3. Click Save.
