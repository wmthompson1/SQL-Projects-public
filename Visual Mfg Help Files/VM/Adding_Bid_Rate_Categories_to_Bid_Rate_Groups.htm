Adding Bid Rate Categories to Bid Rate Groups

To add bid rate categories to bid rate groups:

  1. Select Admin, Application Global Maintenance.

  2. Select Maintain, Bid Rate Maintenance, Bid Rate Group Categories.

  3. In the Bid Rate Group field, select the group to which you are adding categories.

  4. Click Insert.

  5. Double-click the category ID browse button and select the category.

  6. Click Save.
