Establishing Part Price Lists

You can establish prices for parts used in equipment maintenance. A Part Price ID can have multiple parts. Use Part Price IDs to group together parts with similar prices and discounts. For example, an ID could consist of five parts, all of which have a price of 3.00-4.00 and a discount of 3-5%. Of course, a Part Price ID can also be made up of parts with disparate prices and discounts. There are no restrictions on the parts you can include in a Part Price List ID as long as they are valid VISUAL parts.

  1. From the Maintenance menu, select Part Price List Maintenance.

The Part Price List dialog box appears.

  1. If you are only editing an existing profile, select the profile from the Part Price ID list box.

  2. For new profiles, enter a name for the profile into the Part Price ID box.

  3. For new price profiles, click the Insert button.

A new line appears in the table.

  1. Double-click the <Part ID> column header to view the part table and select a part. All parts currently existing in your VISUAL database appear in this table.

VISUAL auto-fills the part’s description in the Description field.

  1. Enter a price for the part. You can enter a maximum of 10 numeric characters in the field.

  2. If you plan to offer a discount on the part, enter a discount for the part in the Discount field. After tabbing out of the field, VISUAL places a % after the discount value.

Enter as many lines as necessary. If you want to copy an existing ID to another ID, click the Copy button.

The Copy Part Price List dialog box appears.

Enter a unique ID or select one from the list box.

  1. Click the Save button. VISUAL adds the new ID if you specified a new ID or modifies the ID if you specified an existing ID.

When you are finished adding, modifying, or deleting Part Price ID information, click the Save button. Click the Close button to return to the Equipment Maintenance window.
