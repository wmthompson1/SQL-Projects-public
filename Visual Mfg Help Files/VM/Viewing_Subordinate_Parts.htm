Viewing Subordinate Parts

The Material Planning Window allows you to easily work with each level of the bill of materials for a fabricated part.

To View Subordinate Parts:

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site to use. If you are licensed to use a single site, this field is unavailable.

  2. Select the fabricated part whose subordinate parts you want to view.

You can do this by entering the Part ID, or by selecting from the browse or search list. The part must be a fabricated part, unless you are using the Use Selected Material Requirement Order option (see below).

  1. Select Subordinate Parts from the Info Menu, then select one of three choices from the pop out menu for the determination of subordinate parts:

Use Engineering Master - VISUAL uses the current engineering master for the Part ID to get the list of material requirements for the part. Any line currently selected in the planning window has no effect.

Use Selected Material Requirement Order - This allows viewing of subordinate parts for a work order requiring the current purchased or fabricated part. Before using this option, select the line with a requirement pegged to the work order of interest. This moves up a level, and allows material planning for all of the sibling parts of the current part.

Use Selected Work Order - VISUAL uses a specific Work Order to determine material requirements. Before using this option, select the line containing the desired order. This option is useful if a work order for a standard part has been customized, and may contain additional material requirements not specified in the master.

The Subordinates Parts List dialog box appears, listing all the subordinate parts based on your selection above.

  1. Select a Subordinate Part to view. Double-click the part’s line or select the line and click View Part.

You can also view the parent part (the part requiring the subordinate parts) of the subordinate parts listed, by pressing the View Parent Part button. This is especially useful when using the Use Selected Material Requirement Order option.

  1. Use the up-arrow and down-arrow buttons to the left of the Part ID field to navigate among all subordinate parts.

Press the down-arrow button to automatically view the next subordinate part in the list.

Press the up-arrow button to view the previous part.
