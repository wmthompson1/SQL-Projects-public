Setting a Customer Order Default Status

Choose a default status for new customer orders VISUAL creates.

  1. From the Edit menu, select Customer Order Defaults.

The Customer Order Defaults dialog box appears.

  1. Select the appropriate option button.

Firm - VISUAL creates customer orders with a status of firmed. Firmed customer orders are not ready for the floor. Work may not begin on them until their status moves to Released. 

Release - VISUAL creates customer orders with a status of released. Released work orders are ready for the floor; work may begin on them at any time.

  1. Click the Save button to commit settings.
