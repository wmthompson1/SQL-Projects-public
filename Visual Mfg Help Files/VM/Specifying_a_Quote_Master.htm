Specifying a Quote Master

  1. Specify the quantity in the Quantity field.

This indicates a lot size for manufacturing of the part. This may be independent of any of the quoted quantities. This field defaults to 1.

This quantity does not enter into cost calculations for different quoted quantities. Those costs are always computed based on a lot size equal to the quantity being quoted. However, you may want to set this quantity if you know that you will be producing the part in a particular lot size.

  1. If necessary, fill in Engineered by, Engineer Date, Drawing #, and Rev information.

  2. If you want VISUAL to forward schedule any placed work order from the release date, check the Forward Schedule check box.

  3. To add an operation to the master, click the Insert button located directly under the operation table.

A new line appears after any current lines. VISUAL automatically assigns the operation sequence number. You can change the sequence number if you want to insert the operation between two existing operations.

  1. If an operation type has been specified in Shop Resource Maintenance for the desired operation, type its name into the Operation Type field.

You may also select from the existing operation types by selecting the field and activating the drop-box menu. This automatically completes all information defined in the operation type. Use of operation types for building quotes is highly recommended.

  1. If you are not using an operation type, enter the Resource ID for the operation. You may type the ID, or select from a list by double-clicking the column header. Entry is required; each operation must specify a Resource ID.

  2. Enter or check the remaining time and cost items.

  3. Click the Save All button to save the Quick Quote information.
