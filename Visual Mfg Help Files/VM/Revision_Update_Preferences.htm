Revision Update Preferences

Set update revision preferences to apply to parts and document ECN incorporation.

  1. From the Options Menu, select Revision Updates.

The Revision Update Preferences dialog box appears.

  1. To update part revisions on incorporation of ECNs, select the Update Part Revision on incorporation of ECN check box.

After completing the ECN, VISUAL updates the part revision automatically according to your stage numbering scheme.

  1. To update document revisions on incorporation of ECNs, select the Update Document Revision on incorporation of ECN check box.

After completing the ECN, VISUAL updates the document revision automatically according to your stage numbering scheme.

  1. Select the point in the ECN at which the revision update should occur:

Authorization – Select the Authorization option button to update revisions after authorization stage tasks are completed.

Implementation – Select the Implementation option button to update revisions after implementation stage tasks are completed.

Approval – Select the Approval option button to update revisions after approval stage tasks are completed.

Distribution - Select the Distribution option button to update revisions after distribution stage tasks are completed.

For each new ECN, VISUAL selects the Update After check box next to the appropriate task stage.

  1. Click the Close button to return to the ECN Entry.
