Editing a Linked Object

  1. From the Manufacturing Window object dialog box, select Picture/Object from the Edit menu.

The linked object appears in the Set Picture/Object window.

  1. Double click the object in the Set Picture/Object window, or select Edit Object from the Edit menu).

The application in which the object was created opens with the linked file loaded and ready for editing.

  1. Make the desired changes to the document and save the changes in the application. Remember that you are changing the original file and must save it when finished.

Now you need to check the links for the files you just edited. By default links are automatically updated in the Manufacturing Window.

  1. Click the Links button.

The Links dialog box appears.

The automatic button should be on, indicating that changes to linked files automatically take place. For more information on maintaining linked files refer to the Windows User’s Manual.

  1. Click the Ok button to close the Links dialog box and return to the Set Picture/Object dialog box.

  2. Click the Save and Exit button to accept the linked object and return to the Manufacturing Window.
