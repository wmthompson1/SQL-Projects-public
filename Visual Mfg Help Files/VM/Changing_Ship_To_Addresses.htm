Changing Ship To Addresses

  1. Click the Ship Addr button.

The Ship To Addresses dialog box for the vendor appears.

The table at the top of the dialog lists the alternate addresses, if any, for the vendor.

  1. To use an alternate address for this purchase order, select the line of the appropriate address and click the Use Selected Ship To Address button.

  2. If you want to enter Intrastat information for the shipping address, click the Intrastat tab and enter information in the following fields:

Port of Arrival - Enter the place to which goods are received for this shipping address.

Country ID - Enter the Country ID for the shipping address.

Nature of Transaction - Enter the nature of transaction for this shipping address.

Mode of Transport - Enter the method by which goods are delivered to this shipping address.

Siret Number - Enter the siret number for this shipping address, if necessary.

  1. Click the Save button to save the new information.

To change back to the vendor’s main address, click the Clear Ship To on Order button.

The dialog box automatically closes. The selected address now appears below the Purchase From address in the Purchase From/Ship To window at the top of the Purchase Order Entry window.
