Report Output Options

These options are available for most reports:

Print - Select this option to send the report to a printer.

View - Select this option to view the report on screen. See Viewing Reports.

File - Select this option to save the report as a text file.

E-Mail - Select this option to attach the report to an E-Mail. If the PDF Format check box is available, select the check box to attach the report as a PDF. If you clear the check box or the check box is not displayed, the report is attached as an RTF.
