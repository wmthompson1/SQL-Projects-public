Viewing Cost Simulation G/L Entries

If you use a simulation to update costs in a site whose parent entity uses the standard costing method, you can view the G/L transactions that were generated when the costs were updated.

  1. Select Inventory, Cost Simulation.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site to use for the simulation. If you are licensed to use a single site, this field is unavailable.  

  2. Click the Simulation ID browse button.

  3. In the browse dialog, select the Include Updated Simulations check box.

  4. Select an updated simulation.

  5. Select Info, G/L Entries. The transactions made when the simulation was applied are displayed.
