Maintaining Estimate to Complete Figures

As you complete the work on your projects, you may want to revise the estimated time and cost to complete figures for the remaining project tasks. VISUAL allows you to do this at a project wide level (all tasks will be completed at 90% of original budget) or at the cost category level (senior engineering time in May will be 90 hours instead of 100).

To initialize a project’s first Estimate to Complete figures, you must do so from the Budget Maintenance window.

To calculate Estimate to Complete (ETC) figures:

  1. Open the Project Summary window.

  2. From the Maintain menu, select the ETC option.

The Maintain ETC dialog box appears.

  1. Click the Project ID button and select the project for which you want to calculate Estimate to Complete figures.

  2. If you want to designate an ETC version, enter an ID in the ETC Version field. If you would like to use the numbering parameters you've set up, leave the field blank.

  3. Click the Version Date field to select a date for the ETC.

  4. If the ETC has been linked to a formal EAC, the ID of the linked EAC appears in the Linked EAC Version field.

  5. If you want to specify a range of WBS Codes for this calculation, click the Starting WBS button and select the code on which you want VISUAL to start its calculations. Click the Ending WBS button and select the code on which you want VISUAL to end its calculations.

  6. Select the following check boxes as appropriate:

Locked – Select the Locked check box if you do not want to allow any changes to the ETC version. When you select the Locked check box, the Delete Row and Insert buttons are inactive. The system selects the Locked check box if the linked EAC is locked.

Current Version – Select the Current Version check box to designate the ETC as the actively used version. If you previously marked another ETC version as the current version, the system clears the Current Version check box on the previous version when you select a new current version.

Baseline – The system selects the Baseline check box if the EAC linked to the ETC is the baseline version. The system maintains this check box; you cannot select it or clear it.

  2. Use the scroll bar to locate the fiscal period to which you want to add costs. If the appropriate fiscal period does not appear in the lower pane, use the Insert Start Period and Insert End Period buttons to add periods to the table.

  3. Click in corresponding column for the information you want to add and enter the amount or number of hours for your new estimated costs.

  4. Click the Save button.

VISUAL calculates the Estimate to Complete figures for the project you selected and populates the Project Summary window with the results.
