Enabling Intrastat

Intrastat is an optional feature in VISUAL; you must have the correct serial number to enable Intrastat. If you have questions, please contact Infor Global Solutions Customer Support.

If you are running a version of VISUAL with an Intrastat-enabling serial number, you still need to enable Intrastat in Accounting Entity Maintenance:

  1. Select Admin, Accounting Entity Maintenance.

  2. In the Entity ID field, select the entity for which you are enabling Intrastat.

  3. Click the Intrastat tab.

  4. Select the Intrastat Enabled check box.

  5. Click Save.
