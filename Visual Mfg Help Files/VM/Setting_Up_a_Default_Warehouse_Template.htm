Setting Up a Default Warehouse Template

After you create the warehouse you would like to use as the default, designate the warehouse as the default through the Defaults tab of VISUAL’s Application Global Maintenance window. When you create a new project and a new project warehouse, VISUAL will copy the default warehouse’s characteristics.

To set a default warehouse to copy:

  1. From the main window, select Application Global Maintenance from the Admin menu.

  1. Click the Defaults tab.

  2. In the Projects section of the Defaults tab, click the Default Warehouse to Copy button and select the appropriate warehouse.

  3. Click the Save button on the toolbar.
