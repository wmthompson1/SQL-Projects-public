Assigning Supply from Returned Shipments with Delivery Schedule Lines

If a customer delivery schedule line that you have shipped has links of demand attached to it, you can assign supply quantities to those links during the return process as well as you can assign ship quantity to supply links when you are shipping the delivery schedule.

  1. With the customer order delivery schedule for which you are entering return receipt information in the Customer Order Shipment Returns dialog box line item table, place the cursor in the Qty Returned column.

The Delivery Schedule Returns dialog box appears automatically. The delivery schedule line appears in the line item table.

  1. Enter the return value in the Quantity Returned column.

  2. Highlight the line and click the Supply button.

The Assign Return to Supply dialog box appears.

  1. Enter a value in the Assign Quantity column. This value cannot be greater than the Link Issued Quantity. If none of the allocated "link" quantity is issued, it is impossible for you to assign supply against a return quantity that was never issued.

  2. Click the Ok button.

The Delivery Schedule Returns dialog box reappears.

  1. Click the Ok button.

The Customer Order Shipment Returns dialog box reappears with the return quantity in the Qty Returned column.

  1. Click the Save button to receive the return delivery schedule quantity.
