Setting Up Price Book

Before you can begin using VISUAL Price Book, you must register the database. Typically, your system administrator would set up your system files, including database registration. See the VISUAL Price Book installation directions for more information.

After you register the appropriate database, you must run VISUAL Price Book to set up the Topology table.

To access the Topology table:

  1. Start VISUAL Price Book.

  2. In the Navigation Pane of the VISUAL Price Book window, double-click the Administration icon. The Topology table appears.

  3. On the VISUAL Price Book main toolbar, click the Insert Row button.

  4. Enter a Site ID. This is a user-defined field and is typically used to identify the code of the location signed onto the VISUAL Price Book database.

  5. Enter a Site Type. The Site Type is identification for the kind of site signed onto the VISUAL Price Book database. This allows you to keep track of the various locations and sites by industry or department.

  6. Enter a Site Name. This is identification for the location signed onto the VISUAL Price Book database. This is generally the name of the company or department.

  7. Enter an Instance Name. This is an identification of the database connected to the VISUAL Price Book database. Typically, this is the identification code of the database against which your ERP system runs.
