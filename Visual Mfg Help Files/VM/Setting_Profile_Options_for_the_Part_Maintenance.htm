Setting Profile Options for the Part Maintenance

You can set security for the different components of Part Maintenance from the Program Security dialog box.

  1. Highlight Part Maintenance from the Program Security dialog box.

  2. Click the Pgm Components button.

The Program Security dialog box appears for Part Maintenance, listing all of the components for Part Maintenance.

  1. Click in the Access field for the different components and set the access.

  2. Click the Save button to save the access information.
