Using WBS Code Dates as Frozen Schedule Dates

You can use the dates associated with your WBS codes in the project header card as frozen schedule dates. To enable this function, open Preferences Maintenance. In the Scheduler section, find the entry, FreezeOpsWithWBSDates. Set the value to Y (the line will be FreezeOpsWithWBSDates=Y). The WBS dates will be treated as frozen schedule dates
