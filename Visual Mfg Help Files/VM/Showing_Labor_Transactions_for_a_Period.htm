Showing Labor Transactions for a Period

If traceability for a fabricated part is being reported during Labor Ticket Entry, then you can view labor ticket transactions for a lot number.

To do this, select the trace record from the Part Trace Maintenance table, then click the Show Labor Trans button. You can also select Show Trace Labor Transactions from the Options menu.

The Traceable Part Labor Tickets dialog box appears, with one line for each associated labor transaction.

The following information appears for each transaction:

Trans # - Transaction ID of the labor ticket. You could enter this number in Labor Ticket Entry to call up the ticket.

Quantity - The Quantity Completed reported for the ticket.

Type - The type of labor transaction, usually Run.

Work Order ID - The Base ID/Lot ID and Operation Sequence Number of the work order operation being reported against. This is the operation marked Begin Traceability At This Operation.

Hours Worked - The Elapsed Time from the labor ticket.

Transaction Date - The date of the labor ticket.

Employee ID - The ID of the employee who performed the labor.

Resource ID - The shop resource associated with the operation.
