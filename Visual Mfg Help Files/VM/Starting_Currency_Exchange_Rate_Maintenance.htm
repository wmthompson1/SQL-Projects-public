Starting Currency Exchange Rate Maintenance

Currency Exchange Rate Maintenance is available from the Maintenance menu of the Infor VISUAL Main window.

To access the Currency Exchange Rate Maintenance window:

  1. Select Currency Exchange Rate Maintenance from the Sales menu.

  2. The Currency Exchange Rate Maintenance window appears with no information in it.

  3. The Currency Exchange Rate Maintenance window contains both entry fields and action buttons. Use these fields and buttons while working in Currency Exchange Rate Maintenance.
