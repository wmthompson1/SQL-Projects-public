Freezing Rates

If you would like to preserve burden amounts in a certain period even if the Use for Actuals rate is changed, you can freeze the rates. When you freeze rates, costing uses the same rate ID that it previously used on any given transaction within that period. To freeze rates:

From the Rate Management window, select Maintain, Period Table....

In the Period Table, select the Freeze Rates check box for the appropriate period.

Costing will apply whichever rate was last used on a transaction, preventing changes to burden amounts.
