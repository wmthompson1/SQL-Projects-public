Testing Analytics

To test the performance of an analytic:

  1. Select File, Dashboard.

  2. Select Edit, Edit Analytic.

  3. In the Configuration ID field, select the analytic. The amount of time it took to construct the analytic is displayed above the tabs.

  4. To view the data used to construct the analytic, click the Chart Data tab.

  5. To view all visible columns from your data source, click the Detail Data tab. To populate the table, select the Populate Detail check box.
