Viewing Costing Information

To view costing information for your projects:

  1. If you want to run a cost analysis for a specific project, click the Project ID button and select a project ID from the list. Select the version of the project you want.

  2. Click the Costs to Check arrow and select the costs you want to appear in the analysis table.

You can select:

  • Purchases

  • Purchase Material

  • Purchase Services

  • WIP

  • WIP Labor

  • WIP Issues

  • Adjustments

  • Project Summary

  1. In the Selection section select:

Starting Order ID - The first order you want VISUAL to include in the analysis.

Ending Order ID - The finishing order you want VISUAL to include in the analysis.

Posting Date - The posting date you want VISUAL to use for the analysis.

  1. In the Options section select the following settings:

  • Transactions

  • Detail

  • Exceptions Only

  1. If you selected Exceptions Only, enter the following settings:

Plus/Minus % - Enter the percentage amount you want VISUAL to include in the analysis.

Plus/Minus Amount - Enter the value you want VISUAL to include in the analysis.

Plus/Minus Hours - Enter the number of hours you want VISUAL to use for the analysis.

  1. If you selected Project Summary in the Costs to Check section, make the following selections in the Proj Summary Selection section:

  • Material

  • Labor

  • Adjustments

  • Payables

  • General Journal

  1. Click the Run toolbar button.

VISUAL populates the table with the information you requested.
