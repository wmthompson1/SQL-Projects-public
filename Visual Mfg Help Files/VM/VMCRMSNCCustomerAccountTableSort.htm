Sorting the Customer/Account table in the Synchronize Customer/Account Utility

You can modify the Customer/Account table by filtering columns, moving columns and prioritizing a column sort based on contact information.

To sort the Customer/Account table:

  1. Navigate to the Infor VISUAL installation directory and double-click VMCRMSNC.EXE. If the default VISUAL installation directory is used, the Synchronize Customer/Account Utility launches from  C:\Infor\VISUAL\VISUAL MFG.

  2. Select Options > Preferences.

  3. Select the customer/account information identifier on which to set the sort priority, and click a priority number in the sort selection area, or click Clear Selection to reset a sort priority. You can set sort priorities on up to eight customer/account identifiers.

  4. Select the check box in the Descending column to perform a descending sort from Z to A or from the largest number to the smallest number. Clear the check box to perform an ascending sort from A to Z or from the smallest number to the largest number.

  5. Click Ok to confirm your sort priorities.
