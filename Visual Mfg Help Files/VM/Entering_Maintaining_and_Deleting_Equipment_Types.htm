Entering, Maintaining, and Deleting Equipment Types

Use the Equipment Types dialog box to enter and maintain equipment types.

  1. From the Maintenance menu, select Equipment Type Maintenance.

The Equipment Types dialog box appears.

  1. To enter a new Equipment Type, click the Insert button.

Enter information in the following fields:

Type - The type of equipment this is. Enter any alphanumeric combination as a unique identifier of the type.

Description - A description of the type. Be detailed in your descriptions. For example, use this field to supply a cryptic alphanumeric equipment type such as 44RR with a description that greatly clarifies actual equipment characteristics and use.

  1. Click the Save button to commit information.

  2. To modify information, place the mouse pointer in the appropriate column in the appropriate line and manually edit the information. When complete, click the Save button.

  3. To delete an equipment type, highlight the appropriate line and click the Delete button.

VISUAL places an X at the beginning of the line.

  1. Click the Save button to delete the line.

  2. Click the Close button to return to the Equipment Maintenance window
