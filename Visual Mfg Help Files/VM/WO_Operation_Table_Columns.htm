W/O Operation Table Columns

Depending on your table preferences the following columns may be available:

  • Sub ID

    The subassembly number of the operation.

  • Sequence No

    The sequence number of the operation.

  • Resource ID

    The resource ID for the operation. This information comes from the database.

  • Test ID

    The test ID assigned to the operation.

  • Test Description

    The test description assigned to the operation.

  • Test Status

    The status of the test assigned to the operation.

  • Start Quantity

    The calculated start quantity of the operation.

  • Completed Quantity

    The quantity completed for this operation.

  • Reported Quantity

    The quantity reported Quality.

  • Quantity to Report

  • Start Hours

    The calculated run duration of the operation in hours.

  • Completed Hours

    An estimated time in hours spent on this operation determined from the run duration and quantity completed.

  • Reported Hours

    The time for this operation in hours reported to Quality.

  • Hours

    The quantity of hours to report.

  • Service Resource

    Operation information from the database.

  • Drawing No

    Operation information from the database.

  • Drawing Rev No.

    Operation information from the database.
