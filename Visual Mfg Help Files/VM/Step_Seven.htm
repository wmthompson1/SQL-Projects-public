Step Seven

Analyze Project Summary Window Information

Use the tools available to you in the Project Summary and Project Maintenance window to assess transactions posted to projects.

Project Summary Estimate at Completion/Estimate to Complete Comparison

The Project Summary EAC/ETC Comparison reports the percent change between the current version EAC and a summarized ETC. Use this report to determine which formal EACs need to be updated. You can also use this report to identify any unusual costs that have been reported to the actual costs and updated to the ETC.

To run the report:

  1. From the Project menu, select Project Summary. The Project Summary window appears.

  2. From the File menu, select Print EAC/POC Report. The Print EAC/POC Report window appears.

  3. To view all projects, leave the Project ID fields blank.

  4. In the Filter box, leave the Project Type field blank to view all project types in the report.

  5. In the Summarize up to field, enter the last day of the period you are closing.

  6. Click Ok. The EAC/POC report appears.

If the actual amounts do not match the formal amounts, you may want to update your EAC for the project.

Print the report to use as a guide.

Project Budget vs. Actual Report

The Project Budget vs. Actual report compares the budgeted costs to the current actual costs.

To run the report:

  1. From the Project Summary window, select File, Print Budget Info. The Print Budget Info window appears.

  2. To view all project budgets, leave the project ID fields blank. To view a range of projects, enter a project ID in the Starting Project ID field and Ending Project ID field. All projects that fall between those two IDs (inclusive) will appear in the report.

  3. Click Ok. The report outputs.

Other Analysis Tools

You can use the Show Source for Actuals option under the Info menu in the Project Summary window to view the details behind the actual amounts.
