Linking Documents to Trace IDs

Trace documents are documents you can link to Trace IDs. You can print a Trace IDs linked document at any time after you have assigned the Trace ID.

To link documents to Trace IDs,

  1. In the Traceability for Part window, select the Trace ID to which you want to link a document.

The row appears highlighted.

  1. Click the Documents button.

The Document Reference window appears.

  1. Click the Insert button.

The first available row appears selected.

  1. Double-click the <Document ID> column header and select the document you want to link to this Trace ID from the List Documents dialog box that appears.

  2. When you have finished linking documents to the Trace ID, click the OK button.

The List Documents dialog box closes and the name of the document you selected appears in the Document ID column.

  1. Click the Save button.

  2. If you want to link documents to any other Trace IDs, link them in the same manner.

  3. When you have finished linking documents to Trace IDs, click the Close button.
