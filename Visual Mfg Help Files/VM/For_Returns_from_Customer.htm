For Returns from Customer

  1. Enter Return Information in Shipping Entry.

When returning parts, first enter the Qty Returned, and check the Warehouse ID, and Location ID for each line.

  1. Enter Part Traceability.

For each line containing a traceable part, select the line and click the Part Traceability button to call up the Traceability dialog box.

Total Quantity shows the Qty Returned entered on the line. You must now specify which lot numbers have been returned. Your original identification, in the form of a trace label or other marking, should be available.

  1. Set the Return Quantities.

The table displays one line for each lot number of the part that has an unreceived quantity.

The system automatically fills in the Quantity fields so as to distribute the total Qty Returned across the lot numbers as needed.

It is possible that these are not the actual lot numbers being returned. You can adjust the quantities by zeroing out the quantities for those lot numbers that aren’t being returned, and filling in the correct quantities of those lot numbers that are being returned. You are accessing lot numbers from the trace labels that were attached earlier, or from other paperwork containing the lot numbers that were shipped.

You can click the Insert button to create a new lot number specifically for a returned quantity. However, lot numbers are usually assigned before the material is shipped.

You can keep track of how much of the total Qty Returned you have accounted for by watching the Numbered Quantity field at the upper right corner of the screen. This field is the total of all entries in the Quantity column. You usually want this quantity to equal the Total Quantity.

You are allowed to have Numbered Quantity be greater than or less than Total Quantity. When you press Close, you are warned, and given the option to cancel or accept the override. If you choose to override, Qty Returned for the line is automatically adjusted to the Numbered Quantity.

  1. Enter comments, if necessary.

You can enter any comments about the return in the Comments field. Note that these are in addition to, or replace, any original comments that were made. You can modify the Expiration Date at this point, if needed.

  1. Click the Close button to exit the Traceability dialog.

  2. Repeat this procedure for each line, the click the Save button to complete the return transaction.

If necessary, you can re-enter Part Traceability as many times as needed before completing the transaction.
