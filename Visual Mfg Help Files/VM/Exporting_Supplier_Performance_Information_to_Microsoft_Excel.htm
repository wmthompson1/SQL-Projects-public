Exporting Supplier Performance Information to Microsoft Excel

You can export the contents of the Supplier Performance table to Microsoft Excel.  To export to Excel:

  1. Select Purchasing, Supplier Performance.

  2. Click Refresh to generate the graph and populate the table.

  3. Select File, Send to Microsoft Excel. The data displayed in the table is added to a spreadsheet. In addition, this information is sent:

  4. Unit Price

  5. Standard Unit Cost

  6. Order Date

  7. Promise Date

  8. Promise Ship Date
