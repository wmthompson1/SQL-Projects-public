Material Planning Window - Allocations Table

The line item table contains the following columns.

Demand Type - The type of demand source to which you have allocated supply. For example, CO stands for Customer Order.

Demand Order ID - The ID of the order to which you have allocated supply

Demand Sequence Number - The line number of the order. For example, if you allocated supply to one order with seven line items, there would be seven lines for this order, each with a numeral in this column so you can differentiate one from another.

Demand Number - If the demand link is to a work order material operation demand, the demand sequence number of the operation demand appears here.

Supply Type - The type of supply you allocated to the order line. For example, I stands for inventory; PO for purchase order.

Supply Order ID - The ID of the supplying order.

Supply Sequence Number - The line number of the supplying order. If you allocated from only one line, only the number one appears.

Supply Number - The number of the supply link.

Warehouse ID - The Warehouse ID of both the supplying and demanding order. Both must be the same in order for VISUAL to allow allocations to or from order lines.

Allocated Quantity - The quantity allocated to the order line from the supplying order.

Received Quantity - The quantity of the order line you have received.

Issued Quantity - The quantity of the order line you have issued.

Supply Mode - The mode of supply.

User ID - The ID of the user who entered the order and committed the allocations.
