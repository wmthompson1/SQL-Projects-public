Viewing Document Revision Histories

  1. To review a revision history for a document, select Revision History from the Info menu.

The Document Revision History (Document ID) dialog box appears.

VISUAL displays one revision per line.

The table contains the following fields:

Stage ID – If you specified a stage for the revision, the stage code appears in this field. Stages apply to revision numbering. Click here for more on stages in numbering profiles.

Revision ID – The Revision ID you specified for this revision.

ECN - If an ECN applies to this revision, its ID appears in this field.

User ID – The ID of the user signed into the database when the revision was created.

Revision Date – The date on which the revision occurred.

Comments – Comment text regarding the revision. To view comment text, double-click the Comments field.

The Revision Comments dialog box appears. This is a read-only dialog box. After viewing comments, click the Close button.

  1. To open ECN Entry, click the ECN Entry button.

  2. To print a history report, select the appropriate print output from the print list and click the Print button.
