Step Nine

Complete Project Billing and Progress Billing Manufacturing Invoicing

Process all project billing for milestones and progress billing.

To process project billing:

  1. From the Project menu, select Project Billing Entry. The Project Billing Entry window appears.

  2. In the Order ID field, select a customer order ID to invoice. Details of the order appear in the appropriate

  3. fields.

  4. Select the row for which you would like to create billing events.

  5. From the Edit menu, select Billing Events. The Edit Billing Events window appears.

  6. Review the information in the line items. If a line item is eligible to be invoiced, a check mark appears in the Generate Invoice field.

  7. Click Save to generate the invoice.

Repeat the process for each project progress invoice.

For commercial transactions:

  1. From the Contracts menu, select Progress Billing Entry. The Progress Billing Entry window appears.

  2. Select an Order ID in the Order ID field. VISUAL populates information for the order in the window.

  3. Select the row for which you would like to create billing events.

  4. From the Edit menu, select Edit Billing Events or Edit Milestones, depending on the type of progress billing you have set up (this example uses billing events). The window appears.

  5. If a billing event has been triggered through the costing utility, a check mark appears in the Triggered field.

  6. Click the Close button to return to the Progress Billing Entry window.

  7. To generate an invoice, select Print Invoices from the File menu. The Print Invoices window appears.

  8. In the Date Setting field, select the date that will appear on the invoice. Your choices are current, event date, or a date you specify.

  9. Select the report’s format in the Format box. Depending on your system, you may have more than one choice in the Format box.

  10. Select the appropriate print options.

Print Only - Do Not Save Invoice # - Select this option if you do not need to use the invoices VISUAL generated.

You may not need to use them if you are using pre-numbered forms.

Use Pre-numbered Forms - Select this option if you are using forms with preprinted invoice numbers. If you select this option, enter the starting form number in the Starting Form Number field.

Use Current Exchange Rates - Select this option to have VISUAL use the latest exchange rates from Currency Exchange Rate Maintenance. See Currency Exchange Rate Maintenance for more information on creating exchange rates.

  1. Click Ok to print the invoice.
