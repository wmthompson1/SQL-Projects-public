Setting Exchange Rates for Individual Purchase Orders

Use Purchase Order Entry to set rates for individual orders. When you enter a purchase order for a vendor that uses a foreign currency, Purchase Order Entry automatically obtains the current exchange rate from Currency Exchange Rate Maintenance. You can, however, fix the exchange rates for individual orders manually. This allows you to set the exchange rates on a per order basis.

  1. With the appropriate order open in the Purchase Order Entry window, select Order Exchange Rates from the Edit menu.

The menu item is only enabled when the current order has a non-system currency ID.

The Edit Exchange Rates for Current Order dialog box appears.

The dialog box displays the customer’s currency and the current sell and buy rates for the customer’s currency.

  1. Enter the new buy and sell rates in the New Sell Rate and New Buy Rate fields.

You can click the Get Current Rates button to call the current rates into the New Sell Rate and New Buy Rate fields.

  1. Click the Ok button to save your changes.

You return to the Purchase Order Entry window.
