Selecting Forecasts Advanced Material Planning

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site to use. If you are licensed to use a single site, this field is unavailable.

  2. Select Master Production Schedule from the File menu.

The Master Production Schedule dialog box appears, listing all of the forecasts for the part.

  1. Select the forecast you want to use, then click the Copy to MPS button.

VISUAL populates the Current MPS row based on the forecast.

  1. Click the Save button.

The next time you run MRP, the current MPS should equal the MRP.

If the Order Policy for this Part ID, as set in Part Maintenance, is not Master Schedule, VISUAL warns you using this function. You can set up a forecast and master production schedule for any part, but MRP is only use it to generate planned orders if the parts order policy is Master Schedule.
