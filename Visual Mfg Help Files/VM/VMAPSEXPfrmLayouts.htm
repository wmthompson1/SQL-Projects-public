About the Up Data Extraction Layouts Window

Use the Data Extraction Layouts window to specify the VISUAL database fields to include in export files. This window is used with Advanced Planning and Scheduling functions.

Starting the Data Extraction Layouts Window

In the VISUAL executables directory, double-click VMAPSEXP.EXE.

Select Maintain, Data Extraction Layouts.

Tasks

This table shows the tasks that you can complete in the Data Extraction Layouts window:

Task

Description

Setting Up Data Extraction Layouts

Use this procedure to specify the database columns to use in data extraction layouts.

Fields

Type of data

Select PLN to set up the material planning layout. Select SCH to set up the scheduling layout.

File Style

Click Fixed or Delimited.

Record

This field is available if you clicked Delimited. Specify how to indicate a new record in the file. Select Newline if you want each record to begin on a new line. Select Null (Fixed Length) if you want your data to appear all on one line.

Field

Specify how to separate fields.

Header tables tab

Specify the tables that contain header-level information.

Line items tables tab

Specify the tables that contain line-level information.
