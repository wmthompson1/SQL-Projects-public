Specifying HTS Duty Percentages by Country of Origin

  1. Complete one of these steps:

  2. Select Admin, Application Global Maintenance. Select Maintain, Harmonization Codes.

  3. Select Admin, Accounting Entity Maintenance. Click the Order Mgt tab. Click HTS Codes.

  1. Highlight the line for which you want to add duty percent by country information and click the Duty by Country button.

  2. Click Insert.

  3. Specify this information:

Country of Origin - The country from which the shipment originates.

Duty % - The duty percent for that country.

  1. Click Save.
