Using the Order Management Window Cross-Selling Parts Window

Cross-selling parts are parts that you sell as accompaniments to the primary part of a customer order. These parts may be akin in function or have such similar design specifications that selling one with the other is a logical or perhaps customer-mandated practice. Cross-selling parts are not, however, replacements for the primary part.

From the Tools menu of the Order Management Window, select Cross-Selling Parts.

The Cross-Selling Parts window appears.

The following fields in the header section summarize the current customer order:

Part ID - The primary part of the customer order.

Warehouse ID - The ID of the warehouse for the customer order line.

ShipTo ID - The customer’s default ShipTo ID. Enter a default ShipTo ID in Customer Maintenance under the General tab.

The following columns appear in the right pane of the window:

Cross-selling Part ID - The ID of the cross-selling part.

Part Description - A description of the cross-selling part.

Unit Price - The price per unit of the cross-selling part.

Unallocated Demand - The available quantity of the part that you have not yet allocated to demand.
