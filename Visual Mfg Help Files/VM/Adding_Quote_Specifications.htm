Adding Quote Specifications

Quote specifications are specific to an individual quote and there can only be one specification per quote. If you wish to add several notes to a specification, use Quote Notes.

  1. With the appropriate quote open in the Estimating Window, click the Quote Specs button.

The Quote Spec dialog box appears.

  1. Enter the information for the quotation.

You can enter an unlimited amount of information.

Use the Paste button to paste text that you copied from another location into the dialog box.

Use the Copy button to copy text in this dialog box to the clipboard.

  1. Click Close to close the specification dialog box and return to the Estimating Window.

Note that you can still access the fields in the Estimating Window while using the Spec dialog box.

If you are pasting information into the dialog box the entire contents of the dialog is replaced with the incoming text.
