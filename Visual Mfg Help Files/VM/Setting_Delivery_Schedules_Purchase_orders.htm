Setting Delivery Schedules

You can create and edit delivery schedules using the Edit Delivery Schedule dialog box.

  1. Select the purchase order line.

  2. Click the Del Sched button.

  3. The Edit Delivery Schedule dialog box appears.

  4. Click the Insert button.

  5. Enter the part quantity in the Quantity column.

  6. Enter the appropriate ship date in the Desired Recv Date column.

  7. Enter the release number in the Release Number column.

  8. Enter the Ship To ID in the Ship To ID column.

  9. Enter the appropriate information in the user defined columns.

  10. Click Ok to save any changes to the Edit Delivery Schedules dialog box.

Deleting Delivery Schedules

To delete a delivery schedule, select the delivery schedule you want to delete and click the Delete button. Click Ok to save any changes to the delivery schedules dialog box.
