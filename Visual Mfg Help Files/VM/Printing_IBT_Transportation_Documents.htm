Printing IBT Transportation Documents

Transportation Document: A status report that details the location and value of inventory and why you are shipping it.

  1. From the File menu, select Print Transportation Document. The Print Transportation Document dialog box appears.

The current IBT’s Packlist ID appears in the Packlist ID field.

  1. From the Ship Reason list box, select the reason for this shipment. Only the ship reasons applying to Interbranch Transfers appear in this list box. Enter IBT Ship Reasons in Site Maintenance.

In Site Maintenance, under the Shipment Tracking tab, you must also specify if the selection of Ship Reason Codes are required in the printing of transportation documents. If you specified that they are optional, you do not have to select a Ship Reason Code at this time. If you specified that they are required, you must select a Ship Reason code before VISUAL will print the document.

  1. From the Carrier ID list box, select a Carrier ID or click the Carrier ID button to select one from the line item table. Enter carriers in Application Global Maintenance.

  2. Select Freight Terms: Prepaid, Billed, or Collect.

  3. Select an output for the document:

Print - Select this to view a print preferences dialog box after clicking Ok.

View - Select this to view the report on your screen after clicking Ok.

File - Select this to save the report to a file after clicking Ok.

E-mail - Select this option to send the report in a Rich Text Format (.RTF) through electronic mail. Upon saving, a Send - Print dialog box appears that you can use to specify to whom you want to send the QRP report information.

  1. Click Ok.
