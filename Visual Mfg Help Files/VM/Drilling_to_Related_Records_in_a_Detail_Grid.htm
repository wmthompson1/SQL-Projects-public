Drilling to Related Records in a Detail Grid

If a piece of data in the detail grid is underlined when you place your cursor on it, you can click the data to open up the related maintenance window. For example, if you are viewing the Bookings vs. Shipment Detail grid, you can click a customer ID to open the customer’s record in Customer Maintenance.

To drill to related data in a detail grid:

  1. Select File, Dashboard.

  2. In the dashboard, select File, Open and select the dashboard that contains the detail grid.

  3. Click the ID to view the record in its maintenance window.
