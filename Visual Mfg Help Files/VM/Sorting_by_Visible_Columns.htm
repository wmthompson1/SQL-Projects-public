Sorting by Visible Columns

You can click any visible column to initiate a sort of the data in the list by that column. You can sort the data by each column in either ascending or descending alphabetical order.

To sort by column:

  1. Click the column header for the column by which you want to sort.

  2. Select Refresh Ascending or Refresh Descending from the Options menu to re-sort the table.
