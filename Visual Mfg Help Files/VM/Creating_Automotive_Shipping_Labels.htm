Creating Automotive Shipping Labels

VISUAL’s shipping label program allows you to pre-print labels to attach to containers prior to the manufacture of parts. You can print labels for the Customer, Shipto ID, and Part ID. You can also sequentially pre-assigns serial lot numbers (lot Trace IDs) to print on the labels.

When printing labels, VISUAL prints them in a set of one master label and x container labels. Each master label coincides with one skid. All containers on the same skid must have the same serial number. You can specify the number of copies of the same master label and the number of copies of the same container label you will need for each container on the skid.
