Setting Purchase Order Entry Preferences

To set the Purchase order Entry child window’s preferences:

  1. Select the Preferences option from the Options menu.

The Preferences dialog box appears.

  1. Click the Default Tab arrow and select the tab you want to appear when you first open the Purchase Order Entry window.

  2. In the Default Status section, select what status you want when you enter a new purchase order.

If you select Release, only users with the appropriate permissions are allowed to save purchase orders with Released status.

  1. If you want to use VISUAL’s auto dispatch services, select the Auto Dispatch Services check box.

With this preference set, VISUAL automatically dispatches the entire service purchase order quantity. Upon saving, VISUAL dispatches the line’s entire Order Quantity.

  1. Click the Ok button.
