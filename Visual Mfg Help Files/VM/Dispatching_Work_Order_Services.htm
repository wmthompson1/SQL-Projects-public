Dispatching Work Order Services

You can use the Service Planning window to dispatch parts from a work order to outside service vendors.

  1. Select a line in the Service Planning window.

  2. Select Dispatch W/O Service from the Edit menu.

The Dispatch W/O Service dialog box appears.

  1. Enter the part amount you want to dispatch in the Dispatch Quantity field. 

  2. Click the Dispatch button.

The Service Dispatch window appears.

A new line appears in the Service Dispatch window for your particular dispatch for the work order.

  1. Click the Disp All button to dispatch the specified part amount.

  2. Click the Save to save any changes to the Service Dispatch window.
