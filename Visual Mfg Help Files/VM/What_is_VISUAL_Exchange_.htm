Using the Accounting Menu

If you have applied an Infor VISUAL Global Financials license to your VISUAL database, the Accounting Menu becomes available. Use the items available on the Accounting Menu to maintain information such WIP costs and financial calendars.

Current Infor VISUAL users who do not use Infor VISUAL Financials, can use these tools to establish a baseline of data for importation into VFIN.

These features apply to the Global Financials .NET product. VISUAL Exchange, a tool facilitating communication between your VISUAL database and a Global Financials database, is necessary to migrate this data to Global Financials. Before proceeding, verify that VISUAL Exchange is installed and configured properly.

These applications are available in the Accounting Menu:

  • Post Manufacturing Journals

  • Progress Billing Entry

  • Accounting Window

  • Costing Utilities

  • Close/Reopen Maintenance

  • WIP Maintenance

  • Financials Application Global Maintenance
