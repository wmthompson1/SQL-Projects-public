Entering Counts and Recounts

After parts have been counted and quantities entered on the tags, you are ready to enter the resulting counts.

To begin this process:

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site where you conducted the count. If you are licensed to use a single site, this field is unavailable.

  2. In the table, select the physical inventory count you are entering.

  3. Select Enter Counts from the File menu.

The Enter Counts dialog box appears.

  1. Change the Count Date, if necessary.

The Count Date defaults to the current date.

  1. Select the appropriate entry options.

Enter Counts allows you to enter information for a large number of tags. You can select two data entry options:

Automatic Save When Filled In - If you select this option, you can save an entered count by pressing the Enter key rather than tabbing to the Save button, or using the mouse.

Automatic Advance to Next Tag - If you select this option, VISUAL automatically enters the next tag number after you save the current one.

  1. Enter a Tag Number.

If the tag is not a blank tag, the Part ID, Location ID, and part description automatically appear. You may want to verify this information against the printed tag, but it is not editable.

If the tag is a blank tag, you must enter the Part ID and Location ID.

  1. Enter the Quantity counted for the tag.

  2. If you specified user dimensions when you initialized the count, the appropriate dimensions are inserted after you specify a quantity. You can override these dimensions.

  3. Save the tag.

If you selected Automatic Save When Filled In, press Enter to complete the tag. Otherwise, click Save.

If Automatic Advance to Next Tag is selected, the next tag number is automatically called up. Otherwise, enter it manually and continue.

Tabbing past the Clear button clears the tag number and prepares the dialog for the next entry. This is useful when you are not using the automatic save feature.

Entering Recounts

You cannot modify tag quantities after you enter them. If you want to re-enter some quantities, or you are performing an actual recount, use the recount function by selecting Enter Recounts from the File menu.

The dialog box that appears looks and behaves exactly like the Enter Counts dialog box, with only the following exceptions:

  • You can only use it for a tag after you have entered an initial count using Enter Counts.

  • You can re-enter a quantity for a tag as many times as necessary.
