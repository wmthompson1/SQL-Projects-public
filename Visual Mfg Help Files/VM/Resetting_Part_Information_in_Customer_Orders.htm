Resetting Part Information in Customer Orders

  1. Either with a Part ID displaying in the Part ID field of the Part Maintenance window or no part currently displaying, select Reset Part Information from the Maintain menu.

The Reset Part Info dialog box appears.

See Resetting Part Information for more information on setting general part reset options.

  1. Select the Customer Orders tab.

  2. Select the According to Customer Order Status box to select which customer order statuses you want the part information update to affect: Firmed, Released, Closed, Exclude Closed Lines Cancelled, or All. Skip this step if you want the update to be date-driven only. See next step

You can select any combination of these options; selecting All makes the previous five options unavailable.

  1. Select the According to Customer Order Date box to activate the date fields. Enter dates in the On or After or On or Before fields. Skip this step if you want the update to be status-driven only. See previous step.

  2. Click the Ok button to begin the part information update process.
