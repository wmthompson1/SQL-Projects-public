Finding Capacity-constrained Resources

Typically, manufacturing lines only have one or two resources that are true capacity-constrained resources (CCRs). DBR’s planned load tools assist you in determining which of your resources are your CCRs. CCRs are used in a DBR environment only. An Easy Lean environment does not have CCRs.

To find and set up the CCR, use these procedures:

  • Setting Up Shop Resources for Monitoring

  • Identifying the CCR

  • Analyzing Planned Load Information

  • Setting Up Buffers for the CCR
