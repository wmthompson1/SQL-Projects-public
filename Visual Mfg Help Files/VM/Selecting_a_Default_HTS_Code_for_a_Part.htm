Selecting a Default HTS Code for a Part

You can enter HTS code information per part in Part Maintenance. However, you must establish HTS Codes and Countries of Origin in Application Global Maintenance under the Order Management tab first.

  1. With the appropriate Part ID displaying in the Part Maintenance window, select the Order Management tab and then click the HTS Code button.

The Harmonized Tariff Schedules dialog box appears.

The HTS Codes that you established in Application Global Maintenance appear in the line item table.

  1. Either double-click a line or select a line and click the Ok button.

The HTS Code appears in the HTS Code field.
