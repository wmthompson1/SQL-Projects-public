Establishing Preferred Vendors

You can establish price and preference information for each vendor that supplies a part. Although you can purchase any part from any vendor defined in Vendor Maintenance, use Vendors Supplying this Part to set up explicit preferred and alternate vendors for each purchased part. Preferred vendors are the suppliers that your company has researched and found to be best in quality, capability to deliver the part, price, or a combination of all three.

If you are licensed to use multiple sites, you must add preferred vendors at the site level.

Before you can assign a preferred vendor, part information for the new part must exist and the unit of measure that the vendor sells in must exist in the Units table. It is required that the vendor is present in the Vendor table.

  1. Select Inventory, Part Maintenance.

  2. If you are licensed to use multiple sites, click the Site ID arrow and select the Site ID for which you are adding vendors. If you are licensed to use a single site, this field is unavailable.

  3. Click the Part ID browse button and select the Part ID. You must select a purchased part to activate access to the Vendors Supplying Part dialog box.

  4. Select Maintain, Vendors Supplying This Part. The dialog box shows the vendors that are currently assigned to the part.

  5. Click Insert.

  6. Double-click the Vendor ID browse button and select a vendor from the list.

  7. To set this vendor as the default vendor for this part, click Set Preferred Vendor. You can only select one preferred vendor for each part.

A check mark appears in the Pref column of the selected vendor. This automatically updates the preferred vendor field in defaults and applies to all entries in the browse table with that Vendor ID.

  1. Click Save.

  2. If you have finished adding vendors for a part, click Close.
