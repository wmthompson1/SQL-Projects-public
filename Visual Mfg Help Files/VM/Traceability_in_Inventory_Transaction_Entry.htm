Traceability in Inventory Transaction Entry

In general, all inventory transactions for traceable parts require traceability information to be entered. This is controlled by settings in the When to Apply section of the part’s traceability profile.

General Procedure for Using Traceability in Inventory Transaction Entry:

These steps are identical for all types of inventory transactions. See the following sections for specifics of each transaction.

  1. Enter transaction data in Inventory Transaction Entry.

  2. Click the Part Traceability button to enter Part Traceability information.

The Part Traceability button becomes active only when you enter the Part ID for a traceable part.

The Traceability dialog box appears.

Total Quantity displays the Quantity that you entered for the transaction. This is the quantity that you must assign or attribute to one or more lots, serial numbers, or other trace IDs.

The Warehouse ID and Location ID specified in the transaction appear at the top of the dialog box. You can change it here, if necessary.

  1. Create and/or Enter Lot Numbers.

Depending upon the transaction, different lot numbers and quantities appear; you may have to enter a new lot number, or set the quantities for the ones that are there. In all cases, VISUAL distributes the transaction Quantity across the existing lots automatically.

  1. Set the quantities.

You are not required to use the default lot numbers or quantities. You can always add, modify, and delete lot number quantities.

  1. Enter Properties and Comments for New Lots.

When you create a new lot number, you must enter all properties at that point; afterwards, you can only change them using Part Trace Maintenance. If a property is marked as Required in the traceability profile for the part, you cannot complete the transaction until you enter a value for the property.

You may always enter or edit the Comments or Expiration Date, as necessary.

  1. Click Close to exit the Traceability dialog box.

If Numbered Quantity does not equal Total Quantity, you can abort the close, or override the quantity. If you override, the receipt Quantity for the transaction resets to the Numbered Quantity.

All the fields for the transaction are uneditable after you exit the traceability dialog. This is because you have set traceability information for this set of data. If you want to cancel or change the transaction, you must click the Clear button and begin again. If necessary, you can re-enter Part Traceability as many times as needed before completing the transaction.

  1. Click the Save button to complete the transaction.
