Importing Workflows

Use the Import menu to open a workflow file that has been previously exported.

To import a workflow:

  1. If necessary, open VISUAL Workflow.

  2. On the File menu, select Import.

  3. In the Warning message, do one of the following:

  • Click Yes to save the open workflow document before importing a new document.

  • Click No to import a workflow document without saving the open document.

  • Click Cancel to return to the open workflow document without importing a document.

  1. In the Create New Workflow dialog box, enter information in the following fields:

Type - Select Predefined.

Application Area - Select the application area of the workflow you want to import. The application area you select must be the same as the application area of the file you want to import.

Template Name - Type the name you want to use for this template.

Usage - Select All Documents if you want VISUAL to apply this template each time you create a workflow or Single Document if you want VISUAL to use this template for a specific document.

Document ID - If you selected All Documents in the Usage list, go to the next step. If you selected Single Document in the Usage list, type the exact document ID (for example, enter the order ID if you want to create a workflow for a specific purchase order); then click the OK button.

The Associated DB Table field is "view only" and lists the database associated with the VISUAL application that you selected in the Application Area field.

Status - Select Active to make the template available or Inactive to not make the template available at this time.

  1. Click Ok.

  2. In the Select an Import File window, select the file you want to import; then click the Open button.

The imported workflow appears in the work area.
