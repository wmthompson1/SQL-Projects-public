Changing Your Password

Valid VISUAL users can change their passwords at any time.

  1. While signed into VISUAL under your user name, select Change Password from the Security menu.

  2. Enter your old password in the password field.

This assures VISUAL that you have the right to change it. The system replaces the characters you type with asterisks.

  1. Enter the new password you want to use in the New Password field.

The system replaces the characters you type with asterisks.

  1. To verify the new password, enter the password again in the Repeat Password field.

  2. Click OK to commit the change, or Cancel to abort it.
