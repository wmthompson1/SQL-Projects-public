Viewing Analytic Data

When you display an analytic, you can view the underlying data used to create the analytic.

  1. Select File, Dashboard.

  2. In the dashboard, select File, Open and select the dashboard to use.

  3. To view a single data point, perform one of these steps:

  4. To view data for a single bar or pie wedge, place your cursor over the bar or wedge. For bars, the value is displayed. For wedges, the percentage represented by the wedge is displayed.

  5. In line graphs, a circle is displayed to indicate the data points. Place your cursor over the circle to view the value.

  1. To view the underlying data for the entire graph, right-click in the graph and select Detail Data.
