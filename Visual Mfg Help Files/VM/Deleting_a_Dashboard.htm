Deleting a Dashboard

Use caution when deleting a dashboard. When you delete a dashboard, the dashboard is permanently deleted. You cannot restore it. If you delete a shared dashboard, the dashboard is deleted for all users.

To delete a dashboard:

  1. Select File, Dashboard.

  2. In the dashboard, select File, Open.

  3. Select the dashboard to delete.

  4. Click Ok.

  5. Click Delete. A message is displayed warning you that the active dashboard will be deleted.

  6. To delete the dashboard, click Yes.
