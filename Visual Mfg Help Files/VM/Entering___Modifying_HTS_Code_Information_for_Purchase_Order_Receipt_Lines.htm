Entering & Modifying HTS Code Information

During the receipt of a purchase order, you have the option of entering an HTS Code per line for a part that does not have a default HTS Code or modifying the part’s default HTS Code that you set in Part Maintenance.

  1. Open the purchase order that you want to receive.

  2. Scroll down the Purchase Receipts line item table until you find the HTS Code column header.

  3. Double-click <HTS Code> to select an HTS code from the table.

VISUAL either replaces the existing HTS Code or sets this as the purchase receipt line’s default HTS Code.

  1. Click the Save button to commit these new or modified settings.
