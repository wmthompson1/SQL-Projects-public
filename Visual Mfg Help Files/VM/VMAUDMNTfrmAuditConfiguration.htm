About the Audit Configuration Window

Use the Audit Configuration Window to specify the information to track.

Starting the Audit Configuration Window

  1. Select Admin, Audit Maintenance.

  2. Select File, Configure Auditing.

Tasks

This table shows the tasks that you can complete in the Audit Configuration window:

Task

Description

Working with the Audit Configuration Toolbar

Use this task to learn about the Audit Configuration toolbar.

Configuring Audit Files

Use this task to specify the database columns to track.

Fields

Table List

This field lists the tables that you can audit.

Columns table

After you select a table in the Table list, this information about the columns in the table is displayed:

Tracked

Select this check box to track changes to the column

Table

The name of the table

Column

The name of the column

Primary Key

Indicates whether the column is a primary key of the table.

Type Definition

The type of data that is stored in the column.

Insert

Select this check box to track when users add data to the column.

Update

Select this check box to track when users edit data in the column.

Delete

Select this check box to track when users delete data in the column.

Audit Report Times

Select this check box to track the duration of reports. The start time and end time of any report run is recorded. If you select this check box, all reports are tracked. You cannot choose which reports to track.
