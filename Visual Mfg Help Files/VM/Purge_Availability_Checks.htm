Purge Availability Checks

Use the Purge Availability Checks option to remove any non-released and non-firmed work orders and all engineering masters and quote masters that have been used to determine availability against a particular schedule.

The system automatically purges these availability checks when you run the Global Scheduler. This option allows you to delete availability checks without running the Global Scheduler. This frees up the shop resources for new availability checks or new work orders that you want to schedule.

You can delete availability checks as they are done, however, you may elect to keep them instead. This is generally the case when you are checking the availability on several competing projects at once and need the capacity allocated to each one to be there to take capacity from the other related projects. This option is here to remove these simultaneous checks so that the additional checks may be made without this competing load on the system.

  1. Highlight the line in the Global Scheduler that you want to purge.

  2. Select Purge Availability Checks from the File menu.

The Purge Availability Checks dialog box appears. The Schedule ID for the schedule you selected appears in the dialog box field.

  1. Click Ok to start the purge process.

Click Cancel to return to the Global Scheduler window without processing the schedule.
