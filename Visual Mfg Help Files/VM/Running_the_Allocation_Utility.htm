Running the Allocation Utility

  1. From the File menu, select Start.

As the Utility carries out your allocation instructions, a dialog box displays the progress of the allocation session. As the blue bar increases in length, the percent complete also increases.

  1. Click the Abort button at any time to stop the allocation session.

If the allocation session was successful, a message appears.

  1. Click the Ok button to return to the Allocation utility window.
