Printing Commissions Reports

This report prints commissions information for all Firm, Released, and Closed customer orders in a specified date range. Depending on the options that you select in the report dialog, the Sales Rep ID, Commission Percent, Order Amount, and resulting Commission amount are displayed. If commission sharing has been set up, the ID of the rep that shares commissions is also displayed.

  1. Select Sales, Sales Commission Report.

  2. Click Print Commissions Report.

  3. If you are licensed to use multiple sites, click the Site ID(s) arrow and select the sites to include in the report. If you are licensed to use a single site, this field is unavailable.

  4. Specify the date range for the report:

  • To specify a date range, specify dates in the Starting Date and Ending Date field.

  • To print month to date and year to date summary information, select the Print Month to Date/Year to Date Summary check box. When you select this check box, the Starting Date and Ending Date fields become inactive. The report is run for January 1 of the current year to the current date. You can sort the report by Customer ID or Customer Name.

  1. In the Sequence section, specify how to group the information in the report. After you click a sequence option, the label for the Starting ID and Ending ID fields change. Click one of these options: Order ID, Customer ID, Customer Name, Part ID, Product Code, Commodity Code, Sales Rep, and Territory.

  2. To restrict the information displayed in the report, specify codes or IDs in the Starting and Ending IDs/Codes fields. The report includes the codes or IDs you specify and any code or ID that alphabetically or numerically between the two fields.

  3. Click the Report Currency arrow to select a currency from the list. If you are printing a report for a single site, the entity currency associated with the site is inserted. If you are printing a report for multiple sites, the drop-down list shows the tracking currencies shared by the selected sites. If you are printing the report for multiple sites, the sites you select must share at lease one tracking currency. If no shared tracking currencies are found, “Not Available” is inserted. You cannot run a the report if a shared tracking currency is not found. Clear site selections until a shared tracking currency ID is displayed in the drop-down list.

  4. From the output list, select the output format for the report. If you select email, select the PDF Format check box to attach a PDF to the email instead of an RTF file. See Report Output Options.

  5. Click Ok.
