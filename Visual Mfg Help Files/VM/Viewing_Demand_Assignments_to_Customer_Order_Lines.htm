Viewing Demand Assignments to Customer Order Lines

Using the Customer Order Entry window you can assign supply from six sources - coproducts, inventory, purchase order delivery schedules, purchase orders, interbranch transfers, and work orders - to the current customer order line. You can also, however, assign supply to a customer order line item from other applications. For example, you can assign coproduct supply to a customer order line item using the Coproduct dialog box. The difference in this case being that you are assigning to the demand source from the supply source, rather than assigning from the supply source to the demand source.

With the appropriate customer order line item highlighted, select Assign Supply to Customer Order Line from the Edit menu.

The Supply Links dialog box appears.

If you have assigned any supply links to the current customer order line using another application the links appear in the Supply Links dialog box line item table. Regard this information the same way that you do a line of supply that you have assigned to the current customer order line using one of the preceding six procedures.
