Data Redaction

CAUTION: If you redact a data subject's data, the data cannot be recovered.

When you redact a data subject’s data with the Individual Privacy program, text strings are replaced by Xs and dates are set to null. In Contact Maintenance, Gender and Marital Status are set to N.

If your search for a data subject yielded multiple results, you must redact all of the results to remove the data subject’s information from the database. For example, if a data subject’s name is found in the First Name or Last Name fields in Contact Maintenance and on a customer order, two lines are displayed for the data subject in the Individual Privacy table. To redact the data subject’s data from both documents, you must select both lines in the table before selecting Redact Personal Information from the File menu.
