Printing Transportation Documents - Service Dispatch Entry

You must enable Shipment Tracking in Site Maintenance before you can print transportation documents.

A Transportation Document is a status report that details the location and value of inventory and why you are shipping it.

  1. From the File menu, select Print Transportation Document.

The current Order ID appears in the Order ID field. Service Dispatch line items appear in the line item table.

The table contains the following columns:

Line # - The line number of the service dispatch.

Package Type - The type of package in which you dispatched the service.

Number of Cartons - The number of cartons you used to dispatch the service.

Weight - The weight of the dispatch line item.

  1. From the Ship Reason list box, select a reason for the dispatch.

  2. From the Carrier ID list box, select the appropriate carrier.

You can also click Carrier ID to select from a list of carriers.

  1. From the Freight Terms section, select the appropriate freight payment option.

  2. Click Ok.

The document appears for you to view before printing.
