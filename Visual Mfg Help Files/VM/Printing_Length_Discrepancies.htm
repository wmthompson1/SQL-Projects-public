Printing Length Discrepancies

For those of you who have used, or are using, VMDI in earlier versions of VISUAL, the print length discrepancies function allows you to print a report of the fields that have changed beginning with version 6.2x. Rather than VMDI automatically changing the lengths, you can choose to keep the older field lengths. This may help you keep some of your older EDI maps functionally accurate.

  1. From the Options menu, select Print Length Discrepancies or click the Print Length Discrepancies button on the main toolbar.

  2. Select an output for the length discrepancies report. See Report Output Options.

  3. Click the Print button.

  4. Click the Apply to DB button to apply field changes to the layout.
