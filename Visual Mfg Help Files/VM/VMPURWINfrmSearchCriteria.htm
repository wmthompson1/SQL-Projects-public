Using the Search Criteria Tool

The Search Criteria tool allows the selection of criteria that determines the parts and vendors that will be examined in the views—use the Search Criteria tool to select what you want to appear in your child windows.

The Search Criteria tool has a basic set of selections. If you do not make any selections, or do not save any settings as default, and open a tool or view, VISUAL has default search settings with which it populates what you have opened.

Using the Search Criteria tool:

  1. Select the Search Criteria option, from the Tools menu of the Purchase Management window.

The Search Criteria tool opens.

  1. In the Sequence section, select what you want VISUAL to use to limit the range of your search.

You can select:

  • Part ID

  • Product Code

  • Commodity Code

  • Planner User ID

  • Buyer User ID

The range fields change depending on your sequence selection.

If you want to limit the range of information appearing in your child windows, enter the starting and ending criteria you want to use.

If you want all of your parts to appear, leave both fields blank.

  1. On the Standard tab, select the warehouses you want to appear in your child windows.

If you have a large number of parts and warehouses, VISUAL will take a long time to populate the Material Planning child window. The object of using the Search Criteria tool is to focus on a set criteria for the current demand.

  1. If you are working with consigned inventory, click the Consigned Inv tab and select the Consigned Warehouses you want to appear.

Select the type of warehouses and locations you want to appear. You can select:

  1. Customer Warehouses

  2. Customer Locations

  3. Vendor warehouses

  4. Vendor Locations

  1. Select your material resource planning requirements for this criteria by clicking the MRP arrow and selecting the option you want. You can select:

Y – If you want only parts that are included in a material resource plan, select the Y option.

N – If you want only parts that are not included in a material resource plan, select the N option.

Nothing – If you want all parts regardless of their material resource planning status to appear, select nothing.

  1. Select the following criteria:

Parts With Shortages Only – If you want only parts with shortages to appear in your child windows, select the Parts With Shortages Only check box.

Fabricated/Purchased – Select which parts you want to appear:

Fabricated Parts Only

Purchased Parts Only

Fab/Pur Parts Only

Either, but not both

At Or Below Order Point – If you want VISUAL to include parts with quantities that are at or below their set order points , select the At Or Below Order Point check box.

Above Order Up To Qty – If you want VISUAL to include parts with quantities that are above their order points, select the Above Order Up To Qty check box.

Include Obsolete Parts – If you want VISUAL to include your obsolete parts, select the Include Obsolete Parts check box.

The following settings apply only to the Material Planning child window:

Net Available Quantity – If you want VISUAL to include only your available stock quantities, select the Net Available Quantity check box.

Deduct Safety Stock – If you want VISUAL to subtract a part’s safety stock value from the total quantity, select the Deduct Safety Stock check box.

Include On Hold Orders – If you want VISUAL to include orders with an On Hold status, select the Include On Hold Orders check box. This setting applies only to the Material Planning child window.

Include Planned Orders – If you want VISUAL to include all of your future planned orders, select the Include Planned Orders check box.

Include Unreleased Orders – If you want VISUAL to include Unreleased orders, select the Include Unreleased Orders check box.

  1. In the Part Selection section make the following selections:

All Parts – If you want all parts to appear, select the All Parts option.

Parts With Material Requirements Dated Between – If you want parts that have demand requirements with a specified time period, select the Parts With Material Requirements Dated Between option and select the Starting and Ending dates you want to use.

Parts With Customer Orders Dated Between – If you want parts that have customer order demand requirements with a specified time period, select the Parts With Customer Orders Dated Between option and select the Starting and Ending dates you want to use.

  1. In the Netting Type section, select the netting type you want to appear in the Material Planning child window. You can select:

  2. Net By Required Date

  3. Net Using MRP Rules

  1. Click in the Days field and enter the number of days you want VISUAL to look ahead for demand. VISUAL normally nets for an infinite horizon—VISUAL includes All demand regardless of how far ahead it finds demand. Using the Planning Horizon option, you can restrict the range to manage the quantity of records appearing. Leaving this field blank indicates no horizon.  This setting applies only to the Material Planning child window.

  2. If you want to limit the range of supplying vendors appearing in your Vendor History and Vendor Comparison child windows, click the appropriate browse buttons and select the starting and ending Vendor IDs you want to use.

  3. If you want to apply your selections but keep the Search Criteria tool open, click the Apply button.

  4. If you want to apply your selections and close the Search Criteria tool, click the Ok button.

Saving Search Criteria Settings

To save your settings so that the Search Criteria tool opens with those settings already selected (by default), make your selections and click the Save as Default button.
