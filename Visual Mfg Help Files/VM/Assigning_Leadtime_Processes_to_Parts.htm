Assigning Leadtime Processes to Parts

To assign a leadtime process to a part:

  1. Select Inventory, Part Maintenance.

  2. In the Site ID field, select the site where the leadtime process is used.

  3. In the Part ID field, specify the part.

  4. In the Process Type field, select the leadtime process to use for the part.

  5. Click the Save button.
