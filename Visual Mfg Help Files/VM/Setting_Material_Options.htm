Setting Material Options

Using the options on the Material tab you can select how you want VISUAL to handle the material for scheduling.

To set material scheduling options:

  1. Click the Material tab.

  2. Select how you want VISUAL to check material:

Part ID – If you want VISUAL to strictly look for the supply of the Part ID regardless of the warehouse in which it resides, select the Part ID option.

Associated warehouse, then set of universal warehouses – If you want VISUAL to look in the associated warehouse before checking your universally planned warehouses, select this option.

  1. If you want VISUAL to use your inventory supply before creating orders to fill that demand, select the Use Inventory Supply First check box.

  2. If you have finished making changes to your scheduling preferences, click the OK button.
