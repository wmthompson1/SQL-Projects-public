Using Provisional Rate Billing Inquiry

To run provisional rate billing inquiries:

  1. From the Project menu of the VISUAL main window, select the Provisional Rate Billing Inquiry option.

  2. Click the Project ID button and select the project on which you want to run a Provisional Rate Billing Inquiry.

  3. Select the By CLIN check box to view billing information by Contract Line Item Number.

  4. Select the By WBS check box to view billing information by WBS code.

  5. In the Enter Date Range section, select the start and End date ranges for the inquiry you are running.

  6. Click the Rate ID arrow and select the rate you want VISUAL to use for the inquiry.

  7. Click the Run button on the toolbar or select Run from the File menu.

VISUAL calculates the Provisional Rate Billing Inquiry for the project, rate, and date range you entered.

To access nested rate analysis information, double-click a category. A secondary dialog box appears with individual lines for each account within the selected category.
