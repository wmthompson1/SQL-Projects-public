Printing the Project Summary Chart

After you generate a project summary chart, you can print the chart to a printer.

To print the chart:

  1. Select Project, Project Summary Window.

  2. Select View, Project Summary Chart.

  3. Specify the chart parameters. See Viewing the Project Summary Chart.

  4. Click Print. Only the chart is printed. The values used to construct the chart are not printed, even if you are viewing them in the Project Summary Chart dialog.
