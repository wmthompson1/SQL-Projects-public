Adding Your Own Help Information

All topics within Infor VISUAL Online Help feature a User-defined Help button (located at the bottom of the window) that allows you to either create or open one of five associated .txt files to which you can add your own comments, instructions, and general text to the Help file. Proper use of these files eliminates the need for annotations.

DO NOT change the name or location of the user-defined files. If you change a file name, the association with the help file is lost.

To access a User-defined file:

  1. Select a topic within the Help file.

  2. If there is a user-defined text file associated with the topic, a button labeled User-defined Help appears at the bottom of the topic.

  3. Click the User-defined Help button.

A secondary window appears, displaying the appropriate reference information in non-scrolling region of the window (the teal section) along with a series of 5 buttons, each with a unique label. These labels are the names of the user-defined files you can create by clicking the button.

If the file already exists—that is, if you have already accessed this user-defined file and saved it—the file opens.

If the file does not exist, you are prompted to either create the new file or cancel the operation.

  1. Enter the appropriate information in the text file.

  2. To save the file, select File, Save. Save the information to the current file.
