Entering Order Management Information for Customer Shipping Addresses

If you are licensed to use multiple sites, you must select a site on the Order Mgt tab is the Customer Maintenance window before accessing the Alternate Addresses dialog box to specify this information. If you do not select a site ID, the Order Mgt tab is not displayed in the Alternate Addresses dialog box.

  1. Select Sales/Contracts, Customer Maintenance.

  2. Click the Customer ID browse and select the customer for whom you are setting up shipping addresses.

  3. Click the Order Mgt tab.

  4. In the Site ID field, select the site that ships products to the customer.

  5. Select Maintain, Shipping Addresses.

  6. Click the Order Management tab.

  7. In the table, select the address.

  8. Click the Primary Warehouse button to select a primary default warehouse for this ShipTo ID. This becomes the default shipping warehouse for this ShipTo ID.

  9. Click the Priority Code button to select a priority code.

VISUAL fills the Fill Rate field with the fill rate associated with the priority code.

  1. From the Fill Rate Type list, select a Fill Rate type.

L (Line Level) - When unavailable inventory for a part on an order line item reaches the fill rate threshold, VISUAL cancels the line item.

W (Whole Order) - When unavailable inventory for a part on an order line item reaches the fill rate threshold, VISUAL cancels the whole order.

  1. Click the Alternate Warehouses button to select alternate warehouses for this address. If the primary warehouse is unavailable, these secondary warehouses are used. This browse button is available only if you selected the Allocations Require Warehouse/Location option in Accounting Entity Maintenance.

  2. Select the Consolidate Orders box if you want to allow order consolidation.

  3. Select the Compliance Labeling box if you require that this customer’s ShipTo address have compliance labeling.

  4. Click the Save button to save the information.
