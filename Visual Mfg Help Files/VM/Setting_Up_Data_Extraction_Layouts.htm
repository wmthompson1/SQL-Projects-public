Setting Up Data Extraction Layouts

To set up Data Extraction layouts:

  1. Click the Data of Type arrow and select the data you want to include in this layout.

You can select PLN or SCH

  1. Select the file style.

You can select Fixed or Delimited.

If you select Delimited:

  • Click the Record arrow and select how you want each record to appear.

  • Select Newline if you want each record to begin on a new line.

  • Select Null (Fixed Length) if you want your data to appear all on one line.

  1. Click the Field arrow and select how you want VISUAL to separate your data fields. You can select:

  2. Asterisk

  3. Comma

  4. Tab

  5. Null (Fixed length)

  1. On the Header Tables tab, click the arrow and select the table containing data you want to include.

  2. Click the Add button.

The table name appears in the list. To remove the name from the list, click the Remove button.

  1. On the Line Item Tables tab, lick the arrow and select the table containing data you want to include.

  2. Click the Add button.

The table name appears in the list. To remove the name from the list, click the Remove button.

  1. When you have finished adding information to your layout, click the Save button.
