Generating Purchase Orders from Vendor RFQs

For information about generating internal buy/sell orders from vendor RFQs, see Generating Internal Purchase Orders from Vendor RFQs.

Use this feature to generate a purchase orders from a vendor RFQ, which is a document generated by a vendor in response to a request for a quote.

You can use this feature without a purchase order record in the window.

If you are licensed to use multiple sites, you can generate a purchase order from a requisition in the same site only. You cannot use a requisition created in one site to generate a purchase order in a second site.

To generate orders from requisitions:

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site to use. If you are licensed to use a single site, this field is unavailable.

  2. From the Edit menu, select Generate Order from RFQ....

The Generate Purchase Order from RFQ dialog box appears.

  1. Click the Request ID button to select the vendor RFQ you want to use to generate the purchase order.

If the RFQ contains any lines eligible to become purchase order lines, they appear in the line item table.

Before a RFQ line can be used to create a purchase order, it must have a quantity and at least a default vendor unit price.

VISUAL supplies the appropriate vendor in the Vendor ID field.

  1. If appropriate, enter an ID for the requisition into the New Order ID field. If you use auto-numbering, leave this field blank for VISUAL to assign the order the next available ID.

  2. Specify an Order Date, a Desired Receive Date, and an Original Promise Date for the order. The current system date appears by default in each date field.

  3. If appropriate, click the Warehouse ID button to choose a warehouse for the order.

  4. Select the Copy RFQ Specs to Purchase Order check box to copy RFQ specifications to the purchase order.

Select the Copy RFQ Header Information check box to copy RFQ header information to the purchase order.

  1. From the table, select which lines you want to add to the purchase order. Click the Select All button to select all lines; click the Deselect All button to clear all lines. To select multiple, noncontiguous lines, hold down the CTRL key while clicking the appropriate lines. 

  2. Click the Ok button to create the purchase order.

VISUAL processes the lines and returns you to the Purchase Order Entry window, displaying the new order.

  1. Complete the order following standard purchase order entry procedures.
