Using Drag n’ Drop in the Order Management Window

In the Order Management window, you can manipulate customer orders using typical drag-and-drop procedures.

For example, if you want to include a cross-selling part in a customer order, and you have already entered and saved an order line for the primary part, follow this simple procedure to drag and drop a cross-selling part to the current customer order.

If you are licensed to use multiple sites, make sure that the part you drag to a line exists in the site ID on the customer order header. If the part does not exist in the site, you are not allowed to replace the part on the line.

  1. With the order for which you are currently entering information open, open the Part Information child window and double-click the Part Info icon, then either double-click the cross selling parts icon or click the expansion box to view the cross-selling parts for the primary order part.

  2. Locate the cross-selling part you want to include in the customer order and select the line.

  3. With the appropriate cross-selling part selected, click and hold down the left mouse button until a box appears on the line.

  4. Continuing to hold down the left mouse button, drag the cross-selling part into the Customer Order Entry window. This is "dragging."

  5. Drag the box anywhere onto the Customer Order Entry window and release the left mouse button. This is "dropping."

VISUAL adds a new order line for this cross-selling part to the current customer order. The new part now has the focus in the Part Availability child window.

  1. Maximize the Customer Order Entry window and scroll down if necessary to verify that VISUAL has added a new line to the order. Select information for this line as you would any other customer order line.

See Customer Order Entry for more information on entering customer order header and line item information.
