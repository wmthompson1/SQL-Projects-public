Viewing Expected Throughput Inquiry Graphs Tabularly

You can view the information from an Expected throughput Inquiry graph in a table format by clicking the button next to the graph styles list box.

Upon clicking the button, a table appears in place of the graph. The selected Schedule IDs appear in the left hand column and the time ranges (daily, weekly, or monthly) for which you requested utilization information each represent one column in the table. The dollar amount of expected throughput appears for the selected time period as well.
