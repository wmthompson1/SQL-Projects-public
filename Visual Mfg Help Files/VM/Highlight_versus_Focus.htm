Highlight versus Focus

You can highlight a work order to differentiate it from other work orders in the schedule. Highlighting a work order then allows you to view the details of that work order. Highlighted work orders appear in the Scheduling Window as yellow bars.

When you click an operation, it is given a dotted-line outline. In Windows terminology, this operation has the focus. Many Scheduling Window functions act upon operations that have the focus. In fact, the Highlight Work Order option highlights the work order that has the focus. An operation that has the focus is often referred to as the focus operation.
