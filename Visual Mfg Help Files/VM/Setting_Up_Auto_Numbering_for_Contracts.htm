Setting Up Auto Numbering for Contracts

Use Auto Numbering to generate contract numbers for the purchase contracts you set up in Vendor RFQ Entry.

For more information on setting up purchase contracts, refer to Entering a New Request For Quote, Entering Quotes From Vendors, and Viewing Purchase Contracts.

  1. Select Edit, Purchase Contract Auto Numbering.

  2. If you are licensed to use multiple sites, click the Site ID arrow and select the site for which you are setting up auto-numbering. If you are licensed to use a single site, this field is unavailable.

  3. Fill in the next sequential number that you want used for the numeric portion of the Contract ID. This number may be up to 7 digits long.

  4. Assign any alphanumeric prefix or suffix. These are case sensitive; you should use capitals for all letters.

  5. Enter the total number of digits in the numeric portion of the ID in the Number of Decimal Digits field.

VISUAL uses this number to generate the proper number of leading zeros, if you select that option.

  1. If you want leading zeros to be included in a number, check the Show Leading Zeroes check box.

When you select this option, all IDs have the same number of characters. If you clear this option, the system omits the place-holder zeroes. For example, the system would change ID CTR-0004 to CTR-4.

  1. Click the Save button to save the changes, or Cancel to close the dialog box without saving the changes.
