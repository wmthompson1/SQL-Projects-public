Editing an Existing Quote

You can edit all fields and line items of a quotation until to print the final quote. After you print the quote, VISUAL automatically locks the quote.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site that contains the quote. If you are licensed to use a single site, this field is unavailable.

  2. Double-click the Quote ID button and select the quote.

  3. Edit any field or line item as necessary.

In particular, you may wish to periodically update the following fields:

Won/Lost - Enter this date when the order is placed, or lost. VISUAL automatically updates this field if you use Customer Order Entry to automatically generate an order.

Followup - Update this date to the next date you wish to follow up this quote for some reason.

Probability - Update this percentage as your knowledge of the sales situation changes.

Expect - Update this date as your knowledge of the sales situation changes.

Depending on the probability of an order, the quote master may be enhanced on an ongoing basis, as it is prepared to be converted to a work order.

  1. Click the Save button to save the changes.
