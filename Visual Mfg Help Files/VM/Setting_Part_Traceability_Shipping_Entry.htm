Setting Part Traceability Shipping Entry

If a part in the shipment requires traceability information, the Trace check box on the line is selected. To specify traceability information:

  1. Select Sales, Shipping Entry.

  2. In the Order ID field, select the order that you are shipping.

  3. Select the line containing the traced part and select Edit, Part Traceability.

  4. In the dialog, the trace IDs for the part are displayed. Specify the quantity to ship.

  5. Click Close.

  6. Continue to complete the transaction or transaction line items.
