Specifying Reporting Information for Shipment of Goods Reports

To specify which "statistics" will be required for shipment of goods reporting purposes:

  1. Click the Dispatch Format button

The Intrastat Report Format dialog box appears.

  1. Select the appropriate check boxes in the Show section. You can select: Tariff Code, Excise Price, Region, Country, Port of arrival, Port of transshipment and Siret Number.

If you check an item, VISUAL does not generate the reports until all Intrastat line items have the checked item.

  1. In the Export ID field, enter a valid Export ID to generate reports to file.

  2. Click the Ok button.

The Intrastat Report Format dialog box closes.

  1. In the Frequency section, select how often you want to generate reports. You can select from the following radio buttons: Monthly, Quarterly or Annually radio button.

  2. Click the Save button to save the information.
