Using the Calendar Button

The Calendar button is an easy way to enter dates into the system. Whenever a date is required in VISUAL, you find the Calendar icon next to the date field.

To use the calendar:

  1. Click the Calendar button.

You can also right-click the calendar button to view a menu containing various time-saving options.

  1. A calendar for the current month appears.

  2. Select a date by double clicking on it.

  3. To scroll ahead to future months, click the right arrow.

To scroll back to previous months, click the left arrow.

When you select a date, the calendar closes. The date you have chosen appears in the date field.
