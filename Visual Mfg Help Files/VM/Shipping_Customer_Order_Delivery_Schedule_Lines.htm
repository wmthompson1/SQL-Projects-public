Shipping Customer Order Delivery Schedule Lines

You can ship customer orders using the Delivery Schedule Shipment dialog box. You must use the Delivery Schedule Shipment dialog box with customer order lines that contain delivery schedules.

  1. Select the customer order line from the Shipping Entry window line item table.

  2. Click the Del Sched button.

The Delivery Schedule Shipment dialog box appears.

  1. Select the delivery schedule line you want to ship.

  2. Enter the portion of the Quantity Ordered that you want to ship in the Ship Quantity column. By default, VISUAL fills this column with the quantity of the order; edit this field as necessary.

  3. Click Save to ship the delivery schedule line.
