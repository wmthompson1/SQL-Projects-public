Sending Infor VISUAL/Financials QRP Files in a Rich Text Format

  1. From the Print destination list box of the current print dialog box, select E-mail.

The Send Mail - Print window appears.

  1. Click the To button to select the E-mail address of the person to whom you are sending the report information.

  2. If appropriate, click the CC button to select the E-mail address of the person whom you want to receive a carbon copy (CC) of the report.

  3. If appropriate, enter a subject heading in the Subject field.

  4. Check the Preview before sending box if you want to view the file/message before VISUAL delivers it.

  5. Enter any necessary message or text in the below field.

  6. Click the Mail button to send the file.

If you checked Preview before Sending, the file appears in a standard Windows mail format as the recipient will view it.
