Viewing Customer Order Information from the Tools Menu

The seven child windows available from the VISUAL Order Management Customer Order Entry Tools menu allow you to view part and order information for each customer order line.

For a customer order, you can view:

  • Part Availability

  • Part Alternate Warehouses

  • Cross-Selling Parts

  • Substitute Parts

  • Customer Order History

  • Current Customer Order

  • WIP/VAS

Besides being separate functions that you can selectively use while entering customer order information, these seven child windows also comprise the Order Management Window.

See the later section on the Order Management window for detailed information on how to use these child windows.
