Starting Workflow

To start VISUAL’s Workflow Designer:

  1. If necessary, start VISUAL.

On the Admin menu of VISUAL’s main window, click Workflow Designer.

  1. The Workflow Designer window opens with the Open Existing Workflow dialog box.

  2. Do one of the following:

  • To open an existing Workflow template, refer to Opening an Existing Workflow Template.

  • To create a new Workflow template, refer to Creating a New Workflow Template.
