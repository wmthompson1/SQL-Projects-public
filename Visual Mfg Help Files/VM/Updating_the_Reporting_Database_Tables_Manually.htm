Updating the Reporting Database Tables Manually

To update the reporting database tables manually:

  1. Select Admin, Application Global Maintenance.

  2. Select Maintain, Visual Reporting Data.

  3. In the Update Type section, click Manual.

  4. In the Start Date and End Date fields, specify the date range for the information to load. Leave both fields blank to load all existing data. To load all data after a certain date, specify a date in the Start Date field only.

  5. In the table, select the data types to load. All data types except for VISUAL menu are used in the Dashboard metrics. You can use SHIFT+click and CTRL+click to select multiple rows.

  6. Click Save.

  7. Click Load.
