Creating an Estimate for a Piece Tracked Part

To create an estimate for a piece tracked part:

  1. Select Sales, Estimating Window.

  2. If you are licensed to use multiple sites, click the Site ID down arrow and select the site to use. If you are licensed to use a single site, this field is unavailable.

  3. Specify information in the estimate header as you would for a standard part.

  4. To add a line item, click Insert.

  5. Specify the piece tracked part you are quoting.

  6. Click in the Quantities Quoted field. In the Quote Quantities dialog box, specify the pieces you are quoting only. Leave the Quantity column blank; the quantities are calculated based on the conversion factor you set up for the part. Click Ok.

  7. Specify the required dimensions for the part in the Length, Width, and Height fields.

  8. Complete the quote line as you would for a standard part.

  9. Click Save.
