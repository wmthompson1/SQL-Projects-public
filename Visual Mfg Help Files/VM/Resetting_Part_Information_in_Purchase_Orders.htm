Resetting Part Information in Purchase Orders

Because purchase order lines do not have drawing numbers or revisions, VISUAL only updates specs associated with a purchase order line.

  1. Either with a Part ID displaying in the Part ID field of the Part Maintenance window or no part currently displaying, select Reset Part Information from the Maintain menu. The Reset Part Info dialog box appears.

See Resetting Part Information for more information on setting general part reset options.

  1. Select the Purchase Orders tab.

Select the According to Customer Order Status box to select which customer order statuses you want the part information update to affect: Firmed, Released, Closed, Exclude Closed Lines Cancelled, or All. Skip this step if you want the update to be date-driven only. See next step.

  1. You can select any combination of these options; selecting All makes the previous five options unavailable.

  2. Select the According to Purchase Order Date box to activate the date fields. Enter dates in the On or After or On or Before fields. Skip this step if you want the update to be status-driven only. See previous step.

  3. Click the Ok button to begin the part information update process.
