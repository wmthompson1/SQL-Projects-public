Setting Event Detection

You can set what VISUAL does with the detected events.

  1. Select Preferences from the Options menu.

The Message Maintenance dialog box appears.

  1. Select the appropriate Event Detection radio button:

Process detected events - VISUAL performs the selected action when the event is triggered.

Place detected events on hold - VISUAL holds the selected events for you to process at a later time.

Disable event processing - VISUAL temporarily disables event processing for all events.

  1. Set the frequency at which VISUAL checks for detected events by entering a value in the Minutes field.

  2. Select the following option check boxes:

Combine Messages if More Than - Combines your messages. Enter a number in the field to indicate the number of messages to include.

VISUAL waits until it reaches the specified number of messages before performing the specified action.

Eliminate Duplicates Created Within - Removes duplicate messages if VISUAL creates them within a specified amount of time. Enter the number of minutes in the field.

This is helpful if you make several changes to a document and save after each change. If you select this option, Message Maintenance sends only one message for the changes, instead of one message for each change.

Limit Processing Time Slice to - Message Maintenance processes as many events as possible within the time limits you set in the field. The remaining events remain in the queue until the next time VISUAL checks for detected events.

  1. Click the Ok button to save your preferences.
