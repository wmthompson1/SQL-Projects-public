Deleting Reporting Service Print Job Configurations

To delete a reporting service print job configuration:

  1. Select Admin, Application Global Maintenance.

  2. Select Maintain, Report Service Configurations.

  3. Select the configuration to delete.

  4. Click Delete. An X is displayed in the row header, indicating that the row will be deleted.

  5. To complete the deletion, click Save. To cancel the deletion and retain the configuration in your database, click Delete again. The X is removed from the row header.
