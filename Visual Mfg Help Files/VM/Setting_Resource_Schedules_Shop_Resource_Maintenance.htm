Setting Resource Schedules Shop Resource Maintenance

The Resource Calendar allows you to define the shifts and hours that a resource is available. In addition to setting up shift hours, you can set up exceptions to the available hours. Establishing exceptions to the standard work shift allows the system to make automatic adjustments for schedule exceptions. The Global Scheduler uses this information. Entries in the Resource Calendar override the settings in the Global Calendar in Application Global Maintenance.

  1. With the appropriate resource in the Shop Resource Maintenance window, select Resource Weekly Calendar from the Edit menu.

The Weekly Calendar dialog box appears.

If you had not set up the weekly calendar for this resource to affect all schedules, the calendar is blank.

The Resource ID for the selected resource appears in the Resource ID field.

The schedule ID field is a drop down menu of available Schedule IDs. By default **All** appears in this field. The **All** indicates that all schedules are to use this calendar. You can also set up a calendar to be specific to a schedule ID by selecting the schedule to which the calendar applies.

  1. Click in the 1st Shift Start field next to the day you want to set. Enter the time that the 1st shift starts.

By default, the system assumes that the 1st shift starts in the morning (AM). Therefore, for speedy entry of time, enter the number. If the shift starts at 7:00:00 AM, you only need to enter 7. VISUAL completes the rest of the entry for you automatically.

  1. Enter the number of hours that your company can use the resource for each of the shifts.

If you cannot use a resource on a day or during a shift, leave the field blank or put a zero (0) in it.

  1. Click the Save button to save the weekly scheduling information for the resource.

Click the Delete button to delete a weekly schedule.
