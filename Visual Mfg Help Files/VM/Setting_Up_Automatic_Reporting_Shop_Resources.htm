Setting Up Automatic Reporting Shop Resources

  1. Select Eng/Mfg, Shop Resource Maintenance.

  2. In the Site ID field, select the site that uses the resource.

  3. Double-click the Resource ID browse button and select the resource, or create a new resource.

  4. Click the Other tab.

  5. Select the Automatic Reporting check box.

  6. Click Save.
