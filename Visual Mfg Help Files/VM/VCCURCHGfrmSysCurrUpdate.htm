Updating an Entity Currency

The current entity currency appears in the read-only Current Entity Currency field.

  1. If you are licensed to use multiple entities, click the Entity ID arrow and select the entity whose entity currency you are changing. If you are licensed to use a single entity, this field is unavailable.

  2. From the New Entity Currency list box, select the currency to which you want to convert.

    Only tracking currencies are possible choices.

  1. Click the Start button or press the F7 key to start the currency update procedure.

    There is no way to reverse this process. It is imperative that you make a backup before you begin.

VISUAL performs the update.
