Adding Purchase Parts from a Work Order or Project

Use this feature to add purchase parts and services from a work order or project to the body of a purchase requisition.

This feature works with or without a requisition record in the window.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site making the purchase. If you are licensed to use a single site, this field is unavailable.

  2. To add lines to an existing requisition, click the Requisition ID browse button and select the ID. To create a new requisition, leave this field blank.

  3. Either with a requisition in the window or working from a blank window, select Add Purchase Parts from WO/Project from the Edit menu.

The Add Purchase Parts from WO/Project dialog box appears.

  1. To add a BOM from a work order, select the Work Order option button. To add a BOM from a project, select the Project option button.

  2. Click the Base ID button to select a work order or project.

Work orders do not have to be released to be eligible for selection. All valid work orders and projects appear.

The Lot ID/Eng ID and Split ID/Version is automatically supplied (if you chose a project).

Do not modify the Lot ID/Eng ID and Split ID/Version. If you enter another value, your work order/project may not be recognized.

  1. From the Include section, select which elements of the work order/project to add to the requisition.

Parts – Select the Parts option button to add only parts from the selected work order/project to the requisition.

Services – Select the Services option button to add only services from the selected work order/project to the requisition.

Both – Select the Both option button to add both parts and services from the selected work order/project to the requisition.

  1. Select the Ok button to add the materials/services to the requisition.

If you are creating a new requisition, click the Save button after the BOM elements are added to the body of the requisition.
