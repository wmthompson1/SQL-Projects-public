Printing Customer Service Impact Reports

To print customer service impact reports:

  1. From the Customer Service Impact graph dialog box File menu, or the Customer Service Impact Details dialog File menu, select Print.

  1. Select the print destination for the report.

Select Print to output the report to a printer.

Select File to output the report as a text file. You can then edit the report using a text editing application.

Select E-mail to send the report via e-mail. Select the PDF Format check box to send the report as a PDF. Clear the check box to send the report as an RTF file. When you generate the report the system attaches

Select View to output the report to your screen for viewing.

  1. Click Print.
