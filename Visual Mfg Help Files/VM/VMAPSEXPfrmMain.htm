About the APS Export Utility Window

If you are licensed to use APS functionality, use the APS Export Utility window to export files that can be imported into your advanced planning and scheduling system.

Starting the APS Export Utility Window

In the VISUAL executables directory, double-click VMAPSEXP.EXE.

Tasks

This table shows the tasks that you can complete in the APS Export Utility window:

Task

Description

Using the APS Export Utility

Use this procedure to export data to a file that can be imported into your advanced planning and scheduling system.

Fields

Site ID

The site from which you are exporting data.

Type of Data

The type of data you are exporting.

Outbound Rule

Specify whether to replace the existing export file or to append the data to the existing export file.

Skip masters with obsolete parts

Select the check box to exclude obsolete parts from the export.

Schedule IDs

Specify the schedules to export.

Output File Directory

Specify the location of the output file.
