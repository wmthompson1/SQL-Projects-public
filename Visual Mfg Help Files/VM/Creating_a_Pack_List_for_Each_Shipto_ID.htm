Creating a Pack List for Each Shipto ID

VISUAL allows you to create a different pack list for each Ship to ID. If a customer order has three different lines, each with a different Shipto ID, VISUAL creates three pack lists. The procedure depends on whether you print a pack lists for the order before shipping it. If you do print a pack list for the order before shipping it, VISUAL detects the unique Shipto IDs and creates the separate pack lists for them before shipment. If you do not print pack lists for the order before shipment, VISUAL, as it starts the shipping process, asks you to confirm the creation of a range of pack lists, one for each Shipto ID.
