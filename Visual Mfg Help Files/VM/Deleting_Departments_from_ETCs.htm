Deleting Detail Rows from ETCs

If you want to delete a row from an ETC:

  1. With the Maintain ETC window open and the appropriate project selected, click the row you want to delete.

  2. Click the Delete Row button.

An X appears in the row header to indicate you have marked the department for deletion.

Because VISUAL does not delete the row until you click the Save button, you can un-mark the row by clicking the Delete Row button again.

  1. Click the Save button.

  2. When you have finished deleting departments for this project ETC, click the Close button.
