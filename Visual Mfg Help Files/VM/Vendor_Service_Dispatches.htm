Vendor Service Dispatches

If you have a vendor populated in the PO Entry child window, the Vendor Service Dispatches window populates with that vendor.

If you are licensed to use multiple sites, you can view this information on a site-by-site basis only.

To view your vendor service dispatches:

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site to use. If you are licensed to use a single site, this field is unavailable.

  2. Select the Vendor Service Dispatches option from the Info menu.

  3. Click the Vendor ID browse button and select the vendor you want to appear in the Vendor Service Dispatches window.

  4. If you want to sort the information in the window, select the sort option you want to use from the Sort section. You can select:

  1. Service ID

  2. Purchase Order ID

  3. Dispatch ID

  4. Vendor Service ID

  5. Order Date

  6. Dispatch Date

  7. Service Part ID

  8. Promise Date

  9. Desired Receive Date

  1. In the Order Sequence section, select the sequence on which you want VISUAL to sort your information. You can select Ascending or Descending.

  2. If you want to exclude all of your fully received dispatches, select the Exclude Fully Received check box.

Modifying Desired Receive Dates

If you want to modify the PO Desired Receive Date for a line on an order from within the Vendor Service Dispatched window, change the date for the line and order you want and click the Save button. VISUAL saves the new date to the purchase order.
