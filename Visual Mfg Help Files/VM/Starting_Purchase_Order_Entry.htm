Starting Purchase Order Entry

Purchase Order Entry is available from the VISUAL Main menu.

To start Purchase Order Entry:

Select Purchase Order Entry from the Purchasing menu.

The Purchase Order Entry window appears with no information in it.

The Purchase Order Entry window contains both entry fields and action buttons.

The Order Entry window is divided into two major sections: Purchase Order Header information and the Purchase Order Line Item Table.

Purchase Order Header Information

This section contains basic information such as Order Date, Order ID, Vendor ID, shipping specifications, contact and status. You can also use the Calendar object to select Order Date and Desired Receive Date, and Original Promise Date.

Purchase Order Line Item Table

This table contains one line for each separate purchase on the purchase order. You can configure this table; you can show, hide, resize, rename, and move columns to suit your individual needs.
