Editing Information in Projects

Using the same dialog boxes and windows you used to enter the original information. You can edit header, leg, task, and material information. Because Project IDs, Leg IDs, and Sequence Numbers define various objects within your project, you cannot make any changes to them.

There are several methods for editing information within projects:

  • Select the object and from the Edit menu, select Edit

  • Select the object, and press CTRL-E keys

  • Double-click the Object

  • If the right-click menu is active, right-click the object and select the Edit option.

All of these methods open the appropriate dialog box in which you can make your edits. If you have several edits to make, VISUAL allows you to open several dialog boxes at the same time and click between them without first closing one and opening another. For example, if you are editing several tasks, you can open the Edit dialog box for one and click the next task when you want to edit it. You can also open all of the Edit dialog boxes (header, task, leg/material) at the same time and when you click a new object, VISUAL highlights the appropriate Edit dialog box.
