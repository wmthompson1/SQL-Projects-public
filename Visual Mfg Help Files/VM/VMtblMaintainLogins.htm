Using the Signed In Users Window

The Sign Ins window lists all currently signed in users. If you are system administrator, you can use the Sign Ins window to sign out any inactive VISUAL users.

Because machine crashes, power failure, and other events can cause users to terminate operation of the VISUAL application they are using but remain signed into the system, you may find it necessary for an administrator to use the Sign Ins window to log users off so that they may continue their work.

Starting the Signed In Users Window

To start the Signed In Users window, select Admin, Signed In Users. The table shows all currently signed in users. The table is sorted by User ID and then by Program for each user.

Signing Users Out

To sign out inactive users:

  1. Select the line for the user and click the Delete button.

  1. Click the Save button.

A dialog box is displayed requesting verification of the deletions.

  1. Click the Yes button.

Users are signed out when the next attempt to save data or run reports. They must sign in to VISUAL again to use the system.
