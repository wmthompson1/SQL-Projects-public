Using a Right Mouse Click to Append an Existing Order Line to a Current Order

You can append an existing order line to a current order by right-clicking on the appropriate order line in the Customer Order History window.

  1. In the VISUAL DDP Customer Order Entry window, open the order to which you want to append existing order line information.

  2. If not already open, the Order Management window; then open the Current Customer Order window and the Customer Order History window.

  3. In the Customer Order History window, double-click the icon.

All existing orders for that customer appear.

  1. Click the plus box next to the order that contains the order line you want to append to the current order.

  2. Highlight the order line and perform a right mouse click.

An option box appears. 

  1. Left-click Append Line.

VISUAL appends the existing order line information to the current customer order.

Maximize the VISUAL Order Management Customer Order Entry window to view the appended order information.

  1. Refresh the Current Customer Order window display to graphically view the current, appended order.

  2. Edit this new customer order information as necessary and then click the Save button.
