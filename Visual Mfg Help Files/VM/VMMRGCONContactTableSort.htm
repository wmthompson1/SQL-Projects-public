Sorting the contact table in the Contact Merge Utility

You can modify the contact table by filtering columns, moving columns and prioritizing a column sort based on contact information.

To sort the contact table:

  1. Select Options > Preferences.

  2. Select the contact information identifier on which to set the sort priority, and click a priority number in the sort selection area, or click Clear Selection to reset a sort priority. You can set sort priorities on up to eight contact identifiers.

  3. Select the check box in the Descending column to perform a descending sort from Z to A or from the largest number to the smallest number. Clear the check box to perform an ascending sort from A to Z or from the smallest number to the largest number.

  4. Click Ok to confirm your sort priorities.
