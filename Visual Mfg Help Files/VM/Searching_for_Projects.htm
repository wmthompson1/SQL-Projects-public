Searching for Projects

After you have been using Infor VISUAL for Project-Based Manufacturers for a while you will have built many projects. In order for you to easily find a particular project of interest, you can use VISUAL’s search function.

To search for a project:

  1. Open any project in the Project Window.

  2. From the File menu, select Search. A list of all of your projects appears. If this list is too large to fit in the dialog box, a scroll bar appears.

  3. Click the Search button on the toolbar.

The Search dialog box appears.

  1. Enter the appropriate search criteria.

  2. Click the Ok button.

If a project exists that meets your search criteria, that project appears in the Project window.

If a project does not exist and you want to enter a new search, click the New button on the toolbar.
