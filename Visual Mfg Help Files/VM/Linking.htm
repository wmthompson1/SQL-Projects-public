Linking

When you link an object to a Manufacturing Window item, an image of the source document appears in the Manufacturing Window item. A link is established between the source document and the Manufacturing Window item.

You can only link objects that you have saved to a file. For example, if you open AutoCAD and create a drawing, you must save it before you can establish a link to it.

When changes are made to a linked Source document these changes will appear in EVERY document that it is linked to. If you have a source drawing that needs to be modified differently in each place it is placed, it is better to Embed the picture.

  1. From the appropriate Manufacturing Window object dialog box, select the Picture/Object from the Edit menu.

The Set Picture/Object window appears.

  1. Start the application for the object you are embedding, select the drawing, and copy it to the Clipboard.

  2. Switch back to the Manufacturing Window.

  3. Select Paste Link from the Edit menu in the Set Picture/Object window.

The picture appears in the display area of the Set Picture/Object window.

  1. Click the Save and Exit button to accept the linked object and return to the Manufacturing Window.
