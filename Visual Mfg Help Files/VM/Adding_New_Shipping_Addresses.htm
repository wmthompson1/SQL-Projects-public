Adding New Shipping Addresses

To add a new ship to address for the customer:

  1. Select Shipping Address from the Edit menu.

The Shipping Addresses dialog box appears.

  1. Click the Insert button.

A new line appears in the table.

  1. If you want to copy address information from an existing address, select the existing line and click Copy, then select the new line and press Paste. Fill in or modify the address information in the fields at the bottom of the dialog. You cannot enter the fields directly into the table.

  2. If necessary, click the Intrastat tab and enter Intrastat information for the new Ship To Address.

  3. Click the Save button to save the new address.

  4. Click the Close button to return to the Customer Order Entry window.
