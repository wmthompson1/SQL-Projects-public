Changing the Status of the Order Header

If any lines in the order do not have a status of Inherit, you can decide how to update the status of the lines when you change the status of the header. You can update all lines to the new status, update only those lines that match the original status of the order header, or update only the header status.

If all lines in the order have a status of Inherit, then the Change Line Status dialog box is not displayed. All lines with a status of Inherit are considered to have the same status as the order header.

  1. Select Sales, Customer Order Entry.

  2. If you are licensed to use multiple sites, click the Site ID arrow and select the site that contains the order. If you are licensed to use a single site, this field is unavailable.

  3. Click the Our Order ID button and select the order.

  4. In the Status field in the header, select the new status.

  5. In the Change Line status dialog, complete one of these actions:

  6. To update all lines with the new status, click All Lines, and then click Ok.

  7. To update lines whose status matches the original status of the header, click Lines with Status of [Original Header Status], and then click Ok. [Original Header Status] is replaced with the actual status value.

  8. To update only the header status, click Cancel.
