Individual Privacy Report

The Individual Privacy report contains this information:

  • The data group where the data subject’s data was found

  • The primary key of the document that contains the data subject’s data

  • The data subject’s name

  • The data subject’s address

  • The data subject’s phone number

  • The data subject’s email

You can generate the report for more than one data subject. If the data subject’s information is used on multiple documents in your database, you can include multiple documents in the report.
