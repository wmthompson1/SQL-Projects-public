Using Automotive Features in the Order Management Window

In previous versions of Infor ERP VISUAL Enterprise, the Order Management Window did not support EDI transactions. Release 6.3 allows you to accomplish every EDI task you can accomplish using the standard Customer Order Entry window. For more information on using these features, consult the sections of this manual that focus on the Customer Order Entry window, and adapt it to the Order Management Window.
