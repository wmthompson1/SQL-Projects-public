Reports Used to Reconcile Balance Sheets

This report is used to reconcile project transactions. It corresponds to step eleven in the Closing Procedures. The Labor Ticket Report and Project Billing Report are also used to reconcile project transactions.

Cash G/L Accounts Report

To print the Cash G/L Accounts Report:

  1. From the Cash menu, select Cash Book. The Cash Book window appears.

  2. From the File menu, select Print G/L Accounts. The Print G/L Accounts window appears.

  3. In the Print Selection box, select By Accounting Period.

  4. In the End Date of Accounting Period, enter any date that falls within the period you would like to close.

VISUAL automatically populates the end date of the period.

  4. Click Ok. The report outputs.

A/R Aged Receivables

To print the A/R Aged Receivables report.

  1. From the Receivables menu, select Invoice Entry. The Invoice Entry window appears.

  2. From the File menu, select Print Aged Receivable. The Print Aged Receivables window appears.

  3. If you are licensed to use multiple sites, click the Site ID(s) arrow and select the sites to include in the report. If you are licensed to use a single site, this field is unavailable..

  4. In the Receivable Account field, select the account for which you would like to view information. Select All to view all accounts.

  5. In the As of Date field, enter the last day of the period you closed.

  6. In the Report Type box, select the level of detail for the report. Your choices are Summary, Detail, and Detail with payment history.

  7. In the Aging basis box, select Invoice Date to start the aging clock from the invoice date. Select Due Date to start the aging clock from the due date.

  8. In the Sort By box, select the sort order for the report. Select Currency ID to sort the report by currency ID. Select Customer ID to sort by Customer ID or Customer Name to sort by customer name.

  9. In the Invoices box, select All Invoices to view all invoices in the report. To view invoices of a particular currency only, select All Invoices of Currency, then click the arrow to select the currency.

  10. Select the exchange rate for the report. Select Current rate to use the current rate or Invoice rate to use the exchange rate as of the date of the invoice.

  11. In the Customers box, select All Customers to view all customer aging. To choose a specific customer, select Customer, then click the Customer ID button to select the customer.

  12. Click Ok. The report outputs.

A/P Aged Receivables

To print the A/P Aged Receivables reports:

  1. From the Payables menu, select Invoice Entry. The Invoice Entry window appears.

  2. From the File menu, select Print Aged Payables. The Print Aged Payables window appears.

  3. If you are licensed to use multiple sites, click the Site ID(s) arrow and select the sites to inlcude in the report. If you are licensed to use a single site, this field is unavailable. .

  4. In the Payable Account field, select the account for which you would like to view information. Select All to view all accounts.

  5. In the As of Date field, enter the last day of the period you closed.

  6. In the Report Type box, select the level of detail for the report. Your choices are Summary or Detail.

  7. In the Aging basis box, select Invoice Date to start the aging clock from the invoice date. Select Due Date to start the aging clock from the due date.

  8. In the Sort By box, select the sort order for the report. Select Currency ID to sort the report by currency ID first. Select Vendor ID to sort by vendor name or Customer ID to sort by customer name.

  9. In the Invoices box, select All Invoices to view all invoices in the report. To view invoices of a particular currency only, select All Invoices of Currency, then click the arrow to select the currency.

  10. Select the exchange rate for the report. Select Current rate to use the current rate or Invoice rate to use the exchange rate as of the date of the invoice.

  11. In the Vendors box, select All Vendors to view all customer aging. To choose a specific customer, select Vendor, then click the Vendor ID button to select the customer.

  12. Click Ok. The report outputs.

Post Manufacturing Journal – Inventory Balance Report

To print Inventory Balance reports:

  1. From the Ledger menu, select Post Manufacturing Journals. The Post Manufacturing Journals window appears.

  2. From the Reports menu, select Inventory Balances. The Print Inventory Balance Report window appears.

  3. Leave the Starting and Ending Part ID fields blank to view all parts.

  4. In the Sequence box, select G/L Account ID. This will make reconciling the report with the General Ledger easier.

  5. Click Ok. The Inventory Balance Report outputs.
