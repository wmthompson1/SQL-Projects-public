Viewing Current MPS for a Master-Scheduled Part

Click anywhere on the Current MPS line. The MPS Standard for Part dialog box appears. You cannot edit any of the values in these columns.

The line item table contains the following columns:

Order Quantity - The work order quantity.

Want Date - The date on which the customer wants the order.

Firmed - The date on which you firmed the order.
