About Contact Maintenance

Use Contact Maintenance to enter and maintain contacts. You can associate contacts with vendors and customers or create stand-alone contacts that have no links to customers or vendors.

Starting the Contact Maintenance Window

Select Sales, Contact Maintenance or Purchasing, Contact Maintenance.

When you access Contact Maintenance, active contacts are displayed in the table. To view inactive contacts instead, select View, Inactive Contacts.

Fields

Contact ID

The ID for the contact. To use auto-numbering, leave the field blank.

First/Last Name

The honorific, first name, and last name of the contact.

Middle Name/Initial

The middle name or initial of the contact.

Position

The title or position of the contact.

Salutation

The salutation by which to address the contact.

Web User ID

The ID that the contact uses to access your website. This may be the direct contact’s user ID, or a generic one for anyone at the customer or vendor location to use.

Web Password

The password associated with the Web User ID. Depending on your database engine, and whether you enforce strong passwords, this may or may not be case sensitive. Refer to your database software documentation for information about passwords.

Confirm Password

Reenter the web password.

Active

Select this check box to activate the contact. Clearing the check box maintains the contact information, but sets the contact status to inactive. See Designating Contacts as Active or Designating Contacts as Inactive for more information.

Phone/Extension

The telephone number, with area code and extension, for the contact. This may be a direct line to the contact, or a generic company line. This is not a required field. If the contact prefers not to be contacted by this phone number, select the Do Not Call check box.

Mobile Phone – If available, enter the telephone number, with area code, for the contact’s mobile (cellular) phone. You can enter up to 20 numeric characters in this field. It is not a required field. If the contact prefers not to be contacted by this phone number, select the Do Not Call check box.

Fax

The fax machine number of the contact.

Country Dial Code

The country dial code of the contact.

E-Mail

To specify the contact’s e-mail address, click the Edit URL button and specify the address. After you specify an address and exit the field, the field is inactive and a link is added to the address. Click the link to open an e-mail addressed to the contract.

If the contact prefers not to be contacted by e-mail, select the Do Not E-mail check box.

Social Network URL

Click the arrow and select a type of social media website, then click the Edit URL button to specify the address.

Address Tab

Name – The name you specified on the General tab is inserted. You cannot edit this field.

Street Address

The street address of the contact.

City, State, Zip

The city, state, and zipcode of the contact.

Country

The country in which the contact is located.

Profile Tab

Date of Birth

The contact’s date of birth.

Marital Status

The contact’s marital status.

Preferred method of contact

The method that the contact wants you to use to communicate.

Gender

The contact’s gender.

Customers Tab

The table on the Customers tab includes these columns:

Customer ID

The ID of the customer that the contact represents

Name

The name of the customer that the contact represents

Primary Contact

The check box is selected if the contact is the primary contact for the customer.

Vendors Tab

The table on the Vendors tab includes these columns:

Vendor ID

The ID of the vendor that the contact represents

Name

The name of the vendor that the contact represents

Primary Contact

The check box is selected if the contact is the primary contact for the vendor.

User Defined Tab

User defined field layout ID

The ID of the user-defined field set.

User Defined Fields

The user-defined information that is required by your company.

Specifications Tab

Use the Specifications tab to enter additional information regarding the contact.
