Adding Rejection Codes to the Database

Before creating any new purchase requisitions, add rejection codes to the database. Rejection codes help you classify rejected purchase requisitions task/ECN tasks by any quantifiable measure you choose to impose.

Purchase Requisition Entry and ECN Entry share a common Rejection Code table. This feature is available from both the ECN tab in Accounting Entity Maintenance.

  1. From the main VISUAL menu, select Accounting Entity Maintenance from the Admin menu.

  1. Click the Defaults tab.

Requisition controls appear in the Requisition Maintenance section.

  1. Click the Rejection Codes button.

The Maintain Task Rejection Codes dialog box appears.

  1. Click the Insert button to add a new rejection code.

  2. Enter information as follows:

Code – Enter an alphanumeric identifier for the rejection code. You are supplied with several predefined codes.

Description - Enter a description for the code.

  1. Repeat this procedure for each code you want to add.

  2. Click the Save button to commit the rejection codes to the database.

  3. Click the Close button to return to the main window.
