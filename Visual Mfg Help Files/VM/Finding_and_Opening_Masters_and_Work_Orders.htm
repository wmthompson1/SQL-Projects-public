Finding and Opening Masters and Work Orders

To edit a master, you must first call it into the Manufacturing Window. Use the Open and Search commands for this purpose.

  1. Select Open from the File menu.

The Manufacturing Window Search dialog box appears.

  1. Select the radio button for Engineering Master, Work Order, or Quote Master.

You must select the proper type — you cannot open an engineering master if you select the Work Order button. You can specify a default type in Preference Maintenance. See Specifying the Default Document Type in the Open Dialog.

  1. Select the size preferences and display mode here, so you don’t need to immediately access the View menu after opening.

Select the radio buttons for size preference and display mode. These are the same as the View menu functions.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site in which the master was created. If you are licensed to use a single site, this field is unavailable.

  2. Enter the Base ID, Lot ID or Engineering ID, and Split ID (work orders only).

At this point, you can also choose to Search, rather than enter the IDs. See the next section for information on searching.

You can also create a new master/work order by clicking on the New button.

  1. Click the Ok button to open the master/work order.

If there is already a master/work order in the Manufacturing Window, it is closed automatically. If an edit to an object was in process, you are allowed to confirm this action.
