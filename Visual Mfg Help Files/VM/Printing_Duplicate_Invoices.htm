Printing Duplicate Invoices

You can print duplicate customer invoices for events.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site whose invoices you want to print. If you are licensed to use a single site, this field is unavailable.

  2. From the File menu, select Print Duplicate Invoices. The Print Invoice dialog box appears.

The Reprintable Invoices dialog box also appears.

The invoices that you can reprint appear in the line item table.

  1. Highlight the invoice you want to reprint and click Ok.

The Print Invoices dialog box reappears.

  1. Enter in the necessary information. See Printing Invoices.
