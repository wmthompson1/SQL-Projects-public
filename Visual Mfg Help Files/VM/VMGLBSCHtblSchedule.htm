Creating a New Schedule

You can create as many global schedules as you need. Each can have its own set of resource calendars and capacities. Remember, the production schedule is the one used to drive the shop. You can view the others in the Scheduling Window.

  1. Click on the Insert button to open a blank line in the Scheduler table.

  2. Enter a Schedule ID in the Schedule ID column. Press the Tab key to continue to the next column.

  3. If you are licensed to use multiple sites, click the Site ID arrow and select the site to use for this schedule. If you are licensed to use a single site, this field is unavailable.

  4. Click on the check box in the Infinite Capacity column if this schedule is to have infinite capacity. Infinite capacity has the maximum number of supported units of capacity for every shop resource. Internally, this equates to 4094 units of capacity for each resource.

  5. Click on the check box in the Materials Checked column if you want VISUAL to check material availability for this schedule. Press the Tab key to continue to the next column.

  6. Click on the check box in the Services Checked column if you want VISUAL to check service availability for this schedule. Press the Tab key to continue to the next column.

  7. Click on the check box in the Active column if this schedule is to be included when you choose the Global Schedule All Active option from the File menu. Press the Tab key to continue to the next column.

  8. Click Save. VISUAL completes the remaining fields when you run the Global Scheduler.
