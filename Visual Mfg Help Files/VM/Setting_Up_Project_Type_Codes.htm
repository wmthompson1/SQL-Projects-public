Setting Up Project Type Codes

Project Types represent the types of projects you will be entering.

For example:

  • Individual Government contract

  • Long term commercial contracts that you are able to progress bill

  • Capital projects such as building expansion efforts

  • Indirect projects such as ISO certification efforts

Because you can use Project Type codes to segregate project costs in your General Ledger, you should give careful consideration to the codes you define.

To set up Project Type codes:

  1. In Project Maintenance, click the Maintain menu and select Project Types. Or, from the Direct Cost G/L Mapping and Burden Mapping Maintenance window, click the Maintain menu and select Project Types.

The Project Type table appears.

  1. Click the Insert button.

The next available row in the table appears highlighted.

  1. Enter the Code and Description you want to use.

If you will be using this project type to specify a non-project oriented job, click the Use for Commercial button.

VISUAL calculates burdens using the same methodology regard less of your project or non-project (commercial) work. You may want to set up a project type for non-project work so that VISUAL can correctly burden your non-project costs and assign those costs to the appropriate Burden G/L account.

You can only mark one project type as Use for Commercial. You cannot change the Used for Commercial designation if there are transactions associated with any project type code.

  1. Click the Save button.

  2. When you have finished entering Project Types, click the Close button.
