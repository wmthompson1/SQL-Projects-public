Associating VMX Files to the Main VISUAL/Financials Application

Refer to Windows® help documentation for information on where and how to enter new file types.

To associate file types, you must specify an action and an application used to perform that action.

For an action, enter Open.

For an application used to perform that action, enter the location of your main VISUAL application - Vm.exe.

For example, if the directory where your executables are installed is located in a folder called VISUAL710, enter this:

C:\Infor\VISUAL710\Vm.exe

After Windows® verifies that you have entered the correct file path name, it adds that file type to its list of recognized file types.

Now, when you attempt to open a VMX file, Windows® associates the action (Open) with the application (Vm.exe) and performs the actions necessary to open the main Infor VISUAL application.
