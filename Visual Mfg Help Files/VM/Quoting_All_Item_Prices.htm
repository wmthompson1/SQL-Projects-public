Quoting All Item Prices

The Quote All Item Prices option, located in the Edit menu, provides a convenient way to manage quoted unit prices for all line items.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site that contains the quote. If you are licensed to use a single site, this field is unavailable.

  2. Select the quote.

  3. Select Quote All Item Prices from the Edit menu.

A dialog box appears, containing a table that shows quoted price information for all line items and quantities.

  1. For each line item, you can edit the Quoted Unit Price and Trade Discount %.

Additionally, you can quickly set the Discount % on all line items by entering a Discount Code or by clicking on the Disc Code button and selecting from the browse list.

You can define Discount Codes in Customer Maintenance. When you enter the Discount Code, the Trade Discount Percentage for all line items and Extension are automatically populated.

Specify the unit of measure by selecting from the drop down list.

  1. Click the Save button to save any changes or Close to cancel the changes.

    Changes in this dialog will affect the Quoted Unit Price shown in the Prices dialog for each individual line item. Any changes will override unit prices that may have been calculated from costs. This feature is most useful as a quick way to quote prices when not automatically calculating prices. You can then view resulting profit margins by using the Cost Detail report.
