Removing Commands from Groups

To remove a command from a group:

  1. Select File, Personal Menu from the main VISUAL menu.

  2. In the Personal Menu, click the group containing the command you want to remove.

  3. Right-click the command you want to remove and select Customize from the right-click menu.

  4. Click the Remove Current Item button.

VISUAL removes the command from the Personal Menu.

  1. Click the OK button.

The customize dialog box closes.
