Importing an Updated Excel Spreadsheet into Quick Quote

If you update the quote information in Excel, you must re-import the spreadsheet to update the Quick Quote.

To re-import a spreadsheet:

  1. Select Sales, Estimating Window.

  2. Open the quote.

  3. Select the line item you are quoting.

  4. Select Edit, Quick Quote.

  5. Click the Copy From button.

By default, the last spreadsheet that you selected for the quote is selected. You can select a different spreadsheet.

  1. Click Ok. The Quick Quote is updated. If you selected a different spreadsheet for the quote, the spreadsheet is attached to the quote as a document attachment. The original spreadsheet is also retained as a document attachment.
