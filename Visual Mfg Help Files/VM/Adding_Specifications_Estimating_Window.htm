Adding Specifications

Specifications allow you to add free form text to quotations. This text is printed on Quotations and can specify and document special needs, notes, and agreements for the customer quote. You might use specifications to specify packaging requirements, shipping details, configuration information, or other options that need to be specified along with a customer order.

There are two types of quotation specifications (specs, for short): Quote Specs and Line Specs. Quote Specs are associated with the entire quote, and Line Specs with individual line items.

You do not need to close Specification windows after each use. They can remain open for the whole session, and their contents change as the current quote and selected line item change. You must select the field (the spec line quote) for the change to occur.
