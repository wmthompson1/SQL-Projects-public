Defining Label Groups

Use label groups to combine labels of like type into groups, which you can then print as you would individual labels.

  1. From the Maintain menu, select Label Groups.

The Label Groups dialog box appears.

  1. If this is a new label group, enter a unique ID into the Label Group ID field. To choose an existing label group, click the Label Group ID button and select the appropriate group.

  2. From the Label Group Type list box, select a label type for this group. All labels in a group must be the same type. VISUAL enforces sameness if you try to choose a label of a different type.

  3. Provide a description for the group in the Description field. To choose an existing group by Description, click the Description button and select the appropriate group.

  4. Begin adding labels to the group by clicking the Insert button. Enter information as follows:

<Label ID> - Double-click the <Label ID> column header to choose a label to add to the group. Only labels of the type you specified in step 3 are eligible for selection.

Label Printer – If the label has a default printer, VISUAL auto-fills this field with the printer name. See defining thermal labels for more information.

Copies – If the label has a default number of copies setting, VISUAL auto-fills this field with the value. See defining thermal labels for more information.

Multiplier – If the label is has a multiplier preference set for it, VISUAL selects this check box. See defining thermal labels for more information.

  1. Click the Save button to save the label group.

To delete a label from a group, select the label, click the Delete button, and then click the Save button.

VISUAL removes the label from the group.

  1. Click the Close button to return to the main window.
