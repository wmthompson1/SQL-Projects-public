Setting Up Discontinuous Operations

The Global Scheduler supports discontinuous scheduling of operations. You can set up a shop resource so that operations using that resource may be scheduled discontinuously by default. To setup a shop resource to schedule discontinuously:

  1. Click the Other tab in Shop Resource Maintenance.

  2. Select the Schedule Discontinuous check box, then enter the minimum number of hours of run that should be scheduled in one segment.

The segments provide the default values for operations that use that resource.

  1. Click the Save button to save the changes.

VISUAL can now schedule discontinuous operations for operations that use this resource. You can override these default values in the Manufacturing Window.
