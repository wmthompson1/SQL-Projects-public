Entering Cross-Selling Part information for a Part

Cross-selling parts are parts that through similar structure, functionality, or price, you can logically group or sell with the focus part of a customer order. The use of cross-selling parts is exclusive to the Order Management module. You can only use cross-selling parts in the Order Management window

  1. From the Order Management tab, click the Cross-Selling Parts button.

The Cross-Selling Parts for Part dialog box appears.

  1. Click Insert to start adding line item information.

The first line is now active.

  1. Enter information in the following fields:

Cross-Selling Part ID - The ID of the part that you want to classify as a cross-selling part. Either manually enter a Part ID or double-click the Part ID column header to select one form the line item table.

Cross-Selling Part Description - The description of the cross-selling part. VISUAL fills this field automatically with the accompanying part description after you select a Cross-Selling Part ID.

  1. When you are done entering cross-selling part information for a part, click the Save button to commit the cross-selling part information.
