Setting the Global Hard Release Dates Preference

Be sure that you want the Global Scheduler to consider all work order release dates as hard release dates in all schedules -- new or existing -- before you make this selection. This setting is global and you cannot override it.

From the Scheduling tab, you can set a preference for the Global Scheduler to treat all work order release dates as hard release dates in all schedules.

With this global hard release date flag set, VISUAL enforces the hard release date for all schedules and all work orders, independent of what you have specified for the individual schedule or work order.

When you run the Global Scheduler, the Global Schedule dialog box has the "Treat release dates as hard" check box both selected and unavailable to ensure that it your global preference remains enforced.

With this option set, the Global Scheduler schedules no activity on any work orders in any schedule before the work order’s hard release date, which you can specify per work order in the Manufacturing window during the creation of work orders.

To set this global preference:

Select the Treat all release dates as hard in all schedules check box.
