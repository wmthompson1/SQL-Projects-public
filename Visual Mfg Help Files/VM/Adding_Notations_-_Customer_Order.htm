Adding Notations - Customer Order

Notations (notes, for short) are free form text used for internal documentation purposes. Although they are associated with the customer order in much the same way as specs, they are fundamentally different:

  • Notes are exclusively for internal use of your business. They are not included on any outgoing paperwork.

  • Individual note entries are automatically date/time stamped, and maintained as separate entries. This is different from specs, where you can modify the entire block of text at any point as a single object.

There are two types of notes associated with Customer Order Entry:

Order Notes - Like order specs, these are associated with the order as a whole. Use Order Notes for internal dialogs concerning this particular order.

Customer Notes - These are running notes associated with the current Customer ID of the Customer Order. If no customer is selected, these notes are not active. Note that these notes are not linked to any current customer order, but to the current customer. Use Customer Notes for internal dialogs regarding the customer.

All notations in Infor VISUAL function in the same way.
