VISUAL AutoRun Services

You can use VISUAL AutoRun services to schedule certain long-running processes at convenient times. After you install the services, use the appropriate dialog to schedule the service.

Each service runs a process associated with a particular window. After the services are installed, schedules are set up in the window where the processes would ordinarily be run. For example, the Costing Service runs costing in the Costing Utilities window, and you set up the schedule to run the Costing Service in a dialog available in the Costing Utilities window.
