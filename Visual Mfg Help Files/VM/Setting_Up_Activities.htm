Setting Up Activities

Before you enter activities, set up activity categories and activity types. Use activity categories to describe the topic of the activity. For example, activity categories could include customer credit, sales planning, or vendor contracts. Use activity types to describe how the activity is conducted. For example, activity types could include phone call, e-mail, or meeting.

If you use VISUAL CRM, then the categories and types you define in VISUAL can also be used in VISUAL CRM. Likewise, the categories and types you define in VISUAL CRM can be used in VISUAL.
