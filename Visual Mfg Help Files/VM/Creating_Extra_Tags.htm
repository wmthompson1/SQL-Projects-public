Creating Extra Tags

During a count, you may discover that you need more blank tags. This is not the same as printing tags again; you need new tag numbers.

To do this:

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site where you conducted the count. If you are licensed to use a single site, this field is unavailable.

  2. In the table, select the physical inventory count for which to create new tags.

  3. Select Create New Tags from the Edit menu.

The Create Tags dialog box appears.

You can only use this option to add tags to the current count, so the Count ID that appears in the dialog box is not editable.

Tags per group automatically default to the value specified when the count was initialized. You cannot modify this value.

  1. Enter the Starting Tag Number.

The Starting Tag Number defaults to the first unused number in the existing count. You can use a higher number, if needed.

  1. To define the total number of tags, enter the Ending Tag Number.

  2. Click Ok to create the tags.

You can use the standard Print Tags command to print these new tags.
