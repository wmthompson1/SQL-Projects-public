Printing Transaction Report

The Transaction report allows printing of trace records and their associated transactions. It is essentially a printed version of the Part Trace Maintenance table, with options to print Show Inventory Transactions and Show Labor Transactions.

To print this report:

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site ID to use for the report. If you are licensed to use a single site, this field is unavailable.

  2. Select Print Transaction Report from the file menu.

The Print Trace Transactions dialog box appears.

  1. Enter the part range to print.

You can print the report for a single part, a range of parts, or all parts. To print for a single part, enter its Part ID in the Starting Part ID and Ending Part ID fields. You can specify a closed or open-ended range by entering one or both of these fields. Starting and Ending Part IDs are included in the range.

  1. Choose the print options.

If you want the information for each Part ID to print on its own group of pages, check the Page Break on Part ID check box.

Normally, alpha properties, numeric properties, and comments print with the report. To suppress printing of one or both, check the corresponding box.

  1. Choose the Detail type.

To print the associated inventory transactions for each trace record printed, check the Inventory check box. To print labor transactions, check the Labor check box.

  1. Enter the date range.

If you are printing inventory and/or labor transactions, you can specify a date range, and only transactions dated within that range print.

You can enter a closed or open-ended date range by filling in one or both of the Starting Date and Ending Date fields. The Start and End date are included in the range.

  1. Select the print output.

Select either Print, View, or File.

  1. Click Ok.

If you are printing the report, a standard Windows dialog box appears.

  1. Make the appropriate selections, then click Ok to print the file.

For each trace record of each part, the Lot Number, Received Quantity, Issued Quantity, Reported Quantity, and Assigned Quantity print, along with the Expiration Date. Properties and comments also print, if not suppressed.

If you selected Inventory transaction printing, Transaction Number, Type, Quantity, Work Order ID, Purchase Order ID, Customer Order ID, and Warehouse Location print for each transaction that falls within the specified date range.

If you selected Labor transaction printing, the Transaction Date, Type, Quantity, Work Order ID, Employee ID, and Resource ID appears for each labor transaction that falls within the date range.
