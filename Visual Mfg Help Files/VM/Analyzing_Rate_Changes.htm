Analyzing Rate Changes

As you changes the rates you use in billing and revenue recognition, it is important you understand the impact those changes have on your cost elements.

To run rate change analysis:

  1. From the Project menu of the Infor VISUAL main window, select Rate Change Impact Analysis.

The Current and Applied Billing Rates for the project you selected automatically appear.

  1. Click the Project ID button and select the project for which you want to run a rate analysis.

  2. If you want the information sorted on this report by CLIN, select the By CLIN check box.

  3. If you want this report to contain information from a specific date rate, use the calendar buttons in the Enter Date Range section to select the Start and End fiscal periods.

  4. In the Select Rate to Compare section, select the rate you want to compare by selecting the appropriate option. You can select either the Current Billing Rate ID or the Applied Billing Rate ID.

  5. In the Compare with this Rate section, click the Select Rate ID arrow and select the rate to which you want to compare the Current or Applied Billing Rate.

  6. Click the Run button on the toolbar.

The rate you selected is compared against the Current or Applied rate for this project and report appears with the appropriate figures.
