Assigning Purchase Order Delivery Schedule Supply to Work Order Material Requirement Demand

  1. With the appropriate delivery schedule line item highlighted, click the Demand button.

The Assign to Demand dialog box appears.

  1. Click Insert to begin adding P/O delivery schedule line information.

  2. From the Type list box, select RQ.

  3. Double-click <Demand Base ID> to view a list of all work orders in your database. 

The Work Orders dialog box appears.

  1. Select the work order to which you want to allocate supply and click Ok.

The necessary information is loaded in the Assign to Demand dialog box line item table.

  1. Double-click <Seq #/Line #> to view a list of work order material requirements to which you can assign P/O delivery schedule line supply.

The Work Order Material Requirement Demand dialog box appears.

  1. Using the search and sort options in the top portion of the dialog box, select what lines of work order material requirement you want VISUAL to display in the line item table and click Apply.

According to how you configured the search and sort options all qualifying work order material requirement lines appear.

  1. Select the appropriate line and click the Ok button.

VISUAL loads the necessary information in the Assign to Demand dialog box line item table.

  1. In the Allocate Quantity column, enter the quantity of the P/O delivery schedule line that you want to allocate to work order material requirement demand.

The quantity that you enter in the Assign Quantity column can not be greater than the Demand Unallocated Quantity of the work order material requirement or the Available Quantity of the P/O delivery schedule line.

  1. Click Ok to commit the allocation of supply from the P/O delivery schedule line to work order material requirement demand.
