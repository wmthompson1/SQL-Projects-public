Allocating Quantities from the Advanced Material Planning Window Line item Table Detail Dialog Boxes

You can allocate quantities to supply and demand using the Advanced Material Planning Window’s Order Details dialog boxes.

You can perform the following allocations:

  • Allocate Supply to Customer Order Lines

  • Allocate Supply to Outgoing Interbranch Transfers

  • Allocate Supply to Work Order Material Requirements

  • Allocate Purchase Order Quantities to Demand

  • Allocate Inventory Quantities to Demand

  • Allocate Incoming Interbranch Transfer Quantities to Demand
