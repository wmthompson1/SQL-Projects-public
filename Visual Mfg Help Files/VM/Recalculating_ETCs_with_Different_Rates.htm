Recalculating ETCs with Different Rates

Because it is important for you to be able to predict how a project’s costs change, you can recalculate your projects Estimate to Complete costs using different rates.

To recalculate ETCs:

  1. With the Maintain ETC window open and the appropriate project selected, click the Rate ID button and select the rate you want to use.

  2. Click the Apply New Rates to ETC button.

VISUAL recalculates the Project’s ETC figures based on the new rate you selected and upon successful calculation, a dialog box appears.

VISUAL only calculates ETCs for the project you selected, if you want to recalculate ETCs for all of your projects, you must do so in the Rate Maintenance window..
