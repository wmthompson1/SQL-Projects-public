Generating Requisitions from RFQs

Use this feature to generate a purchase requisition based on a vendor RFQ, which is a document generated by a vendor in response to a request for a quote.

If you are licensed to use multiple sites, you can create a requisition from an RFQ within the same site only. You cannot create a requisition from an RFQ entered in a different site.

You can use this feature without a requisition record in the window.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site to use. If you are licensed to use a single site, this field is unavailable.

  2. From the Edit menu, select Generate Requisition from RFQ....

The Generate Purchase Requisition from RFQ dialog box appears.

  1. Click the Request ID button to select the vendor RFQ you want to use to generate the requisition.

If the RFQ contains any lines eligible to become requisition lines, they appear in the line item table.

Before a RFQ line can be used to create a requisition, it must have a quantity and at least a default vendor unit price.

VISUAL supplies the appropriate vendor in the Vendor ID field.

  1. If appropriate, enter an ID for the requisition into the New Order ID field. If you use auto-numbering, leave this field blank for VISUAL to assign the order the next available ID.

  2. Specify an Order Date, a Desired Receive Date, and an Original Promise Date for the order. The current system date appears by default in each date field.

  3. If appropriate, click the Warehouse ID button to choose a warehouse for the order.

  4. Continue completing header information as follows:

Buyer - Specify a Buyer for the requisition. If a buyer is specified on the RFQ, VISUAL displays it here. Change it as necessary.

FOB - Specify an FOB code for the requisition. If an FOB code is specified on the RFQ, VISUAL displays it here. Change it as necessary.

Ship Via - Specify a Ship Via code for the requisition. If a ship via code is specified on the RFQ, VISUAL displays it here. Change it as necessary.

User ID - Specify a User ID for the requisition. If a User ID is specified on the RFQ, VISUAL displays it here. Change it as necessary.

Entity ID - Specify an Entity ID for the requisition. If an Entity ID is specified on the RFQ, VISUAL displays it here. Change it as necessary.

  1. Select the Copy RFQ Specs to Purchase Requisition check box to copy RFQ specifications to the requisition.

Select the Copy RFQ Header Information check box to copy RFQ header information to the requisition.

  1. From the table, select which lines you want to add to the requisition. Click the Select All button to select all lines; click the Deselect All button to deselect all lines. To select multiple, noncontiguous lines, hold down the CTRL key while clicking the appropriate lines. 

  2. Click the Ok button to create the requisition.

VISUAL processes the lines and returns you to the Purchase Requisition Entry window, displaying the new order.

  1. Complete the order.
