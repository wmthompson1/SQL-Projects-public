Editing Packlists & ASNs

After creating a packlist for an order, you can edit its contents, append service charge lines, or edit an ASN you created during shipment.

Before you can edit a packlist there must be a packlist to edit. To do this, either choose to print a packlist before shipping, or ship the lines as you normally would. In either case, VISUAL creates a packlist you can then edit.

  1. Either by Order ID or packlist ID, select the order with the packlist you want to edit.

  2. If you have not already shipped the order, print a pack list for it before shipment, or ship it.

  3. After VISUAL generates the packlist and ships the order, click the Edit Pack list button on the main toolbar.

The Edit packlist dialog box appears. For general information on using edit packlist features, click here.

The Order ID and the packlist ID appears in read-only reference fields at the top of the dialog box.

  1. Click the Other tab to enter general information for the packlist.

  2. Click the Insert button to append a service charge to the line. Service charges are additional charges, such as expediting or installation fees, attached to an order.

  3. If VISUAL generated an ASN for this order at the time of shipment, and you want to change ASN details, click the Edit ASN toolbar button.

The Advance Ship Notice dialog box appears.

  1. If appropriate, enter information into the following fields:

Exped Trans Code - Enter an Exped Trans Code for the ASN.

Carrier ID - From the Carrier IDlist box, select a valid VISUAL Carrier ID. Enter Carrier IDs through Application Global Maintenance.

Exped Trans Auth - Enter an Exped Trans Auth for the ASN.

Trans Method Code - Enter a Trans Method Code for the ASN.

Lading Code - Enter a Lading Code for the ASN.

Lading Quantity - Enter Lading Quantity for the ASN.

Ship ID (Pool Point) - Enter a Ship ID for the ASN.

Equip Desc Code - Enter an Equip Desc Code for the ASN.

Equip Initial - Enter an Equip Initial for the ASN.

Air or Pro Number Type - Enter an Air or Pro Number Type for the ASN.

Air or Pro Number - Enter an Air or Pro Number for the ASN.

  1. If the ASN is on hold, the ASN on Hold check box is selected. This means that VISUAL will not send the ASN until you have activated it.

If you want to put the ASN on hold, select this check box.

It is possible to make ASN on Hold a default selection, either by customer or by Ship To ID.

If appropriate, select a User Defined Layout ID for the ASN from the User Defined Layout ID list box.

VISUAL updates the User Defined field labels and you can add information as necessary.

  1. Click the Save toolbar button to save the edited ASN information.
