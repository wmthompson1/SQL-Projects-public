Selecting Customer Fill Rate Types

With the appropriate Customer ID in the Customer ID field, click the Order Management tab.

From the Fill Rate Type list box, select a Fill Rate type.

L (Line Level) - When unavailable inventory for a part on an order line item reaches the fill rate threshold, VISUAL cancels the line item.

W (Whole Order) - When unavailable inventory for a part on an order line item reaches the fill rate threshold, VISUAL cancels the whole order.
