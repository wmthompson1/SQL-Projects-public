Selecting a Default Country of Origin for a Part

You can set a default Country or Origin for each part in Part Maintenance. You can override the default country of origin in Customer Order Entry.

  1. With the appropriate Part ID displaying in the Part Maintenance window, select the Order Management tab and then click the Default Country of Origin button.

The Default Country of Origin dialog box appears.

The system lists any country used in a customer record, customer ship-to address, customer address, vendor record, vendor address, and vendor remit-to address.

  1. Either double-click a line or select a line and click the Ok button.

The Country of Origin appears in the Default Country of Origin field.

If you use this part in Customer Order Entry, the system inserts the default country of origin in the Country of Origin column in the lines table.
