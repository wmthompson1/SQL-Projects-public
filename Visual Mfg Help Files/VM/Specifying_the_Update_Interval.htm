Specifying the Update Interval

To specify how often the DBR Dashboard updates buffer information:

  1. Select Tools, Options.

  2. To refresh the information in the dashboard automatically, select the Refresh Automatically check box. After you select the check box, specify how frequently to check for new information in the Refresh Every fields. To refresh the information manually, clear the Refresh Automatically check box.

Regardless of the setting you choose, you can refresh the information in the dashboard at any time by clicking the Refresh button on the toolbar.
