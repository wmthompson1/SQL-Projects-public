Attaining Statistical Control

To attain a state of statistical control:

  1. Take measurements for a number of subgroups.

Usually 25 subgroups is optimal, but 10 subgroups is also sufficient, especially where production runs are short. If you only use 10 subgroups, however, the proper location of the control limits is defined with considerably less precision.

  1. Log any process changes (material, worker, tool, method, person) during the collection of data.

  2. Compute trial control limits from this data.

Use the Define Control Limits item of the Edit menu in the Test Input/Analysis window.

  1. Chart the data using the graphing tools and see if all points are within the control limits.

  2. Run the decisions tests to determine if the process is in statistical control.
