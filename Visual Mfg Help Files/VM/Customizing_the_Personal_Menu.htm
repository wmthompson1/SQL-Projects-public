Customizing the Personal Menu

You can add commands and groups of commands to your personal menu customizing it to meet your exact needs. The Personal Menu supports almost all executable commands as well as providing you with all of the VISUAL programs you may want to use.

  1. With the Personal Menu open, right-click anywhere in the Personal Menu.

  2. From the right-click menu, select Customize.

The Customize dialog box appears.

  1. Begin to customize the personal menu. See the following sections for more information.
