Purging Test Results

VISUAL saves test results in your database. You may want to purge these tests to make room for new tests.

  1. Select Purge Test Results from the Edit menu.

The Test Purge window appears.

  1. In the Test ID field, enter the test ID for the test you want to purge, then click the button.

The information for the test appears in the window.

  1. Select the Test ID line items you want to delete.

You can click the Select All button to select all Test ID line items. Click the Select All button again to de-select all.

  1. Click the Delete Selected button.

  2. Click the Save button to complete the purge.
