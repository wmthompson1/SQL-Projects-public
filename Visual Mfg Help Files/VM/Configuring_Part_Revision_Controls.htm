Configuring Part Revision Controls

Before assigning Rev IDs to parts, take time to establish a revision-numbering scheme.

Whether you use the Infor VISUAL ECN module or a manual revision scheme to control part revisions, use the Part Maintenance window to establish part master revision preferences.

  1. Click the Part ID button to select the part you want to configure.

  2. Click the Config Mgt tab.

  3. If you use a manual numbering scheme to control part revisions, enter a revision level for the part into the Revision ID field.

If you previously generated revision numbers using Revision Numbering Maintenance, you may want to select a stage first, which allows you to choose a Revision ID associated with the stage.

  1. From the Stage list box, select a Stage for the revision. Stages are predefined in Revision Numbering Maintenance and apply to the various levels of part production and subsequent maintenance revision.

    If you select a stage with associated revision numbers, VISUAL changes the Revision ID field to a Revision ID button, which you can then click to browse a list of all predefined revision numbers.

  1. If this part has an associated Drawing #, enter it into the Drawing # field.

  2. If the drawing is also undergoing a revision, enter a Revision ID for the drawing into the Drawing Revision field.

  3. To browse for a drawing file, click the Cabinet button; to open a file, click the Open Folder button.

  4. If, as mentioned in the introduction to this topic, you use the Infor VISUAL ECN module to control part revisions, select the ECN Controls Revision check box.

This disables the Revision ID and Stage fields.
