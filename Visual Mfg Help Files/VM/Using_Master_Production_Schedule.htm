Using Master Production Schedule

The Master Production Schedule function has two main uses:

  • Setting and maintaining a forecast of demand for the current part.

  • Setting and maintaining the part’s master production schedule for MRP to use when generating planned orders.

If you are licensed to use multiple sites, set up master production schedules by site. Select a site ID in the Material Planning window before accessing the Master Production Schedule dialog box. .

To perform these both of these functions, use the Master Production Schedule command from the File menu.

The Master Production Schedule dialog box appears.

If the application global setting for Material Requirement Planning >By Release Date check box is selected, the demand order’s release date will be used to determine the forecast bucket to include the demand. If it is unchecked, it will use the required date.
