Printing Call Reports

If you only want to print a specific call, use the File menu option Print Call Detail. If you want to print a more detailed report containing multiple calls, use the Print Call List report. This feature offers greater flexibility in defining parameters for a report. For example, you can print a report by date range, urgency code, or escalation code.

  1. From the File menu, select Print Call List.

The Call Reporting dialog box appears.

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the sites to include in the report. If you are licensed to use a single site, this field is unavailable.

  2. Specify a date range for the report by clicking the calendar buttons or manually entering a date range.

  3. Select a Report by option button. Choices include, customer, engineer, area, equipment, and call. VISUAL sequences the report according to this setting.

  4. Narrow the scope of the report more by choosing the appropriate Select Only options. By allowing you to select specific components of existing calls, VISUAL can compile a call report that exclusively includes only calls with, for example, a specific urgency code, escalation code, or any of the other possible selections.

Equip ID - Click the Equip ID button to select a piece of equipment. VISUAL compiles the report with calls for this piece of equipment only.

Type - Click the Type button to select a type of equipment. VISUAL compiles the report with calls with this type of equipment only.

Model - Click the Model button to select a model. When you select a model, VISUAL auto-fills the Type field as well. VISUAL compiles the report with calls with this model and type of equipment only.

Fault Type - Click the Fault Type button to select a fault type. VISUAL compiles the report with calls with this fault type only.

Fault Code - Click the Fault Code button to select a fault code. VISUAL compiles the report with calls with this fault code only.

Status - From the list box, select a status. VISUAL compiles the report with calls with this status only.

Urgency - Click the Urgency button to select an urgency code. VISUAL compiles the report with calls with this urgency code only.

Escalation - Click the Escalation button to select an escalation code. VISUAL compiles the report with calls with this escalation code only.

Engineer - Click the Engineer button to select an engineer (resource). VISUAL compiles the report with calls with this engineer (resource) only.

  1. Select how you want to output the report:

Print - Select Print if you want VISUAL to print the report. After clicking Ok, you can specify print preferences and begin printing.

View - Select View if you want to view the report. After clicking the Ok button, VISUAL opens Centura Report Builder and displays the report.

File - Select File if you want to save the report to file. After clicking Ok, VISUAL allows you to name the file; VISUAL then saves the file in your VISUAL directory.

E-mail - Select E-mail if you want to send the report through electronic mail in a Rich Text Format (.RTF). After clicking Ok, VISUAL displays the report as an attachment and allows you to specify to whom you want to send and/or CC the file.

  1. Click the Print Setup button to specify which printer you want VISUAL to use when printing Unplanned Maintenance reports.

  2. Click the Ok button to begin the printing process.

  3. Click the Cancel button to return to the Unplanned Maintenance window.
