Specifying a Customer Type for a Customer

You can assign customers types in Customer Maintenance. Customer types act as unique identifiers of customers and their ordering needs and habits through the use of priority rankings. Types can also have a default reallocation preference setting of No or Yes and a default reallocation fence value. Set reallocation preferences and reallocation fence values per Customer Type in Accounting Entity Maintenance. See Entering Customer Types.

  1. Select Sales, Customer Maintenance.

  2. Click the Customer ID browse button and select the customer.

  3. Click the Order Management tab.

  4. Click the Customer Type button.

  5. Select the appropriate line and click OK or double-click the line and the order type appears in the Customer Type field.
