Troubleshooting Toolbar Buttons for Macros Stored on the Workstation

If the macro attached to a toolbar button cannot be found, this message is displayed: Macro name is not valid. The reasons you may receive this message include, but are not limited to:

  • The macro file has been renamed

  • The macro file has been deleted from the workstation

  • The macro button refers to a macro stored in the database, but you now store macros on the workstation.

If you no longer use the macro, you can remove the toolbar button. If you still use the macro, update the toolbar button with the correct macro file name, or rename the macro to its original name.
