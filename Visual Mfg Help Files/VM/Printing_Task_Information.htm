Printing Task Information

Use this feature to print requisition task reports or send a .rtf list through e-mail to another approval team member or individual.

  1. From the File menu, select Print or click the Print button on the main toolbar.

A Print Report dialog box appears.

  1. Select an output for the report:

Print - Select Print to output the file to a printer.

View - Select View to view the report before printing.

File - Select File to save the information to a text file.

E-Mail - Select E-Mail to send the report in a Rich Text Format (.RTF) through electronic mail.

  1. Click Ok to output the report to the specified destination.

If e-mailing the report, the system attaches the file to a Microsoft Outlook e-mail. For more information on addressing and sending the e-mail, refer to your Microsoft Outlook user documentation.
