What is Examining Triggers?

Using VISUAL’s Examine Triggers tool, you can check your database for missing, additional, disabled, and audit triggers.

The Examine Triggers tool compares the list of required triggers to those it finds in the database.

Any entries you find in the Found Triggers, Additional Triggers, and Disabled Triggers fields are cause for concern because the identified triggers indicate there could be a problem with your database.

Only users with administrator privileges have the authority to run the Examine Triggers program.
