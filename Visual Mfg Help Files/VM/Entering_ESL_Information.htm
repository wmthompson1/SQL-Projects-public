Entering ESL Information

A European Sales List (ESL) is a VAT report that is based on goods reported under Intrastat. You can enter the following ESL information:

ESL Export ID - Enter a valid export ID if you are generating the report to file.

Next Sequence # - Enter the next number for the ESL report.

Frequency - Specify how often you want to generate the report by selecting the appropriate radio button. You can select Monthly, Quarterly, or Yearly.

Click the Save button to save the information.
