Adding and Deleting Vendor Withholding Codes

Use Vendor Maintenance to add and delete vendor withholding codes.

Adding Vendor Withholding Codes

  1. Select Withholding Codes from the Maintain menu.

  2. Click the Insert toolbar button.

A new line appears in the line item table.

  1. In the Code column, enter a unique identifier for this withholding code.

  2. In the Description column, enter a unique description for this withholding code.

  3. In the Tax Percent field, enter the withholding tax percentage for this withholding code.

  4. In the Tax G/L Account ID field, enter the G/L account to which this tax is posted.

  5. In the Tax Allowance Percent field, enter the total amount that can be exempt from withholding tax.

  6. Click Save.

Deleting Vendor Withholding Codes

  1. In the Withholding Codes dialog, select the withholding code line to delete.

  1. Click the Delete toolbar button.

An X appears in the row header indicating that you have marked it for deletion.

  1. Click Save.
