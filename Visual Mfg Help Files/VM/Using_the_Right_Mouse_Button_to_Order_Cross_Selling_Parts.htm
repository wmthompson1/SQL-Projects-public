Using the Right Mouse Button to Order Cross-Selling Parts

You can order a cross-selling part by right clicking on the part that you want to add to the current customer order.

  1. In the VISUAL Order Management Customer Order Entry window, highlight the order line that contains the primary part. This is the part for which you are ordering cross-selling parts.

  2. If not already open, the Order Management Window; then open the Current Customer Order Window and the Cross-Selling Parts Window. Tile or cascade the two so you can clearly view both windows.

  3. In the Cross-Selling Part window, double-click the icon.

If there are any, cross -selling parts for the primary part appear.

  1. Highlight the cross-selling part that you want to add to the current customer order and perform a right mouse click.

An Order Cross Selling Part message box appears.

  1. Left-click the message box to allocate the supply.

The VISUAL Order Management Customer Order Entry Window appears with a line appended to the current customer order for the cross-selling part.

  1. Enter an order quantity for the part and any other necessary information and then click the Save button to commit the order.
