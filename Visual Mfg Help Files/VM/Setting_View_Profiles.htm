Setting View Profiles

The Order Management Window supports multiple views. You can establish views that allow you instant access to general or specific information regarding the current customer order.

  1. In the View field, type a name for the view.

  2. From the Tools menu, select the window or windows that you want to appear when you select this view from the list box. You must open all the windows that you want to comprise the view. VISUAL places a check mark (4) next to windows that you select from the Tools menu.

  3. When you have opened all the necessary windows, click the Save button to the immediate right of the Views field.

Deleting View Profiles

  1. From the View profile list box, select the profile you want to delete. The selected View appears on the Order Management Window.

  2. Click the Delete button. VISUAL prompts you to confirm the deletion of the view.

  3. Click Yes to remove the View from the database. This removes the View profile from the database, and returns you to the <Default> view. Click No to save the View profile and remain in the Order Management Window.
