Showing Operation Details

This feature allows you to view a specific operation in a work order.

  1. Focus on a particular operation by pointing and clicking on the appropriate operation.

  2. Select Show Operation Detail from the Info menu.

You can also display an operation by double clicking on the desired operation.

A window containing detail information about that operation appears.

All of the information in the top sections of the window is identical to that defined in the Manufacturing Window Operation dialog box. Start and Finish are the dates determined by the Concurrent Scheduler. The Quantity shown is the End Quantity, and Completed is the quantity reported complete from Labor Ticket Entry.

  1. Use the NEXT or PREV buttons to show the subsequent or proceeding operation in the same Work Order leg.

For example, if you have a Work Order with two legs (i.e. sub_ids), each having operations 10, 20, 30, and 40, and you are looking at operation 20 of sub ID 1 (the second leg), using NEXT displays operation 30. If you choose PREV, operation 10 appears.
