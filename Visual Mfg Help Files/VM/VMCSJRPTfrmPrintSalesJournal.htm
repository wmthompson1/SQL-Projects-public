About the Customer Order Sales Journal Report

Use the Customer Sales Journal to review the amounts that you have invoiced customers

You can print a detailed report or a summary report. The detailed report lists individual transactions based on the sort sequence. The summary report groups the transactions based on the sort sequence. If you include multiple entities and sites in the report, totals are provided by each site and entity.

Starting the Customer Order Sales Journal Window

Select Sales, Customer Sales Journal.

Printing Customer Order Sales Journals

Use this procedure to generate customer sales journal reports.

Fields

Site ID(s)

If you are licensed to use multiple sites, click the arrow and select the sites to include in the report. If you are licensed to use a single site, this field is unavailable.

Starting Invoice Date and Ending Invoice Date

The date range for the report.

Starting ID and Ending ID

The label for these fields changes based on the sequence option that you select. For example, if you select Customer ID in the Sequence section, the field labels are Starting Customer ID and Ending Customer ID. Use these fields to print the report for a range of IDs or records. To include all IDs in the report, leave the fields blank. To print the report for one ID, specify the same ID in the Starting ID and Ending ID fields.

If you click Due date in the Sequence section, these fields are not displayed.

Report Currency

The currency for values in the report. Select ** Native ** to view amounts in each order's currency. If you select ** Native **, grand totals are not provided. If you select sites in more than one accounting entity, then the currency list is filtered to show the currencies that the entities share.

Report output

Select the output format for the report. See Report Output Options.

Summary

Click this option to view a summary report. A summary report provides invoice totals based on the sort sequence that you selected. The report is grouped by site and entity.

Detail

Click this option to view a detailed report. A detailed report includes the individual transactions that make up each subtotal.

Print Uninvoiced

Select this check box to include uninvoiced shippers in the report.

Sequence

Specify how to sort the report. For example, if you click Part ID, the report shows customer orders for each part ID.
