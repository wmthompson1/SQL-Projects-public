Printing Order Point Reports

The Order Point report provides a management tool for parts that are not planned by MRP, and have specified order points.

To run this report:

  1. If you are licensed to use multiple sites, click the Site ID browse button and select the site to use for the report. If you are licensed to use a single site, this field is unavailable.

  2. Select Print Order Point Report from the File menu.

The Print Order Point Report dialog box appears.

  1. Enter the Starting and Ending Part IDs.

These default to the current part. If necessary, modify these to print the desired range. If you want to print all parts, leave both fields blank.

  1. Select the Print Mode.

If you check the Parts Below Order Point Only check box, then only those parts whose total on hand quantity is less than or equal to the part’s Order Quantity appear in the report.

  1. Choose the output type from the drop down menu. Select either Print, View, or File.

  2. Click Ok.

If you select Print as the output type, a standard Windows Print dialog box appears.

  1. Click Ok to print the reports.

For each part, the Part ID, Description, Product Code, Commodity Code, Unit of Measure, Allocated Quantity, On Order Quantity, and Order Point appear. Allocated quantity is the total quantity of outstanding demands. On Order Quantity is the total quantity of outstanding supply orders.
