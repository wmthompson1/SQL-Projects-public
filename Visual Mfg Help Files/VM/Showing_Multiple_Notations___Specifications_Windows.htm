Showing Multiple Notations & Specifications Windows

You can have multiple notations and specifications windows open at once. For example, in Purchase Order Entry, you can simultaneously show Vendor Notations, Order Notations, and Order Specifications.

There are two options to manage the display of these windows, both available in the Notes menu: tiling and cascading.

Tiling Notations/Specifications

VISUAL stacks all open notation and specification windows vertically on the right side of the screen, so that each window is fully uncovered. This is useful if you have a higher resolution display, because it allows you to utilize the screen space.

Cascading Notations/Specifications

Starting at the top-left of the screen, VISUAL stacks all open notation and specification windows one on top of the other, each one shifted down and to the right. You can see the title bar of each window. To use a window, click its title bar to move it to the front. This option is useful when you do not have a large amount of screen space.

Resizing Notations and Specifications Windows

You can resize Notations windows to any size. The text portion of the window grows or shrinks in both directions.

Saving Notation and Specification Window Positions and Sizes

You can choose to have the size and position of these windows saved. They will be saved each time you move them.

To control this option, select Notation Preferences from the Notes menu. There is a check box to control saving of window positions and another to control saving of sizes. If the boxes are not checked, VISUAL does not save the sizes/positions.
