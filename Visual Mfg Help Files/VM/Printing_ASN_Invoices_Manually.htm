Printing ASN Invoices Manually

This feature is intended for use with VISUAL Automotive. EDI users may want to reprint invoices for which they have generated ASNs.

Although VISUAL automatically generates Invoice IDs for consolidated orders, you can still use the Invoice Forms window to generate an invoice.

If VISUAL generates the ASN number but for some reason is not able to print the invoice (for example, a printer malfunction), Invoice Entry (Vminvgen.exe) has an option that allows you to print an invoice for the ASN, using the ASN number as the invoice number.

To print ASN Invoices:

  1. From the main VISUAL menu, select Invoice Forms from the Sales menu.

The Invoice Forms window appears.

  1. From the File menu, select Print One ASN Invoice.

The Pack lists-By Invoice ID dialog box appears. 

A list of all consolidated pack lists, by Invoice ID, appears in the table.

  1. Select the Invoice you want to print, and click the Ok button.

The Print Options dialog box appears.

  1. Select Print Options and click the Ok button to print the invoice.
