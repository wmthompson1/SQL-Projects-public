Changing Sub-Menus

To change program setting in a Sub-menu Menu:

  1. Open the Maintain Menu window and select the Heading that contains the Sub-menu you want to change.

  2. In the Menu Entries section, select the Sub-menu name.

  3. In the Sub-menu section, make the changes you want to the Sub-menu programs.

  4. Click the Save button.

When you have finished making changes, click the Close button.
