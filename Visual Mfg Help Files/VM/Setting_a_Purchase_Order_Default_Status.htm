Setting a Purchase Order Default Status

You can set the default status of purchase orders, either Firmed or Released.

  1. From the Options menu, select Preferences.

The Preferences dialog box appears.

  1. Select Firm if you want VISUAL to create purchase orders with a Firmed status. Firmed purchase orders exist for planning purposes, but do not have any work performed on them.

Select Released if you want VISUAL to create purchase orders with a Released status. Released purchase orders are orders that you are set to begin producing on the scheduled start date.

  1. Click the Ok button to commit this preference.
