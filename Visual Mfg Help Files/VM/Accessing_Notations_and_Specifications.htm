Accessing Notations and Specifications

Edit Notations and Specifications in pop-up text windows. You can access this text window in the following ways:

Push Buttons

Some applications that access notes or specs have one or more push buttons near the bottom of the window to access the notes. For example, Accounts Receivable Invoice Entry has Cust Notes, and Invoice Notes buttons.

Notations

Most applications that access notations have a Notes menu. This menu contains options to view the different types of notes and specs available. This menu also contains options for controlling the display of notes and specs.
