Inventory Transactions Report

Use the Inventory Transactions report to review the inventory transactions for the month.

  1. From the Inventory menu, select Inventory Transaction Entry. The Inventory Transaction Entry window appears.

  2. From the File menu, select Print. The Print Inventory Transactions window appears.

  3. In the Print Inventory Transactions window, click the Site ID(s) arrow and select the sites to use.

  4. In the Starting Date field, enter the first day of the period you would like to close. In the Ending Date field, enter the last day of the period you would like to close.

  5. In the Type box, use the check boxes to determine the types of inventory transactions you would like to view.

  6. Select Receipts, Issues/Shipments, and Adjustments to view all inventory transactions.

  7. In the Sequence box, select a sort order for your report. Your choices are:

    • Transaction ID

    • Part ID

    • Customer Order ID

    • Purchase Order ID

    • Work Order ID

    • Warehouse/Location

    • Warehouse/Location/Part ID

For projects, you may want to sort the report by Warehouse/Location to view transactions by project (since project warehouse IDs are identical to project IDs).

  1. Select the Include Obsolete Parts to include parts that you may have recently put out of service.

  2. Click Ok. The report appears.
