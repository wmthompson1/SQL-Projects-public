Printing Open Requisition Reports

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site to view. If you are licensed to use a single site, this field is unavailable.

  2. Select Print Open Requisitions... from the File menu.

The Print Open Requisitions dialog box appears.

  1. Enter an outer date range for the report into the Dated through field.

VISUAL will print all open purchase requisitions with a desired receive date up to and including this date. Any open purchase requisitions that have a later date do not print.

  1. Select the appropriate Vendors radio button.

Select the Print Current Vendor option button if you want to include only purchase requisitions for the selected vendor. Click the Print All Vendors option button to print open requisitions for all vendors.

When printing this report without a requisition in the window, the "Print Current Vendor" option is not available

  1. Select a sequence for the open requisition report:

You can sequence the report by Order ID, or Part ID.

If you sequence by Order ID or Vendor ID, one line prints for each requisition, showing vendor, date, and shipping information. Additionally, one line prints for each requisition line item, showing Part/Service ID, Description, Quantity, and Unit of Measure. If you sequence by Vendor ID, Order ID lines appear by Vendor.

If you sequence by Part ID, a section prints for each Part ID or Service ID, showing ID and description.

  1. Select the Delivery Schedules box to include delivery schedule lines in the report.

  2. Choose the output destination by clicking on the arrow next to the output field. Select one of the following:

Print - Select Print to output the file to a printer.

View - Select View to view the report before printing.

File - Select File to save the information to a text file.

E-Mail - Select E-Mail to send the report in a Rich Text Format (.RTF) through electronic mail.

  1. Click the Ok button to output the report to the specified destination.

If e-mailing the report, the system attaches the file to a Microsoft Outlook e-mail. For more information on addressing and sending the e-mail, refer to your Microsoft Outlook user documentation. If you are sending a PDF, VISUAL starts the distiller, converts the document to PDF, and attaches it to the e-mail.
