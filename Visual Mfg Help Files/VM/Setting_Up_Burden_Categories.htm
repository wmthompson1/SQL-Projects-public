Setting Up Burden Categories

Burden categories are groupings of similar burdens. Assigning burden categories helps you better assign specific sets of burdens for individual and multiple projects.

To set up Burden Categories:

  1. From the Project menu, select G/L Burden Mapping Maintenance.

  2. From the Maintain, select Burden Categories....

The Burden Category Maintenance window appears.

  1. Click the Insert button.

A new row appears highlighted.

  1. Click in the ID column and enter an ID for this burden category.

  2. Click in the Description column and enter a description for this burden category.

  3. Click in the Apply Type column and select the type of burden you want to apply from the list that appears when you click the arrow.

Direct Overhead - Select Direct Overhead if the burden covers direct overhead costs.

Direct G & A - Select Direct G & A to assign a type of direct general and administrative costs.

COM Overhead - Select COM Overhead if the burden covers overhead related to cost of money.

COM G & A - Select COM G & A if the burden covers general and administrative costs related to the cost of money.

  1. Click in the Cat Pool column and select the Cat Pool you want to use for this Burden Category.

  2. Click in the Cat Base column and select the Cat Base you want to use for this Burden Category.

  3. Click the Save button.

  4. When you have finished entering your burden categories, click the Close button.
