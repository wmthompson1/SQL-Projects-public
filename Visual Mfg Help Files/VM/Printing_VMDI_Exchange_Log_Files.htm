Printing VMDI Exchange Log Files

The log files provide detailed information about your most recent VMDI Exchange sessions. Log files include:

  • VMDI Exchange Session ID

  • Session Date and Time

  • User who initiated the session’s Imported Transactions

  • Key, Version and description of the VMDI Generate Layout

For Key PLN

  • Customer ID

  • Forecast ID

  • Forecast Date

For Key CPO

  • Customer ID

  • Order ID

  • Status: INSERTED or UPDATED

For Key CSH

  • Customer ID

  • Payment ID

  • Imported Transactions

  • Key, Version and description of the VMDI Generate Layout

  • Total number of transactions/documents in the imported file

For Key INV

  • Customer ID and Invoice Numbers exported

For Key ASN

  • ASN #, BOL # and Packlist #’s that were referenced on it

For Key ACK

  • Customer Number being acknowledged

  • Status Original acknowledgement or Change Acknowledgement

For Key VPO

  • Vendor ID and Vendor Purchase Order Number exported

To print log file reports:

  1. Select Print Logs from the VMDI Exchange File menu.

The Print Log File dialog box appears.

  1. Click the Log File button to locate the Vmdixchg.log file.

Select a print output.

View - Select this option if you want to view the log file report before printing it.

File - Select this option to output the log file report to a specified directory.

Print - Select this option if you want to print the log file report. Specify print settings and VISUAL prints the report to the appropriate printer

E-mail - Select this option if you want to send the report through electronic mail. VISUAL allows you to use your default mail application and you have access to all of your personal address books.

  1. Click the Ok button to output the log file report to the selected print destination.
