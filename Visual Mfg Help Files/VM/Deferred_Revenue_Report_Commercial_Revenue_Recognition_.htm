Deferred Revenue Report (Commercial Revenue Recognition)

The Deferred Revenue report shows revenue recognition transactions for the period.

  1. From the Receivables menu, select Progress Billing Entry. The Progress Billing Entry window appears.

  2. In the Progress Billing Entry window, click the Site ID arrow and select the site to use.

  3. From the File menu, select Print Deferred Revenue Status Report.

  4. In the Starting Date field, enter the first day of the period you would like to close. In the Ending Date field, enter the last day of the period you would like to close.

  5. In the Customers box, select All Customers to view information for all customers. This provides a complete view of the period.

  6. In the Orders box, select All Orders to view information for all orders. This provides a complete view for the period.

  7. In the Sequence box, select a sort order for the report. Your choices are by Order ID or by Customer.

  8. In the Type box, select Detail. The detailed report provides more granularity to the information.

  9. Click Ok. The report outputs.
