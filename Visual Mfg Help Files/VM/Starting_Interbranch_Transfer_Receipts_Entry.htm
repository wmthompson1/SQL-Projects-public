Starting Interbranch Transfer Receipts Entry

From the Inventory menu, select Interbranch Transfer Receipts Entry.

The Interbranch Transfer Receipts Entry window appears with no information in it.

The Interbranch Transfer Receipts Entry window contains both entry fields and buttons.

The window consists of two main sections: a header section in which you enter general shipment receipts information, and a line item table in which you enter information specific to the individual receipt of transfers.

The line item table contains the following columns:

IBT Line Number - The line number of the IBT line as it appears in the original IBT order.

Reference - Any referential text regarding the material you are transferring.

Quantity Ordered - The quantity of the material you are transferring.

Quantity Shipped - The quantity shipped to this transfer.

Total Quantity Received - The total quantity to date that you have received.

Quantity Received - The portion of the Quantity Ordered that you want to receive. This number can be more or less than the Quantity Ordered. If so, VISUAL prompts you to verify the amount over or short.

Length, Width, Height – If you are transferring piece tracked parts, the dimensions of the part are inserted. You cannot edit this field.

Dimensions UM – If you are transferring piece tracked parts, the dimension unit of measure is inserted. You cannot edit this field.

Piece Count – If you are transferring piece tracked parts, the number of pieces specified in Inter Branch Transfer Entry is inserted. You cannot edit this field.

Total Pieces Shipped – If you are transferring piece tracked parts, the number of pieces previously shipped is inserted. You cannot edit this field.

# Pieces to Receive – Specify the number of pieces to receive. After you specify the number of pieces, the quantity of the shipment is calculated and inserted in the Quantity Received field.

Total Pieces Received – If you are transferring piece tracked parts, the number of pieces previously received is inserted. You cannot edit this field.

Total Quantity Allocated - The portion of the Quantity Ordered that you have received up to this time.

Total Quantity Fulfilled - The portion of the allocated quantity you have fulfilled to date.

U/M - The unit of measure of the part.

Last Receive Date - The date on which you last received a IBT shipment quantity into this warehouse location.

Location ID - The Location ID in the To Warehouse into which you are receiving this shipment. You enter this Location ID in IBT Entry and can change it in IBT Shipping Entry. You cannot change the Location ID at this point.

Warehouse ID - The ID of the warehouse into which you are currently receiving the IBT shipment.

Receive On - The date on which you received this IBT Shipment.

Close Short - VISUAL selects this check box if you have received only a partial quantity of the line and opted to close the line with a balance, or "short."

For Project and A&D users

The following required fields also appear:

WBS Code - If this transfer refers to a project, the WBS Code appears.

Dept ID - If this transfer refers to a project, the default Department ID appears for the WBS Code.

Proj Sub ID Ref - If this transfer refers to a project, the default Sub ID Ref for the project appears.

Proj Seq No Ref - If this transfer refers to a project, the default Seq No Ref for the project appears.

You cannot save project related Interbranch Transfers without the relevant project information.
