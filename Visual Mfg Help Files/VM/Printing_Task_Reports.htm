Printing Task Reports

The Tasks report allows you to view tasks of all statuses for the User currently signed into the VE database. To view tasks for another user, you must sign out and then sign in again using the appropriate User ID and password.

  1. To print a task report, click the Print button or select Print Tasks from the File menu.

The Print Report dialog box appears.

  1. Specify a print output destination:

Print - Select Print to output the file to a printer.

View - Select View to view the report before printing.

File - Select File to save the information to a text file.

E-Mail - Select this option to send the document through electronic mail. If you want to convert this document to PDF (Portable Document Format), select the PDF Format check box.

  1. Click the Ok button to output the report to the specified destination.

If you selected E-Mail, the system attaches the file to a Microsoft Outlook e-mail. For more information on addressing and sending the e-mail, refer to your Microsoft Outlook user documentation. If you are sending a PDF, VISUAL starts the distiller, converts the document to PDF, and attaches it to the e-mail.
