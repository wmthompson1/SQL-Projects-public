Installing a VISUAL Service

Before you install a service, make sure you meet the prerequisites. Then, use procedures described in this section to install services.

This table shows the services you can install to run VISUAL processes:

Service EXE

What does this service do?

How many times can I install this service?

VRDTASVC.EXE

Loads data into the Reporting database tables

Once per database

VMSCHSVC.EXE

Runs a schedule

Once per site

VMDBRSVC.EXE

Runs a schedule in a DBR-enabled database

Once per site

VRPTSVC.EXE

Prints Work Order Travellers

Once per database

VMCSTSVC.EXE

Runs costing

Once per site

VMMRPSVC.EXE
