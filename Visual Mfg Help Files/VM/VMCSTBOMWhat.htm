What is the Costed Bill of Material Report?

The Costed Bill of Material report is similar to the Bill of Material report, but only prints BOMs for engineering masters, and provides additional cost information.

For each part, this information is displayed in the report for each material requirement on its engineering master:

  • Part ID and Description

  • Quantity Required

  • Unit Cost

  • Total Cost = Unit Cost x Quantity Required

  • BOM Level

  • Dimensional Information.

If any parts included in the report are piece tracked, the number of pieces, the length, width, and height dimensions, and the dimension unit of measure appear.

  • Additionally, the lot size and total production hour for the part are shown, along with totals for Unit Cost and Total Cost.
