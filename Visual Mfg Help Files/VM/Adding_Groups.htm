Adding Groups

You can add new groups to your Personal Menu. You can use this feature to combine in one group the applications you use most.

  1. With the customize dialog box open, click the Add New Group button.

The Edit Group dialog box appears.

  1. In the Group Caption field, enter a caption for the group.

  2. If you want to select a different icon to appear next to the group name, click the Icon arrow and select the icon you want to use.

  3. Click the Ok button.

  4. Click the Apply button.

  5. Add commands to your new group by selecting the commands you want in the right pane and clicking the Add Selection button.

  6. When you have finished setting up your new group, click the OK button.

The new group bar is added to your Personal Menu.

Removing Groups

To remove groups from your Personal Menu:

  1. In the Personal Menu, right-click the group you want to remove and select Customize from the right-click menu.

  2. Click the Remove Group button.

  3. The group is removed from the Personal Menu.

  4. Click the OK button.

The customize dialog box closes.
