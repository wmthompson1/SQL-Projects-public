Changing Ship to Addresses

You can change Ship To addresses using Service Dispatch Entry.

  1. Click the Ship To Addr button.

The Ship To Addresses dialog box for the vendor appears.

The table at the top of the dialog lists the alternate addresses, if any, for the vendor.

  1. To use an alternate address for this purchase order, select the line of the appropriate address and click the Use Selected Address button.

To change back to the vendor’s main address, click the Use Vendor Address button.

The dialog box closes. The address you selected now appears at the top of the Service Dispatch Entry window.

Editing Ship To Addresses

You can also use the dialog box to add, modify, and delete alternate Ship To addresses.

Adding Ship To Addresses

  1. Click the Insert button.

A new line appears in the table.

  1. Fill in the address information in the fields at the bottom of the dialog box.

  2. Click the Save button to save the new address.

Deleting Ship To Addresses

  1. Select the address to delete and click the Delete button.

  2. Click the Save button to permanently delete the address.

Modifying Ship To Addresses

  1. Select the address to modify.

  2. Modify the fields that appear at the bottom of the dialog box, as necessary.

  3. Click the Save button to save the modifications.
