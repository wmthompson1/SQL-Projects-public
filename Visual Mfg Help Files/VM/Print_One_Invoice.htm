Print One Invoice

Use the Print One Invoice window to view shipments eligible to be invoiced.

  1. From the Receivables menu, select Invoice Forms. The Invoice Forms window appears.

  2. In the Invoice Forms window, click the Site ID arrow and select the site to use.

  3. From the File menu, select Print One Invoice. The Print One Invoice window appears.

The window shows orders that have been shipped but have not yet been invoiced. Review your records to see if an invoice should be issued.
