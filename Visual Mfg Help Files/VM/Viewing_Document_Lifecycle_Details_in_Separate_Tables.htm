Viewing Document Lifecycle Details in Separate Tables

You can view document details in separate, floating tables. When you use a separate table, you can sort the columns and you can export the table to Excel. You cannot customize the order or size of columns in floating tables.

To view document details as a separate table:

  1. Select Info, Lifecycle Document Viewer.

  2. Click the Show Detail button.

  3. Click the Plus icon in the table that contains the detail information to view. The information is displayed in a separate table.

To export the information to Microsoft Excel, select File, Send to Microsoft Excel.

To close the separate table, select File, Exit.
