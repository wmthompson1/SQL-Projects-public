Entering License Keys

The license key you need depends on the version of Infor VISUAL you purchased. License keys activate certain components throughout the system that allow the user to take full advantage of special functionality. If you have any questions about license keys or expanding the functionality of your Infor VISUAL system, contact your Infor Global Solutions Partner for more information.

What is a License Key?

A license number is like a serial number. It allows licensed users of Infor VISUAL to enjoy all the benefits of the modules they have purchased.

Licensing is mandatory. Infor VISUAL does not operate without valid licenses, except in the case of a 30-day temporary license. To enter a license number for your Infor VISUAL system, or any of its components:

  1. From the Admin Menu, select Enter New License Key.

The Enter License Key dialog box appears.

  1. From the Product options, select the appropriate option button.

  2. In the License Key data field, enter the supplied license key to activate the product.

  3. Click the Ok button to commit the license number.

If VISUAL accepts the license you provided, you return to the main menu. If VISUAL rejects the license number you provided, a warning message appears.
