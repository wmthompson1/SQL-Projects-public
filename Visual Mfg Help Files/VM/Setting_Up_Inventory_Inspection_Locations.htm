Setting Up Inventory Inspection Locations

Because you may receive Universally Planned material into several locations within several warehouses, you may want to set up your project warehouses as Part and Warehouse inspection points.

To set up inspection locations:

  1. From the main VISUAL menu, select Part Maintenance from the Inventory menu.

  2. Click the Part ID button and select the part you want to set up.

  3. From the Maintain menu, select Warehouse Locations.

The Warehouse Locations for Part dialog box appears.

All of the warehouse locations you have previously set up for the part you selected appear.

Click the row header for the warehouse whose attributes you want to change.

The row appears highlighted.

  1. If you want all parts added to this location to be inspected, click the Set part inspect button.

VISUAL selects the check box in the Part Inspect column. VISUAL also automatically changes the status of the parts within this location to On Hold until you release them after your inspection.

  1. If you want to designate the location you selected as an Inspection Location, select the Set whs inspect button.

VISUAL selects the check box in the Whs Inspect column.

  1. Click the Save button.

  2. When you have finished making changes to your part warehouse locations, click the Close button.
