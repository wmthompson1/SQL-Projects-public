Placing a Purchase Order for a Piece Tracked Part

Use Purchase Order Entry to order a piece-tracked part. Unlike ordinary parts, which you order by quantity, you order piece-tracked parts by number of pieces. For piece-tracked parts, the number of pieces multiplied by the dimensions equals the quantity. VISUAL automatically calculates the quantity for you based on the Purchase U/M.

  1. Select Purchase Order Entry from the Purchasing menu of the VISUAL Main window.

The Purchase Order Entry window appears.

  1. If you are licensed to use multiples sites, click the Site ID arrow and select the site to use. If you are licensed to use a single site, this field is unavailable.

  2. In the Vendor ID field, enter the Vendor ID for the vendor from whom you are ordering the part.

  3. Enter any other necessary purchase order information. See Purchase Order Entry for more information on creating new purchase orders.

  4. Click the Insert button.

VISUAL adds a new line to the line item table.

  1. In the Part ID column, enter the Part ID for the dimensional part you are ordering.

When you tab out of the Part ID column, VISUAL fills in the dimensional unit of measure for the part and enters a value of 1 for any dimension not required for the part.

  1. In the #Pieces column, enter the number of pieces you are ordering.

  2. In the Length, Width, and Height dimensions columns, enter the required dimensions.

  3. Click the Save button.

VISUAL computes the quantity for the order, based on the stock unit of measure for the part, the required dimensions, and the required number of pieces.

In the example above, the order requires ten 3x5 pieces of the part, which equals 150 square feet. If you stock the part in 10x10 sheets, you need to order two 10x10 sheets (200 square feet) to supply the required quantity. VISUAL automatically calculates the required quantity for the order.
