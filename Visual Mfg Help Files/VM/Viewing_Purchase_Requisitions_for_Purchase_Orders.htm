Viewing Purchase Requisitions for Purchase Orders

After creating a purchase order from a requisition, you can view details of the requisition/purchase order relationship using this feature.

  1. Select the purchase order you want to view purchase requisition information for and select Show Purchase Order Requisitions from the Info menu.

The Show PO Requisition dialog box appears, displaying the specified purchase order in both the Starting and Ending fields.

This dialog box allows you to specify a range of requisitions, a starting ID and an Ending ID.

  1. Click the Starting PO ID button to choose the starting purchase order.

Click the Ending PO ID button to choose the ending purchase order.

  1. Click the Ok button to retrieve results for the range.

  2. Review the table for details of the requisition/purchase order relationship. The table includes the following fields:

Purchase Order ID - The ID of the purchase order. If there is more than one line in the order, the Order ID appears in multiple rows.

Purchase Order Line # - The line in the purchase order.

PO Vendor ID - The Vendor ID from the purchase order.

PO Desired Receive Date - The desired receive date from the purchase order.

Requisition ID - The ID of the requisition used to create the purchase order.

Requisition Line # - The requisition line the purchase order is linked to.

Requisition Quantity - The quantity of the requisition line.

Purchase Order Quantity - The quantity of the purchase order line.

Requisition Status - The current status of the requisition line. This is not necessarily the overall status of the requisition, which supports a header status as well as a status per line.

Purchase Order Status - The current status of the purchase order.

Requisition Buyer - The Buyer specified on the requisition. The Requisition Buyer and Purchase Order Buyer must be the same to create a purchase order from a requisition.

Purchase Order Buyer - The Buyer specified on the purchase order. To create a purchase order from a requisition, the requisition and the purchase order must share a common Buyer.

  1. To view a requisition order in Purchase Requisition Entry, select a line and click the Requisition button. In addition, you can double-click a line.
