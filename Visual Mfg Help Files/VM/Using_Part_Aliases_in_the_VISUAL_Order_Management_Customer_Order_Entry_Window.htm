Using Part Aliases in the VISUAL Order Management Customer Order Entry Window

Enter Part Alias Types in Application Global Maintenance under the Order Management tab. Then, assign parts Alias IDs in Part Maintenance. See Application Global Maintenance and Part Maintenance for more information.

Part aliases are alternate names, abbreviations, or other designations for a Part ID.

Order entry clerks may refer to these alternative part names when dealing with customers instead of using cryptic Part IDs. The ID you use in stocking the item may not convey what the part is, especially if it is an unrecognizable alphanumeric string. An alias could be the actual part name.

For example, Part ID CAM-XX1234 could equal Part Alias ID "X11 Camera."
