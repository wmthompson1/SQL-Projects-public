Showing Work Order Audit Information

Use the Show Work Order Audit option to display the work order audit information for all work orders in the current schedule ID. You must have activated the Work Order Audit function in the Concurrent Scheduler to view work order audits.

  1. Select Show Work Order Audit from the Info menu.

The Work Order Audit dialog box appears.

  1. You can view detailed information on an individual work order by selecting the appropriate work order and clicking on the Show Detail button.

  2. Click the Close button to close the dialog box.
