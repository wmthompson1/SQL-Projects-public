Window Table Buttons

When a table is displayed in the body of a window, a Table toolbar is also displayed. The buttons on Table toolbars have ToolTips that are displayed when you rest the cursor on the button. These toolbar buttons are typically displayed on the Table toolbar:

Insert – Click the Insert button to insert a line into the line item table.

Delete – Select a row, and then click the Delete button to remove a line from the line item table.

Repeat Row Information – Click this button to make a copy of a line. To use this button, select the line to copy, then click the Repeat Row Information button. The system inserts a duplicate line at the end of the line item table.
