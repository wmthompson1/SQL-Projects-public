Specifying Subordinate Departments

You must be licensed to use Project/A&D functionality to complete this procedure.

For reporting purposes, you may find it useful to have VISUAL report a project’s budget and actual costing figures to a single department.

To set up subordinate departments:

  1. Complete one of these tasks:

  • Select Admin, Application Global Maintenance. Then, select Maintain, Accounting Tables, Department Table.

  • If you are licensed to use Project/A&D functionality, select Project, G/L Burden Mapping Maintenance. Then, select Maintain, Departments.

  1. Select the parent Department ID and click the Subordinate Departments button.

  2. Click Insert.

  3. Click the Department ID browse button and select the subordinate department. VISUAL reports costs from subordinate departments to the parent department.

  4. Click Save.

  5. Click Close.
