Using the APS Import Utility

To use the APS Import Utility:

  1. Start the APS Import Utility by using Windows Explorer or My Computer to navigate to the directory where your executables are installed, and double-click the VMAPSIMP.EXE program name.

  2. Select the components you want to import and enter the full path name where your import files reside.

  3. Click the Start toolbar button.

If you have entered all of the paths correctly, VISUAL Imports the information from your import files into your database.

If you are licensed to use APS functionality, use the APS Import Utility window to import files that from your advanced planning and scheduling system into VISUAL.
