Viewing Warehouse Locations for Parts

You can view a complete list of all of the locations where the selected part is allowed to be stored. This includes Warehouse and Location IDs and descriptions, the status and quantity of the part in each location and other information.

  1. Select Inventory, Part Maintenance.

  2. If you are licensed to use multiple sites, click the Site ID arrow and select the site to use. If you are licensed to use a single site, this field is unavailable.

  3. In the Part ID field, specify the ID of the part to view.

  4. Select Info, Locations for This Part.
