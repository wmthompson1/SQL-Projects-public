Generating IDs in the Cust/Accts to Process table in the Synchronize Customer/Account Utility

If you are using the Synchronize Customer/Account Utility to create new records, use the Preferences dialog to specify how to generate the IDs for the new records.

To set the default generation of VE IDs and CRM IDs:

  1. Navigate to the Infor VISUAL installation directory and double-click VMCRMSNC.EXE. If the default VISUAL installation directory is used, the Synchronize Customer/Account Utility launches from C:\Infor\VISUAL\VISUAL MFG.

  2. Select Options, Preferences.

  3. In the New Customers section, specify how to generate the ID when you create a new customer to link to a CRM account. Click one of these options:

Next Number Generation - Click this option to use the next number generation format specified for customers in VISUAL Customer Maintenance.

Account ID - Click this option to use the CRM account ID as the VISUAL Customer ID.

  1. In the New Accounts section, specify how to generate the ID when you create a new account to link to a VISUAL customer. Click one of these options:

Next Number Generation - Click this option to use the next system-generated number for the account.

Customer ID - Click this option to use the VISUAL Customer ID as the CRM Account ID.

  1. Click Ok to confirm your ID Generation preferences.
