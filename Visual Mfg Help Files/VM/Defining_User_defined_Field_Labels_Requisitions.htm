Defining User-defined Field Labels

Use this feature to assign values to the 10 user-defined labels allotted to each requisition. As these fields are user-defined, entry is optional, but may be of some use by providing you extra space to add miscellaneous text and comments to a requisition.

  1. From the Edit menu, select User Defined Fields or click the User Defined Fields button on the Main toolbar.

The User Defined Field Labels dialog box appears.

  1. From the User defined field Layout ID list box, select the appropriate label set.

    Enter Label Sets in Vendor Maintenance, Part Maintenance, and Customer Maintenance.

VISUAL displays the appropriate fields.

  1. Enter appropriate text into the adjacent fields.

  2. Click the Ok button to save the values to the requisition.
