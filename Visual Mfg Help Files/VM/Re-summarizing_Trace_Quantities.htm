Re-summarizing Trace Quantities

If quantities in the trace table need to be recalculated for issues with historical trace information, this is the option you use. The option is only active when one user is logged into the VISUAL database.

Only the SYSADM user can perform this function.

For assistance with this administrative task contact Infor Support at https://concierge.infor.com.

To re-summarize trace quantities:

  1. Select Inventory, Part Trace Maintenance.

  2. Select either a trace profile or a specific trace ID .

  3. Select Maintain, Re-summarize Trace Qty.

  4. The current part or trace ID is displayed. To re-summarize only the selected part, click Ok. To re-summarize all parts, clear the information in the dialog and then click Ok.
