Payment Register

To review the accounts payable transactions for the month, run the Payment Register.

  1. From the Payables menu, select Payment Entry. The Payment Entry window appears.

  2. From the File menu, select Print Payment Register. The Print Payment Register window appears.

  3. In the Print Payment Register dialog box, click the Site ID(s) arrow and select the sites to use.

  4. In the Starting Date field, enter the first day of the period you would like to close. In the Ending Date field, enter the last day of the period you would like to close.

  5. In the Sort By box, select the sort order for the report. You choices are:

  • Control No

  • Payment No

  • Payment Batch No

  1. Select the types of payments you would like to view. Your choices are:

  • Cleared Payments

  • Uncleared Payments

  • Void Payments

We recommend selecting both Cleared and Uncleared payments for a complete view of the payments for the month.

  1. Click Ok. The report outputs.
