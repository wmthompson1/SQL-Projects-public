Viewing Dependent Demand for a Part

Click anywhere on the Dependent Demand line. The Dependent Demand Details for Part dialog box appears. You cannot edit any of the values in these columns.

The line item table contains the following columns:

Order ID - The Work Order ID of the order for which the part is a material. 

Sub ID - The Sub ID of the work Order.

Operation # - The Operation # with which the material is associated.

Piece # - The place where the part appears in the work order sequence.

Status - The status of the order.

Order Quantity - The quantity of the order line.

Quantity Due - The quantity that you must produce to deliver to the customer.

Want Date - The date on which the customer wants the order.

Warehouse ID - The ID of the warehouse from which the order originates.
