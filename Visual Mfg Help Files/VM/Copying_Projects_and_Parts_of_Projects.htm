Copying Projects and Parts of Projects

There are several methods you can use to copy objects within Infor VISUAL for Project-Based Manufacturers :

The Project window supports the drag and drop functions. You can copy projects, work orders, and masters (or parts of them) by clicking on the object you want to copy and dragging it to its new location. VISUAL copies object information from the source to the new location.

If you are dragging information between multiple projects, you must open and arrange side by side the appropriate projects or masters. For more information on working in multiple windows, refer to your Infor VISUAL documentation.

  • Use the Copy From option on the Edit menu. Select the object to which you want to copy and from the Edit menu, select the Copy From option. Select the Project or Master you want to copy. VISUAL copies the appropriate information to your project and attaches it to the object you selected.

  • Use the Copy and Paste options on the Edit menu. Click the object you want to copy and from the Edit menu, select Copy. Click the object to which you want to paste the information and from the Edit menu, select Paste. VISUAL copies the appropriate information to your project and attaches it to the object you selected.

  • A progress indicator appears and if the operation was not successful or the paste types were incompatible, an error dialog box appears.

  • If you try to drag a project, work order, or master to another window in which there are identical sequence numbers, a message box appears warning you that pasting would cause a duplicate key for an operation.

It is important to remember that when VISUAL performs a copy and paste function, it copies operations and materials but does not copy quantities and other header fields, such as engineering drawings or revision numbers. VISUAL does not copy labor tickets or materials issues/ links to the new project.
