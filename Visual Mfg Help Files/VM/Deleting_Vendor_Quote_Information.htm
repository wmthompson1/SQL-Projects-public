Deleting Vendor Quote Information

To delete the pricing that a vendor provided for an RFQ:

  1. Select Purchasing, Vendor RFQ Entry.

  2. If you are licensed to use multiple sites, click the Site ID arrow and select the site where the RFQ was created. If you are licensed to use a single site, this field is unavailable.

  3. Click the Vendor ID drop-down button and select the vendor whose quote you are deleting.

  4. Click the Delete Current Vendor button.

  5. Click Save.
