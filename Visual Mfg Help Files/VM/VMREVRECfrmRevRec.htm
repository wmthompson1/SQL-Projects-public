Using Project Revenue Recognition

To use the Project Revenue Recognition window:

  1. To start the Project Revenue Recognition window, select it from the Project menu of VISUAL's main window.

  2. Click the Site ID arrow and select a site to use.

  3. Click the Transaction ID arrow and select the transaction you want to view. You can also select a Project ID.

The revenue totals appear in the tab and the breakdowns appear in the table.

If you want to view the figures in a different currency, click the Currency ID and select the currency in which you want to view the figures.
