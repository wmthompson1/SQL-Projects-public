About the Sales Commission Report Window

Use the Sales Commission Report window to access the sales commissions reports.

Starting the Sales Commission Report Window

Select Sales, Sales Commission Report.

Tasks

This table shows the tasks that you can complete in the Sales Commission Report window:

Task

Description

Printing Commissions Reports

Use this report to print commissions information for all Firm, Released, and Closed customer orders in a specified date range.

Printing Commissions Actuals

Use this report to print commissions for each sales rep based on shipped orders.

Printing Commissions Distribution

Use this report to view how projected commissions are broken down for each Sales Rep or for each order.

Fields

There are no fields in this window. Click a button to open the dialog for the report.
