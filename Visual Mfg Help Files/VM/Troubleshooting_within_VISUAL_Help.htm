Troubleshooting VISUAL Help and User Guides

Before you contact Technical Support with problems with your Help file:

  • Make sure you have the latest Online Help file from Infor Support Portal. See Updating Help and User Guides.

  • Make sure you are running the online help file on a client machine. You cannot run the online help file from a network location. In order for the VISUAL application to run your Help file, it must reside in your application directory.

  • Make sure the user guide PDFs reside in your VISUAL application directory.
