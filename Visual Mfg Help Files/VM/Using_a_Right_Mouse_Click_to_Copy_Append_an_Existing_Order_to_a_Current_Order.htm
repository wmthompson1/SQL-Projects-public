Using a Right Mouse Click to Copy/Append an Existing Order to a Current Order

You can copy/append an entire existing order or specific lines from existing orders to a current order by right-clicking the whole order or an order line in the Customer Order History child window.

  1. In the Order Entry child window, open the order to which you want to copy or append a line.

  2. If not already open, open the Customer Order History child window. Tile or cascade the two windows so you can see the VISUAL add the order lines.

  3. In the Customer Order History window, double-click the Customer icon.

All existing orders for the customer appear.

  1. Either highlight the order you want to copy from (this will copy/append the whole order), or click the plus box next to the order and highlight the specific line from which you want to copy.

  2. Highlight the order or order line you want to copy from and click the right mouse button.

If you highlight an order, an option box appears. 

If you highlight a single order line, another option appears.

  1. Select Copy/Append or Append line as necessary.

VISUAL copies the order/line information to the customer order.

  1. Maximize the Order Entry child window to view the copied/appended order information.

  2. Edit this new customer order information as necessary and then click the Save button on the Customer Order Main toolbar.
