Setting Up Location/Part Associations

You can set up associations between warehouse locations and specific part IDs using Warehouse Maintenance. The association indicates that the part can be stored in, received to, or issued from the location. Each part must have at least one assigned location, if shipping, receiving, or inventory transactions are to be done for the part.

You can also make this assignment in Part Maintenance.

Using the Locations/Parts command allows you to:

  • Review and modify all the locations for one part.

  • See if the status of the parts in location/parts is available, unavailable, or on hold.

  • Lock the part for that particular warehouse location.

  • Place particular parts on hold. This status prevents these parts from being issued or transferred.
