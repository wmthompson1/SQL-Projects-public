Creating Thermal Label Groups

Label groups allow you to combine individual labels into one group.

  1. From the Maintain menu, select Label Groups.

The Label Groups dialog box appears. 

  1. Enter an ID for the Label Group ID in the Label Group ID field, or click the Label Group ID button to browse the LABEL_GROUP table and choose an existing ID.

  2. From the Label Group Type list box, select a group type. Automotive users may want to use the PART_SHIPPING table.

  3. Enter a description for the label group in the Description field.

  4. Click the Insert button to begin adding individual labels to the group.

  5. Enter information into the following fields:

<Label ID> - Double-click the <Label ID> column header to browse the LABEL_FORMAT table and select a label to include in this group.

Label Printer - VISUAL auto-fills this field with the Label IDs default thermal printer. See Creating Individual Thermal Labels for more information on establishing thermal label printer defaults.

Copies - VISUAL auto-fills this field with the label’s default copy number.

  1. Click the Save button to save the label group.
