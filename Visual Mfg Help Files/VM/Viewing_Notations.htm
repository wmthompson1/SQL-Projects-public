Viewing Notations

You can view and edit two types of notations from the Manufacturing Window:

  • Part Notations are associated with the Part ID of the master/work order — the same set of notations are seen for every master/work order for that Part ID.

  • Work Order/Master Notations are associated with the current master/work order.

Access Notation windows through the File menu. You can leave these windows open and display the notations for the current Work Order and Part ID.
