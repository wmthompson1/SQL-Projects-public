The file does not exist. If you want to use User Defined Help information within this help file, you must use the default text files for the current function.

To create the appropriate text file:

Click one of the below five buttons (the name of the UDF file is the button label) to continue.

A Notepad message box appears prompting you to create the file. Click Yes to create the file in the appropriate location with the correct file name.

If you click No on the message box, Notepad opens with an Untitled filename. Do not save this file.
