Reopening Accounting Periods (Global Financials License)

You can reopen a closed accounting period. For example, if you need to post an adjustment to a closed period, you can reopen the period to post the adjustment transactions.

You cannot reopen a locked period in the Close/Reopen Period window. To reopen a locked period, see Financials Application Global Maintenance.

  1. Select Accounting, Close/Reopen Period.

  2. In the Site ID field, select the ID of the site whose period you are reopening.

  3. In the End Date of Beginning Acct Period field, specify a date in the period that you are reopening. The date is adjusted to the last date of the period. After you select a date, the ID of the fiscal period and the current status of the period are displayed.

  4. Click Reopen Period.
