Selecting Customer Primary Warehouses

With the appropriate customer in the Customer Maintenance window, click the Order Management tab.

  1. Click the Primary Warehouse button to view a list of warehouses for the customer.

The Warehouses dialog box appears.

  1. Highlight the warehouse that you want to be the customer’s default primary warehouse.

  2. Click the Ok button.

VISUAL sets this as the customer’s default primary warehouse.
