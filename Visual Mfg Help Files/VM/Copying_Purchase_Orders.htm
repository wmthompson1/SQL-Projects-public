Copying Purchase Orders

You can copy information from saved purchase orders to new purchase orders. Use this option to avoid repetitive entry of purchase order information.

If you are licensed to use multiple sites, you can copy purchase orders within the same site only. You cannot copy a purchase order from one site into a second site.

To use the copy function:

  1. If you are licensed to use multiple sites, click the Site ID arrow and select the site to use. If you are licensed to use a single site, this field is unavailable

  2. With the purchase order from which you want to copy open, select Copy Purchase Order from the Edit menu.

The Copy Purchase Order dialog box appears.

The Order ID of the order you are copying appears in the Copy from PO ID field.

  1. Enter a Copy to PO ID. This will be the Order ID of the new order. If you have set auto-numbering preferences, VISUAL uses the next sequential number.

  2. Select the Copy PO Line Items box if you want VISUAL to copy the line items from the current purchase order to the new purchase order.

  3. Select the Copy PO Line Item Links box if you want VISUAL to copy demand link information from the current purchase order to the new purchase order.

  4. Select the Copy Delivery Schedule box if you want VISUAL to copy delivery schedule information from the current purchase order to the new purchase order.

  5. Select the Close Order box if you want to close the purchase order after VISUAL copies it.

  6. Click the Copy button.

VISUAL copies the Purchase Order.

When VISUAL has completed copying the purchase order, it asks you if you want to view the new purchase order.

Click the Yes button to view the new purchase order.

If you selected the Close order box, the status of the purchase order from which you created the new customer order is now Closed.

The Create Internal Order check box, Use Cust VAT Tax check box, Internal Customer ID field, and SO Tax Shipto ID field are used for buying and selling between accounting entities. See Copying Internal Purchase Orders.
