Setting Information on the Menu Tab

Use the Menu Security tab to specify to which menus the user has access. If you deny a user access to a menu, the user also cannot access any of the applications listed in the menu.

To specify menu security settings:

  1. Verify that the correct User ID is displayed in the User ID field.

  2. Click the Menu Security tab.

  3. Specify which menus you would like the user to access. Specify Yes to allow access and No to deny access.

  4. Click Save.
