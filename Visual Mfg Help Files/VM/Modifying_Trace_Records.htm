Modifying Trace Records

You can modify the Properties, Comments, Expiration Date, and Quantity Assigned for any trace record by modifying the corresponding field.

Modifying the Quantity Assigned field provides an easy way to manipulate preassigned quantities when you are setting up a group of lot numbers.

Modifying properties and comments allows administrative corrections and adjustments to trace records. You should use caution when changing these fields; the original values are usually entered in the normal course of traceability, and comprise an important part of the traceability record.

Click Save to commit the changes.
