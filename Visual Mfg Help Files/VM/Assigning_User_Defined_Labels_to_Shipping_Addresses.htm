Assigning User Defined Labels to Shipping Addresses

You can define user defined Layout IDs for customer Shipto IDs. User defined fields are static fields in the database you can use to store special information about customer shipping addresses.

To define user defined labels:

  1. From the VISUAL main menu, select Customer Order Entry from the Sales menu.

The Customer Order Entry window appears.

  1. Click the Our Order ID button to select the order for which you want to define user-defined labels.

If you want to create a new sales order, refer to order entry procedures.

The necessary fields are auto-filled, and all sales order lines are displayed.

  1. From the Edit menu, select Shipping Address. You can also right-click the window itself (anywhere but on the line item table) and select Shipping Address from the pop-out menu.

The Shipping Address (Customer) dialog box appears.

  1. Select the Shipto ID line for which you want to specify a Layout ID.

  2. Click the User Defined tab.

  3. From the User Defined Field Layout ID list box, select the appropriate user defined label.

If a customer has multiple Shipto addresses, you may want to take the time to establish multiple Layout IDs. For example, if a Shipto ID has certain restrictions on times of delivery or the maximum amount of cargo it can accept, create a label with fields apposite to the Shipto ID’s unique requirements.

You cannot create new User Defined Field Layout IDs from this screen. To create a new Layout ID, use one of three applications that support the creation of new Layout IDs. Those applications are: Customer Maintenance, Vendor Maintenance, and Part Maintenance.

The field labels are changed to display the Layout ID.

  1. Click the Save button to save the User Defined Field Layout ID to the Shipto ID.
