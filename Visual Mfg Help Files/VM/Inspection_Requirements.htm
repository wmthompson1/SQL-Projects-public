Inspection Requirements

If you are working with IQM, you can set the requirement for inspections on your labor tickets and inspection locations by selecting the appropriate check boxes:

Require Completed Inspections on Labor Tickets – Select this check box if you want your users to select the Quality Data Collection Complete check box in the Labor Ticket Entry window before they can save the labor ticket.

Require Completed Inspections for Return/Release – Select this check box if you want your users to select the Quality Data Collection Complete check box in the Receiving Inspection dialog box of the Purchase Receipt Entry window.
