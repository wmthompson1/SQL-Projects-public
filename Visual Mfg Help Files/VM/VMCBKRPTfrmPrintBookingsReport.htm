About the Customer Bookings Report Window

Use the Customer Booking Report to print information for all Firm, Released, and Closed customer orders in a specified date range. Order amounts are totaled for each group in the sorting sequence. For example, if sequencing by Customer ID, total order amounts for each customer are shown.

Backorders are not included in this report. See the Customer Backlog Report.

Starting the Customer Bookings Report Window

Select Sales, Customer Bookings Report.

Printing the Customer Bookings Report

Use this procedure to generate customer booking reports.

Fields

Site ID(s)

If you are licensed to use multiple sites, click the arrow and select the sites to include in the report. If you are licensed to use a single site, this field is unavailable.

Starting Date and Ending Date

The date range for the report. Dates are based on the customer order date.

If you select the Print Month to date/Year to date summary check box, January 1 of the current year is inserted in the Starting Date field, and the current date is inserted in the Ending Date field.

Starting ID and Ending ID

The label for these fields changes based on the sequence option that you select. For example, if you select Customer ID in the Sequence section, the field labels are Starting Customer ID and Ending Customer ID. Use these fields to print the report for a range of IDs or records. To include all IDs in the report, leave the fields blank. To print the report for one ID, specify the same ID in the Starting ID and Ending ID fields.

Report Currency

The currency for values in the report. Select ** Native ** to view amounts in each order's currency. If you select ** Native **, grand totals are not provided. If you select sites in more than one accounting entity, then the currency list is filtered to show the currencies that the entities share.

Print Month to date/Year to date summary

Select this check box to print month to date and year to date totals. If you select this option, then you can sequence the report by customer ID, customer name, or none. If you click None, then the report shows summary totals that are not broken down by customer.

If you select this check box, January 1 of the current year is inserted in the Starting Date field, and the current date is inserted in the Ending Date field.

Print Month to date summary

Select this check box to print month to date totals. If you select this option, then you can sequence the report by customer ID, customer name, or none. If you click None, then the report shows summary totals that are not broken down by customer.

Report output

Select the output format for the report. See Report Output Options.

Sequence

Specify how to sort the report. For example, if you click Part ID, the report shows customer orders for each part ID. If you selected the Print Month to date/Year to date summary or Print Month to date summary check boxes, you can sort the report by Customer ID, Customer Name, or None.
