# Viewing Baseline and EAC Comparisons

VISUAL gives you the ability to compare any version
of an EAC to its original baseline target. This allows you to see
view the impact those different versions have on the contracts profitability.

To view Baseline and EAC Comparisons:

1. In the Project Summary
   window, select the project whose percent of completion values
   you want to view.
2. From the Info menu,
   select the Baseline/EAC Comparison option.

The version appearing in the Baseline/EAC
Comparison dialog box is the version you have designated as current
in the Project EAC/Target Costs window.

3. If you want to select
   a different version, click the Version ID
   arrow and select the version you want.
4. When you have finished
   viewing your Baseline/EAC Comparisons, click the Close
   button.
