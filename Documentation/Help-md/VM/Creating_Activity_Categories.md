# Creating Activity Categories

To create an activity category:

1. In an application that
   supports activities, select Edit, Activity Entry.
2. In Activity Entry, select
   Maintain, Activity
   Categories.
3. Click Insert.
4. Specify this information:

Category ID
Specify an ID for this category.

Description
Specify a description for the ID.

5. Click Save.
