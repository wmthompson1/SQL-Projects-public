# Resizing Browse Window Columns

In addition to adding and removing columns and assigning
column positions in the Browse window, you can also adjust column
widths.

1. Place the pointer on
   the separator between two column headers.

The pointer changes to a double-arrow with
a vertical line, indicating movement of the divider between the columns.

2. Click
   and drag the separator to resize the column.

VISUAL resizes the column as you drag the
dividing line.

3. Select Save
   Widths and Positions from the Options
   menu to save the new column widths.
