# Creating the Project Warehouse

If you created the project warehouse through Project
Maintenance, you do not need to perform this step. Only perform this
step if you cleared the Create Warehouse check box when creating the
project in Project Maintenance.

1. Select Warehouse
   Maintenance from the Inventory menu. The Warehouse Maintenance
   window appears.
2. In the Warehouse ID
   field, enter the ID of your project. Make sure that the ID you
   enter for the warehouse is identical to the project ID.
3. In the Description field,
   enter a description of the warehouse.
4. In the Country ID field,
   enter the ID of the country where the warehouse is located. You
   can enter a country only if the Intrastat option is enabled.
5. In the Region ID field,
   enter the ID of the region where the warehouse is located. You
   can enter a region only if the Intrastat option is enabled.
6. If the warehouse is
   a VISUAL Distribution warehouse, select the VDW
   Installed check box. VDW warehouses can only be used in
   conjunction with VISUAL Distribution.
7. Click the Independently
   Planned Warehouse check box. Project warehouses must be
   independently planned.
8. Select the MRP
   Exempt check box if you would like to exclude the warehouse
   from material resource planning. You may want to select this check
   box upon the conclusion of a project to prevent material resource
   planning from using the warehouse during processing.
9. If the warehouse supports
   WIP/VAS, select the WIP/VAS supported
   check box. WIP/VAS must be globally activated in Application Global
   Maintenance.
10. Click the Inventory
    Not Shared arrow to define whether or not the warehouse shares
    its inventory. Select Y if the warehouse
    does not share inventory. Select N if
    the warehouse does share inventory. Select P
    if only the program manager can transfer inventory to other warehouses.
    Specify the program manager in the Contract General dialog box
    in Project Maintenance.
11. Click the Insert
    Row button. A new entry appears in the location table.
12. Enter the Location ID
    for the location you are creating within the warehouse.
13. Select the type for
    the location from the Type drop down box.

The five types are: Regular, Floor Stock,
In Transit, From Customer, and From Vendor. The From Customer and
From Vendor types are used with consigned inventory. For more information
on consigned inventory, see the "Consigned Inventory" chapter
in the Infor VISUAL Inventory User's Guide.

14. Enter a description
    for the location in the Location Description column.
15. You can continue to
    enter additional locations by clicking the Insert
    Row button and repeating steps 11 through 13.
16. Click the Save
    button to save the warehouse information.
