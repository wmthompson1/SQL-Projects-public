# Selecting Customer Alternate Warehouses

1. With the appropriate
   Customer ID in the Customer ID field, click the Order
   Management tab.
2. Click the Alternate
   Warehouses button to enter alternate warehouses for a customer.

The Alternate Warehouses for Customer dialog
box appears.

3. Click the ShipTo
   ID button to view a list of valid ShipTo IDs for the customer.
4. Select a ShipTo ID and
   click Ok.
5. With the appropriate
   ShipTo ID in the ShipTo ID field, click Insert
   to begin adding Alternate Warehouse information.
6. Double-click <Warehouse
   ID> to select a warehouse and then enter a rank for
   the warehouse.

Repeat this procedure for each alternate
warehouse.

7. Click the Save
   button to commit alternate warehouse information.
