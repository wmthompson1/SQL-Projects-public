# Dispatching Purchase Order Line Services

You can use the Service Planning window to dispatch
purchase order line services.

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site to use. If you are licensed to use a single site, this field
   is unavailable.
2. Click the Service ID
   arrow and select the service to dispatch.
3. Select Dispatch
   P/O Service from the Edit menu.

The Dispatch Purchase Order Line Service
dialog box appears.

4. Enter the part amount
   you want to dispatch in the Dispatch Quantity field.
5. Click the Dispatch
   button.

The Service Dispatch window appears.

A new line appears in the Service Dispatch
window for that particular dispatch for the purchase order.

6. Click the Disp
   All button to dispatch the specified part amount.
7. Click the button to
   save any changes to the Service Dispatch window.
