# Reviewing Quick Quote Spreadsheet Data

After you create a Quick Quote using an Excel spreadsheet,
you can edit the information in the Quick Quote dialog. To review
the original information imported from the spreadsheet, click the
View Excel Data button. A grid that contains
the imported information is displayed.
