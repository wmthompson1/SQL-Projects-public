# Entering and Deleting Intrastat Port of Transshipment Information

The port of transshipment is the place from which
goods will be shipped.

1. Click the Port
   of Transshipment button.

The Port of Transshipment dialog box appears.

2. Click the Insert
   button.

A new line appears in the line item table.

3. In the Code column,
   enter a numeric code for the port of transshipment.
4. In the Description column,
   enter a description for the new port of transshipment code.
5. Click the Save
   button to save the new information.

## Deleting Port of Transshipment Codes

1. Click the Port
   of Transshipment button.

The Port of Transshipment dialog box appears.

2. Select the port of transshipment
   code you want to delete, then click the Delete
   button.

An x appears in the row header beside the
code you are deleting.

3. Click the Save
   button to save the new information.
