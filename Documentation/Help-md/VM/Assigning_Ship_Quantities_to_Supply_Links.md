# Assigning Ship Quantities to Supply Links

|  |  |
| --- | --- |
| POSTIT.gif | You must assign supply links to customer order lines in Customer Order Entry before you ship them. If you have not attached supply links to a customer order line, this option is of no use to you. |

1. If you are licensed
   to use multiple sites, click the Site ID
   arrow and select the site in which you placed the order. If you
   use a single site, this field is unavailable.
2. Click the Order
   ID browse button and select the appropriate customer order.
3. Select the customer
   order line that contains the supply links to which you want to
   assign ship quantities highlighted in the Shipping Entry Window
   line item table, select Assign Ship Quantities
   to Supply Links from the Edit menu.

The Assign to Supply dialog box appears.

The line item table contains a list of the
customer order's supply links.

The line item table contains the following
columns:

Supply ID - The
ID of the supply link.

Assign Quantity
- The portion of the Allocated Quantity that you want to assign to
the supply link.

Link Unreceived Quantity
- The "Link" or allocated quantity that you have yet to
receive.

Desired Receive Date
- The date that you want to receive the supply link line. If the supply
link is a P/O, then it's the day you want to receive the P/O.

Supply Unfulfilled Quantity
- The quantity of the supply link line that you have yet to fulfill.

Supply Quantity
- The order quantity of the supply link line.

Supply Received Quantity
- The quantity of the supply link line that you have received.

Link Allocated Quantity
- The quantity of the supply link line that you have allocated to
the customer order line at hand.

Link Received Quantity
- The linked quantity received so far.

Link Issued Quantity
- The linked quantity received so far.

Supply Part ID
- The ID of the part for the supply link line. This is usually the
same as the Part ID of the C/O line.

Vendor ID - The
ID of the vendor if the supply link is a P/O or P/O delivery schedule
line.

Vendor Part ID
- The Vendor ID of the part.

4. In the Assign Quantity
   column, enter the ship quantity you want to assign to the supply
   link.
5. Click the Ok
   button to commit the assignment of supply to the ship quantity.

| POSTIT.gif | After you fully ship a customer order, you cannot assign ship quantities to supply links. If you select the option from the Edit menu after shipping the line, the Assign To Supply dialog box still appears with the supply links for the C/O line in the line item table, but the Ok button is unavailable. The order is closed after it is shipped. |
