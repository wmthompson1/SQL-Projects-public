# Printing Formats

The format of Estimating Window reports can be modified
using Centura Report Builder. The following QRP report files apply:

## Print Quotation

Standard  VMESTFRM.QRP

User-Defined 1 VMESTUD1.QRP

User-Defined 2 VMESTUD2.QRP

User-Defined 3 VMESTUD3.QRP

## Proforma Report

By Month  VMESTRP1.QRP

By Sales Rep VMESTRP2.QRP

By Territory VMESTRP3.QRP

By Cust, Month VMESTRP7.QRP

## Cost Detail Report

With Pricing VMESTRP4.QRP

Without Pricing VMESTRP8.QRP

## Quote Log

Sorted by Cust VMESTRP6.QRP

No Sort by Cust VMESTRP5.QRP
