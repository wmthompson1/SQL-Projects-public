# Amtsbi.exe Debug/Testing Notes

1. At the menu prompt you
   can type DB (uppercase) and it will
   prompt you to change the connected database alias to another alias
   defined in the ODBC Agent
2. At the menu prompt you
   can type LOG (uppercase) and it will
   prompt you for a log file name (example: amtsbi.log). This file
   can be analyzed to diagnose problems.
3. Type or scan QUIT
   at the menu prompt to quit/exit the program
4. Type or scan RESTART
   or CANCEL at any prompt to return to
   menu prompt
5. Type or scan SHUTDOWN
   at any prompt to quit/exit instantly
6. Closing the window before
   unloading afnb could cause the PC to hang

The programs, VMBTSRCV.exe, VMBTSINV.exe, VMBTSSHP.exe, and VMBTSPHY.exe
must execute on a client to actually complete the transactions. Each
of these can be executed independently and more than one can be executed
at a time. Each scanned transaction isnt complete until it has been
posted by one of these programs.
