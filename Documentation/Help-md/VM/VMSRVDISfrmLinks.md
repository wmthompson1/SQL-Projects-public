# Viewing Linked Work Orders

To view linked
work orders for your outside service dispatches:

1. In the Outside Service
   Dispatch Entry window, select the dispatch in which you are interested.
2. Select the row header
   for the part for which you want to view linked work orders.
3. Click the View
   Links toolbar button.

The Linked Work Orders dialog box appears
populated with a list of the work orders linked to the part you selected.
