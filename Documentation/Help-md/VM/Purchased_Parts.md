# Purchased Parts

Purchased parts have a Material account and possibly
a Burden account, if you are using purchase burden. Like the Fabricated
Parts, you can set up the Product Code Accounts as the defaults, or
VISUAL uses the Inventory default account from the Interface table.
Also like the fabricated parts, if you do not assign an account number
to the purchased part, the default account is used.
