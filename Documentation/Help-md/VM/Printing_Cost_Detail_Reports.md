# Printing Cost Detail Reports

The cost detail report provides you with a single
printed (or viewed) report containing all information relevant to
the estimated costs, calculated prices, and quoted prices for all
line items and quantities quoted. Having set up the quantity, engineering,
and price information, this report summarizes the situation for you.

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site that contains the quote. If you are licensed to use a single
   site, this field is unavailable.
2. Select Print
   Cost Detail from the File menu.

The Print Cost Detail dialog box appears.

3. Choose the output destination
   from the drop down menu. Select one of the following:

Print - To output
the file to a printer, select this option. VISUAL sends the file to
the selected default printer.

View - To see
what the printed output will look like before it is printed, select
View. When the information appears on the screen, you can scroll through
it and print it if you need a hard copy.

File - To save
the information to a text file, select File from the pull down menu.

4. Select the line items
   to print.

You may print cost detail for one or more
selected quote line items, or for the whole quote.

If you wish to print for selected lines,
select the desired lines before choosing Print Cost Detail. Then check
the Selected Lines Only check box.

5. If you want cost calculations
   to include consideration of quantity breaks offered on purchased
   parts, click the Use Qty Breaks on Purchased Material check box.

This should match your selection of this
feature in the Prices dialog box. If it doesnt estimated costs used
in this report may be different from those used to calculate unit
prices.

6. If you want to print
   the price information that you specified in the Prices dialog
   box, check the Print Pricing Information check box.
7. If printing pricing
   information, then you can override the markups specified in the
   Prices dialog box by checking the Override Markup dialog box.

This allows you to try some different markups
and see the effects on your margin.

The Estimating Window allows you to specify
different markups for each quoted quantity. However, you can only
override using the same markups for all quantities.

8. Click Ok.

A standard Windows dialog box appears.

9. Click Ok
   to print.

A separate report section, on its own set
of pages, is printed for each quantity of each line item selected.

For each operation of the quote master or
engineering master, Material, Labor, Burden, and Service cost estimates
are printed, along with setup and run hours. Setup and Run hours are
totaled.

For each material requirement, Material,
Labor, Burden, and Service costs are printed, along with quantity
required.

Next, totals are printed for each of these
categories, along with a total cost and resulting unit cost for the
quantity. This is labeled the pre-GSA cost, your cost before any markups.

If printing pricing information, the following
additional information is provided:

GSA expense for each cost category is shown
next. GSA expense is computed by multiplying the GSA percent by each
pre-GSA cost, and subtracting that amount from the cost. Next, Total
Cost is printed, that being the pre-GSA cost plus the GSA expense.

The same format is followed to add in markups,
to come to a Calculated Price for each category, a Total Calculated
Price, and a resulting effective unit price.

Percent Profit expresses Total Markup as
a percentage of Total Calculated Price.

Percent Markup expresses Total Markup as
a percentage of Total Calculated Cost.

The same calculations are shown for the
Quoted Price, which may be different from the one calculated.
