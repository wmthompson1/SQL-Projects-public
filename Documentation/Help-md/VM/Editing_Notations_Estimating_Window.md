# Editing Notations

To edit a notation entry:

1. Use the navigation arrows
   to the left of the dialog box to move to the entry that you want
   to edit.
2. Click the Edit
   button.

The entry becomes highlighted.

3. Make the necessary changes.
4. Click the Save
   button to save your changes.
5. Click the Close
   button to close the Notations dialog box and return to the Estimating
   Window.
