# Setting Timer Options

1. You can specify the
   amount of time you want VISUAL BTS to wait between checks for
   new transactions.
2. From the Timer Options
   menu, select Set Timer Period.

The Set Timer Period dialog box appears.

3. In the available field,
   enter the period of time in milliseconds that you want VISUAL
   BTS to allow to lapse before it checks for new transactions.

If you enter 1,000 here, VISUAL BTS checks
for new received signals from the barcode reader every second. The
milliseconds default unit allows you to check for new transactions
virtually continuously.

4. Click the Ok
   button to set this preference.
