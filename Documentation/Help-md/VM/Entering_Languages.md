# Entering Languages

It is possible that you may need to specify multiple
languages when dealing with international customers. Some international
companies have requirements for issuing documentation in the home
language of the company with which they are dealing. For example,
a situation may call for an order acknowledgement in German, a packlist
in French, and an invoice in Italian. Use Customer Maintenance to
enter and maintain Language IDs and descriptions.

1. From the Edit menu,
   select Languages.

The Languages dialog box appears.

2. Click the Insert
   button.

A new line appears in the table.

3. Enter a two character
   alphanumeric identifier as an ID for the language.

This ID must be unique.

4. Enter a description
   for the language. Maximum length of entry is 20 characters.
5. Click the Save
   button to commit the information.
6. To copy language information,
   select a line and click the Copy button.

To paste the information, click the Insert button and then click the Paste
button.

Because Language IDs must be unique, VISUAL
pastes the Language ID description only.

7. Click the Close
   button to return to the main Customer Maintenance window.
