# Printing G/L Transactions

The G/L Transactions Report provides invoice activity
to the G/L by accounting period or batch. You have the option of printing
posted or un-posted transactions. This allows you to print unposted
transactions to review them before posting them to the ledger. You
can also use this report to detail postings within a particular batch
or accounting period.

1. From the File menu,
   select Print G/L Transactions or click
   the Print G/L Transactions button on
   the main toolbar.

The Print G/L Transaction dialog box appears.

2. Click the Standard
   tab, and specify this information:
3. Click the Site
   ID arrow and select the site to use.
4. Select one of the following
   options from the Print Selection section.

By Accounting Period
- Select this option to print G/L Transactions by accounting period
(time period).

By Batch Number
- Select this option to print G/L Transactions by batch number (range
of single batch IDs). This option is only available when you are printing
posted transactions.

5. If you requested transactions
   by Accounting Period, enter the following information:

End Date of Accounting
Period - Click the Calendar button
to select any date within the accounting period, or enter the end
date in the field provided.

VISUAL determines the last date of the accounting
period, and displays the month and year when you tab out of the field.

6. If you choose Batch
   ID (for posted transactions only), enter the following information.

Starting Batch ID
- Click the Browse button to select a starting
batch ID from the list, or enter the starting batch ID in the field
provided.

If you want a single batch, enter that Batch
ID in both fields.

If you want the report to include all batches,
leave both fields blank.

If you specify a Batch ID in the Starting
field and leave the Ending field blank, VISUAL begins the report with
the Starting Batch and prints all subsequent batches.

Ending Batch ID
- Click the Browse button to select an ending
batch ID from the list, or enter the ending batch ID in the field
provided.

If you want a single batch, enter that Batch
ID in both fields.

If you want the report to include all batches,
leave both fields blank.

If you specify a Batch ID in the Starting
field and leave the Ending field blank, VISUAL begins the report with
the Starting Batch and prints all subsequent batches.

7. Click the arrow beside
   the Currency ID field to select a Currency ID from the list.

8. Enter one of the following
   options in the Sequence section.

G/L Account ID
- Select this option to sort the report by G/L Account ID.

Voucher ID, G/L Account
ID - Select this option to sort the report by Voucher ID and
G/L Account ID.

Batch ID, G/L Account
ID - Select this option to sort the report by Batch ID and
G/L Account ID. This option is only available when printing the report
by batch number.

Batch ID, Invoice ID
- Select this option to sort the report by Batch ID and Invoice ID.
This option is only available when printing the report by batch number.

9. Enter one of the following
   options in the Transactions section.

Print Unposted Txs
- Select this option to print unposted transactions.

Print Posted Txs
- Select this option to print posted transactions.

10. Click the arrow to select
    one of the following output types from the list.

Print - Select
this option to output the file to a printer. The file will be sent
to the selected default printer in Printer Setup. When the Print dialog
box appears, click the Ok button.

View - Select
this option to see what the output will look like before you print
it. When the information appears on the window, you can scroll through
and printed if you want a hard copy.

File - To save
the report to a text file. When you click Ok, the Print to File dialog
box appears requiring you to enter a destination directory and filename.
If a destination is not specified, VISUAL places the text file in
the same directory that the program files were placed in when the
database was installed (i.e. the default VMFG).

Click the Include field
names on first output line check box if you want to output
column labels.

Click the Ok button
and VISUAL creates the file that can be edited outside of Infor VISUAL
Financials using any text editor.

E-mail - Select
this option to send the report as an .RTF attachment through your
electronic mailing system.

11. Select the Current
    User Only checkbox to view G/L transactions that were created
    only by the user. Leave the checkbox unchecked to view all user
    G/L transactions.
12. Click the Advanced
    tab, and specify this information:

Report Calendar ID
-

Fiscal Year -

Site ID (range)
-

13. Click the Print
    Setup button to modify printer options.
14. Click Ok
    to activate the print option.

VISUAL prints the G/L Transactions report.
