# Adding Specifications

Specifications allow you to add free form text to
a purchase order. This text is printed on Purchase Order Acknowledgments.
You can use this text to specify and document special needs, notes,
and agreements for the purchase order. These might be used to specify
packaging requirements, shipping details, configuration information,
or other options that need to be specified along with a purchase order.

There are two types of purchase order receipt specifications (specs,
for short): Order Specs and Line Specs. Order Specs are associated
with the entire order, and Line specs with individual line items.
