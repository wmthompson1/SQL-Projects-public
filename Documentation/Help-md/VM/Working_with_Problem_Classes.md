# Working with Problem Classes

Setting up problem classes and assigning them to your instances
of buffer intrusions helps you to analyze your buffers and the effects
they have on your production line.

If you are licensed to use multiple sites, you can set up problem
classes on an enterprise-wide basis only.

Use these procedures to set up and use problem classes:

* [Setting
  Up Problem Classes](Setting_Up_Problem_Classes.md)
* [Assigning
  Problem Classes to Buffers](Assigning_Problem_Classes_to_Buffers.md)
