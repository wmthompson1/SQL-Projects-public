# Viewing Carton Information

After you have selected the customer's order in which
you are interested, you can view the carton information for that order
by clicking the Carton Tracking toolbar
button.

The Carton Tracking table appears populated with the appropriate
information.

You can change the carton information appearing in the table by
changing the following information:

Customer Order ID

Carton ID

Tracking Number

Create Date
