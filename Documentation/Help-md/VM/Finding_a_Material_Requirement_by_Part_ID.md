# Finding a Material Requirement by Part ID

It may be difficult to locate a material requirement,
because the Part ID is usually the information of interest. The Find
Part ID function allows you to locate material requirements by Part
ID.

1. Select Find
   Part ID from the Edit menu under
   Parts.

The Find dialog box appears.

2. Enter the Part ID you
   are looking for in the Search For field.
3. Click the Search
   button.

If VISUAL finds the part in a requirement,
it automatically selects the requirement and places it in the home
position.

If VISUAL does not find the part, the message
End of Part List Encountered in Search appears.

4. You can continue to
   search for other instances by reselecting Find Part ID.

VISUAL retains the prior search string.

5. Click the Search
   button again to find the next requirement.

You can also use wildcards in the search
string to search for any parts with a particular pattern.

For example, B% searches for all Part IDs
that start with the letter B.

You can continue to search for other instances
by reselecting Find Part ID. The prior search string is retained.
Press Search again to find the next requirement.
