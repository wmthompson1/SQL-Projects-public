# Entering Customer Notes

You can enter information regarding a customer in
the Customer Notations note box. A time stamp informs the reader of
how current the note is; the copy and paste options allows you to
duplicate entries; the edit option allows you to update and modify
existing customer notes.

1. Click the Cust
   Notes button located at the bottom of the window. The Customer
   Notations note box appears.
2. Click New
   to clear the note box. VISUAL displays the current time date and
   time.
3. Enter the appropriate
   text.
4. Click the Save
   button.
