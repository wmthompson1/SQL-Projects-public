# Viewing Work Order Material Requirement Demand Allocation Information for a Purchase Order Line Item

After you have either fully or partially allocated
demand to a work order material requirement using the preceding procedures,
you can view allocation information on the purchase order line level.

For each purchase order line item to which you have assigned demand,
the following appears:

* In the Base
  ID column, if there is only one material requirement link,
  the Work Order ID that the material requirement is part of appears.

In the Base ID
column, if there are multiple material requirement links, Multiple
Links appears.

* In the Lot
  ID column, if there is only one material requirement link,
  the Lot ID of the work order which the material is part of appears.
* In the Split
  ID column, if there is only one material requirement link,
  the Split ID of the work order which the material is part of appears.
* In the Sub
  ID column, if there is only one material requirement link,
  the Sub ID of the work order which the material is part of appears.
* In the <Seq
  #> column, if there is only one material requirement
  link, the Seq # of the material requirement in the work order
  appears.
* In the <Piece
  #> column, if there is only one material requirement
  link, the Piece # of the material requirement in the work order
  appears.
* In the above five columns,
  if there is more than one demand material requirement link, nothing
  appears.
* In the Link
  Part ID column, if there is only one material requirement
  link, the Part ID of the purchase order line appears.

In the Link Part ID
column, if there are multiple links of material requirement, nothing
appears.

* In the Link
  Warehouse ID column, if there is only one link of material
  requirement, the Warehouse ID of the purchase order line appears.

In the Link Warehouse
ID column, if there are multiple links of material requirement,
nothing appears.

* In the Link
  Quantity Unallocated column, if there is only one material
  requirement link, the quantity of the purchase order line that
  you have yet to assign to demand to appears.

In the Link Quantity
Unallocated column, if there are multiple links of material
requirement, nothing appears.

* In the Total
  Qty Allocated column, the quantity thus far of the purchase
  order line that you have allocated to sources of demand appears.
