# Renumbering Piece Numbers

VISUAL assigns material requirement piece numbers
sequentially as you add requirements and legs to an operation. When
a single operation has a large number of materials, you may want to
resequence the requirements by Part ID. This makes the list more manageable.

You can only perform this operation on a master, because a work
order could have references to existing piece numbers.

To renumber piece numbers:

1. Select the operation
   whose pieces you wish to renumber.
2. Select Renumber
   Pieces by Part ID from the Edit
   menu under Parts.

VISUAL renumbers the pieces and refreshes
the display.
