# Backflushing Limitations

If you specify an auto-reporting resource on an operation,
then these features are unavailable:

* Part
  traceability. The Begin Traceability at this operation
  check box on the Setup/Run tab of the operation is unavailable.
  You cannot use part trace functionality with auto-reporting resources.
* Inspection.
  The Required on Setup and Required on Run check boxes on the Scrap/Yield
  tab of the operation card are unavailable. You cannot require
  inspection for auto-reporting resources.
