Searching and Viewing ECN Histories




# Viewing ECN History

After you have entered a large amount of ECNs into
your database, you may find it helpful to use the ECN History window
to search for a particular items ECN history.

1. Select Eng/Mfg,
   ECN History.
2. In the Line Type
   section, click the type of item whose ECNs you want to view.
3. Click
   the ID button and select the item whose
   ECN history you want to view.
4. To
   filter the ECN History list by status, use the Status section.
   For example, if you only select Completed, only those ECNs that
   you have completed appear in the table.
5. After you make your
   selections, this information is displayed in the table:

ECN ID - The ID
of the ECN.

Site ID - The
site where the ECN was created.

Assigned To -
The user to whom the ECN is assigned.

Status - The current
status of the ECN.

Desired Want Date
- The date by which the ECN should be complete.

Completed On -
The date that the ECN was completed.

Type - The type
of ECN.

Reason - The reason
for the ECN.

Locked - Indicates
that the ECN is locked. When an ECN is locked, the subject of the
ECN, such as a part, cannot be used in transactions.

Priority - The
priority code given to the ECN.

Origin - The user,
customer, or vendor who requested the ECN.

Origin Type -
The type of ECN.

Reason - The reason
for the ECN.
