# Repeating Purchase Order Line Items

At any time as you enter purchase order line information,
you can instruct VISUAL to insert a new line into the purchase order
with the identical information of a line youve entered to that point.
Use this feature to avoid the arduous reentry of the same purchase
order line information.

1. Select the line you
   want VISUAL to repeat.
2. Click the Repeat
   button located underneath the line item table.

VISUAL inserts a new line in the line item
table with the identical information of the selected line.

3. Continue to click the
   Repeat button for each new line you
   want VISUAL to insert.

## Copying Delivery Schedule Lines

When you use the Repeat Line function to copy an order line, you
can also copy the delivery schedule.

To copy the delivery schedule along with the line, you must specify
your preference in Preferences Maintenance. Use these settings:

1. Select Admin,
   Preferences Maintenance.
2. Click the Insert
   button.
3. Specify these settings:

Section Specify
PurchaseEntry.

Entry Specify
CopyDelSchedOnRepeatLine.

Value Specify
Y.

4. Click the Save
   button.

When you elect to copy delivery schedules, the delivery schedule
is copied when you use the Repeat Line function. You can then edit
the part ID as necessary.
