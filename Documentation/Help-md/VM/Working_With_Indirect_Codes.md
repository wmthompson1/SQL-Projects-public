# Working With Indirect Codes

Indirect labor codes specify the accounts to which
indirect labor (i.e. non-work order related) is charged. The employee
can then use the bar code associated with the indirect code when charging
time to something other than a work order (i.e. training, meetings,
etc.).

## Adding New Indirect Codes

1. Select Indirect
   Codes from the Edit menu.

The Indirect Codes dialog box appears.

2. Click the Insert
   button.

A new line appears at the end of the table.

3. Enter the new Indirect
   ID in the Indirect Code column.

If you are using Barcode Labor Ticket Entry,
enter indirect labor codes using UPPERCASE alphanumeric characters
only. Barcode scanning devices can not read indirect codes than contain
lowercase characters.

4. Enter a description
   of the new Indirect ID in the Description field.
5. Enter the G/L Account
   ID and Code for this Indirect ID.

These entries must already exist in your
database. You can select this information from existing tables by
double clicking in the column heading. This calls up a table of available
information.

6. Click the Save
   button to save the new Indirect code.

## Deleting Indirect Codes

To delete an Indirect ID, highlight the line and click the Delete button.

## Copying Indirect Code Information

To copy information from one indirect code to a new line:

1. Select the information
   you want to copy and click the Copy
   button.
2. Click the Insert
   button to insert a new line.
3. Click the Paste
   button.

VISUAL pastes all of the information (except
the indirect code) into the new line.

4. Enter the Indirect code
   to be associated with the new entry in Indirect Code field.
