# Show Purchase Order RFQs

Use the Show Purchase Requisitions window to view
the purchase requisitions associated with your purchase orders.

To view your purchase order requisitions:

1. Select the Show
   Purchase Order RFQs option from the Info menu.

The Show PO RFQs window appears. If you
have selected a PO in the Purchase Order Entry child window, the PO
ID appears in the Starting and Ending PO ID fields.

2. Click the starting and
   ending PO ID browse buttons and select he range of POs for which
   you want to view requisitions.
3. To view requisitions
   for a single PO, select the same PO IDs for the starting and ending
   PO IDs.
4. Select the requisition
   line you want to view.
5. Click the RFQ
   button.

The Vendor RFQ Entry window appears populated
with the RFQ you selected.

You cannot make changes to the requisition
because a PO exists.
