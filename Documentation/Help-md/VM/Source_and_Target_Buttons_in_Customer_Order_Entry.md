# Source and Target Buttons in Customer Order Entry

When you are viewing a transaction, you can use the
source and target toolbar buttons to view related transactions. Use
the source button to view the previous transaction in the lifecycle.
Use the target button to view the next transaction in the lifecycle.

In the Customer Order Entry window, the Source button is labeled
Quote. It is available if an estimate was used to generate the order.
Click the button to open the estimate. The Target button is labeled
Packlist. It is available if you have shipped any portion of the customer
order. Click the button to open Shipping Entry.
