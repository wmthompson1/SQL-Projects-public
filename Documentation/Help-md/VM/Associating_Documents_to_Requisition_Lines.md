# Associating Documents to Requisition Lines

To associate a document to a requisition line:

1. Select the appropriate
   requisition, highlight the appropriate line, and select Line Document
   Reference from the Edit menu.

The Document Reference dialog box appears.

2. Click the Insert button
   to add a new line to the table and then double-click the <Document ID> column header to choose
   a document.

Your choice is restricted by displaying
only those documents with the appropriate reference typein this case
"Requisition."

You can, however, choose any document, regardless
of its reference type. To view all documents in the database, select
the All Reference Types check box.

3. Select the appropriate
   document and click the Ok button.

The document is added to the table.

4. Click the Save
   button to commit the document reference to the requisition line.
