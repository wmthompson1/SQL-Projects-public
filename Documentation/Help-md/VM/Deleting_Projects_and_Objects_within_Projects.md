# Deleting Projects and Objects within Projects

From time to time you will find the need to delete
an object in a project. For example, you have built a leg and find
that the contract has changed or your (or your customers) needs have
changed.

The process for deleting a project is the same as the process for
deleting an object in a project.

To delete an object in a project:

1. Select the object you want to
   delete.

A small arrow appears to the right of the
object you selected.

2. Click the Delete
   icon on the toolbar or from the Edit menu and select the Delete option.

A Confirm Delete dialog box appears.

3. Click the Yes
   button.

A progress indicator appears and VISUAL
deletes the object you selected from the project.
