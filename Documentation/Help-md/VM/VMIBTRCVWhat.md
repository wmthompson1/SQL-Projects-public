# What is Interbranch Transfer Receipt Entry?

After you have entered and shipped an interbranch transfer, use
Interbranch Transfer Receipts Entry to receive the shipped IBT quantities.
IBT Receipts Entry is similar to Purchase Receipt Entry, which you
use to receive purchase orders.

Though similar, IBT Receipts Entry does not support the following
Purchase Receipts Entry functions:

* Print Trace labels.
* Receiving Inspection.
* Returns.
