# Editing Commands in Group Menus

To edit the commands within your groups:

1. Select File,
   Personal Menu from the main VISUAL menu.
2. In the Personal Menu,
   select the group that contains the command you want to edit.
3. Right-click the command
   you want to edit and select Customize
   from the right-click menu.

The Customize dialog box appears.

4. Click the Edit
   Current Item button.

The Edit Command Item dialog box appears
populated with the current arguments.

5. Modify the commands
   arguments as necessary.
6. When you have finished
   modifying the command arguments, click the OK
   button.

The Edit Command Item dialog box closes.

7. Click the OK
   button.

The Customize dialog box closes and your
changes appear in the Personal Menu.
