# Setting Unit of Measure Conversions for a Service

You can set and modify unit of measure conversion
factors for services using the Unit of Measure Conversion dialog box.
This dialog box is the same as the Unit of Measure Maintenance window
available from the Infor VISUAL Maintenance menu.

1. With a service
   displaying in the Service ID field of the Service Maintenance
   window, select Unit Of Measure Conversion
   from the Edit menu. The Unit of Measure Conversion dialog box
   appears.
2. From the top table,
   select a unit of measurement. Your selection appears in the first
   gray field.
3. From the bottom
   field, select a second unit of measure. Your selection appears
   in the second gray field.

For example, this conversion is between
inches and feet: How many inches does it take to equal one foot?

4. In the Conversion
   Factor field, enter a numeric expression to satisfy the metric
   relationship.

The outcome can be expressed
thusly:

There are 12 inches in a foot.

You could have also entered
some equation or other combination of integers and operations that
through solution also equals the wanted value.

For example, instead
of 12, you could enter:

100-67+22-43

This, of course,
also equals 12.

If you select
the same unit of measure from the top table and the bottom table,
the conversion factor field becomes unavailable. Because a unit must
equal itself, the value 1 appears. You cannot edit this.

5. Click
   the Save button to commit the conversion
   factor.
