# Selecting a Customer Order Allocation Level

Select a Customer Order Allocation level by clicking
on the appropriate level of allocation support for customer orders:
None, Full, or Partial.

Full Allocation  This selection requires
you to make sure that each customer order has allocated supply before
accepting the order.

Partial Allocation  This selection allows
you to allocate supply to the order, but makes the allocation optional.
