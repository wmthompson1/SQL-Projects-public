# Deleting Operations

To delete an operation, select it and select Delete from the Manufacturing Window Edit
menu.

The system prompts you to confirm the deletion. All material requirements
and legs that are attached to the operation are automatically deleted
as well. Start, end, and material quantities are automatically recalculated.
