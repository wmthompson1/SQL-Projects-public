# About the Examine Triggers Window

Use the Examine Triggers window to compare the triggers
found in your database with a list of required triggers. The Examine
Triggers utility identifies missing, additional, disabled, and audit
triggers.

## Starting the Examine Triggers Window

Select Admin, Examine
Triggers.

## Task

This table shows the task that you can complete in the Examine Triggers
window:

| Task | Description |
| [Examining Triggers](Examining_Triggers.htm) | Use this procedure to identify inconsistences between the triggers in your database and a master list of triggers. |

Audit triggers that have been added to the database. Audit triggers
can be added by system administrators and are not considered to be
incorrect.
