# Print Formats

You can modify the format of the travellers and picklists
using Centura Report Builder. The following QRP report files apply:

|  |  |
| --- | --- |
| Traveller Report | VMTRVRPT.QRP |
| Picklist Report |  |
| By Location Sequence | VMTRVRP3.QRP |
| By Piece No. Sequence | VMTRVRP2.QRP |
