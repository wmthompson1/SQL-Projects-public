# Jumping Up to the Parent Part ID

Material Planning allows you to view parent Part IDs
for a material requirement in a separate Material Planning Window.

1. Select a line containing
   a demand order that is a work order.

You cannot select a customer order, because
there is not a parent part associated with that type of demand.

2. Select Jump
   Up from the Info menu.

A separate Material Planning Window opens
with the parent part information for the selected requirement. At
this point, the two Material Planning Windows are tiled on the screen.
The original window takes up the top half of the screen, and the new
window the bottom.

You can "Jump Up" again in either
window.

VISUAL continues to tile windows horizontally.
With a high-resolution monitor, it is possible to comfortably view
portions of three or four planning windows at once.

3. When finished with the
   new window, select Exit from its File menu to close just that window.
