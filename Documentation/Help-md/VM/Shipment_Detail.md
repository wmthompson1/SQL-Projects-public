# Shipment Detail

This [dashboard](VMDBOARDWhat.htm) detail grid lists
shipment lines.

Date filter applies to: Actual Ship Date

Filters: Customer ID, Sales Rep ID, Territory,
Product Code, Commodity Code, Site ID

Red text: n/a

Details: Customer ID, Customer Name, Packlist
ID, Ship Ln No., Ship Del Ln No., Cust Order ID, CO Line No, CO Del
Ln No, Order Qty, Shipped Qty, Selling U/M, Amount Shipped, Desired
Ship Date, Part ID, Part Desc., Service Charge ID, Sales Rep, Territory,
Product Code, Commodity Code, Currency ID, Site ID.
