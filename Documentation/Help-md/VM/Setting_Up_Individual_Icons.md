# Setting Up Individual Icons

It is possible to set up an individual Windows icon
to start a single VISUAL application. This is useful for users who
may only need a relatively small subset of applications. For example,
a user in purchasing might frequently use Vendor Maintenance and Purchase
Order Entry. Rather than start up the main menu to access these programs,
you can set up two icons to invoke the programs directly. When you
invoke a program in this way, database login takes place when you
start each application.

Consult your Windows documentation for information on creating program
icons.
