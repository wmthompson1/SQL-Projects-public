# Traceability in the Service Planning Window

When using the Outside Service Planning window, you
can view the Trace IDs you have created by clicking the Trace
Details icon on the toolbar or selecting Trace
Details on the Edit menu.

The Service Trace Details window opens populated with the appropriate
Tarce IDs for the current part in the Service Planning window.
