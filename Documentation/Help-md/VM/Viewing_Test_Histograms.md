# Viewing Test Histograms

You can view a histogram of the test results in Test
Definition.

1. Highlight the test step
   line item for which you want to view the histogram.
2. Click the Histogram
   button.

A histogram of the test results appears
in a window.

You can select different views of the histogram
from the Views menu and other options from the Options menu.
