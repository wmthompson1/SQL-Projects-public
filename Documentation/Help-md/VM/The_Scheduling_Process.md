# The Scheduling Process

After you have set up and verified shop resources
and part lead times, you can begin the scheduling process. This involves
the following steps:

## Scheduling Process Steps

### Scheduling the Job

1. Start the Concurrent
   Scheduler.
2. Set Concurrent Scheduler
   preferences.
3. Create a New Schedule
   or activate an existing schedule.
4. Select the appropriate
   scheduling options.
5. Run the schedule.

### Viewing the Results

1. Open the Scheduling
   Window.
2. Load schedule you just
   created.
3. View the schedule.
