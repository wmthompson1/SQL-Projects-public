# Viewing Part Pictures

If a part specified in a requisition line has an associated
drawing or graphic, you can view it in Purchase Requisition Entry.

To view a part picture:

Select the line containing the part and select Picture
from the Edit menu, or click the Picture
button on the table toolbar.

The Picture dialog box appears, displaying the picture associated
to the part.
