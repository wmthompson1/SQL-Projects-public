# Processing Selected Transfer and Issue Rows

You can process select rows by highlighting the line
you want to process in the pending transactions table and selecting
Process Selected Rows from the File menu.

If you are using the VBTS for the first time, it may be a good idea
to use this feature from time to time to monitor selected rows and
in the process develop an understanding of the transaction processors
and their intricacies.

In addition, you can use this feature as a debugging tool. After
scanning a label or a barcode on a report, you can observe the transaction
in the Pending Transactions table and verify that the data displaying
is the data you scanned. When an error occurs, you can instantly identify
it and take the appropriate steps to rectify the discrepancy.

From the File menu, select Process Selected Row.

VISUAL BTS processes the selected row. The standard VISUAL Purchase
Receipt Entry window appears instantaneously and then disappears.
The transaction is now processed.
