# Using Control Charts

Control charts can monitor the aim and variability
and thereby continually check the stability of the process. This check
of stability in turn helps to assure that the statistical distribution
of the product characteristic is consistent with quality requirements.

Use control charts to:

* Attain a state of statistical
  control for a process (i.e. to eliminate variation due to assignable
  causes).
* Monitor a process.
* Determine process capability.
  A process must be in statistical control before process capability
  measurement can occur.)
