# Editing Customer Information

You can edit customer information using Customer Maintenance.

1. Enter the Customer ID
   in the Customer ID field, or select the customer from the Customers
   Table.
2. Make any necessary changes
   to the customer information.

You can change any field in the Customer
Maintenance window. Changing the Customer ID creates a new customer.

3. Click Save
   to save your changes.
