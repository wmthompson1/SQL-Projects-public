## WIP/VAS Child Window

In the WIP/VAS child window, WIP/VAS IDs for parts in
warehouses are represented by a hammer icon. If no WIP/VAS is available
for a part in a warehouse, there is nothing to view in the WIP/VAS child
window.

With an order open in the Customer Order Entry child window, highlight
an order line and select WIP/VAS from the Tools
menu.

The WIP/VAS child window opens with the part from the order at the top
of the explorer tree and any WIP/VAS available for the part in the warehouse
underneath, with each WIP/VAS available in the warehouse represented by
a hammer icon. To change the part, click the Options
button, click the Part tab, and then select
another part. This part now has the focus in the WIP/VAS child window.
VISUAL retains the Warehouse ID from the order when searching for WIP/VAS
for the part. To view WIP/VAS availability for the part in another warehouse,
you must open an order with the Warehouse ID and then select a part.
