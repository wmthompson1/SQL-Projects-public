# Viewing Work Order Status

You can view work order status information for your
vendor orders using Vendor Inquiry.

1. Call up the order for
   which you want to check work order status into the Vendor Inquiry
   window.
2. Select Show
   Work Order Status from the Info menu.

A dialog box appears, showing all of the
work orders for that vendor order.

3. View the work order
   status information.
4. Select a work order
   to view, then click the View Operations
   button.

A dialog box appears, showing all of the
operations for the work order.

5. When you have finished
   viewing the operation information, click the Close
   button.

You return to the Work Orders dialog box.

6. Click the Close
   button.

You return to the information window for
that order.

7. Click the Close
   button to return to the Vendor Inquiry window.

Click the Next
or Prev buttons to view the next or previous
order from the Search Results table.
