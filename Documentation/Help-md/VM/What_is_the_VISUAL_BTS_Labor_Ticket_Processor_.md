# What is the VISUAL BTS Labor Ticket Processor?

New in release 6.1.3 is VISUAL ALTS. The new Automated
Labor Transaction System application is a real-time, hardware-independent
labor tracking system that you can fully integrate into Infor VISUAL.
Use the program to track shop floor operations and labor using barcode-scanning
devices (typically RF). VISUAL ALTS (Vmbtslab.exe) will eventually
replace the current VISUAL central labor (wedge) module, Vmbclabr.exe.
The new module has all of the same functionality as the current wedge
offering, but is hardware independent and is coded into the Material
Barcode executable, so it maintains the look and feel of that module.
For more information on [VISUAL
Material Barcoding](What_is_VISUAL_BTS_Material_Tracking_.htm), see the previous section.

VISUAL/ALTS supports the following general features:

* Simple DOS-based reader
  applet for use on many different Barcode reader platforms.
* ODBC Compliant connections
  to any and all databases.
* Completely customizable
  user interface for a variety of reader formats.
