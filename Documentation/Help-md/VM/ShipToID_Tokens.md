# ShipToID Tokens

If you add a shiptoID token to a template, the shipto ID is replaced
with the corresponding address.

This table shows the shiptoID tokens that are available and where
they can be used:

| Token | Template |
| %OrderShipToID | Customer Order |
| %OrderLineShipToID | Customer Order |
| %PacklistShipToAddrNo | Shipment |
| %OrderLineShipToID | Purchase Order |
