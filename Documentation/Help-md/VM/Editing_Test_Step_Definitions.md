# Editing Test Step Definitions

You can edit existing test step definitions in Test
Definition.

1. Highlight the test step
   line item that you want to edit.
2. Click the Edit
   button.

The Test Step Definition dialog box appears
for that step.

3. Make any changes to
   the test step definition.
4. Click
   Ok to save the changes.
