# Specifying Customer Shipping Defaults

Use the Shipping tab to define a customers general shipping preferences.
This is a sub-procedure of Adding a New Customer.

1. With the customer for
   whom you want to define shipping preferences in the window, click
   the Shipping tab.
2. Enter information into
   the following fields, as follows:

Carrier ID - From
the Carrier ID list box, select a preferred carrier code for shipments
to this customer. Add carrier codes to the database through Application
Global Maintenance.

Default Shipto ID
- Click the Default Shipto ID button to
select a default Shipto ID for VISUAL to use when preparing shipments
to this customer.

Free on Board
- From the Free on Board list box, select an FOB code for shipments
to this customer. Add FOB codes to the database through Application
Global Maintenance.

Ship Via - From
Ship Via list box, select a ship via code for shipments to this customer.
Add ship via codes to the database through Application Global Maintenance.

Customer Accepts Partial
Shipments - If the customer accepts partial shipments, select
the Accepts Partial Shipments check box.

Accepts Early Order X
Days Early - If the customer accepts early orders, select the
Accepts Early Orders check box and then
enter the number of days a shipment may be early into the adjacent
field, which is unavailable until you select the check box.

Pallet Details Required
- Select the Pallet Details Required check
box if pallet details are necessary for all shipments to his customer.
In order for VISUAL to generate pallet details, specify carton information
for a part.

Match Invoice ID to Pack-list
ID - Select the Match Invoice ID to Pack-list
ID if you want VISUAL to use the same ID for both the pack-list
and A/R invoice.

Choose a Customs Documentation option:

Do Not Print -
Select the Do Not Print option button if
you dont want to print customs documentation for this customer.

Print by Packlist
- Select the Print by Packlist option button
if you want to print customs documents for this customer by pack-list.
Only consolidated pack lists (see the Shipping Entry section) are
eligible for printing as customs documents.

Print by Bill of Lading
- Select the Print by Bill of Lading option
button if you want to print customs documents for this customer by
bill of lading.

3. Click the Save
   button or select Save from the File
   menu.
