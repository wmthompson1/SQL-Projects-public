# Viewing Back Orders

You can view back order information for your customer
orders using Customer Inquiry.

1. Call up the order for
   which you want to check back orders into the Customer Inquiry
   information window.
2. Select List
   Back Orders from the Info menu.

A dialog box appears, listing all of the
back orders for the current order.

3. Select a back order
   to view, then click the View Order button.

A Customer Inquiry Information window appears
with the information for that order.

4. When you are finished
   viewing the information for that order, click the Close button.

You return to the Backorders dialog box.

5. Click Close
   button to return to the Customer Inquiry window.

Click the Next
or Prev buttons to view the next or previous
backorder from the search results table.
