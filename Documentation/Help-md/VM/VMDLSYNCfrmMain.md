# Starting the VISUAL Data Import Utility Interactively

You must open the VISUAL Data Import Utility from
directory where your executables are installed.

1. Using Microsoft Windows
   Explorer locate the directory in which your executables are installed.
2. Sorting by type, locate,
   select, and then double-click VMDLSYNC.exe.
   The VISUAL Data Import Utility appears.
