# Allocating Supply Quantities from Incoming Interbranch Transfers and Purchase Orders

1. With the Advanced Material
   Planning Window open and a Part ID selected, left-click anywhere
   on the Firmed Order Details line. The
   Firmed Order Details for Part dialog box appears.
2. Select the Incoming
   Interbranch Transfer or Purchase Order from which you want to
   allocate quantities.

In the Order ID column, Interbranch Transfer
Order IDs are preceded by WH; Purchase Order
IDs by P.

3. From the Edit
   menu, select Allocate. The Assign to
   Demand dialog box appears.

Any allocations to demand, or "demand
links" that you have made from the selected interbranch transfer
or purchase order appear in the line item table.
