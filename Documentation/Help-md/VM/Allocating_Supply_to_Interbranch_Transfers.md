# Allocating Supply to Interbranch Transfers

With the IBT line to which you wan to allocate supply
highlighted, from the Edit menu select Allocations
and then Allocate Supply to Interbranch Transfer.
The Supply Links dialog box appears.

You can allocate supply to interbranch transfers from these sources:

* Co-product supply
* Inventory
* Purchase Order Delivery
  Schedule supply
* Purchase Order supply
* Interbranch Transfer supply
* Work Order Supply
