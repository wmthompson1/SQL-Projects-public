# Printing Inventory Out Reports

An Inventory Out report lists the status of all inventory
that you still own but have moved off-site. Service Dispatches and
Inter Branch Transfers are the two transaction types that allow you
to ship inventory to other locations but still retain ownership of
the shipped materials.

1. From the Inventory menu,
   select Inventory Out Report.

The Print Inventory Out dialog box appears.

2. If you are licensed
   to use multiple sites, click the Site ID(s)
   arrow and select the sites to use. If you are licensed to use
   a single site, this field is unavailable.
3. Enter the dates between
   which you want the report to print information in the Starting
   Date and Ending Date fields.
4. Click the Starting
   Part ID button to select the part with which you want VISUAL
   to begin compiling report information.
5. Click the Ending
   Part ID button to select the part with which you want VISUAL
   to finish compiling report information.
6. Click the Report
   Currency arrow and select the currency for the report.
   If you are printing a report for a single site, the entity currency
   associated with the site is inserted. If you are printing a report
   for multiple sites, the drop-down list shows the tracking currencies
   shared by the selected sites. If no shared tracking currencies
   are found, Not Available is inserted. Either clear site selections
   until a shared currency is found, or run the report in native
   currency.
7. Select how you want
   VISUAL to output the report:

Print - Select
this to view a print preferences dialog box after clicking Ok.

View - Select
this to view the report on your screen after clicking Ok.

File - Select
this to save the report to a file after clicking Ok.

E-mail - Select
this to send the report in a Rich Text Format (.RTF) through electronic
mail. Upon saving, a Send - Print dialog box appears that you can
use to specify to whom you want to send the QRP report information.

8. Select the Service
   Dispatches box to view service dispatch information only.
9. Select the Inventory
   Transfers box to view inventory transfer information only.

|  |  |
| --- | --- |
| POSTIT.gif | You can select both boxes. You must, however, select one of the boxes in order to view, print, or send the report. |

10. Select how you want
    VISUAL to sort the information it gathers and puts in the report:
    By Date, Reason Code, or Part ID.
11. Enter Ship Reason Codes
    in Site Maintenance in the Shipment Tracking tab.
12. From the File menu,
    select Print/View.

Depending on your selection above, VISUAL
now presents a print preferences dialog box (Print), displays the
report on your screen (View), or displays a dialog box that lets you
specify where you want to save this report (File).
