# About the Close/Reopen Period Window

This window is available to licensees of Infor VISUAL
for .NET Global Financials.

Use the Close/Reopen Maintenance to manage the status of accounting
periods. Transactions cannot be posted to a closed period.

## Starting the Close/Reopen Period Window

Select Accounting, Close/Reopen
Period.

## Tasks

This table shows the tasks that you can complete in the Close/Reopen
Period window:

| Task | Description |
| [Closing Accounting Periods (Global Financials)](Closing_Accounting_Periods_(Global_Financials).htm) | Use this procedure to close a period. |
| [Reopening Accounting Periods (Global Financials License)](Reopening_Accounting_Periods_(Global_Financials_License).htm) | Use this procedure to reopen a closed period. |

## Fields

Site
ID

The ID of the site whose accounting periods you are managing.

End
Date of Beginning Acct Period

The end date of the period.

Fiscal
Period

The ID of the period.

Journal
types

After you click the Test button, the number of unposted and posting
candidate transactions for each journal type are displayed. You must
post all transactions before closing a period. Use Post Manufacturing
Journals to post purchase receipts, work in process journal, adjustments,
sales shipments, and indirect labor. Use WIP Maintenance to post WIP
Maintenance transactions. Use Project Journal Adjustment Entry to
post project adjustments.
