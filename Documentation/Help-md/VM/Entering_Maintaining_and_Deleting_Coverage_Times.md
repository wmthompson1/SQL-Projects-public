# Entering, Maintaining, and Deleting Coverage Times

Use the Coverage Time Maintenance dialog box to enter
and maintain coverage times information.

1. From the Maintenance
   menu, select Coverage Hours Maintenance.

The Coverage Hours Maintenance dialog box
appears.

2. To enter a new range,
   click the Insert button.
3. Enter information in
   the following fields:

Start Time - The
start time of the maintenance coverage shift. VISUAL assigns AM unless
you specify PM.

End Time - The
end time of the maintenance coverage shift. VISUAL assigns AM unless
you specify PM.

Coverage Description
- A description of the coverage type. Be detailed in your descriptions.
For example, enter text that helps to clarify what a maintenance engineer
may do during this type of maintenance coverage shift.

4. Click the Save
   button to commit the information.
5. To modify information,
   place the mouse pointer in the appropriate column in the appropriate
   line and manually edit the information. When finished, click the
   Save button.
6. To delete a maintenance
   coverage range, highlight the appropriate line and click the Delete button.

VISUAL places an X at the beginning of the
line.

Click the Save
button to complete the deletion.

7. Click the Close
   button to return to the Equipment Maintenance window.
