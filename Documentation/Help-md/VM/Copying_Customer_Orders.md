# Copying Customer Orders

You can copy information from saved customer orders
to new customer orders. Use this option to avoid repetitive entry
of customer information.

If the customer that is specified on the order is inactive, you
cannot copy the order.

If you are licensed to use multiple sites, you can copy customer
orders within the same site only. You cannot copy a customer order
from one site to a second site.

1. Select Sales, Customer Order Entry.
2. If you are licensed
   to use multiple sites, click the Site ID
   arrow and select the site that contains the order. If you are
   licensed to use a single site, this field is unavailable.
3. With the customer order
   from which you want to copy open, select Copy
   Customer Order from the Edit menu. The Copy Customer Order
   dialog box appears. The Order ID of the order you are copying
   appears in the Copy from CO ID field.
4. Enter a Copy to CO ID,
   or leave the field blank to use auto-numbering.
5. Select the Copy
   Commissions Sharing box if you want VISUAL to copy commissions
   sharing information from the current customer order to the new
   customer order.
6. Select the Copy
   Line Items box if you want VISUAL to copy the line items
   from the current customer order to the new customer order.
7. Select the Copy
   Line Item Link Info box if you want VISUAL to copy line
   item link info from the current customer order to the new customer
   order. This option is only available if you select the Copy Line
   Items box.
8. Select the Copy
   Delivery Schedules box if you want VISUAL to copy delivery
   schedule line items from the current customer order to the new
   customer order.
9. Select the Close
   Order box to close the order after VISUAL copies it.
10. Click the Copy
    button. VISUAL copies the Customer Order.

The Customer Order is copied. The default
status that is set up in Customer Order Entry Preferences is used
as the header status on the new order. If you copied the order lines,
the order lines have a status of Inherit

If the original customer order referenced
inactive ship-to IDs, a message is displayed. The inactive ship-to
IDs are not copied to the new order and are left blank.

When VISUAL has completed copying the customer
order, it asks you if you want to view the new customer order. Click
the Yes button to view the new customer
order.

The header status of the new order is the
default status that is specified in customer order preferences. The
lines have a status of Inherit.

If you selected the Close order box, the
status of the customer order from which you created the new customer
order is now Closed.
