# Adding Departments

1. Complete one of these
   tasks:

* Select Admin, Application Global
  Maintenance. Then, select Maintain,
  Accounting Tables, Department
  Table.
* If you are
  licensed to use Project/A&D functionality, select Project,
  G/L Burden Mapping Maintenance. Then,
  select Maintain, Departments.

2. Click the Insert
   button.
3. Specify this information:

ID Enter an
ID for the department into the ID field.

Description
Enter a description of the ID into the Description field.

4. If you are licensed
   to use Project/A&D functionality, specify this information:

Default To make
the department the default department, highlight the line and click
the Set Default Department button.

Summarize to Department
If the department has a subordinate department (click [here](Specifying_Subordinate_Departments.htm)
for more on subordinate departments), the department ID appears in
this column as a summarized or reporting subordinate.

Material Cost Cat ID
- Double-click the Material Cost Cat ID
column header to select a material cost category ID for the department.

Labor Cost Cat ID
- Double-click the Labor Cost Cat ID column
header to select a labor cost category ID for the department.

Service Cost Cat ID
- Double-click the Service Cost Cat ID column
header to select a service cost category ID for the department.

Use the Project Summary window to enter
and maintain cost category codes for materials, services, and labor.

5. Click the Save
   button.
