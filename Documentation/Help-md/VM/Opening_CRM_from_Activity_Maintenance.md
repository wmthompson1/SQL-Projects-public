# Opening CRM from Activity Maintenance

You can open records related to CRM Tasks directly
from Activity Maintenance. Use the drill-down button in the Related
ID column to open the CRM record. See [Drilling
to Related IDs from Activity Maintenance](Drilling_to_Related_IDs_from_Activity_Maintenance.htm).

You can also open CRM Tasks in CRM Task Maintenance
directly from Activity Maintenance. Use the drill-down button in the
Activity ID/CRM Task ID column to open a task in CRM Task Maintenance.

To use these feature, you must specify CRM connection settings in
Preferences. See [Specifying
CRM Connection Settings](Specifying_CRM_Connection_Settings_for_Activities.htm).

In addition, you must also be a CRM user to open CRM. See the CRM
User Guide.
