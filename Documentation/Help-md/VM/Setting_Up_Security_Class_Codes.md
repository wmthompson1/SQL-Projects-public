# Setting Up Security Class Codes

Security Class codes represent the levels of security
with which you want to class your projects. For example, Classified
and Unclassified.

To set up Security Class codes:

1. In Project Maintenance,
   click the Maintain menu and select Security
   Class Table.

The Security Class table appears.

2. Click the Insert
   button.

The next available row in the table appears
highlighted.

3. Enter the Code and Description
   you want to use.
4. Click the Save
   button.

When you have finished entering Security
Class codes, click the Close button.
