# Setting Workflow Preferences

To access the Workflow Preferences dialog
box, select Preferences from the Options
menu.

You can select the following preferences for
Workflow:

Rotation Angle
Enter the angle to which you want VISUAL to rotate the Workflow window.

Delete on right-click menu
If you want the delete command to appear on the right-click menu,
select the Delete on Right-click menu check box.

Logging Level
Enter the number of undo levels you want VISUAL to track. The greater
the quantity of undo levels, the greater the memory usage.

Make Tasks in
Click the arrow and select where you want VISUAL to make your Workflow
tasks. You can select:

VISUAL
Task List If you want VISUAL to create tasks only in VISUALs
task list, select the VISUAL Task List option.

Outlook
If you use Microsoft Office Outlook and want VISUAL to create
tasks only for Outlook, select the Outlook
option.

Both
If you want VISUAL to create tasks in VISUALs task list and in
Outlook, select the Both option.

ECN and Purchase Requisitions
Tasks Click the arrow and select where you want VISUAL to
create your Engineering Change Notice and Purchase Requisition tasks.
You can select:

* Task Table
  Only
* Outlook and
  Task Table

In the E-mail Settings (SMTP) section, set
the following outgoing e-mail server information:

Auto Discover
If you want VISUAL to detect your settings, select the Auto
Discover check box. If you select this setting, you do not
need to make any other settings.

Server Name Enter
the name of your outgoing e-mail server.

Email Address
Enter the e-mail address you want VISUAL to use as a From e-mail address.

MS Exchange User Name
If you are using MS Exchange, enter the User Name.

Display Name Enter
a From name.
