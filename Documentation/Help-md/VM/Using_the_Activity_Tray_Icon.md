# Using the Activity Tray Icon

You can access Activity Maintenance directly from
the Activity tray icon. You can also hide the icon.

To access Activity Maintenance, perform one of these procedures:

* Click the icon.
* Right-click and select Activity
  Maintenance.

To hide the icon, right-click and select Exit.
The icon is redisplayed if a new activity is detected.
