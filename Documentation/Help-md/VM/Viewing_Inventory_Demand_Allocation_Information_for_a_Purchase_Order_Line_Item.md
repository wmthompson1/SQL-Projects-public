# Viewing Inventory Demand Allocation Information for a Purchase Order Line Item

After you have either fully or partially allocated
demand from a purchase order line to inventory using the preceding
procedures, you can view the allocation information on the purchase
order line level.

For each purchase order line item from which you have attached links
to demand, the following appears:

* In the Base
  ID column, if there is only one inventory link, the Warehouse
  ID appears.

In the Base ID
column, if there are multiple inventory links, Multiple Links appears.

* In the Link
  Part ID column, if there is only one inventory link, the
  Part ID of the link appears.

In the Link Part ID
column, if there are multiple inventory links , nothing appears.

* In the Link
  Warehouse ID column, if there is only one inventory link,
  the Warehouse ID of the part from the purchase order line that
  you allocated to inventory appears.

In the Link Warehouse
ID column, if there are multiple inventory links, nothing appears.

* In the Link
  Quantity Unallocated column, if there is only one inventory
  link, the quantity of the purchase order line that you have yet
  to assign to demand to appears.

In the Link Quantity
Unallocated column, if there are multiple inventory links,
nothing appears.

* In the Total
  Qty Allocated column, the quantity thus far of the purchase
  order line that you have allocated to sources of demand appears.
