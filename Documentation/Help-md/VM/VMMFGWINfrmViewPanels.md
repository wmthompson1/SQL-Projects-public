# Using View Panels

In the Manufacturing Window there are three View Panels
you can use to view information about the current operation, material,
or leg card. Because View Panels are floating panels, you can place
them anywhere on the screen you want. View panels and information
panels are for information purposes onlyyou cannot modify the information
appearing in them

To use View Panels:

From the view menu, select the panel you want
to use.

You can use:

Work Order View Panel

Operation View Panel

Material View Panel

The check mark appears next to the option
you select and the View Panel appears populated with the current card's
information.

If you use the Close Window button in the
top right corner to close the panel, it will appear automatically
when you select the next card of the same type. To stop using the
panel, select the same option in the View menu. The check mark clears
and the panel closes.

## Configuring View Panels

By adding or removing data fields to View Panels, you can tailor
the panel to contain the information that is most relevant for your
needs.

To configure View Panels:

1. Click the Configure
   menu and select one of the following options:

For Operation View Panels:

* Operation Info
* Concurrent Resource
  Infor
* Service Quotes
  Info
* ECN Detail Infor

For Work Order View Panels:

* Work Order Info
* Part Info
* Coproduct Info
* Variable Info
* ECN Detail Info
* Document Info

For Material and Leg View Panels:

* Requirement
  Info
* Part Info
* ECN Detail Info

The Configure Table Window appears populated
with all of the available data fields. The Include column indicates
if the field is hidden from view in the panel or is visible.

2. Select the column you
   want by clicking in the row header.
3. To display a field,
   select the Include check box. To hide
   a field, clear the Include check box.
4. When you have finished
   changing what appears in the panel, click the Ok
   button.
