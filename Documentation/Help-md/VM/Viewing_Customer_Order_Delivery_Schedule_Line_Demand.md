# Viewing Customer Order Delivery Schedule Line Demand Allocation Information for a Purchase Order Line

After you have either fully or partially allocated
demand from a purchase order line to a customer order delivery schedule
line item using the preceding procedures, you can view allocation
information on the purchase order line.

For each purchase order line item from which you have attached links
to demand, the following appears:

* In the Base
  ID column, if there is only one link of customer order
  delivery schedule demand, the Customer Order Delivery Schedule
  ID appears.
* In the Base
  ID column, if there are multiple links of customer order
  delivery schedule demand, Multiple Links
  appears.
* In the Link
  Part ID column, if there is only one link of customer order
  delivery schedule line demand, the Part ID of the purchase order
  line that you have allocated supply from appears.
* In the Link
  Part ID column, if there are multiple links of customer
  order delivery schedule line demand, nothing appears.
* In the Link
  Warehouse ID column, if there is only one link of customer
  order delivery schedule line demand, the Warehouse ID of the customer
  order delivery schedule line appears.
* In the Link
  Warehouse ID column, if there are multiple links of customer
  order delivery schedule line demand, nothing appears.
* In the Link
  Quantity Unallocated column, if there is only one link
  of customer order delivery schedule line demand, the quantity
  of the customer order delivery schedule line to which you have
  yet to assign demand appears.
* In the Link
  Quantity Unallocated column, if there are multiple links
  of customer order delivery schedule line demand, nothing appears.
* In the Total
  Qty Allocated column, the quantity thus far of the purchase
  order line that you have allocated to sources of demand appears.
