# Document Lifecycle in Purchase Order Entry

The Lifecycle Document Viewer shows you documents
linked to the sales or purchasing document that you are currently
viewing. You can drill down to view all linked documents.

You can access the Purchasing lifecycle from Purchase Order Entry.
These documents are included in the Purchasing lifecycle:

* Purchase requisitions
* Purchase orders
* Work orders linked to purchase
  orders
* Purchase receipts
* AP Invoices
* AP Payments

To access the Lifecycle Document Viewer, select Info,
Document Lifecycle.

See [Viewing Document Lifecycles](Viewing_Document_Lifecycles.htm).
