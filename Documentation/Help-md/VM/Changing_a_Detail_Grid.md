# Changing a Detail Grid

After you add a detail grid, you can change the filters
for the detail grid or select a different detail grid to display.
To change a detail grid:

1. Select File,
   Dashboard.
2. In the dashboard, select
   File, Open
   and select the dashboard that contains the detail grid.
3. Right-click the detail
   grid and select Grid Settings.
4. Change the information
   and filters as needed.
5. Click Ok.
