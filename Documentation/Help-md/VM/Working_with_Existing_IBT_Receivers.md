# Working with Existing IBT Receivers

Unlike many other Infor VISUAL objects, you cannot
modify receivers. Each Save or Receive All operation creates a new
receiver. If you try to modify header information on an existing receiver
and click Save, VISUAL prompts you with
a dialog box.

VISUAL interprets Save as an attempt to
receive more goods (to create a new receiver). If you enter new Quantity
Received values when an existing receiver is in the window, another
VISUAL message appears.

Click Yes to create a new receiver. Click
No to cancel the operation.

If you enter a new quantity in the Quantity Receive column after
you have fully received an IBT line, another message box appears.
Click Yes to continue. VISUAL displays the
above message box; click No to discontinue
the process.
