# Deleting a Part in a Single-site Environment

You can permanently remove parts from the database.
Always be sure you want to remove a part before deleting it.

|  |  |
| --- | --- |
| WARNING.gif | Deleting a part permanently removes the information from the database. A deleted part is not recoverable. VISUAL does not allow deletion if the part is used anywhere else in the system or has any transactions against it. |

To delete a part:

1. Call the part you want
   to delete to the Part Maintenance window.
2. Click the Delete
   button.

A dialog box appears, asking you to confirm
the deletion.

3. Click Yes
   to continue, or No to cancel the deletion.

The Part Maintenance window still contains
the part information, but the part has been removed from the database.

4. Click the Clear
   button to reset the window.
