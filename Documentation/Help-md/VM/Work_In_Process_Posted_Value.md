# Work In Process Posted Value

To view the actual posted costs for your work in process,
run the Work in Process Posted Value report.

1. From the Eng/Mfg menu,
   select Work In Process Report. The Work
   In Process Report window appears.
2. Click Print
   Work In Process. The Print Work In Process Value window
   appears.
3. Click the Site ID(s)
   arrow and select the sites to use.
4. In the Selection box,
   select Use Posted Value. The options
   in the WIP costing method box become unavailable.
5. Select Include
   Active ECNs for a complete report.
6. Click Ok.
   The report outputs.
