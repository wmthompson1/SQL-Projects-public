# Assigning a Work Order to a Purchase Order Item

The Assign to Work Order option allows you to manage
multiple work order links. Use it to purchase materials in quantities
to serve work orders and designate purchased materials for jobs.

1. Call up a previously
   created purchase order by entering the PO ID or by clicking on
   the Our Order ID button and selecting
   a PO from the table.
2. Highlight the line of
   interest.

The line item must be for the purchase of
a material, not a service.

3. Select Assign
   to Work Order from the Edit menu.

The Assign to Work Order dialog box appears.

The table displays one line for each link
to a work order, including any link that was created directly on the
PO line.

4. To add a new link:

   1. Click the Insert
      button. A new line appears
      in the table.
   2. Enter Base ID, Lot ID, Split ID for the
      new link.
   3. Enter Seq. #, Piece #.
   4. Click Save to
      save the new link.

5. To modify a link:

   1. Highlight the line to modify.
   2. Change information as appropriate.
   3. Click Save to
      save the changes.

6. To delete a link:

   1. Highlight the line to delete.
   2. Click Delete
      to mark the line for deletion.
   3. Click Save to
      permanently delete the line.

When multiple links are present, the Base
ID field of the line item displays the message Multiple Links. You
can view the links using Assign to Work Order.
