# Adding Carriers

Use Application Global Maintenance to maintain commercial
transport carriers.

1. Select Admin,
   Application Global Maintenance.
2. Select Maintain,
   Carriers.
3. Click Insert.
4. Specify this carrier
   information:

ID - Specify a
unique identifier for the carrier.

Name - Specify
the name of the carrier.

Shipper ID - Specify
the identification of the shipper as assigned by the carrier.

Default Route
- Specify the default routing information.

Default Shipper Instructions
- Specify the standard instructions from the shipper to the carrier.

Default COD Fee
- Specify the standard fee of the carriers charges for collecting
COD payments.

Address fields
- Specify the carrier's address.

Vat Registration
- Specify the registration number of the shipper.

Parcel ID - Specify
a Parcel ID for this carrier. This ID is used to help identify the
carrier for the UPS Connect interface.

Web URL - Specify
the carrier's web address.

5. Click Save.
