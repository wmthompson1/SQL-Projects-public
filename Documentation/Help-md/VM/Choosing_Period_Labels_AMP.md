# Choosing Period Labels in the Advanced Material Planning Window

Use the Show First Day of Period option in the View
menu to specify how to label planning periods. To use the first day
of the period is as the label, select View, Show First Day of Period
until a check mark is displayed next to the option. To use the last
day of the planning period, select View, Show First Day of Period
until the check mark is cleared.

Planning period labels are used as column headers in these areas:

* The Advanced Material Planning
  window grid
* The Master Production Schedule
  dialog grid
* The Planning Forecast dialog
  grid
* The Material Planning Report,
  when run from the Advanced Material Planning window
