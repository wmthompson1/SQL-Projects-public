# Removing Groups from the Personal Menu

To remove groups from your personal menu:

1. Select File,
   Personal Menu from the main VISUAL menu.
2. In the personal menu,
   right-click the group to remove and select Customize
   from the right-click menu.
3. Click the Remove
   Group button.
4. Click the OK
   button.
