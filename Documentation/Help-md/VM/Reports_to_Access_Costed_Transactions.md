# Reports to Access Costed Transactions

Use these reports to assess the transactions that
may be or have been affected by the Costing Utilities.

## Inventory Valuation Report

Run the Inventory Valuation Report to view a break
down of the total value for all parts in your current inventory.

You should run the report twice: once to review project transactions
and once to review non-project transactions. To run the report:

1. From the Inventory menu,
   select the Inventory Valuation Report.
   The Inventory Valuation Report window appears.
2. In the Report Type box,
   select Show Cost Layer Detail. When
   you select this option, the Cost Layer Detail Options and Transaction
   Detail Options become available.
3. In the Selection Criteria
   box, select Part ID if you are running
   your commercial inventory valuation. Select Warehouse
   ID if you are running your project inventory valuation.
4. If you selected Warehouse
   ID in step 3, choose a sort order for your report.

* Warehouse/Part
* Warehouse/Location/Part
* Warehouse/WBS/Part
* Warehouse/Location/WBS/Part

5. If you chose Part
   ID in the step 3, the sort order for the report by default
   is part ID.
6. Select an option in
   the Cost Layer Detail Options box. Your choices are:

* Show all parts
  in range
* Show parts
  with zero cost layers only
* Show parts
  w/neg bal cost layers

We recommend selecting Show all parts in
range.

7. Click the Print
   button. The report outputs.

## Work In Process Posted Value

To view the actual posted costs for your work in process,
run the Work in Process Posted Value report.

1. From the Eng/Mfg menu,
   select Work In Process Report. The Work
   In Process Report window appears.
2. Click Print
   Work In Process. The Print Work In Process Value window
   appears.
3. In the Selection box,
   select Use Posted Value. The options
   in the WIP costing method box become unavailable.
4. Select Include
   Active ECNs for a complete report.
5. Click Ok.
   The report outputs.

## General Journal Report

To view all general journal transactions, run the
General Journal Report.

1. From the Ledger menu,
   select General Journal Entry. The General
   Journal Entry window appears.
2. From the File menu,
   select Print General Journal. The Print
   Journals window appears.
3. In the End Date of Beg
   Acct Period field, enter any date of the period you are reviewing.
   VISUAL automatically populates the field with the last day of
   the period. In the End Date of End Acct Period field, enter any
   date of the period you are reviewing. VISUAL automatically populates
   the field with the last day of the period.
4. In the Sequence box,
   select the order for the report. Your choices are By Account ID,
   By Batch ID and By Batch Type.
5. In the Type box, select
   Detail. When you select detail, the
   options in the Detail box become available.
6. In the Detail box, select
   Costing Detail.
7. Click Ok.
   The report outputs.
