# Accessing Customer Order Specifications

Shipping Entry allows you to view and modify Order
Specifications and Line Specifications associated with the selected
customer order.

To access Order Specs for the current customer order, select Order Specifications/Notes from the Notes menu,
or press Shift+F5.

To access Line Specs for a customer order line, highlight the line
and select Line Specifications/Notes from
the Notes menu, or press Shift+F6.

The notes can remain called up for the whole session. Their contents
change as the current customer order and selected line item change.

You can make changes to these specifications. Any changes are saved
when you click Save or Ship
All.

It is important to realize that you are changing specifications
for the customer order itself, not for the packlist.
