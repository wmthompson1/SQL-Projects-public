# Deleting a Traceability Profile

You can only delete a Part Trace Profile if no traceability
records exist for the Part ID using the profile.

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site that owns the profile. If you are licensed to use a single
   site, this field is unavailable.
2. Call the profile into
   the window.
3. Click the Delete
   button.

The system detects if there are traceability
records for the part, and does not allow deletion if there are.

The window does not clear. If you want to
re-save the profile, click the Save button.
