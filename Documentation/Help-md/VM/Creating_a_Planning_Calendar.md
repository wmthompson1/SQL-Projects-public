# Creating a Planning Calendar

The Planning Calendar dialog box presents each calendar
as a series of rows in a table window. This dialog box has the necessary
Insert, Delete, Save and Close commands, as appropriate.

The data elements handled are:

Calendar # - The CALENDAR\_NO, primary
key of the row.

Description A descriptive entry to remind
the user the purpose of the calendar.

Planning Origin - The Planning Origin
of the calendar. All dates are determined as a function of this date.

Days in Period - The size, in calendar
days, of the planning period.

Total Periods - The number of periods
in the planning horizon. Maximum value is 256.

(Range 1) Days/Period - The size, in calendar
days, of the display range 1 periods.

(Range 1) # of Periods - The number of
periods in display range 1.

(Range 2) Days/Period - The size, in calendar
days, of the display range 2 periods.

(Range 2) # of Periods - The number of
periods in display range 2.

(Range 3) Days/Period - The size, in calendar
days, of the display range 3 periods.

(Range 3) # of Periods - The number of
periods in display range 3.

The total maximum value of all range counts is 256.

This dialog box also supports setting the current calendar at this
workstation.

Click Save to save any changes to the
Planning Calendar dialog box.

## Deleting a Planning Calendar

To delete a existing Planning Calendar, select that particular Planning
Calendar and click the Delete button in
the Planning Calendar table. Click Save
to save any changes to the Planning Calendar dialog box.
