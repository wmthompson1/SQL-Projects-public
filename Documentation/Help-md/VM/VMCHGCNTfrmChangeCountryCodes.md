# About the Change Country Utility Window

If you use free-form country codes in addresses, you
can use the Change Country utility to search for free-form country
codes and replace them with two-letter ISO codes. The two-letter format
is required in BODs.

When you replace a free-form code, all instances of
the code are replaced. You cannot replace the free-form code on certain
documents and leave it intact on others.

The Change Country utility is not available in the VISUAL menu structure.
It must be opened directly from the VISUAL executables directory.
Only the SYSADM user can access the utility.

You must sign in as the SYSADM user to access the utility.

## Starting the Change Country Utility Window

In the VISUAL executables directory, double-click VMCHGCNT.EXE.
Sign in as the SYSADM user.

## Fields

Country ID

 The free-form country ID that you want to replace.

Change Country table

Use the table to specify the ISO code to use in place of the free-form
codes that match your search terms. The table includes these columns:

The Old Country ID / Country
column

The free-form country codes that match the
search criteria.

New Country ID / Country
column

The ISO code to use in place of the free-form
code.
