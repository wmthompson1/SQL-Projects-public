# Viewing IBT Demand Links

You
can view IBT demand links that you assigned in IBT Entry.

Possible demand links include:

* Customer Orders
* Customer Order Delivery
  Schedules
* Work Order Material Requirement
* Interbranch Transfers

Use this feature as a reference to view demand links
that you know exist and as a query tool to view see if there are demand
links from the current IBT to one of the above four sources of demand.

With an IBT in the IBT Receipts Entry window, click the Show Links
button. The Linked Demand dialog box appears.

Order - The ID of the order - Customer
Order, Customer Order Delivery Schedule, Work Order Material Requirement,
or IBT - to which this IBT is linked.

Quantity Allocated - The quantity of the
IBT line that you have allocated to demand.

Quantity Received - The quantity of the
IBT line that you have received.

Quantity Required - The quantity of the
linked order. If there is a demand link to customer order for 25,000,
25,000 appears in this column.

Quantity Issued - If the source of demand
is a work order material requirement, the portion of the allocated
quantity that you have issued to the released work order.

Date Required - The date by which or on
which the linked order needs to be fulfilled.

Resource ID - If the source of demand
is a work order, the Resource ID of the operation.
