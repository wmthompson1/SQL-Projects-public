# BTS-PC Physical Inventory / Cycle Counting

These transactions are similar in scope and provide
the same interface for either one. The standard VISUAL Manufacturing
Physical Inventory window is used and the readers replace the data
entry required to enter the counts of the inventory from the tags
and/or sheets.
