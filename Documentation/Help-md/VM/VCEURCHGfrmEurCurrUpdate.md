# Updating Euro Currency Banks

After you become a member of the EU, you can use the
Euro Currency Update utility to update your Customers, Vendors, and
banks to use Euros.

To update your currency:

1. Navigate to the directory
   where your executables are installed, and double-click the VCEURCHG.exe program name.
2. Select what you want
   to update.

You can select:

Customers

Vendors

Bank
Accounts

A
list of what you selected appears in the table.

3. Select the items you
   want to update or click the Select All
   button.
4. From the File menu,
   select the Start option.

On successful completion of the update process,
a notification dialog box appears.
