# Using the Accounting Menu

If you have applied an Infor VISUAL Global Financials
license to your VISUAL database, the Accounting Menu becomes available.
Use the items available on the Accounting Menu to maintain information
such WIP costs and financial calendars.

Current Infor VISUAL users who do not use Infor VISUAL Financials,
can use these tools to establish a baseline of data for importation
into VFIN.
