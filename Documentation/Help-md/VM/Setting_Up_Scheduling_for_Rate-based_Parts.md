# Setting Up Scheduling for Rate-based Parts

To set up scheduling settings for rate-based production:

1. Select Admin,
   Site Maintenance.
2. In the Site ID field,
   select the site where you produce rate-based parts.
3. Click the Scheduling
   tab.
4. Select the Use
   All Supply Before Applying Lead-time in Material Checks
   check box.
5. Click Save.
6. Select Scheduling,
   Global Scheduler.
7. In the row that contains
   the production schedule for the site you selected in step 2, select
   the Materials Checked check box.
8. Ensure that the row
   that contains the production schedule is selected, then select
   Options, Schedule
   Preferences.
9. Click the Allocations
   tab.
10. Select the Allocate
    fabricated materials check box.
11. Click Ok.
