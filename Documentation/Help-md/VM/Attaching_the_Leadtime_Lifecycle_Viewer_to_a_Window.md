# Attaching the Leadtime Lifecycle Viewer to a Window

You can attach the lifecycle display to the edge of
the application window. Drag the lifecycle to within a few pixels
of the edge of the window. When you release the mouse button, the
lifecycle is attached to the edge. You can attach the lifecycle to
any edge of the window. When you move the window, the lifecycle is
moved with it.
