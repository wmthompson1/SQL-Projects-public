# Selecting a Measurement Basis for the Overall Percent Complete Calculation

1. Perform one of these
   steps:
2. Select Sales, Estimating Window.
3. Select Sales, Customer Order Entry.
4. Select Sales, Order Management
   Window.

2. Select the quote or
   order.
3. If the quote or order
   has multiple lines, select the line.
4. Select Info,
   Leadtime Lifecycle.
5. To measure percent complete
   based on the number of steps in the process, select View, % Overall
   by Process. To measure percent complete based on the number of
   days in the process, select View, % Overall by Days.
