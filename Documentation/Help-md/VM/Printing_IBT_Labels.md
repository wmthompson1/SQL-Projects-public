# Printing IBT Labels

You can print an address label to affix to the shipment.

1. Select File,
   Print Labels.
2. In the Number of labels
   (each line), specify the number of labels to print.
3. In the Format section,
   select the label format to use.

There is one standard format and up to three
user-defined ones. The user defined options are only available if
you have previously created the form layouts.

4. Click the OK
   button.
