# Project Summary Estimate at Completion/Estimate to Complete Comparison

The Project Summary EAC/POC Comparison reports the
percent change between the current version EAC and a summarized ETC.
Use this report to determine which formal EACs need to be updated.
You can also use this report to identify any unusual costs that have
been reported to the actual costs and updated to the ETC.

To run the report:

1. From the Project menu,
   select Project Summary. The Project
   Summary window appears.
2. From the File menu,
   select Print EAC/POC Report. The Print
   EAC/POC Report window appears.
3. In the Print EAC/POC
   Report dialog box, click the Site ID arrow and select the site
   for which to print a EAC/POC Report.
4. To view all projects,
   leave the Project ID fields blank.
5. In the Filter box, leave
   the Project Type field blank to view
   all project types in the report.
6. In the Summarize up
   to field, enter the last day of the period you are closing.
7. Click Ok.
   The EAC/POC report appears.

If the actual amounts do not match the formal amounts, you may want
to update your EAC for the project.

Print the report to use as a guide.
