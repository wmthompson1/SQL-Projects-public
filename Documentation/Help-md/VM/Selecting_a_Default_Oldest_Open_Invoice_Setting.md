# Selecting a Default Oldest Open Invoice Setting

Select a default setting for defining the oldest past due customer
invoice. The system uses this information to help determine if the
customer has an unpaid invoice that exceeds your receivable aging
threshold that you specify in Customer Maintenance. The setting you
specify in Accounting Entity Maintenance is applied to any new customers
you create. You can override the setting in Customer Maintenance.
Select one option:

Determined by Invoice Date If you select
this option, the system uses the date of the invoice to identify the
oldest past due invoice.

Determined by Due Date If you select
this option, the system uses the date by which the full amount of
the invoice should be paid to identify the oldest past due invoice.
