# What is VISUAL BTS Material Tracking?

VISUAL/BTS is a sophisticated software application
that can help you manage materials through the use of barcode hardware.

## VISUAL/BTS features:

* Graphical User Interface
  (GUI) for label design and printing
* Simple DOS based reader
  applet for use on many different Barcode reader platforms
* ODBC Compliant connections
  to any and all databases
* Standard industry compliant
  labels and thermal label printers
* Completely customizable
  user interface for a variety of reader formats

You can perform the following tasks using VISUAL/BTS :

* Purchase Receipt Entry
* Material Movement within
  Facilities
* Issue of Raw materials to
  Workorders
* Receipt of Finished Goods
  to Inventory
* Physical Inventory and Cycle
  Counting
