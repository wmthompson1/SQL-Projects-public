# Setting Service Charge Defaults

Use Site Maintenance to set any service charges your
company may charge.

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site for which you are setting up service charge defaults. If
   you are licensed to use a single site, this field is unavailable.
2. Select Service
   Charge Defaults from the Maintain
   menu.

The Service Charge Maintenance dialog box
appears.

3. In the ID field, enter
   an ID for the service charge.
4. In the Description field,
   enter an description of the service charge.
5. In the Unit Price field,
   enter the unit price of the service charge.
6. In the Trade Discount
   % field, enter any discounts.
7. In the Commission %
   field, enter commissions.
8. From the Product Code
   drop down box, select the product code for which you are setting
   this service charge.
9. From the Commodity Code
   drop down box, select the commodity code for which you are setting
   this service charge.
10. In the Sales Tax Grp
    ID field, enter a sales tax group ID for the service charge.
11. In the Revenue Acct
    ID field, enter the account into which the service charge will
    be received.
12. Click the Save
    button to save the service charge information.
