# Opening Multiple Engineering Masters

To open multiple Engineering Masters in the Manufacturing windows:

Right-click the card referencing the master you want to open and
select Open Engineering Master from the
right-click menu.

## Controlling How New Windows Appear

To control how new menus appear:

1. Select MDI Window Preferences
   from the Options menu.

The MDI Child Window Management dialog appears.

2. Select where you want
   VISUAL to open the new window.

You can select either:

New Window - If
you want the old windows o remain when the new window opens, select
New Window.

Replace Current Window
- If you want VISUAL to close the old window and replace it with the
new window, select Replace Current Window.

3. If you selected New
   Window, select where you want the new window to appear.

You can select:

Show active window maximized
- VISUAL minimizes (but not closes) all open windows and the new window
appears maximized.

Tile Vertical
- VISUAL keeps all windows open and fits them in the Manufacturing
Window one above the other.

Tile Horizontal
- VISUAL keeps all windows open and fits them in the Manufacturing
Window side by side.

Cascade - VISUAL
keeps all windows open and stacks each window on top of the previous
one but slightly to the right and below the previous window.

4. Click the OK
   button
