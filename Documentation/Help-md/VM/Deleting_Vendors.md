# Deleting Vendors

|  |  |
| --- | --- |
| WARNING.gif | Deleting a vendor permanently removes the information from the database. It cannot be recovered. You cannot delete the vendor if a PO Receiver, A/P Payable, or any other object refers to it. |

1. Call the vendor you
   want to delete into the Vendor Maintenance window.
2. Click Delete.

A dialog box appears, asking you to confirm
the deletion.

3. Click Yes
   to continue, or No to cancel the deletion.

The Vendor Maintenance window still contains
the vendor information, but the vendor is not longer in the database.

4. Click Clear
   to reset the window.
