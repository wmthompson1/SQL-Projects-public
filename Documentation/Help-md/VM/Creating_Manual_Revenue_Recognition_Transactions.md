# Creating Manual Revenue Recognition Transactions

If a projects contract type requires you to enter revenue recognition
transactions manually, use the following steps:

1. From the Project menu,
   select Project Revenue Recognition.
   The Project Revenue Recognition window appears.
2. Leave the Transaction
   ID field blank. VISUAL populates this field for you when you save
   the record.
3. In the Project ID field,
   enter the ID of the project for which you are recognizing revenue.
   VISUAL populates the Current EAC and Revenue Basis fields.
4. Select the currency
   for the revenue transaction by clicking the Currency
   ID arrow and selecting a currency from the list.
5. Enter the date of the
   transaction in the Transaction Date field.
6. In the table below the
   header, change the G/L account ID, if necessary. Double-click
   the G/L Account ID column header and
   select a new account.
7. Enter the credit and
   debit amounts for this transaction. As you enter information in
   the table, VISUAL updates the information in the Totals tab.

After you post the revenue recognition transaction
to the General Ledger, you cannot edit the information in the table.

8. When you have finished
   entering information, click the Save
   button.
9. To post the revenue
   transaction to your general ledger, click the Post
   to G/L button.
10. Click the Close
    button to exit the Project Revenue Recognition window
