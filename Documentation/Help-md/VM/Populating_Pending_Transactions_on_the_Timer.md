# Populating Pending Transactions on the Timer

After you have set you timer preferences, you can
make it so the timer checks for pending transactions now eligible
for completion.

From the Timer Options menu, select Refresh
on Timer.

A check mark appears next to the menu selection.

Your timer settings now dictate when VISUAL BTS checks for new purchase
receipt transactions.
