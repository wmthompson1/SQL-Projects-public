# Viewing Current MRP for a Part

Click anywhere on the Current MRP line. The Planned
Order Details for Part dialog box appears. You cannot edit any of
the values in these columns.

The line item table contains the following columns:

Order ID - The ID of the order.

Order Quantity - The work order quantity.

Want Date - The date on which the customer
wants the order.
