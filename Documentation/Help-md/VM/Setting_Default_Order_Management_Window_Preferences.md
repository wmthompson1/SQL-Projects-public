# Setting Default Order Management Window Preferences

You can set the default status of new customer orders
to Firm, Released, or On-hold. VISUAL assigns only new orders this
status; existing order are not affected by this setting. You can also
set how you want the Order Management window to appear, if at all,
upon opening the application.

To set these preferences:

1. From the Order management
   window Options menu, select Preferences.

The Preferences dialog box appears.

2. Select the appropriate
   status option:

Firm - Select
this option button if you want the default status of all new orders
to be Firm. A firmed order, until you change its status, exists for
planning purposes only.

Planners and schedulers use this designation
to test scheduling and planning schemes. When the order has been placed
in a schedule, or approved, the status normally changes to Released
and work begins

Released - Select
this option button if you want the default status of all new orders
to be Released. Released orders are orders on which work may begin
immediately.

On-hold - Select
this option button if you want the default status of all new orders
to be On-hold. Work cannot begin on on-hold orders until the status
changes appropriately. You may want to set this as the default if
you are currently employing an order fulfillment scheme that makes
few checks on available resources and materials. A status of On-hold
assures that work will not begin until you specify otherwise.

3. Enter the number of
   ship days leadtime you want the Order Management window to display.
4. Click the Ok
   button to commit these preferences.
