# Allocating Supply Quantities to Customer Order Demand

1. With the Advanced Material
   Planning Window open and a Part ID selected, left-click anywhere
   on the Customer Demand line. The Customer
   Order Details for Part dialog box appears.
2. Select the customer
   order to which you want to allocate supply.
3. From the Edit
   menu, select Allocate. The Supply Links
   dialog box appears.

Any allocations of supply, or "supply
links" that you have made to the customer order appear in the
line item table.
