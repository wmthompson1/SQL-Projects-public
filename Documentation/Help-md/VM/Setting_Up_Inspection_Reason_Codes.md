# Adding Inspection Reason Codes

Use Inspection reasons to specify why parts and materials
are inspected.

1. Select Admin,
   Application Global Maintenance.
2. Select Maintain,
   Reasons Maintenance, Issue/Return/Transfer
   Reasons.
3. Click Insert.
4. Specify an ID and description
   of the inspection reason.
5. Click Save.
