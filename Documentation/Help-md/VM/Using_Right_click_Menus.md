# Using Right-click Menus

Right-click menus allow you to open properties dialog
boxes, delete items, and manage objects in the work area.

To use right-click menus:

1. Right-click the work
   area, rule, or step.

A menu appears.

2. Select an item from
   the menu.

For example, to access the properties of
an object in the work area, right-click the object and select Properties. The Properties dialog box appears.
