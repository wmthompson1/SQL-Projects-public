# Maintaining Unit of Measure Categories

Note: Unit of Measure Categories
will be used with Infor Evolve components. They are not currently
used within VISUAL.

Use Unit of Measure Categories to classify your units of measure.

1. From the Maintain menu, select UOM
   Categories.
2. Click Insert.
3. Enter a code and description for the category.
4. Click Save.
