# Setting Exchange Rates for Individual Purchase Requisitions

When you initiate a purchase requisition for a vendor
that uses a foreign currency, VISUAL uses the vendors default currency
and the current exchange rate from Currency Exchange Rate Maintenance
to calculate the value of the requisition. Use this feature to override
the default price lookup and fix the exchange rates on a requisition-by-requisition
basis.

1. With the appropriate
   order open in the Purchase Order Requisition Entry window, select
   Set/Reset Exchange Rates... from the
   Edit menu.

The Effective Exchange Rates dialog box
appears.

The dialog box displays the currency of
vendor from the requisition and the current sell and buy rates for
the specified currency.

2. Enter the date from
   which you want to check currency valuations into the Effective
   Date field or click the calendar icon to select a date.
3. Make any necessary modifications
   to the appropriate currency line.

For example, if the vendors default currency
is Lira, adjust the value of either the Foreign to USD or USD to Foreign
field to reflect a new or projected exchange rate.

4. Click Close
   when you are done editing or viewing exchange rates.
