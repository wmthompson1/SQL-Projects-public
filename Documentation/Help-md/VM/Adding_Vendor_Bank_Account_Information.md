# Adding Vendor Bank Payment Information

Enter Payment information for the Vendor Bank Account
on the Payment tab.

In the Vendor Bank Accounts dialog box, click the Payment
tab.

Enter the following information:

Bank Account ID Enter or select the
ID for the Bank Account to which you want to add payment information.
If you select a Bank Account ID on the Address tab, then click the
Payment tab, the same Bank Account ID carries over.

Description Enter a brief description
to help you identify the payment bank. The Description you enter here
appears on the table that appears when you click the Bank Account
ID browse button on the Vendor Maintenance Payment tab. Also, this
is the description that appears to the right of the Bank Account ID
field on the Payment tab. This information appears on both the Address
and Payment tabs.

Bank ID Enter an identification code
for the bank to use to help you identify the bank. This may be the
branch name of the bank, or some other name.

Account # Enter your companys account
number. This was provided to you when you open your account at the
bank.

Account Type Enter the type of account
you have at the bank. This may be a checking account, for example.

Branch ID Enter the ID for the specific
branch where your bank account is located. This may be your bank routing
number.

IBAN Enter the International Bank Account
Number for this bank.

BIC Enter the Bank Identification Code
for this bank.

If you have not already done so, enter the Address information for
your bank on the Address tab, as described in [Entering
Vendor Bank Account Information](Entering_Vendor_Bank_Information.htm).

To save the Vendor Bank account Payment information, click Save. To exit the dialog box, click Close.
