# For Transfer Quantities Between Locations

A transfer is essentially an automatic double-adjustment,
an adjust out from one location and an adjust into another. This applies
when the Adjustments option is checked in When to Apply section for
the parts traceability profile

You cannot adjust between lot numbers; you can only move their quantities
between locations.

All lot numbers for the Part ID that has unissued quantities appear.
You can only transfer material that is still in inventory. Total Quantity
shows the transfer quantity.

Enter the actual quantities of lot numbers being transferred. Zero
out any quantities for lot numbers that are not being transferred.
