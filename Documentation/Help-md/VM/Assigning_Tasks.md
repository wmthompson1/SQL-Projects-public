# Assigning Tasks

To assign tasks:

* In Purchase Requisition
  Entry, assign a user in the Assigned To field or assign a user
  or group to one of the four approval fields on the Approvals tab.
  See [Initiating Purchase Requisitions](VMREQENTfrmPurchaseReq.htm).
* In ECN Entry, assign a user
  in the Assigned To field or assign a user or group to one of the
  teams on the ECN header. See [Entering
  Basic Information into ECNs](Entering_Basic_Information_into_ECNs.htm).
* In Workflow, create a rule
  that consists of a task. Assign users or groups to the task. See
  [Adding Rule Properties](Adding_Rule_Properties.htm).
