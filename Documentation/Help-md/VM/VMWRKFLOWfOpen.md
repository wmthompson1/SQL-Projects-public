# Starting Workflow

To start
VISUALs Workflow Designer:

1. If necessary, start
   VISUAL.

On the Admin menu of VISUALs main window,
click Workflow Designer.

2. The Workflow Designer
   window opens with the Open Existing Workflow dialog box.
3. Do one of the following:

* To open an existing
  Workflow template, refer to [Opening
  an Existing Workflow Template](Opening_an_Existing_Workflow_Template.htm).
* To create a
  new Workflow template, refer to [Creating
  a New Workflow Template](Creating_a_New_Workflow_Template.htm).
