# Table Section- Purchase Receipt Entry

Depending on your table preferences the following
columns may be available:

* [Ln#](javascript:void(0);)
* [Reference](javascript:TextPopup(this))

  Displays the contents of the Misc Ref/Descr column from
  the customer order. If the line is for an inventory part,
  Reference will be labeled Part, followed by the part description.
  If the line is linked to a work order, Reference will be labeled
  job, followed by the work order Base ID/Lot ID, and the Part
  ID, if there is one. If the line is a miscellaneous item,
  Reference will be labeled Misc.
* [Trc](javascript:TextPopup(this))

  This indicates whether the part is traceable.
* [Vendor
  Part ID](javascript:TextPopup(this))

  The vendors specification, if any, for the ordered part.
  This is defined in Vendors Supplying this Part, in Part Maintenance.
  This simplifies ordering when your internal part numbering
  scheme is different than the vendors catalog number.
* Quantity Ordered
* P.O. Pieces
* P.O. Length
* P.O. Width
* P.O. Height
* [Total
  Qty Dispatched](javascript:TextPopup(this))

  The total quantity that was dispatched for this order.
* [Total
  Qty Recv](javascript:TextPopup(this))

  The total quantity received on this receipt. A numeric entry
  is required. The default is zero.
* Quantity Received
* [Recv
  to Inspect](javascript:TextPopup(this))

  Check this box if the current referenced line item requires
  inspection upon receipt.
* [# Pieces
  Received](javascript:void(0);)
* [Length](javascript:TextPopup(this))

  Specify the length dimension in this field.
* [Width](javascript:TextPopup(this))

  Specify the width dimension in this field.
* [Height](javascript:TextPopup(this))

  Enter the height dimension in this field.
* [Dimensions
  UM](javascript:TextPopup(this))

  Specify the unit of measure that you are using for dimensional
  inventory. You can also click on the Dimensions UM button
  and select a unit of measure from the table.
* [Unit
  of Measure](javascript:TextPopup(this))

  The unit of measure for this line item.
* [Last
  Recv Date](javascript:TextPopup(this))

  Maintained by the system. This column indicates the date
  the line item was received.
* [Location
  ID](javascript:TextPopup(this))

  Location within the selected warehouse where the part is
  stored. This is an automatic field, but can be changed. If
  the entry is changed, it must match one of the locations where
  the part is stored.
* [Warehouse
  ID](javascript:TextPopup(this))

  Specify the warehouse ID of the warehouse whose parts you
  want to count. When you Initialize Physical Count, only parts
  belonging to this warehouse are included in the count list.
* Receive On
* Promise Date
* [Close
  Short](javascript:TextPopup(this))

  Check this box if the current referenced line may be closed
  short.
* [Del Sched](javascript:void(0);)
