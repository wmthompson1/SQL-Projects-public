# Configuring the Customer Service Impact Detail Dialog Box Line Item Table

You can configure the Customer Service Impact Detail
dialog box line item table to hide or show columns.

From the Customer Service Impact Detail dialog box Options menu,
select Configure Line Item Table or right-click
anywhere on the dialog boxs surface and select the same from the
pop-out menu

The Configure Table window appears.

Using the up and down arrows, scroll through the window.

If you want to hide a visible column, highlight the line and click
the Make Hidden button.

If you want to make visible a hidden column, highlight the line
and click the Make Visible button.

To hide or make visible selected columns that are not contiguous,
hold down the Ctrl key and left mouse, click
each line you want to effect, and then click the Make
Hidden or Make Visible button as appropriate.

To hide or make visible every column at once, click the Select
All button and then click the Make Hidden
or Make Visible button as appropriate.
