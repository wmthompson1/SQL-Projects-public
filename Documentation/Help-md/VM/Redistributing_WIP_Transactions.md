# Redistributing WIP Transactions

If you make any changes or revisions to a WIP transaction,
select this feature to recalculate and redistribute the transactions.
VISUAL provides this functionality as a way to "refresh"
and update the database tables. You can only apply to a previously
saved un-posted transaction.

If you want to apply a change, you must execute the Redistribute
Transactions option for each transaction as necessary.

1. Choose the Transaction
   ID containing the lines you want to redistribute.
2. Make the necessary adjustments.
3. Click the Save
   toolbar button to save the changes.
4. With the un-posted transaction
   open, select Redistribute Transactions
   from the File menu.

VISUAL automatically updates the distribution
record (invoice) of the modifications.
