# Setting Notification Maintenance Window Preferences

To set the Notification Maintenance window
preferences:

1. Select Options,
   Preferences in the Notification
   Maintenance window.

The Preferences window
is displayed.

2. Select the appropriate
   options from these fields:

* Default
  Tab - Select one of the options from the list. Possible
  values:
* Customer
  Order
* Shipment
* Cash Receipt
* Purchase
  Order
* PO Receipt
* Cash Disburse
* Email
  AR Invoices

3. Manual
   Override for Notifications - Select the check boxes for
   the documents to override the automatic notification email. Possible
   values:

* Manually
  Send Customer Order Notification
* Manually
  Send Shipment Notification
* Manually
  Send Cash Application Notification
* Manually
  Send Purchase Order Notification
* Manually
  Send Receiver Notification
* Manually
  Send Cash Disbursement Notification

3. Add
   Prompt for Manual Override for Notification - Select the
   check boxes for the documents that you want a reminder prompt
   to send the notification when you are saving that document. Possible
   values:
4. Prompt
   on Customer Order Save for Manual Notification
5. Prompt
   on Packlist Save for Manual Notification
6. Prompt
   on Cash Receipt Save for Manual Notification
7. Prompt
   on Purchase Order Save for Manual Notification
8. Prompt
   on Receiver Save for Manual Notification
9. Prompt
   on Cash Disbursement Print for Manual Notification

3. Click Ok
   to commit the selected preferences.
