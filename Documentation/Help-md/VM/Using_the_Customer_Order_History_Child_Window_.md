# Using the Customer Order History Child Window

Use the Customer Order History window to view a customers
order history, instantly recall and open a customer order, regardless
of how long a period of time has passed since order fulfillment, and
append and copy an entire order or a line from that order to the current
order. Because customers may often order the same parts or materials,
the copy/append feature makes order entry nearly effortless by allowing
you to transfer existing customer default and preference information
to new orders.
