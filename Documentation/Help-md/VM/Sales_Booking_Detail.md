# Sales Booking Detail

This [dashboard](VMDBOARDWhat.md) detail
grid lists customer orders.

Date filter applies to: Customer Order
Date

Filters: Customer ID, Sales Rep ID, Territory,
Product Code, Commodity Code, Part ID, Site ID

Red text: n/a

Details: Customer ID, Customer Name, Cust
Order ID, CO Line No, CO Del Ln No, Cust Order Date, Desired Ship
Date, Status, Order Qty, Selling U/M, Unit Price, Trade Disc. Pct.,
Amount Ordered, Currency ID, Part ID, Part Desc., Service Charge ID,
Sales Rep, Territory, Product Code, Commodity Code, Site ID.
