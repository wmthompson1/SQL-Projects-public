# Vendor Performance Detail

This [dashboard](VMDBOARDWhat.htm) detail
grid shows the number of purchases you received early, the number
received on time, and the number received late.

Date filter applies to: Received Date

Filters: Type (On Time, Early, Late),
Vendor ID, Buyer, Product Code, Commodity Code, and Part ID, Site
ID

Red text: n/a

Details: Type (On Time, Early, Late),
Days Diff., Vendor ID, Vendor Name, Receiver ID, Rcvr Ln No, Rcvr
Del Ln, Purc Order ID, PO Ln No, PO Del Ln No, Purc Order Date, Desired
Recv Date, Received Date, Received Qty, Order Qty, Purchase U/M, Status,
Buyer, Product Code, Commodity Code, Part ID, Part Desc., Service
ID, Service Desc., Site ID
