# Renumbering Requisition Lines

Use this feature to renumber lines in the table portion
of the Purchase Requisition Entry window after deleting a single line
or multiple lines.

To renumber line items, select Renumber Lines
from the Edit menu.

VISUAL refreshes the display and renumbers the remaining in ascending
order.
