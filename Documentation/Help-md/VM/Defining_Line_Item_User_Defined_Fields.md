# Defining Line Item User Defined Fields

Unlike header-level user definable fields, you cannot
assign Label IDs to line-item level fields. You can, however, change
column labels by using the Configure Line Item Table option to manually
modify column titles.

1. From any of the applications
   that support field-level user defined fields, select Configure
   Line Item Table from the Options menu.
2. The Configure Line Item
   Table dialog box appears.
3. Scroll down until you
   can see the user-defined columns.

They appear as colUSER\_1, colUSER\_2, and
so on.

4. Place the cursor in
   the Title field and modify the default Title as appropriate. For
   example, you could change a user-defined title to Color and use
   the column to describe part color schemes.

Depending on how many fields you want the
line item table to contain, modify fields 1-10 as necessary.

5. When finished, verify
   that the columns are visible and click Save.

The fields now appear in the line item table.
