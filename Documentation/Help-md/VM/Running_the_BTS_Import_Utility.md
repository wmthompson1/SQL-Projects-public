# Running the BTS Import Utility

1. From the directory in
   which you keep your VISUAL BTS executables, double-click Vmbtsimp.exe.

The BTS Import Utility appears.

2. From the File Format
   section, select either the CSV or XML option button.

This is the type of file you want to import.
You are not bound to import this type of file; VISUAL opens up an
Open dialog box with the Files of Type selection
set to either of these two possible choices.

3. Click the Open
   button on the main toolbar.

The Open dialog box appears.

4. Locate the appropriate
   file. If you cant find the file, adjust your Files of Type preference
   and continue to search. A text file search is possible as well.

The path and the file appear in the File
Name data field.

5. If the imported document
   contains a transaction type, select the Transaction
   Type in File check box.

If the file does not contain a transaction
type, clear the check box and select the transaction type from the
below list box. VISUAL attaches this transaction type to the import
file.

6. After verifying that
   all information is correct, click the Import
   button on the main toolbar.
7. After importing the
   file, VISUAL displays the file for you to view. Verify that the
   information imported correctly and click the Ok
   button to return to the main window.
