# Using the Close All Function

It is possible to access several functions within
Infor VISUAL without closing other functions. You may decide that
you want to close all open windows and continue with your operations.
The Close all function from the Infor VISUAL
File menu allows you to exit from all active windows from Infor VISUAL,
however, the Infor VISUAL Main window remains open.
