# Marking Billing Events for Invoice Generation

When you reach the point at which you need to generate
an invoice for billing events, you must first mark the event as being
ready for invoicing.

To mark billing events:

1. Open the Project Billing
   window.
2. Select the Order ID
   containing the event you want to mark.
3. Select the line containing
   the billing event you want to mark and click the Edit
   Billing Events icon on the toolbar.
4. For the billing event
   you want to mark, select the Generate Invoice
   check box.
5. When you have finished
   marking billing events, click the Save
   and Close buttons.
