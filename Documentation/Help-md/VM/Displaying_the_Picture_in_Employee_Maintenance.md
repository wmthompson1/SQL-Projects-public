# Displaying the Picture in Employee Maintenance

If you [add
a picture](Adding_a_Picture_in_Employee_Maintenance.htm) to an employee record, you can display the picture in
the Employee Maintenance window. To display the picture, set up the
ShowPicture preference setting.

To set up the preference:

1. Select Admin,
   Preferences Maintenance.
2. Click Insert.
3. Specify this information:

Section Specify
EmployeeMaintenance.

Entry Specify
ShowPicture.

Value Specify
Y.

4. Click Save.
