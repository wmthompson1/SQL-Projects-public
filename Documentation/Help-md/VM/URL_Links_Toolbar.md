# URL Links Toolbar

You can replace the VISUAL splash screen with a URL. You can display
an external web site, a custom HTML page you create, a SharePoint
site, a browse-based application, and so on. You can define up to
three URLs to use, and you can switch between the websites using toolbar
buttons that you specify.

Set up URL links in Preferences Maintenance.

Note: To be displayed properly in VISUAL,
web sites must be compatible with Internet Explorer 7 or earlier.
