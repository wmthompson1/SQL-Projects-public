# Copying a Preference Template to a User

Only a user with system administration privileges can perform this
procedure.

After you create a preference template, you can copy the template
settings to a user. When you copy a template to a user, the user inherits
the preferences in the template. You can then edit the individual
users preferences as necessary.

You can only copy a preference template to a user that has no preferences
specified. If you would like to copy a template to a user that has
preferences set, you must first delete all of the users preferences.
Refer to Deleting All Preferences from a User.

To copy a preference template to a user:

1. Select Admin,
   Preferences Maintenance.
2. Select Edit,
   Copy Preferences.
3. Click the Copy
   From arrow and select the template to copy.
4. In the Copy to User
   ID field, specify the ID of the user.
5. Click Copy.
   The system copies the template settings to the specified user.
