# Using VISUAL Price Book

VISUAL Price Book is a distributed architecture that
uses Windows DNA. Within Infor VISUAL, the Order Management Window
interfaces with VISUAL Price Book. Additionally, VPB relies on source
customer, customer order, and part information from your ERP software.
Though VPB uses a distributed architecture, all applications behave
in an integrated fashion. Generally, all applications share the same
information rather than duplicating it between the various databases.
In some cases, however, certain application objects have a meaning
or definition specific to their native database.
