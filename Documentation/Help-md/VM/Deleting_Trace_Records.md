# Deleting Trace Records

Deleting trace records is only allowed when no inventory
transactions reference them. This is useful for removing preassigned
lot numbers that you no longer use. Quantity Received, Quantity Issued,
and Quantity Reported must all be zero.

To delete one or more trace records:

1. Select the rows you
   wish to delete.

Hold down the shift key if you wish to select
individual, noncontinuous rows.

2. Click the Delete
   Lines button.
3. Click the Save
   button to delete the trace record.

VISUAL informs you if any of the trace records
have associated transactions. The deletion does not take place.
