# Starting Task Maintenance

Task Maintenance is available from two menus: the
Purchasing menu and the Eng/Mfg menu.

To start Task Maintenance:

From the Purchasing Menu, select Task Maintenance.

From the Eng/Mfg menu, select Task Maintenance.

The Task Maintenance window appears, displaying all tasks for the
user currently signed into the database.
