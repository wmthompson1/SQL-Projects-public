# Reopening the Customer Order

When you open the customer order after running the
Allocation utility, "Multiple Links" appears in the Supply
Type column in the Customer Order Entry window line item table. This
indicates that VISUAL obtained supply for this customer order from
more than one source. In this case, inventory, a purchase order, and
an unreleased work order.

With the Customer Order entry window open, highlight the order line
and select Assign Supply to Customer Order line.

The Supply Links dialog box appears.

The three links of supply the Allocation Utility created appear
in the line item table. See Customer Order Entry for more information
on establishing [supply
links to customer orders](Assigning_Supply_to_Customer_Order_Lines.htm).
