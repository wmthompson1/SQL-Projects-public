# Making WIP Adjustments

|  |  |
| --- | --- |
| POSTIT.gif | Start the WIP Adjustment Entry window by double-clicking on the file name in your VM program directory. |

To enter WIP transactions:

1. Click the New
   toolbar button to create a new transaction.

The current employee's ID appears in the
Employee ID field.

2. Click the Site
   ID arrow and select the site to use.
3. Click the Transaction
   Date Calendar button and select the appropriate date. The
   current date appears
4. Click the Calendar
   button beside the Posting Date field to select a posting date,
   or enter the posting date in the field provided.
5. Click the Work
   Order ID button and select the work order against which
   you want to make this transaction.
6. Make the appropriate
   count changes to the lines of the work order you selected.
7. Click the Save
   toolbar button.
