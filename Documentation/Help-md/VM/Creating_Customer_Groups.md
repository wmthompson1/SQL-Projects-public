# Creating Customer Groups

Each customer ID can belong to only one customer group.

To add a customer group:

1. Select Sales,
   Customer Group Maintenance.
2. In the header, specify
   this information:

Customer Group ID
Specify an ID for this customer group.

Group Name Specify
a description of the customer group ID.

Bill to Address
Specify the bill to address for the customer group.

3. To add customer records
   to the customer group, click the Insert
   button.
4. Double-click the Customer ID browse button and select the customer
   to add to the group. Each customer ID can belong to only one group.
5. Click the Accounting
   tab and specify these settings:

Allow Invoice Over Payments
Select this check box to allow this customer group to make payments
that exceed the invoice balance.

Allow Closed Period Invoice
Adjustments Select this check ox to allow adjustments to
invoices for customers in this customer group even if the period is
closed.

6. Click Save.
