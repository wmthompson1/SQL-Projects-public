# Individual Privacy Report

The Individual Privacy report contains this information:

* The data group where the
  data subjects data was found
* The primary key of the document
  that contains the data subjects data
* The data subjects name
* The data subjects address
* The data subjects phone
  number
* The data subjects email

You can generate the report for more than one data subject. If the
data subjects information is used on multiple documents in your database,
you can include multiple documents in the report.
