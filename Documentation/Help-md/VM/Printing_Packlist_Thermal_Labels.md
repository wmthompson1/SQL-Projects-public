# Printing Packlist Thermal Labels

|  |  |
| --- | --- |
| IMPORTNT.gif | You must define Label formats in the VISUAL Automated Material Tracking System Barcode Labels/Printer Setup Utility - VMBTSLP. If you are not using VAMTS, disregard this section. If you are using VAMTS LP, click [here](What_is_the_Label_Printer_Setup_Utility_.md) for more information on defining Label formats using VISUAL Database tables. |

If you are using VAMTS, you can print packlist thermal labels for
barcoding purposes in Shipping Entry.

1. From
   the File menu, select Print Packlists.

The Print Packlists dialog box appears.

Click the Thermal Labels
button.

The Print Labels dialog box appears

2. Click
   the Label ID button
   to select the Label Format you want to use for packlist thermal
   labels.
3. In
   the # of Labels fields, enter the number of labels you want to
   print.
4. Using
   the Thermal Printers list box, select the appropriate thermal
   printer. This is the thermal printer in your network neighborhood
   to which you want to send the print order.
5. In
   the Spool Into section, click the browse button to select where
   you want your thermal print order to spool before printing.
6. Click
   Ok when
   you are finished setting Packlist Thermal Label Printing information.
