# Specifying 3PL Customers

|  |  |
| --- | --- |
| POSTIT.gif | The 3PL tab is not visible unless you have the appropriate VISUAL Distribution license key. |

Use the 3PL (Third Party Logistics) tab to specify valid third-party
customers who take part in distribution of goods to the primary customer.

Some customers operate as a 3PL warehouse, meaning that an outside,
or 3rd-party, owns the
inventory handled at the distribution centers. The warehouse owners
rent the use of the facilities and equipment. In order to distinguish
inventory owned by different parties, VISUAL allows you to specify
an Owner ID to accompany the inventory at all stages of the life cycle.
For 3PL customers, Owner IDs exist on purchase orders, customer orders,
and all inventory transactions in between.

To specify 3PL customers for a customer:

1. Click the 3PL
   tab.
2. Click the Insert
   button to add a new line to the table.
3. Double-click the <Sell to Customer ID> column header
   to select the 3PL customer participating in the delivery of goods
   to the primary customer.

VISUAL auto-fills the Sell to Customer Name
field with the customers proper name.

4. Click the Save
   button on the main toolbar or select Save
   from the File menu to commit the 3PL customer association to the
   customer record.
