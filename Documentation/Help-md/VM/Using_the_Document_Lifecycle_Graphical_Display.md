# Using the Document Lifecycle Graphical Display

If the Tree View is displayed when you open the Document Lifecycle
Viewer, select View, Show Graphical Display.

A card is displayed for each document in the lifecycle. You can
customize the size of the cards and the information displayed on the
cards. On certain cards, a completion meter is shown.

Lines are drawn between cards to show the connections between the
transactions.

If a document is attached to a transaction, then a paper clip is
displayed next to the card. Double-click the paper clip to view the
documents. If an activity is attached to a transaction, then the activity
button is displayed next to the card. Double-click the button to view
the activities attached to the transaction. See [What
is Activity Maintenance](VMACTMNTWhat.md).

In the Graphical View, you can perform these actions:

* [View
  information about documents in the lifecycle](Viewing_Information_about_Documents_in_the_Lifecycle.md)
* [Customize
  information on the cards](Customizing_Document_Lifecycle_Graphical_Display_Card_Information.md)
* [Customize
  the size of the cards](Customizing_Document_Lifecycle_Graphical_Display_Card_Size.md)
* [Customize
  the color of the cards](Customizing_Document_Lifecycle_Graphical_Display_Card_Color.md)
* [Attach
  the Lifecycle Viewer to a window](Attaching_the_Lifecycle_Viewer_to_a_Window.md)
