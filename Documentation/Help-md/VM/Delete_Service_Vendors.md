# Delete Service Vendors

To delete an existing service vendor, select that
particular Vendor ID and click the Delete
button. Click the Save button to
save any changes to the Vendors Supplying Service dialog box.

You cannot delete the currently selected Preferred Vendor. To delete
the vendor that is currently selected as the preferred vendor, change
to a different Preferred Vendor before clicking on the Delete
button.
