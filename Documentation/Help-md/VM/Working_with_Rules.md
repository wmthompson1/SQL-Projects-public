# Working with Rules

Rules determine the path of the workflow. Using the
Rule Properties dialog box, you can set up approval tasks (which appear
in VISUALs Task Maintenance application) for a specific user or a
group of users. These users must sign off electronically on the activity
before the workflow proceeds to the next step.

You can also assign Authorizations to restrict the enactment of
a workflow step to a specific user or group of users.
