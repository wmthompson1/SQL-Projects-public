# Showing Part IDs

You can view Part IDs in Operation Bars and at the
top of display windows by selecting Show Part from the Options menu.
Before Part IDs appear, though, you must select Refresh
from the File menu.

Many of the menu selections under the Options menu also have corresponding
buttons on the Toolbar. See "Using the Toolbar" later in
this section.
