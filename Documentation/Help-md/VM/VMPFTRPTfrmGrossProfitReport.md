# Printing/Viewing Gross Profit Report

To print or view Gross Profit Reports:

1. Select Gross
   Profit Report from the Eng/Mfg menu.

The Gross Profit Report window appears.

2. If you are licensed
   to use multiple sites, click the Site ID(s) arrow and select the
   sites to include in the report. If you are licensed to use a single
   site, this field is unavailable.
3. In the Starting
   Ship Date field, specify the starting ship date.
4. In the Ending
   Ship Date field, specify the ending ship date.
5. Click the Report
   Currency arrow and select the currency for the report.
   If you are printing a report for a single site, the entity currency
   associated with the site is inserted. If you are printing a report
   for multiple sites, the drop-down list shows the tracking currencies
   shared by the selected sites. If no shared tracking currencies
   are found, Not Available is inserted. You cannot generate the
   report if a shared currency is not found. Clear site selections
   until a shared currency is found.
6. Select the appropriate
   report sequence in the Selection Criteria area of the window.

You can sequence the report by the following
fields: Order
ID, Customer ID, Customer Name, Part ID, Product Code, Commodity Code. You can select a range for Order
ID, Customer ID, Customer Name, and Part ID. You can select a single
code, or all codes, for Product Code and Commodity Code.

The Starting/Ending field names change according
to the sequence you select.

7. Enter the range to print
   by selecting the Starting
   and Ending
   IDs.
8. Select the appropriate
   Show options.

Open
Orders/Closed Orders
- You can choose to print only open orders (firm or released) or only
closed orders by checking the appropriate box. If you select both,
all open and closed orders print.

Estimates
- Cost estimates for Material, Labor, Burden, and Service print along
with the actuals discussed above.

9. Select the output for the report.

Select Print to
output the report to a printer.

Select View to
output the report to your screen for viewing.

Select File to
output your report as a text file. You can then edit the report using
a text editing application.

10. Select Print/View
    from the File menu.

If you selected Print
as the output, a standard Windows dialog box appears.

11. Select the Summary check box to print summary reports.

Only the totals by order and sequence group
print.

12. Make the appropriate
    selections then click Ok.

VISUAL prints the report.
