# Activities in Sales Order Entry Window

Activities are available in Sales Order Entry. Use
activities to manually create reminders about follow-up items or to
record information about interactions you have with customers.

To add a new activity, select Edit, Activity Entry. See [Creating
Activities](VMACTENTfrmActivityEntry.htm).

To view existing activities, select View,
Activities. To view all activities related
to the sales cycle, select View, Activities
from a blank Sales Order Entry window. To view all activities related
to a particular order, select the order in the Sales Order Entry window
before selecting View, Activities.
See [Reviewing Activities](VMACTMNTfrmActivities.htm).
