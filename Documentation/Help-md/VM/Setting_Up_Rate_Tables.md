# Setting Up Rate Tables

When you perform budget calculations, VISUAL uses
the direct labor rates and burden factors you set up for each period
in the Rate Maintenance window. During actual cost calculations, VISUAL
uses only the burden factors from your rate table and uses the labor
rates from Employee maintenance table.

To set up rate tables:

1. From the Project menu
   of the Infor VISUAL main window, select Rate
   Maintenance.

The Rate Maintenance window appears.

2. Click the Rate
   ID button and select the rate whose table you want to set
   up. Click the OK button.
3. Click the Period
   ID button and select the period you want to use for this
   table.

The dates for the period you select appear
in the Start and Finish date fields.

4. Click the Insert
   Row button on the toolbar.

A new line appears highlighted in the table
and the cursor appears in the Cost Cat ID column.

If you use a Rate ID/Period ID combination
in any of your budget spreads Rate Table Locked appears to the right
of the End Date field of the Rate Maintenance window. VISUAL also
disables the Insert, Delete, and Save functions. A new line appears
highlighted in the table and the cursor appears in the Cost Cat ID
column.

5. Double-click the Cost/Burden Cat ID column header and select
   the cost or burden ID you want to use for this line of the rate
   table.
6. Click in the Rate column
   and enter the rate you want to assign to this Rate ID.

The description and rate type for the Category
ID you selected automatically appear.

7. Click in the Rate column
   and enter the rate you want to use for this line of the rate table.

When you have finished entering lines in
your rate table, click the Save button on
the toolbar.

## Deleting Lines From Rate Tables

You cannot delete Rate IDs you have previously used in any projects
budget.

To delete lines from Rate tables:

1. Open the Rate Maintenance
   window and select the Rate ID that contains the rate you want
   to delete.
2. Click the row header
   for the rate you want to delete.

The row appears highlighted.

3. Click the Delete
   Row button on the toolbar.

An X appears in the row header to indicate
you have marked this Rate for deletion.

4. Click the Save
   button.

The Rate Maintenance window deletes the
rate information for the rate and period you selected.

If you do not want to delete a Rate you
have marked for deletion, click the Delete
button again.

When you have finished deleting Rates, click
the Close Window button, or from the File
menu, select Exit.
