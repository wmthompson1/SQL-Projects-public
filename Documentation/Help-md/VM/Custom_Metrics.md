# Custom Metrics

You can create custom metrics based on any table, set of tables,
or views in the database. To create a custom metric, you first define
a data source. Then, you use the data source to build a metric.

You can create two types of metrics: [analytics](About_Analytics.md)
and [detail grids](About_Detail_Grids.md).

  Data Sources](Creating_Data_Sources.md)

  Analytics](Building_Analytics.md)

  Detail Grids](Building_Detail_Grids.md)
