# Controlling Access to the Macro Edit Menu

A system administrator can prevent users from accessing the macro
Edit menu. If users cannot access the Edit menu, they cannot add,
edit, or delete macros.

1. Sign in to VISUAL as
   a system administrator.
2. Select Security,
   Profile/User/Group Security.
3. Open the profile, user,
   or group ID for which you are setting edit permissions.
4. Click the Status
   tab.
5. To allow users to access
   the macro Edit menu, select the Edit Macros
   check box. To prevent users from accessing the Edit menu, clear
   the check box.
6. Click Save.
