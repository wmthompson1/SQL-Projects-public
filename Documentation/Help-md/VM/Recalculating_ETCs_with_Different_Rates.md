# Recalculating ETCs with Different Rates

Because it is important for you to be able to predict
how a projects costs change, you can recalculate your projects Estimate
to Complete costs using different rates.

To recalculate ETCs:

1. With the Maintain ETC
   window open and the appropriate project selected, click the Rate ID button and select the rate you want
   to use.
2. Click the Apply
   New Rates to ETC button.

VISUAL recalculates the Projects ETC figures based on the new rate
you selected and upon successful calculation, a dialog box appears.
