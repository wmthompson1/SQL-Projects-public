# Reviewing Notification Pending for Sales Transactions

You can view a list of notifications for an order that are eligible
for email but have not yet been sent. To view the list.

1. Select Sales,
   Customer Order Entry.
2. Select the sales order.
3. Select Info,
   Notification Pending. For each notification,
   this information is displayed:

Program ID  The
ID of the program that generated the notification.

* VMORDENT
   Customer Order Entry
* VMSHPENT
   Shipping Entry
* VFARCENT
   Cash Application

Doc User ID 
The ID of the program that generated the notification.

Customer ID 
The ID of the customer on the sales order.

Customer Name 
The name of the customer on the sales order.

Document ID 
The ID of the transaction that requires a notification to be sent.

Site ID  The
Site ID of the transaction that requires a notification to be sent.

Doc Create Date 
The Create Date of the transaction that requires a notification to
be sent.

Order Contact Name 
The Contact Name on the sales order associated with the Document ID.
