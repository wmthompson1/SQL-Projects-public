# Deleting Profiles

To delete a profile:

1. Select the line you
   want to delete and click the Delete
   button.

VISUAL marks the line with an X, to indicate
it will be deleted.

If you change your mind, you can click Delete again to remove the mark. You cannot delete
a numbering profile being used by one or more Part Trace Profiles.

2. Click the Save
   button to delete the profile.
