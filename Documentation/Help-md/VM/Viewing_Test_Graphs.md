# Viewing Test Graphs

You can view a graph of the test results in Test Definition.

1. Highlight the test step
   line item for which you want to view the graph.
2. Click the Graph
   button.

A graph of the test results appears in a
window.

You can select different views of the graph
from the Views menu and other options from the Options menu.
