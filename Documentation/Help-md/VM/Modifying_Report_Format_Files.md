# Modifying Report Format Files

You can modify the format of Manufacturing Window
Reports using Centura Report Builder. The following QRP report files
apply:

Print Costs  VMMFGRP1.QRP

Print Material Availability VMMFGRP4.QRP

Print Engineering Report

Long Form  VMENGRP1.QRP

Short Form  VMENGRP2.QRP
