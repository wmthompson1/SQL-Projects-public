# Entering Shipping Addresses Vendor Maintenance

This function allows you to maintain multiple shipping
addresses for your company. These addresses are not specific to a
vendor they can be accessed for each purchase order created. See
Purchase Order Entry for more information.

Shipping Addresses are created and edited in the same way as Purchase
Addresses.

1. Select Shipping
   Addresses from the Edit menu.

The Ship To Addresses dialog box appears.

2. Click the Insert
   button.

A new line appears in the Shipping Addresses
table.

3. Enter the Ship to ID
   in the Ship to field and the information for the shipping address
   in the fields at the bottom of the window.

The information automatically appears in
the table as you tab through the address fields. You can also copy
and paste information from one line to another.

4. If you want to enter
   Intrastat information for the shipping address, click the Intrastat
   tab and enter information in the following fields:

Port of Arrival
- Enter the place to which goods are received for this shipping address.

Country ID - Enter
the country ID for the shipping address.

Nature of Transaction
- Enter the nature of transaction for this shipping address.

Mode of Transport
- Enter the method by which goods are delivered to this shipping address.

Siret Number -
Enter the siret number for this shipping address, if necessary.

5. Click Save to save the
   new information.

## Deleting Shipping Addresses

To remove a shipping address completely, highlight the address line
and click Delete.
