# What is the Gross Profit Report?

This report lists orders shipped in a specified financial
period or date range, and shows the following information for each
selected packlist shipped:

Sales Amount

Labor, Material, Burden, Service, and Total Costs

Gross Profit  = Sales Amount - Total Cost

Gross Profit % = 100 x Gross Profit / Sales Amount

Costs and profits are summarized by order and by sequence group
(ex: Customer, Product Code, etc.)

## Options

### Report Type

The Period Report prints results for a specified fiscal period.
The To Date report prints results based on a specified shipping date
range.

### Fiscal Period

This specifies the financial period to print the report for. By
entering any date in the period, VISUAL will automatically determine
the Fiscal Period. This only applies to a Period report.

### Shipping Date Range

Specifies the starting and ending shipping dates that you want to
report on. Only applies to a To Date report.

### Summary/Detail

If Summary is selected, only the totals by order and sequence group
will be printed.

### Open Orders / Closed Orders

You can choose printing of only open orders (firm or released) or
only closed orders by checking the appropriate box. If both are selected
OR deselected, then all open and closed orders will print.

### Estimates

Cost estimates for Material, Labor, Burden, and Service are printed
along with the actuals discussed above.

### G/L Transactions

If this option is selected, total costs and revenue that have actually
been posted to the G/L for each packlist will also be shown.

### Sequencing

The report can be sequenced by the following fields: Order ID, Customer
ID, Customer Name, Part ID, Product Code, Commodity Code. You can
select a range for Order ID, Customer ID, Customer Name, and Part
ID. You can select a single code, or all codes, for Product Code and
Commodity Code.
