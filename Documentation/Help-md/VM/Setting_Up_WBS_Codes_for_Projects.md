# Setting Up WBS Codes for Projects

Each project requires at least one WBS Code to which you can assign
tasks.

To specify WBS codes, use either the Project Maintenance Window
or the Project Window. To set up WBS codes using the Project Maintenance
window, see [Entering
Information on the WBS Codes Tab](Entering_Information_on_the_WBS_Codes_Tabpm.md). To set up WBS codes using the
Project Window, see [Creating
the Header Card](Creating_a_New_Project_in_the_Project_Window.md).
