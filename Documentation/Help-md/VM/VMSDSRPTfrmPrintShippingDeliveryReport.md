# Printing Shipping/Delivery Reports

To print the Shipping/Delivery report:

1. Select Shipping/Delivery
   Report from the Sales menu.

The Print Shipping/Delivery Report window
appears.

2. If you are licensed
   to use multiple sites, click the Site ID(s)
   arrow and select the sites to include in the report. If you are
   licensed to use a single site, this field is unavailable.
3. Enter the range of the
   Order Due Date for the orders you want
   to print.
4. Select the appropriate
   sort sequence.

VISUAL supports eight different sort sequences
for the standard report: Due Date, Order ID, Customer ID, Customer
Name, Part ID, and Work Order ID.

5. Specify a range for
   all of these sequences in the Starting/Ending fields.

The Starting/Ending field names change according
to the sequence you have selected.

You can select:

Due
Date

Order
ID

Customer
ID

Customer
Name

Part
ID

Supply
ID

6. Select the appropriate
   subsequence to further confine the report range.

You
can select:

Order
ID

Customer
ID

Customer
Name

Part
ID

Supply
ID

7. Select the output for the report.

Select Print to
output the report to a printer.

Select View to
output the report to your screen for viewing.

Select File to
output your report as a text file. You can then edit the report using
a text editing application.

8. Select Print/View
   from the File menu.

If you selected Print as the output, a standard
Windows dialog box appears.

9. Make the appropriate
   selections then click Ok.

VISUAL prints the report.
