# Setting Default Thermal Labels for Parts

Set up thermal labels to use barcode scanners to process part shipments
and other transfers.

To set up default thermal labels:

1. Select Inventory,
   Part Maintenance.
2. In the Part ID field,
   specify the part to set up.
3. Select Maintain,
   Label Settings.
4. Click the Label
   ID browse button and select the label to use for this part.
   Use the Label Printer Setup Utility to specify labels.
5. Click the Label
   UM browse button and select a unit of measure for the Label
   ID.
6. Click Ok.
