# Modifying Print Formats

The format of Physical Inventory tags and reports
can be modified using Centura Report Builder. The following QRP report
files apply:

|  |  |
| --- | --- |
| Tags | VMPHYFRM.QRP |
| Tag List | VMPHYRP1.QRP |

Tag Comparison Report

|  |  |
| --- | --- |
| By Tag Number | VMPHYRP3.QRP |
| By Part ID | VMPHYRP4.QRP |
| By Location ID | VMPHYRP5.QRP |
| Part Comparison Report | VMPHYRP6.QRP |
