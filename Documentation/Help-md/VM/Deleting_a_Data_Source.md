# Deleting a Data Source

You can delete a data source only if it is not used
as the source for a Detail Grid or Analytic. If a data source is used
for a Detail Grid or Analytic, you must either specify a different
data source to use or delete the detail grid or analytic.

To delete a data source:

1. Select File,
   Dashboard.
2. Select Edit,
   Edit Data Source.
3. In the Data Source ID
   field, select the data source.
4. Click Delete.
5. Click Yes to
   confirm the deletion. The data source is deleted and the Edit
   Data Source dialog is cleared.
