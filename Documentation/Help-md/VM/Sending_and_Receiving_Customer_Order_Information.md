# Sending and Receiving Customer Order Information

Use the Send To option to send customer order information
from the current window to another Infor VISUAL user with the same
database. For a full explanation of sending and receiving Customer
Order information in the form of Infor VISUAL VMX Files, click [here](Sending_Infor_VISUAL_EnterpriseFinancials_Information.md).
