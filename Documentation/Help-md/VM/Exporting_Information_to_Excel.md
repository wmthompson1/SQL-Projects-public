# Exporting Detail Grid Information to Excel

To export the contents of a detail grid to Microsoft
Excel:

1. Select File,
   Dashboard.
2. In the dashboard, select
   File, Open
   and select the dashboard that contains the detail grid.
3. In the detail grid,
   select the rows to export.
4. Right-click and select
   Microsoft Excel.
