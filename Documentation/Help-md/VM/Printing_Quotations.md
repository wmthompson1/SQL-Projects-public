# Printing Quotations

Printed quotations are usually sent to prospects/customers.
You can use VISUALs standard format or up to three user-defined formats.

If you are licensed to use multiple sites, you can
view information for multiple sites in the same proforma report, provided
that the sites share the same parent accounting entity.

1. If you are licensed
   to use multiple sites, click the Site ID
   arrow and select a site to view in the report. When you access
   the dialog box, you can choose to include in the report any site
   with the same parent accounting entity. If you are licensed to
   use a single site, this field is unavailable.
2. Select Print
   Quotation from the File menu.
3. If you are licensed
   to use multiple sites, click the Site ID(s)
   arrow and select the sites to include in the report. You can include
   any sites with the same parent entity as the site selected in
   the Estimating Window. If you are licensed to use a single site,
   this field is unavailable.
4. Click Print
   Current Quote to print the Quotation currently in the Estimating
   Window.

Click the Print All Quotes Dated Between
radio button to print out the quotations over a range of dates.

The quotes in the date range must have a
status of Ready, once the quotes are printed, the status is changed
to Printed. These dates are entered in the dates fields.

5. If you selected the
   Print Quotes Dated Between radio button, enter the Dated From
   and the Dated Through dates.
6. Select a format. User-defined
   formats are only available if the forms have been created.
7. If the quote is for
   an international customer, select a Language ID for the quote.
   See [Entering Languages](Entering_Languages.htm) and
   [Specifying Customer
   Language Forms](Specifying_Customer_Language_Forms.htm).
8. Select the Draft
   Mode check box to print a draft of the quote. If you select
   the check box, the quote is not locked after you print it.
9. Specify how to output
   the report. See [Report Output
   Options](Report_Output_Options.htm).
10. To print the documents
    that you have associated with this customer, select the Print Associated Documents check box.
11. Click Ok.
    If you are outputting the report to a printer, a standard Windows
    printer dialog box is displayed. Make the appropriate selections
    and click Ok.
