# What is the Visual Reporting Data Loader Service?

The Reporting Data Loader Service updates the reporting tables on
the days and times you specify. The service is run against a specific
VISUAL database. If you have multiple VISUAL databases and would like
to run the service on all databases, then you must install the service
once for each database.

To access the Visual Reporting Data Loader Service executable, locate
VRDTASVC.EXE in your VISUAL executable directory. Right-click VRDTASVC.EXE
and select Run as Administrator.

Use the Visual Reporting Data Loader Service executable to perform
these procedures:

* [Installing
  the Visual Reporting Data Loader Service](VRDTASVCfrmMain.htm)
* [Removing
  the Reporting Data Loader Service](Removing_the_Reporting_Data_Loader_Service.htm)

In addition to these procedures, the SYSADM user can set up the
service schedule and monitor the service. To perform these procedures,
use the Visual Reporting Data dialog available in Application Global
Maintenance. See these procedures:

* [Scheduling
  the Reporting Data Loader Service](Scheduling_the_Reporting_Data_Loader_Service.htm)
* [Monitoring
  the Reporting Data Loader Service](Monitoring_the_Reporting_Data_Loader_Service.htm)
