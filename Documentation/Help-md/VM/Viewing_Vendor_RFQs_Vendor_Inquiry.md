# Viewing Vendor RFQs

If a purchase order started as an [RFQ](Generating_Purchase_Orders_from_Purchase_Requisitions.md),
you can view the original vendor RFQ using this feature.

1. With the appropriate
   purchase order displaying, select Show Purchase
   Order RFQs from the Info menu.

The Show PO RFQs dialog box appears, displaying
the specified purchase order in both the Starting and Ending fields.

If the purchase order has a linked RFQ,
it appears in the table. If there is no linked RFQ, no entries appear.

2. To widen the scope of
   the search, click the Starting PO ID
   and Ending PO ID buttons to choose a
   range of purchase orders.
3. Click the Ok
   button to retrieve results for the range.

VISUAL displays any qualifying RFQs.

4. Review the table for
   details of the requisition/purchase order relationship. The table
   includes the following fields:

Purchase Order ID
- The ID of the purchase order. If there is more than one line in
the order, the Order ID appears in multiple rows.

Purchase Order Line #
- The line in the purchase order.

PO Vendor ID -
The Vendor ID from the purchase order.

PO Desired Receive Date
- The desired receive date from the purchase order.

RFQ ID - The ID
of the RFQ used to create the purchase order.

RFQ Line # - The
RFQ line the purchase order is linked to.

Purchase Order Quantity
- The quantity of the purchase order line.

RFQ Line Status
- The current status of the RFQ line.

Purchase Order Status
- The current status of the purchase order.

RFQ Buyer - The
Buyer specified on the RFQ.

Purchase Order Buyer
- The Buyer specified on the purchase order.

5. To view a requisition
   record in Vendor RFQ Entry, select an entry and click the RFQ button or double-click a line.
