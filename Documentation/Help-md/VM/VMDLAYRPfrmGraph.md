# Viewing Resource Delay Reports as Graphs

If you want to view your Resource Delay report as
a graph, set up the report and from the Info menu, select Graphs.
You can also press the F4 key.

To can make the following changes to your graphed report:

Starting
Date - If you want to change the starting date of the report,
click the calendar button and select the date you want.

Delay By - If you want to change the delay
on which the graph is based, you can select the Resource
and Material check boxes.

Shop
View - If you want to change the shop view (group), click the
Shop View drop down box and select the view you want

Resource
- If you want to change the resource depicted in the report, click
the Resource arrow and select the resource you want to use.

Schedule
- If you want to select a different shop schedule, click the arrow
and select the schedule you want to use.

Graph
Type - If you want to change the style of graph that appears,
click the Graph Type arrow and select the style you want to use.
