# Calling Up Customer Tables

The customer table contains the customer name, ID,
and address information. The table can be called up in either name
or ID order. Once called up, the table can be browsed or searched.

The customer table contains the customer name, ID, and address information.
You can call up the table in either name or ID order. Once called
up, you can browse and search the table.

1. To call up the customer
   table in ID order, click the Customer ID
   button.

To call up the customer table in name order,
click the Names button.

Both tables function and operate the same
way.

The only difference between the ID and names
table is the order in which the information appears. These tables
provide an excellent tool to use for finding customers. You can sort
the tables in either ascending or descending order.

If both numeric and alpha characters are
included in the list, the numeric character appears first in the listing.
If lower case characters appear, the lower case characters appear
before upper case characters. Therefore, if the customer list contains
all of these types of characters it appears as follows: Numbered IDs
first, Lower case alpha character second, and Upper case alpha character
last.

2. To view the list in
   descending order, select Refresh Descending
   from the Options menu.

The list resorts to display the numeric
values from largest to smallest and alpha characters from Z to A.

3. Select the name of a
   customer and click the Select or Select/Close.

Select selects
the customer and returns the information to the Customer Maintenance
window for editing while preserving the browse set.

Select/Close also
selects the customer and returns the information to the Customer Maintenance
window for editing, but does not preserve the browse set.

To close the table and return to the Customer
Maintenance window without selecting a customer, click the Close
button.
