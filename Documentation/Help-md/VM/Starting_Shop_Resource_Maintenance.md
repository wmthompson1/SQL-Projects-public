# Starting Shop Resource Maintenance

Shop Resource Maintenance is available from the VISUAL
Main window.

To access the Shop Resource Maintenance window:

Select Shop Resource Maintenance from
the Eng/Mfg menu.

The Shop Resource Maintenance window appears with no information
in it. The Shop Resource Maintenance window contains both entry fields
and action buttons.
