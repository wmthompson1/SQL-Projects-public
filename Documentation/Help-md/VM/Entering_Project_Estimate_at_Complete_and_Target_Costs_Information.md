# Entering Project Estimate at Complete and Target Costs Information

Because the life of your contracts can span months
or even years, it is probable that your cost will deviate from their
estimated values. VISUAL provides you with a method of saving your
changes for later comparison. This functionality also allows you to
create several EACs in advanceperhaps based on figures you know might
changeand specify the one you want to use as a current version.

You may find it easier to decide on a numbering sequence before
you start so that the versions make sense to you in the future.

To create a new version EAC:

1. Select EAC/Target
   Costs from the Maintain menu of the Project Maintenance
   window.
2. Click in the EAC Version
   field and enter the version number you want to enter for project.
3. Click the Calendar
   button and enter the date for this version.
4. If you want to specify
   this version as being the current version, select the Current
   Version check box.

   |  |  |
   | --- | --- |
   | POSTIT.gif | If you have previously set an EAC version as being current, you must first clear the Current Version check box before you can specify this version as being current. |

5. Enter the EAC information
   in the appropriate fields.
6. Click the Save
   button.
7. When you have completed
   creating new EACs, click the Close button.
