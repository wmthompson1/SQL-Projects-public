# Deleting Macros

If you store macros on workstations, any user can
delete a macro at any time. Use caution when deleting a macro from
a workstation. If you delete a macro from a workstation, it is deleted
for all users of that workstation.

If you store macros in the database, you can delete macros that
you created. You cannot delete macros created by the SYSADM user that
have been assigned to your profile.

To delete a macro:

1. From an application
   that supports macros, select Macros,
   Edit.
2. In the Macro Name field,
   specify the macro to delete.
3. Click Delete.
