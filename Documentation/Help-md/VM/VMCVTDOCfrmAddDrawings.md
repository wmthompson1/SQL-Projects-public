# Adding Part Drawings to Document Maintenance

To add Drawing IDs and File Names to Document Maintenance:

1. Navigate to your installed
   VISUAL directory and double-click the VMCVTDOC.EXE file name.
2. Specify
   the database to which you want to connect.
3. If
   you want to create links from your part records to new Document
   IDs, select the Create Links from Parts to
   New Document IDs check box.
4. Click Start.
