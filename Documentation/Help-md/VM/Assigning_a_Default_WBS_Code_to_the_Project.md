# Assigning a Default WBS Code to the Project

After you set up your project master in the Project Window, return
to Project Maintenance to assign a default WBS code/Sub ID/Seq No
combination to your project.

You can also assign the default WBS Code in the Project Window.

1. From the Project Menu,
   select Project Maintenance. The Project
   Maintenance window appears.
2. Click the Project
   ID browse button and select the project to which you want
   to assign a default WBS code. The information for the project
   appears in the window.
3. Click the Default
   WBS browse button to select the WBS code/Sub ID/Seq No
   combination you would like to use as the default.

Only WBS codes with tasks assigned to them
are eligible for selection in the Default WBS field. If you have not
assigned a task to a WBS code, the code will not appear in the WBS
browse table.

4. Click the Save
   button.
