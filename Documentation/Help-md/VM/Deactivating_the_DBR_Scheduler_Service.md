# Deactivating the DBR Scheduling Service

To deactivate the service for a particular schedule ID:

1. Select Scheduling,
   DBR Scheduler.
2. In the table, select
   the schedule that you are running with the DBR Scheduling Service.
3. Select File,
   Set as Scheduled. The ID of the schedule
   you selected is displayed.
4. Clear the Enabled
   check box.
5. Click Save
