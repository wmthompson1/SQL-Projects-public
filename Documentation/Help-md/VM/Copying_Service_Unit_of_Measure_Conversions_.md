# Copying Service Unit of Measure Conversions

You can copy unit of measure conversion information
from one service to another.

1. With a service displaying
   in the Service ID field of the Service Maintenance window, select
   Unit Of Measure Conversion from the
   Edit menu. The Unit of Measure Conversion dialog box appears.
2. If you want, add a new
   measurement line (see section previous) or establish a new conversion
   factor (see section before previous) before copying any information.
3. With the conversion
   factor selected that you want to copy to the other service, click
   the Copy to Other servicebutton. The
   Copy Service Conversion Factors dialog box appears.
4. Click the Service
   ID button to select the service to which you want to copy
   this unity of measure conversion information.
5. Click the Ok
   button. VISUAL copies the unit of measure conversion to the selected
   service.
