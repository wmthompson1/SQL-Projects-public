# Printing FIFO Distributions Reports

Use the FIFO Distributions report to view incoming
costs of selected parts against the distribution of those parts. For
example, if you receive a quantity of ten parts at a cost of $1.00
the first month and a quantity of five parts at a cost of $1.25 the
second month, you have a total quantity of 15, but with different
costs. The FIFO Distributions report shows you where those parts went,
on a "first in-first out" basis. For example, a work order
that requires seven of those parts receives the seven parts at a cost
of $1.00. The work order that requires eight of those parts receives
the first three at $1.00 and the remaining five at $1.25.

1. Select Print
   FIFO Analysis from the File menu.

The Analyze FIFO Distributions dialog box
appears.

2. If you are licensed
   to use multiple sites, click the Site ID
   arrow and select the site to view in the report. If you are licensed
   to use a single site, this field is unavailable.
3. Specify the part range
   for the report by entering the appropriate part IDs in the Starting
   and Ending Part ID fields.
4. Select the appropriate
   output type from the output combo box.

Select Print to print the report to your
printer.

Select View to view the report on your monitor.

Select File to print the report to a file.

4. Specify the date range
   for the report in the Transactions Dated On or After field. You
   can also select a date by clicking on the calendar tool.
5. Click the Print
   button to output the report.
