# Adding a Line Item

1. With the appropriate
   request for quote open in the Vendor RFQ Entry window, click the
   Insert button.

A new line is added to the table section.
An arrow appears in the row header of the new line.

2. Enter the item information
   (see "[Entering a New Request
   for Quote](VMRFQENTfrmRFQ.md) ").
3. Click Save
   to update the request for quote.
