# Viewing Contention Inquiry Graphs as a Table

You can view the information from a Contention Inquiry
graph in a table format by clicking the button next to the graph styles
list box.

Upon clicking, a table appears in place of the graph. The selected
Schedule IDs appear in the left hand column and the resources for
which there is contention information each represent one column in
the table.
