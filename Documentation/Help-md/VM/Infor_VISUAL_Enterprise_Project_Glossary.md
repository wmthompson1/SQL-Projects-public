# Infor VISUAL Project Glossary

The glossary contains some common A&D terms.

## A

ACWP - Actual Cost of Work Scheduled.
The costs you have reported to the project to date.

Acceptance Point - Where the quality assurance
take place.

Actuals - What projects actually cost
you.

Administered By - Person or office administering
the contract. You can usually find this information

in Contract Section G under Contract Administration Office (CAO).

## B

BCWP - Budget Cost of Work Performed.
The difference between your budget costs and your actual costs to
datethis is the projects Earned Value.

Burden Categories - Groups of like burdens.
For example, any person or group whose burden you want to capture.

Burden Groups - Groups of associated burdens.

## C

CLIN - Contract Line Item Numbers.

CQA - Contract Quality Assurance.

CV - Cost Variance. The difference between
the earned value of the project and the actual value you have reported
to the project.

CWS - Budget Cost of Work Scheduled. The
budget for your project.

Commodity Code - Industry codes identifying,
by category, the products a company produces.

Contract Type - The types of contract
with which you deal. For example: B & P, Contract, and Internal.

Cost Categories - Groups of like costs.
For example, any person or group whose cost you want to capture.

Cost Groups - Groups of associated costs.

## D

DD250 - Receiving and Inspection Report
(DD250).

Departments - Groups to which you report
budget and actual costs "by department". See also "Subordinate
Departments" on page A6.

Distribution - Budget distributions schemes.
VISUAL uses in your project budgeting calculations. Also known as
Time Phased Budget Planning.

Division - Individual part of your company
used for reporting purposes.

Drawing File - Name of a drawing file.

Drawing Number - Identifying number of
the drawing associate with a project or part of a project.

Drawing Revision - Revision number of
the drawing.

## E

EAC - Estimate to Completion. An estimate
costs to complete the project.

ECN - Engineering Change Notification.

Earned Value Quantities - Earned value
of tasks based on the completeness of the linked work orders.

Earning Code - Groups to which you report
costs.

Engineered Date - Date on which you engineered
this product.

Estimate to Complete - Project costs from
the date you select to the completion of your project.

ETC - Estimated Time to Complete. An estimate
of the time required to complete the project.

## F

Forward Schedule from Release Date - Scheduler
works forward from the release date, trying to schedule the work order
for completion as soon as possible.

## I

Interval Percentages - Number of intervals
used for the distribution. The number of lines refers to the quantity
of intervals and the interval percentage is the amount of the total
assigned to the interval.

## M

Macros - Small script-based programs used
to simplify complex tasks by automating time-consuming processes and
eliminating the chance of incorrect data entry.

Marked For - At the receiving end, the
person or department responsible for the receipt of the part.

Mod # - Modification Numbers. Used to
identify customer changes to their contracts.

## P

Pay Made By - Organization, department,
or person responsible for making contract payments to you.

Period - Time period you want a Rate to
be effective.

Pict/Object - Graphic file you can associate
with the project. Defining a graphic association provides shop floor
access to the object.

Planning Parameters - Policies by which
VISUAL plans a material.

Posting - Sending journal entries to the
VISUALs general ledger.

Prime Contractor - Contractor primarily
responsible for the contract.

Priority - Importance levels for your
projects.

Product Code - Codes identifying products
by their category.

Proj Ref Sub ID - Sub ID for the project.

Proj Ref Seq No - Sequence number for
the project.

Project Manager - Person responsible for
the management of the project.

Project Type - Type of projects you enter.
For example, Bid and Proposal, Contract, Indirect, and Capital.

## R

Resolution Factor - Used for factoring
in any additional burdens. Applied to hours or amounts

Rate - Actual amounts VISUAL applies to
projects.

Receiving Inspection Locations - Location
at which your customer wants to have the product of the project received
and inspected.

Retro Burden - Process of spreading the
changes in Overhead Burden Rates over the year-to-date figures of
a project.

## S

SV - Schedule Variance. Difference between
the budget and earned value of the project at the time you calculated
the schedule variance.

Security Class - Security levels for your
projects. For example, Unclassified, Secret, and Top Secret.

Segment - Individual parts of a company
used for reporting purposes.

Shipped From - Address from which you
ship items on a project.

Shipped To - Usually the receiving station
at the site. Block 15 of the contract.

Subordinate Departments - Several departments
that report their budget and actual costs to one department.

Summary Code - Types of work that VISUAL
can summarize in project reports.

## T

Task - Assignment that must be finished
before the project can be complete.

Terms and Conditions - The policies and
requirements under which you agree to accept contracts.

Time Phased Budget Planning - Different
budget distributions schemes VISUAL uses in project budgeting calculations.

Traveller - List of components, parts,
and materials needed to complete a project.

Treat Release Date as Hard - Scheduler
does not attempt any scheduling of the work order until the release
date.

## V

VAC - Variance at Completion. Difference
between complete budget and the total reported costs for a project.

Variables - Named items with changeable
values. Variables primarily define material requirements in terms
of variable dimensions.

## W

WBS Code - Work Breakdown Structure Code.

WBS Contract Type - Type of contract to
which the WBS refers.

Warehouse Defaults - Template VISUAL copies
when you create a project warehouse.
