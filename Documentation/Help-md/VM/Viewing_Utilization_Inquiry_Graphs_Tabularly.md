# Viewing Utilization Inquiry Graphs as a Table

You can view the information from a Utilization Inquiry
graph in a table format by clicking the button next to the graph styles
list box.

Upon clicking, a table appears in place of the graph. The selected
Schedule IDs appear in the left hand column and the Resource IDs for
which you requested utilization information each represent one column
in the table.
