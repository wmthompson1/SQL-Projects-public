# In-Context View Panels in Purchase Requisition Entry

You can view the Vendor View Panel and Part View Panel
in Purchase Requisition Entry.

To access the Vendor View Panel, select View,
Vendor View Panel. Use the Vendor View panel
to view information about your outstanding transactions with the vendor
in the parent accounting entity of the site specified in the header.
You can view summary information, detail information, and a chart.
For information about setting up the summary and detail views, see
[In-context View Panels](In-context_View_Panels.md). For
information about the chart, see [Vendor
View Panel Chart](Vendor_View_Panel_Chart.md).

To access the Part View Panel, select View,
Part View Panel. The Part View Panel chart
shows information about the current inventory levels for the selected
part in the site specified in the header. The chart contains one line
and four bars. For information about setting up the summary and detail
views, see [In-context View Panels](In-context_View_Panels.md).
For information about the Part view panel chart, see [Part
View Panel Chart](Part_View_Panel_Chart.md).
