# Setting the Vendor Service Calendar Exceptions

You can set the outside service vendors calendar
exceptions using the Vendor Service Calendar Exceptions dialog box.

1. Click the Vendor
   Calendar button in the Scheduling tab.

The Vendor Service Calendar Exceptions dialog
box appears.

2. Enter the dates the
   outside service vendors schedule varies in the Start Date and
   End Date columns.
3. Enter the percent of
   the day worked in the Percent of Day Worked column. The percent
   in based on the outside service vendors operation run rate.
4. Click Save
   to save any changes to the Vendor Service Calendar Exceptions
   dialog box.
