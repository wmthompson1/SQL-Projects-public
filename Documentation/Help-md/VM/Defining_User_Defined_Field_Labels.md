# Defining User Defined Field Labels

This option allows you to define the label for each
field in the User defined section of the Part Maintenance window.
This changes the labels for all users of the system.

1. Select User
   Defined Field Labels from the Options
   menu.

User Defined Field Labels dialog box appears.

2. Enter the appropriate
   label in the fields next to the User Defined Fields titles.
3. Click Save
   to exit and save the new field labels.

Click Close to
close the dialog box and return to the Part Maintenance window.

Click the Clear
button to clear all of the fields.

4. Click the Delete
   button to delete the label set.
