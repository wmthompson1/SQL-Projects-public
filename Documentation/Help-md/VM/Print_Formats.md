# Print Formats

You can modify the format of the Material Planning
Window reports using Centura Report Builder. The following QRP report
files apply:

## Material Netting Report

 By Part ID - VMPLNRP1.QRP

 By Product Code - VMPLNRP3.QRP

 By Commodity Code - VMPLNRP4.QRP

 By Planner - VMPLNRP5.QRP

 By Buyer - VMPLNRP6.QRP

 Order Point Report - VMPLNRP2.QRP
