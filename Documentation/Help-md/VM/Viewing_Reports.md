# Viewing Reports

VISUAL uses a report-formatting tool called Report
Windows for all of its printed reports. With few exceptions, any VISUAL
output that you can print you can also view on the screen. This is
useful for previewing printed information, but the real benefit is
in the ability to read any report on-screen. The viewer has these
controls:

## Scroll Bars

To scroll the current page, use the horizontal and vertical scroll
bars as necessary.

## Page Buttons

To go to the next page, use the single-right
arrow button.

To go to the previous page, use the single-left
arrow button.

To go to the last page, use the double-right
arrow button.

To go to the first page, use the double-left
arrow button.

Paging commands are also located in the File menu. You can also
go to any page directly by selecting Go to Page, and entering a page
number.

## Zoom Icon

Two view modes are available: Actual Size and Fit in Window. Use
the magnifying glass icon to switch between modes. You can also select
the mode from the View menu.

## Print Icon

To print the report from this window, press the printer icon, or
select Print from the Print menu.

## Exiting Report

To close the report window, select Close Report
from the File menu. This also usually closes the VISUAL Print dialog.
