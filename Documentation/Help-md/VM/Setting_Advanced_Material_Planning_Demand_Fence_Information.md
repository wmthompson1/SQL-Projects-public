# Setting Advanced Material Planning Demand Fence Information

In the Repetitive area of the General tab section,
enter the Repetitive Manufacturing demand fences. The Site Maintenance
demand fence determines the number of days in a real demand period
for parts without demand fences in either a product code or part master
and with an order policy of Master Schedule. If a part has a product
code with a demand fence, VISUAL uses the product code demand fence
for the part during MRP; if a part has a demand fence defined in its
part master, VISUAL uses the part master demand fence over both product
code demand fence and Site Maintenance demand fence values.
