# Starting Sales Tax Group Maintenance

Sales Tax Group Maintenance is available from the
Sales Menu of the Infor VISUAL Main window.

To access the Sales Tax Group Maintenance window:

Select Sales Tax Group Maintenance from
the Sales menu.

|  |  |
| --- | --- |
| POSTIT.gif | With VAT enabled, this menu option does not appear. |

An empty Sales Tax Group Maintenance window appears.

The Sales Tax Group Maintenance window is made up of both entry
fields and action buttons.
