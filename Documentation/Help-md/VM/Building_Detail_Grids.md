# Building Detail Grids

After you create a data source, you can use the data
source as the basis for a detail grid.

To build a detail grid, follow these basic steps:

1. Specify a configuration
   ID, description, data source, and title.
2. Select the grid type.
   The type that you select determines the other information that
   you specify in the dialog.
3. Specify the columns
   to include in the detail grid.
4. Test the detail grid.
5. Activate the detail
   grid.

See [Creating Detail Type
Grids](Creating_Detail_Type_Grids.md), [Creating
Top N and Bottom N Detail Grids with Individual Values](Creating_Top_N_and_Bottom_N_Detail_Grids_with_Individual_Values.md), or [Creating
Top N and Bottom N Detail Grids with Aggregated Values](Creating_Top_N_and_Bottom_N_Detail_Grids_with_Aggregated_Values.md).
