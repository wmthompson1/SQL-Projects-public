# Opening Advanced Browse Tables for Editing

You can use this feature for the advanced browse associated
with the primary key of the window only. For example, you can edit
only the Customer advanced browse in Customer Maintenance.

To edit information in an advanced browse table:

1. Access the advanced
   browse table.
2. Select File,
   Open for Edits.
3. Edit information as
   needed.
4. Select File,
   Save.
