# Printing Customer Order Sales Journals

To print the Customer Order Sales Journals:

1. Select Sales,
   Customer Sales Journal.
2. If you are licensed
   to use multiple sites, click the Site ID(s)
   arrow and select the sites to include in the report. If you are
   licensed to use a single site, this field is unavailable.
3. In the Starting Invoice
   Date and Ending Invoice Date fields, specify the date range for
   the invoices to include in the report.
4. In the Sequence section,
   click a sequence option. After you click a sequence option, the
   label for the Starting ID and Ending ID fields change. You can
   click one of these options: Order ID, Customer ID, Customer Name,
   Part ID, Product Code, Commodity Code, Sales Rep, Territory, Invoice
   ID, G/L Account ID, Tax ID, and Tax Authority ID.
5. To restrict the information
   displayed in the report, specify codes or IDs in the Starting
   and Ending IDs/Codes fields. The report includes the codes or
   IDs you specify and any code or ID that alphabetically or numerically
   between the two fields.
6. Click the Report Currency
   arrow to select a currency from the list. If you are printing
   a report for a single site, the entity currency associated with
   the site is inserted. If you are printing a report for multiple
   sites, the drop-down list shows the tracking currencies shared
   by the selected sites.

If you are printing a summary report, the
sites you select must share at least one tracking currency. If no
shared tracking currencies are found, Not Available is inserted.
You cannot run a summary report if a shared tracking currency is not
found. Clear site selections until a shared tracking currency ID is
displayed in the drop-down list.

If you are printing a detail report, you
can print the report in native currency if a shared tracking currency
is not found.

7. To include uninvoiced
   amounts in the report, select the Print Uninvoiced
   check box.
8. To print a summary report,
   click Summary. To print a detailed report,
   click Detail. Summary reports include
   subtotals based on the sequence. Detailed reports include the
   individual transactions that make up the subtotals.
9. From the output list,
   select the output format for the report. If you select email,
   select the PDF Format check box to attach
   a PDF to the email instead of an RTF file. See [Report
   Output Options](Report_Output_Options.md).
10. Click Print.
