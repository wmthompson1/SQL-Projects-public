# Print Purchase Order List

Purchase order lists give you a complete listing of
purchase order activity.

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site to use. If you are licensed to use a single site, this field
   is unavailable.
2. Select Print
   Purchase Order List from the File
   menu.

The Print Purchase Orders dialog box appears.

3. Click the appropriate
   Orders radio button

Click the Current Order
button to print a purchase order report for the current order only,
or click the All Orders button to print purchase order reports for
all orders.

If you select the Current
Order button, skip to step 8.

This prints the current order in report
format. To print a PO to transmit to the vendor, use Print Acknowledgment.

4. Enter the starting date
   and ending date for the period you want to print.

This range includes all purchase orders
that took place during the time period you enter.

5. If you selected the
   Vendor ID Sequence radio button, you can further narrow the list
   by entering a starting and ending Vendor ID.

If you do not enter a Vendor ID, all vendors
with in the specified date range print. You can print an open-ended
list by specifying a starting or ending Vendor ID only. The range
includes the specified starting and/or ending vendor.

6. Select the status of
   the purchase orders to print.

You can select any or all of the available
status options.

7. Select the list sequence.

If you click the Order ID radio button,
the list appears in Order ID sequence. If you click the Vendor ID
option button, the list appears in Vendor ID sequence.

8. Choose the type of report
   to generate, Summary or Detailed.

The Detailed format prints most of the information
for each PO, including all addresses and all line items.

The Summary format prints a single line
for each PO, listing Order ID, Vendor ID, Vendor Name, Order Date,
Recv Date, Status, and Total Amount.

The Delivery Detail format prints a most
of the information for the delivery schedule.

9. Choose the output destination
   by clicking on the arrow next to the output field. Select one
   of the following:

Print - Select
this option to output the file to a printer.

View - Select
this option to preview what the printed output will look like when
printed.

File - Select
this option to save the information to a text file.

10. If you selected the
    Print in Step 8, click Ok to output
    the information.

A standard Windows printer dialog box appears,
allowing you to select the number of copies to print.

11. Make the appropriate
    selection and click Ok.
