# VISUAL BTS Physical Inventory Count Table

The Pending Transaction portion of the window contains
the following columns. The Completed Transaction line item table contains
the same columns, with the addition of column Transaction ID. After
VMBTSBI has successfully processed a transaction, the transactions
ID appears here.

ID - The ID of the transaction.

Trans Type - The type of the transaction.
Receipt appears here.

Trans Date - The date on which you scanned
in the purchase order.

Physical Count ID - The ID of the physical
count.

Tag # - The tag number of the part

Part ID - The ID of the part.

Quantity - The quantity that you are issuing
or transferring

User ID - The ID of the user who is currently
scanning.

Trace ID - The ID of the trace for the
part.
