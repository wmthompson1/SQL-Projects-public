# Adding SIC Codes

Standard Industrial Classification codes classify
various companies by the products they produce. You can enter and
delete SIC codes from the SIC Table.

1. Select Admin,
   Application Global Maintenance.
2. Select Maintain,
   Table Maintenance, SIC
   Table.
3. Click Insert.
4. Specify a SIC code and
   a description.
5. Click Save.
