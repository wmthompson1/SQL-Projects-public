# Specifying Email Lists for Documents

To specify the recipients of emailed reports:

1. To set up customer recipients,
   select Sales, Customer
   Maintenance.

To set up vendor recipients, select Purchasing, Vendor Maintenance.

2. Select the ID of the
   customer or vendor.
3. Select Maintain,
   Email Documents.
4. In the dialog, a row
   for each contact and site combination is displayed. For each contact,
   select which documents to send.
5. The email specified
   for the contact on the contacts record is displayed. To send
   the email to a different address, specify the address in the Send
   Docs to: Email Addr column.
6. Click Save.
