# Saving a Query

If you run the same query frequently, you can save the query input
for future use. Queries are saved by user. You can access your saved
queries from any workstation, provided that your User ID is used to
sign into the system.

To save a query:

1. Specify the query in
   the Query Input panel.
2. Specify a name for the
   query in the Query Name field.
3. Click Save.

To delete a saved query, specify the query in the Query Name field,
and then click Delete Query.
