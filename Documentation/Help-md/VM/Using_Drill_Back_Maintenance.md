# Using Drill Back Maintenance

In certain integrations, such as with Infor Ming.le, you can drill
back from the integrated environment to VISUAL. For example, you may
be able to drill back from a metric displaying purchase orders directly
to Purchase Order Entry in VISUAL. For more information, see the appropriate
integration or configuration guide, such as the Infor
VISUAL Configuration Guide for Infor Ming.le.

Note: You must be signed
in as the SYSADM user to access Drill Back Maintenance.

1. To access Drill Back
   Maintenance, from the Admin menu, select Drill
   Back Maintenance.
2. The table lists all
   of the drill back views. To change the VISUAL program associated
   with the view, double-click the Program Name
   column header and select the program you would like to use. For
   example, for the PurchaseOrderView, you could choose to drill
   back to the Purchase Management Window instead of the Purchase
   Order Entry window. The system inserts the Program ID associated
   with the Program Name in the Program ID field.
3. When you are finished
   changing the default settings, click the Save
   button.

## Drillback Service

To drill back from an application to VISUAL, you must install the
HTTP2VM service. You can install this service from the VISUAL installation
media. For more information, see the Infor VISUAL Applications Installation
Guide.
