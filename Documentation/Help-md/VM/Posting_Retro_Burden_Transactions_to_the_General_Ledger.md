# Posting Retro Burden Transactions to the General Ledger

If you are satisfied with the transactions displayed in the Retro
Burden G/L Transactions report, post the transactions to the general
ledger. To post the transactions, select the Post All button on the
toolbar. If you do not post the transactions, the unposted transactions
generated by the Retro Burden utility will be erased the next time
you run costing.

Use the Inquiry / Print G/L Transactions feature of the Project
Summary window to view how burden amounts are distributed.
