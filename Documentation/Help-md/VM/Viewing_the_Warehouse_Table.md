# Viewing the Warehouse Table

The Warehouse table contains basic information on
all warehouses in the warehouse master file. You can browse and search
this table.

1. Click the Warehouse
   ID button in the Warehouse Maintenance window. To view
   the table in Warehouse Description order, click the Description
   button.

The Warehouses dialog box appears.

2. Sort the table in either
   ascending or descending order by clicking on the Descending check
   box.

To view the list in descending order, click
the Descending check box. When checked, the list re-sorts to display
the numeric values from largest to smallest and alpha characters from
Z to A.

If both numeric and alpha characters are
included in the list, the numeric character appears first in the listing.
If lower case characters are present, the lower case characters appear
after upper case characters. Therefore, if the warehouse list contains
all of these types of characters it appears as follows: Numbered IDs
first, Upper case alpha character second, and Lower case alpha characters
last.
