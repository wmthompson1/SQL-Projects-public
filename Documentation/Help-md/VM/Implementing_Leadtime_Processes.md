# Implementing Leadtime Processes

Complete these procedures to implement leadtime processes:

1. [Plan
   how to measure leadtime steps.](Planning_Leadtime_Processes_Steps.md) You can use built-in functionality,
   SaveProcess macros, or a combination of the two.
2. After you decide how
   to measure leadtime steps, [specify
   the UpdateProcessActivity preference in Preferences Maintenan](Specifying_Leadtime_Process_Preferences.md)ce.
3. If you are using SaveProcess
   macros, [create the macros](Creating_SaveProcess_Macros.md).
4. In Part Maintenance,
   [create leadtime processes](Creating_Leadtime_Processes.md).
5. In Part Maintenance,
   [assign leadtime
   processes to parts](Assigning_Leadtime_Processes_to_Parts.md).
