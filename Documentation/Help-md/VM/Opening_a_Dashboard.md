# Opening a Dashboard

When you open a dashboard, the dashboard you open
is displayed instead of the dashboard you are currently viewing. You
can view only one dashboard at a time.

To open a dashboard:

1. Select File,
   Dashboard.
2. In the dashboard, select
   File, Open.
   In the table, all dashboards you created are listed. If other
   users created shared dashboards, then these dashboards are also
   listed.
3. Select the dashboard
   to open.
4. Click Ok.
