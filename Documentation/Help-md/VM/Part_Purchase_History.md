# Part Purchase History

If you are licensed to use multiple sites, you must
specify a Site ID before viewing Part Purchase History.

To view Part Purchase History:

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site to use. If you are licensed to use a single site, this field
   is unavailable.
2. Select the Part
   Purchase History option from the Info menu.

The Part Purchase History window appears.

3. If you are licensed
   to use multiple sites, click the Site ID(s) arrow and select the
   sites to use. The drop-down menu is limited to sites that belong
   to the same entity as the site you selected in Part Maintenance.
   If you are licensed to use a single site, this field is unavailable.
4. Click the Part
   ID browse button and select the part you want to appear
   in the Part Purchase History window.
5. If you want to sort
   the information in the window, select the sort option you want
   to use from the Sort section. You can select:
6. Order Date
7. Receive Date
8. Purchase date

6. In the Sort Sequence
   section, select the sequence on which you want VISUAL to sort
   your information.  You can select Ascending
   or Descending.

## Viewing Purchase Order Line Details

To view purchase order details from the Part Purchase
History window:

1. In the Part Purchase
   History window, select the line containing the purchase order
   in which you are interested.
2. Click the View
   button.

The order information appears in the Purchase
Order Entry child window.

## Exporting Information from the Part Purchase History Window

To export information from the Part  Purchase
History window to Microsoft Excel:

1. Populate the Part Purchase
   History window.
2. Select the Send
   to Microsoft Excel option from the File menu.

VISUAL opens Excel and populates it with
your current parts information.

To export information from the Part  Purchase
History window to an XML file:

1. Populate the Part Purchase
   History window.
2. Select the Send
   to XML option from the File menu.

The Send to XML dialog box appears.

3. Click the File Name
   browse button and navigate to the folder you want to use.
4. In the Export File dialog
   box, enter the name of the file you want to use.
5. In the Export File dialog
   box, click the Save button.

The folder you selected and file name you
entered appears in the Send to XML dialog box.

6. In the XML to Write
   section, select what you want to send to XML. You can select:

Schema If you
want to send your information to an .xsd file, select the Schema
option.

Document If
you want to send your information to an .xml file, select the Document option.

Both If you
want to send your information to both .xsd and .xml files, select
the Both option.

7. In the Tags section,
   select how you want VISUAL to tag your XML information. You can
   select:

Use Column Name
If you want VISUAL to use the column name from the table window
to tag your XML information, select the Use Column
Name option.

Use Item Name
If you want VISUAL to use the item name from the code, for example,
dfsID as opposed to Part ID to tag your XML information, select the
Use Item Name option.

8. Click the Export
   button.

If the export is successful, Export Complete
appears at the bottom of the Send to XML dialog.
