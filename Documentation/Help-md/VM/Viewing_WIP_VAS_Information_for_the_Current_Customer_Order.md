# Viewing WIP/VAS Information for the Current Customer Order

1. With the appropriate
   customer order open in the VISUAL Order Management Customer Order
   Entry window, highlight the line that contains the Part ID and
   Warehouse ID for which you want to view WIP/VAS information.
2. Maximize the Order Management
   window and select WIP/VAS from the Tools
   menu.

The WIP/VAS (for Part in Warehouse) window
appears.

3. Double-click the icon.

TheWIP/VAS IDs for the focus part in the
focus warehouse appear.

These are the WIP/VAS IDs for the focus
part in the focus warehouse.
