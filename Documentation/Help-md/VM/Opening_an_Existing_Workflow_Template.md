# Opening an Existing Workflow Template

To open an existing Workflow template:

1. In the Open Existing
   Workflow dialog box, click the Application Area arrow and select
   the application area of VISUAL for which you want to open a template.
2. Click the Template
   Name button and select the template you want to open.
3. Click the Ok
   button.

The Work Area appears with the selected
template.
