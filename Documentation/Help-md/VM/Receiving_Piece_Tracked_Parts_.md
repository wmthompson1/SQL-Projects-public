# Receiving Piece Tracked Parts

Receive piece-tracked parts using Purchase Receipt
Entry.

1. Select Purchase
   Receipt Entry from the Purchasing menu of the VISUAL Main
   window.

The Purchase Receipt Entry window appears.

2. If you are licensed
   to use multiples sites, click the Site ID arrow and select the
   site to use. If you are licensed to use a single site, this field
   is unavailable.
3. In the Order ID field,
   enter the Order ID for the order containing the piece-tracked
   part.

You can also click the Order ID button and
select the order from the Purchase Orders table.

When you enter the Order ID, the vendor
information appears for the order.

4. In the Quantity Received
   column, enter the quantity of the inventory part that you are
   receiving.
5. In the # of Pieces column,
   enter the number of pieces of the inventory part that you are
   receiving.
6. In the Length, Width,
   and Height columns, enter the dimensions of the inventory part
   that you are receiving.
7. Click the Save
   button to save the purchase receipt.

For more information on receiving purchase
orders, see Purchase Receipt Entry.
