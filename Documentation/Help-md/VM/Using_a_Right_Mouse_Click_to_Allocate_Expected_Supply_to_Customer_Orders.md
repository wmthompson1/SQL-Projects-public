# Using a Right Mouse Click to Allocate Expected Supply to Customer Orders

To perform this operation, you must have the Current
Customer Order window and Part Availability window open. The part
that you want to allocate to the customer order line must be displaying
in the Part Availability window.

1. In the Part Availability
   window, double-click the icon and then click the Expected
   plus box.

The three sources of supply - PO, WO, IBT
- appear.

2. Depending upon which
   source of expected supply you want to allocate from, click the
   plus box next to the appropriate line.

The expected lines of supply appear.

3. Right-click the line
   from which you want to allocate supply.

An Allocate message box appears.

4. Left-click the message
   box to allocate the supply.

VISUAL allocates supply from the line to
the current customer order line. In the Current Customer Order window,
a plus box appears next to the line to which you allocated supply.
If you click this plus box, the ID of the supplying source from which
you allocated the supply appears.
