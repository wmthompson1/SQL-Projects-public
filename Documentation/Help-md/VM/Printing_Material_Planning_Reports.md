# Printing Material Planning Reports

The Material Planning Report is essentially a printed
summary of the information available on the Material Planning Window.

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site ID to use for the report. If you are licensed to use a single
   site, this field is unavailable.
2. Select Material
   Planning Report from the File
   menu.

The Print Material Planning Report dialog
box appears.

3. Enter the Starting and
   Ending Part IDs for the parts.
4. Select the appropriate
   part options:

Parts with Shortages
Only - Only prints the report for parts that have a negative
Projected Quantity.

Fabricated Parts Only
- Click this radio button to report only on fabricated parts.

Purchased Parts Only
- Click this radio button to report on purchased and fabricated parts.

Fab/Pur Parts Only
- Click this radio button to report on parts that are both purchased
and fabricated.

Either, but Not Both
- Click this radio button to report on purchased or fabricated parts,
but not both.

5. Choose the appropriate
   Netting Options.

Net by Required Date
- This option nets the demand and supply order by Date Required
and Due Date.

Net using MRP rules  -
This option nets the demand and supply orders by standard MRP rules.

Net by Release date -
This option nets the demand and supply order by Release Date and Due
Date.

Choose from Net Available Quantity and Deduct
Safety Stock check boxes.

6. Set the Print Horizon.
   The Print Horizon sets the number of days forward from the current
   date that you want to net. The Material Planning Window always
   nets for an infinite horizon. For the report, you can restrict
   the range to manage the size of the report. Leaving this field
   blank indicates no horizon.
7. Choose the Report Sequence.

Choose from the five different options for
sequencing the report: Part ID, Product Code, Commodity Code, Planner
User ID, or Buyer User ID.

8. Choose the Netting Type.

Select either Net by Required Date or Net
Using MRP Rules.

9. Choose the Part Selection.

You can select:

All Parts - To
print material planning reports for all parts.

Parts With Material Requirements
Dated Between - To constrain the report based on the date range.
Enter the Starting and Ending dates

Parts With Customer Orders
Dated Between - To focus on demand for a particular period
of time.

Enter the Starting and Ending dates.

10. Choose the output type
    from the drop down menu. Select either Print, View, or File.
11. Click Ok.

If you select Print as the output type,
a standard Windows Print dialog box appears.

12. Click Ok
    to print the reports.

The report format is similar to the columns
of the Material Planning Window. Detail Quantities are not shown,
and the total quantities for demand and supply orders appear for each
Part ID.
