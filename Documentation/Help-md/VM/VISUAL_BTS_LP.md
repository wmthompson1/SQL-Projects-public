# VISUAL BTS-LP

VISUAL BTS-LP provides a GUI window to print labels
based on criteria set forth in the VISUAL BTS-LP setup program. There
are no limits as to what labels can be printed.

These include:

* Automotive Industry Action
  Group (AIAG)
* Uniform Code Council (UCC)
* Health Industry Bar Code
  Council (HIBCC)
* Automatic Identification
  Manufacturers (AIM)
* LOGistics Applications of
  Automated Marking And Reading Systems (LOGMARS)
* Committee for Commonality
  in Blood Banking Automation (CCBBA)
* American National Standards
  Institute (ANSI)
* Uniform Product Code (UPC)
* And assorted other standard
  label types as well as any custom labels.
