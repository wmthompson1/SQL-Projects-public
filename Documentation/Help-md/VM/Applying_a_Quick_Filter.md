# Applying a Quick Filter

To apply a quick filter in an advanced browse dialog:

1. Access the appropriate
   advanced browse dialog.
2. Click Quick
   Filter. A row is added to the top of the table.
3. Use the row to specify
   your search criteria. You can use [wildcards](Advanced_Browse_Wildcards.htm)
   to indicate missing characters and operators to search for ranges.
4. If you are using dynamic
   filtering, the rows that match your search criteria are displayed
   as you enter the criteria. See [Using
   Dynamic Filtering](Using_Dynamic_Filtering.htm). If you are not using dynamic filtering,
   click Quick Filter to search the table.
