# Viewing Trace Details in the Material Planning Window

When using the Material Planning window, you can view
the trace information for the Warehouses and Locations for the current
Part ID.

To view trace information in the material Planning window:

1. Select Inventory,
   Material Planning Window.
2. If you are licensed
   to use multiple sites, click the Site ID
   browse button and select the site to use.
3. In the Part ID field,
   click the Part ID button and select
   the part.
4. Select Info,
   Trace Details.
5. To view trace details
   for specific warehouses and locations,select the warehouses and
   locations.

If you only want to view Trace IDs with
quantities greater than zero, select the Show Quantity
Remaining >0 check box.

If you want to view trace history of the
part, select the Show trace history check
box.

6. When you have finished
   viewing your Trace IDs, click the Close
   button.
