# Sending Data to Microsoft Project

Use VISUALs Microsoft Project Interface window to
export data to and import data from Microsoft Project. After you
have set up your project and exported its data to Microsoft Project,
you can add task dependencies and change critical task information
as you need, then import that new information back to VISUAL and save
the project as a baseline.

By using Infor VISUAL and Microsoft Project to prepare your project
plans, you can extend your ability to provide detailed project information
to your customers and any non-Infor VISUAL users.

You can access VISUALs Microsoft Project Interface window from
either the Project Summary window (File menu) or the Project Window
(Send to Microsoft Project button).

There are three basic steps to exporting your VISUAL project information:

1. Set up your project
   with correct work breakdown structure.
2. Establish a budget for
   your project and assign cost categories to your tasks.
3. Transfer data to Microsoft
   Project.
