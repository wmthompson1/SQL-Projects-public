# Step Six

## Update ETC with Actuals

After you post the manufacturing journals, update
project ETCs with actual costs. To update ETC with actuals:

1. From the Project window,
   select Update ETC with Actuals. The
   Update ETC with Actuals window appears.
2. In the Start Period
   field, enter any date in the period you are closing. VISUAL populates
   the field with the first day of the period. In the End Period
   field, enter any date in the period you are closing.

VISUAL populates the field with the end
date of the period.

3. Click the Run
   button. The Update ETC window appears.
