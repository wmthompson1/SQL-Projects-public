# Entering & Modifying HTS Codes and Country of Origin Information for Customer Order Shipment Lines

In Customer Order Entry, during the entry of a cutomer
order, you have the option of entering an HTS Code per line and a
Country of Origin per line. After order entry and order fulfillment
are complete, and now that you are ready to ship these lines to the
customers, you can override or set these two options at the shipment
line level.

1. Open the customer order
   that you want to ship.
2. Scroll down the Shipping
   Entry line item table, until you find the HTS
   Code column header.
3. Double-click <HTS
   Code> to select an HTS code from the table.

VISUAL either replaces the existing HTS
Code or sets this as the lines default HTS Code.

4. Double-click <Country
   of Origin> to select a Country of Origin from the table.

VISUAL either replaces the existing Country
of Origin or sets this as the lines default Country of Origin.

5. Click the Save
   button to commit these new or modified settings.
