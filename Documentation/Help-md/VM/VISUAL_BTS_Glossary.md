| --- | --- |
| 2D (two dimensional) symbol | a machine-readable symbol which must be examined both vertically and horizontally to read the entire message. A 2D symbol may be one of two types of machine-readable symbols: a Matrix Symbol or a Stacked Symbol. 2D symbols differ from linear bar codes in that they have the ability for high data content, small size, data efficiency, and error correction capability. |
| Accredited Standards Committee (ASC) | a committee that defines the structure of EDI Transaction Sets in the United States. |
| Advance Ship Notice (ASN) | an EDI transaction where the supplier provides the customer with shipment details prior to the actual arrival of the shipment. |
| Alphanumeric | a character set which contains letters, numbers, and usually function characters such as punctuation marks. |
| American National Standards Institute (ANSI) | a non-governmental organization responsible for the development of voluntary national standards. |
| Aperture | the opening in a physical baffle that establishes the field of view in an optical system. |
| ASCII | the character set and code described in American National Standard Code for Information Interchange, ANSI X3.4-1983. Each ASCII character is encoded with 7 bits (8 bits including parity check). The ASCII character set is used for information interchange between data processing systems, communication systems and associated equipment. The ASCII set consists of 128 characters. |
| Autodiscrimination | the capability of a reader to automatically recognize and decode multiple machine-readable symbologies. |
| Automatic Identification Manufacturers Association (AIM) | an international industry association and for automatic identification & data capture. |
| Average Narrow Element Width | the average of the widths of the narrow bars and Narrow spaces in a bar code symbol as calculated by the following formula:  X=(b + s) / 2  X=average narrow element width  b=average of all narrow bars in a bar code symbol b=Sb/nb  Sb=sum of narrow bar widths  nb=number of narrow bars in a bar code symbol.  s=average of all narrow spaces in a bar code symbol  s=Ss/ns Ss=sum of narrow space widths (excluding intercharacter gaps)  ns=number of narrow spaces in a bar code symbol. |
| Average Wide Element Width | the average wide element width is inherent in the calculation of the element width ratio. See "Element Width Ratio." |
| Bar | the darker elements of a bar code symbol, either narrow or wide in dimension. |
| Bar Code | an automatic identification technology which encodes information into an array of parallel rectangular bars and spaces of prescribed widths. |
| Bar Code Character | a single group of bars and spaces which represent an individual letter, number, or function (e.g. start/stop). |
| Bar Code Density | the number of characters which can be represented in a linear unit of measure. Bar code density is often expressed in characters per inch (CPI). SPI is a function of the "X" dimension, element width ratio, and intercharacter gap. These three parameters must be defined when describing density as CPI. |
| Bar Code Reader | a device used to decode a bar code symbol. |
| Bar Code Symbol | a combination of bar code characters including quiet zones, start/stop characters and data characters which form a complete, scannable entity. Code 39 symbols also contain intercharacter gaps and may contain check characters and data identifier characters. |
| Bar Code Symbol Width | the distance between the outside edges of the minimum quiet zones. Sometimes called "symbol length". |
| Bar Height | the bar dimension perpendicular to the bar width. |
| Bar Width | the thickness of a bar measured from the leading edge to the trailing edge of the same bar. |
| Bidirectional | a bar code capable of being read correctly in opposite scanning directions, e.g. from start character to stop character or from stop character to start character. |
| Binary | a number system that uses only 1s and 0s. |
| Building Block | the building block is the basic unit of the label format. |
| Carrier | in a transaction, the party that provides freight transportation services. |
| Carrier Tracking Number | a unique number used by the carrier, customer, and/or supplier to track freight while consigned to the Carrier the shipping label number may be used for this process. |
| Character | Data character - a letter digit or other member of a character set like ASCII.  Symbol character a unique pattern of light and dark elements which is defined for that symbology there is not necessarily a one-to-one or unique correlation between symbol characters and data characters. |
| Character Set | those characters available for encodation in a particular automatic identification symbology. |
| Check Character | a character included within a message whose value is used for the purpose of performing a mathematical check to ensure the accuracy of that message. |
| Code 39 | for the purposes of this guideline, Code 39 (also known as Code 3 of 9) shall mean the symbology as specified by AIM Uniform Symbology Specification Code 39. |
| Code Set | the specific assignment of data characters to symbol character values. |
| Coding | a recognized set of abbreviations or conventions for representing specific elements of data. |
| Common Item Pack | a pack which contains all like items, i.e., same part/item numbers. |
| Compaction Algorithm | a method of encoding data in away that reduces the number of bytes required for storage, typically without loss-this method allows the data to be reconstructed to their original form. |
| Compliance Indicator | a specified character or string of characters indicating that the message that follows complies with an industry, regional, or international standard. |
| Container | Receptacle or a flexible covering for shipment of goods such as a box, bag, package or pallet. |
| Container ID | alphanumeric field used by the shipping company to identify the shipment. |
| Customer | in a transaction, the party that receives, buys, or consumes an item or service. |
| Customer Part Number | the part number as defined by the customer. |
| Data Element | the smallest named item of information that can convey data, analogous to a field in a data record or a word in a sentence. |
| Data Element Separator | the special character used to separate data elements in a data format. |
| Data Field | a message consisting of a data identifier immediately followed by its associated data. |
| Data Identifier (DI) | a specified character string within a bar code symbol or other data field which defines the general category or specific use of the data following it in the symbol Data Identifiers are defined in the ANSI MH10.8.2, Data Application Identifier Standard. |
| Data Stream | for the purposes of this document, a data stream is the string of data resulting from the reading of a single 2D symbol or a string of concatenated 2D symbols. |
| Data Structure | the organization of data elements in a message. |
| Data Universal Number System (D-U-N-S) | assigned by Dun and Bradstreet. |
| Decoder | the electronic subsystem which receives signals from the scanner and performs the algorithm to interpret signals into meaningful data. |
| Density | the number of data characters that can be represented in a unit of measureLinear bar code density is often expressed in characters per inch (CPI); 2D symbol density is often expressed in characters per square inch (CPSI). |
| Depth of Field | the distance between the maximum and minimum plane in which a scanner is capable of resolving a bar code symbol. |
| Diffuse Reflection | the sum of the components of the reflected light which emanates in all directions from the reflecting surface. |
| Discrete Code | a bar code symbology where characters are independent and the spaces between characters (intercharacter gaps) do not encode information. |
| EDI Message Data | the data communicated between trading in a standard format and syntax, e.g., ANSI ASC X 12 or UN/EDIFACT. |
| Electronic Data Interchange (EDI) | for the purposes of this document, EDI shall mean the computer communication of data between trading partners. |
| Element | in a bar code symbol, a single bar or space. |
| Element Width | the thickness of an element measured from the leading edge of an element to the trailing edge of the same element (See X Dimension). |
| Element Width Ratio | the ratio of average wide element widths to average narrow element widths within a bar code symbol as calculated by the following formula:  N= (B + S)/2X  N=element width ratio  B=average width of all wide bars in a bar code symbol B=SB/nB  SB=sum of wide bar widths  nB=number of wide bars in a bar code symbol  S=average width of all wide spaces in a bar code symbol S=SS/nS  SS=sum of wide space widths (excluding intercharacter gaps and quiet zones)  nS=number of wide spaces in a bar code symbol  X=average narrow element width |
| Error Correction | a reader/decoder use of mathematical schemes to reconstruct or replace damaged or missing symbol characters to enable the reading of the symbol data. |
| Error Correction Characters | symbol characters that are used for error correction and detection-These characters are calculated automatically from the other symbol characters. |
| Error Correction Level | an indicator of the number of characters used in symbology for error correction-Higher levels of error correction allow for correcting greater potential symbol damage. |
| Error Detection | the automatic determination that a decoded messages content is incorrect - Error detection will keep the symbol from being decoded as erroneous data. |
| Error Detection Characters | symbol characters which are reserved for error detection-These characters are calculated automatically from the other symbol characters. |
| Highlighting Line | horizontal divider lines placed above and/or below building block or blocks Highlighting lines are easily distinguishable from the horizontal separator lines used to separate other building blocks. This visual difference may be the result of using a thicker line chosen by the labeler. |
| Human-Readable Interpretation | the letters, digits or other characters associated with specific symbol characters and printed along with the linear bar code or 2D symbol. |
| Individual Part | a single part, item, or material purchased, manufactured, and/or distributed. |
| Interchange Control Header (ISA) | to start and identify an interchange of one or more functional groups and interchange-related control segments of an EDI transmission. |
| Intercharacter Gap | the space between two adjacent bar code characters. |
| International Article Numbering Association | (EAN) (formerly the European Article Numbering Association) the international organization that administers the manufacturer and item numbering scheme most commonly used for retail bar coding internationally (see also UCC). |
| Item | a single part or material purchased, manufactured, and/or distributed. |
| Kit | a set of components for a single assembly part, packaged together as a single part number, for inclusion into one assembly. |
| Label | a card, strip of paper, etc. marked and attached to an object to convey information. |
| Labeler | term to identify the organization responsible for the labeling of a Unit Load/Transport Package (UL/TP). |
| Like Parts Pack | pack which contains all like items, i.e., same part/item number. |
| Linear Symbol | a one-dimensional bar code symbol such as Code 39 or U.P.C. - An array (linear sequence) of variable width rectangular bars and/or spaces, arranged in a predetermined pattern, following specific rules, to represent elements of data; these bar and space patterns are referred to as characters. A bar code symbol typically contains a leading quiet zone, start character, data characters including a check character (if any), stop character, and a trailing quiet zone. |
| Lines Per Block (LPB) | unit of measure defining the height of text characters. |
| Manufacturer | actual producer or fabricator of an item; not necessarily the supplier in a transaction. |
| Master Label | a label used to identify and summarize the contents of a multiple pack of common items. |
| Master Load | a multiple pack or unit load of common items (sharing a single part number), such as a pallet. |
| Master Pack | unit load containing common (like parts) items. |
| Matrix Symbol | a two-dimensional array of regular polygon-shaped cells where the center-to-center distance of adjacent elements is uniform - The arrangement of the cells represents data and/or symbology functions. Matrix symbols may include recognition patterns that do not follow the same rule as the other elements within the symbol (e.g., Data Matrix and MaxiCode). |
| Maximum Bar Reflectance | the maximum reflectance within any bar when the measuring aperture is wholly contained within a bar. |
| Maximum Space Reflectance | the maximum reflectance within any space when the measuring aperture is wholly contained within a space. |
| Message | a contiguous stream of data elements, including formatting characters and delimiters, to be encoded in a symbol or set of symbols. |
| Message Envelope | consists of a message header and a message trailer that delimits the start and end of a data stream in a given message. |
| Message Header | a character or group of characters that defines the start of a Message Envelope. |
| Message Trailer | a character used to identify the end of a Message Envelope. |
| Mil | a unit of measure equal to 0.001 inch, commonly used to refer to width of bar code elements, e.g., 0.0075 inch would be expressed as 7.5 mils. |
| Minimum Bar Reflectance | the minimum reflectance within any bar when the measuring aperture is wholly contained within a bar. |
| Minimum Reflectance Difference (MRD) | the difference between the minimum space reflectance value (darkest space) and the maximum bar reflectance value (lightest bar) as measured across the entire bar code symbol. |
| Minimum Space Reflectance | the minimum reflectance within any space when the measuring aperture is wholly contained within a space. |
| Mixed Item Label | label used to identify and summarize the contents of a mixed item pack. |
| Mixed Item Pack | a pack containing items with different part/item numbers. |
| Mixed Load | a multiple pack or unit load of mixed items (different part numbers), such as a mixed-container pallet. |
| Mixed Load Label | a label used to designate mixed item shipping packs. |
| Multiple Pack | a pack containing smaller packages (subpacks) of items. |
| Nanometer | a unit of measure used to define the wavelength of light, equal to 10-9 meters. |
| Non-Standard Quantity Pack | a pack which contains variable quantities of like items. |
| Opacity | the property of a material to obstruct the transmission of light and prevent show-through. |
| Overhead Characters | characters encoded in the symbol that are not data. |
| Pack, Package or Load | a unit which provides protection and containment of items plus ease of handing by manual or mechanical means. Examples of containers or packs which normally are disposable include bags, cartons, cartons on pallets and pallet boxes. Examples of containers or packs which are returnable/reusable include bins (with mesh or solid sides and ends), racks (plain or with special dunnage), racks with wire mesh sides and ends, tubes and drums. |
| Part | an identifiable item which has a unique name and/or number assigned to it (see also Individual Part). |
| Part Number | a unique code which identifies a single part, individual part, assembly, component, or kit. |
| Print Contrast Signal (PCS) | the ratio of the reflectivities between bars and spaces of a bar code symbol, commonly expressed in percent. |
| Quantity | indicates the number of parts, item or other units of measure in the container. |
| Quiet Zone | an unprinted area on the substrate which precedes the start character of a bar code symbol and follows the stop character. Sometimes called the "clear area". |
| Reader | a device comprising of a scanner and a decoder. |
| Recognition Pattern | a unique pattern, specific to each symbology, that does not represent data; its purpose is to locate the symbol within the readers field of view-The recognition pattern may aslope used for other purposes. (Also known as a finder pattern). |
| Reflectance | the ratio of the amount of light of a specified wavelength, or series of wavelengths, reflected from a test surface to the amount of light reflected from a barium sulfate or magnesium oxide standard. |
| Resolution | in a bar code reading system, the narrowest element dimension which can be distinguished by a particular device. In a bar code printing system, the narrowest average element width which can be printed by a particular device. |
| Scanner | a device which converts optical information into electrical signals for input to a reader. |
| Self-Checking | a bar code symbology using a checking algorithm which can be applied to each character to guard against undetected errors. |
| Serial Number | string of numeric or alphanumeric characters that is not repeated within 365 days to a single customer. |
| Ship-From | on a transport level, the address of the location where the carrier will return the shipment if the container is undeliverable. |
| Shipping Pack | a pack used for shipping items from one facility to another. |
| Shipping/Parts Identification Label | label or tag used to identify the contents of a shipping container. |
| Ship-To | address of the location where a carrier will deliver the freight. |
| Show-Through | the generally undesirable property of a substrate that permits underlying markings to be seen. |
| Sortation | the process by which an automated material handling system routes containers in a distribution environment. |
| Space | the lighter elements of a bar code symbol, either narrow or wide in dimension. |
| Space Width | the thickness of a space measured from the leading edge to the trailing edge of the same space. |
| Spectral Response | the variation in sensitivity of a reading device to light of different wavelengths. |
| Specular Reflection | the mirror-like reflection of light from a surface. |
| Spot | the undesirable presence of ink or dirt in a space, intercharacter gap or quiet zone (cf "Void"). |
| Standard Quantity Pack | pack which contains the same quantity of like items. |
| Structure | the order of data elements in a message. |
| Subpack | one of the smaller packs that make up a larger pack. |
| Substrate | the surface on which a bar code symbol is printed. |
| Supplier | in a transaction, the party that produces, provides, or furnishes a product or service. |
| Supplier ID | the numeric or alphanumeric data used to identify the supplier. |
| Supplier/Vendor ID | the numeric or alphanumeric code used to identify the supplier/vendor. |
| Symbol | a graphic array of light and dark elements that forms a complete scannable entity. |
| Symbology | the study of bar code symbols, particularly the architecture parameters of encoded characters. |
| Symbology Identifier | an optional, pre-defined sequence of characters that may prefix transmitted data from a linear bar code or 2D symbol reader indicating the symbology read and any options enabled in the reader or special features of a symbology encountered (e.g., presence of "]u1" for Maxi code). |
| Symbology Overhead | features and structure (such as finder patterns and quiet zones) used to aid in finding and decoding the symbol. |
| Syntax | the way in which data as combined to form messages - Syntax also includes rules governing the use of appropriate identifiers, delimiters, separator characters, and other non-data characters within the message. Syntax is the equivalent of grammar in spoken language. |
| Tag | label (card) that is attached to a shipping container. |
| Trading Partner Label (TPL) |  |
| Trading Partners | all members within the channels of distribution within an industry (carriers, customers, suppliers, and intermediaries). |
| Transport Container | container intended for the transportation and handling of one or more articles, smaller containers, or bulk material. |
| Transportation Tracking | the process in which packages in route between shipper and receiver are located at any point in time. |
| Uniform Code Council (UCC) | the standards association of the U.S. retail industry The UCC sets the standard for U.P.C., the Uniform Product Code used for point of sale scanning in retail. UCC in the U.S. works with EAN internationally (see also EAN). |
| Unique Container Identification | supplier identification and a container serial number that, together, uniquely identify the container to trading partners (sometimes referred to as license plate). |
| Unique Package Identification | a code or combination of codes that uniquely identifies a package to all trading partners using an accepted and defined supplier ID and package ID number. |
| Unit Load | one or more transport containers or other items held together by means such as strapping, interlocking, glue, shrink wrap, or net wrap, making them suitable for transport, stacking, and storage as a unit. |
| Unit Load or Transport Package (container) (UL/TP) |  |
| Unit Pack | the first tie, wrap, or container of a single item a unit pack may be an item packaged singly or a kit of items to be added to a single assembly. |
| United Nations/EDI For Administration, Commerce, and Trade (UN/EDIFACT) | the acronym for the international data standard for electronic business messages. |
| Verifier | a device to measure certain bar code attributes. |
| Void | the undesirable absence of ink in a bar (cf "Spot"). |
| "X" Dimension | the average width of the narrow bars and narrow spaces in a bar code symbol. See "Average Narrow Element Width". |
| "Y" Dimension | the intended height of the elements dictated by the application, or symbology specification, or both. |
