# Saving a Default Dashboard Configuration

After you decide which tools to view in the window, you can save
the configuration as your default configuration for quick access later.

To save your default configuration:

1. Choose the tools to
   display in the window.
2. Click the Save
   button.

As you analyze your buffer information, you might close certain
tools and open others. To restore the dashboard to your default settings,
click the View drop-down button and re-select Default.
