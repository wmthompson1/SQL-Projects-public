# Modifying Form Formats

The format of Vendor RFQ Entry reports can be modified
using Centura Report Builder. The following QRP report files apply:

## Print Request for Quote

|  |  |
| --- | --- |
| Standard | VMRFQFRM.QRP |
| User-Defined 1 | VMRFQUD1.QRP |
| User-Defined 2 | VMRFQUD2.QRP |
| User-Defined 3 | VMRFQUD3.QRP |

## Print Request for Quote Log

|  |  |
| --- | --- |
| Vendors | VMRFQRP1.QRP |
| Parts | VMRFQRP2.QRP |
| Vendor Quotes | VMRFQRP3.QRP |
