# BTS-LP Label Print Utility - Designing Barcode Labels for Thermal Printers

This utility allows the user to design and define
labels within specific categories. These categories allow the definitions
to produce labels based on their usage and data requirements. The
label categories are:

* Raw Materials and/or Received
  Materials Labels
* Parts Labels
* Traceability Labels
* Finished Goods Labels
* Shipping Labels
* Warehouse/Location Labels

These categories allow for a structure to better determine the data
sources as well as segmentation of the printing duties into their
respective areas. Therefore, the scope of application interaction
is between VISUAL BTS, and Infor VISUALs Order Management, Work Order
Management and Purchase Order Management Systems, and the VISUAL Distribution
Center Management System (DCMS).

## Loftware - The Essential Thermal Label Design Software

Loftware is a label design and print utility for
barcode thermal printers that is a cornerstone to the label print
utility. Through the use of Loftware we are able to shortcut the
process of specific label designs and printer intricacies. Through
the use of the ActiveX control we are leveraging their printing abilities
and setting the groundwork for future development using the Microsoft
COM/DCOM techniques.
