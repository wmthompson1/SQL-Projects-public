# Entering Part Aliases for Customer Order Lines

As you are entering information for a customer order
line, instead of double-clicking <Part ID>
to select a Part ID from the part table, enter the parts alias in
the Part ID field. When you tab or move the cursor to enter more information,
VISUAL replaces the Part Alias with the Part ID proper.
