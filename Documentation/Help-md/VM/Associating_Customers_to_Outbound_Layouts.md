# Associating Customers to Outbound Layouts

You need to associate each outbound layout with one
or more VISUAL Customer IDs or Vendor IDs so that when you run the
layout through VMDI Exchange, it knows whose data (invoices, ship
notices, acknowledgments, purchase orders, and purchase receipts)
to export. To do this:

1. If you havent done
   so already, save your outbound layout by clicking on the Save button.
2. Select Associate
   from the File menu.

The Associate Outbound Format dialog box
appears.

3. Select which customers
   or vendors will use this layout when exporting data.

You can click the Select
All button to select all of the customers or vendors listed
and the Unselect All button to deselect
all the customers or vendors you have selected.

4. Click the Save
   button, then the Close button.

The customers or vendors you selected are
now associated with this layout. When this layout is selected in the
VMDI Exchange window, data for all customers or vendors who have been
associated with this layout will be exported.
