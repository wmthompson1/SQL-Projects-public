# Activating the Links Toolbar

To activate the links and the Links toolbar:

1. Select Admin,
   Preferences Maintenance.
2. Select Maintain,
   Links Preferences.
3. In the Links Preferences
   dialog box, select the Links Enabled
   check box.
4. Exit VISUAL, and then
   relaunch VISUAL.
5. On the main menu, select
   File, Links.
   The Links toolbar is displayed. The URL specified for the first
   button displayed in the toolbar is shown in place of the VISUAL
   splash screen.
