# Setting Profile Options for the Manufacturing Window

The Manufacturing Window can allow separate access
to different types of masters.

1. Highlight Manufacturing
   Window from the Program Security dialog box.
2. Click the Set
   Profile Options button.

The Set Manufacturing Window Profile Options
dialog box appears.

3. Select from the following
   read-only options:

Engineering Masters
- This sets the profile string in the Program Security window to E=RO.

Quote Masters
- This sets the profile string in the Program Security window to Q=RO.

Work Orders -
This sets the profile string in the Program Security window to W=RO.

If you select all three check boxes, the
profile string the Program Security window changes to Read Only.

4. Click Ok
   to set the profile options, or Cancel
   to close the dialog box without saving the profile options.
