# Specifying a Filter for the Data Source

To specify a filter for the data source:

1. Select File,
   Dashboard.
2. Select Edit,
   Edit Data Source.
3. In the Data Source ID
   field, select the data source.
4. Click the Filter
   tab.
5. Specify the SQL statement
   to use as the filter. The statement you specify is appended after
   a WHERE clause in the query.
6. Click Save.
