# Printing an Analytic

When you print an analytic, your only option is to
send the analytic to a printer. You cannot view, e-mail, or print
the analytic to a file.

To print an analytic:

1. Select File,
   Dashboard.
2. In the dashboard, select
   File, Open
   and select the dashboard to use.
3. Click in the analytic
   to select it.
4. Click Print.
5. Select your printer,
   and then click Ok.
