# Choosing a Button Display

When you first open the view panel, a toolbar is displayed at the
bottom. You can use the buttons on the toolbar to select the information
type to view. For example, to view the chart, click the Chart toolbar
button.

You can display the buttons in an accordion view. In the accordion
view, the buttons are larger and stacked one on top of another. Perform
one of these steps:

* Drag up the bar that separates
  the toolbar from the information. As you drag the bar up, buttons
  are added to the accordion view.
* Click the arrow in the toolbar
  and select Show More Buttons. Repeat this step to display additional
  buttons.

To display the buttons in a toolbar, perform one of these steps:

* Drag down the bar that separates
  the buttons from the information. As you drag the bar down, buttons
  are removed from the accordion view and added to the toolbar.
* Click the arrow in the toolbar
  and select Show Fewer Buttons. Repeat this step to remove all
  buttons from the accordion view and add them to the toolbar.
