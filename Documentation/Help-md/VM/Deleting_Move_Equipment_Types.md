# Deleting Move Equipment Types

You can delete a move equipment type only if it is not used on a
move request record.

1. Select Inventory,
   Move Request.
2. Select Maintain,
   Move Equipment Type.
3. Select the row that
   contains the ID that you are deleting.
4. Click Delete.
   The line is marked for deletion.
5. Click Save.
   The ID is deleted.
