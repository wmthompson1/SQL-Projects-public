# Accessing Purchase Receipt Entry

You can access Purchase Receipt Entry directly from the Purchase
Order Entry window. Right-click in the Purchase Order Entry header
and select Receiving from the menu. If you selected a purchase order
in the Purchase Order Entry window, the system inserts the details
of the purchase order into the Receiving window. If the Purchase Order
Entry window is blank, then the Receiving window is also blank.
