# Printing Capability Reports

You can print the results of Quality tests in three
different report formats: Work Order, Part, and Test.

1. From the Eng/Mfg menu,
   select Capability Report.
2. Select Work
   Order, Part, or Test
   as a Report Type for the report.
3. Enter a starting and
   ending date range in the Starting and Ending Date fields.
4. Depending upon your
   Report From selection, click the Starting Work
   Order ID button to select the work order, part, or test
   with which you want to start the report.
5. Depending upon your
   Report Form selection, click the Ending Work
   Order ID button to select the work order, part, or test
   with which you want to end the report.
6. Select the Summary
   check box if you want VISUAL to print a summary report instead
   of a detailed report.
7. Select how you want
   VISUAL to group the report: By Subgroup Number, Work Order ID,
   Part ID, or Test ID.

You can select only one option at a time.
Unless your Report Type selection is Test, the test option button
is not available.

8. Select what you want
   to appear on the report in addition to the standard information:
   Specifications, Current Limits, and Calculated Limits. You can
   select all three or any combination thereof.
9. Specify the output format
   for the report. See [Report
   Output Options](Report_Output_Options.htm).
10. From the File menu,
    select Print/View or click the Print button.
