# Copying Detail Grids

You can create a new detail grid by copying an existing
detail grid. To copy a detail grid:

1. Select File,
   Dashboard.
2. Select Edit,
   Edit Detail Grid.
3. In the Configuration
   ID field, select the detail grid.
4. Click Copy.
5. In the New Configuration
   ID field, specify the ID of the new detail grid.
6. Click Ok.
   The new detail grid is loaded in the dialog.
