# Adding and Deleting Defect Classifications

Use Quality Maintenance to add and delete defect classifications.

## Adding Defect Classifications

1. Select Defect
   Classification from the Edit menu.

The Defect Classification Coding dialog
box appears.

2. Click the Insert
   button.

A line appears in the list of the defect
classifications.

3. In the Name column,
   enter a name for the new defect classification.
4. Click the button to
   save the new defect classification.

## Deleting Defect Classifications

1. Select Defect
   Classification from the Edit menu.

The Defect Classification Coding dialog
box appears.

2. Select the defect classification
   you want to delete.
3. Click the Delete
   button.

An x appears to the left of the defect classification
you are deleting.

4. Click the Save
   button to complete the deletion.
