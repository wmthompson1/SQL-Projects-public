# Resetting Material Requirement Costs in the Costed Bill of Material Report Window

When you reset material requirements costs, costs
in engineering masters, quote masters, and work orders are updated
based on the costs that are specified in Part Maintenance.

You can also reset material requirement costs in Part Maintenance.

1. Select Eng/Mfg,
   Costed Bill of Material Report.
2. In the Site ID field,
   select the site where you are updating material requirement costs.
3. Select Edit,
   Reset Material Costs.
4. Complete one of these
   tasks:

* To reset costs
  for all parts in the site that you selected in step 2, click All Parts.
* To select
  the parts whose costs you are resetting, click Selected
  Parts. Click the Insert Row button,
  then click the Part ID browse button
  to select the parts.

The Current Part Only option is not available.

5. Specify the types of
   masters to update:
6. To update
   engineering masters, select the Engineering
   Master material requirements check box.
7. To update
   work orders, select the Work Order material
   requirements check box.
8. To update
   quotes, select the Quote Master material requirements
   check box.

6. If you selected the
   Work Order material requirements check
   box, you can reset material costs based on the status of each
   material requirement. Select the According
   to Material Requirement Status check box and select the
   statuses to use:

* Unreleased
* Firmed
* Released
* Closed
* Cancelled
* All

7. Select the According
   to W/O or Master creation date check box, and then use
   the On or after and On or before fields to specify the create
   date of the masters that you are updating. To update all masters,
   specify today's date in the On or before field and leave the On
   or after field blank.
8. Click Ok.
