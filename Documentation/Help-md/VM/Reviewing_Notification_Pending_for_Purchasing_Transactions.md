# Reviewing Notification Pending for Purchasing Transactions

You can view a list of notifications for an order that are eligible
for email but have not yet been sent. To view the list.

1. Select Purchasing,
   Purchase Order Entry.
2. Select the purchase
   order.
3. Select Info,
   Notification Pending. For each notification,
   this information is displayed:

Program ID  The
ID of the program that generated the notification.

* VMPURENT
    Purchase Order Entry
* VMRCVENT
    Purchase Receipt Entry
* VFAPCENT
    Payment Entry

Doc User ID 
The ID of the user who created the transaction that requires a notification
to be sent.

Vendor ID  The
ID of the vendor on the purchase order.

Vendor Name 
The name of the vendor on the purchase order.

Document ID 
The ID of the transaction that requires a notification to be sent.

Site ID  The
Site ID of the transaction that requires a notification to be sent.

Doc Create Date 
The Create Date of the transaction that requires a notification to
be sent.

Order Contact Name 
The Contact Name on the purchase order associated with the Document
ID.
