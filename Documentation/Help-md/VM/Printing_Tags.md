# Printing Tags

The next phase in the physical inventory process is
the printing of tags to manage the count. You can manage the count
in one of two ways:

* Use Print Tags to print
  individual tags to record counts and mark inventory.
* Use Print Tag List to print
  all tags in report format, and enter counts on this report. This
  is less common than printing individual tags.

To print tags:

1. If you are licensed
   to use multiple sites, click the Site ID
   arrow and select the site that contains the physical inventory
   count. If you are licensed to use a single site, this field is
   unavailable.
2. Select Print
   Tags from the File menu.

The Print Tags dialog box appears.

3. Enter the Physical Count
   ID.

When a count is in progress, this field
automatically defaults to that Count ID. You can also use this option
to print or view tags from a historical count that has been saved.

4. Select the print output.

You can select Print, View, or File.

5. Choose the appropriate
   Sort By radio button.

You can sort the tags by tag number or Location
ID/Part ID.

6. Click Ok.

A standard Windows dialog box appears.

7. Make the appropriate
   selections and click Ok.

VISUAL prints the tags.

The standard tag format shows Tag Number,
Location ID, Part ID, Part Description, and Unit of Measure. There
are also blank lines for entering Quantity and Counted by information.
