# Printing Commissions Distribution

This report shows details of how projected commissions
are broken down for each Sales Rep, or for each order. For Order ID
sequencing, each booked order is listed, along with the order amount
and commission amount for each Sales Rep. For Sales Rep sequencing,
each Sales Rep is listed, along with the order amount and commission
amount for each order. Amounts are totaled for each order or each
Sales Rep.

The report shows customer order line item amounts and commissions,
and the standard or override commission sharing currently defined
for the sales rep. Invoices and payments are not referenced. To report
on actual commissions paid, use the [Commissions
Actuals](Printing_Commissions_Actuals.htm) report.

1. Select Sales,
   Sales Commission Report.
2. Click Print
   Commissions Report.
3. If you are licensed
   to use multiple sites, click the Site ID(s)
   arrow and select the sites to include in the report. If you are
   licensed to use a single site, this field is unavailable.
4. In the Sequence section,
   click a sequence option. After you click a sequence option, the
   label for the Starting ID and Ending ID fields change. You can
   click Order ID or Sales Rep.
5. Use the Starting ID
   and Ending ID fields to specify the records to include in the
   report. To view a range of records, specify the first ID in the
   Starting ID field and the last ID in the Ending ID field. To view
   a single record, specify the same ID in both fields. To view all
   records, leave the fields blank.
6. Click the Report
   Currency arrow to select a currency from the list. If you
   are printing a report for a single site, the entity currency associated
   with the site is inserted. If you are printing a report for multiple
   sites, the drop-down list shows the tracking currencies shared
   by the selected sites.

If you are printing a summary report, the
sites you select must share at least one tracking currency. If no
shared tracking currencies are found, Not Available is inserted.
You cannot run a summary report if a shared tracking currency is not
found. Clear site selections until a shared tracking currency ID is
displayed in the drop-down list.

If you are printing a detail report, you
can print the report in native currency if a shared tracking currency
is not found.

7. From the output list,
   select the output format for the report. If you select email,
   select the PDF Format check box to attach
   a PDF to the email instead of an RTF file. See [Report
   Output Options](Report_Output_Options.htm).
8. Click Ok.
