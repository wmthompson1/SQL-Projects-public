# Printing Thermal Labels from Customer Order Entry

Customer Order Entry allows you to print thermal labels
for customer orders and customer order acknowledgments. Before you
can print a label, though, you must create the appropriate label types
to use in the window. The default label type for the Customer Order
Entry Window is SHIPPING. See the section on creating thermal labels
and the cross-reference chart in the thermal label printing section
for more information.

1. From the main Sales
   menu, select Customer Order Entry.

The Customer Order Entry window appears.

2. From the File menu,
   select Print Acknowledgment or Print Customer Orders.

The Print dialog box appears.

3. Click the Thermal
   Labels button.

The VISUAL Print Thermal Labels dialog box
appears.

4. Click the Label
   ID button to choose the label you want to print for this
   customer order/acknowledgment.

VISUAL limits your choice to Label IDs with
SHIPPING label types.

5. Click the Label
   Group ID button to choose the Label Group to print. If
   you are printing only one label, do not choose a group. VISUAL
   limits your choice to either a Label ID or a Label Group ID.

VISUAL auto-fills the read-only Label Type
field with SHIPPING.

6. If you are printing
   a group, click the Insert button and
   add any new labels, if appropriate.
7. Specify a print quantity
   in the Print Qty. field.
8. Verify that you thermal
   label printer is configured properly, and then click the Print Labels button.
