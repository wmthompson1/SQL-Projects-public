# Viewing Gross Demand Information for Short Parts

You can view gross demand detail for short parts,
or parts that you have insufficient quantities of to meet order demand
during specific planning periods. When you select this option, VISUAL
displays information for all parts that come up "short."

If you are licensed to use multiple sites, click the Site ID drop-down
button and select the site to use. If you are licensed to use a single
site, this field is unavailable.

Select a part in the Part ID field.

To view short part gross demand information, select the Short Parts
check box located under the part information tabs in the Advanced
Material Planning window.

VISUAL processes each part and displays the results for short parts
in the line item table.

The point at which quantities of the part become short are set off
by blue.
