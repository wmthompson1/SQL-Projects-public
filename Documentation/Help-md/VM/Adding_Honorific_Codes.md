# Adding Honorific Codes

Honorifics precede an individuals name. Examples include Mrs.,
Mr., and Dr.

To add an honorific code:

1. Select Admin,
   Application Global Maintenance.
2. Select Maintain,
   Table Maintenance, Honorific
   Table.
3. Click Insert.
4. Specify the honorific.
5. Click Save.
