# Entering Part Alias Types

1. From the Maintenance
   section, click the Part Alias button.

The Part Alias Types dialog box appears

2. Click the Insert
   button to begin adding line item information. Enter information
   into the following fields:

Part Alias Type
- A unique identifier for the part alias type.

Description - A
description of the part alias type.

3. Click the Save
   button when you are finished entering part alias type information.
