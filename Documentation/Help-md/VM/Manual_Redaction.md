# Manual Redaction

The Individual Privacy program is a tool that you
can use to help you redact personal data from your database. However,
the program does not search all areas of the database that could contain
personal information. You may need to manually delete data through
the standard VISUAL interface.

See [Individual
Privacy Program Limitations](Individual_Privacy_Program_Limitations.htm).
