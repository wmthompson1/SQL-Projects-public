# Working With Earning Codes

Earning codes specify the Earning code of an employee.
Earning codes copy to labor tickets during creation and for the subsequent
payroll processing. You must first define the Earning code by selecting
Earning codes from the Edit
Menu.

## Adding New Earning Codes

1. Select Earning
   Codes from the Edit menu.

The Earning Codes dialog box appears.

2. Click the Insert
   button.

This adds a new line to the end of the table.

3. Enter a new earning
   code in the Earning Code column.
4. Enter a description
   for the earning code.
5. Click the Save
   button to save the new earning code.

## Deleting Earning Codes

To delete an Earning Code, highlight the earning code line and click
the Delete button.

## Copying Earning Code Information

To copy information from one Earning Code to a new line:

1. Select the information
   you want to copy and click the Copy
   button.
2. Click the Insert
   button to insert a new line.
3. Click the Paste
   button.

VISUAL pastes all of the information (except
the Earning Code) into the new line.

4. Enter the earning code
   to be associated with the new entry in Earning Code field.
