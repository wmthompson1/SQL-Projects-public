# Repricing Requisition Line Items

Use this feature to update unit prices on purchase
requisition lines with the price from the preferred vendor quantity
break table.

1. Select the line or lines
   you want to reprice.
2. Select Reprice
   Lines from the Edit menu.

A dialog box appears.

3. Click the Yes
   button to reprice the lines.

VISUAL refreshes the lines with the latest
prices from the vendor.
