# Leadtime View Panel Charts

Use the [Leadtime
View Panel](Viewing_the_Leadtime_View_Panel_Charts.htm) to view information about the leadtime performance
for a particular sales order or quote. You can view two charts: the
Standard vs. Actual (Days) chart and the Percent Complete chart.

The Standard vs. Actual (Days) chart shows a comparison of your
actual performance and the standards that you specified in the Process
Type dialog.

The Percent Complete chart shows the percentage complete of each
step in the process.
