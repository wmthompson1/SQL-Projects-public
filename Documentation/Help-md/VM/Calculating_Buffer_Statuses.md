# Calculating Buffer Statuses

You should perform this procedure each day.To calculate your buffer
statuses:

1. Select Scheduling, Buffer Management.
2. If you are licensed
   to use multiple sites, click the Site ID
   arrow and select the site whose buffers you are calculating.
   If you are licensed to use a single site, this field is unavailable.
3. Click Calc
   Buffer Status.
