# Netting by Release Date

Select Net By Release Date from
the View menu to sequence demand and supply
orders together by Release Date and Due Date. They are only aligned
if they fall on the same date. This is like Net By Required Date however
it uses the demand orders release date instead of the required date.
Refer to [Net By Required Date](Netting_By_Required_Date.htm)
for additional information.

The release date for customer orders is the customer
order line promise ship date or customer order header promise ship
date. If both are null, the desired ship date will be used.

The release date for material requirement demand is
the work orders release date. If the work order release date is null,
the work orders want date is used.
