# VISUAL BTS Material Movements, Issue, and Work Orders Table

The Pending Transaction portion of the window contains
the following columns. The Completed Transaction line item contains
the same columns, with the addition of column Transaction ID. After
VMBTSBI has successfully processed the transaction, the ID of the
transaction appears here.

ID - The ID of the transaction.

Trans Type - The type of the transaction.

Trans Date - The date on which the transaction
occurred.

Part ID - The ID of the part that you
are transferring or issuing.

Base ID - The Base ID of the work order
- if there is one - to which this material is allocated.

Lot ID - The Lot ID of the work order
- if there is one - to which this material is allocated.

Split ID - The Split ID of the work order
- if there is one - to which this material is allocated.

Sub ID - The Sub ID of the work order
- if there is one - to which this material is allocated.

Quantity - The quantity that you are issuing
or transferring

From Warehouse - The warehouse from which
you are transferring or issuing the parts.

From Location - The location in the warehouse
from which you are transferring or issuing the parts.

To Warehouse - The ID of the warehouse
where you are receiving the goods.

To Location - The ID of the location in
the warehouse into which you are receiving the goods.

Operation Sequence Number - The operation
sequence number of this material in a work order.

Requirement Piece Number - The requirement
piece number of the material in a work order

User ID - The ID of the user who is currently
scanning.
