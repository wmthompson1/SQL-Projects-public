# Creating Quotes for Existing Projects

You can specify project information in the Estimating Window to
link a quote to a project. The information you specify links the quote
to the project.

In the Estimating Window header, specify information as you would
in a non-project quote. Then, click the Project tab. Click the Project
Info button and select the project to associate with the quote. The
system inserts the Project Type, the default WBS code, and the Project
Warehouse from Project Maintenance.

In the line table, specify information as you would in a non-project
quote. If you have built the project structure in the Project Window,
you can enter the following project-specific information:

WBS Code Double-click the WBS Code browse
button and select the WBS code to associate with the line. The system
inserts the WBS Description and CLIN number associated with the WBS
code.

Proj Ref Seq No Double-click the Proj
Ref Seq No browse button and select the sequence number to associate
with the line. The system inserts the Proj Ref Sub ID based on your
Proj Ref Seq No selection.
