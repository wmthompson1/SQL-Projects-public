# Editing Saved Work Order Traveller Print Settings

To edit an existing print settings profile:

1. Select Eng/Mfg,
   Work Order Traveller.
2. In the Option Name field,
   specify the name of the profile.
3. Make edits as necessary.
4. Select File,
   Save Print Option.
