# Adding Specifications

Specifications allow you to add free form text to
quotations. This text is printed on Quotations and can specify and
document special needs, notes, and agreements for the customer quote.
You might use specifications to specify packaging requirements, shipping
details, configuration information, or other options that need to
be specified along with a customer order.

There are two types of quotation specifications (specs, for short):
Quote Specs and Line Specs. Quote Specs are associated with the entire
quote, and Line Specs with individual line items.
