# Copying Approval Groups

Use the Copy Group dialog box to copy group information
to a new or existing group. You can copy group information within
a site only. You cannot copy group information in one site to group
information in another site.

To copy a group

1. Select Security,
   Profile/User/Group Security.
2. Select Edit,
   Copy Group.
3. In the Site ID field,
   specify the site where you are copying groups.
4. In the Copy From section,
   specify the group to copy.
5. In the Copy To section,
   specify the group to which you are copying information. You can
   select an existing group or specify a new group ID. If you select
   an existing group, the existing information is overwritten with
   the Copy From group information.
6. Click Ok.
   The group information is copied.
