# Detail Grid Types

The detail grid type determines the measurement scope
of the analytic. You can select these analytic types:

Detail  The Detail type shows all
results from your data source.

Top N The Top N type shows the largest
results found in a value column that you specify.

Bottom N  The Bottom N type shows
the smallest results found in a value column that you specify.
