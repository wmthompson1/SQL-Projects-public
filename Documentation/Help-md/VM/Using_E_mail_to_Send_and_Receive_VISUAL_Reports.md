# Using E-mail to Send and Receive VISUAL Reports

In addition to the three traditional destination outputs
of Print, View, and File, is a fourth option, E-Mail, which you can
use to send and receive VISUAL .QRP data. VISUAL sends the information
from the reports with the same file name as the report QRP file but
with an .RTF (Rich Text Format) file extension.

The E-mail print destination is available in all applications that
support report printing.
