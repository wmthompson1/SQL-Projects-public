# Deleting Activity Categories

To delete activity categories.

1. In an application that
   supports activities, select Edit, Activity Entry.
2. In Activity Entry, select
   Maintain, Activity
   Categories.
3. Select the line to delete.
4. Click Delete.
5. To complete the deletion,
   click Save. To cancel the deletion,
   click Delete again.
