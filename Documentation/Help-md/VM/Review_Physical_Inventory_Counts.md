# Review Physical Inventory Counts

As the count progresses and is completed, you may
want to analyze progress as a whole.

The Review All Counts command allows you to view and manipulate
physical inventory count information by tag group. Print Tag Comparison
and Print Part Comparison provide analysis of on-hand quantities vs.
counted quantities.

## Reviewing Count Information

If you are licensed to use multiple sites, click the
Site ID arrow and select the site where you conducted the count. If
you are licensed to use a single site, this field is unavailable.

In the table, select the physical inventory count to review.

To review count information for a physical count, select Review
All Counts from the File menu.

The Review Counts dialog box appears.

Use this window to review the current count, or a previously saved
one. If a count is in progress, the Count ID defaults to the current
count. You can enter the Count ID for a previous count. The Warehouse
ID for the count appears.

The table displays all tags for one group at a time.

Select either Group Number or Location ID/Part ID from the Review
Sequence section.

The following columns appear; you can modify them
as noted:

Group Number - Select this option if you
want to view counts by group number. When a new count is called in,
the Group Number defaults to 1.

Location ID/Part ID - Select this option
if you want to view counts by Location ID. You can enter the number
of a location or select one from the drop down menu.

Tag Number - Identifies the tag, cannot
be modified.

Location ID - The Location ID for the
part count. Modification is allowed, but is not usual, because you
would not create a new tag, but would instead change the location.

Part ID - The Part ID that was counted
for the location. Modification is allowed, but is uncommon.

Description - Description of the part.
This field is not editable.

Count - The Quantity entered for the tag
in Enter Counts or Enter Recounts. This value can be entered or modified
here. This is equivalent to using Enter Counts or Enter Recounts.

Status - Tags have two status values:
Active or Void. Void is used when you want to ignore (tags cannot
be deleted) the count for a particular part. A Voided tag is essentially
ignored in comparisons and adjustments of inventory unless explicitly
included. To activate or void one or more tags, select the line or
lines, and press the Activate or Void
button. You cannot enter or re-enter counts for voided tags.

User ID - The ID of the VISUAL user who
entered the Count or Recount.

To commit the changes, click the Save
button.
