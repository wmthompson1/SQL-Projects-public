# Using the Customer View Panel

Use the
Customer View Panel to view information about the current customer
whose order you are entering or viewing. Because the Customer View
Panel is a floating panel, you can place it anywhere on the screen
you want. view panels and information panels are for information purposes
onlyyou cannot modify the information that appears in them

To use the Customer View Panel:

From the view menu, select Use
Customer View Panel.

The check mark appears next to the option
and the Customer View Panel appears populated with the current customers
information.

If you use the Close Window button in the
top right corner to close the panel, it will appear automatically
when you select the next customer. To stop using the panel, select
the Use Customer View Panel option in the
View menu. The check mark clears and the panel closes.

## Configuring the Customer View Panel

By adding
or removing data fields to the Customer View Panel, you can tailor
the panel to contain the customer information that is most relevant
for your needs.

To configure the Customer View Panel:

1. Click the Configure
   Table Window menu.

The Configure Table Window appears populated
with all of the available data fields. The Include column indicates
if the field is hidden from view in the panel or is visible.

2. Select the column you
   want by clicking in the row header.
3. To display a column,
   select the Include check box. To hide
   a column, clear the Include check box.
4. When you have finished
   changing what appears in the panel, click the Ok
   button.
