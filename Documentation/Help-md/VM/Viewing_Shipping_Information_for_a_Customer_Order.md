# Viewing Shipping Information for a Customer Order

You can view general shipping information for a customer
order by clicking the Ship Info button located
underneath the line item table.

The Shipments dialog box appears.

The line item table contains the following columns:

Packlist ID - The Packlist ID that either
you or VISUAL assigned to the customer order. If you have not yet
shipped the order, nothing appears in this column.

Shipped Date - If you have shipped it,
the date on which you shipped the order. If you have not yet shipped
the order, nothing appears in this column.

Invoice ID - The Invoice ID that either
you or VISUAL assigned to the customer order. If you have not yet
shipped the order, nothing appears in this column.

Invoiced Date - The date on which either
you in VISUAL assigned the order an Invoice ID. If you have not yet
shipped the order, nothing appears in this column.

Shipped Quantity - The quantity of the
order line that you have shipped. If you have not yet shipped the
order, nothing appears in this column.

Click Close when you are finished viewing
customer order shipment information.
