# Adding Commands to Existing Groups

To add commands to a group, you must select the group
to which you want to add commands.

The right half of the dialog box contains a drop-down list of common
groups of commands available to you for use in your Personal Menu.
When you select a group, a list of available applications appears.
For example, if you select the Sales group, a list of common Sales
applications appears.

To add commands to groups:

1. Select File,
   Personal Menu from the main VISUAL menu.
2. Right-click and select
   Customize.
3. From the drop-down list,
   select the group that contains the item you want to add to the
   displaying menu bar.
4. If you want the new
   command to have an button different than the default button, click
   the down-down arrow on the left and select the button you want
   to use.
5. In the right pane, select
   the command you want to use and click the Add
   Selection button.
6. After adding new commands,
   click OK.

The Customize dialog box closes and VISUAL adds the new command
to the bottom of the group you selected in your Personal Menu.
