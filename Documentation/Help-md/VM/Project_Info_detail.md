# Project Info

This [dashboard](VMDBOARDWhat.md) detail
grid is available only if you are licensed to use projects/A&D
functionality. This detail grid shows basic information about your
projects, such as the type of project, the customer, and target price,
target cost, and target profit.

Double-click a line to open the [Project
Summary](Project_Info.md) analytic. When the analytic is opened, the budget amount
is displayed. You can select other amounts to view in the analytic.

Date Filter Applies to: n/a. A date filter
is not available.

Filters: Project ID, Customer ID, Site
ID

Red text: n/a

Details: Project ID, Project Desc., Project
Type, Proj Type Desc., Customer ID, Customer Name, Target Price, Target
Cost, Target Profit, Currency ID, Site ID
