# PLN CUMM Shipped Quantities - Example 1

| Quantity: | Discrete |
| Subtract Intransit Qty: | NOT Marked |
| Start Proration from Transm Date: | NOT Marked |
| Include Today in Proration: | NOT Marked |
| Sample of example HDR lines from PLN0044.VDI file: | |
| HDRGATERU 243615999 20020304000000000000021787000003761-04 | |
| HDRGATERU 243615999 20020311000000000000032714000003761-04 | |
|  |  |
| The two VDI HDR lines listed above are read as follows: | |
| Quantity of 21787 is forecasted for the week of 03/04/2002, where 3/4 is a Monday | |
| Quantity of 32714 is forecasted for the week of 03/11/2002, where 3/11 is a Monday | |
|  |  |
| After importing via Vmdixchg, the customer forecast records on the Customer Forecasts screen would show up allocated as follows: | |
|  | |
| DATE | QTY |
| 3/4/2002 | 4359 |
| 3/5/2002 | 4357 |
| 3/6/2002 | 4357 |
| 3/7/2002 | 4357 |
| 3/8/2002 | 4357 |
| 3/11/2002 | 6543 |
| 3/12/2002 | 6543 |
| 3/13/2002 | 6543 |
| 3/14/2002 | 6543 |
| 3/15/2002 | 6542 |
|  |  |
| (4359+4357+4357+4357+4357 = 21787) | |
