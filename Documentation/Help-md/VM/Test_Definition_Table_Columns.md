## Test Definition Table Columns

Depending on your table preferences the following columns may be available:

[Step No](javascript:void(0);).

[Seq No](javascript:TextPopup(this))

The sequence number of the operation being purchased to on the work
order/leg specified above. This is required for all purchases to jobs.

[Type](javascript:TextPopup(this))

Operation types are defined in Shop Resource Maintenance, and act
as operation templates.

[Description](javascript:TextPopup(this))

Specify the full description of the item or service.

[Status](javascript:TextPopup(this))

The following status conditions are available:

Firm - Generally indicates that the order
has not yet been completed or placed. There is some leeway in your
interpretation of this status.

Release - Indicates that the order has
been printed/placed. Firmed orders can have their status automatically
changed to Release through the Print Acknowledgment function.

Closed - Indicates all material that will
be received has been received.

Canceled - Indicates that the order will
not be placed, or goods will not be received because of a canceled
order. Used as an alternative to deletion to retain a record of the
PO.

On Hold - The status of the order is currently
on hold; that is, the order is not currently being processed.

[Dimension Type](javascript:TextPopup(this))

There are three dimension types: Mean +/-, Max/Min, and Min/Max.

[Aim](javascript:TextPopup(this))

The aim or target engineering specification.

[LSL](javascript:TextPopup(this))

The lower specification limit.

[USL](javascript:TextPopup(this))

The upper specification limit.

[Limit ID](javascript:TextPopup(this))

A user defined ID of limits.

[Limit Description](javascript:TextPopup(this))

A description of the limits.

[Central Line
X](javascript:TextPopup(this))

The central line of the control chart.

[LCL X](javascript:TextPopup(this))

The lower control limit.

[UCL X](javascript:TextPopup(this))

The upper control limit.

[Central Line
RS](javascript:TextPopup(this))

The central line of the range or standard deviation
control chart.

[LCL RS](javascript:TextPopup(this))

The lower control limit of the range or standard deviation control
chart.

[UCL RS](javascript:TextPopup(this))

The upper control limit of the range or standard deviation control
chart.

[R or S](javascript:TextPopup(this))

Whether Range or Standard deviation is used in XBar control charts.

[Gauge ID](javascript:TextPopup(this))

The gauge ID is a unique identifier for the gauge.

[Gauge Desc](javascript:TextPopup(this))

Description of gage used for this test step.

[Subgrp Size](javascript:TextPopup(this))

Subgroup size (i.e. number of measurements per subgroup). (The 3
in 3 out of 8 pieces).

[Sample Size](javascript:void(0);)

[Freq Measure](javascript:TextPopup(this))

The type of measure used in sample size. (The pieces in 3 out of
8 pieces, or the minutes in 3 out of 8 minutes).

[Defect Class](javascript:TextPopup(this))

The defect classification, e.g. Critical, Major, Minor, etc.

[Create Date](javascript:TextPopup(this))

The date the test was first defined and saved.

[Last Update](javascript:TextPopup(this))

The date the test was last updated.
