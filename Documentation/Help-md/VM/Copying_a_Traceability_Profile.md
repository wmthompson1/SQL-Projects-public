# Copying a Traceability Profile

You can very quickly copy a traceability profile to
a large number of parts using the Copy Profile to Other Parts command.

If you are licensed to use multiple sites, you can copy a profile
to any other part assigned to the site selected in the profile. You
cannot copy a profile from a part in one site to a part in another
site.

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site that owns the profile. If you are licensed to use a single
   site, this field is unavailable.
2. Call the profile that
   you wish to copy into the window.
3. Select Copy
   Profile to Other Parts from the Edit menu.

The Copy Profile to Parts dialog appears.

The dialog box is similar to the standard
Part ID table. You can select the display of Fabricated, Purchased,
or All parts using the radio buttons. Use the search feature to limit
the number of parts to which you want to copy the profile.

4. If you want to replace
   profiles that might already be defined for parts, check the Replace
   Existing Profiles box.

If you want to copy the profile to all the
parts in the list, click the Copy to Listed Parts
button.

If you want to copy only selected items
in the list, select each part. You can select a noncontinuous list
of parts by holding down the CTRL key as you make your selections.
While you hold down the CTRL key, each click selects or clears that
line only, without affecting the others.

When you have selected the desired parts,
click Copy to Selected Parts.

VISUAL copies the current trace profile
to the parts. If you clicked on Replace Existing Profiles, any previously
defined profile for a selected part is overwritten; otherwise, only
parts without profiles receive the new profile.
