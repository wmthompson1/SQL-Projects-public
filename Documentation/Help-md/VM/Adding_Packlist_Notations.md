# Adding Packlist Notations

Notations (notes, for short) are free form text used
for internal documentation purposes. Although they are associated
with the customer order in much the same way as specs, they are fundamentally
different:

* Notes are exclusively for
  internal use of your business. They are not included on any outgoing
  paperwork.
* Individual note entries
  are automatically date/time stamped, and maintained as separate
  entries. This is different from specs, where you can modify the
  entire block of text at any point as a single object.

All notations in Infor VISUAL function in the same way.

1. With the appropriate
   packlist open in the Customer Shipments window, select Packlist
   Notations from the Notes menu.

The Packlist Notations dialog box appears.

2. Click the New
   button to make a new entry.

A line appears with the current date and
time for the entry.

3. Enter the appropriate
   information.

You can enter an unlimited amount of information.

Use the Paste
button to paste text that you copied from another location into the
box.

Use the Copy button
to copy text in the dialog box to the clipboard.

4. Click the Save
   button to save the Notes.
5. Click the Close
   button to close the Notations dialog box and return to the Customer
   Shipments window.
