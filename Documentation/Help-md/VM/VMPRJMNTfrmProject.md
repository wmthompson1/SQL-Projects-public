# Entering Project Information

To enter basic information:

1. Using the appropriate
   buttons, select the following information:

Site ID - Click the Site ID
arrow and select a site to use for this project. The name of the selected
site is displayed next to the field.

Project
Type - Click the Project Type button
and select the type of project you are entering.

Division
- Click the Division button and select the
division you want to use for this project.

Business
Line - Click the Business Line button
and select the type of business you want to associate with this project.

Customer
- Click the Customer button and select the
name of the customer you want to use for this project.

Contract
Type - Click the Contract Type button
and select the contract type you want to associate with this project.

2. Using the appropriate
   drop-down arrows, select the following information:

Summary
Code - Click the Summary Code drop-down
arrow and select the appropriate summary code for this project.

SIC
- Click the SIC arrow and select the appropriate
SIC for this project.

Status - A status
of Unreleased automatically appears in the Status field.

Default
WBS - Click the Default WBS browse button and select the WBS
Code you want VISUAL to use for this project.

Default
Ref Sub ID - Click the Default Ref Sub ID browse button and
select the WBS Code you want VISUAL to use for this project.

Default
Ref Seq No - Click the Default Ref Seq No browse button and
select the WBS Code you want VISUAL to use for this project.

For more information on using default IDs,
refer to [Enforcing Referential
Integrity](Enforcing_Referential_Integrity.md).

3. Select the appropriate
   check boxes:

Create Project Warehouse - Because your
project must have a warehouse with the exact project name, the Create
Project Ware house appears automatically selected. If you do not want
to create a warehouse for this project at this time, clear the Create
Project Warehouse check box. If you clear the check box, you
must create a warehouse is Warehouse maintenance with the same ID
as the project.

Gov't
Related - If this project is government related, select the
Gov't Related check box.

DCM
Cognizance - If you want this project to be DCM (Defense Contract
Management) cognizant, select the DCM Cog nizance
check box.

4. Click the Save
   icon on the toolbar.
