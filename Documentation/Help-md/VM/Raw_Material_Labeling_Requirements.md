# Raw Material Labeling Requirements

In order for material to flow through a facility,
it should be sufficiently labeled with barcodes. These barcode labels
vary in size shape and available information. There are, however,
standard compliance labels that provide the necessary information.
VISUAL BTS interact with many label types and provide suitable means
to receive materials via any of these various label types and scenarios.
