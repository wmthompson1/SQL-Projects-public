# Starting the Estimating Window

The Estimating Window is available from the VISUAL
Main window.

To access the Estimating Window:

Select Estimating Window from the Sales or Eng/Mfg menu.

The Estimating Window appears with no information in it.

The Estimating Window contains both entry fields and buttons and
is divided into two distinct sections: prospect information and part
definition. The upper section (prospect information) contains a main
Quote ID and general prospect/customer information. The lower section
details the parts to be quoted. It is in the lower section where the
quantities, price breaks, and markup information specific to the Part
ID appears.
