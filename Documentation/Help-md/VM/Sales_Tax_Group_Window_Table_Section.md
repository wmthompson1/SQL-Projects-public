# Sales Tax Group Window Table Section

Depending on your table preferences the following
columns may be available: [Sales
Tax ID](javascript:TextPopup(this)), [Description](javascript:TextPopup(this)),
Percent Set, [G/L
Account ID](javascript:TextPopup(this)), [G/L
Account Description](javascript:TextPopup(this)), [Returns (G/L Account
ID)](javascript:void(0);) and [Returns (G/L Account Description)](javascript:void(0);)

The description of the G/L Account.

The G/L Account ID to which you want Sales Tax dollars posted.
Once the account is entered, the description is automatically
retrieved from the G/L Account table. An entry is not required.
If no account is specified, the default A/R Invoice Sales Tax
account in the G/L interface will be used.

Specify the full description of the item or service.

Assign a Unique ID to be associated with the individual taxes
within a Tax group. A Tax ID can be assigned to multiple tax groups.
If you change the Percent Set in one Tax ID, all of the Tax Groups
that have that Sales Tax ID in it will also be changed. An entry
is required.
