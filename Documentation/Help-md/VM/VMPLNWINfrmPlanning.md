# Starting the Material Planning window

The Material Planning Window is available from the
VISUAL Main window.

To access the Material Planning Window:

Select Material Planning Window from the
Inventory menu.

The Material Planning Window appears with no information in it.

The Material Planning Window is made up of both entry fields and
action buttons.

## Viewing Material Issues in the Material Planning Window

In the non-advanced mode, click the Material
Issues toolbar button. VISUAL opens the Material Issues dialog
box populated with all of the material issues for the current card.

You can print the Material Issues list by selecting the print destination
and clicking the Print toolbar button. If you want to convert the
list to a PDF document and attach it to an e-mail, select the PDF Format check box.

## Viewing Inventory Transactions from the Material Planning Window

From the right-click menu select Inventory Transactions
option. The Inventory Transactions dialog shows the inventory transactions
for the part selected in the material planning window. If you are
licensed to use multiple sites, the Inventory Transactions dialog
shows the transactions for the currently selected site only. To view
inventory transactions that occurred during a particular time period,
specify the date range of the time period in the From Date and Thru
Date field. To increase the From Date value by one day, click Next
Date. To decrease the Thru Date value by one day, click Prev Date.
Use the date fields at the top of the window to window with the appropriate
information from the selected line in the Material Planning window.
