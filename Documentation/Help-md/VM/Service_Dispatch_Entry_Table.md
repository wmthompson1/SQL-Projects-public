# Service Dispatch Entry Table

Depending on your table preferences the following
columns may be available:

* [Ln
  #](javascript:TextPopup(this))

  The system numbers each line consecutively. If lines are
  deleted you can select the Renumber Lines option from the
  Edit menu.
* [Reference](javascript:TextPopup(this))

  Displays the contents of the Misc Ref/Descr column from
  the customer order. If the line is for an inventory part,
  Reference will be labeled Part, followed by the part description.
  If the line is linked to a work order, Reference will be labeled
  job, followed by the work order Base ID/Lot ID, and the Part
  ID, if there is one. If the line is a miscellaneous item,
  Reference will be labeled Misc.
* [Service
  Part ID](javascript:TextPopup(this))

  A unique identifier for the service part.
* [Vendor
  Service ID](javascript:TextPopup(this))

  A Unique identifier for the service that the vendor provides.
* Quantity Ordered
* [Total
  Qty Dispatched](javascript:TextPopup(this))

  Specify the total quantity of the dispatched item. A numeric
  entry is required. The default is zero.
* [Total
  Qty Recv](javascript:TextPopup(this))

  The total quantity received on this receipt. A numeric entry
  is required. The default is zero.
* [Dispatch
  Quantity](javascript:TextPopup(this))

  The quantity that was dispatched.
* [Unit
  of Measure](javascript:TextPopup(this))

  The unit of measure for this line item.
* [Actual
  Freight](javascript:TextPopup(this))

  Specify the appropriate quantity in the Actual Freight column.
* [Last
  Disp Date](javascript:TextPopup(this))

  Specify a date in the Last dispatched date column.
* [Desired
  On](javascript:TextPopup(this))

  Specify the desired date in this column.
* Promise Date
* [Shipping
  Weight](javascript:TextPopup(this))

  This is the shipping weight of one unit. It is intended
  to be used on an acknowledgment or pack list to give you some
  idea of the shipping weight.
* [Links](javascript:TextPopup(this))

  If any of these items are linked, enter the links in this
  column.
