# Viewing Current Customer Order Information

1. Double-click the icon
   in the Current Customer Order window.

All lines for the current customer order
appear.

2. Highlight the order
   line for which you want to view information.

The following columns appears in the right
pane:

Demand ID - The
ID of the supplying source. If you are issuing supply from inventory,
a Warehouse ID appears in this column; if supply is coming from a
purchase order, the Purchase Order ID appears here.

Allocated - The
quantity of supply that you have allocated to the order line to meet
its quantitative requirements for customer order fulfillment.

Received - If
the supplying source of the customer order is an expected purchase
order or purchase order delivery schedule quantity, when you receive
the purchase order or purchase order delivery schedule line into a
warehouse location, VISUAL fills this column with the amount towards
the order line that you received from the purchase order, even though
the purchase order may have been for substantially more.

Issued - If the
supplying source for the customer order a work order, after you finish
the work order and issue the parts into inventory, VISUAL fills this
columns with the amount towards the order line that you have issued,
even though the work order may have been for substantially more.

Amount Ordered
- The dollar amount of the order line.

Amount Shipped
- The dollar amount of what you have shipped from the order line.
