# Changing a Project's Status

To change a projects status:

1. Open the project you
   would like to close in the Project window.
2. Select the header card
   of the project.
3. From the Change Status
   menu, select the status for the project. The Change Status dialog
   box appears.
4. Select the Change
   status of subordinate materials and operations checkbox,
   then click Ok.
5. VISUAL changes the projects
   status.
