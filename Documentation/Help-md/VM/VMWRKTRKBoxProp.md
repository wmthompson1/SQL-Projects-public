# Viewing Step Status Information

To view step status information:

1. Right-click the step
   for which you want to view status information.

The Step Properties dialog box appears.

2. View status information
   in the following fields:

Status Indicates
whether the workflow step is complete or in process.

Started On The
date when VISUAL first evaluated this step. This date does not change.

Last notified The date when VISUAL last
evaluated the step. This date changes each time VISUAL evaluates the
step.

Completed The
date when VISUAL completed the step.
