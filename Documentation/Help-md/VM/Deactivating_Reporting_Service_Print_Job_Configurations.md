# Deactivating Reporting Service Print Job Configurations

To deactivate a reporting service print job configuration:

1. Select Admin,
   Application Global Maintenance.
2. Select Maintain,
   Report Service Configurations.
3. Clear the Enabled
   check box.
4. Click Save.

If you are deactivating more than one print job, you must click
save after each time you clear the Enabled check box.
