# Allowing Excess Supply on Work Orders

During processing, the DBR scheduler adjusts work orders that have
excess supply if you have given the scheduler permission to do so.
You can prompt the DBR scheduler to ignore selected work orders during
this process, even if the work orders have excess supply. The work
orders you designate will not be adjusted by the scheduler.

To allow excess supply on work orders:

1. Select Eng/Mfg,
   Manufacturing Window.
2. Open the work order.
3. On the header card,
   select Allow Excess Supply.
4. Click the Save
   button.
