# A/R Cash Receipts

To review the cash receipts for the period you would
like to close, run the Cash Receipts report.

1. From
   the Receivables menu, select Cash Application.
   The Cash Application window appears.
2. From
   the File menu, select Print Cash Receipts.
   The Print Cash Receipts window appears.
3. In
   the Starting Date field, enter the first day of the period you
   would like to close. In the Ending Date field, enter the last
   day of the period you would like to close.
4. In
   the Sort by section, select Payment ID
   to view the report by payment ID, or Deposit
   ID to view the report by Deposit ID.
5. The
   Print Active Payments check box is selected
   by default. If you select Print Void Payments,
   you can de-select the Print Active Payments
   check box. Select the Summary check
   box if you would like to view a summary report; we recommend leaving
   the check box blank to view a detailed report.
6. Click
   Ok. The report outputs.
