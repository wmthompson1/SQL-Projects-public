# Sorting the Contact table in the Synchronize Contact Utility

You can modify the Contact table by filtering
columns, moving columns and prioritizing a column sort based on contact
information.

To sort the Contact table:

1. Navigate
   to the Infor VISUAL installation directory and double-click VMCRMSNC.EXE.
   If the default VISUAL installation directory is used, launch the
   Synchronize Customer/Account Utility from C:\Infor\VISUAL\VISUAL
   MFG.
2. Select
   Maintain, Sync Contacts.
3. Select **Options,
   Preferences**.
4. Select the contact information
   identifier on which to set the sort priority, and click a priority
   number in the sort selection area, or click Clear
   Selection to reset a sort priority. You can set sort priorities
   on up to eight contact identifiers.
5. Select
   the check box in the Descending
   column to perform a descending sort
   from Z to A or from the largest number to the smallest number.
   Clear the check box to perform an ascending sort from A to Z or
   from the smallest number to the largest number.
6. Click
   Ok to confirm
   your sort priorities.
