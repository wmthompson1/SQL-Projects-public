# Activating the Data Source

To activate the data source:

1. Select File,
   Dashboard.
2. Select Edit,
   Edit Data Source.
3. In the Data Source ID
   field, select the data source.
4. Click the Status drop-down
   button and select Active.
5. Click Save.
