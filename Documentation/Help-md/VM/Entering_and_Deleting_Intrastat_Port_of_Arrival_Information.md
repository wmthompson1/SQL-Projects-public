# Entering and Deleting Intrastat Port of Arrival Information

The port of arrival is the place in which goods are
received.

To add new port of arrival codes:

1. Click the Port
   of Arrival button.

The Port of Arrival dialog box appears.

2. Click the Insert
   button.

A new line appears in the line item table.

3. In the Code column,
   enter a code for the port of arrival.
4. In the Description column,
   enter a description for the new port of arrival code.
5. Click the Save
   button to save the new information.

## Deleting Port of Arrival Codes

1. Click the Port
   of Arrival button.

The Port of Arrival dialog box appears.

2. Select the port of arrival
   code you want to delete, then click the Delete
   button.

An x appears in the row header beside the
code you are deleting.

3. Click the Save
   button to save the new information.
