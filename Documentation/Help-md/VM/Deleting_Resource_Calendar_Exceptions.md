# Deleting Resource Calendar Exceptions

You can delete existing resource calendar exceptions.

To delete an exception:

1. Highlight the exception
   line and click the Delete button.

VISUAL places an X to the left of the exception
line that you just deleted.

2. Click the Save
   button to complete the deletion. Click the Close
   button to close the dialog box without deleting the resource.
