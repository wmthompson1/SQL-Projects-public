# Printing the Quote Log

The Quote Log gives you the ability to analyze quotes
that have been entered into the database.

If you are licensed to use multiple sites, you can
view information for multiple sites in the same proforma report, provided
that the sites share the same parent accounting entity.

1. If you are licensed
   to use multiple sites, click the Site ID
   arrow and select a site to view in the report. When you access
   the dialog box, you can choose to include in the report any site
   with the same parent accounting entity. If you are licensed to
   use a single site, this field is unavailable.
2. Select Print
   Quote Log from the File menu.
3. If you are licensed
   to use multiple sites, click the Site ID(s)
   arrow and select the sites to include in the report. You can include
   any sites with the same parent entity as the site selected in
   the Estimating Window. If you are licensed to use a single site,
   this field is unavailable.
4. Enter the starting date
   and ending date for the period desired.
5. Choose how you want
   the report sequenced.

Use Quote ID or Quote Date to have quotes
sequenced in the general order they were entered.

Use Expected Date to place the quotes in
the sequence in which the order is expected. This is useful for directing
attention to quotes with upcoming decisions being made.

Use Follow-up Date to sequence quotes by
the follow up date that was previously set. If you maintain this date
whenever you call up a quote, youll be able to generate a list of
quotes in the order in which you should follow them up.

Use Expiration Date if you wish to focus
on those quotes that are coming near their expiration date.

You can also set the primary sort to be
by Customer ID, by checking the Sort by Customer Box. This will output
quotes in Customer ID order, with all of a customers quotes sequenced
as specified in Sequence By.

5. Select which status
   types you want to include. You can choose as many as necessary.
6. Choose the output destination
   by selecting the arrow next to the output field. Select one of
   the following:

Print - To output
the file to a printer, select this option. VISUAL sends the file to
the selected default printer.

View - To see
what the printed output will look like before it is printed, select
View. When the information appears on the screen, you can scroll through
it and print it if you need a hard copy.

File - To save
the information to a text file, select File from the pull down menu.

7. If you selected Print
   as the output type, select Ok to output
   the information to your printer.
8. Click Ok.

A standard Windows printer dialog box appears,
which allows you to select the number of copies to be printed or viewed.

8. Make the desired selections
   and click Ok.

VISUAL prints the report.
