# Showing Backlog Line Items Only

Select Show Backlog Only
from the Options menu to view only backlogged line items. When this
option is on, only the items from the order that have not been fully
shipped appear in the line item table. A check mark indicates that
Show Backlog Only is on.

When this option is NOT on, the system displays all line items belonging
to the order, whether they have been shipped or not.
