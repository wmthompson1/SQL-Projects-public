# Using Drag n' Drop to Order Cross-Selling Parts

You can add an order line for a cross-selling part
to the current customer order by dragging the part into the Current
Customer Order window.

1. In the VISUAL Order
   Management Customer Order Entry window, highlight the order line
   that contains the primary part. You are ordering cross-selling
   parts for this part.
2. If not already open,
   the Order Management Window; then open the Current Customer Order
   Window and the Cross-Selling Parts Window. Tile or cascade the
   two so you can easily perform a drag n drop operation.
3. In the Cross-selling
   Part Window, double-click the icon.

If there are any, cross -selling parts for
the primary part appear.

4. Highlight the cross-selling
   part that you want to add to the current customer order and hold
   down the left mouse button until the drag n drop box appears.
5. Drag the box into the
   open Current Customer Order Window and release it on top of the
   displaying current customer order.

The VISUAL Order Management Customer Order
Entry Window appears with a line added to the current customer order
for the cross-selling part.

6. Enter a quantity for
   the line and other necessary information and then click the Save button to commit the order.
