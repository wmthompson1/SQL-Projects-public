# Modifying Customer Service Impact Graphs

Use the options in the header portion of the window.
Refer to the following list for more information.

Base Date - You can change the base date
VISUAL uses in the compiling of customer service impact information
by manually editing this field or clicking the calendar icon to select
a date. After selecting a date, you must click the Refresh button
or press F5 for the change to take effect.

Use Selected Customers - Select this check
box to enable the selecting of specific customers in the specified
schedules

Edit List - Click this button to select
specific customers for whom you want to view customer service impact
graphic information in the current schedules. You must select the
Use Selected Customers check box for these settings to have any effect.
After selecting customer to include, you must click the Refresh button
or press F5 for the change to take effect.

Days, Dollars, Dollar/Days, Opportunity Lost
- Select any of these buttons to change the graphic display. See the
definition of each earlier in this section. You can select only one
at a time. After selecting a button, you must click the Refresh button
or press F5 for the change to take effect.

Monthly, Weekly, Daily - Select any of
these buttons to change the time frame of the customer service impact
graph. For example, selecting Monthly, produces a graph of the selected
schedule by month. After selecting a button, you must click the Refresh
button or press F5 for the change to take effect.

Schedule IDs - Select from a list of valid,
active schedules. You can select more than one schedule at a time.
Doing so allows you to compare schedules against one another. After
selecting a schedule or multiple schedules, you must click the Refresh
button or press F5 for the change to take effect.

Graph Types - Select from Area, Area Step,
Bar, Bar Horizontal, Doughnut, Line, Line Step, Pie, Spline, and Spline
Area. After selecting a graph type, you must click the Refresh button
or press the F5 key before the display changes.

Toggle Button - Click this button to view
graph information tabularly.
