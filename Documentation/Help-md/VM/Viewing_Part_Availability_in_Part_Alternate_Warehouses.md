# Viewing Part Availability in Part Alternate Warehouses

Double-click the icon and the plus boxes as appropriate
to view alternate warehouse part availability information for the
part in the current customer order.

The procedures for viewing part availability information in this
window are the same as the procedures described previously in the
section "Using the Order Management Window Part Availability
Window."

See that previous section for more detailed information on viewing
part quantities and using drag n drop and right-click allocations.
