# Changing Ship from Addresses

To change Ship From addresses:

1. Click the Ship
   from Addr button.

The Ship from Addresses dialog for the vendor
appears.

The table at the top of the dialog lists
the alternate addresses, if any, for the vendor.

2. To use an alternate
   address for this dispatch, select the line of the appropriate
   address and click the Use Selected Address
   button.

The dialog automatically closes. The selected
address now appears in the Purchase From window at the top of the
Service Dispatch Entry window.

## Editing Ship From Addresses

You can also use the dialog box to add, modify, and
delete alternate addresses.

## Adding Ship From Addresses

1. Click the Insert
   button.

A new line appears in the table.

2. Fill in the address
   information in the address fields.
3. Click the Save
   button to save the new address.

## Deleting Ship From Addresses

1. Select the address to
   delete and click the Delete button.
2. Click the Save
   button to permanently delete the address.

## Modifying Ship From Addresses

1. Select the address to
   modify.
2. Modify the fields that
   appear at the bottom of the dialog, as necessary.
3. Click the Save
   button to save the modifications.
