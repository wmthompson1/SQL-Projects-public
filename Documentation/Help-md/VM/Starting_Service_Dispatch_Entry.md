# Starting Service Dispatch Entry

Service Dispatch Entry is available from the VISUAL
Main window.

To start Outside Service Dispatch Entry, select Outside
Service Dispatch Entry from the Purchasing menu.

The Service Dispatch Entry window appears.

The Service Dispatch Entry window contains fields and buttons for
entering dispatch information.
