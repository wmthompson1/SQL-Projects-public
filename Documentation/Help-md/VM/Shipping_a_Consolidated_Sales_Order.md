# Shipping a Consolidated Sales Order

When you ship a consolidated sales order, you can
ship sales order lines that match the site ID you specify on the Shipping
Entry header. Consider this example:

Sales Order Header Site ID: MMC

Sales Order Line 1 Site ID: MMC

Sales Order Line 2 Site ID: MMC-NH

If you select MMC on the Shipping Entry header, you can only ship
items from line 1 of the sales order. If you have not yet received
the items for site MMC-NH, a message is displayed informing you that
the customer order will remain open.

If you select MMC-NH on the Shipping Entry header, you can only
ship items from line 2 of the sales order.
