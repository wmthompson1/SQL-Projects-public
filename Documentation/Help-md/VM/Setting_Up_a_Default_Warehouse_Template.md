# Setting Up a Default Warehouse Template

After you create the warehouse you would like to use
as the default, designate the warehouse as the default through the
Defaults tab of VISUALs Application Global Maintenance window. When
you create a new project and a new project warehouse, VISUAL will
copy the default warehouses characteristics.

To set a default warehouse to copy:

1. From the main window,
   select Application Global Maintenance
   from the Admin menu.

2. Click the Defaults
   tab.
3. In the Projects section
   of the Defaults tab, click the Default Warehouse
   to Copy button and select the appropriate warehouse.
4. Click the Save
   button on the toolbar.
