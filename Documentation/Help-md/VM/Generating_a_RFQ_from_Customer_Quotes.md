# Generating a RFQ from Customer Quotes

You can generate an RFQ from a customer quote. You
must select a site ID to use this feature, but do not load an RFQ
in the window. If an RFQ is selected, the dialog box does not appear.

1. If you are licensed
   to use multiple sites, click the Site ID
   arrow and select the site where you created the customer quote.
   If you are licensed to use a single site, this field is unavailable.
2. Select Generate
   from Customer Quote from the File menu.
3. Enter the Customer Quote
   ID.

If the customer does not exist in the system,
the system prompts you with a dialog box.

3. Click Ok
   to generate the vendor request for quote.
