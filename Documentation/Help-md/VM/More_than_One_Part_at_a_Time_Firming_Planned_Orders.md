# More than One Part at a Time Firming Planned Orders

Planned order firming supports multiple parts. You
may specify either a product code or a range of part numbers to cause
the system to select parts and their planned orders eligible for firming.
This results in a display similar to the table shown above, except,
in addition to the columns used for one part at a time, VISUAL also
shows the Part ID. The Vendor, and Type columns appear if any of the
selected parts require them.
