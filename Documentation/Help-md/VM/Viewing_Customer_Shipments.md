# Viewing Customer Shipments

You can view shipments and invoice information for
your customers using Customer Inquiry.

To view customer shipments:

1. Call up the order for
   which you want to check shipments into the Customer Inquiry information
   window.
2. Select List
   Shipments/Invoices from the Info
   menu.

The Shipments dialog box appears, listing
all of the shipments for the current order. You can view the ship
date, the quantity shipped, reference information, the invoice ID,
and the invoice date.

3. Click the Close
   button when you are finished.
