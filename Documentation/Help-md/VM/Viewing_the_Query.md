# Viewing the Query

After you specify the tables, columns, and filters for your data
source, use the Query tab to review the SQL statement that was constructed.
You cannot edit the statement directly on the Query tab. To edit the
statement, change your selections on the Tables, Columns, and Filter
tabs.

To review the query:

1. Select File,
   Dashboard.
2. Select Edit,
   Edit Data Source.
3. In the Data Source ID
   field, select the data source.
4. Click the Query tab.
