# Deleting Intrastat Information Lines

To delete an Intrastat information line:

1. Click the row header
   of the line you want to delete.
2. Click the Delete
   Row icon on the toolbar.

An X appears in the row header indicating
you have marked the row for deletion.

3. Click the Save
   icon on the toolbar.

VISUAL removes the line from your database.
