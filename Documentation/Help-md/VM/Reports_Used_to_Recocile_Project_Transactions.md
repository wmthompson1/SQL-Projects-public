# Reports Used to Reconcile Project Transactions

Use these reports to reconcile project transactions.
The Project Billing Events Report is also used.

## EAC/POC Report

To run the report:

1. From the Project menu,
   select Project Summary. The Project
   Summary window appears.
2. From the File menu,
   select Print EAC/POC Report. The Print EAC/POC Report window appears.
3. To view all projects,
   leave the Project ID fields blank.
4. In the Filter box, leave
   the Project Type field blank to view all project types in the
   report.
5. In the Summarize up
   to field, enter the last day of the period you are closing.
6. Click Ok.
   The EAC/POC report appears.

If the actual amounts do not match the formal
amounts, you may want to update your EAC for the project.

Print the report to use as a guide.

## Printing Project WIP Balance Reports

To print WIP balance reports for your non-commercial
projects:

1. From the Ledger menu,
   select Post Manufacturing Journals.
   The Post Manufacturing Journals window appears.
2. From the Reports menu,
   select Project WIP Balances. The WIP
   Balance Inquiry for Projects window appears.
3. In the End Date of Accounting
   Period field, enter any date within the accounting period you
   would like to view, or click the Calendar button to select a date.
   VISUAL adjusts the date to the last date of the period and determines
   the Fiscal Period.
4. In the Report Currency
   field, enter a currency by clicking the arrow and selecting a
   currency. You can choose from the entity currency or any tracking
   currency.
5. In the Status box, select
   the Released Only check box to view
   information for projects that have been released. Clear the check
   box to view all projects.
6. Select a sequence in
   the Sort box. The fields in the window change depending upon the
   sort order you choose. VISUAL sorts the inquiry or report by the
   sequence you choose. Your choices are:

* G/L Account
  ID
* Project
  ID
* Project
  Type

7. If you selected G/L
   Account ID in the Sort box, enter information in the following
   fields:

Starting Account ID and
Ending Account ID - Click the Browse button to select a starting
account ID and an ending account ID, or enter account IDs in the Account
ID fields. The report displays information for all the accounts that
fall within the range you specify. To print all accounts, leave these
fields blank.

If you selected Project ID in the Sort box,
enter information in the following fields:

Starting Project ID and
Ending Project ID - Click the Browse button to select a starting
project ID and an Ending project ID, or enter project IDs in the project
ID fields. The report displays information for all the accounts that
fall within the range you specify. To print all projects, leave these
fields blank.

If you selected Project Type in the Sort
box, enter information in the following fields:

Starting Project Type
ID and Ending Project Type ID - Click the Browse button to
select a starting project type ID and an ending project type ID, or
enter project type IDs in the Project Type ID fields. The report plays
information for all the project types that fall within the range you
specify. To print all project types, leave these fields blank.

8. In the Mode box, select
   Inquiry Only to view the results of
   your query in the table. Select Report Only
   to output the results of your query as a report.
9. If you select Inquiry Only in step 8, the results of the
   query appear in the table when you click Ok.. If a plus sign appears
   in the Sum column, you can double click the row header to view
   more detailed information. You can continue to drill down to individual
   transactions.
10. If you select Report Only in step 8, select the output for
    the report. Click the arrow and choose one of the following options
    from the menu:

Print - Select
this option to output the file to a printer.

View - Select
this option to see what the printed output looks like before you print
it.

File - Select
this option to save the WIP Balances Report to a text file. You can
edit the report outside of VISUAL using any text editor.

E-mail - Select
this option to send the report as an .RTF (rich text file) attachment
through your electronic mailing system.

11. Select the type of report
    you would like to view. You can view a summary report, summary
    with sub-totals report, or a detailed report with or without burden
    detail.

Summary Report
- Select the Summary box to output a summary report. The summary report
by G/L account displays the G/L account ID, the account description,
the total debit amount, and the total credit amount. The summary reports
by project or project type shows the project ID or project type ID,
project or project type description, status, type, material cost,
labor cost, burden cost, service cost, and total value.

Summary with Sub-Totals
Report - Select the Summary w/Sub-Totals check box to output
a Summary with Sub-Totals Report. The summary with sub-totals reports
show the same information as the summary reports, but the information
in broken down by transaction.

Detailed Report
- Clear the Summary box to view a detailed report. Detailed reports
show all of the transactions by type flowing in and out of the G/L
accounts. The information includes transaction ID, transaction date,
project ID, and project summary ID along with the debit and credit
amount for each line item. Detailed reports by project show all the
transactions by type for each project. Transactions include purchased
materials, WIP issues, WIP labor, vouchers, and inventory adjustments.
Detailed reports by project type show the same information as detailed
reports by project ID, only the transactions are shown by project
type.

Detailed Report with
Burden Detail - You can only print detailed reports with burden
detail for project IDs and project types. The reports show the same
information as the detailed reports, but burden costs are broken down
by burden category ID.

12. Click the OK
    button to output the information.

## Post Manufacturing Journal PO Accrual Report

To print PO Accrual reports:

1. From the Ledger menu,
   select Post Manufacturing Journals.
   The Post Manufacturing Journals window appears.
2. From the Reports menu,
   select Print PO Accrual. The Print PO
   Accrual window appears.
3. In the Print Select
   box, enter a date that falls within the period you would like
   to close in the End Date of Accounting Period field. VISUAL adjusts
   the date to the last day in the period and populates the Fiscal
   Period field.
4. Select the Summary
   check box to view a summary report. Leave the Summary check box
   blank to view a detailed report.
5. Click Ok.
   The report outputs.
