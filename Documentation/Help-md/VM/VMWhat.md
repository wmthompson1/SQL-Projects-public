# What Is Security Maintenance

Use VISUAL Security modules to add users, specify password requirements,
and manage user access to VISUAL executables.

After your database management system is installed, there is at
least one default system management account that you can use to sign
into the database and add other users. This account has full privileges,
and is used as the starting point to set up other users and their
privileges.
