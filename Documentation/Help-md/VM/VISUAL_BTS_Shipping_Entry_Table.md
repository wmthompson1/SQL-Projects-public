# VISUAL BTS Shipping Entry Table

Both the Pending Transaction and Completed Transaction
portions of the window contain the following columns:

ID - The ID of the transaction.

Trans Type - The type of the transaction.

Trans Date - The date on which you scanned
in the purchase order.

Customer Order ID - The ID of the customer
order that you are shipping. See Shipping Entry for more information.

Line Number - The number of the line that
you are shipping as it appears in the customer order.

Quantity - The quantity of the customer
order line.

From Warehouse - The ID of the warehouse
from which you are shipping the goods.

From Location - The ID of the location
in the warehouse from which you are shipping the goods.

User ID - The ID of the user who is currently
scanning.
