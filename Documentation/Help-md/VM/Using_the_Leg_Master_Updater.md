# Using the Leg/Master Updater

If you have created a leg in a master from another master, you can
use the Leg Updater to update the leg if you change the original master.
Similarly, you can apply changes you make to a leg to the original
master.

If you are licensed to use multiple sites, legs and masters are
updated within the same site only. You can not use this function to
update a master or leg in one site from a master or leg in a second
site.
