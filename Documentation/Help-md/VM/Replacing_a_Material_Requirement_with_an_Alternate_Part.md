# Replacing a Material Requirement with an Alternate Part

You can use the Alternate Parts dialog box to replace a material
requirement specified on a work order with an alternate part.

Define alternate parts for work orders in [Part
Maintenance](VMPRTMNTfrmPart.htm).

If you are licensed to use multiple sites, the alternate part and
the part it replaces must both exist in the selected site.

To replace a material requirement with an alternate part:

1. Select Inventory,
   Material Planning Window.
2. Open a part.
3. Select the line that
   contains the part you want to replace.
4. Select Info,
   Alternate Parts.
5. Click the alternate
   part you want to use, and drag it to the Material Planning Window.
6. The system asks you
   to confirm that you want to replace the part. Click Yes
   to continue.

If the part is a piece-tracked part, the system replaces the original
part ID and usage UM on the material card with the part ID and the
usage UM of the alternate part.
