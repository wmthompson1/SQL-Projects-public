# Restoring Lifecycle Card Colors to the Original Settings

To restore the cards to the original colors used when
VISUAL was delivered:

1. In an application that
   supports the document lifecycle, select Info,
   Document Lifecycle
2. In the Graphical Display,
   select Options, Color
   Preference.
3. Click Restore
   Defaults. If you accessed the Color Preferences dialog
   from a purchasing module, then the purchasing transaction cards
   are reset. If you accessed the Color Preferences dialog from a
   sales module, then the sales transaction cards are reset.
