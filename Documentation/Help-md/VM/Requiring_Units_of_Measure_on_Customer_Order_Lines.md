# Requiring Units of Measure on Customer Order Lines

You can set up Preferences Maintenance to require users to specify
a unit of measure on the order line.

Usually, unit of measure is populated from the part record or from
the customer pricing table. Use this preference to prevent users from
accidentally deleting the unit of measure.

1. Select Admin,
   Preferences Maintenance.
2. Specify this information:

Section To set
up the preference in Customer Order Entry, specify OrderEntry. To
set up the preference in the Order Management Window, specify OrderEntryWindow.

Entry Specify
ReqSalesUnitOfMeasure.

Value Specify
Y to require units of measure on customer order lines.

3. Click Save.
