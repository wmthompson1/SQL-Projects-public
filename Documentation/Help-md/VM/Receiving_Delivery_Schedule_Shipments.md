# Receiving Delivery Schedule Shipments

You can receive shipments using the Delivery Schedule
Shipment dialog box. You must use the Delivery Schedule Shipment dialog
box with purchase order lines that contain delivery schedules.

1. Select the purchase
   order line from the Purchase Receipt Entry line item table
2. Select the Del
   Sched button.

The Delivery Schedule Shipment dialog box
appears.

3. Select the delivery
   schedule line you want to use.
4. If you do not want to
   receive the whole quantity, enter the quantity that you want to
   receive in the Recv Quantity column.
5. Click the Ok
   button to receive the shipment.
