# Setting Global Values

Use Quality Maintenance to set global variables such
as the minimum subgroup size (for XBar/Range tests) and the smallest
allowable subgroup before VISUAL can use standard deviation instead
of range.

1. In the Minimum Subgroup
   Size field, enter the minimum number of pieces for a subgroup.
2. In the Std Dev Min field,
   enter the smallest allowable subgroup necessary for using standard
   deviation instead of range.

The minimum subgroup size must be at least
1.

3. Enter the percent of
   specification you want the new control limits to default to.

The number must be between 1 and 100.

VISUAL creates control limits using the
data that you entered for a test step.

4. Click the Save
   button to save the new information.
