# Using Live Updates to Update the Reporting Database Tables

To update the reporting database tables anytime a record affecting
the metrics is changed:

1. Select Admin,
   Application Global Maintenance.
2. Select Maintain,
   Visual Reporting Data.
3. In the Update Type section,
   click Live.
4. Click Save.
