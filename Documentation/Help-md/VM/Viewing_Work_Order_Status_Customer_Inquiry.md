# Viewing Work Order Status

You can view work order status information for your
customer orders using Customer Inquiry.

1. Call up the order for
   which you want to check work order status into the Customer Inquiry
   information window.
2. Select Show
   Work Order Status from the Info
   menu.

A dialog box appears, showing all of the
work orders for that customer order.

3. View the work order
   status information.
4. Select a work order
   to view, then click the View Operations
   button.

A dialog box appears, showing all of the
operations for the work order.

5. When you have finished
   viewing the operation information, click the Close
   button.

You return to the Work Orders dialog box.

6. Click the Close
   button.

You return to the information window for
that order.

7. Click the Close
   button to return to the Customer Inquiry window.
