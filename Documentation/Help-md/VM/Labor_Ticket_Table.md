# Labor Ticket Table

The labor ticket table displays a list of the labor tickets associated
with the project. To view the labor ticket table, select Labor
Tickets... from the Info menu. The Labor Tickets table appears.

To output the report, select the View
arrow to select the output method, then click the Print button.
