# Purchase Orders - Printing Formats

The format of Purchase Order Reports can be modified
using Centura Report Builder. The following QRP report files apply:

## Print Acknowledgment

|  |  |
| --- | --- |
| Standard | VMPURFRM.QRP |
| User Defined 1 | VMPURUD1.QRP |
| User Defined 2 | VMPURUD2.QRP |
| User Defined 3 | VMPURUD3.QRP |

## Print Purchase Order List

|  |  |
| --- | --- |
| Summary | VMPURRP4.QRP |
| Detail | VMPURRP5.QRP |

## Print Open Purchase Orders

|  |  |
| --- | --- |
| By Order ID | VMPURRP1.QRP |
| By Vendor ID | VMPURRP2.QRP |
| By Part ID | VMPURRP3.QRP |
| By Service ID | VMPURRP6.QRP |
