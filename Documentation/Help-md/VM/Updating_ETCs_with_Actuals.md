# Updating ETCs with Actuals

To update ETCs with your actual figures:

1. From the Project menu
   of the Infor VISUALs main screen, select Update
   ETC with Actuals.
2. In the Start Period
   field, enter any date in the period you are closing. VISUAL populates
   the field with the first day of the period. In the End Period
   field, enter any date in the period you are closing. VISUAL populates
   the field with the end date of the period.
3. Click the Run
   button. The Update ETC window appears.

All ETCs are updated with current cost information.
