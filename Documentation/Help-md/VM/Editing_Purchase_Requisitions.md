# Editing Purchase Requisitions

Because requisitions are precursors to purchase orders,
you are free to modify requisition header and line information until
the status changes to Ordered, by which point the requisition has
been used successfully to generate a purchase order. After VISUAL
changes a requisition status to Ordered, the requisition is closed
to further editing.

To edit a purchase requisition at any time before the status reaches
Ordered, choose the appropriate Requisition ID and use the procedures
in the previous sections to modify the header or lines as necessary.
