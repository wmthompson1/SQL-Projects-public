# Indirect Labor Journals

The Indirect Labor journal contains a listing of the
summary transactions created by the Costing Utilities as a result
of processing labor tickets which are not for a specific work order.

Indirect labor tickets are posted in a transactional manner similar
to the adjustment journal.
