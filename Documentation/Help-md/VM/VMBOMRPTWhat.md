# What is the Bill of Material Report?

This report provides an indented bill of materials
for a range of engineering masters or work orders.

For each master or work order, the following data is reported for
each material requirement:

* BOM Level (0 is top level
  of part, 1 is main leg requirement, 2 is leg or fabricated part
  required by main leg, etc.)
* Part ID
* Quantity Required per Unit
* Fixed Quantity Required
* Scrap %
* Dimensional Information

If any parts included in the report are piece tracked, the number
of pieces, the length, width, and height dimensions, and the dimension
unit of measure appear.
