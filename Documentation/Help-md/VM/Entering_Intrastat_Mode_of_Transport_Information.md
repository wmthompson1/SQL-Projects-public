# Entering Intrastat Mode of Transport Information

The mode of transport is the method by which the goods
will be shipped.

1. Click the Mode
   of Transport button.

The Mode of Transport dialog box appears.

2. Click the Insert
   button.

A new line appears in the line item table.

3. In the Code column,
   enter a numeric code for the new mode of transport.
4. In the Description column,
   enter a description for the new mode of transport code.
5. Click the Save
   button to save the new information.

## Deleting Transport Codes

1. Click the Mode
   of Transport button.

The Mode of Transport dialog box appears.

2. Select the mode of transport
   code you want to delete, and then click the Delete
   button.

An X appears in the row header beside the
code you are deleting.

3. Click the Save
   button to save the new information.
