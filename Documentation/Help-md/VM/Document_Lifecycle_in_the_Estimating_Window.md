# Document Lifecycle in the Estimating Window

The Lifecycle Document Viewer shows you documents
linked to the sales or purchasing document that you are currently
viewing. You can drill down to view all linked documents.

You can access the Sales lifecycle from the estimating window. These
documents are included in the Sales lifecycle:

* Estimates
* Customer Orders
* Work Orders linked to Customer
  Orders
* RMAs
* Shipments
* A/R Invoices
* A/R Payments

To access the Lifecycle Document Viewer, select Info,
Document Lifecycle.

See [Viewing Document Lifecycles](Viewing_Document_Lifecycles.htm).
