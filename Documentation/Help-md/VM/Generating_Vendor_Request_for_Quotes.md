# Generating Vendor Request for Quotes

1. If you are licensed
   to use multiple sites, click the Site ID
   arrow and select the site that contains the quote. If you are
   licensed to use a single site, this field is unavailable.
2. Click the Quote ID browse
   button and select the quote.
3. Select Generate
   Vendor RFQ from the File menu.

The Generate from Customer Quote dialog
box appears.

4. Click Ok.

VISUAL generates the Vendor RFQ from the
customer quote.

Generation of an RFQ depends on the following
during estimating:

* Existence of one or more
  quote masters having one or more purchased material requirements.
* The material requirements
  must have a vendor ID directly specified, or the part required
  must have one or more vendors, which supply the part. The former
  is setup in the Manufacturing Window and/or Quick Quote. The latter
  is setup during part definition.
