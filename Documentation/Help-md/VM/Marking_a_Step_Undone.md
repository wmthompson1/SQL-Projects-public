# Marking a Step Undone

With appropriate permissions, you can revert the state
of an authorization to an undone condition.

To mark a step undone:

1. Right-click the rule
   that you want to authorize.
2. In the menu, select
   Mark Undone.

The workflow reverts to an undone condition.
