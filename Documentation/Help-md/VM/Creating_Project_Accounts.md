# Creating Project Accounts

To record the financial involvement of a project, you must set up
project accounts that you can specify for your projects accounting
purposes.

To create a project account:

During your account set up phase, select the Project Account check
box.

You should treat the Fiscal Year column as the year in which the
burden takes place.

For more information on Creating Accounts, refer to your VISUAL
Financials documentation.
