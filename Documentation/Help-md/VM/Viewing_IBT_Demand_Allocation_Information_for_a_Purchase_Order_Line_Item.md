# Viewing IBT Demand Allocation Information for a Purchase Order Line Item

After you have either fully or partially allocated
demand to a IBT using the preceding procedures, you can view allocation
information on the purchase order line.

For each purchase order line item from which you have assigned demand,
the following appears:

* In the Base
  ID column, if there is only one IBT link, the IBT ID appears.
* In the Base
  ID column, if there are multiple IBT links, Multiple
  Links appears.
* In the <Seq
  #> column, if there is only one IBT link, the value
  1 appears.
* In the <Seq
  #> column, if there is more than one IBT link, nothing
  appears.
* In the Link
  Part ID column, if there is only one IBT link, the Part
  ID of the purchase order line appears.
* In the Link
  Part ID column, if there are multiple IBT links, nothing
  appears.
* In the Link
  Warehouse ID column, if there is only one IBT link, the
  Warehouse ID of the purchase order appears.
* In the Link
  Warehouse ID column, if there are multiple IBT links, nothing
  appears.
* In the Link
  Quantity Unallocated column, if there is only one IBT link,
  the quantity of the IBT to which you have yet to allocate supply
  appears.
* In the Link
  Quantity Unallocated column, if there are multiple IBT
  links, nothing appears.
* In the Total
  Qty Allocated column, the quantity thus far of the purchase
  order line that you have allocated to sources of demand appears.
