# Entering Information on Common Tabs

There are several dialog boxes that use identical
tabs. Enter information on these tabs as you enter information in
the dia log box.

## Entering Information on Quantities Tabs

If you are entering information for a manufactured part, use the
Quanti ties tab to enter additional information about the part.

1. Click the Quantity
   tab and enter the following:

Quantity Per
Click in the Qty Per field and enter the quantity of this part per
unit that you want.

To select at what point you want the Quantity
Per to apply, select either the Qty per (start qty) or Qty per (start
qty) option buttons.

Fixed Quantity
If you do not want VISUAL to calculate start or fin ish quantities
based on Quantity Per and Scrap values, click in the Fixed Qty field
and enter the value you want to use.

Scrap % Click
in the Scrap % field and enter the percentage value you want to use
for the amount of scrap you want VISUAL to use in its calculations.

Dimensions If
you are using dimensional inventory, click in the Dimensions field
and enter the formula for the dimensions of this part.

2. Click in the Length,
   Width, and Height fields and enter the appropriate values.

Reference #
Click in the Reference # field and enter a reference num ber.

Issued Quantity
Click in the Issued Qty field and enter the quantity you want to
issue to this leg.

Usage Unit of Measure
Click the Usage U/M button and select
a unit of measure for this part.

3. Click the Specifications
   tab.

The Specifications tab appears.

4. Click in the Specifications
   field and enter any comments or notes you want about this leg.

Enter any appropriate engineering information
for the following:

Drawing Number
Click in the Drawing # field and enter a drawing number for this
leg.

Rev If the drawing
number you entered has a specific revision number, enter it in the
Rev # field.

Drawing File
If the drawing number you entered has a picture file, navigate to
and select the appropriate file.

## Entering Information on Vendor Tabs

Use the Vendor tabs to override any specific vendor information
for the part you have described on this leg.

You can override the following vendor information:

Vendor ID Click the Vendor
ID button and select the vendor you want to use for the part
described on this leg.

Vendor Part ID Click the Vendor
Part ID button and select the part you want to use. The manufacturers
part name and ID automatically appear.

## Specifying Drawing Numbers and Files

If you have a drawing (and a picture of that drawing) you want to
spec ify for an object in the Project window, you can enter the drawing
num ber, revision, and file name in the appropriate information card.

To specify drawing information:

1. Open the edit dialog
   box for the item for which you want to spec ify a drawing.

For example, if you want to specify a drawing
for a task, open the edit dialog box for that task by double-clicking
the task or select ing Edit from the Edit menu.

2. Select one of the following:
3. For Tasks, click the
   Other tab.

For Legs, click the Specifications
tab.

For Headers, click the General
tab.

The appropriate tab appears.

4. Click in the Drawing
   # field and enter the drawing number.

Click in the Rev field and enter the drawings
revision number.

Click the File
button.

The Navigation dialog box appears.

5. If you know the location
   of the file you want to specify, you can click in the Drawing
   # field and enter the appropriate path.
6. Navigate to and select
   the file you want. Click the Save button.

The Navigation dialog box closes.

When you have finished specifying your drawing
information, click the Save button on the
toolbar.

## Using User Defined Field Labels

To help you capture additional information about each of the steps
in a project, you can use User Defined Field Labels. After you have
set up a label set, you can select a set and enter the appropriate
information that VISUAL saves along with your project.

Use Infor VISUALs Customer, Part, or Vendor Maintenance win dows
to set up your User Defined Field Labels. For more information on
setting up User Defined fields, refer to your Infor VISUAL documentation.

To select User Defined Field Labels:

1. Open the Edit dialog
   box for the item on which you want to use User Defined Field Labels.

For example, if you want to use User Defined
Field Labels for a task, open the Edit dialog box for that task by
double-clicking the task or selecting Edit
from the Edit menu.

2. Click the User
   Defined tab.
3. Click the User
   Defined Field Layout ID arrow and select the set of labels
   you want to use.

The field labels you selected appear.

Click in the appropriate field and enter
the relevant information as the label defines.
