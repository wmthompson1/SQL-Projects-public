# What are Customizable User Defined Fields

You may find that most of the windows you use in VISUAL contain
all the information you want while others lack fields unique to your
company or circumstances. You can use VISUAL's Customizable User Defined
Fields to create, columns in tables and tabs on windows to capture
the information you want. VISUAL stores this information in your database
for use in your reports.

Any user with System Administrator privileges can create and edit
customizable user defined fields.

You can add Customizable User Defined Fields to the following programs:

| Program | Located on Menu | Type of Customizable UDF |

| VFARIENT - A/R Invoice Entry | Edit | Tab & Line |

| VMMFGWIN - Mfg Window - Material Card | Edit | Tab |
| VMMFGWIN - Mfg Window - Leg Card | Edit | Tab |
| VMORDENT - Customer Order Entry | Edit | Tab, Line, & Delivery Schedule |
| VMORDWIN - Customer Order Window | Edit | Tab, Line, & Delivery Schedule |
| VMPRJMNT - Project Maintenance | Maintain | Tab |
| VMPRTMNT - Part Maintenance | Maintain | Tab |
| VMPURENT - Purchase Order Entry | Edit | Tab, Line, & Delivery Schedule |

| VMSHPENT - Shipping Entry | Edit | Tab, Line, & Delivery Schedule |
| VMSRVENT - Outside Service Entry | Maintain | Tab |
| VMUPMNT - Unplanned Maintenance | Edit | Tab |
| VMVNDMNT - Vendor Maintenance | Maintain | Tab |

Refer to the table above to determine the type of user-defined information
you can add to a window. If a type of user-defined information is
not available in a window, the system disables the appropriate Insert
button. For example, if you are adding user-defined information to
Customer Maintenance window, the system does not give you the option
to add a Delivery Schedule or Line Item user-defined field, since
that type of information does not apply to the Customer Maintenance
window.
