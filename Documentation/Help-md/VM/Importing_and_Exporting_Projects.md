# Importing and Exporting Projects

Importing and exporting information as ASCII text
files give you the advantage of being able to use your project information
in other pro grams.

You can use Importing and Exporting to:

Import and Export to and from other systems. Many other systems
output and accept text file formats. You may be able to export information
from another system and import the information into a VISUAL Project.
Sometimes you may need to reformat that text before you can import
it.

Transferring information between databases. Using the Importing
and Exporting features, you can transfer information from a project
into a different database.

## Exporting Project Information

Use the Export command to save the current Project to a file on
your computer.

1. If the project you want
   to export is not open, open it.
2. From the Edit menu,
   select the Export option.

The Export dialog box appears.

3. Enter a unique file
   name for the information you are exporting.

VISUAL automatically assigns a file extension
of .ENG. If you want to specify a different file extension, enter
the appropriate extension after the name.

4. If the directory where
   you want to save the file is not the current directory, navigate
   to and select the directory into which you want to export the
   project information.
5. Click the Save
   button.

VISUAL saves your project information to
the file you specified and a message appears, indicating that the
information is being copied.
