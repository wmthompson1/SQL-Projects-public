# What is Part Trace Profile Maintenance?

Part Traceability is an optional Infor VISUAL module
that provides tracking of purchased or fabricated materials by lot,
serial number, or other identifier. This applies to all points in
the material cycle: receipt of raw materials, return to vendor, issue
of raw materials to work orders, reporting completed parts, receipt
of work orders to inventory, shipment of finished goods, and inventory
adjustments.

Two applications are used to setup and apply part traceability.
Part Trace Profile Maintenance is used to describe how you want traceability
to work for a part. Part Trace Maintenance is used to report, maintain,
and print traceability records. Additionally, a Part Traceability
button and dialog are used in all applications that are involved in
material transactions.
