# Adding Requisition Category Type Codes to the Database

Before creating any new purchase requisitions, add
requisition category type codes to the database. Category type codes
help you classify purchase requisitions by any quantifiable measure
you choose to impose. For example, requisition types can reflect an
"as needed" procurement policy or a broader system of product
procurement.

1. In the Purchase Requisition
   Entry window, select Category Type Table
   from the Maintain menu.

The Requisition Category Type Maintenance
dialog box appears.

2. Click the Insert
   button to add a new category code.

The next available line is activated and
an arrow is placed in the first column.

3. Enter information into
   the following fields as follows:

ID - Enter an
ID for the category type into the ID field. Enter a range of IDs to
cover all types of requisitions.

Description -
Enter a description for the ID.

4. Repeat steps 2 and 3
   for each new category type code.
5. Click the Save button
   to save the category codes to the database.
6. Click the Close button
   to return to the main window.
