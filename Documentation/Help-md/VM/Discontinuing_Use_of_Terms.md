# Discontinuing Use of a Term

If you would like to discontinue the use of a term without deleting
the term from your database, clear the Active check box. The term
will no longer be available for selection.

1. Select Admin,
   Application Global Maintenance.
2. Select Accounting
   Maintenance, Terms Table.
3. Select the row that
   contains the term that you want to deactivate.
4. Clear the Active
   check box.
5. Click Save.
