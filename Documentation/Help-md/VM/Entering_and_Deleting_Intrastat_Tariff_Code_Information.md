# Entering and Deleting Intrastat Tariff Code Information

Tariff codes define the class of goods being transported.

1. Click the Tariff
   button.

The Tariff dialog box appears.

2. Click the Insert
   button.

A new line appears in the line item table.

3. In the Code column,
   enter a numeric code for the new tariff.
4. In the Description column,
   enter a description for the tariff code.
5. Click the Save
   button to save the new information.

## Deleting Tariff Codes

1. Click the Tariff
   button.

The Tariff dialog box appears.

2. Select the tariff code
   you want to delete, and then click the Delete
   button.

An X appears in the row header beside the
code you are deleting.

3. Click the Save
   button to save the new information.
