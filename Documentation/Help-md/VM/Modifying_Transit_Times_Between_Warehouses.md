# Modifying Transit Times Between Warehouses

After you save Transit Time information, you cannot
delete or modify a From and To warehouse combination.

You may, however, need to modify the number of days it takes for
transit between two warehouses for a number of reasons.

Call the warehouses into the appropriate fields and, if a transit
time already exists between the two, manually change the value that
appears in the Transit Days Field to the new value.
