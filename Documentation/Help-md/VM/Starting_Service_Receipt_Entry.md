# Starting Service Receipt Entry

Service Receipt Entry is available from the VISUAL
Main window.

To access the Service Receipt Entry window:

Select Service Receipt Entry from the
Purchasing menu.

The Service Receipt Entry window appears with no information in
it.

The Service Receipt Entry window contains both entry fields and
buttons.
