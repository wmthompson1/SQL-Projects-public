# Entering Vendor and Order Notes

The vendor notes and order notes are basically the
same. The difference between them is that vendor notes are tied to
the vendor and order notes are tied to the order. That is, a note
you enter for a vendor in one order also appears in every other order
placed to that vendor. The notes you enter for an order only appear
in the order in which they were written.

1. With the appropriate
   customer open in the Purchase Order Entry window, click the Order Notes or Cust Notes
   button.

The Notations dialog box appears.

2. Click the New
   button to make a new entry.

A line appears with the current date and
time for the entry.

3. Enter the appropriate
   information.

You can enter an unlimited amount of information.

Use the Paste
button to paste text that you copied from another location into the
box.

Use the Copy button
to copy text in the dialog box to the clipboard.

4. Click the Save
   button to save the Notes.

VISUAL adds the user name to the end of
the notation entry.

5. Click the Close
   button to close the Notations dialog box and return to the Purchase
   Order Entry window.
