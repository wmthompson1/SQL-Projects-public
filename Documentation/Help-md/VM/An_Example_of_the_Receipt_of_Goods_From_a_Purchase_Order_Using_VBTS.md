# An Example of the Receipt of Goods From a Purchase Order Using VBTS

1. From Purchase Order
   Receipt or Purchase Order Entry window, print paperwork and labels
   with barcodes.
2. From Warehouse Maintenance,
   print warehouse and locations with labels.
3. From the appropriate
   report or the actual purchase order quantity itself, scan any
   record of the transaction in BTSbi.exe for:

   * Employee ID
   * Purchase Order ID
   * Purchase Order Line
   * \*To Warehouse ID
   * \*To Location ID
   * \*Trace ID
   * \*Alpha Propertys
   * \*Numeric Propertys
   * Qty
4. Validate that BTSBI
   has processed the transaction by looking in VMBTSRCV.EXE for the
   appropriate record in the completed transactions portion of the
   window.
5. Purchase Receipt Entry
   for transaction.
6. Check Part Maintenance
   for warehouse locations in Warehouse Maintenance or the Manufacturing
   Window for the correct inventory update.
