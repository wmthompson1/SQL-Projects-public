# Starting the Costing Tools Window

You can access the Costing Tools window from the Admin
menu of the VISUAL Main window.

To start Costing Tools:

Select Costing Tools from the Admin menu.

The Costing Tools window appears with no information in it.

The Costing Tools window contains both entry fields and action buttons.
