# What is the Project Summary Window?

The Project Summary window provides a platform from which you can
manage your project budgets, apply those budgets, and review the results.
You can also view the results of your actual costs and compare them
to your budgeted figures thereby allowing you greater accuracy in
your budgeting.

Use the Project Summary window to:

* Set up Budgets
* Maintain Cost and Burden
  tables
* Set up your G/L and Burden
  Mappings
* Spread Budgets
* Calculate Earned Value
* Estimate Completion Dates
* Calculate Retro Burden
