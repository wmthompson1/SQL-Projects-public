# Opening a Related Document in the Document Lifecycle Tree View

To open a related document in the Tree View:

1. Select Info,
   Lifecycle Document Viewer.
2. Double-click the document
   ID in the document list or double-click the line for the document
   in the detail table. The linked document is displayed in the module
   in which it was created.
