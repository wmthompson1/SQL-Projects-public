# Controlling Work Orders Printed

There are several options for selecting which work
orders have travellers or picklists printed:

## Printing Single Work Orders

By specifying Base ID, Lot ID, Split ID, and Sub ID,
you can print a new traveller and/or picklist for a work order, regardless
of its status. You can select a single leg to print, or All. This
allows individual legs to be started before the main leg, or in parallel
with it.

To print a single work order, you must fill in all four of fields.

## Printing by Release Date Range

You can specify an open or closed range for release
dates by indicating an earliest and/or latest release date in the
Earliest Release Date and Latest Release Date fields.

If you leave the Earliest Release Date field blank, VISUAL includes
all firm and released work orders, up to and including any specified
Latest Release Date. Likewise, if you leave the Latest Release Date
field blank, VISUAL includes all work orders with release dates on
or after the specified earliest date. If you leave both fields blank,
all work orders, without regard to release date, are included.

Release Date Range in combination with the other options listed
below determines which work orders are actually printed.

## Printing New Releases Only

If you select Print New Releases Only, only work orders
that were auto released (see below) from a Firm status are printed.
Otherwise, all Firm and Released work orders are printed.
