# Deleting Cost Simulations

|  |  |
| --- | --- |
| WARNING.gif | Deleting Costs Simulations completely removes then from your database. Make sure you want to delete the simulation before you do so. |

To delete cost simulations:

1. Select Inventory,
   Cost Simulation.

2. If you are licensed
   to use multiple sites, click the Site ID
   arrow and select the site to use for the simulation. If you are
   licensed to use a single site, this field is unavailable.
3. Click the Simulation
   ID browse button and select the simulation to delete.
4. Click the Delete
   button.
5. To delete the simulation,
   click Yes.
