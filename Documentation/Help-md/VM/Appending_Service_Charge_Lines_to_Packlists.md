# Appending Service Charge Lines to Packlists

Service charges are items that are either ordered
by a customer or items charged to the customer in fulfillment of the
order. Service charges are not stocked parts.

An example of a service charge is a crating charge, a cost passed
on to the customer for packing or crating the parts on the order for
shipment.

1. Select Sales,
   Shipping Entry.
2. If you are licensed
   to use multiple sites, specify the site from which you shipped
   the order.
3. In the packlist ID field,
   specify the packlist ID.
4. Select Edit,
   Edit.
5. Specify this information:

Service Charge ID
- Double-click the column header to select a Service Charge ID for
the line. Enter Service Charge IDs in Site Maintenance.

Shipping U/M -
The shipping unit of measure for the service charge.

Ship Qty - The
quantity of the service charge that you are shipping.

Unit Price - The
price per unit of the service charge.

Discount % - The
discount percent of the service charge line. Set this while entering
service charge information in Application Global Maintenance.

Actual Freight
- The dollar amount of freight charge for the service charge line.

Shipping Weight
- The weight of the service charge line.

Weight U/M - The
shipping weight unit of measure.

NMFC ID - The
National Motor Freight Code ID. Double-click the column header to
select an ID from the list. Enter NMFC codes in Application Global
Maintenance.

Package Type -
The type of package you are using to ship the service charge line.

Commission % -
The commission percentage of the sales rep.

VAT Code - If
an international order, the Value Added Tax ID, of the service charge
line. Enter VAT Codes in Accounting Entity Maintenance.

G/L Rev Acct ID
- The general ledger account for the service charge line.

Packlist Long Description
- A description, in long form, of the line and/or its contents.

Ship Dimensions
- A description - numeric or otherwise - of the shipping dimensions
for the service charge line.

HTS Code - Harmonized
Tariff Schedule. Select an HTS code if this is an international order
and duty percents apply. Enter HTS Codes in Application Global Maintenance.

Country of Origin
- The country from which the order originated. Enter Countries of
Origin in Application Global Maintenance.

Service Charge Site ID
- The site ID associated with the service charge.

6. Click the Save
   toolbar button to save the new service charge line.
