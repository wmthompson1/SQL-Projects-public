# Fix Synonyms and Grants

When VISUAL fixes synonyms and grants, it creates
the public synonym to all tables in the database so that all users
will have access to the tables in the database. For the public synonym
to work, you must have at least one privilege to each table. After
VISUAL generates the public synonym, it then grants a select to all
users for all tables.

To fix synonyms and grants:

1. Select Admin,
   Fix Synonyms and Grants.
2. Click Start
   to begin the procedure.
3. A dialog box is shown
   warning you that this procedure will only work if all other users
   are disconnected from the database. Click Yes
   to continue. A meter displays the progress through the database
   tables. During the procedure, you can click Abort
   to stop activity.
