# Using Online Help Toolbar Buttons

The buttons along the top of the help window provide
additional help functions.

Help window buttons include:

|  |  |
| --- | --- |
| Hide/Show | Click this button to toggle between hiding and showing the left pane of the Help window. |
| Back/Forward | Click the Back button to go back to the previous topic. If you are viewing topic B, then click the Back button to view topic A, you can then click the Forward button to return to topic B. |
| Print | Click this button to print the current topic. If you select a book level heading, you have the option of printing all of the topics within the book. For example, if you clicked the Online Help heading then click Print, the system offers you the option of printing all of the topics that fall under the Online Help heading or printing only the selected topic. |
