# Setting Up Shop Groups

The Shop Groups function allows you to maintain the
list of members for a group type resource.

If you are licensed to use multiple sites, the site
ID of the resources you add to the group must match the site ID of
the group. If the site ID does not match, you are prompted to add
the resource group member to the site. For example, say you created
a group shop resource in site MMC called MMC GROUP. All resources
that you add to MMC GROUP must also exist in site MMC. If you assigned
MMC GROUP to MMC and MMC-2, then all shop resources you assign to
MMC GROUP must exist in both MMC and MMC-2.

If your shop resource group is assigned to multiple
sites, you are prompted to add member resources to all sites to which
the shop resource group is assigned.

If you are licensed to use a single site, you can
add any shop resource to a shop group without limitation.

1. Select Shop
   Groups from the Edit menu

The Shop Groups dialog box appears.

2. Select the Resource
   ID for the group for which you are maintaining the member list
   from the pull down menu available.
3. Click the Insert
   button

A new row appears in the table. VISUAL sets
the sequence number for the first row to 10.

4. Enter the Resource ID.
   You can also double click the Resource ID column header and select
   from the Resource ID table.
5. Repeat steps 3 and 4
   until you have added all the resources to be added in a Group.
6. Click the Save
   button to save the created or edited Shop Group.

It is important to maintain sequencing for
Sched One types of group resources.

The Global Scheduler attempts to use member
resources in the sequence they appear in the list.

Use the Delete
button to delete the resources from a particular Group Resource ID.

Use the Copy button
to copy the member list from one Group Resource ID to another.
