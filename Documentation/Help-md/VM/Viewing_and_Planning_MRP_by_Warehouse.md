# Viewing and Planning MRP by Warehouse

You can view material resource planning for a part
in a specific warehouse. From the View menu, highlight MRP by Warehouse.
VISUAL places a check next to the option.

A Warehouse ID browse button appears between the header section
and the top line item table.

Click the Warehouse button to view a list
of warehouses or manually enter a Warehouse ID in the field.

Orders with the specified Part ID and Warehouse ID appear in the
appropriate line item tables.
