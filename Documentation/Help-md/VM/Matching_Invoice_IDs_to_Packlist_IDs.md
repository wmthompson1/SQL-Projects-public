# Matching Invoice IDs to Packlist IDs

Optionally, you can use the packlist ID as the invoice ID. Set up this functionality at the customer level.

1. Select Sales, Customer Maintenance.
2. Click the Customer ID browse button and select the customer whose invoice IDs should match the packlist IDs.
3. Click the Shipping tab.
4. Select the Match Invoice ID to Packlist ID check box.
5. Click Save.
6. When you are ready to ship an order to a customer, print a Before Packlist ID. If you do not print a Before Packlist ID, then the invoice ID will not match the packlist ID:

1. Select Sales, Shipping Entry.
2. If you are printing the packlist for a particular order, specify the ID of the order in the Order ID field. Otherwise, leave this field blank.
3. Select File, Print Pack Lists.
4. In the Print section, click Before Shipment.
5. If you are printing the packlist for the order that you selected in the main Shipping Entry window, click Current Order. Otherwise, use the filters to print packlists. See [Creating Packlist IDs Before Shipment](Creating_Packlist_IDs_Before_Shipment.htm).
6. Click Ok.
7. If the packlist is printed correctly, click Yes in the dialog.
8. When you are prompted to save the packlist ID, click Yes.

7. Complete the shipment.
8. Generate the invoice for the shipment. The invoice ID is the same as the packlist ID.
