# Using a Right Mouse Click to Allocate Expected Quantities to Customer Orders

To perform this procedure, you must have the Order
Entry child window and the Part Availability child window open. Tile
the two so dragging between them is easy. The part you want to allocate
to the customer order line must be the focus part in the Part Availability
child window.

1. In the Part Availability
   child window, double-click the part
   icon double click the Available parts
   icon, double-click the Part ID icon,
   expand the warehouse icon (the Warehouse ID on the order line),
   then click the Expected plus box. Expected orders appear.
2. In the left pane, find
   the expected order you want to allocate from, select it, and right-click
   it.

The Allocate option box appears.

3. Click Allocate.

If there are no conflicts, VISUAL allows
the allocation from the expected order to the current customer order.
In the Customer Order Entry child window, you can verify the supply
by looking at the Supply Type column in the line table. For example,
if the expected order was a purchase order, PO appears in the column.
