# Entering Part Aliases

Part aliases are alternate names, abbreviations, or
other designations for a Part ID. The use of part aliases is exclusive
to the Order Management module. You can only use Part Aliases in the
Order Management window.

1. With the part for which
   you want to establish part aliases displaying in the Part ID field
   in the Part Maintenance window, click the Part
   Aliases button from the Order Management
   tab.

The Alias for Part dialog box appears.

2. Click the Insert
   button to start adding line item information.
3. Enter information in
   the following fields:

Alias ID - The
ID of the alias for this part.

Description -
A description of the alias for this part.

Type - The type
of alias. Select an Alias from the list box. Enter Alias types in
Application Global Maintenance.

4. Click the Save
   button when you are finished entering part alias information.
