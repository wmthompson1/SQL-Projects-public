# Postponing Activity Reminders

Use the Snooze buttons to redisplay [activity
reminders](Using_the_Reminder_Dialog.md) at a later time.

To postpone a reminder for a single activity:

1. Select the line.
2. Specify when to redisplay
   the reminder. Select a value from the drop-down menu.
3. Click Snooze.
   The activity is removed from the table. The amount you specified
   in the snooze time field is added to the reminder time on the
   activity.

To postpone all reminders, specify when
to redisplay the reminders and then click Snooze
All. All activities are removed from the table.

4. After the snooze interval
   has elapsed, a reminder is redisplayed.
