# Configuring Barcode Printing Options

If you regularly
include barcodes with your packlist reports or print to the same thermal
label printer when you print thermal labels, you can make the inclusion
of barcodes with packlist reports and thermal label printer information
a default setting.

1. From
   the Options menu, select Configure Barcode
   Print Options.

The Configure Barcode Print Options dialog
box appears.

2. From
   the Thermal Printer list box, select the thermal printer you want
   to make the default thermal label printer.
3. In
   the Thermal Printer Number field, enter the number of the thermal
   printer.

This serves as an ID for the thermal printer.

4. In
   the Number of Thermal Labels field, enter the number of thermal
   labels that you typically print or want to set as the default.
5. Click
   the browse button to select the directory into which you want
   the thermal labels to spool before printing.
6. Select
   the Include Barcodes as Default
   check box to make
   the inclusion of barcodes with packlist reports the default.

With this default, the Print
Barcodes check box in the Print Packlists dialog boxes is selected
automatically. You can override this setting at any time by clearing
the check box before printing a packlist report. See [Printing
Packlists](Printing_Packlists.md) for more information.
