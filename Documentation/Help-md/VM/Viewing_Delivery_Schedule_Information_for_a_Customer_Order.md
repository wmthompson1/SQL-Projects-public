# Viewing Delivery Schedule Information for a Customer Order

If a customer order contains delivery schedule lines,
you can view delivery schedule information for that customer order
line.

|  |  |
| --- | --- |
| POSTIT.gif | The Delivery Schedule button is unavailable unless the highlighted line is a delivery schedule line. |

1. With the order for which
   you want to view delivery schedule information highlighted, click
   the Delivery Schedule button located
   underneath the line item table.

The Shipments dialog box appears.

The line item table contains the following
columns:

Line Number -
The line number of the delivery schedule as it appears in the customer
order.

Desired Ship Date
- The date by which the customer wants shipment of the delivery schedule
line to occur.

Order Quantity
- The order quantity of the delivery schedule line.

Shipped Quantity
- The quantity of the delivery schedule line that you have shipped
to the customer.

ShipTo ID - The
customers ShipTo ID.

Allocated Quantity
- The quantity of the delivery schedule line that you have allocated
to demand.

Fulfilled Quantity
- The quantity that has been fulfilled by the supplying source.

2. Click Close
   when you are done viewing delivery schedule information.
3. Click Ship
   Info to see general shipping information for the delivery
   schedule line.

See the next section for more information
on viewing general shipping information for customer order lines.
