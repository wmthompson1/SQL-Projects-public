# An Example of Common Sense Throughput

Imagine that you are the manager of a new manufacturing
company. At first, your company plans to make only two products, PRODUCT
A and PRODUCT B. You know that you can sell 50 PRODUCT As per week
at $180.00. You know that you can sell 25 PRODUCT Bs at $200. You
need to know how much profit your company will make the first week.

## Here are the market conditions:

Marketplace

|  |  |  |
| --- | --- | --- |
|  | Price | Quantity |
| PRODUCT A | $180.00 | 50 per week |
| PRODUCT B | $200.00 | 25 per week |

Material Costs

|  |  |
| --- | --- |
| PPV | $25.00 each |
| RAW L | $30.00 each |
| RAW M | $35.00 each |
| RAW N | $40.00 each |

Operating Expense

$6000 per week

Capacity

|  |  |
| --- | --- |
| RESOURCE W | 2,400 minutes/week |
| RESOURCE X | 2,400 minutes/week |
| RESOURCE Y | 2,400 minutes/week |
| RESOURCE Z | 2,400 minutes/week |
