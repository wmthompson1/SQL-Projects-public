# Using the Right Mouse Button to Insert a Substitute Part into the Current Customer Order

Because a substitute part is similar to the part that
it replaces, you may want to order that part in addition to the primary
part and not only as a substitute. You can insert an order line for
a substitute part into the current customer order by right-clicking
on the substitute part in the Substitute Part window.

1. In the VISUAL Order
   Management Customer Order Entry window, highlight the order line
   that contains the primary part.
2. If not already open,
   the Order Management Window; then open the Current Customer Order
   Window and the Substitute Parts Window. Tile or cascade the two
   windows so that you can easily view both windows.
3. In the Substitute Part
   Window, double-click the icon.

If there are any, substitute parts for the
primary part appear.

4. Highlight the substitute
   part that you want to add to the current customer order and perform
   a right mouse click.

An Options box appears.

5. Left-click Insert
   Substitute Part to insert the substitute part into the
   current customer order.

The VISUAL Order Management Customer Order
Entry Window appears with the new order line for the substitute part
added in the line item table.

6. Enter a quantity for
   the line and other necessary information and then click the Save button to commit the order.
