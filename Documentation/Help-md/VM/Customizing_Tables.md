# Customizing Tables

Using the Configure Table tool, you can select which
data columns you want to appear.

1. In the view panel, click
   the Table Properties button.

The Configure Table Window appears populated
with all of the available data fields. The Attribute Column indicates
if the field is hidden from view in the panel or is visible.

2. Select the column you
   want by clicking in the row header.
3. If the field is hidden
   and you want to make it visible, select the Include
   check box.

If the field is visible and you want to
hide it, clear the Include check box.

4. If you want to rename
   the column, click in the title column and enter what you want
   to appear in the table. Use an at character (@) if you want
   your title to appear on separate lines.
5. When you have finished
   changing what appears in the panel, click the Ok
   button.
