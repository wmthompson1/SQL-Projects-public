# Setting the Display Date

Use the Date option to set the earliest displayed
date to any date after the "as of" date of the current schedule
ID. Using this option can be quicker than scrolling.

1. Select Date
   from the View menu.

The Set Start Date dialog box appears.

2. Enter the start date,
   or click the calendar icon to bring up a calendar. Select a date
   from the calendar.
3. Click Ok
   to set the start date, or Cancel to
   close the dialog box without setting the start date.
