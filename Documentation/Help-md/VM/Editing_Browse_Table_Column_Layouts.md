# Editing Browse Table Column Layouts

You can set the position, width, title, and visibility
of columns.

To do this:

1. Select Edit
   Columns from the Browse window Options
   menu, or right click in the Browse window and select Edit
   Columns from the pop-up menu.

The Edit Columns dialog box appears.

The Edit Columns dialog box displays the
list of columns available, the column titles, check boxes for including
columns in the table, and the position of the columns in the table.

2. Make the appropriate
   edits to the columns.

To add a column to the table, click in the
Include check box, then enter a position in the Position column. This
assigns the columns placement in the table, numbered from left to
right.

To remove a column from the table, click
in the Include check box to remove the check mark.

To change the order of the columns, change
the numbers in the Position column as appropriate.

VISUAL places a check mark to the left of
the Column Name, indicating that you have edited that column.

|  |  |
| --- | --- |
| POSTIT.gif | If you have set up User Defined Fields, you can also include them in the columns appearing in the browse dialog box. You can use User Defined information for search and query functionality. |

3. Click Ok
   to apply the changes to the Browse table.

VISUAL refreshes the Browse table to display
your changes.
