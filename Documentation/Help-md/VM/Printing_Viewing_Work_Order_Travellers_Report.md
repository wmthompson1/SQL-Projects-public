# Printing/Viewing Work Order Travellers Report

## Options

### Operation Write-Ins

Operation Write-ins provide for up to four data entry areas on each
operation printed. To be filled in by operators, supervisors, or other
production personnel. The labels for these fields are specified in
the Operation Write-in fields in the upper right of the dialog. The
dialog starts up with these defaults, which can be overridden if desired.
The four write-ins appear horizontally for each operation on their
own line. A blank underline appears to the right of each label.

### Operation Barcodes

If Barcode Labor Entry is being used, this option allows printing
of a barcode encoding workorder Base ID, Lot ID, Split ID, Sub ID,
and Operation Number. The Code 39 barcode font must be installed on
the system where the traveller is being printed in that type of barcode.

### Customer Order Info

If a customer order is linked to the work order, this option will
print customer order information in the header section of the traveller.
This includes Order ID, Order Date, Ship Via, Customer PO Number,
Phone, FAX, Customer Part ID, Customer Name and Address, Ship To Address,
and the Drawing/Revision from the order line.

### Part Location

This option will show all valid part locations for each part requirement.
If the requirement is for an auto-issue part, this is indicated.
