# Viewing Master Production Schedules

The Schedule IDs that you enter here appear in the
Schd ID list box in the main Material Planning
Window and in the Schd ID list box that
appears at the top of the other Master Production Schedule dialog
box - the one that allows you to enter demand forecast information.
