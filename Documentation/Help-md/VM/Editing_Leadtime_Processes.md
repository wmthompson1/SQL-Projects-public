# Editing Leadtime Processes

You can edit a leadtime process provided that the
process type has not been used in a quote or order.

1. Select Inventory,
   Part Maintenance.
2. Select Maintain,
   Process Type.
3. Select the process type.
4. Make your edits.
5. Click the Save
   button.
