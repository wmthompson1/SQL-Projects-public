# Restoring Default Template Settings

To restore the templates and available tokens to the
default settings:

1. Select Admin,
   Notification Maintenance.
2. Select Options,
   Reset Defaults.
3. Select the templates
   that you want to reset.
4. To reset the list of
   available tokens that are associated with the templates that you
   selected, select the Reset Associated Default
   Tokens check box.
5. Click Ok.
