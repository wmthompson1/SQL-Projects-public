# Hiding Inactive Shipping Addresses

Use Preferences Maintenance to specify whether [inactive shipping addresses](Inactivating_Shipping_Addresses.htm)
are hidden in the List Ship-to-IDs browse table and the Shipping Addresses
dialog.

If you show both active and inactive shipping addresses, you are
warned if you attempt to select an inactive address on a record.

1. Select Admin,
   Preferences Maintenance.
2. Click Insert.
3. Specify this information:

Section Specify
CustomerMaintenance.

Entry Specify
HideInactiveShipToIDs.

Value Specify
Y to hide inactive addresses. Specify N to show inactive addresses.

4. Click Save.
