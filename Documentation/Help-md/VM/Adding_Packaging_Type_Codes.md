# Adding Packaging Type Codes

Packaging codes define the types of packaging to use
for parts.

1. Select Admin, Application Global Maintenance.
2. Select Maintain,
   Packaging Types.
3. Click Insert.
4. Enter the new packaging
   type code.
5. Click Save.
