Part ID




## Part ID

This field contains the identification code for the current item. This
ID is used throughout VISUAL to identify the current part. The alphanumeric
ID must be unique, and between 1 and 30 characters. You can enter the
Part ID manually, or select it from the browse by clicking the Part
ID button.
