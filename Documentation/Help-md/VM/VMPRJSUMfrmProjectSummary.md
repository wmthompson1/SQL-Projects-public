Using the Project Summary Window




# Using the Project Summary Window

The Project Summarys main window contains a table
that, depending on your selections, provides you with quick access
to important project information:

* After you have spread the
  projects budget, your Budgeted costs and hours
* After you have run the costing
  utilities, the projects actual costs and hours
* After you have calculated
  earned value quantities and amounts, your projects Earned Value
* Schedule variance
* Cost variance
* Budget variance
* Estimate to complete

VISUAL provides several viewing methods for your data:

* Together or separately
* In monthly buckets or in
  summary form
* For single projects or many

VISUAL also gives you the ability to access different layers of
information down to the source transactions from which VISUAL derives
the actual costs.

To view your project status in the Project Summary window:

1. From the Project menu
   of the Infor VISUAL main window, select Project
   Summary.

The Project Summary window appears.

2. Click
   the Site ID arrow and select the site
   to use.
3. Click
   the Project ID button and select the
   project for which you want to view a summary.
4. If you have created
   several versions of the project you selected, click the Versions
   arrow and select the specific version you want to view.
5. In
   the Summary Types section, select the type of summary you want
   to appear in the table.

As you select summary types the appropriate information appears in
the table.

If you select a Variance summary type, the
Variance Threshold section activates allowing you to specify the following:

Type Click the arrow and select the type of
threshold you want to use for this summary. You can select Fiscal
Period, Year-To-Date, or Cumm-To-Date.

Amount
Enter the amount of variance you want to use for this summary.

## Using Project Filters

In the Project Summary window, you have the option
of viewing more than one project at a time by either selecting the
projects you want to view or by selecting project type. You also have
the option of further selecting project data by department.

For example, if you are rough cut capacity planning, you can use
project filters to select All project types with an Engineering type
department. By selecting a Summary Type of Budget Spread, you can
easily view the expected number of engineering hours required on a
monthly basis across all off your projects.

 
 
 

To set project filters:

1. Open the Project Summary
   window and click the Project Summary tab.
2. Click the Project Info
   button.

The Project Info Filters dialog box appears.

3. Use the tabs to select
   the types of filters you would like to use. You can enter filter
   information on a single tab or on multiple tabs to retrieve the
   information you would like to view. Use the arrow buttons to move
   project IDs to the Selected Codes box. The tabs are:

Project ID Click
the Project ID tab to specify individual projects you would like to
view. The IDs of all the projects appear in the Available Codes box.

Project Type
Click the Project Type tab to select the specific project types you
would like to view.

Department ID
Click the Department ID tab to select the specific department types
you would like to view.

Business Line
Click the Business Line tab to select the specific business lines
you would like to view.

Division Click
the Division tab to select the specific divisions you would like to
view.

Customer Click
the Customer tab to select the specific customers you would like to
view.

Cost Category
Click the Cost Category tab to select the specific cost categories
you would like to view.

WBS Click the
WBS tab to select the specific WBS codes you would like to view.

Cost Group - Click
the Cost Group tab to select the specific cost groups you would like
to view.

4. When you have finished
   building your filter set, click the Ok
   button.

The information for the filter set you selected
appears in the Project Summary window.

5. Select the Summary Type
   you want to use.

The appropriate information appears in the
Project Summary window for the Summary Type you selected.
