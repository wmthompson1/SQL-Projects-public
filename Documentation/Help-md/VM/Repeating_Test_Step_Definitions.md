# Repeating Test Step Definitions

You can repeat test step definitions in Test Definition.

1. Highlight the test step
   line item that you want to repeat.
2. Click the Repeat
   button.

VISUAL copies the test step line item and
places the copy at the bottom of the table.

3. Click Ok
   to save the changes.
