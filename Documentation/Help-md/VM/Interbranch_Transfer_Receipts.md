# Interbranch Transfer Receipts

For Project and A&D users, the following required
fields also appear in the Interbranch Transfer Receipts line item
table:

WBS Code - If this transfer refers to
a project, the WBS Code appears.

Dept ID - If this transfer refers to a
project, the default Department ID appears for the WBS Code.

Proj Sub ID Ref - If this transfer refers
to a project, the default Sub ID Ref for the project appears.

Proj Seq No Ref - If this transfer refers
to a project, the default Seq No Ref for the project appears.
