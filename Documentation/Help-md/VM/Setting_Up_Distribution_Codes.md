# Setting Up Distribution Codes

Use the Distribution Code Maintenance window to set
up the budget distribution schemes you want to use for your projects.

|  |  |
| --- | --- |
| POSTIT.gif | Assign your Distribution Codes using the Budget Maintenance window. (Maintain menu of the Project Summary window.) |

Distribution intervals are the intervals on which you want VISUAL
Enterprise Projects/A&D to assign your budget costs to the appropriate
budget line.

For example, if you want to set up a simple two line 60%/40% distribu
tion scheme, enter 60 in line one and 40 in line two. When you assign
this to a $1000 budget line that is for two weeks, Project distributes
$600 for the first week and $400 for the second week.

If you have a ten-line (10% each line) distribution scheme and assign
it to a $1000, ten-week budget line, Project distributes $100 each
week.

Remember, your total Interval Percentages must add to 100%.

To enter Distribution Codes:

1. From the Project menu
   of VISUALs main window, select Distribution
   Code Maintenance.

The Distribution Code Maintenance window
appears.

2. Click in the Distribution
   ID field and enter an identification number for this Distribution
   Code.
3. Press the TAB key or
   click in the Description field and enter a description for this
   Distribution Code.
4. Click the Status
   arrow and select the appropriate status for the Distribution Code.
5. Click the Insert
   Row button on the toolbar.

A blank row appears highlighted in the table.

6. Click in the Interval
   Percentage column and enter the percentage you want to use for
   this interval.
7. If you want to add more
   intervals, press the TAB key or click the Insert
   Row button on the toolbar.

A new highlighted row appears in the table
where you can enter the appropriate percentage.

8. When you have completed
   entering all the interval percentages for this Distribution Code,
   click the Save button on the toolbar.

The Distribution Code Maintenance window
saves the new code to your database.

If your Interval Percentages do not total
100%, a dialog box appears. Click the OK
button and correct the offending Interval Percentages.

9. If you want to enter
   more Distribution Codes, click the New
   button on the toolbar and enter the appropriate information.
10. When you have finished
    entering Distribution Codes, click the Close
    Window button or, from the File menu, select the Exit but ton.
