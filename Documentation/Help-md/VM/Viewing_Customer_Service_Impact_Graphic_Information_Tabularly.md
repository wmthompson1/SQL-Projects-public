# Viewing Customer Service Impact Graphic Information in a Table

As you are viewing a customer device impact graph,
you can click the Table toggle button to view the graphic information
in a table.

The line item table above shows the same information a graph shows.
Schedule IDs appear to the far left and, according to the time preference
you currently have set, each column represents a day, a week, or a
month.

Click the button again to switch back to the graph.
