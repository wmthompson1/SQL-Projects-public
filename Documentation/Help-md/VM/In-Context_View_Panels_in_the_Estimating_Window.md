# In-Context View Panels in the Estimating Window

You can view the Customer View Panel and Part View
Panel in the Estimating Window.

To access the Customer View Panel, select View,
Customer View Panel. Use the Customer View
panel to view information about the selected customer. You can view
summary information, detail information, and a chart. For information
about setting up the summary and detail views, see [In-context
View Panels](In-context_View_Panels.htm). For information about the chart, see [Customer
View Panel Chart](Customer_View_Panel_Chart.htm).

To access the Part View Panel, select View,
Part View Panel. For information about setting
up the summary and detail views, see [In-context
View Panels](In-context_View_Panels.htm). For information about the Part view panel chart,
see [Part View Panel Chart](Part_View_Panel_Chart.htm).

To access the Process Activity View Panel, select View, Process
Activity View Panel. The charts show information about the performance
of your sales process compared to a standard that you set up. For
information about the charts, see [Viewing
the Leadtime View Panel Charts](Viewing_the_Leadtime_View_Panel_Charts.htm).
