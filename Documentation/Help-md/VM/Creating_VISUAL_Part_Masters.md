# Creating VISUAL Part Masters

1. Select the Create
   VISUAL Part check box.

Make sure that you VISUAL database name,
User ID, and Password are correct. This is the VISUAL database into
which you are importing the data.

2. When you are ready to
   import the text file into your VISUAL database, click the Start! button.
3. Using the Search dialog
   box that appears, locate and select the text file that you want
   to import into your VISUAL database. Keep these text files in
   your VISUAL directory.
4. Click the Open
   button.

The VISUAL Data Import Utility imports the
specified text file, creates the part master in your Infor VISUAL
database.

5. After clicking, the
   Start button becomes the Stop
   button. Click the Stop button to stop
   an import session.

If you click the Stop
button, VISUAL asks you to confirm the action.
