# Starting Physical Inventory Count

Physical Inventory Count is available from the VISUAL
Main window.

To start Physical Inventory Count:

Select Physical Inventory Count from the
Inventory menu.

The Physical Inventory Count dialog box appears.

You can have multiple counts in process at any time. You can enter
counts for only one active physical count ID at a time.
