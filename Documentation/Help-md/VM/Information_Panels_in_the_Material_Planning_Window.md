# In-context View Panels in the Material Planning Window

You can view three information panels in the Material Planning Window:
the Part view panel, the Vendor view panel, and the Part Sites panel.

To access the Part view panel, select View,
Part View Panel. Information is shown for
the site selected in the Site ID field. For information about the
Part view panel chart, see [Part
View Panel Chart](Part_View_Panel_Chart.md). For information about how to use view panel,
see [In-context View Panels](In-context_View_Panels.md).

The Vendor view panel is displayed only if you are viewing a purchased
part in the Material Planning Window. To access the Vendor view panel,
select View, Vendor View
Panel. Information for the default vendor for the selected
part is displayed. Totals shown are for the parent entity of the site
specified in the header. For information about the Vendor view panel
chart, see [Vendor View Panel
Chart](Vendor_View_Panel_Chart.md).

To access the Part Sites panel, select View,
Part Sites. The sites assigned to the part
are displayed.
