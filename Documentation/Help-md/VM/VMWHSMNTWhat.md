# What is Warehouse Maintenance?

Warehouse Maintenance provides functions to create,
edit, and delete warehouse and warehouse location information. You
must create at least one warehouse for the system to use during inventory
transaction activities. You should do this during implementation and
before you begin entering your parts through Part Maintenance.
