# Order Management Window Buttons and their Function

|  |  |
| --- | --- |
|  | Save Button |

This button, located to the immediate right of the View field, allows
you to save a personalized view of the Order Management Window. For
example, if you want to save a view of four cascaded or tiled windows,
enter a unique view name in the View field and click this Save
button.

|  |
| --- |
| Delete Button |

The Delete button on the Order Management
Window deletes View properties you previously set up. When you click
the Delete button, you are prompted to confirm
the deletion of the View.

|  |
| --- |
| Refresh Button |

Rather than waiting for the specified period of time between system
refreshes to elapse, you can refresh the current display by clicking
the Refresh icon, located under the toolbar.

|  |
| --- |
| Options Button |

The options feature allows you to specify refresh, customer, and
part preferences. From the Tools menu, select Options,
or click the Options icon located under
the toolbar in every child window.
