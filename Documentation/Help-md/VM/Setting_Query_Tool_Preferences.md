# Setting Query Tool Preferences

Define your display preferences for the Database Tree shown in the
Query Tool.

To set your preferences from within the Query Tool window:

1. Select Admin,
   Query Tool.
2. Select Options,
   Database Tree Display Options.
3. Select the Show
   data type next to column name check box to display the
   data type and size associated with each column. The data type
   and size are displayed in parentheses after the column name. If
   you do not select this check box, the data type and size are not
   displayed.

Knowing the data type and size is useful
both for SQL command usage and for improving database query results.

4. Click OK
   to save your preference, or Cancel to
   exit without a changed display. The Database Tables and Columns
   panel is refreshed with your updated display preference.
