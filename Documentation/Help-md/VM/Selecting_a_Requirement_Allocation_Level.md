# Selecting a Requirement Allocation Level

Select a Requirement Allocation level by selecting
the appropriate level of allocation support for work order requirements:
None, Full, or Partial.

Full Allocation  This selection requires
you to make sure that work order requirements have allocated supply
before you firm or release the requirement.

Partial Allocation  This selection allows
you to allocate supply to the work order requirement, but makes allocation
optional.
