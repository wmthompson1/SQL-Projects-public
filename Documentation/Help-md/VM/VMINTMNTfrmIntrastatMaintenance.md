# Specifying Additional Intrastat Information

Use the Additional Intrastat Maintenance window to
specify any additional intrastat information you want to appear on
your reports.

To set up addition Intrastat information:

1. From the Inventory menu
   of the Infor VISUAL main screen, select Intrastat
   Maintenance.

   |  |  |
   | --- | --- |
   | POSTIT.gif | For this option to appear on the Inventory menu, Intrastat must be enabled in [Accounting Entity Maintenance](VMAENMNTfrmAccountingEntity.htm). You may need to restart VISUAL to view the menu option. |

The Additional Intrastat Maintenance window
appears.

2. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site to view. If you are licensed to use a single site, this field
   is unavailable.
3. In the Movement Type
   section, select the type of movement, either Dispatch or Arrival, to which this line refers.
4. In the Date Range, use
   the calendar buttons to select the range of dates that apply to
   this line.
5. Click the Insert
   Row button on the toolbar.
6. Enter the following
   information in the appropriate columns:

Reference - A
short description or reference.

Movement Date
- Date on which this movement takes place.

Tariff Code / Commodity
Code - Double-click the column header and select the Tariff
or Commodity code.

Excise Price -
The Excise charge.

Delivery Terms
- Click the arrow in the row to select the Terms of the delivery.

Nature of Trans
- Double-click the column header and select the type of transaction
to which this line refers.

Mode of Transport
- Double-click the column header and select the type of transportation
used.

Net Mass - Weight
of the shipment.

Supplement Unit
- Number of units.

To Country - Country
to which the shipment is going.

Orig Country -
Country of origin.

From Country -
Country from which this shipment is sent.

Region - Region
of the country from which this shipment is sent.

Port of Arrival
- Name of the port in which this shipment first arrived in the country.

Port of Transshipment
- Intermediate country used between shipping and receiving.

Number of consignments
- Number of agents selling items in the shipment for the shipper.

7. Click the Save
   button on the toolbar.
