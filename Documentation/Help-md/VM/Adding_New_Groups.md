# Adding New Groups

You can add new groups to your Personal Menu. You
can use this feature to combine in one group the applications you
use most.

1. Select File,
   Personal Menu from the main VISUAL menu.
2. Right-click and select
   Customize.
3. With the customize dialog
   box open, click the Add New Group button.
4. In the Group Caption
   field, enter a caption for the group.
5. If you want to select
   a different button to appear next to the group name, click the
   Button arrow and select the icon you
   want to use.
6. Click the OK
   button.
7. On the Customize window,
   click the Apply button.
8. Add commands to your
   new group by selecting the commands you want in the right pane
   and clicking the Add Selection button.
9. After setting up your
   new group, click the OK button.
