# Viewing Test Step results

You can view the results of test step definitions
in Test Definition.

1. Highlight the test step
   line item for which you want to view the test results.
2. Click the Results
   button.

The Test Step Results dialog box for the
test appears.

You can click the Exceptions
button to view test step exceptions. You can also view test notes,
subgroup notes, result notes capabilities, exceptions, XBar, and graphs
and histograms of test results by clicking on the appropriate buttons.
