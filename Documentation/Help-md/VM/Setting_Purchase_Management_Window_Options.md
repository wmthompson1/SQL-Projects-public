# Setting Purchase Management Window Options

Using the Options window, you can select options for
the following:

Colors You can set color options for
your Vendor Rating and Receive Statuses.

Part History You can set your initial
view and graph options.

Vendor Comparison You can set up your
initial view and graph options

## Setting Color Options

To enable you to quickly discern the ratings of your
vendors in the Vendor History child window, set different colors for
the various rankings of your vendors on-time performance.

To set up your color options:

1. Select the Options
   option from the Tools menu.

The Options dialog opens.

2. Click the Colors
   tab.
3. In the Vendor Rating
   section, enter the Lower percentage values you want to use from
   Best, Good, Bad, and Worst. As you enter each value, VISUAL enters
   the next Upper % value. For example, enter 90 in the Best Lower
   % field and VISUAL enters 89 in the Good Upper % field when you
   move to another field.
4. Click the Color
   browse button and select the color you want to use for the rating.
5. In the Color dialog
   box, click the OK button.

The Color dialog box closes and the color
appears in the Colors tab.

6. In the Apply Rating
   Colors To section, select to what you want to apply the color.
   You can select Text or Background.
7. In the Receive Status
   Colors section, click the Color browse
   button and select the colors you want to use for your received
   statuses. You can set colors for:
8. Fully Received
9. Partially
   Received
10. Not Received

8. In the Apply Status
   Colors To section, select to what you want to apply the color.
   You can select Text or Background.
9. If you want to enable
   a different color bold text or background, in the Highlight Colors
   section, select the Enable Text or Enable Background check boxes, and select
   the color you want to use.
10. If you want to save
    your settings as your default settings, select the Save
    as Default check box.
11. If you want to close
    the Options dialog and apply the changes, click the Ok
    button.

If you want to apply your changes without
closing the Options dialog, click the Apply
button.

## Setting Up Part History Options

To set up your Part History options:

1. Select the Options
   option from the Tools menu.

The Options dialog opens.

2. Click the Part
   History tab.
3. In the Initial View
   section, select what view you want to appear when you first select
   a Part ID in the Part History child window. You can select either
   Graph or Data.
4. In the Periods Per Page
   field, enter the number of periods you want to appear when you
   first view a Part History graph.
5. If you want VISUAL to
   apply the same color as you select for the first point to all
   of your data points, select the Apply Color
   Ones Color To All check box.
6. Using the Graph Controls,
   set up how you want your Part History graph to appear.
7. If you want to save
   the settings as your default Part History settings, select the
   Save as Default check box.
8. If you want to close
   the Options dialog and apply the changes, click the Ok
   button.

If you want to apply your changes without
closing the Options dialog, click the Apply
button.

## Setting Up Vendor Comparison Options

To set up your Part History options:

1. Select the Options
   option from the Tools menu.

The Options dialog opens.

2. Click the Vendor
   Comparison tab.
3. In the Initial View
   section, select what view you want to appear when you first select
   a Part ID in the Part History child window. You can select either
   Graph or Data.
4. In the Periods Per Page
   field, enter the number of periods you want to appear when you
   first view a Part History graph.
5. In the Vendors Per Page
   field, enter the numbers of vendors you want to appear for each
   period.
6. Using the Graph Controls,
   set up how you want your Part History graph to appear.
7. If you want to save
   the settings as your default vendor comparison settings, select
   the Save as Default check box.
8. If you want to close
   the Options dialog and apply the changes, click the Ok
   button.

If you want to apply your changes without
closing the Options dialog, click the Apply
button.
