# Copying a Data Source

You can create a new data source by copying an existing
data source. You can copy custom and vendor data sources. If you copy
a vendor data source, the new data source has a scope of custom.

1. Select File,
   Dashboard.
2. Select Edit,
   Edit Data Source.
3. In the Data Source ID
   field, select the data source.
4. Click Copy.
5. Specify the name of
   the new data source.
6. Click Ok.
