# Setting Program Preferences in Application Global Maintenance

Use the Program Preferences dialog in Application
Global Maintenance to specify the tab that is displayed when you access
Application Global Maintenance and to specify your preferred customer
order entry and purchase order entry program.

You can access purchase order entry and customer order entry from
several windows in the system. For example, you can open Purchase
Order Entry from the Manufacturing Window. When you open Purchase
Order Entry from the Manufacturing Window, the program you specify
as the preferred purchase order entry program is opened.

If a purchase order entry or customer order entry window is already
open when you access a purchase order or customer order from another
application, then the window that is already open is used to display
the order. The preferences that you specified in Application Global
Maintenance are ignored.

1. Select Admin,
   Application Global Maintenance.
2. Select Options,
   Preferences.
3. Specify this information:

Default Tab
Click the drop -down arrow and select the tab you want to appear when
you first open the Application Global Maintenance program.

By selecting the following settings, you
can set the requirement that VISUAL uses the Purchase Management Window
or Order management window when viewing order history. For example,
if you are in Part Maintenance and view a parts purchase history,
you can double-click the PO ID column and VISUAL opens either the
Purchase Management Window or the Purchase Order Entry window depending
on the setting.

Preferred C/O Entry Program
Click the arrow and select the program you prefer to use for Customer
Order entry. You can select either the Customer
Order Entry program or the Order Management
program.

Preferred P/O Entry Program
Click the arrow and select the program you prefer to use for Purchase
Order entry. You can select either the Purchase
Order Entry program or the Purchase Management
Window.

4. Click Done.
