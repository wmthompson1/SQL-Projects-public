# Setting Up Information in Direct Cost G/L Mapping and Burden Mapping Maintenance

Use the Direct Cost G/L Mapping and Burden Mapping Maintenance window
to set up the following information:

* Business Lines
* Project Types (you can also
  access Project Types from the Project Maintenance window)
* Department Codes (you can
  also access Department Codes from Application Global Maintenance)
* Cost Categories
* Cost Groups
* Burden Categories
* Burden Groups
* General Ledger Burden Mappings

Use the GL/Burden mapping window to specify the accounts you want
to use for which departments and the burdens you want VISUAL to use
for its calculations.

To access the Direct Cost G/L Mapping and Burden Mapping Maintenance
window, select Project, G/L Burden Mapping Maintenance.

The Direct Cost G/L Mapping and Burden Mapping Maintenance window
appears.
