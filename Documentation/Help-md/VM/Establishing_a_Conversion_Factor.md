# Establishing a Conversion Factor

The conversion factor is what establishes the ability
of the system to convert from a receiving or purchasing unit to a
stocking unit.

1. Select the From (upper)
   unit of measure you are converting.
2. In the lower table,
   select the unit of measure To which you want to convert.

3. Move the cursor to the
   Conversion Factor field and enter the conversion factor.

Remember that you are converting the Upper
unit to the Lower unit. In the example below, a conversion factor
of 0.3937 is entered to convert inches to centimeters.

Note that you can covert units in both ways,
but the conversion factors are not always the same. For example, converting
feet to inches is not the same as converting inches to feet.

4. Click the Save
   button to make the conversion factor permanent.

To view established conversion factors in
the future, select the upper and lower units for which you want to
see the conversion factor.
