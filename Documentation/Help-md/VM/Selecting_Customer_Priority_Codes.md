# Selecting Customer Priority Codes

With the appropriate customer in the Customer ID field,
click the Order Management tab.

1. Click the Priority
   Code button.

The Customer Priorities dialog box appears.

Enter the information that appears here
in Application Global Maintenance. Click here for more information.

2. Highlight the appropriate
   line and click Ok.

The Priority Code appears in the Priority
Code field.

The Disbursement fill rate, a value called
"Fill Rate" in the Priority Code line item table, also appears
in the Disbursement fill rate field. This value specifies how much
of the available quantity in a warehouse is available to the customer
at the time of order fulfillment. For example, if a customer fill
rate is 100, VISUAL allows the customer access to 100% of the available
stock; if the customer fill rate is 70, VISUAL allows the customer
access to 70% of available stock. VISUAL uses this rate when it figures
Quantity Available to Customer in the VISUAL Order Management Window.
