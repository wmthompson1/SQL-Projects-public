# Print Formats Part Trace Maintenance

The format of Traceability Reports and Labels can
be modified using Centura Report Builder. The following QRP report
files apply:

|  |  |
| --- | --- |
| Transaction Report | VTTRCRP1.QRP |
| Location Report | VTTRCRP2.QRP |
| Part Trace Labels |  |
| Standard | VMTRCLBL.QRP |
| User Defined 1 | VMTRCUD1.QRP |
| User Defined 2 | VMTRCUD2.QRP |
| User Defined 3 | VMTRCUD3.QRP |
