# Making Manual Changes to ETCs

If you want to manually make ETC adjustments, double-click
the project summary line you want to adjust.

For labor cost categories, enter the hours you want in the appropriate
month. VISUAL changes the labor cost and associated applied overhead
accordingly. Adjusting costs (labor, material, or ODC) causes VISUAL
to adjust the associated applied overhead accordingly. You can also
adjust each burden category amount as you need.
