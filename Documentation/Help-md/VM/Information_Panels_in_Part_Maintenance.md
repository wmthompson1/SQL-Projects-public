# In-context View Panels in Part Maintenance

You can view two information panels in Part Maintenance: the Part
Panel and Part Sites panel.

For information about the Part view panel chart, see [Part
View Panel Chart](Part_View_Panel_Chart.md). For information about how to use the Part view
panel, see [In-context View Panels](In-context_View_Panels.md).

To access the Part Sites panel, select View,
Part Sites. See [Using
the Sites for Part Panel](Using_the_Part_Sites_Panel.md).
