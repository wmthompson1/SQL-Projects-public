# Modifying Projects

If you copy projects from templates or other projects
or VISUAL masters, you will need to modify them to meet the needs
of the project or contract on which you are working.

If you modify projects and want to keep the original project information
for your records, VISUAL provides a version control mechanism to help
you track your changes.
