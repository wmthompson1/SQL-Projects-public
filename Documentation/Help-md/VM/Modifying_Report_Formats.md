# Modifying Report Formats

You can modify the format of Inventory Transactions
Entry reports using Centura Report Builder. The following QRP report
files apply:

## Print Inventory Transactions

|  |  |
| --- | --- |
| By Transaction ID | VMINVRP1.QRP |
| By Part ID | VMINVRP2.QRP |
| By Customer Order ID | VMINVRP3.QRP |
| By Purchase Order ID | VMINVRP4.QRP |
| By Work Order ID | VMINVRP5.QRP |
| By Warehouse/Location | VMINVRP6.QRP |
