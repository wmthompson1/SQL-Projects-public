# Forecasts

You can also use AMP to build multiple forecasts and
display each forecast separately in the main AMP window. Build forecasts
separately from the Master Production Schedule (MPS).

You can build the MPS using a single forecast or multiple forecasts,
customer demand, independent demand and gross demand.
