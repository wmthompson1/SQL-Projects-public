# Filtering the RMA Line Item Table

After you create an RMA, you can filter the RMA line
item table to display only the customer order lines for which you
have authorized return quantities. To display only lines with authorized
return quantities, select Options, Show Authorized Lines Only until a check mark
is placed next to the menu item. To show all lines in the customer
order, select Options, Show
Authorized Lines Only until the check mark is removed from
the menu item.

You can filter the line item table for Credit, Repair, and Replacement
RMAs that were not converted from an Evaluation RMA.
