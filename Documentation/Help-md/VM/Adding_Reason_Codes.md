# Adding Adjustment Reason Codes

Adjustment Reason Codes allow you to classify common
inventory adjustment transactions by type and apply to the transfer
of goods between customers and 3PL customers involved in the distribution
process.

1. Select Admin,
   Application Global Maintenance.
2. Select Maintain,
   Reasons Maintenance, Adjustment
   Reason Table.
3. Click Insert.
4. Specify this information:

ID Specify an
ID for code. IDs must be unique.

Description
Specify a description for the code.

G/L Account ID
- Click the browse button and select a general ledger account to apply
adjustment costs to when this reason code is specified.

5. Click Save.
