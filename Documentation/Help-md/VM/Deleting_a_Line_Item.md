# Deleting a Line Item

1. With the appropriate
   request for quote is open in the Vendor RFQ Entry window, highlight
   the line item that you want to delete.
2. Click the Delete
   button.

An X appears in the row header of the item.

3. Click Save
   to remove the item and update the quotation.
