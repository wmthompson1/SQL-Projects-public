# What is Customer Inquiry?

The Customer Inquiry window provides an easy and versatile
way to view different types of database information about your customer
orders. You can view Order IDs, Customer Part IDs, Invoice IDs, Quote
IDs, and others.

To make the inquiry easier to use and more specific, several search
tools are included. As with other search tools used through out the
system, you can perform searches using a wide variety of options.
