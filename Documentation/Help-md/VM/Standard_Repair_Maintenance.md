# Standard Repair Maintenance

Standard repairs are repairs that you perform routinely.
Enter standard repair types into your database so you dont have to
reenter the same information each time the need for the repair arises.

1. Select Maintain,
   Standard Repairs.
2. Click the Insert
   button to add a new row to the table.
3. Enter information in
   the following fields:

Eng ID - Double-click
the column header to select an Engineer ID for the repair. VISUAL
assigns all standard repairs an Engineer ID of 0.

Part ID - The
Part ID of the standard repair. You cannot select a Part ID other
than the one that appears when you select an Engineer ID.

Equip Type - Double-click
the column header to select an Equipment Type for the repair. Enter
equipment types in Equipment Maintenance. Because equipment models
contain equipment types as part of their profile, you can skip this
step if you know which model you want to assign to the standard repair.

Equip Model -
Double-click the column header to select an Equipment Model for the
repair. You can select an equipment type and model with one step by
selecting an equipment model. Model IDs contain an equipment type.
Enter equipment models in Equipment Maintenance.

Fault Type - Double-click
the column header to select a fault type for the standard repair.
Because fault codes contain fault types as part of their profile,
you can skip this step if you know which fault code you want to assign
to the standard repair.

Fault Code - Double-click
the column header to select a fault code for the standard repair.
Any fault code selection overrides the fault type, if the fault type
is different than the fault type in the fault code.

4. Click the Save
   button to commit the information and save the standard repair
   type.
5. Click the Close
   button to return to the Unplanned Maintenance window.
