# About the Customer Backlog Report Window

The Customer Backlog report prints information on
all partially or totally unshipped orders, including those with status
of Firm, Released, or On Hold. Total revenue value of the unshipped
portion of each order is shown, along with totals for each sequence
group.

## Starting the Customer Bookings Report Window

Select Sales, Customer
Backorder Report.

## Fields

Site
ID(s)

If you are licensed to use multiple sites, click the arrow and select
the sites to include in the report. If you are licensed to use a single
site, this field is unavailable.

Starting Due Date
and Ending Due Date

The date range for the report. Dates are based on the Desired Ship
Date that is specified on the customer order.

Starting ID and Ending ID

The label for these fields changes based on the sequence option
that you select. For example, if you select Customer ID in the Sequence
section, the field labels are Starting Customer ID and Ending Customer
ID. Use these fields to print the report for a range of IDs or records.
To include all IDs in the report, leave the fields blank. To print
the report for one ID, specify the same ID in the Starting ID and
Ending ID fields.

If you click Due date in the Sequence section, these fields are
not displayed.

Report Currency

The currency for values in the report. Select \*\*
Native \*\* to view amounts in each order's currency. If you
select \*\* Native \*\*, grand totals are not
provided. If you select sites in more than one accounting entity,
then the currency list is filtered to show the currencies that the
entities share.

Report
output

Select the output format for the report. See [Report
Output Options](Report_Output_Options.htm).

Suppress
amount

Select this check box to omit monetary amounts on the report.

Order
Amount Based On

If you clear the Suppress amount check box and also click Order
ID, Customer ID, or Customer Name in the Sequence section, specify
how to value the orders. Click Qty to be shipped to value the orders
based on the sales prices of the outstanding items. Click Value uninvoiced
to value the orders based on the amount remaining to be invoiced.

Sequence

Specify how to sort the report. For example, if you click Part ID,
the report shows customer orders for each part ID. If you selected
the Print Month to date/Year to date summary or Print Month to date
summary check boxes, you can sort the report by Customer ID, Customer
Name, or None.
