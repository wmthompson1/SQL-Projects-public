# Creating a Leadtime Process Template

To create a template:

1. Select Inventory,
   Part Maintenance.
2. Select Maintain,
   Process Type.
3. Select File,
   Create Template.
4. To customize the template,
   edit the description and process steps. Do not edit the Process
   Type field. The Process Type must be TEMPLATE.
5. Click Save.
