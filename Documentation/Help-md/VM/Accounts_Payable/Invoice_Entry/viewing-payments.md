# Viewing Payments

Use the Payments/Disbursements inquiry to view vendor payments, check run details, and reconcile to bank records.

Common tasks:

1. Filter by `Vendor ID`, `Date Range`, `Payment Method`, or `Batch ID`.
2. Review payment status (Issued, Cleared, Voided) and reference numbers.
3. Drill into a payment to see applied vouchers and any unapplied debit/credit adjustments.

Bank reconciliation tips:

- Export payment lists (CSV) for the bank reconciliation tool or accounting system.
- Match cleared checks/ACH items by reference number and amount; flag any discrepancies for investigation.
