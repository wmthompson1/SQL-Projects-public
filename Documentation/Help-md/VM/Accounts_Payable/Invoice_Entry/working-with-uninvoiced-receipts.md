# Working with Uninvoiced Receipts

Uninvoiced receipts are goods receipts that have not yet been matched to vendor invoices. Process them to avoid orphan cost items.

Steps to identify and process:

1. Run the `Uninvoiced Receipts` inquiry or report and filter by `PO`, `Receipt Date`, or `Vendor`.
2. For each receipt, determine whether an invoice is expected and whether it should be created from the receipt (see `Creating A/P Invoices from Purchase Order Receipts`).
3. If no invoice is expected (internal transfer, sample), document the reason and close the receipt as per policy.
4. If an invoice is expected but missing, contact the vendor and place a hold or create a voucher for the expected amount if policy allows.

Notes:

- Aging of uninvoiced receipts should be monitored; long-uninvoiced receipts may indicate missing vendor invoices or process gaps.
