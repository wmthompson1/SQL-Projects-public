# Manually Entering an A/P Voucher

This page covers manual voucher entry (three-level hierarchy under Invoice Entry → Vouchers).

Steps:

1. Open `Invoice Entry → Vouchers → New Voucher`.
2. Select `Vendor ID`, `Site ID`, and enter `Voucher ID` (or leave blank if auto-numbered).
3. Add line items or select matching receipts to attach to the voucher.
4. Verify GL distributions and totals, then save. Optionally print or post according to workflow.
