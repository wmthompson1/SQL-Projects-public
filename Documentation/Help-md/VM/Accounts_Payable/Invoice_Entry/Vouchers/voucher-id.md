# Voucher ID

If you are paying against an invoice, enter the Voucher ID of that invoice in this field. You must create the Voucher ID if it does not already exist. The Voucher ID is not a required field if you are not paying against an invoice.

## Voucher ID Button

Click the Voucher ID button to browse existing Voucher IDs. The Voucher ID is a unique identifier assigned to the transaction on the current window. You can use the Auto Numbering feature to set up automatic numbering for vouchers; when auto-numbering is enabled you may leave the Voucher ID blank during entry and a number will be assigned when you save the transaction.

## Voucher Notes Button

Voucher notes are associated with the voucher as a whole and can be used for internal comments relevant to the voucher. Access Voucher Notes by pressing the Voucher Notes button or selecting `Voucher Notations` from the Notes menu.
