# Posting A/P Vouchers to the G/L

Posting transfers voucher amounts to the General Ledger. Follow your cycle (daily vs batch) and period close rules.

Posting checklist:

1. Confirm all vouchers included in the posting run are complete and approved.
2. Reconcile voucher totals with the A/P trial balance and the batch control total.
3. Select the posting date and ensure it maps to the correct accounting period (do not post to a closed period).
4. Execute `Post` or `Post Batch` and capture the posting log or journal reference number.
5. Verify the corresponding GL entries and reconcile the A/P control account.

Post-post activities:

- Run an A/P trial balance and confirm totals match the GL control account.
- Investigate any GL/AR discrepancies immediately and follow reversal/adjustment procedures if required.
