# Creating a Payment

Create payments to remit vendor balances via check, ACH, or other payment methods.

Steps to create a payment batch:

1. Gather approved vouchers to pay and create a payment batch (by date, vendor group, or payment run).
2. Select payment method (Check, ACH, Wire) and configure bank account and payment date.
3. Generate a proposed payment file and preview totals; resolve any vendor holds or exceptions.
4. Print checks or export ACH file per bank format; mark payments as Issued when completed.
5. Update payment statuses once cleared by the bank.

Notes:

- Use payment proposal tools to net multiple vouchers for the same vendor into a single payment.
- Keep a copy of the payment file and reconciliation report for audit.
