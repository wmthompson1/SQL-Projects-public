# Adding Line Items Not Associated With the Purchase Order

Sometimes you must add charges (freight, handling, adjustments) that are not present on the originating PO. Use this procedure to ensure proper accounting.

Steps:

1. In the voucher or invoice line area, choose `Add Misc Line` (or similar) and select the appropriate GL account or distribution.
2. Enter a clear description (e.g., "Freight - carrier XYZ"), the amount, and any tax code if applicable.
3. If the amount should be allocated to multiple cost centers, use the distribution split feature to allocate portions accordingly.
4. Save and ensure totals match the voucher header before posting.

Notes:

- Avoid adding non-PO items without proper supporting documentation and approval.
- Track such items using a consistent description standard to aid future audits and reporting.
