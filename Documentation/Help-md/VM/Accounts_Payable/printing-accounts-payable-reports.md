# Printing Accounts Payable Reports

Common A/P reports and how to print/export them:

- **A/P Aging** — shows vendor balances by aging bucket; print monthly for vendor review and cash planning.
- **Open Vouchers** — list of unpaid vouchers; useful before payment runs to confirm payables.
- **Payment Register** — shows payments issued for a period or batch; required for bank reconciliation.
- **Vendor Activity** — transactions for a vendor over a date range.

Printing and export tips:

- Use the application `Reports` menu to select filters and output format (PDF, CSV).
- For large datasets or scheduled reports, use the `SQL_Reports` queries in the repository and run them in the reporting database or a BI tool.
