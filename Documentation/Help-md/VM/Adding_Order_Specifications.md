# Adding Order Specifications

Order specifications are specific to an individual
customer order. There can only be one specification per order.

1. With the customer order
   open in the Customer Order Entry window, click the Order
   Specs button.

The Order Spec dialog box appears.

2. Enter the information
   for the customer order.

You can enter an unlimited amount of information.

3. Use the Paste
   button to paste text that you copied from another location into
   the box.
4. Use the Copy
   button to copy text in the dialog box to the clipboard.

The entire contents of the dialog box are
copied to the clipboard. If you are pasting information Into the dialog
box, The current text of the dialog box is replaced with the incoming
text. If you wish to paste without replacing the entire text, use
the Shift-Insert key combination.

5. Click Close
   to close the specification dialog box and return to the Customer
   Order Entry window.
