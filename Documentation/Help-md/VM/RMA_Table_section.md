# RMA Table section

Depending on your table preferences the following
columns may be available:

* [Ln
  #](javascript:TextPopup(this))

  The system numbers each line consecutively. If lines are
  deleted you can select the Renumber Lines option from the
  Edit menu.
* [New
  Order](javascript:TextPopup(this))

  If you are creating a new order for this RMA, this is the
  line number for the item in the new order.
* Quantity Ordered
* [Shipped
  Qty](javascript:TextPopup(this))

  The original quantity shipped to the customer.
* [Rtn
  Qty Authorized](javascript:TextPopup(this))

  The quantity that your company has authorized the customer
  to return.
* Quantity Returned
* Quantity Re-shipped
* [Job
  ID](javascript:TextPopup(this))

  The original orders Job ID for this item.
* [Lot
  ID](javascript:TextPopup(this))

  The original orders Lot ID for this item.
* [Split
  ID](javascript:TextPopup(this))

  The original orders Split ID for this item.
* Part ID
* [Service
  Charge ID](javascript:TextPopup(this))

  The original orders service charge for this item, if applicable.
* [Misc
  Ref/Descr](javascript:TextPopup(this))

  A field reference for or a description of the item from
  the original order.
* [U/M](javascript:TextPopup(this))

  The selling unit of measure for this item.
* [Unit
  Price](javascript:TextPopup(this))

  The unit price for the item. This automatically defaults
  to the price set for that customer or discount code. If you
  do not have anything set up for this customer, the Unit Price
  defaults to the unit price in Part Maintenance.
* [Disc
  %](javascript:TextPopup(this))

  The discount percent to be applied to this item. If you
  specified a Discount Code for this item, this column defaults
  to the discount percentage for that code.
* [Extension](javascript:TextPopup(this))

  The extended price for the item, computed by: (Unit Price
  - Unit Price x Disc%/100) x Quantity.
* [Ship
  Date](javascript:TextPopup(this))

  The original ship date for this item.
* Promise Date
* [Freight](javascript:TextPopup(this))

  The freight charges for this item.
* [Comm%](javascript:TextPopup(this))

  The default commission percentage for the
  Sales Rep for this order.
* [Tax
  Group ID](javascript:TextPopup(this))

  If you specified a Product Code with a defined Sales Tax
  Group, the Tax Group ID automatically appears here.
* Product Code
* [Commodity
  Code](javascript:TextPopup(this))

  If a Commodity Code exists for this item,
  it appears in this column.
* [Revenue
  Account ID](javascript:TextPopup(this))

  The G/L Revenue Account ID associated with this line item.
* [Revenue
  Acct Descr](javascript:TextPopup(this))

  A description of the Revenue Account ID.
* [Cust
  Part ID](javascript:TextPopup(this))

  A Customer Part ID for this item, if applicable.
* [Drawing
  ID](javascript:TextPopup(this))

  The Drawing ID for this item, from the work order.
* [Rev](javascript:TextPopup(this))

  The drawing revision number for this item.
* [Ln
  Closed Short](javascript:TextPopup(this))

  This check box indicates that the line VISUAL has closed
  this item without having received the entire order quantity.
* [Specs](javascript:TextPopup(this))

  This check box indicates that there are specs for this item.
  To view the specs, highlight the line, then lick on the Line
  Specs button. The Specs dialog box appears.
* Piece Tracked.
