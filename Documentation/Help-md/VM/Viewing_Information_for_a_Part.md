# Viewing Information for a Part

After you select a part, click the Part
Info tab. The Part Info tab contains the following read-only
information:

The Part Info tab contains the following read-only information:

Order Policy - The order policy for the
part that you set in Part Maintenance. One of these appears: Not Planned,
Discreet, Period Supply - # of days, Fixed EOQ, Master Schedule. See
Part Maintenance for more information.

Leadtime - The number of days that you
need to replenish the part. You set this value in Part Maintenance.

Safety - Extra amount of a part that you
plan to keep in excess of real demand to cover fluctuations in demand
caused by unplanned orders, scrap, or other events.

Total Unit Cost - The material cost of
the part. See Part Maintenance for more information.
