# Viewing & Setting Customer Terms and Currency IDs in the Monetary Tab

1. With the Customer ID
   for whom you want to view and set default Terms information appearing
   in the Customer ID field, click the Monetary
   tab.

The customers default terms appear in the
field; the customers currency appears beneath the field.

2. Click the Terms
   button to edit customer information.

The Terms dialog box appears.

3. When you are finished
   setting Customer Term information, click the Ok
   button to commit the settings.
4. If the customer requires
   a currency change, click the arrow and select a Currency ID from
   the list box.
