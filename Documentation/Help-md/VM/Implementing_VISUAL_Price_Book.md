# Implementing VISUAL Price Book

The structure of the VISUAL Price Book menus and interface
makes implementation of VPB very intuitive. Following the structure
of the navigation pane, the files you must configure are presented
in a logical order. The foundation of VISUAL Price Book is the Price
matrix. In order to make VISUAL Price Book as flexible and easy to
use as possible, there are several "entry points" to the
price matrix. This is to provide access to the price matrix from as
many directions as possible.

After you set up the price matrix, typically you would configure
Discounts and Commissions. Because of the flexibility of VISUAL Price
Book, you may want to customize the files you need to configure, suiting
them to your particular needs.

Following the configuration of the various matrices, you must set
up and maintain formulas. There are several standard pricing models
available to you during formula creation, through a series of formula
templates. Templates allow for quick programmability of VISUAL Price
Book for common pricing formulas. These templates allow for quick
and easy customizability. You can, of course, create formulas from
scratch.
