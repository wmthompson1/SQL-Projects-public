# Setting a Default IBT Status

You can select the default status of an IBT: Firmed or Released.

1. From the Options menu,
   select Preferences.
2. Select either the Firm radio button or the Release
   radio button.
3. Click the Ok
   button when you are finished.

VISUAL sets your default status.

Each new Interbranch transfer you create
now has this as its default status.
