# Resetting Part Information in Material Requirements and Work Orders

1. Either with a Part ID
   displaying in the Part ID field of the Part Maintenance window
   or no part currently displaying, select Reset
   Part Information from the Maintain menu.

The Reset Part Info dialog box appears.

See [Resetting
Part Information](Resetting_Part_Information_Master.htm) for more information on setting general part
reset options.

The Requirements
tab appears by default.

2. Select the Engineering
   Master Material Requirements box to reset information only
   for parts/material requirements as they appear in Engineering
   Masters.

Select the Work Order
Material Requirements box to reset information only for parts/material
requirements as they appear in Work Orders.

Select the Quote Master
Material Requirements box to reset information only for parts/material
requirements as they appear in Quote Masters.

3. Select the Include
   WO Headers and Sub IDs box to include work order header
   cards and Sub IDs in the part information reset. VISUAL only updates
   work order headers and Sub IDs if you select a type of requirement.
4. Select the According
   to Material Requirement Status box to select which material
   requirement statuses you want the part information update to affect:
   Unreleased, Firmed,
   Released, Closed,
   Cancelled, or All.
   Skip this step if you want the update to be date-driven only.
   See next step.

You can select any combination of these
options; selecting All makes the previous
five options unavailable.

5. Select the According
   to Work Order or Master Creation Date box to activate the
   date fields. Enter dates in the On or After
   or On or Before fields. Skip this step
   if you want the update to be status-driven only. See previous
   step.
6. Click the Ok
   button to begin the part information update process.
