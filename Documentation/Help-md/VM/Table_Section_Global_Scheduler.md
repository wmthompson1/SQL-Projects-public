# Table Section Global Scheduler

Depending on your table preferences the following
columns may be available:

* [Schedule
  ID](javascript:TextPopup(this))

  This is a unique identifier that has been assigned to a
  schedule. You may have multiple Schedule IDs. You can have
  multiple "what-if" schedules in addition to your
  standard production schedule.
* [Infinite
  Capacity](javascript:TextPopup(this))

  Capacity is expressed as the number of simultaneous operations
  which can be worked on at a given resource. For instance,
  if you have 5 horizontal milling machines called resource
  ID MILLING you would set its capacity to 5 for each shift
  that you have those machines available. Each shift may have
  a different capacity.
* [Materials
  Checked](javascript:TextPopup(this))

  There are two possible responses allowed in this data field:

  Yes - When scheduling, check the
  material availability for a specified schedule.

  No - When scheduling, without
  checking the material availability for a specified schedule.
* [Services
  Checked](javascript:TextPopup(this))

  This check box is checked if the Scheduler checks the availability
  of services.
* [Hard
  Release](javascript:TextPopup(this))

  Select this check box if you want VISUAL to schedule all
  the work order in this Schedule ID with hard release dates.
  See [Setting
  the Global Hard Release Dates Preferences](Setting_the_Global_Hard_Release_Dates_Preference.htm).
* [Active](javascript:TextPopup(this))

  If this box is not checked then the employee is considered
  inactive. You cannot enter labor against an inactive employee
  ID. Also, you may optionally suppress inactive employees for
  the Employee Report available through Employee Maintenance.

  The default setting is "active".
* [Run
  Start Date](javascript:TextPopup(this))

  This displays date and time information about this schedule
  the last time it was run. That is, the start date and time
  is displayed. The information may help you determine any future
  time allotment that should be made.
* [Run
  End Date](javascript:TextPopup(this))

  This displays date and time information about this schedule
  the last time it was run. That is, the end date and time is
  displayed. The information may help you determine any future
  time allotment that should be made.
* [Schedule
  As of Date](javascript:TextPopup(this))

  This displays date and time information about this schedule
  the last time it was run. That is, the schedule as of date
  and time is displayed. The information may help you determine
  any future time allotment that should be made.
* [Work
  Orders](javascript:TextPopup(this))

  The number of work orders scheduled the last time it was
  run is displayed here.
* [Operations](javascript:TextPopup(this))

  The number of operations scheduled the last time it was
  run is displayed here.
* [Requirements](javascript:TextPopup(this))

  The number of material requirements scheduled the last time
  it was run is displayed here.
