# Maintaining Markups

Maintain Markups within VISUAL Price Book on the Markup
Matrix window. This configures the amount of rebate for materials
sold at your company.

To access the Markup Matrix window:

1. With VISUAL Price Book
   open, from the navigation pane, double-click the Markups icon.
   The Markup Matrix window appears. Depending on the settings you
   used on the Tools menu, your default
   values may vary.
2. From the Context drop-down
   list, select the markup context. Make your selection from: Commodity,
   Customer, Customer/Part, Customer Type, Customer/Product, Customer
   Type/Product, Market, Market/Part, Part, Part Group, or Product.
   The Context selection determines the values that appear in the
   Markup Matrix table.

If you have previously configured markups,
you can view the entire list by clicking the Browse button to the
left of the Context ID field. This opens a table that contains all
the existing markups.

3. In the Effectivity Date
   field, enter the date on which the Markup Matrix was created.
   VPB uses this information for filtering purposes. Do not confuse
   the Effectivity Date in the header with the Effective Date in
   the table. By default, VISUAL Price Book inserts the current system
   date. You can change this value manually by typing in the effectivity
   date manually, in Windows system format, or you can click the
   Calendar button and select the date from the pop-up calendar.
4. From the Break Type
   drop-down list, select the type of price break to add to the matrix.
   Make your selection between Amount (dollar value) and Quantity
   (numeric value).
5. Select the Show
   Effectivity check box if you want to see the Effectivity
   results on the Markup Matrix table. By default, VISUAL Price Book
   does not select this check box.
6. In the Currency ID field,
   enter the identification code for the currency for which you are
   creating the markup matrix. By default, VPB uses the default entity
   currency. You can enter the currency ID manually, or select it
   from the browse table by clicking the Browse button.
7. Depending on your Context
   selection, enter the appropriate value for the identification
   code. For example, if you select Part Context, you would enter
   the Part ID whose matrix table you want to view. If a cost matrix
   exists for the identification code, VISUAL populates the matrix
   table with the appropriate values.
8. Enter the appropriate
   information on the table. The necessary information varies based
   on the selections made in the header portion of the Markups Window.
   See "Adding Markup Matrix Table Information" for further
   information.
9. Click the Save
   button on the main VISUAL Price Book toolbar to save your changes.
