# Specifying a Drawing Number

If there is a specific drawing associated with this
operation, you can enter its drawing number and revision number in
the fields in the Other section of the Operation dialog box.

1. Enter the drawing number
   in the Drawing # field.
2. Enter the revision number
   in the Rev field.
3. Click the Save
   button to save the new information.
