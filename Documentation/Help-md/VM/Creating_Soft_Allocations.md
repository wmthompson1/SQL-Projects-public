# Creating Soft Allocations

You can execute Soft Allocations Manually any time
you want.

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site to use.
2. Select Create
   Soft Allocations from the File menu.
3. Click Yes
   to create the Soft Allocations.
