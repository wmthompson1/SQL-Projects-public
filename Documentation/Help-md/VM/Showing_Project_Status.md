# Showing Project Status

To show the status of each task within the project:

Select a Project ID in the Project Summary window and select an
Actuals summary.

From the Info Menu select the Show Project Status
option.

A secondary window appears showing, by WBS, all the work orders
and A/P invoices associated with the project.
