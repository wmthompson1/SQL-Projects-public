# Deleting Translations

You can delete individual translation strings or an
entire language pack.

## Deleting Translation Strings

You can delete a VISUAL string and its translation from a language
pack:

1. Select Admin,
   Translation Maintenance.
2. In the Language field,
   specify the language pack.
3. Select the row to delete.
4. Click the Delete
   Row button.
5. Click Save.

## Deleting a Language Pack

1. Select Admin,
   Translation Maintenance.
2. In the Language field,
   specify the language pack.
3. Click the Delete
   Language Pack button.
4. In the confirmation
   dialog, click Yes.
