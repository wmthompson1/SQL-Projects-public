# Setting Up Priority Codes

Priority codes represent the various levels of priority
you want to use for your projects. For the Aerospace and Defense industry,
you may want to enter your DCAA priority codes.

To set up Priority codes:

1. In Project Maintenance,
   click the Maintain menu and select Priority
   Table.

The Priority table appears.

2. Click the Insert
   button.

The next available row in the table appears
highlighted.

3. Enter the Code and Description
   you want to use.
4. Click the Save
   button.
5. When you have finished
   entering Priority codes, click the Close
   button.
