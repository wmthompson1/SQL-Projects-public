# Linking Co-products to New Work Orders

You can link co-products to new and existing work
orders. For new work orders, you can only specify the main part in
[Customer Order Entry](VMORDENTWhat.md) enter the co-products
to the new work order in the Manufacturing Window.

1. Select Customer
   Order Entry from the Tools menu
   of the Infor VISUAL Main window.

The Customer Order Entry window appears.

2. Enter the customer order
   ID in the Our Order ID field. You can also click the Our
   Order ID button and select an order from the Orders table.

The information for the order appears in
the Customer Order Entry window.

3. Click the Insert
   button to add a new line item to the table.

A blank line appears in the table.

4. Enter the quantity of
   the part created in this work order in the Quantity column.
5. Enter a unique work
   order ID in the Job ID column.

Because you are entering a new work order
with a unique ID, a dialog box appears, asking if you want to create
the new work order.

6. Select the appropriate
   job status for the new work order (Unreleased, Firmed, or Released),
   then click Yes.

A check mark appears in the New W/O field.

7. Tab to the Part ID field
   and enter or select a part.

If you double click the Part ID column header,
a dialog box appears, asking if you want to browse the Parts table
by Part ID or Part Description. Notice that the Co-products radio
button is grayed out. This is because you are entering a part for
a new work order.

When you select a part, VISUAL fills in
the part information for the part.

8. Enter the unit price
   for the part in the Unit Price column.
9. Click the Save
   button to save the new line item.

Now you need to open the Manufacturing Window
and add the co-products to the new work order.

10. From the Customer Order
    Entry window, select Tools from the File menu, then Manufacturing
    Window from the Tools pop out menu.

The Manufacturing Window search dialog box
appears.

11. Enter the Base ID for
    the work order you just created.

The new work order appears in the Manufacturing
Window.

12. Enter the co-products
    for this work order.
