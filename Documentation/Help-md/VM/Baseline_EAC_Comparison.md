# Baseline/EAC Comparison

The Baseline/EAC Comparison window shows the difference between
the baseline EAC and the current version of the EAC. Use the comparison
to assess whether your current costs are in line with your original
estimate.

Keep in mind that VISUAL does not update the EAC automatically.
You must create the current version manually. The Baseline/EAC Comparison
is only as accurate as your current EAC, so update the current EAC
carefully.

To view the Baseline/EAC Comparison window, select Baseline/EAC
Comparison from the Info menu. The Baseline/EAC Comparison
window appears.

To view a different EAC, click the Version ID
arrow and select the EAC from the list.
