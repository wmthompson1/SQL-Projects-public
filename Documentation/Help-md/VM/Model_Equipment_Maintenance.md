## Model

Click the Model button to select a type for
the equipment. Click [here](Entering_Maintaining_and_Deleting_Equipment_Models.htm)
for more information.

If you select an equipment type, you may not be able to select a model.
Equipment models require an equipment type as part of their definition,
so selecting a certain type does not necessarily mean that a model exists
with the same type in its profile. Base the piece of customer equipment
on an existing equipment type or a specific model. If you select a type
with no corresponding model, you cannot choose a model without changing
the type.
