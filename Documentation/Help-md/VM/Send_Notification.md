# Send Notification

When the option to manually send notification on a given document
is selected, you can manually send an email for an existing customer
order, packlist, cash receipt, purchase order, receiver or AP payment.

To send email notifications use the Send Notification
option from the File menu of the
respective window. You can also use the Send Notification
icon from the toolbar menu. The Send Notification
menu option and icon are only enabled if the window has manual notifications
selected. The Send Notification option is
available in these windows:

* Customer Order Entry
* Order Management
* Shipping Entry
* AR Cash Application
* Purchase Order Entry
* Purchasing
* Purchase Receipt Entry
* AP Payment Entry
