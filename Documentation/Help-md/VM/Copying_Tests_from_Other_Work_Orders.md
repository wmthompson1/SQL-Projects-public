# Copying Tests from Other Work Orders

You can copy tests from existing engineering masters
from the Work Order and Operation Input window.

1. Click the Copy
   Tests from Work Order button.

The Copy Tests from Another Work Order dialog
box appears.

There are two sections in this dialog box:
the From section and the To section. The current work order ID appears
in the To section.

2. Enter the work order
   ID for the work order from which you want to copy the tests. You
   can click the Base/Job ID button and
   select a work order from the Work Orders table.
3. Enter the lot ID and
   split ID in the Lot ID and Split ID fields, then select the sub
   ID from the Sub ID drop down boxes.
4. Click the Save
   button.

VISUAL copies the tests from the existing
work order to the current work order.
