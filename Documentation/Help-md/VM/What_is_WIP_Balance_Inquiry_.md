# What is WIP Balance Inquiry for Projects?

Use the Work In Process (WIP) Balance Inquiry for
Projects from the Post Manufacturing Journal window to review your
Work In Process balances for a specified period.  You can use
this report to reconcile your WIP balances after closing a period.
You can use the inquiry window to print a WIP Balance report or view
the information in a table within the inquiry window.
