# Step Thirteen

## Close the Period

When you have finished reconciling your transactions, you are ready
to close the period.

1. From the Ledger menu,
   select Accounting Window. The Accounting
   Window appears.
2. From the Maintain menu,
   select Close/Reopen Period. The Close/Reopen
   Period window appears.
3. In the End Date of Beginning
   Acct Period field, enter any date within the period you would
   like to close.

VISUAL populates the field with the end
date of the period.

4. Click the Test
   button to verify that all transactions for the period have been
   posted. If necessary, investigate any unposted transactions.
5. If the test runs successfully,
   click the Close Period button to close
   the period.
