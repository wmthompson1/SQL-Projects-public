# Adding Notes and Specs

You can add notations and specifications for both
the test and the test step line items.

To add notes and specs to the line items, select the line item for
which you want to add the notes or specs, then click the Specs
or Notes buttons located directly below
the line item table.

To add notes and specs for the test, click the Test
Specs or Test Notes buttons located
in the lower right corner of the window.
