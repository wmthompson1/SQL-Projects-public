# Viewing Test Step Capabilities

You can view the test capabilities for a test step
using Test Definition.

1. Highlight the test step
   line item for which you want to view capabilities.
2. Click the Capabilities
   button.

A window showing the test capabilities appears.

You can view the test capabilities information
in this window. You can also view part notes, work order notes, operation
notes, and test notes result notes by clicking on the appropriate
buttons. Click the Quantitative button to view a histogram of the
test results.
