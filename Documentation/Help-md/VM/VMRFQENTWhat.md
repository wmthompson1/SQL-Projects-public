# What is Vendor RFQ?

Vendor Request for Quote Entry makes it easy to request,
obtain, and record quotations on parts needed to fulfill a customer
order. Vendor RFQ Entry allows you to list quantities for a quote
for one or more quantities with unit price.

You should normally create a customer quotation first, engineer
it sufficiently to be able to cost it and identify the necessary parts
to complete it, and then generate a request for quote. Alternately,
you can create request for quote and process it without a customer
quotation involved.
