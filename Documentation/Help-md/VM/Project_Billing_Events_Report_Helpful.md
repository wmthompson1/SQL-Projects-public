# Project Billing Events Report

Use this report to review project billing transactions.

1. From the Project menu,
   select Project Billing Entry. The Project
   Billing Entry window appears.
2. In the Project Billing
   Entry window, click the Site ID arrow
   and select the site to use.
3. Select File,
   Print Project Billing Events Report.
   The Print Project Billings Event window appears.
4. In the Starting Date
   field, enter the first day of the period you want to close. In
   the Ending Date field, enter the last day of the period you want
   to close.
5. In the Show Events box,
   select Future Billing Events, Current Billing Events, and Processed
   Billing Events. Selecting all check boxes will give you
   a complete view of the billing events that are set to occur during
   the month.
6. In the Customers box,
   select All Customers. When you select
   All Customers, VISUAL selects All Orders
   in the Orders box by default.
7. Select a sort order
   for your report in the Sequence box. Your choices are by Order ID and by Customer
   ID.
8. Click Ok
   to run the report. The report outputs.
