# Drilling to Related IDs from Activity Maintenance

You can open the record related to the task directly from Activity
Maintenance. The application that is opened depends upon the related
type. This table shows each related ID type, which application is
opened when drilling to an ID, and which window is opened when drilling
to an ID.
