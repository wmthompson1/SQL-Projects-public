# Creating Records

The focus of most applications is the creation and
maintenance of a specific record. For example, each maintenance application
maintains all of one type of record. Sometimes, an application also
works with sub-records. For example, Product Codes are separate records
that you maintain from the Part Maintenance window.

To create a new record, open the appropriate application (for example,
Part Maintenance to create a part). When you first open an application,
the window should be empty and ready for you to create a new record.
If the window contains a previous record, click the New button to
ready the window for entering a new record.

After you have entered information in all of the required fields,
click the Save button or select File,
Save to create the record.
