# Working with Authorizations

The Authorization dialog box allows you to restrict
the enactment of a workflow step to a specific user or users only.

To set up authorization lists:

1. After clicking the Authorization button, the Authorization dialog
   box appears.
2. Click the Insert
   button.

The User/Group ID arrow appears.

3. Click the arrow and
   select a User or User Group from the list.

   |  |  |
   | --- | --- |
   | IMPORTNT.gif | If you select a group, the names of all the members of that group appear in the list. |

4. Click the Save
   button.
