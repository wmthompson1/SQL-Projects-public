# Working with the Leadtime Process Dialog

After you set up process types and assign them to
parts, performance leadtime information is automatically recorded.
If you use built-in functionality, then information is updated based
on events such as releasing a sales order. See [About
Built-in Leadtime Functionality](About_Built-in_Leadtime_Functionality.htm). If you use macros, then information
is updated based on conditions in the macro.

Use the Process Activity dialog to view the information. You can
edit the actual days, % complete, start date, and end date values.
You can also export the information to Excel or to an XML file.

See these topics:

* [Editing
  Information in the Leadtime Process Dialog](Editing_Information_in_the_Leadtime_Process_Dialog.htm)
* [Exporting
  Process Leadtime Information](Exporting_Process_Leadtime_Information.htm)
