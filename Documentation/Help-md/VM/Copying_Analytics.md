# Copying Analytics

You can create a new analytic by copying an existing
analytic. To copy an analytic:

1. Select File,
   Dashboard.
2. Select Edit,
   Edit Analytic.
3. In the Configuration
   ID field, select the analytic.
4. Click Copy.
5. In the New Configuration
   ID field, specify the ID of the new analytic.
6. Click Ok.
   The new analytic is loaded in the dialog.
