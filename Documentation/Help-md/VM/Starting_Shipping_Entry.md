# Starting Shipping Entry

Shipping Entry is available from the Infor VISUAL
Main menu.

To access Shipping Entry:

Select Shipping Entry from the Sales menu.

The Shipping Entry window appears with no information in it.

The Shipping Entry window is made up of both entry fields and action
buttons.
