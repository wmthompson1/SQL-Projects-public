# Using the Order Management Window

The Order Management Window functions in a manner
similar to Microsoft Windows Explorer. The window is divided into
two separate "panes": a tree and a detail. When you click
an icon in the left pane, the details for the current icon appear
in the right pane. This is to say that the information for the next
level "down" appears on the detail pane.

The left pane, or tree, functions much like the "folder"
portion of Windows Explorer. Functions with subordinate information
have a plus sign (+) to the left of the icon. You can expand the function
by double clicking the icon, or by single clicking on the plus box.
When you expand a function, the plus sign changes to a minus (-) sign,
indicating that you can collapse the function. When you focus on a
particular function of the tree pane, its details appear in the right
pane.

The right pane, or details, varies, depending on the function on
which you place focus. In general, the details indicate, in a table
format, information related to the function you select. You can modify
the width of the columns as well as the sort order, but you cannot
change the column order. See [Working
with Tables](Editing_Browse_Table_Column_Layouts.htm) for more information on setting column widths.
