# Freezing Project Schedules

You can prevent your projects from being rescheduled
by Freezing the project.

To freeze or unfreeze projects:

Open the Project window and open the appropriate project.

From the File menu, select Freeze Current project.

To Unfreeze projects, select the Unfreeze Current
Project option.
