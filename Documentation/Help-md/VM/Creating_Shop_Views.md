# Creating Shop Views

You can create different views to be available while
using the Scheduling Window. These different views contain a set of
resources you define under a particular View ID. Use Shop Views to
define a set of resources you want to view at one time in the Scheduling
Window. A resource can belong to many views.

1. Select Shop
   Views from the Edit menu.

The Shop Views dialog box appears.

2. Enter a new View ID
   in the appropriate data field.

If you want to add some more resources to
the existing view, select that particular View ID from the pull down
menu available.

3. Click the Insert
   button.

A new row appears in the table. VISUAL sets
the sequence number for the first row to 10.

4. Enter a valid Resource
   ID.

You can double click the Resource ID column
header to select from a table of resources.

5. Repeat steps 3 and 4
   to enter all the resources to be added to the Shop View.
6. Change the sequence
   numbers to determine the top-to-bottom sequence in which you want
   the resources in this view to appear in the Scheduling Window.
7. Click the Save
   button.

VISUAL saves your Shop View.

Use the Delete
button to delete the resources from a particular view.

Use the Copy button
to copy a view to another.
