# BTS-BI Basic Inventory Transactions

These transactions are for tracking basic material
movement within a facility. They are the core of the Infor VISUAL
BTS system for real-time inventory tracking. This module allows tracking
of inventory from the point that it enters a facility, through processing
and out the shipping dock. You can use these transactions independently
and restart them at any time depending on the Infor VISUAL Database
server.

## Telnet Server

Though API-Fusion by Odyssey Software fits very well
with the VISUAL Automated Material Tracking Systems operational requirements,
you can perform the same barcoding functions off of a telnet session.
A Telnet NT Server is relatively inexpensive and available off the
shelf from Microsoft, Seattle Labs, Georgia Softworks, and other
various other software producers. Consult your Infor Global Solutions
Partner for more information.
