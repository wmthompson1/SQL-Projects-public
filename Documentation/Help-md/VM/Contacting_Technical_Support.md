# Contacting Support

If
you have questions about Infor products, go to the Infor Customer
Portal at  [https://customerportal.infor.com/csmcore/](https://concierge.infor.com/).
If
we update this document after the product release, we will post the
new version on this website. We recommend that you check this website
periodically for updated documentation <https://docs.infor.com/en-us>.To
access documentation on the Infor Documentation Central, select **All
Product > Visual**. If you
have comments about Infor documentation, contact [Infor.Documentation@infor.com](mailto:Infor.Documentation@infor.com)
