# Displaying a Task Icon in the Task Tray

You can set up VISUAL to display an icon in the Windows Task Tray
any time a user has an open task.

To set up the Task Tray icon:

1. Select Admin,
   Preferences Maintenance.
2. Scroll to the Visual
   Mfg and locate the TaskTrayIcon entry. If the entry does not exist:

1. Click Insert Row.
2. In the Section field, enter Visual
   Mfg.
3. Click the Entry browse button and
   select TaskTrayIcon.

3. In
   the Value field, specify one of the following:

Y Specify Y to display the tasks icon. If you select Exit
in the task icon right-click menu, the system hides the icon until
you are assigned a new task.

T Specify T to toggle the tasks icon. If you select Exit
in the task icon right-click menu, the system hides the task tray
icon until you launch vm.exe again.

N Specify N if you do not want to view the task tray icon.
The system also hides the task tray icon if you leave the value blank.

4. Click Save.

## Accessing Task Maintenance from the Task Tray Icon

If the system displays the Task Tray icon, double-click the icon
to open Task Maintenance. You can also right-click the Task Tray icon
and select Open Task Maintenance.

To hide the icon, right-click the icon and select Close.
