# Starting Interbranch Transfer Entry

From the Inventory menu, select Interbranch
Transfer Entry. The Interbranch Transfer Entry window appears
with no information in it.

The Interbranch Transfer entry window contains both entry fields
and buttons.

The window consists of two main sections: a header section in which
you enter general warehouse and shipment information, and a line item
table in which you enter information specific to individual transfers
of parts.
