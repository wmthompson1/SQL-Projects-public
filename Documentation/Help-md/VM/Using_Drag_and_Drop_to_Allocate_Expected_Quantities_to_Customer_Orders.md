# Using Drag n' Drop to Allocate Expected Quantities to Customer Orders

Depending on a customers Priority Code and Disbursement
Fill Rate (the percentage of the total available quantity a customer
has access to) you can allocate expected supply to a customer order
line using the Part Availability windows drag n drop function.

To perform this operation, you must have the Customer Order Entry
child window and the Part Availability child window open. Tile the
two so dragging between the two is easy. The part you want to allocate
to the customer order line must be displaying in the Part Availability
child window.

1. In the Part Availability
   child window, double-click the part
   icon, double click the Available parts
   icon, double-click the Part ID icon,
   expand the warehouse icon of your choice, then click the Expected plus box. Expected orders appear.
2. In the left pane, find
   the expected order you want to allocate from, click the line and
   hold down the mouse button until the drag n drop box appears.
3. Drag the box into the
   open Current Customer Order Window and release the mouse button,
   thereby "dropping" it.

If there are no conflicts, VISUAL allows the allocation from the
expected order to the current customer order. In the Customer Order
Entry child window, you can verify the supply by looking at the Supply
Type column in the line table. For example, if the expected order
was a purchase order, PO appears in the column.
