# Setting Up Piece Tracked Parts in Part Maintenance

Use Part Maintenance to set up new piece tracked parts.

If you are licensed to use multiple sites, piece tracking information
is defined at the tenant level.

1. Select Part
   Maintenance from Inventory menu of the VISUAL Main window.

The Part Maintenance window appears.

2. If you are licensed
   to use multiple sites, click the Site ID arrow and select \*\*Tenant\*\*.
   If you are licensed to use a single site, this field is unavailable.
3. In the Part ID field,
   enter a Part ID for the part you are setting up.
4. In the Description field,
   enter a description for the part.
5. In the Stock UM field,
   enter the stock unit of measure for the part.
6. Enter any necessary
   part information for the part.

See "Adding New Parts" in Part
Maintenance for more information.

7. Click the Piece
   Trk tab.

Piece Track information appears.

8. Select the Part
   is Inventoried Using Piece Tracking check box.

The other check boxes become active.

9. Choose the dimensions
   for the part. You can choose any combination of length, width,
   and height as the dimensions for the part.
10. Enter
    a unit of measure for the dimensions in the Dimensions
    UM field, or click the Dimensions UM
    button to select a unit of measure for the dimension from the
    Units of Measure table.

The Dimensional U/M defaults to the Stock
U/M if you do not enter a Dimensional U/M. The Dimensional U/M pertains
to how you want to view the inventory pieces. Inventory transactions
are posted in the Stock U/M.

11. Click the Save
    button to save the new part.
