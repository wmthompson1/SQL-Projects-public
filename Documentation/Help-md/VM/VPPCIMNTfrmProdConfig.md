# Creating New Configurator Interface Tables

1. If you are licensed
   to use multiple sites, click the Site ID
   arrow and select the site to set up. If you are licensed to use
   a single site, this field is unavailable.
2. From the Product Configurator
   Interface Maintenance window, click the Insert
   Line button.

A new line appears in the table.

3. In the Series ID and
   Model ID columns, enter a Series ID and a Model ID for the part
   you want to configure.

If the same combination of Series ID and
Model ID already exists, VISUAL does not allow you to go to the next
column.

4. In the Part ID column,
   enter or select the appropriate Part ID.

You can double click the Part ID column
to select the Part ID from the Parts browse window.

5. In the Eng Mstr column,
   enter the Eng Master Engineering ID for the part.

You can double click the Eng Master column
to select the engineering master from the Engineering Masters browse
window.

6. Click the Save
   button to save the new values.

## Deleting Rows

To delete a row or rows:

1. Select the row or rows
   you want to delete then click the Delete
   button.

VISUAL marks the deleted rows with an X.

2. Click the Save
   button to complete the deletion.
