# Adding Color

You can add color to detail grids. You can use colors
in these ways:

* You can specify static colors
  for columns. When you specify static colors, the colors are used
  for the column regardless of the data in the column. Use this
  method to draw attention to certain columns or to categorize columns.
  For example, in a grid that shows work order costs, you could
  use a certain color for projected costs and another color for
  actual costs.
* You can specify conditions
  under which a cell has a particular color. Use this method to
  draw attention to a cell when the information meets criteria that
  you specify. For example, you could specify that the text color
  of a cell should be red if the value in the cell is negative.

You can specify a background color and a text color for columns
and cells.
