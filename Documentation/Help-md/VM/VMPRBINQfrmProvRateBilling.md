# Using Provisional Rate Billing Inquiry

To run provisional rate billing inquiries:

1. From
   the Project menu of the Infor VISUAL main window, select the Provisional Rate Billing Inquiry option.
2. Click the Site ID drop-down
   arrow and select the site to use.
3. Click the Project ID button and
   select the project on which you want to run a Provisional Rate
   Billing Inquiry.
4. In the Enter
   Date Range section, select the start and End date ranges for the
   inquiry you are running.
5. Click the Rate ID arrow and select the rate you want
   VISUAL to use for the inquiry.
6. If
   you want VISUAL to run the inquiry by CLIN, select the CLIN
   check box.
7. If
   you want VISUAL to run the inquiry by WBS, select the WBS
   check box.
8. Click the Run
   button on the toolbar or select Run from the File menu.

VISUAL calculates the Provisional Rate Billing Inquiry for the project,
rate, and date range you entered.
