# Milestone Status Report

Use the Milestone Status Report to review progress
billings that are ready to be triggered

1. From the Progress Billing
   Window, select File, Print
   Milestone Status Report. The Print Milestone Status Report
   window appears.
2. In the Starting Date
   field, enter the first day of the period you would like to review.
   In the Ending Date field, enter the last day of the period you
   would like to review.
3. In the Show Events box,
   select Ready to Trigger to review events
   that are ready to be processed. If you would also like to see
   the events that have already been triggered during the time period,
   select the Already Triggered check box
   as well.
4. In the Customers box,
   select All Customers to review a comprehensive
   set of data.
5. In the Sequence box,
   select the sort order for the report. You can choose to sort the
   report by Order ID or Customer
   ID.
6. Click Ok.
   The report outputs.
