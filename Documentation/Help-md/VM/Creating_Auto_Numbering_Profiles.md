# Creating Auto Numbering Profiles

Numbering profiles allow you to create an unlimited
number of auto-numbering definitions that the system can then apply
to traceable part profiles.

A numbering profile consists of the following pieces of information:

Number ID - The unique identifier
for each separate numbering profile, up to 15 characters in length.

Next Number - Auto-numbers consist of
an integer portion, and an optional alphanumeric prefix and suffix.
Next Number specifies the next integer that will be used for this
profile.

Prefix - Specifies any alphanumeric characters
that will precede the integer portion of the number.

Suffix - Specifies any alphanumeric characters
appended to the number.

Digits - If display of leading zeros is
active (see next item), this specifies the total number of digits
in the integer portion of the number.

Leading Zeros - If Y, specifies that leading
zeros are to be included in numbers, to the number of digits specified
by the Digits field. This will keep all auto-numbers the same number
of characters long.

To create an autonumbering profile:

1. Select Auto
   Numbering from the Edit menu.

The Numbering Profiles dialog box appears.
All currently defined numbering profiles appear in the table.

2. Click the Insert
   button.

A new line appears in the table.

3. Enter a new Number ID
   on the new line. It must be different from all the other Number
   IDs.
4. In the Next Number column,
   enter the number to be used as the integer portion of the next
   number assigned.
5. In the Prefix column,
   enter any prefix. In the Suffix column, enter any suffix.

The following fields are optional:

6. In the Digits column,
   enter the total number of digits to show if Leading Zeros is selected.
7. If you wish to suppress
   leading zeros, enter N in the Leading Zeros column; otherwise,
   leave it set to Y.
8. Click the Save
   button to save the new Number ID.
