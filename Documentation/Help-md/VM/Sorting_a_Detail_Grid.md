# Sorting a Detail Grid

You can sort a detail grid by any column.

To sort a detail grid:

1. Select File,
   Dashboard.
2. In the dashboard, select
   File, Open
   and select the dashboard that contains the detail grid.
3. To sort by a column,
    click the header of the column. The grid is sorted by the
   column in descending order. To sort by ascending order, click
   the column header again.

You can also group information together,
and then sort the information within the groups. To create a group,
right-click the column header to use as the group and select Group
By. You can also display a Group Box at the top of the grid and drag
the columns to use as the group to the box.

Use the arrows next to items in the Group
Box to change the sort order of the group.

To remove a group, drag the group name from
the Group Box to the grid header row.
