## ECN Cost Analysis Tab

The Cost Analysis tab contains information boxes related to the costs
for the current ECN.

These costs may be estimated costs, actual costs, or both. You may want
to use this tab to enter the cost estimates so that the individuals responsible
for authorizing the ECN can base their decision on these expected costs.
You may also want to use this tab to enter in the actual costs of the
ECN implementation after the ECN is complete as a way of providing an
audit to the ECN.

Click [here](Entering_Information_on_the_Cost_Analysis_Tab.md)
for more information.
