# Activating Analytics

To activate an analytic:

1. Select File,
   Dashboard.
2. Select Edit,
   Edit Analytic.
3. In the Configuration
   ID field, select the analytic. The amount of time it took to construct
   the analytic is displayed above the tabs.
4. Click the Status
   drop-down button and select Active.
