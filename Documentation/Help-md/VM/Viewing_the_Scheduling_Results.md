# Viewing Scheduling Results

Because scheduling is a batch process, the Global
Scheduler itself is limited in user interaction. The Scheduler takes
the input you enter and processes it to suggest the optimal shop schedule.
Input required from you is limited to some choices about how to produce
the schedule.

The main schedule viewing and analysis tool is the Scheduling Window.
Schedule results also appear in the Manufacturing Window and Material
Planning Window.
