# Deleting a Purchase Order

You can delete a Purchase Order and all of its line
items permanently from the database.

|  |  |
| --- | --- |
| POSTIT.gif | You do not need to close Specification windows after each use. They can remain open for the whole session. Their contents change as the current order and selected line items change. |

1. With the appropriate
   purchase order open in the Purchase Order Entry window, click
   the Delete button.

A dialog box appears, asking you to confirm
the deletion.

2. Click Yes
   to permanently remove the purchase order or No
   to leave the purchase order in the system.

You cannot delete a purchase if a Receipt
Transaction has been charged against it. In order to delete the purchase
order, you must first delete the receipt.

3. Click Save
   to complete the deletion.
