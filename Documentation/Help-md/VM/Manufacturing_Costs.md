# Manufacturing Costs

In order to determine an organization's profits or
losses, the costs associated with the production of the end product
must be accumulated, measured and recorded. VISUAL offers two methods
for generating manufacturing costs, standard cost and actual/job order
cost. The system can be set up to use either option but not both.
This choice has to be made before implementing the software. Establish
the accounting entity parameters in the [Accounting
Entity Maintenance](VMAENMNTWhat.md) window.
