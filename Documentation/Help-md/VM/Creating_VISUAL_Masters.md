# Creating Import Files for the Data Import Utility

The VISUAL Data Import Utility creates a single or
multilevel Engineering, Quote or Work Order master from a tab-delimited
file.

The VISUAL Data Import Utility processes a tab-delimited file that
contains the data needed to create masters in VISUAL. The file contains
several different record layouts for each part of the master or work
order. The file can contain multiple masters.
