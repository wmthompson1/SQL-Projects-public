# Using Animation Mode

Use Animation mode to step through the painting of
the Scheduling Window display one work order or one operation at a
time. The work orders and operations appear in the position at which
each was successfully scheduled, starting with the focus operation.

This is one way to view what the load against capacity was before
an operation was successfully scheduled. Only the load for all operations
that were scheduled prior to the focus operation in the current schedule
ID and the focus operation appear.

If you filtered the Scheduling Window using the options available
in the Info menu, the Animation Mode may not function as you expect.
The Next Operation and Next Work Order buttons consider all scheduled
operations and work orders, not just the work orders or operations
that meet your filter criteria. As a result, clicking the Next Operation
or Next Work Order buttons may not seem to have an effect in the Scheduling
Window. We recommend clearing any filters before you enter Animation
Mode.

1. Select Animation
   Mode from the View menu.

The Scheduling Window removes all information
from the window display.

2. Use the Next
   Operation function from the View menu to display only the
   next operation as it was successfully scheduled (which may be
   the first operation of the next work order in the schedule, if
   the current one is the last one on its work order)

Use the Next Work Order
function to paint the remaining operations of the current work order.

The respective Previous
functions work the process in reverse.

3. Select Animation Mode
   again to exit Animation mode.
