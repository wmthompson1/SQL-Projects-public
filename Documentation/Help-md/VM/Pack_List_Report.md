# Pack List Report

Use this report to review shipments.

1. From the Contracts Menu,
   select Shipping Entry. The Shipping
   Entry window appears.
2. In the Shipping Entry
   window, click the Site ID arrow and
   select the site for the report.
3. From the File menu,
   select Print Pack List Report. The Pack
   List Report window appears.
4. In the Status box, choose
   Shipped.
5. In the Date box, choose
   Actual Ship Date.

The field labels at the top of the window
change to Starting Actual Ship Date and Ending Actual Ship Date.

6. In the Starting Actual
   Ship Date field, enter the first day of the month you would like
   to close. In the Ending Actual Ship Date field, enter the last
   day of the month that you would like to close.
7. In the Type box, choose
   Detailed.
8. Click Ok.
   A Detailed Pack List report appears.

The key information on the Pack List detail
report is the customer, the items shipped, the quantity and the value
of the shipment.
