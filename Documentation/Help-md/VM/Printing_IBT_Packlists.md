# Printing IBT Packlists

You can use a Packlist as:

* A document that accompanies
  the product you are shipping to the To Warehouse. It contains
  shipping information about the IBT.
* A "picklist" for
  preparing shipments.
* A list that shows what is
  remains to be shipped.

1. Select Print
   Pack Lists from the File menu. The Print Pack List dialog
   box appears.
2. If you are licensed
   to use multiple sites, click the Site ID(s) arrow and select the
   sites to view in the report. The site selection applies only if
   you select the Print All IBTs Shipped Between option. If you select
   either of the Print Current IBT options, then only the site associated
   with the IBT is included in the report.
3. Select the range of
   packlists to print:

Print Current IBT Shipper
- If a packlist currently appears in the IBT Shipping Entry window,
you can print it by selecting this option; otherwise, this option
is not enabled.

Print Current IBT-
If a packlist or IBT ID is currently specified in the IBT Shipping
Entry window, you can print all packlists for that IBT order by selecting
this option.

Print All IBTs Shipped
Between - This prints IBT packlists for all IBT orders with
Actual Ship Date between the Dated From and Dated Through dates. Dated
From can be left blank to give an open-ended range. Dated Through
must be specified.

4. If appropriate, select
   the Print Lines Due Through option to
   further constrain the printing of packlist lines.

For Before Shipment reports, this option
filters out packlist lines that have a desired ship date later than
the specified date. If a Ship Date is not specified on the order line,
the Desired Ship Date for the customer order is used. This option
is useful when using Print Current Order to focus on lines due before
a certain date.

5. Select the Print
   Traceable Part Properties check box if you want to include
   traceable part properties in your report
6. Select the print format.

You can choose between the standard format
provided with the application (you can also customize this form),
a laser print format or three other user-defined formats. The User-defined
radio buttons are only active if the forms have been defined. You
may want to create different forms for Before Shipment and After Shipment
lists, and for internal and customer uses.

Select the Print Form
option button to print the report with a preprinted form using a laser
printer. The resulting report is of better quality than the report
you can print using the Standard print format
option. You must have a laser printer, however, to use this format.

7. If necessary, click
   the Print Traceable Part Properties
   check box to print traceable part properties.
8. If you do not at this
   time want to print again packlists that you have already printed,
   click the Do Not Print Previously Printed Packlists
   check box.
9. Select how you want
   VISUAL to output the report:

Print - Select
this to view a print preferences dialog box after clicking Ok.

View - Select
this to view the report on your screen after clicking Ok.

File - Select
this to save the report to a file after clicking Ok.

E-mail - Use the
E-mail option to attach the report to an email. When you select E-mail,
these check boxes become available:

PDF Format Select
this check box to attach the report as a PDF. Clear this check box
to send the report as a RTF.

Send to Document Contacts
If you are generating this report for a consignment shipment, select
this check box to automatically email the report to the vendor and
customer contacts specified for IBT Shipments for the vendor. When
you select this check box, the email is sent without further action
from you. To preview the email before it is sent and to manually specify
the recipients of the email, clear this check box.

10. Click Ok.

A standard Windows dialog box appears, which
allows you to select the number of copies to print.

11. Make the appropriate
    selections in the dialog box and click Ok.
