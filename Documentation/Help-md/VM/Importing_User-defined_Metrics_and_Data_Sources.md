# Importing User-defined Metrics and Data Sources

You can import user-defined content into your database.
You can import both custom content and vendor content. Vendor content
is provided by Infor. You cannot edit vendor content.

1. Select File,
   Dashboard.
2. Select Edit,
   Import Content.
3. Click the browse button
   and select the file to import.
4. Click Open.
5. Click Import.
