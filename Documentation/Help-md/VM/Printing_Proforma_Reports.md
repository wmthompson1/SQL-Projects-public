# Printing Proforma Reports

The Proforma report projects the kind of revenue you
will get for a period of time should you win all quotes above a certain
probability percentage.

VISUAL uses the Win Probability value from the Dates section of
the Estimating Window. It is important to note that there is a data
field on the Quote Price dialog box associated with each quotation
line item called In Proforma. If you select the check box in the In
Proforma column, the associated part and dollar value appear on this
report.

If you are licensed to use multiple sites, you can view information
for multiple sites in the same proforma report, provided that the
sites share the same parent accounting entity.

To output a Proforma Report:

1. If you are licensed
   to use multiple sites, click the Site ID
   arrow and select a site to view in the report. When you access
   the dialog box, you can choose to include in the report any site
   with the same parent accounting entity. If you are licensed to
   use a single site, this field is unavailable.
2. Select Print
   Proforma from the File menu.
3. If you are licensed
   to use multiple sites, click the Site ID(s)
   arrow and select the sites to include in the report. You can include
   any sites with the same parent entity as the site selected in
   the Estimating Window. If you are licensed to use a single site,
   this field is unavailable.
4. Enter the starting date
   and ending date for the appropriate period.

This range includes all quotations for the
time period you enter.

5. Enter the Probability
   value.

For example, if you want to include all
estimates with a probability greater than but not equal to 75%, enter
75 here.

6. Choose the output destination
   by selecting the arrow next to the output field. Select one of
   the following:

Print - To output
the file to a printer, select this option. VISUAL sends the file to
the selected default printer.

View - To see
what the printed output will look like before it is printed, select
View. When the information appears on the screen, you can scroll through
it and print it if you need a hard copy.

File - To save
the information to a text file, select File from the pull down menu.

Click OK to bring
up a dialog box for the entry of a file location and name. Enter the
desired location for the file and whether or not the field names are
to be included in the text file. You can now edit the file outside
of VISUAL using any text editor.

7. Select the list sequence
   sort type. You can select one of the following: by Month, by Sales
   Rep, by Territory, or by Customer, Month.
8. If you selected Print
   as the output, click Ok to output the
   information to your printer.

A standard Windows printer dialog box appears,
allowing you to select the number of copies to print or view.

9. Make the appropriate
   selections and click Ok.

The Proforma Report prints one line for
each quotation you select. Each line contains the Quote ID, Customer
Name, Entity, Sales Rep, Territory, Probability, and the total amount
quoted. These lines are grouped by the sequence specified. For each
unique group in the sequence, the total amount quoted is shown. For
example, if sequencing by month, a total quoted amount for each month
is printed.
