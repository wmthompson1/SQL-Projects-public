# What is the Engineering Report?

The Engineering Report provides a complete printout
of an engineering master, work order, or quote.

When you start this report, the table window shows all engineering
masters, work orders, and quotes. This window is similar to the Search
window provided in the Manufacturing Window. To perform a search,
select Search from the file menu. See the Manufacturing Window for
more information on searching this list.

If you wish to run the report for a selected line or lines, make
the selections, and choose Print/View Selections
from the File menu. To print all lines,
choose Print/View All Lines from the File menu. This option is useful when you have
used a search to limit the number of lines in the list.

## Options

### Short Form

The long form shows essentially all information defined in the Manufacturing
Window for the header, each operation, and each material requirement.
The short form shows Quantity, Setup Hours, and Run Time for each
operation, and Quantity for each material. Drawing IDs and Revs are
also shown. Check the Short Form box to get this report.

### No Customer

This option excludes customer information from the list table. This
can increase the speed of a search.

### Sequencing

You can sequence the table window list by Work Order ID, Customer
ID, Create Date, Want Date, or Release Date. The selected masters
and orders will print in this sequence.

Additionally, you can check the Descending check box to reverse
the order of the listing.
