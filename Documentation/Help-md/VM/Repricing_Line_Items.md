# Repricing Line Items

You can assign the latest part price from an updated
vendor quote to the line items in a purchase order. To do this:

1. Select the line or lines
   you want to reprice.
2. Select Reprice
   Lines from the Edit menu.

 A dialog box appears, asking you if
you want to reprice the line items.

3. Click Yes
   to reprice the lines.

The new prices appear in line item table
of the Purchase Order Entry window.
