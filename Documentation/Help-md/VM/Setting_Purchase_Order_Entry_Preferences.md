# Setting Purchase Order Entry Preferences

To set the Purchase order Entry child windows preferences:

1. Select the Preferences
   option from the Options menu.

The Preferences dialog box appears.

2. Click the Default Tab
   arrow and select the tab you want to appear when you first open
   the Purchase Order Entry window.
3. In the Default Status
   section, select what status you want when you enter a new purchase
   order.

|  |  |
| --- | --- |
| POSTIT.gif | If you select Release, only users with the appropriate permissions are allowed to save purchase orders with Released status. |

4. If you want to use VISUALs
   auto dispatch services, select the Auto Dispatch
   Services check box.

With this preference set, VISUAL automatically
dispatches the entire service purchase order quantity. Upon saving,
VISUAL dispatches the lines entire Order Quantity.

5. Click the Ok
   button.
