# Specifying Capacity Use Information

You can specify minimum and maximum number of resource
units needed for the operation in the Other section of the Operation
dialog box.

1. Enter a minimum number
   of resource units needed for this operation in the Capacity Use
   - Min field.

Enter a maximum number of resource units
this operation will use in the Capacity Use - Max field.

2. Click the Save
   button to save the new information.
