# EDI Requirements

Infor Global Solutions' EDI solution requires Infor
ERP VISUAL Enterprise 5.2 or higher. Infor ERP VISUAL Enterprise/Infor
VISUAL is a 32-bit application using client/server architecture that
runs on Novells NetWare with SQLBase, or Windows NT, with SQLBase,
SQL Server, or Oracle. For Oracle databases, VMDI also runs on a
variety of UNIX platforms. Infor ERP VISUAL Enterprise/Infor VISUAL
also runs on the DB2/400 database on the AS/400. Client support
includes Windows 95 and Windows NT. Infor ERP VISUAL Enterprise
uses a GUI interface and supports both OLE and ODBC.

Translation software is required, which will have its own hardware
requirements. The VDMI module from Infor Global Solutions is required
for integration.

Communication requirements include an Async/Bysync Hayes compatible
modem as well as a method of transferring data (not provided by Infor
Global Solutions), either a Value Added Network (VAN) account, direct
connection to a trading partner, or Internet access.

In the interest of a timely and cost-effective implementation, Infor
Global Solutions will help determine your EDI implementation requirements
by conducting a needs assessment, assisting with an EDI software purchase,
providing services and support, and providing services and support.
