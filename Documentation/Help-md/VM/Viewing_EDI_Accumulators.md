# Viewing EDI Accumulators

You can view and edit EDI order accumulators using
the Customer Order Entry window. Most EDI users use this function
if the order is for a single Part ID.

1. Click the EDI
   tab then select the EDI Release check
   box. Typically, VISUAL selects this check box during an import
   from VMDI Exchange.

This allows you to create outbound transactions
using the Infor VISUAL Data Interchange VMDI module.

This check box must be selected in order
to view order accumulators.

2. From the EDI menu, select
   EDI Order Accumulators.

The Electronic Data Interchange Order Accums
dialog box appears.

The dialog box includes the following fields:

The dialog box includes the following customer
accumulated fields:

Release # - The
customers release number (RAN).

Shipments as of
- The date by which the customer has received the accumulated shipments.

Build Authorization
- The quantity of parts that the customer has authorized your company
to build.

Buy Authorization
- The quantity for which the customer has authorized your company
to purchase materials.

Accumulated Shipments
- The quantity of shipments the customer reports your company has
shipped to them.

The dialog box contains the following accumulator
totals. Either you or VISUAL can enter this information.

Adjustment to Shipments
- The adjustment quantity to the accumulated shipments. Use this field
to adjust the calculated shipment quantity for historical data that
may or may not exist in your VISUAL database.

Adjustment to Releases
- The adjustment quantity for the amount ordered. Use this field to
adjust and reconcile internal order quantity with customer order quantity.

Accumulated Shipments
- The actual shipments from this order. VISUAL calculates this value
when you open the EDI Order Accums dialog box.

Accumulated Releases
- The actual order quantity from this order. This value is only calculated
when you open the EDI Order Accums dialog box.
