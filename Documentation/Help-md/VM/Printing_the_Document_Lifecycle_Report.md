# Printing the Document Lifecycle Report

You can print a document lifecycle report from the Tree view only.
The report shows the lifecycle information you are currently viewing.

To print the report:

1. Select Info,
   Lifecycle Document Viewer.
2. If the Graphical Display
   is displayed when you open the Document Lifecycle Viewer, select
   View, Show Graphical
   Display to clear the check box next to Show Graphical Display.
   The Tree Display is displayed.
3. Select the output method
   from the output drop-down menu.
4. Click Print.
