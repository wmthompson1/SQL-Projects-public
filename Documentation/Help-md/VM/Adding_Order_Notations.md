# Adding Order Notations

Order notations are free form text similar to packlist
notations, except they are associated with customer orders instead
of the packlist.

1. With the appropriate
   order open in the Customer Shipments window, select Order Notations
   from the Notes menu.

The Order Notations dialog box appears.

2. Click the New
   button to make a new entry.

A line appears with the current date and
time for the entry.

3. Enter the appropriate
   information.

You can enter an unlimited amount of information.

Use the Paste
button to paste text that you copied from another location into the
box.

Use the Copy button
to copy text in the dialog box to the clipboard.

4. Click the Save
   button to save the Notes.
5. Click the Close
   button to close the Notations dialog box and return to the Customer
   Shipments window.
