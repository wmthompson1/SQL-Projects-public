# Setting Permissions for Individual Users

You can set up permissions on a user-by-user basis. To set up permissions
for an individual user:

1. In the Security by section,
   select User.
2. Click the User
   ID browse button and select the user for whom you want
   to set up permissions.
3. In the Language ID field,
   specify USA. To assign a language to a user, use the Language
   Assignment dialog. See [Assigning
   Languages to Users](Assigning_Languages_to_Users.md).
4. If you would like this
   user to have system administrator privileges, select the System Administrator check box. Users who
   are system administrators have access to the same applications
   as the SYSADM user.
5. Use the tabs to specify
   additional security settings. Refer to [Setting
   Information on Status Tabs](Setting_Information_on_Status_Tabs.md), [Setting
   Information on the Menu Tab](Setting_Menu_Security.md), Setting Information on the Program
   Tab, and Setting Information on the Allowable Sites tab for more
   information.
6. Click Save.
