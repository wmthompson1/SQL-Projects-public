# Searching Locations

Use the Find button in the
Warehouse Maintenance window to search the locations field. This allows
you to search through a long list of locations using a search string.

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site to use. If you are licensed to use a single site, this field
   is unavailable.
2. Click the Find
   button.

The Find Location dialog box appears.

3. Enter the location name
   in the Search For field, then click the Search
   button.

The search string you enter remains in effect
until you replace it or exit Warehouse Maintenance. You can repeat
the search without clicking on the Find
button by pressing the F3 key. This finds the location entered in
the search string or find the next location that matches the search
string.

If VISUAL finds the location, you return
to the Warehouse Maintenance window with the location highlighted.

If VISUAL does not find the location, a
message box appears indicating that VISUAL could not find the search
string.
