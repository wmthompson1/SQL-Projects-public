# Source and Target Buttons in Shipping Entry

When you are viewing a transaction, you can use the
source and target toolbar buttons to view related transactions. Use
the source button to view the previous transaction in the lifecycle.
Use the target button to view the next transaction in the lifecycle.

In the Shipping Entry window, the Source button is labeled Customer
Order. Click the button to open the order. The Target button is labeled
Invoice. It is available if you have created an invoice for the shipment.
Click the button to open Accounts Receivable Invoice Entry.
