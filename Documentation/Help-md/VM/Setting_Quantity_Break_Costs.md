# Setting Quantity Break Costs

You can set quantity break costs in the Material dialog
box. Use this function for one specific situation only when the
material requirement is for a Quote Master, and for a non-inventory
part. Quantity Break Costs allows you to specify quantity price breaks
and a minimum charge that the Estimating Window can use to project
costs.

1. Click the Quotes
   tab.

The Quantity Break Costs table window appears.
The table has one line for each quantity level specified.

2. Click the Insert
   button to add a line to the table.
3. Enter the Quantity Break
   or Up-To Quantity. Which quantity is used depends upon the setting
   of Purchase Quote Type in Application Global Maintenance.
4. Enter the Unit Cost
   and Fixed Cost for the Quantity. Fixed cost is a onetime cost
   of the purchase, regardless of order quantity.
5. Enter the Minimum Cost
   for the quantity. Any calculated cost based on unit price and
   fixed cost that is lower than this value is increased to this
   cost.
6. Continue to insert lines
   as needed.
7. Click the Save
   button to save all lines.
