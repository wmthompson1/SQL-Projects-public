# Saving Work Order Traveller Print Settings

To save your settings:

1. Select Eng/Mfg,
   Work Order Traveller.
2. In the Work Order Traveller
   window, select the print settings.
3. Select File,
   Save Print Option As.
4. In the Name field, specify
   the name for this print option.
5. To use these print settings
   as your default settings, select the Default
   check box. When you next open the Work Order Traveller, your default
   settings are loaded.
6. Click Save.
