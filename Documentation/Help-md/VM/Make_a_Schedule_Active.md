# Make a Schedule Active

This feature allows the system to act on a schedule
when the Schedule All Active option is selected.

1. Highlight the schedule
   you want to activate or deactivate.
2. Select the Active check
   box to activate the schedule.

A check mark appears in the Active column.

3. Click Save to activate the schedule.
