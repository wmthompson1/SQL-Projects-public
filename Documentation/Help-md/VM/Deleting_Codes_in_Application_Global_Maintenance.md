# Deleting Codes in Application Global Maintenance

To delete a system-wide code:

1. Select Admin,
   Application Global Maintenance.
2. Open the dialog in which
   the code is created. For example, to delete an honorific code,
   select Maintain, Table
   Maintenance, Honorific Table.
3. Select the code.
4. Click Delete.
   An X is displayed next to the code.
5. Click Save.
6. If the code can be deleted,
   a confirmation message is displayed. Click Yes.
   The code is deleted.

If the code cannot be deleted, a message
is displayed and the code is not deleted. Certain codes cannot be
deleted if they are used on a record in your database.
