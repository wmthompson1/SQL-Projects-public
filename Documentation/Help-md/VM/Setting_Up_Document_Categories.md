# Setting Up Document Categories

Use Document Categories to specify what type of document
you are entering. For example, if you want to reference Material Specification
Documents, enter a category similar to Material Specs.

To set up document categories:

1. From the Document Maintenance
   window, click the Document Category
   toolbar button.

2. Click Insert.

The next available row appears highlighted
and the cursor appears in the ID column.

3. Specify a suitable identification
   for this category.
4. Press the TAB key and
   enter a description for this category.
5. Click Save.
6. Click Close.
