# Specifying Calendar Exceptions for Fabricated Parts

If you want Material Requirements Planning to take
your calendar exceptions into account when calculating availability
dates for your fabricated parts, select the Use
Global Calendar and Exceptions for Fabricated Parts check box.
By using this setting you can specify the dates you do not want VISUAL
to use in its calculations.
