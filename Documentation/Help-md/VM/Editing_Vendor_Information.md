# Editing Vendor Information

1. Enter the vendor ID
   into the Vendor ID field.

If you do not know the vendor ID, click
the Vendor ID or Name
button to call up a table of available vendors.

2. Make any necessary changes
   to the vendor information.

You can change any field in the Vendor Maintenance
window except the Vendor ID. Changing the Vendor ID field creates
a new vendor record.

3. Click the Save
   button to save your changes.
