# Exporting Material Availability Information to Microsoft Excel

You can send the contents of the Material Availability
dialog to Microsoft Excel.

1. Select Eng/Mfg,
   Manufacturing Window.
2. Open the work order.
3. Select Info,
   Material Availability.
4. In the table, select
   the rows that you want to send.
5. Select File,
   Send to Microsoft Excel.
