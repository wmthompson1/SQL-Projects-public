# Adding Rule Properties

To add rule properties:

1. If necessary, create
   a rule line.
2. Right-click the rule
   to which you want to add or edit properties.

A menu appears.

3. Select Properties

The Rule Properties dialog box appears.

4. In the Label field,
   enter a name for the rule.

You may want to consider using a name indicative
of what you are attempting to accomplish with this rule.

5. In the Description field,
   enter a description for the rule.

If you are editing a rule property, the
Status, Completed By, and Date/Time fields appear populated.

6. In the Evaluation Sequence
   field, enter a number to indicate the sequence that you want a
   workflow path re-evaluated. Workflow paths are re-evaluated sequentially,
   beginning with 1.

7. If you want to use this
   Rule for assigning tasks, click the Approval
   button and set up the Task Notification list for this rule.

If you want to set up rule statements, click
the Insert button and set up the Rules statements
you want to use.

8. If you have entered
   rule statements and want to include Authorizations, click the
   Authorization button and set up the
   Authorization Users you want to use for this Rule.

9. When you have finished
   setting up the rule, click the Ok button.
