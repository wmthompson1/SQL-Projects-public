# Viewing Customer Information

Information for the customer you selected appears
in the Customer Inquiry Window. You cannot change this information.
You can use the Customer Inquiry Window to browse the group of information
with which you started. For example, if you originally browsed on
a customer name, you can use the Next and Previous record toolbar
buttons to browse that setonly orders for the customer name you selected
appear.
