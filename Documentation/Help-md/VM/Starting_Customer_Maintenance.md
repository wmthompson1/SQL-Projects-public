# Starting Customer Maintenance

Customer Maintenance is available from the VISUAL
Main window.

To access the Customer Maintenance window:

Select Sales, Customer
Maintenance.

The Customer Maintenance window appears with no information in it.

The Customer Maintenance window contains both entry fields and action
buttons.
