# What is the Consignment Usage Report?

When you use parts and material consigned to you,
use the Consigned Usage Report to print a list of those parts. Complete
the report and send it to your vendors or customers notifying them
of the parts you have used.

For more information on printing Consignment Usage reports, refer
to the [Printing Consigned Usage Reports](VMCSNUSGfrmMain.htm)
topic.
