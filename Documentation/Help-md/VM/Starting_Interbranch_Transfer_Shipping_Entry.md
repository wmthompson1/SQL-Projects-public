# Starting Interbranch Transfer Shipping Entry

From the Inventory menu, select Interbranch
Transfer Shipping Entry.

The Interbranch Transfer Shipping Entry window appears with no information
in it.

The Interbranch Transfer Shipping Entry window contains both entry
fields and buttons.

The window consists of two main sections: a header section in which
you enter general warehouse and shipment information, and a line item
table in which you enter information specific to individual transfers
of parts.

IBT Line Number - The line number of the
IBT line as it appears in the IBT order.

Supply Type - If the supply for this IBT
Order Quantity is from anything but inventory, the appropriate abbreviation
appears here. CP (Coproduct) PO (Purchase Order), PD (Purchaser Order
Delivery Schedule), WH (Interbranch Transfer), or WO (Work Order).

Reference - Any referential text regarding
the material you are transferring.

Ship Date - The date on which the parts
leave one warehouse for another.

Quantity Ordered - The quantity of the
material you are transferring.

Quantity Shipped - The quantity of the
material that you successfully shipped to another warehouse.

Ship Quantity If you are shipping a
standard part, specify the quantity to ship to the other location.
This cannot be greater than the Quantity Ordered.

Length, Width, Height If you are transferring
piece tracked parts, the dimensions of the part are inserted. You
cannot edit this field.

Dimensions UM If you are transferring
piece tracked parts, the dimension unit of measure is inserted. You
cannot edit this field.

Piece Count If you are transferring
piece tracked parts, the number of pieces specified in Inter Branch
Transfer Entry is inserted. You cannot edit this field.

Total Pieces Shipped If you are transferring
piece tracked parts, the number of pieces previously shipped is inserted.
You cannot edit this field.

# Pieces to Ship Specify the number
of pieces to ship. After you specify the number of pieces, the quantity
of the shipment is calculated and inserted in the Ship Quantity field.

Quantity Allocated - The portion of the
Quantity Ordered that you have allocated from other supplying sources.

U/M - The unit of measure of the part.

Actual Freight - The amount you pay for
freight charges to convey the materials.

Location ID - The ID of the transit location
in the From Warehouse from which you are shipping the parts. You can
change the Location ID; however the new Transit Location ID you specify
must be a valid Transit Location ID. See Part Maintenance for more
information on setting Location IDs as Transit Location IDs.

Cartons - The number of cartons you use
in the shipment.

Shipping Weight - The weight of the shipment.

Weight U/M - The unit of measure that
you use to weigh the part that you are transferring.

Package Type - The type of package you
want to use to ship the IBT. Enter packaging types in Application
Global Maintenance.

Out Trans ID - The ID of the Adjustment
Out as you ship the parts from the From Warehouse. Transaction IDs
appear in Inventory Transaction Entry.

In Trans ID - The ID of the Adjustment
In after you have shipped the parts in the From warehouse. Transaction
IDs appear in Inventory Transaction Entry.

Unit Price - The price per unit of the
part.

Shipping U/M The shipping unit of measure
for the material.

Shipping Dimensions - Any dimension information
associated with the material for shipping purposes.
