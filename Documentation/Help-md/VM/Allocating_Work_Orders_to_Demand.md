# Allocating Work Orders to Demand

With the work order from which you want to allocate
supply to demand highlighted, select Allocations
from the Edit menu and then Allocate Work Order
to Demand. The Assign to Demand dialog box appears.

You can allocate work order supply to these demand sources:

* Customer Order Delivery
  Schedule demand
* Customer Order demand
* Inventory demand
* Work Order demand
* Work Order Material Requirement
  demand
* Interbranch Transfer demand
