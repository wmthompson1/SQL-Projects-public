# Adding Ship Via Codes

Use the Ship Via Table to enter or delete Ship Via
codes. Ships Via codes represent the various methods you may choose
to ship product to your customers.

1. Select Admin,
   Application Global Maintenance.
2. Select Maintain,
   Table Maintenance, Ship
   via Table.
3. Click Insert.
4. Specify a code and a
   description.
5. Click Save.
