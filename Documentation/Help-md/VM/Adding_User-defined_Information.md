# Adding User-defined Information

To specify the user-defined field values:

1. Access Activity Entry.
   See [Editing
   Activities in Activity Entry](Editing_Activities_in_Activity_Entry.md) or [Creating
   Activities](VMACTENTfrmActivityEntry.md).
2. Select Edit,
   User-defined Fields. Select the label
   set to use, and then specify the values as required by your company.
