# Printing Customs Documents

After creating a pack-list for a shipment, you can
print customs documentation to comply with international shipping
regulations. The .qrp file for this report is vmcusdoc.qrp.

1. From the VISUAL main
   menu, select Shipping Entry.

The Shipping Entry window appears.

2. From the File menu,
   select Print Customs Documents.

The Print Customs Documents dialog box appears.

All pack lists eligible for printing as
customs documents appear in the table.

The Print Customs Documents dialog box contains
the following fields:

Customs Doc ID
- The ID of the customs document, which is the same as the Pack list
ID.

Pack list ID -
The Pack List ID for the order, which is the same as the Customs Doc
ID.

Customer ID -
The ID of the customer who placed the order.

Ship To ID - If
applicable, the Ship To ID for the shipment. If there is no Ship To
ID, VISUAL uses the default customer address.

Total Value -
The total value of the shipment.

A/R Invoice ID
- The A/R Invoice ID for the shipment, if VISUAL has created one.
If you chose to consolidate the Pack List, ASN, and Invoice ID at
the time of shipment, the A/R Invoice ID is the same as the Pack List
ID.

3. Select which pack lists
   you want to print. Multiple selections are possible. To select
   multiple, non-contiguous rows, select lines while holding down
   the CTRL key. To select a continuous
   range, select the first row in the range, press the SHIFT key,
   and then select the last row in the range.
4. Select a print output
   for the customs document:

Print -Select
Print output the customs document to a printer.

View - Select
View to view the customs document on your
screen.

File - Select
File to save the customs document to a directory
on your local machine or on the LAN.

E-mail - Select
E-mail to send the customs document to a
specified electronic mail recipient.

5. Click the Ok
   button to output the report.
