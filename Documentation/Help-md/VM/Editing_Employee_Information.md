# Editing Employee Information

You can edit employee information using Employee Maintenance.

1. Enter the Employee ID
   in the Employee ID field, or select an employee from the Employees
   Table.
2. Make any necessary changes
   to the employee information.

You can change any field in the Employee
Maintenance window except the Employee ID. Changing the Employee ID
creates a new employee.

3. Click the Save
   button to save your changes.
