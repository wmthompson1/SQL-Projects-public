# What is the Estimating Window?

Use the Estimating Window to create and maintain quotations
to present to prospective customers of your company. The Estimating
Window allows you to:

* Estimate costs for a quote
  of one or more parts or jobs, each can be quoted for one or more
  quantities. The unit cost estimates that are generated accurately
  reflect the relationship between fixed costs (ex: setup labor
  and fixed charges for services), and variable costs (ex: run labor
  and material costs). Quantity breaks for material unit costs can
  also be taken into account. If you want, you can view cost details
  at the individual operation level.
* Define "skeleton"
  engineering masters using the Quick Quote feature, to provide
  a basis for estimating custom jobs.
* Price a quote based on calculated
  costs and specified markups. Markups can be individually defined
  for manufacturing and General Services Administration (GSA), and
  for each separate cost category.
* Determine best quantities
  to quote, if desired, based on available unit price breaks for
  materials.
* Print a quotation suitable
  for transmittal to the prospective customer.
* Print a Proforma Report
  of projected revenue, for a given time period, sales rep, or territory,
  based on a specified cutoff probability.

You can quickly convert a quotation prepared in the Estimating Window
into a customer order and work order, using the Customer Order Entry
application.

If you are licensed to use multiple sites, enter quotes on a site-by-site
basis. Estimates you enter in one site can only be converted to orders
in the same site. If you are licensed to use a single site, enter
quotes on an enterprise-wide basis.
