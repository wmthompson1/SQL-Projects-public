# Loading Infinite Schedules

You can quickly display an infinite schedule that
you defined using the Global Scheduler by selecting the Load Infinite
Schedule option from the File menu. The
schedule is added to the existing schedule.

If you have multiple infinite schedules defined, VISUAL picks and
displays the first infinite that occurs in the database. If you select
the option again, VISUAL adds the next infinite schedule to the display.
