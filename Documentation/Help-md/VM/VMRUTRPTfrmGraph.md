# Viewing Resource Percentage Graphs

You can view your Utilization reports in the form
of a graph by clicking on the Graphs option on the View menu.

After you have made your Utilization Report settings, and opened
the Graph dialog box, you can make the following changes to your report:

Site
ID - Click the Site ID arrow and select a site to use.

Starting
Date - You can change the date on which VISUAL will report
your resources.

Show As - You can select either the Utilization
or Unscheduled
Time options.

Shop
View - You can change which group of resources you want VISUAL
to use for your report.

Resource
- You can change the resource you want VISUAL to show in your report.

Schedule
- You can change he schedule you want VISUAL to use for your report.

Graph
Type - You can change the type of graph on which you want to
view your information.
