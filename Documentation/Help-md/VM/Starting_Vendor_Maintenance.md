# Starting Vendor Maintenance

Vendor Maintenance is available from the Infor VISUAL
Main window.

To access the Vendor Maintenance window:

1. Select Vendor
   Maintenance from the Purchasing menu.
2. The Vendor Maintenance
   window appears with no information in it.
3. The Vendor Maintenance
   window contains both entry fields and action buttons.
