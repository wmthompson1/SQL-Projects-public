# Closing Accounting Periods (Global Financials License)

When you have completed transaction processing for
a financial period, close the period. After you close a period, new
transactions cannot be posted to it.

1. Select Accounting,
   Close/Reopen Period.
2. In the Site ID field,
   select the ID of the site whose period you are closing.
3. In the End Date of Beginning
   Acct Period field, specify a date in the period that you are closing.
   The date is adjusted to the last date of the period. After you
   select a date, the ID of the fiscal period and the current status
   of the period are displayed.
4. To check for unposted
   transactions and posting candidates, click Test.
   For each journal type, the number of unposted transactions and
   posting candidates are displayed. All posting candidates are displayed
   regardless of the accounting period. Before you can close a period,
   you must post all unposted transactions. Use WIP Maintenance to
   post WIP Maintenance transactions. Use Project Journal Adjustment
   Entry to post project adjustments. Use Post Manufacturing Journals
   to post all other transaction types.
5. Click the Close
   Period button.

## Reopening Accounting Periods

Reopening a period may be necessary for a variety
of reasons. For example, you may have discovered a batch of un-posted
transactions for a period that has already been closed, or you may
have to post adjustments to a closed period.

VISUAL allows you to reopen a closed period using this feature.
You cannot however, open a locked period or reopen a locked period
from the Close/Reopen Period dialog box. If you need to open a locked
period or reopen a locked period, use Calendar Maintenance.

The Reopen Period function allows you to consciously reopen the
period to post the transactions then use the Close Period function
to close the period again preventing it from being posted inadvertently.

1. From the directory where
   your executables are installed, execute VmClsPer.exe.

The Close/Reopen Maintenance window appears.

2. Click the Site
   ID arrow and select a site to use.
3. When the Close Period
   window appears, the last day of the current (active) period displays
   in the End date of beginning acct period field. Enter the date
   within the period that you want to open, or select a date using
   the Calendar object. Regardless of what date you specify, VISUAL
   converts it to the last day of that period. When you Tab out of
   the field, the Fiscal Period for the date appears. The Status
   for the period displays below it.
4. Click the Reopen
   Period button to open the closed period.
