# Starting the Generate VMDI Layouts Window

You need to define the .VDI file formats using the
Generate VMDI Layouts window before VMDI can support any import or
export activity. Because the necessary data in any given application
depends on the destination or source, VMDI requires you to indicate
exactly which data fields it will use. While you are responsible for
specifying the fields you need from each table, VMDI controls the
sequence of the fields in the tables of each file layout.

To start the Generate VMDI Layouts window, select VMDI
Generate from the EDI menu.

The Generate VMDI Layouts window appears.

The Generate VDMI window contains buttons, pull-down menus, option
buttons and tabs.
