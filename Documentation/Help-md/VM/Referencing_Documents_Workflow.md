# Referencing Documents

The Workflow Document Reference dialog box allows
you add supportive documentation, such as instructions, specifications,
or CAD drawings to a workflow step.

To reference documents in a workflow, you must first add the document
in VISUALs Document Maintenance.

1. In the Step Properties
   dialog box, click the Document Reference
   button.

The Workflow Document Reference dialog box
appears.

2. In the Workflow Document
   Reference dialog box, click the Insert
   button.
3. Enter the ID of the
   document you want to reference or double-click the Document ID
   column heading to open the List Documents dialog box where you
   can select a document ID.
4. Click the Save
   button.
