# Changing Program Security Settings

Because you may not want all of your users to have
access to your Engineering Change Notices, you can use VISUALs Program
Security window to limit access to the individual ECN modules.

You can change access to program components for ECN Entry, ECN,
History, and ECN Task Maintenance.

For more information on limiting access to ECN features, refer to
your VISUAL documentation.
