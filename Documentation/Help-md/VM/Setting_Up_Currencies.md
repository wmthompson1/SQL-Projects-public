# Setting Up Currencies

You can define each currency known to the system (including
the systems own native currency) using Currency Exchange Rate Maintenance.

To define currencies:

1. Select Currency
   Maintenance from the Edit menu.

The Currency Table window appears. It has
a line for each currency defined in the system.

2. Click the Insert
   button to add a new line to the end of the table.
3. Enter a unique ID for
   the new currency in the Currency ID field.
4. Enter a long description
   for the currency in the Name field.

This description appears in windows such
as Purchase Order Entry and Customer Order Entry to show the currency
for the currently selected vendor or customer.

5. Enter a short description
   for the currency in the Short Name field.

This is used in various reports.

6. Select the country of
   the currency from the Country drop down menu. If the country for
   the currency you are adding does not appear in the list, select
   User Defined.

Selecting a country from the list automatically
sets the display symbol and formats for the currency. Selecting Default
selects Windows default currency formatting.

7. If you selected User
   Defined as the country, you must now define the currency format
   and symbol.

Currency Format
- Specifies whether currency symbol precedes or follows the amount,
and whether a space is placed in between.

Negative Currency Format
- Specifies same as Currency Format, plus display of negative amounts.

Currency Symbol
- Specifies the symbol used to represent the currency.

Thousands Separator
- Specifies the symbol used to separate each group of three digits
in an amount. Normal values are a comma or period.

Decimal Digits
- Specifies the standard number of digits to the right of the decimal
point.

8. Click the Save
   button to save the currency.
