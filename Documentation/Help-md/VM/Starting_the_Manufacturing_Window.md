# Starting the Manufacturing Window

The Manufacturing Window is available from the VISUAL
Main window.

To access the Manufacturing Window:

1. Select Manufacturing
   Window from the Eng/Mfg menu.

The standard Open dialog box appears to
allow you to search for and open an existing master/work order, or
create a new one.

2. Create a new master/work
   order or open an existing master/work order.

Creating of masters and work orders is covered
later in this section.

If you wish to open an existing demonstration
master or work order to use while reading the next section, complete
the following steps.

3. Select the appropriate
   Type radio button for Engineering Master, Work Order, or Quote
   Master. You must select the proper type an engineering master
   does not open if you select Work Order. You can specify a default
   setting for the type. See [Specifying
   the Default Document Type in the Open Dialog](Specifying_the_Default_Document_Type_in_the_Open_Dialog.md).
4. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site to use. Masters are created by site.

If you are licensed to use a single site,
this field is unavailable.

5. Select the appropriate
   predefined Size Preference radio button.
6. Select the appropriate
   viewing mode radio button. Select from the following: Graphical
   (1), Graphical (2), Text (1), and Text (2).
7. Enter the appropriate
   IDs.

Enter the Base ID, Lot ID or Engineering
ID, and Split ID (work orders only).

At this point, you can also choose to Search,
rather than enter the IDs manually. You can also create a new master/work
order in this dialog box by clicking on the New
button.

8. Click the Ok
   button.

The master/work order appears in the Manufacturing
Window.
