# Setting Purchase Receipt Thermal Label and Receiver Print Options

Configure print options so that thermal labels and
receivers print automatically after VBTS processes a transaction and
assigns it a status of Completed.

1. With the VISUAL BTS
   Purchase receipts window open, select Print
   Options from the File menu.

The BTS Print Options dialog box appears.

2. Select the Auto
   Print Thermal labels check box if you want VBTS to print
   thermal labels for all purchase receipt transactions when they
   reach a Completed status.
3. Select the Auto
   Print Receiver check box if you want VBTS to print receivers
   for all receipt transactions when they reach a Completed status.
4. Click the Save
   button to commit these settings.
