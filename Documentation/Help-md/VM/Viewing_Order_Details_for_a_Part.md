# Viewing Order Details for a Part

After you select a part, you can click anywhere on
the following lines that appear in the Advanced Material Planning
Window line item table to view information regarding the current part.

If you are licensed to use multiple sites, click the
Site ID arrow and select the site to use. If you are licensed to use
a single site, this field is unavailable.

* Customer Demand
* Dependent Demand
* Current MPS
* Current MRP Plan
* Current Firmed Orders
