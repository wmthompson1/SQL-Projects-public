# Starting Product Configurator Interface Maintenance

The Product Configurator is available from the VISUAL
Main window.

To start the Product Configurator:

Select Configurator Interface Maintenance
from the Sales menu.

The Product Configurator Interface Maintenance window appears.

The Product Configurator Interface Maintenance window contains the
following fields.

Site ID - If you are licensed to use multiple
sites, the Site ID associated with the product configuration. If you
are licensed to use a single site, this field is unavailable.

Series ID - A Series ID for each configuration.
An entry is required.

Model ID - A Model ID for each configuration.
An entry is required.

Part ID - The Part ID for the fabricated
part, which corresponds to the series/model combination to be configured.
An entry is required.

Eng ID - Eng ID for the part entered.
An entry is required.
