# Using Preference Maintenance

You can use the Preference Maintenance window to:

* [Load
  any existing preference values from the VISUAL.ini file](Loading_Values_from_the_VISUAL.ini_file.md)
* [Set
  your own preferences](Adding_Preferences_for_Individual_Users.md)
* [Print
  a list of your preferences](Printing_a_Preferences_Report.md)

In addition to the above tasks, the system administrator can:

* [Set
  default preferences](Adding_Default_Preferences.md)
* [Set
  preferences for individual users](Adding_Preferences_for_Individual_Users.md)
* [Set
  up preference profiles and apply them to individual users](Creating_Preferences_Profiles.md)
* [Set
  up scheduling preferences](Setting_Up_Scheduling_Preferences.md)
* [Print
  a list of global and individual user preferences](Printing_a_Preferences_Report.md)

To access Preference Maintenance, select Admin,
Preference Maintenance.
