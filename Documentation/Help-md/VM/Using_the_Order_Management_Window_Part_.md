# Using the Order Management Window Part Alternate Warehouses Window

You must enter alternate warehouses and associated
ShipTo IDs in Customer Maintenance. The entry of alternate warehouses
are for customers, not for specific parts.

Part Alternate Warehouses are warehouses that also, in addition
to the parts primary warehouse, contain the part in the customer
order: They are potential sources of order fulfillment if the quantities
in the primary warehouse are insufficient, or expected, incoming supply
is already committed or unavailable for some reason.

From the Tools menu of the Order Management Window, select Part Alternate Warehouses.

|  |  |
| --- | --- |
| POSTIT.gif | In order to use the Part Alternate Warehouses for Part window in general, and the windows drag n drop feature, you must have the Current Customer Order window open as well. The Part Alternate Warehouses for Part window is useless unless you have an order open for a customer for whom you have entered alternate warehouses in Customer Maintenance. Also, if either you or VISUAL have already allocated supply to the customer order line, you cannot use this window to allocate supply, as this would create an over allocation of supply to a line that has already been fulfilled. |

The Part Alternate Warehouses for Part dialog box appears.

The IDs of the alternate warehouses appear in the right pane.
