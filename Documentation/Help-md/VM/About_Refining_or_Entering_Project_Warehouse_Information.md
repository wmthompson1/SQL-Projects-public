# About Refining or Entering Project Warehouse Information

If you created a project warehouse when you created
the project, you may want to alter the warehouses information through
Warehouse Maintenance. Otherwise, the project warehouse you created
will have the same information as your project warehouse template.

If you chose not to create the project warehouse when you created
the project, you must create a project warehouse through Warehouse
Maintenance with the same ID as the project.
