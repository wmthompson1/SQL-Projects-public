# Establishing Unplanned Maintenance Work Order Defaults

Enter work order defaults for status, rank, and specify
an internal and external engineering master for VISUAL to use when
processing new work orders.

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site for which you are setting up defaults. You must select a
   site to define the default internal and external work orders for
   the site. If you are licensed to use a single site, this field
   is unavailable.
2. From the Options menu,
   select Preferences, Work
   Order Defaults.
3. Click the Internal
   button to select an engineering master for VISUAL to use for all
   internal (taking place within the shop, or on-site) maintenance
   jobs.

A standard VE search table appears with
all valid VISUAL engineering masters. VISUAL requires that you specify
a default Engineering ID before you can begin processing any work
orders.

4. Click the External
   button to select an engineering master for VISUAL to use for all
   external (taking place outside the shop, or off-site) maintenance
   jobs.

A standard VE search table appears with
all valid VISUAL engineering masters. VISUAL requires that you specify
a default Engineering ID before you can begin processing any work
orders.

5. Select a default status
   for work orders:

Release - Released
work orders are work orders on which you are ready to begin working.
They have been released" to the shop floor.

Firm - Firm work
orders are work orders existing more for planning purposes. Though
they are real work orders, until they are released, the VISUAL Concurrent
Scheduler does not schedule them as it does released work orders.
Scheduler and planners use this designation as a work order place
holder.

6. Specify a rank for work
   orders:

Internal Rank
- Select a default priority for the purpose of scheduling work orders
for internal equipment unplanned maintenance.

External Rank
- Select a default priority for the purpose of scheduling work orders
for external equipment unplanned maintenance.

7. Click the Ok
   button to commit information.
