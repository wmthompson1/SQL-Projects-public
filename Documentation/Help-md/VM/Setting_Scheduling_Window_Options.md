# Setting Scheduling Window Options

You can set a variety of options to customize the
Scheduling Window to your needs. You can find many of these options
under the Options menu.

All of the selections under the Options
menu toggle between being active and inactive. A check mark to the
left of the menu selection indicates that it is currently active.
Select a menu option to activate it; select the menu option again
to deactivate it.

Many of the menu selections under the Options
menu also have corresponding buttons on the Toolbar.
