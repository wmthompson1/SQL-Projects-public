# Sign in

When you access Infor VISUAL from your Start menu
or desktop, you receive Infor VISUAL Sign In dialog box. This dialog
box prompts you to enter the database you want to use, your user name,
and your password.

The Sign In option from VISUAL allows
you to sign onto another users system without exiting the Infor VISUAL
application. When you access this option, you must complete the sign
in process; if you cancel out of it, Infor VISUAL closes, and you
must re-start the application from your desktop or Start menu.
