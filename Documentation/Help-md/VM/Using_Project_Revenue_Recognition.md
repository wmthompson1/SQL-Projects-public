# Using Project Revenue Recognition

To use the Project Revenue Recognition window:

1. To start the Project
   Revenue Recognition window, select it from the Project menu of
   VISUAL's main window.
2. Click the Transaction
   ID arrow and select the transaction you want to view. You can
   also select a Project ID.

The revenue totals appear in the tab and
the breakdowns appear in the table.
