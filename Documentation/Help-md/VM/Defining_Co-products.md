# Defining Co-products

You can define co-product parts manufactured on this
work order. You can define quantities and percentage of cost.

1. Click the Co-Products
   button.

The Co-products dialog box appears.

2. Click the Insert
   button to add a new co-product.

A new line appears in the line item table.

3. Enter the Part ID for
   the co-product.
4. Enter the percentages
   and the quantity in the appropriate columns.
5. Click the Save
   button to save the new co-product.
