# Starting Labor Ticket Entry

Labor Ticket Entry is available from the VISUAL Main
window.

To start Labor Ticket Entry:

Select Labor Ticket Entry from the Eng/Mfg menu.

If you are licensed to use multiple sites, you must first assign
employees to sites before they can enter labor tickets. After you
add employees to sites, the employees can enter labor tickets for
work orders created in their assigned sites.

## Setting Preferences

You can set various Labor Ticket Entry preferences.

1. Select Administrator
   Setup from the File menu.

2. Select these options
   in the dialog:

Automatically Compute
Quantity Complete - Causes quantity complete to be automatically
calculated based on elapsed time.

Use Decimal Time Format
- When entering time in quick 4 digit format, if this box is checked,
the second two digits are interpreted as hundredths of an hour. Otherwise,
theyre interpreted as minutes.

Edit Qty Complete vs.
Standard - If you select this option, VISUAL warns you if the
quantity you report on a labor ticket is above the system calculated
operation quantity, plus the allowable percentage over standard. When
you select this option, the Allowable Percentage Over Standard field
appears. Enter the percentage.

Allow reporting to obsolete
resources Select this option to allow users to report labor
to obsolete resources. If you select this option, the system warns
users that the resource they chose is obsolete. Clear this option
to prevent users from reporting labor to obsolete resources.

3. Click Ok
   to set the preferences.
