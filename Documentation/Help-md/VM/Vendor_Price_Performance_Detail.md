# Vendor Price Performance Detail

This [dashboard](VMDBOARDWhat.htm) detail
grid compares the costs of materials and services received from vendors
with the amounts invoiced for the receipts.

Tip: Group this grid by Purc Order ID,
and then sort the grid by PO Ln No to group received lines for a purchase
order with invoiced lines for a purchase order.

Date filter applies to: Invoice Date

Filters: Type (Received or Invoiced),
Vendor ID, Site ID

Red text: n/a

Details: Type (Received or Invoiced),
Invoice Date, Due Date (Invoices only), Vendor ID, Vendor Name, Voucher
ID (Invoices only), Line No (Invoices only), Purchase Order, PO Ln
No, PO Del Ln No, Received Qty, Purchase U/M (Received only), Amount
Received (for Received lines, the amount on th purchase order line.
For Invoiced lines, the amount invoiced), Currency ID, Site ID.
