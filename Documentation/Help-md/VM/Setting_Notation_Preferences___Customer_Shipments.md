# Setting Notation Preferences Customer Shipments

You can set the notation preferences in Shipping Entry.

1. Select Notation
   Preferences from the Notes menu.

The Notation Preferences dialog box appears.

2. Select the check boxes
   to enable/disable the following:

Temporary Positioning
of Windows - If you move the Notations box to a different location
on your screen, VISUAL saves the new location.

Temporary Sizing of Windows
- If you resize the Notations box on your screen, VISUAL saves the
new size.

3. Click the Ok
   button to set the preferences.
