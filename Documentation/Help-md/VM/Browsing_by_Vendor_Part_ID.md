# Browsing by Vendor Part ID

If you would like to search for a purchase order by vendor part
ID, use the Browse Purchase Orders by Vendor Part ID dialog box.

Select Edit, Browse
Purchase Orders by Vendor Part ID. The system lists the purchase
orders sorted by vendor part ID. Double-click a line in the table
to open the purchase order.
