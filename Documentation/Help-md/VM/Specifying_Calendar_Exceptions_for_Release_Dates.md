# Specifying Calendar Exceptions for Release Dates

If you want Material Requirements Planning to take
your calendar exceptions into account when calculating release dates,
select the Use Global Calendar and Exceptions for
Release Dates check box. By using this setting you can specify
the dates you do not want VISUAL to use in its calculations.
