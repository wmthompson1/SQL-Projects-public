# Changing the Active Status of Customers

By making a customer's account Inactive you can hide
it so that your users cannot use it in any transactions.

Only users with administrator privileges may make changes to inactive
customers.

If you want to view the status of accounts in the search dialog
box, select Edit Columns from the Options
menu and select the Active\_Flag check box
in the Include column.

If you have administrator privileges and you want to change the
active status of an account:

1. Open the Customer Maintenance
   window.
2. Click the Customer
   ID button and select the appropriate customer from the
   list.
3. Select or Clear the
   Active check box.
4. Click the Save
   toolbar button.
