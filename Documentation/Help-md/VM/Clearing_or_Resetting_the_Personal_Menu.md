Clearing or Resetting the Personal Menu




# Clearing Personal Menu Settings

To remove all groups and commands from your Personal
Menu:

1. Select File,
   Personal Menu from the main VISUAL menu.
2. Right-click and select
   Customize.
3. Click Clear.
   All groups and commands are deleted from the menu.
4. Click Ok.
