# Deleting Notation Entries

To delete an entry:

1. Use the navigation arrows
   to the left of the dialog box to move to the entry that you want
   to delete.
2. Click the Del
   button.

A note appears next to the entry stating
that the entry is marked for deletion.

3. Click the Save
   button to remove the entry.
4. Click the Close
   button to close the Notations dialog box and return to the Estimating
   Window.
