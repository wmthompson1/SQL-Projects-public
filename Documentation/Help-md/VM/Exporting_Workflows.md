# Exporting Workflows

Use the Export menu to save the currently open workflow
(including all step and rule properties) to a file on your computer.
The exported file will have a .dia file extension.

To export a workflow:

1. If necessary, open VISUAL
   Workflow.
2. On the File menu, select
   Export.
3. In the Export File window,
   select a location to save the file, type a file name; then click
   the Save button.

VISUAL saves the workflow file to the selected
location.
