# Starting Warehouse Maintenance

Warehouse Maintenance is available from the VISUAL
Main window.

To access the Warehouse Maintenance window:

From VISUALs main window, select the Warehouse
Maintenance option from the inventory menu.

The Warehouse Maintenance window appears.
