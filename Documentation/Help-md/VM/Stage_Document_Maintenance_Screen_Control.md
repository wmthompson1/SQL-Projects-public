## Stage - Document Maintenance

From the Stage list, select a revision stage code. Stage codes apply
to records controlled by manual (you enter a Revision ID) or predefined
(VISUAL generates numbers based on your preferences) revision numbers.

To create stage codes, use [revision
numbering maintenance](Maintaining_Revision_Numbering.md).

If the stage code you select has any predefined revision numbers assigned
to it, VISUAL displays a browse for the Revision field.
