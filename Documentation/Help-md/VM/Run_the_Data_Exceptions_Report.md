# Run the Data Exceptions Report

Before closing the period, use the data exceptions report to identify
missing or invalid entries in your project-related data. These data
errors frequently cause VISUAL's costing utilities to halt. Use the
Data Exceptions report to address these errors before running the
costing utilities.
