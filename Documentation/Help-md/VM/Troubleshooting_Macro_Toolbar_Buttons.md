# Troubleshooting Toolbar Buttons for Macros Stored in the Database

If the macro attached to a toolbar button cannot be
found, this message is displayed: Macro name is not valid. The reasons
you may receive this message include, but are not limited to:

* A system administrator has
  removed the macro from your profile
* The SYSADM user has deleted
  the macro
* The macro button refers
  to a macro stored on the workstation, but you now store macros
  in the database.

If you no longer use the macro, you can remove the toolbar button.
If you still use the macro, contact your system administrator to restore
your access to the macro.
