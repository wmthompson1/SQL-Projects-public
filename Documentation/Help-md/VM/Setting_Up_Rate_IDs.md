# Setting Up Rate IDs

If you are licensed to use multiple sites, set up
rate IDs at the tenant level. The rate IDs you specify can be used
in all sites. You can specify different values for each cost/burden
category ID in each site.

To set up Rate IDs:

1. From the Project menu,
   select Rate Management. The Rate Management window appears.
2. From the Maintain menu,
   select Rate Table.

The Maintain Rate ID window appears.

3. Click the Insert
   button.

A new row at the bottom of the table appears
highlighted. The cursor appears in the ID column.

4. Enter the identification
   you want to use for this rate.

Click in the Description column and enter
a description for this rate.

5. When you have finished
   entering Rate IDs, click the Save button.

The Maintain Rate ID window closes and VISUAL
saves the changes to your rate list to your database.
