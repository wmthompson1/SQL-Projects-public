# Using the Tree Display

If the Graphical Display is displayed when you open the Document
Lifecycle Viewer, select View, Show
Graphical Display to clear the check box next to Show Graphical
Display. The Tree Display is displayed.

In the Tree View, you can perform these actions:

* [View
  document details](Document_Lifecycle_Viewer.htm)
* [Configure
  document details information](Configuring_Lifecycle_Document_Details.htm)
* [View
  document details in separate tables](Viewing_Document_Lifecycle_Details_in_Separate_Tables.htm)
* [Open
  a related document](Opening_a_Related_Document_in_the_Document_Lifecycle_Tree_View.htm)
* [Print
  the Document Lifecycle Report](Printing_the_Document_Lifecycle_Report.htm)
