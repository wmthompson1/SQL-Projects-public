# Deleting Items in Lists

To delete a line item from a list:

1. Open the appropriate
   list.
2. Click the row header
   fro the line item you want to delete.

The row appears highlighted.

3. Click the Delete
   button.

An X appears in the row header indicating
you have marked the line item for deletion.

4. Click the Save
   button.

VISUAL removes the line item from your database.
