# Selecting if Shipment Tracking Reason Codes are Required or Optional

From the Reason Codes section, select either Required or Optional.

If you select Required, VISUAL does not
allow you to print a transportation document of the transaction without
supplying a reason code.

If you select Optional, VISUAL allows
you to print a transportation document of the transaction without
supplying a reason code.
