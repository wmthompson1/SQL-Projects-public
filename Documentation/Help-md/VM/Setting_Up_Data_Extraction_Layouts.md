# Setting Up Data Extraction Layouts

To set up Data Extraction layouts:

1. Click the Data
   of Type arrow and select the data you want to include in
   this layout.

You can select PLN
or SCH

2. Select the file style.

You can select Fixed
or Delimited.

If you select Delimited:

* Click the
  Record arrow and select how you want
  each record to appear.
* Select Newline if you want each record to begin on
  a new line.
* Select Null (Fixed Length) if you want your data
  to appear all on one line.

3. Click the Field
   arrow and select how you want VISUAL to separate your data fields.
   You can select:
4. Asterisk
5. Comma
6. Tab
7. Null
   (Fixed length)

4. On the Header Tables
   tab, click the arrow and select the table containing data you
   want to include.
5. Click the Add
   button.

The table name appears in the list. To remove
the name from the list, click the Remove
button.

6. On the Line Item Tables
   tab, lick the arrow and select the table containing data you want
   to include.
7. Click the Add
   button.

The table name appears in the list. To remove
the name from the list, click the Remove
button.

8. When you have finished
   adding information to your layout, click the Save
   button.
