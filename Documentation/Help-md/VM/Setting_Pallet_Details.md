# Setting Pallet Details

Use Shipping Entry to enter pallet details for your
shipments. Pallet details include the Pallet ID, the number of cartons
on a pallet, and the weight of a pallet.

1. With the appropriate
   packlist open in the Customer Shipments window, select Pallet
   Details from the Edit menu.

The Pallet Details dialog box appears.

2. Enter the following
   pallet information:

Line # - Enter
a line number for the Pallet ID.

Pallet ID - Enter
an ID for the pallet.

Cartons - Enter
the number of cartons on the pallet.

Weight - Enter
the weight of the pallet.

Pkg Type - Enter
the package type.

Qty per Pkg -
Enter the quantity per package.

3. Click the Save
   button to save the pallet information.

## Viewing Pallet Details on Save

You can view and add pallet details when you save a shipment by
activating the Ask for Pallet Details on Save
option from the Options menu.

When you save the shipment, the Pallet Details dialog box appears
with pallet details information.

To add pallet details, click the Insert
button and add pallet information.

To delete pallet details, select the line you want to delete then
click the Delete button.
