# Starting Purchase Receipt Entry

Purchase Receipt Entry is available from the VISUAL
Main window.

To start Purchase Receipt Entry:

Select Purchase Receipt Entry from the
Purchasing menu.

The Purchase Receipt Entry window appears with no information in
it.

The Purchase Receipt window contains both entry fields and action
buttons.
