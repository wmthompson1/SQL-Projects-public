# Setting WIP/VAS Information for Warehouses

|  |  |
| --- | --- |
| IMPORTNT.gif | WIP/VAS (Work In Process/Value Added Service) only applies if you have enabled WIP/VAS in Application Global Maintenance. Click here for more information. |

You must set WIP/VAS information for each warehouse.

1. With the appropriate
   warehouse in the Warehouse ID field, select the WIP/VAS
   supported check box.
2. Click the Save
   button on the main toolbar.

The WIP/VAS button is unavailable until
you save.

3. Click the WIP/VAS
   button.

The WIP/VAS dialog box appears.

4. Enter a Warehouse ID
   or click the Warehouse ID button to
   select a Warehouse ID from the line item table.

The Warehouse ID and description appear
along with any WIP/VAS line items that you may have already entered.

5. Click the Insert
   button to start adding line item information.
6. Enter information in
   the following fields:

Part ID - The
part in the warehouse that you are making eligible for value added
service.

Either enter a Part ID or double-click the
column header to select a part from the PART table.

WIP/VAS ID - The
ID of the value added service or work in process that you are assigning
to the part.

Either enter a WIP/VAS ID or click the column
header and select one from the table. Enter WIP/VAS IDs in Application
Global Maintenance.

Unit Price - The
price per part of the value added service.

The default Unit Price that you established
for this WIP/VAS ID in Application Global Maintenance appears here.
You can, however, change the unit price for each warehouse.

7. Click the Save
   button to commit the new information.
8. Click the Close
   button to return to the Warehouse Maintenance window.
