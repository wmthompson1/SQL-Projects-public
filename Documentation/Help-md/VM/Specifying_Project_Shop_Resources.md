# Specifying Project Shop Resources

When you are selecting resources for the tasks in
your projects, you may find it helpful if only those resources you
have designated as Project Resources appear.

VISUAL uses any overhead rates you enter in Shop resource Maintenance
only for work order estimating purposes. VISUAL does NOT use these
rates when booking labor against shop resources. Maintain all of your
burden rates and mappings in the Project Summary Window.
