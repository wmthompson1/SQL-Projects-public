# Adding National Motor Freight Codes

Use Application Global Maintenance to maintain National
Motor Freight codes.

1. Select Admin,
   Application Global Maintenance.
2. Select Maintain,
   National Motor Freight Codes.
3. Click Insert.
4. Specify this information:

ID - A unique
definition for the code (15 characters maximum).

Article - A classification
of the code (8 characters maximum).

Sub - A sub-classification
of the article (3 characters maximum).

Truck Load Class
- A class for a full truck load (5 characters maximum).

Less Than Truck Load
Class - A code for a partial truck load.

Hazardous - A
check box to indicate that the parts consigned under the code consist
of hazardous material.

Description -
A description of the types of parts to be transported under this code
(250 characters maximum).

5. Click Save
   button.
