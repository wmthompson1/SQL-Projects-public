# Entering a Default Ship To ID and a Primary Warehouse for a Customer

Although it is not imperative that you define a primary
warehouse and a default ShipTo ID for each customer, it is a good
idea to do so. Ship To IDs and primary warehouses are two basic components
of a customer order. Set these two defaults in Customer Maintenance.

1. Open the Order Management
   Window and select the customer for whom you want to set these
   two defaults.
2. Click the Customer
   Maintenance toolbar button. You can also right-click the
   window and select Customer Maintenance from the pop-out menu.

The Customer Maintenance window appears.

3. In the General tab,
   enter a Ship To ID for the customer. Click the Default Shipto
   ID button if you want to choose from a list of IDs already defined
   for the customer.
4. In the Order Mgt tab,
   click the Primary Warehouse button and
   select a primary warehouse for the customer.
5. Click the Save
   button on the main toolbar.
6. Now refresh the Order
   Management Window display to verify the customers default Shipto
   ID and Primary Warehouse. This information appears in read-only
   fields on the Order Mgt tab in the Order Management Window.
