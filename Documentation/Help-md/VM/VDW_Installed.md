## VDW Installed

Unless the [Allocations
Require Warehouse/Location](Enabling_Warehouse_Location_Requirements_for_Allocations.md) option is selected in Application Global
Maintenance, this option is unavailable.

If this warehouse is a VISUAL D
istribution Warehouse, select the VDW
Installed check box.

With this setting active, you must use VISUAL Distribution to perform
shipping and receiving transactions. If you specify a VDW warehouse during
the course of a common Infor VISUAL transaction, VISUAL alerts you to
the fact that the warehouse you just specified is a VDW warehouse and
stops the transaction.
