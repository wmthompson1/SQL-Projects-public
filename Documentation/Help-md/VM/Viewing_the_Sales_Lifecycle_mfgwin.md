# Viewing the Purchasing Lifecycle

If a work order is linked to a customer order, you can view the
[sales lifecycle](Sales_Lifecycle.md) for the order. To
view the lifecycle:

1. Select Eng/Mfg,
   Manufacturing Window.
2. Open the work order.
3. Select Info,
   Sales Lifecycle....
