# Creating a New ETC from an Existing ETC

If you would like to create a new ETC for a project that is identical
to an existing ETC for the same project, you can use the Copy To...
function. You can then update the new ETC as necessary.

1. In the Maintain ETC
   window, click the Project ID button
   to select the project associated with the ETC you would like to
   copy.
2. Click the ETC
   Version button to select the ETC you would like to copy.
3. Click the Copy
   To... button. The Copy ETC window appears.
4. In the Copy to ETC Version,
   enter the ID you would like to use for the new version. Leave
   the field blank if you would like the system to generate an ETC
   version ID based on the settings in the Auto Numbering - EACs
   and ETCs window.
5. Click Save.
   The system opens the new ETC in the Maintain ETC window.
