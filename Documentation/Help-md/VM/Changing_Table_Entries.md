# Changing Table Entries

1. Select the unit of measure
   you need to edit in the To table.

The line selected becomes highlighted, indicating
that you can edit it.

2. Move the cursor to the
   appropriate entry and make the change.

A check mark appears to the left of the
unit of measure.

The From table does not immediately reflect
the changes you make in the To table. First, you must save your changes.

3. Click the Save
   button.
