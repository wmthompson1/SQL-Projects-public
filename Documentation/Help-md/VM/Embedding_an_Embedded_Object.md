# Embedding an Embedded Object

From the Manufacturing Window object window, select
Picture/Object from the Edit
menu.

The embedded object appears in the Set Picture/Object window.

1. Double click the object
   in the picture window (or select Edit Object
   from the Edit menu).

The application opens with the embedded
object loaded and ready for editing.

2. Make the appropriate
   changes to the document and exit the Server application.

When exiting the application, you may have
the opportunity to update or cancel changes to the embedded object.
Select the Update Object option to return to the Manufacturing windows
with the edited object.
