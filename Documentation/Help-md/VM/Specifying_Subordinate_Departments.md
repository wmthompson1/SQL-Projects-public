# Specifying Subordinate Departments

You must be licensed to use Project/A&D functionality
to complete this procedure.

For reporting purposes, you may find it useful to
have VISUAL report a projects budget and actual costing figures to
a single department.

To set up subordinate departments:

1. Complete one of these
   tasks:

* Select Admin, Application Global
  Maintenance. Then, select Maintain,
  Accounting Tables, Department
  Table.
* If you are
  licensed to use Project/A&D functionality, select Project,
  G/L Burden Mapping Maintenance. Then,
  select Maintain, Departments.

2. Select the parent Department
   ID and click the Subordinate Departments
   button.
3. Click Insert.
4. Click the Department
   ID browse button and select the subordinate department.
   VISUAL reports costs from subordinate departments to the parent
   department.
5. Click Save.
6. Click Close.
