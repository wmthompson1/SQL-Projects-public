# Customizing User Defined Format Labels

If you have created user-defined report formats for invoices, you
can customize the labels that are displayed for the formats.

To customize the labels:

1. Select Admin,
   Preferences Maintenance.
2. Click the Insert
   button.
3. Specify this information:

Section Specify
InvoiceGeneration.

Entry Specify
the user-defined label to change. Specify RenameInvUD1, RenameInvUD2,
or RenameInvUD3.

Value Specify
the new label for the user-defined report.

4. Click the Save
   button.
