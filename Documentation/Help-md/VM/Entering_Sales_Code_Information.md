# Entering Sales Code Information

To enter the Sales Code Information:

1. Highlight the row containing
   the appropriate series/model and click the Sales
   Codes button.

The Edit Sales Codes dialogs box appears.

2. Click the Insert
   Line button.

A new line appears in the table.

3. Enter the Sales code
   and corresponding Part ID in the Sales Code and Part ID fields.

The part you select must have an Engineering
Master. You can select the Engineering Master Engineering ID from
the Engineering Master browse window.

4. If necessary, edit the
   copy sequence in the Copy Sequence field.

By default, copy sequence is 0.

5. Click the Save
   button to save the new information.
