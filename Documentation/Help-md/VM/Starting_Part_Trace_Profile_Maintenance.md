# Starting Part Trace Profile Maintenance

Part Trace Profile Maintenance is available from the
Maintenance menu of the Infor VISUAL Main window.

To access the Part Trace Profile Maintenance window:

Select Part Trace Profile Maintenance
from the Inventory menu.

The Part Trace Profile Maintenance window appears.
