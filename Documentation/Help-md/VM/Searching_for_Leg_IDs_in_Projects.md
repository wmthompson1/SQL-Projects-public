# Searching for Leg IDs in Projects

Sometimes Projects can be very large and you may want
to find a particular leg that is not currently on your screen. If
you know the identification number of a Leg, using the search feature
can be easier than manually searching for that leg.

To search for a Leg ID:

1. From the Edit menu,
   select Find Sub ID.

The Find Sub ID dialog box appears.

2. Click in the Search
   For field and enter the Sub ID of the leg you want to locate.
3. If you want to select
   a Leg ID, click the arrow and select an ID from the list.
4. To start the search,
   click the Search button.

If VISUAL finds the Leg ID, the leg "card"
appears at the bottom left of the Project window with an arrow pointing
to it.
