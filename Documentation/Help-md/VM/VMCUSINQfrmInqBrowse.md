# Using the Customer Inquiry Browse Dialog

The Customer Inquiry Browse dialog contains a table that VISUAL
populates with information based on what you enter in the Customer
Inquiry Window. To select a record either double-click the record
you want or click the record and click the Open
toolbar button.
