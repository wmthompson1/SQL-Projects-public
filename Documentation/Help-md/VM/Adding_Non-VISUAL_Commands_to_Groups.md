# Adding Non-VISUAL Commands to Groups

To add non-VISUAL commands:

1. Select File,
   Personal Menu from the main VISUAL menu.
2. Right-click and select
   Customize.
3. Click the Browse
   and Add button.

A search dialog box appears.

4. Locate the program you
   want to add and double-click it.

The Edit Command Item dialog box appears.

5. Complete the following
   fields:

Icon - The icon
for the new menu item. Select an appropriate icon from the icon list
box.

Caption - The
label for the icon. This is optional.

Command - The command line to execute the
program you are adding. This appears by default after you select a
program using the search dialog box.

Arguments - Values
as part of a command you want to include. This is optional.

6. Click the Ok
   button.

The Edit Command Item dialog box closes.

7. When you have finished
   adding new commands, click the OK button.

The Customize dialog box closes and the
new command is added to the bottom of the group you selected in your
Personal Menu.
