# Deleting All Preferences from a User or Template

To delete all preferences from a user:

1. Select Admin,
   Preferences Maintenance.
2. Select Edit,
   Delete User Preferences.
3. Click the drop-down
   button and select the user or template to delete.
4. Click Ok.

The system removes the user or template from Preferences Maintenance.
If you deleted a user, the system removes the user ID from Preferences
Maintenance only; it does not remove the user ID from the database.
