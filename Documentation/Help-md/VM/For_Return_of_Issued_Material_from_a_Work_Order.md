# For Return of Issued Material from a Work Order

This applies when the Issuing/Shipping option in the
When to Apply section is set for the parts traceability profile.

All lot numbers for the Part ID that has an unreceived quantity
remaining appear.

It is unlikely that you would create a new lot number at this point,
as it would have been created upon the initial issue or before that
point.

The system automatically fills in Quantity fields so as to distribute
the total adjustment across the lot numbers in the table. If you wish
to modify issue return quantities, change the Quantity fields as needed.
If the material being returned belongs to another lot number, use
Insert to add a new line, enter that number and the quantity.

You can keep track of how much of the total issue return Quantity
you have accounted for by watching the Numbered Quantity field at
the upper right. This field is the total of all entries in the Quantity
column. You usually want this to equal Total Quantity.

You are allowed to have Numbered Quantity be greater than or less
than Total Quantity. When you press Close,
you are warned, and given the option to cancel or accept the override.
If you choose to override, the issue return Quantity for the transaction
is automatically changed to the Numbered Quantity.
