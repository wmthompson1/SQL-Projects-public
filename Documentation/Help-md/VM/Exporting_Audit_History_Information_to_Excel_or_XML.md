# Exporting Audit History Information to Excel or XML

1. Select Admin,
   Audit Maintenance.
2. Optionally, filter the
   results. See [Filtering Audit
   Results](Filtering_Audit_Results.htm).
3. To send the information
   to Microsoft Excel, select File, Send to Excel. To export the information to
   an XML file, select File, Send
   to XML. See [Sending
   Information to XML](Sending_Information_to_XML.htm).
