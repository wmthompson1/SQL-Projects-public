# Specifying HTS Duty Percentages by Country of Origin

1. Complete one of these
   steps:
2. Select Admin, Application Global
   Maintenance. Select Maintain,
   Harmonization Codes.
3. Select Admin, Accounting Entity
   Maintenance. Click the Order Mgt
   tab. Click HTS Codes.

2. Highlight the line for
   which you want to add duty percent by country information and
   click the Duty by Country button.
3. Click Insert.
4. Specify this information:

Country of Origin
- The country from which the shipment originates.

Duty % - The duty
percent for that country.

5. Click Save.
