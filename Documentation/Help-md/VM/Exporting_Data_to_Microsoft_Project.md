# Exporting Data to Microsoft Project

Use the Infor VISUAL - Microsoft Project Interface
window to export your project information to Microsoft Project.

To export your project information:

1. In either the Project
   window or the Project Summary window, open the project you want
   to send to Microsoft Project.
2. From the File menu,
   select Send to Microsoft
   Project.

The Infor VISUAL - Microsoft Project Interface
appears populated with the data from the project you previously opened.

3. Click the Send to MS Project button.

VISUAL sends the appropriate data to Microsoft
Project, which creates a new MS Project file and populates it with
your VISUAL project information.

If you have previously sent this projects
information to Microsoft Project, VISUAL prompts you with the following
dialog box.

4. If you want to open
   the existing file in Microsoft Project and import the information
   back into your VISUAL project, click the Yes button.
5. If you want to open
   the existing file in Microsoft Project so that you can view the
   information, click the No
   button.
6. If you want to cancel
   the operation and close the dialog box, click the Cancel button.
7. In Microsoft Project,
   click the Save
   button on the toolbar.
