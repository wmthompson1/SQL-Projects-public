# Entering and Deleting Intrastat Country Information

Use the Countries dialog box to define countries,
whether or not Intrastat is required, and what country codes are used
for reporting.

1. Click the Country
   button.

The Countries dialog box appears.

2. Click the Insert
   button to add a new country.

A new line appears in the line item table.

3. In the ID column, enter
   an ID for the new country.
4. In the Description column,
   enter the name of the country.
5. If the new country requires
   Intrastat, click the check box in the Intrastat Required column.
6. In the Intra Country
   column, enter the Intrastat reporting country code for the new
   country.
7. In the ESL Country column,
   enter the ESL reporting country code for the new country.
8. Click the Save
   button to save the new information.

## Deleting Country Information

1. Click the Country
   button.

The Countries dialog box appears.

2. Select the country you
   want to delete, and then click the Delete
   button.

An X appears in the row header beside the
country you are deleting.

3. Click the Save
   button to save the new information.
