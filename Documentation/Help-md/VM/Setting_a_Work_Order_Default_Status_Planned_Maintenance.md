# Setting a Work Order Default Status

Choose a default status for new work orders and an
external and internal rank.

1. Select Work
   Order Defaults from the Edit menu.

The Work Order Defaults dialog box appears.

2. Select the appropriate
   option button.

Firm - VISUAL
creates work orders with a status of firmed. Firmed work orders are
not ready for the floor. Work may not begin on them until their status
moves to Released.

Released - VISUAL
creates work orders with a status of released. Released work orders
are ready for the shop floor; work may begin on them at any time.

3. Enter an Internal and
   External Rank. Use ranks to assign importance to external and
   internal work orders. The VISUAL Concurrent Scheduler uses these
   ranks as priorities when scheduling released work orders. To change
   the priority of individual work orders, use the [VISUAL
   Concurrent Scheduler](Setting_Work_Order_Priorities.md).
4. Click the Save
   button to commit settings.
