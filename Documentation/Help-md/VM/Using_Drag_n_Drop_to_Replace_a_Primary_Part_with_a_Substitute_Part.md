# Using Drag n' Drop to Replace a Primary Part with a Substitute Part

You can replace a part in a customer order with a
designated substitute part by dragging the substitute part from the
Sub stitute Part window into the Current Customer Order window.

1. In the VISUAL Order
   Management Customer Order Entry window, highlight the order line
   that contains the primary part. You are replacing this part with
   the substitute part.
2. If not already open,
   the Order Management Window; then open the Current Customer Order
   Window and the Substitute Parts Window. Tile or cascade the two
   windows so that you can easily perform a drag n drop operation.
3. In the Substitute Part
   Window, double-click the icon.

If there are any, substitute parts for the
primary part appear.

4. Highlight the substitute
   part that you want to replace the primary part in the current
   customer order and hold down the left mouse button until the drag
   n drop box appears.
5. Drag the box into the
   open Current Customer Order Window and release it on top of the
   displaying current customer order.

The VISUAL Order Management Customer Order
Entry Window appears with the primary part line edited to display
the default information for the new substitute part.

For example, if you replace a quantity of
1/4 Plates with an equal quantity of 1/4X3Angle Plates, after you
perform the drag n drop operation and the VISUAL Order Management
Customer Order Entry window appears, the highlighted line that contained
the primary Part ID 1/4Plate now has the substitute Part ID 1/4X3Angle
Plate.

6. Enter a quantity for
   the line and other necessary information and then click the Save button to commit the order.
