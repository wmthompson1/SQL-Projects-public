# Shop Floor Status Report

The Shop Floor Status report displays the work order
status associated with resource details in the shop floor. It displays
the information of Work Order, Part, estimated in quantity, estimated
out quantity, actual quantity in, actual quantity out, scrap quantity,
yield percentage, and available quantity.

To print the Shop Floor Status report:

1. From the Print Production
   Schedule Reports window, click the Shop Floor Status Report button.

The Print Shop Floor Status Report dialog
box appears.

2. If you are licensed
   to use multiple sites, click the Site ID(s) arrow and select the
   sites to include in the report. If you are licensed to use a single
   site, this field is unavailable.
3. Enter the range of work
   orders to include in the report.
4. Select the sequence
   for the report.

You can sequence the report by Work Order
ID or Part ID.

4. Select the status of
   the work orders you want to print.

You can select Unreleased, Firmed, Released,
or Closed.

5. Select the output for
   the report.

Select Print to
output the report to a printer.

Select View to
output the report to your screen for viewing.

Select File to
output your report as a text file. You can then edit the report using
a text editing application.

6. Click the Ok
   button.

If you selected Print
as the output, a standard Windows dialog box appears.

7. Make the appropriate
   selections then click Ok.

VISUAL prints the report.
