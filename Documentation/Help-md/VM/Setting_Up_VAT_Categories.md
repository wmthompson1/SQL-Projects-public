# Setting Up VAT Categories

VAT categories identify the type of item being taxed: material,
service, or other. If you have selected the VAT Categories Required
check box, users are required to select a VAT category for all VAT-related
transactions.

To specify a VAT category:

1. Click VAT
   Category Setup.
2. Click Insert.
3. Specify this information:

ID Specify the
category ID.

Type Specify
the type of item associated with this category. Click the arrow and
specify one of these options:

M Select M if this category is for a material.

S Select S if this category is for a service.

O Select O if this category is for other items.

Description
Specify a description of the VAT category.

4. Click Save.
