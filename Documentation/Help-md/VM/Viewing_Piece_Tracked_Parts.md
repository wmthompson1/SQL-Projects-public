# Viewing Piece Tracked Parts

The Pieces button is only
active if the part you select is piece tracked. To view the individual
pieces for a piece tracked part, click the Pieces button.

The Physical Inventory Pieces Part table appears.

Use this table to view the number of pieces, length, width, height,
and Trace ID of piece tracked parts.
