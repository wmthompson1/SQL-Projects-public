# Selecting Picklist Content Options

In addition to the options listed for the traveller,
you can select the following information for picklists:

## Print Picklist Ops Info

Select this check box to include information about the operation
resource.

## Printing Material Barcodes

If you are using Barcode Material Entry, this option allows printing
of a barcode that encodes work order Part ID with each material requirement
on the picklist. To print a Code 39 barcode you need the Code 39 barcode
font installed on the system printing the traveller.

## Printing Part Locations

Similar to the option for the traveller format, this option prints
all valid part locations, or auto-issue, for each material requirement.

All specified picklists are printed as one print job. Pages break
for work orders if all legs are being printed, all materials for
all legs are printed as one list.
