# Adding Deviation Reason Codes

Use Deviation Reason Codes to specify the reason that
a deviated quantity on a labor ticket was unsatisfactory. Depending
on how sites are set up in [Site
Maintenance](Specifying_Information_on_the_Defaults_Tab_site.md), the use of reason codes on labor tickets may be mandatory.

1. Select Admin,
   Application Global Maintenance.
2. Select Maintain,
   Reasons Maintenance, Deviation
   Reasons.
3. Click the Insert
   button.
4. Specify an ID and description
   of the deviation reason.
5. Click Save.
