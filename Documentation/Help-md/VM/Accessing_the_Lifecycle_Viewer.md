# Accessing the Lifecycle Viewer

To access the Document Lifecycle Viewer, select Info,
Document Lifecycle from any of these applications:

* Purchase Requisition Entry
* Purchase Order Entry
* Purchasing Window
* Purchase Receipt Entry
* AP Invoices
* AP Payments
* Estimating Window
* Customer Order Entry
* Order Management Window
* Return Material Authorization
* Shipment Entry
* AR Invoices
* AR Payments

In the Manufacturing Window, select either Info,
Sales Lifecycle or Info,
Purchasing Lifecycle.
