# Starting the VMDI Exchange Window

The VMDI Exchange window is the application that does
the actual importing or exporting of data. VMDI Exchange imports/exports
the data to and from the .VDI files.

To start the VMDI Exchange window, select VMDI
Exchange from the EDI menu.

The VMDI Exchange window appears.
