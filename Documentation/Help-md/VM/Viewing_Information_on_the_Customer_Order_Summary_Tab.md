# Viewing Information on the Customer Order Summary Tab

Click the Cust Ord Summary tab to view the totals for the customer
orders associated with this project.

Totals are displayed in both the document currency you assigned
for this project and in the entity currency.

You can view the lines from the customer orders by selecting Info, View Customer Orders....

The Customer Orders table lists the lines from all customer orders
associated with the project. To view an individual customer order,
select one of the lines of the customer order you would like to view,
then click the Open Customer Order button. The Customer Order Entry
window opens with information for the selected customer order.
