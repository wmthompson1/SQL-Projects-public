# Printing Shipping Labels

Shipping Entry allows you to print Name/Address labels
with the shipping address of the current packlist.

1. Select Print
   Labels from the File menu.

The Print Labels dialog box appears.

2. Enter the number of
   labels you want to print.

Each label is the same. This provides for
a multiple-container shipment.

3. Select the label format.

There is one standard format and up to three
user-defined ones. The user defined options are only available if
you have created the forms using Centura Report Builder.

4. Click Ok
   to print the labels.
