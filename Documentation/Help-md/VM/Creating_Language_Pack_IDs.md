# Creating Language Pack IDs

1. Select Admin,
   Translation Maintenance.
2. Select File,
   Create Language Pack.
3. Specify this information:

Language ID
Specify the ID to use for the language pack. We recommend using ISO
language and locale codes, such as fr-CA for French Canadian.

Description
Specify a description of the language pack.

4. Click the Save
   button.
