# What are Invoice Forms?

Use the Invoice Forms window to print invoices for
shipped customer orders. VISUAL supports pre-numbered forms or auto
numbering of forms. If you are using Infor VISUAL Financials, you
can also have this process create actual receivable invoices.

## Invoice QRP Files

The format of Invoice Forms can be modified using Centura Report
Builder. The following QRP files apply:

|  |  |
| --- | --- |
| Standard | VMINVFRM.QRP |
| User Defined 1 | VMINVUD1.QRP |
| User Defined 2 | VMINVUD2.QRP |
| User Defined 3 | VMINVUD3.QRP |
