# Viewing and Editing Pallet Details

1. To view and edit pallet
   details for a customer order, select Pallet
   Details from the Edit menu.

The Pack list ID for the customer order
appears in a read-only field at the top of the dialog box.

2. If applicable, click
   the Insert button to add a new pallet
   line.

You cannot add a new pallet line unless
a shipper line exists for it.

3. Modify pallet line information
   if necessary.
4. Click the Save
   button to save any modified pallet lines.
