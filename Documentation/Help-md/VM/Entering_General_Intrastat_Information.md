# Entering General Intrastat Information

Enter general Intrastat information in the following
fields:

Review Intrastat Before Saving Transaction
- If you want to review the Intrastat information before VISUAL saves
the transaction, select the Review intrastat before txn check box.

Country ID - Select the country ID of
your company.

Branch ID - Enter branch ID of your company.

Next Sequence # - Enter next number for
Intrastat report.

Click the Save button to save the information.
