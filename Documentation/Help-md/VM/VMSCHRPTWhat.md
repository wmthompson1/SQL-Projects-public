# What are the Production Schedule Reports?

These reports help the Production Manager and Supervisor
in managing and analyzing the shop floor. You can access all seven
reports by selecting the Production Schedule Report from the Reports
menu. This calls up a dialog box that has one push button for each
report.

Select Production Schedule Report from
the Eng/Mfg menu.

The Print Production Schedule Reports window appears.

The window has seven buttons, one for each report available from
the window.
