# What is the Customer Sales Journal?

Use the Customer Sales Journal to review the amounts that you have
invoiced customers

You can print a detailed report or a summary report. The detailed
report lists individual transactions based on the sort sequence. The
summary report groups the transactions based on the sort sequence.
If you include multiple entities and sites in the report, totals are
provided by each site and entity.
