# Viewing the Tax Groups Table

The Tax Groups table contains basic information on
all tax groups in the sales tax group master file. You can browse
or search the Tax Groups table.

To call up the Tax Groups table:

1. Click the Tax
   Group ID button from the Sales Tax Group Maintenance window.

The Tax Groups table appears, showing all
of the sales tax groups in the database. This table is a convenient
tool for finding tax groups.

You can sort the table in either ascending
or descending order. To view the list in descending order, click the
Descending check box. This re-sorts the list to display the numeric
values from largest to smallest and alpha characters from Z to A.

2. Select a sales tax group
   and click Ok, or click Cancel
   to close the table and return to the Sales Tax Group Maintenance
   window without selecting a group.
