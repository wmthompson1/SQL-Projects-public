# Entering Substitute Part information for a Part

Substitute parts are parts that you designate as reasonable
replacements to an existing part in your inventory. Use substitute
parts to meet order demands when the part you originally intended
to use is unavailable.

1. From the Order
   Management tab, click the Substitute
   Parts button.

The Substitute Parts for Part dialog box
appears.

2. Click Insert
   to start adding line item information.

The first line is now active.

3. Enter information in
   the following fields:

Substitute Part ID
- The ID of the part that you want to use as a substitute for a part.
Either manually enter a Part ID or double-click the Part ID column
header to select one from the line item table.

Substitute Part Description
- The description of the substitute part. VISUAL fills this field
automatically with the accompanying part description after you select
a Substitute Part ID.

Rank - The importance
that you assign to the part. When you need to order substitute parts,
the rank value dictates the need and/or importance of the substitute
part.

4. Click Save
   to commit the substitute part information.
