## Pallet Details Required

If pallet details are required for shipments to this customer, select
the Pallet Details Required check box. If shipping
information, such as carton and skid preferences, has been specified for
this customer, VISUAL automatically generates pallet details at the time
of shipment. Click [here](Automatically_Generating_Pallet_Details_for_Customer_Shipments.htm)
for more information.
