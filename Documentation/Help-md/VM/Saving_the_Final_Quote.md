# Saving the Final Quote

When engineering and price information is completed
to your satisfaction, the quote is ready to be presented to the customer
or prospect.

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site that contains the quote. If you are licensed to use a single
   site, this field is unavailable.
2. Double-click the Quote
   ID button and select the quote.
3. Set the Status to Ready
   by clicking on the Ready radio button
   in the Status section of the Estimating Window.

This indicates that the quote is ready to
be transmitted.

4. If appropriate, click
   the Dates tab and set the Follow-up
   Date appropriately.

This allows you to periodically print a
Quote Log sequenced by a follow-up date, so you can manage follow-up
of quotes.

5. Set the Probability
   % to the estimated probability of a win.

Remember that this value drives the Proforma
Report.

6. Set an Expiration Date,
   if one applies.
7. Set Expect date to the
   projected date of win or loss.
8. Provide a quoted lead-time
   in days in the Quoted Leadtime field.
9. Click the Save
   button to save all information.

The quote is now ready to print.
