# Using a Right Mouse Click to Append an Existing Order Line to the Current Order

You can order WIP/VAS for a part in the current customer
order by right-clicking on the WIP/VAS type in the WIP/VAS window.

1. In the VISUAL Order
   Management Customer Order Entry window, open the order for which
   you want to order WIP/VAS.
2. If not already open,
   open the Order Management window, then the Current Customer Order
   window and the WIP/VAS window. Tile or cascade the two windows
   so that you can easily view and refer to both.
3. In the WIP/VAS window,
   double-click the icon.

All valid WIP/VAS types for that part in
that warehouse appear.

4. Highlight the appropriate
   WIP/VAS line and perform a right mouse click.

An Order option box appears.

5. Left-click Order.

VISUAL orders this WIP/VAS for the entire
quantity of the part.

6. In the VISUAL Order
   Management Customer Order Entry window, click the WIP/VAS
   button to view the ordered WIP/VAS line in the WIP/VAS dialog
   box line item table.
7. Edit the information
   as necessary and click the Save button
   to commit the order of the WIP/VAS.

After you save the order, the price per
part of the WIP/VAS appears in the WIP/VAS Unit price column in the
VISUAL DDP Customer Order Entry window.
