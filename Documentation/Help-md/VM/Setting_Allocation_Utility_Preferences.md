# Setting Allocation Utility Preferences

To set your allocation utility preferences:

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site to use. If you are licensed to use a single site, this field
   is unavailable.
2. In the Starting Part
   ID field, enter the ID of the part with which you want VISUAL
   to begin this allocation session.

The VISUAL Allocation Utility ignores all
Part IDs previous to this.

3. If you want to allocate
   only a range of parts, enter the ID of the part with which you
   want VISUAL to end the allocation session in the Ending Part ID
   field.

If you leave this field blank, VISUAL allocates
all parts with IDs that follow the Starting Part ID.

Leave both these fields blank if you want
the Allocation Utility to consider all parts.

4. In the Starting Warehouse
   ID field, enter the ID of the warehouse with which you want VISUAL
   to begin this allocation session.

The VISUAL Allocation Utility ignores all
Warehouse IDs previous to this.

5. If you want to allocate
   only parts in a range of warehouses, enter the ID of the warehouse
   with which you want the VISUAL Allocation Utility to end the allocation
   session in the Ending Warehouse ID field.

If you leave this field blank, VISUAL allocates
the selected parts in all the warehouses with IDs that follow the
Starting Warehouse ID.

6. Select the appropriate
   inventory allocation options. Both of these selections are optional.
   However, you may select both.

If you select either of these check boxes,
the Allocation Utility considers on-hold and/or unavailable inventory
in addition to on hand inventory when allocating supply to customer
order demand.

Allocate On-hold Inventory
- Select this check box if you want the Allocation Utility to allocate
inventory that is currently on-hold. Classify parts as on-hold in
Warehouse Maintenance.

Allocate Unavailable
Inventory - Select this check box if you want the Allocation
Utility to allocate inventory that is currently unavailable. Classify
parts as unavailable in Warehouse Maintenance.

Reevaluate Partial Allocations
- Select this check box if you want the Allocation Utility to your
past partial allocations.

7. If you want the Allocation
   Utility to consider future supply order during the allocation
   of supply to customer order demand, select from the following
   options:

Allocate Future Supply
Orders - Select this check box if you want the Allocation Utility
to consider supply orders as it allocates supply to customer order
demand.

Relevant supply order types include purchase
orders, work orders, work order coproducts, interbranch transfers,
and possibly planned orders.

Allocate Unreleased Orders
- This option is not available until you select the Allocate
Future Supply Orders check box. Select this check box if you
want the Allocation Utility to allocate unreleased work order quantities
to customer order demand.

Allocate Firmed Orders
- This option is not available until you select the Allocate
Future Supply Orders check box. Select this check box if you
want the Allocation Utility to allocate firmed work order quantities
to customer order demand.

You can select all three future supply options
at the same time.
