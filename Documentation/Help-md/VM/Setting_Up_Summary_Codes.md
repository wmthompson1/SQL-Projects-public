# Setting Up Summary Codes

Summary codes represent the types of project work
you want VISUAL to summarize in your project reports.

To set up Summary codes:

1. In Project Maintenance,
   click the Maintain menu and select Summary
   Table.

The Summary table appears.

2. Click the Insert
   button.

The next available row in the table appears
highlighted.

3. Enter the Code and Description
   you want to use.
4. Click the Save
   button.
5. When you have finished
   entering Summary codes, click the Close
   button.
