# Adding States and Provinces

Use the States/Provinces Table to enter or delete
code and name of States and Provinces. By default, codes for the 50
U.S. states and some territories are displayed in the table. You can
add or delete states and provinces from the default list.

1. Select Admin,
   Application Global Maintenance.
2. Select Maintain,
   Table Maintenance, States
   and Provinces.
3. Click Insert.
4. Specify a code and a
   description.
5. Click Save.
