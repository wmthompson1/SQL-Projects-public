# Adding Detail Rows to ETCs

If you want to add a new detail row for which you want to estimate
costs:

1. With
   the Maintain ETC window open and the appropriate project selected,
   click the WBS code to which you want to add a new cost.
2. Click
   the Insert Row button.
3. Double-click
   the Dept ID column header and select
   the department you want to use.
4. Double-click
   the Cost Cat ID column header and select
   the category you want to use.
5. Scroll
   to the period to which you want to add the cost, and add the cost
   you want.
6. Click
   the Save button.
7. When
   you have finished adding costs to this projects ETC, click the
   Close button.
