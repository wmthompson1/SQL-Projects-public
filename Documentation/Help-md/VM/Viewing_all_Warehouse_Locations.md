# Viewing all Warehouse Locations

You can view a complete list of all warehouse locations. While you
must select a part to access the dialog, the dialog shows all warehouse
locations in your system.

To view all warehouse locations:

1. Select Inventory,
   Part Maintenance.
2. In the Part ID field,
   specify a part.
3. Select Info,
   All Locations. If you are licensed to
   use multiple sites, all sites are displayed in the dialog box.
4. Click Ok.
