# Calculating Earned Value Amounts

VISUAL calculates earned value based on the percent
complete for the project tasks against which you have entered labor
in Labor Ticket Entry.

To post earned value amounts:

1. In the Project Summary
   window, select the project for which you would like to calculate
   earned value amounts.
2. In the Edit menu, select
   Calculate Earned Value.

The Calculate Earned Value window appears.

3. In the Calculate Earned
   Value window, click the Calculate Amount
   button.
4. Click the End
   of Fiscal Period calendar button and select any date within
   the fiscal period.
5. Click the Calculate
   button.

VISUAL
calculates earned value based on the parameters you select. On successful
calculation a dialog box appears requesting you to confirm refreshing
the Project Summary window.

6. Click the Yes
   button.
7. Click the Close
   button.
