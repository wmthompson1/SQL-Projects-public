# Input File Record Types

* MASTER
  - Master header information.
* PART
  - Part master information.
* OPERATION
  - Operation information.
* OPERATION
  RESOURCE - Concurrent resources
* MATERIAL
  - Material requirement information.
* ASSEMBLY
  - Sub-Assembly or leg-header information (for multilevel masters).
* BEGIN-SPEC
  - Indicates the next lines are specs for the last record type
  processed.
* END-SPEC
  - Indicates the end of specs for the last record type processed.
