# Viewing Revision Comment Histories

1. To review a revision
   history for a part, select Revision History
   from the Info menu.

The Revision History (Part ID) dialog box
appears.

VISUAL displays one revision per line.

The table contains the following fields:

Stage ID If
you specified a stage for the revision, the stage code appears in
this field. Stages apply to revision numbering. Click [here](Maintaining_Revision_Numbering.htm)
for more on stages in numbering profiles.

Revision ID
The Revision ID you specified for this revision.

ECN - If an ECN
applies to this revision, its ID appears in this field.

User ID The
ID of the user signed into the database when the revision was created.

Effective Date
The date on which the revision was initiated.

Comments Comment
text regarding the revision. To view comment text, double-click the
Comments field.

2. The Revision Comments
   dialog box appears. This is a read-only dialog box. After viewing
   comments, click the Close button.
3. Click the Documents
   tab.

The Documents tab contains the same fields
as the Part tab, with the exception of the ID field, which contains
the ID of the document linked to the part.
