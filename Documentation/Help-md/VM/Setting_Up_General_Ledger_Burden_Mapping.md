# Setting Up General Ledger/Burden Mapping

After you have set up your project accounts, use the
GL/Burden map ping window to specify the accounts you want to use
for which departments and the burdens you want VISUAL to use for its
calculations.

The GL/Burden mappings you set up depict how VISUAL calculates your
monthly bills and to which accounts it credits those figures.

To set up GL/Burden mappings:

1. From the Maintain menu
   of the Project Summary window, select GL/Burden
   Mapping.

The GL/Burden Mapping Maintenance window
appears. All of your Cost Group IDs automatically appear.

2. Click the Department
   ID button and select the department whose mappings you
   want to set up.
3. Click the Project
   Type button and select the project type you want to apply
   to this mapping.
4. Click in the Debit Account
   column for the Cost Group ID whose mapping you want to set up.
5. Click the <Debit
   Account> row header and select the debit account you
   want to use for this mapping.
6. Click the <Credit
   Account> row header and select the credit account you
   want to use for this mapping.
7. Click the <Burden
   Group ID> row header and select the burden group you
   want to use for this mapping.
8. When you have finished
   setting up your GL/Burden mappings, click the Save
   button.

If you want to set up a new mapping, click
the Insert button and select the Cost Group
ID you want to use for the new mapping.

If you find that you need to enter information
for which you do not have the appropriate Class or Cost Category,
close the Budget Spread Cost Category maintenance window and enter
the information you need in the appropriate window.
