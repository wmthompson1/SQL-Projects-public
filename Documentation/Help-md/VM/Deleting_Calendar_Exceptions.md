# Deleting Scheduling Information

Use this feature to delete scheduling information
for one or more days in the schedule.

1. Select Admin,
   Application Global Maintenance.
2. Click the Scheduling
   tab.
3. Select the rows that
   contain the days whose schedule you want to delete.
4. Click Delete
   Selections. An X is displayed in the header of each row
   that you selected.
5. Click Save.
   The scheduling information is removed from the rows.
