# Starting the Throughput Window

To start the Throughput Window, select Scheduling,
Throughput Window.

The Throughput Window contains the Site ID field, and eight buttons,
one for each Throughput option. If you are licensed to use multiple
sites, you must run each Throughput option by site. If you are licensed
to use a single site, the Site ID field is unavailable, and you must
run each Throughput option on an enterprise-wide basis.

The Throughput Window contains eight buttons, one for each throughput
option.

From the Options menu, select Show Sales Values to view Throughput
Values as sales values, or raw dollar amounts.
