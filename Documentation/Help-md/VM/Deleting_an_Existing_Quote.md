# Deleting an Existing Quote

This operation removes the quotation from the system
including all of its line items and all quote masters.

|  |  |
| --- | --- |
| Stop.gif | All quote masters for a quotation are deleted along with the quotation. If you want to retain the quote master for other purposes, use the Cancel status to cancel the quote, or use the Copy From option in the Manufacturing Window to copy the quote master to another quote master, engineering master, or work order. |

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site that contains the quote. If you are licensed to use a single
   site, this field is unavailable.
2. With the quotation you
   want to delete open in the Estimating Window, select Delete
   from the File menu or click the Delete
   button at the bottom of the window.

Clicking the Delete
button directly below the table places an x at the end of the column,
signifying that the line item has been selected for deletion.

A dialog box appears, asking you to confirm
the deletion.

3. Click Yes
   to permanently remove the quotation or No
   to leave the quotation order in the system.
4. Click the Save
   button to complete the operation.
