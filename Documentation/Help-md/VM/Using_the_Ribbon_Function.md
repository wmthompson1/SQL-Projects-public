# Using the Ribbon Function

The Ribbon function from
the Infor VISUAL File menu displays the VISUAL menu options in a "ribbon"
along the top of your screen, without the Infor VISUAL main window.
This function takes up less space, and provides you with the ability
to size the windows on your system so that you can view the menu options
and the active windows at once.
