# Setting Up GL/Burden Mappings

After you have set up your project accounts, use the
GL/Burden map ping window to specify the accounts you want to use
for which departments and the burdens you want VISUAL to use for its
calculations.

The GL/Burden mappings you set up depict how VISUAL calculates your
monthly bills and to which accounts it credits those figures.

You cannot change the cost group if any transactions reference the
burden map. You cannot change the burden group if any burden transactions
reference the G/L burden map. VISUAL generates burden transactions
during costing.

To set up GL/Burden mappings:

1. From the Project menu,
   select G/L Burden Mapping Maintenance.

The Direct Costs GL/Burden Mapping Maintenance
window appears. All of your Cost Group IDs automatically appear.

2. Click the Department
   ID button and select the department whose mappings you
   want to set up.
3. Click the Project
   Type button and select the project type you want to apply
   to this mapping.
4. Click in Debit
   Account column for the Cost Group ID whose mapping you
   want to set up.
5. Click the Debit
   Account column header and select the debit account you
   want to use for this mapping.
6. Click the Credit
   Account column header and select the credit account you
   want to use for this mapping.
7. Click the Burden
   Group ID column header and select the burden group you
   want to use for this mapping.
8. To add more mappings,
   click the Insert button. Double-click
   the Cost Group ID button and select the cost group ID whose mapping
   you want to set up. Then, repeat steps 4 through 7.
9. When you have finished
   setting up your GL/Burden mappings, click the Save
   button.
