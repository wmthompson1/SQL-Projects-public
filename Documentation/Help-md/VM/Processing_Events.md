# Processing Events

All Message Maintenance events appear in the Process
Events dialog box. From this dialog box, you can delete, print, and
mail event messages.

## Processing New Events

You can manually process new events that VISUAL has not yet processed.

1. Select Process
   Events from the File menu.

The Process Events dialog box appears.

2. Select New from the
   Status drop down menu.

A list of new events appears in the table.

3. Select the event you
   want to process. Click the Select All
   button to select all of the events in the table.

The event line in the table becomes highlighted.

4. Process the event.

Click the Delete
button to delete the event.

Click the Print
button to print the event.

Click the Mail
button to email the event.

If you click the Mail
button, your mail application dialog box appears. Enter or select
the recipient of the message, then send the message.

5. Click Ok
   to complete the operation.

## Processing Completed Events

button to delete the event.

Click the Print
button to print the event.

Click the Mail
button to email the event.

If you click the Mail
button, your mail application dialog box appears. Enter or select
the recipient of the message, then send the message.

## Processing Error Events

Any errors that occur when events trigger appear here. You can view
all of the events with errors here.

1. Select Process
   Events from the File menu.

The Process Events dialog box appears.

2. Select Error
   from the Status drop down menu.

A list of events with errors appears in
the table. You can scroll to the right of the table to see the error
message for the event.

3. Select the event you
   want to reprocess. Click the Select All
   button to select all of the events in the table.

The event line in the table becomes highlighted.

4. Process the event.

Click the Delete
button to delete the event.

Click the Print
button to print the event.

Click the Mail
button to email the event.

If you click the Mail
button, your mail application dialog box appears. Enter or select
the recipient of the message, then send the message.

5. Click Ok
   to complete the operation.
