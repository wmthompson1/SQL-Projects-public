# Understanding Color Codes in Workflow Tracker

You can view the status of each step in the workflow
by the following color code:

Blue A blue dash outline indicates
the starting workflow step.

Black A black dash outline indicates a completed workflow step.

Yellow A yellow dash outline indicates
the current workflow step.
