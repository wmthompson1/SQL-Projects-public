# Setting the Load Horizon

You can control how many days the Scheduling Window
displays load from the current date. If you have operations that may
be scheduled farther out, you may need to adjust the loading horizon.

You must edit the loading horizon BEFORE loading a schedule.

1. Select Loading
   Horizon from the Edit menu.

The Loading Horizon dialog box appears.

2. Enter the load horizon
   in days. Click Ok.
3. Load the schedule ID.

The schedule ID appears in the Scheduling
Window. A solid blue vertical line appears in the window at the point
in the schedule at which the horizon occurs.
