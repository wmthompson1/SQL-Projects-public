# Processing On Hold Tasks

An On Hold task is generated when you reject a task. When the On
Hold task is generated, the On Hold tasks status is set to Pending.
You can change the status to Complete. You cannot use any of the other
statuses with On Hold tasks.

To process an on hold task:

1. Select Eng/Mfg,
   Task Maintenance or Purchasing,
   Task Maintenance.
2. Change the status to
   Complete.
3. Click Save.
