# Deleting Preferences

To delete a preference:

1. Select Admin,
   Preferences Maintenance.
2. If you are the SYSADM
   user, select the ID of the user for whom you are deleting preferences.
3. To search for a preference
   in a particular section, click the Section
   Filter drop-down button and select the section that contains
   the preference that you are editing.
4. Select the row that
   contains the preference to delete, and then click the Delete
   Row button.
5. Click Save.
