# Adding Notations

Notations (notes, for short) are free form text used for internal
documentation purposes. Although they are associated with the purchase
order in much the same way as specs, they are fundamentally different:

* Notes are exclusively for
  internal use of your business. They are not included on any outgoing
  paperwork.
* Individual note entries
  are automatically date/time stamped, and maintained as separate
  entries. This is different from specs, where you can modify the
  entire block of text at any point as a single object.

There are three types of notes associated with Purchase Receipt
Entry:

Receiver Notes - These are associated
with the current Receiver ID. If no Receiver is selected, these notes
are not active.

Vendor Notes - These are running notes
associated with the current Vendor ID of the Purchase Order. If no
vendor is selected, these notes are not active. These notes are not
linked to any current purchase order, but to the current vendor. Use
Vendor Notes for internal dialogs regarding the vendor.

Purchase Order Notes - Like order specs,
these are associated with the order as a whole. Use Order Notes for
internal dialogs concerning this particular order.

All notations in Infor VISUAL function in the same way.
