# Entering and Deleting Intrastat Region Information

Region codes define the locations of warehouses.

1. Click the Region
   button.

The Region dialog box appears.

2. Click the Insert
   button.

A new line appears in the line item table.

3. In the Code column,
   enter a numeric code for the new region.
4. In the Description column,
   enter a description for the region code.
5. Click the Save
   button to save the new information.

## Deleting Region Codes

1. Click the Region
   button.

The Region dialog box appears.

2. Select the region code
   you want to delete, and then click the Delete
   button.

An X appears in the row header beside the
code you are deleting.

3. Click the Save
   button to save the new information.
