# Specifying a Default Tab

If you deal primarily with the fields on a specific
tab in the Part Maintenance window, configure Part Maintenance to
open displaying a preferred tab.

1. Select Preferences
   from the Options menu.
2. From the Default Tab
   list, select the tab you want to have the focus when you open
   Part Maintenance.
3. Click the Done
   button to commit the tab preference.
