# Estimating Process

The Basic steps of the Estimating process are:

1. Set
   up Auto Numbering for Estimates - This is a one-time step
   that usually takes place during implementation.
2. Define
   the quote using the Estimating Window - This is where you
   specify customer or prospect information, probability of winning
   the quote, expected date, and other information. Additionally,
   one line item is created for each part being quoted. Multiple
   quantities can be quoted for each line item.
3. Create
   or Modify Engineering Information - Estimated costs are
   calculated using engineering or quote masters. If the quote is
   for a custom job, you may need to create a quote master using
   the Manufacturing Window or Quick Quote facility.
4. Preparing
   pricing information - The Estimating Window automatically
   computes unit prices to quote, based on estimated costs computed
   by the system and markups specified by you.

You can use the Cost Detail report to analyze
how costs and markups resolve to profit margin.

5. Checking
   schedule availability - If necessary, you can analyze the
   impact on your shop schedule should you place an order for each
   quoted part.
6. Printing
   the quote - You can print quotes in standard or user-defined
   formats. You can then mail or Fax the quote to the customer or
   prospect.
7. Printing
   Proforma - Periodically, you can print a Proforma Report
   to get projected revenue from all quotes for a time period based
   on a cutoff probability of winning the quote.
8. Managing
   quotes - Use the Quote Log report, along with the Follow-up
   Date field, to manage the quote process.
9. Generating
   orders - You can quickly generate customer orders from
   quotes.
