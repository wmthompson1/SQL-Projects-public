# Designating Project Resources

To designate a project resource:

1. Open the Shop Resource
   Maintenance window and enter the appropriate information for your
   resource.
2. Click the Other
   tab.
3. Select the Project
   Resource check box.
4. Click the Save
   icon on the toolbar.
