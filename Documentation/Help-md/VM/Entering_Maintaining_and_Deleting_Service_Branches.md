# Entering, Maintaining, and Deleting Service Branches

Use the Service Branches dialog box to enter and maintain
service branch information.

1. From the Maintenance
   menu, select Service Branch Maintenance.

The Service Branch dialog box appears.

2. To enter a new Service
   Branch, click the Insert button.

Enter information in the following fields:

Branch- An ID
for the branch you want to add. Enter any alphanumeric combination
as a unique identifier of the branch.

Area - The area
in which the service branch is found. Double-click the Area
column header to view a list of valid service areas. If you have not
previously entered any service areas, none appear in the line item
table. Because Service Areas are necessary in the entry of new branches,
pause in this procedure and enter at least one Service Area now. See
[Entering,
Maintaining, and Deleting Service Areas](Entering_Maintaining_and_Deleting_Service_Areas.htm).

Description -
A description of the service branch. Be detailed in your descriptions.

3. Click the Save
   button to commit the information.
4. To modify information,
   place the mouse pointer in the appropriate column in the appropriate
   line and manually edit the information. When finished, click the
   Save button.
5. To delete a service
   branch, highlight the appropriate line and click the Delete
   button.

VISUAL places an X at the beginning of the
line.

6. Click the Save
   button to delete the line.
7. Click the Close
   button to return to the Equipment Maintenance window.
