# Adding Transit Locations on the Fly

Transit locations are locations within warehouses
from which you can ship part quantities. You must supply a Transit
Location ID for each line of an IBT before a record of transfer is
saved. If you have not already set locations within your warehouses
as transit locations, you can specify only one Transit Location ID
on the fly during IBT entry. After this first location has been established
- now the default - you cannot add anymore in this manner. After this,
use Part maintenance to classify locations as Transit locations.

If the From warehouse has no default Transit Location ID, you can
enter a unique Transit Location ID. After you enter the new ID and
press the Tab key, you are asked if you want to create this new Transit
Location ID as a valid Transit Location ID.

If you select Yes, this new ID is saved
as the From warehouses default Transit Location ID.

If you enter an existing non-transit Location ID in the Transit
Location ID field, you are told that this is not a valid Transit Location
ID.

If this message appears, a default Transit Location already exists.
Either accept this Location ID, or open Part Maintenance and classify
the Location you want to use as a Transit Location

See Part Maintenance for more information on setting Location IDs
as Transit Location IDs.
