# Entering a Default Material Code for a Part

Material Codes are unique identifiers for parts that
you ship to international customers. If you are using Shipment Tracking
to track the whereabouts of shipments and inventory holdings, you
may want to assign material codes to parts. However, if you are not
using Shipment Tracking, you can still use this feature. Click for
more information on enabling [Shipment
Tracking](Setting_Shipment_Tracking_Information.htm) and defining [Ship
Reason Codes](Entering_Ship_Reason_Priority_Codes.htm).

1. With the appropriate
   Part ID displaying in the Part Maintenance window, select the
   Order Management tab.
2. Enter a material code
   in the Material Code field and then click the Save
   button.

VISUAL saves this as the parts default
material code.
