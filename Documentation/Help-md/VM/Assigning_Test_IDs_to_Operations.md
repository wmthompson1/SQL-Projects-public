# Assigning Test IDs to Operations

Use Work Order/Operation Test Input to assign tests
to operations.

1. In the table of operations,
   highlight the operation to which you want to assign the test.
2. Click in the Test ID
   column of the operation and enter the test ID. You can also double
   click the Test ID column header and
   select a test ID from the List of Tests table.

A check mark appears to the left of the
operation to which you assigned the test ID.

3. Click the Save
   button to save the assignment.
4. Repeat steps 1 to 3
   to assign tests to other operations. You can assign different
   test IDs to each operation in a work order.
