# Viewing Material Constraint Inquiry Graphs in a Table

You can view the information from a Material Constraint
Inquiry graph in a table format by clicking the button next to the
graph styles list box.

Upon clicking, a table appears in place of the graph. The selected
Schedule IDs appear in the left hand column and the Part IDs for which
there is constraint information each represent one column in the table.
