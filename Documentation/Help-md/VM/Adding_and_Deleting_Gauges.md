# Adding and Deleting Gauges

Use gauges to measure the tolerances for a test. You
can add gauges in the Quality Maintenance window.

1. Click the Insert
   button to add a new line to the list of gauges. You can also select
   Insert Gauge from the Edit menu.

A new line appears in the list.

2. Inn the Gauge ID column,
   enter a gauge ID.

The gauge ID is a unique identifier for
the gauge.

3. Move to the Description
   column and enter a description for the gauge.
4. Move to the Serial #
   Reqd column and enter Y if the serial number is required for this
   gauge, or N if the serial number is NOT required for this gauge.
5. Click the Save
   button to add the gauge.

## Deleting Gauges

You can delete gauges from the Quality Maintenance
window. You CANNOT delete gauges that existing test steps are using.

1. Select the gauge that
   you want to delete.
2. Click the Delete
   button. You can also select Delete Gauge
   from the Edit menu.

A dialog box appears, asking you to confirm
the deletion.

3. Click to confirm the
   deletion.

An x appears to the left of the deleted
gauge.

4. Click the Save
   button to complete the deletion.

If you want to cancel the deletion, click
the Refresh button. Your gauge is not deleted
until you click the Save button.
