# Printing Sales Code Reports

Before you begin, be sure that a printer is properly
installed for use in Windows.

1. Select Print
   Sales Code Report from the File
   menu.

The Print Series/Model Info dialog box appears.

If necessary, you can alter the printer
setup from the print dialog box by selecting the Print
Setup button.

2. If you are licensed
   to use multiple sites, click the Site ID(s) arrow and select the
   sites to view in the report. You can select more than one site.
   If you are licensed to use a single site, this field is unavailable.
3. Enter the Starting Series
   ID and the Ending Series ID in the appropriate fields.

Leave both of these fields blank to print
a report of all series.

4. Select one of the three
   print options from the drop down box.

You can select File, View, or Print. If
you want to print the Report, choose Print.

5. Click Ok.

If you select File, a dialog box appears,
prompting you for the destination file name. Enter the destination
file name and click Ok to write the report
to the other file.

If you select View, the report appears on
your screen.

If you select Print, the Printer Setup dialog
box appears.

6. Select the print range.

You can select all pages or specify individual
pages.

7. Click Ok
   to print the report.
