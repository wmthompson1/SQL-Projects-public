# Step Eight

## Update Formal EAC in Project Maintenance

Based on the information in the EAC/POC report, update
each projects formal EAC.

1. From the Project menu,
   select Project Maintenance. The Project
   Maintenance window appears.
2. Enter the Project ID
   for the project you would like to update. VISUAL populates other
   information for the project in the window.
3. From the Maintain menu,
   select Project EAC/Target Cost. The
   Project EAC/Target Cost window appears.
4. Enter a Version ID in
   the EAC Version field.
5. Enter the end date of
   the period you are closing in the Version Date field.
6. Select the Current
   Version check box.
7. Using the information
   from the EAC/POC report, update the Project EAC.
8. Click Save.
9. Click Close
   to return to the Project Maintenance window.

To compare the current EAC with the Baseline EAC, select Info, Baseline/EAC.
The Baseline and EAC window appears.

You can also view percentage of completion for the project by selecting
Info, Percentage of Completion.
The Percentage of Completion window appears.
