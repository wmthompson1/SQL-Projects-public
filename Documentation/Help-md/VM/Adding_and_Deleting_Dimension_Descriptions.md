# Adding and Deleting Dimension Descriptions

Use Quality Maintenance to insert and delete dimension
descriptions for tests.

## Adding Dimension Descriptions

1. Select Dimension
   Descriptions from the Quality Maintenance Edit
   menu.

The Dimension Description dialog box appears.

2. Click the Insert
   button.

A new line appears in the list of dimensions.

3. Enter the new dimension
   description.
4. Click the Save
   button to save the new description.

## Deleting Dimension Descriptions

You can use Quality Maintenance to delete dimension
descriptions.

1. Select Dimension
   Descriptions from the Quality Maintenance Edit
   menu.

The Dimension Description table appears.

2. Select the description
   you want to delete.
3. Click the Delete
   button.

An X appears to the left of the description.

4. Click the Save
   button to complete the deletion.
