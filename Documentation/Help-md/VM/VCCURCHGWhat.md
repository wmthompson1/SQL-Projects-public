# What is the Entity Currency Update Utility?

Use this utility to convert an entity currency to
another currency, which must be a tracking currency. You should update
the exchange rate to the rate you want to use for the conversion then
click the start button on the main toolbar.

|  |  |
| --- | --- |
| WARNING.gif | There is no way to reverse this process. It is imperative that you make a backup before you begin. |
