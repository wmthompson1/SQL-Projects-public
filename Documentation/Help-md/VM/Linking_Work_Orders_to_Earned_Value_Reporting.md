# Linking Work Orders to Earned Value Reporting

Selecting Report Earned Value Quantities
From This Operation check box to automate the process of reporting
earned value against project linked work orders. VISUAL calculates
earned value against the task to which the work order is linked. Because
the task completion becomes the same percent age as the operation
to which you have linked, you should take care which task you link.

In the above example, operation 20 is 40% complete which makes the
project Task 40 % complete, however, operation 30 and 40 are not complete
and do not need to be in order to see a 100% completion at the task.
