# Creating a Reminder for Another User

If you [create
an activity and assign it to another user](VMACTENTfrmActivityEntry.md), you can specify reminder
information on the activity for the user. The reminder is displayed
only if that user has activated reminder features. If the user has
not activated the reminder features, then no reminder is displayed
for that user. If you want to be able to set up reminders for others,
make sure all users activate reminder features.
