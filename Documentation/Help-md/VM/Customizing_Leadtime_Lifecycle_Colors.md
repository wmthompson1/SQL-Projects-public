# Customizing Leadtime Lifecycle Colors

You can select the colors to use for the lifecycle
cards. You can customize colors by transaction document type.

1. Perform one of these
   steps:
2. Select Sales, Estimating Window.
3. Select Sales, Customer Order Entry.
4. Select Sales, Order Management
   Window.

2. Select the quote or
   order.
3. If the quote or order
   has multiple lines, select the line.
4. Select Info,
   Leadtime Lifecycle.
5. Select Options,
   Color Preference.
6. For each transaction
   document type, use the color picker to specify the colors to use.
7. To save these colors
   as your default colors, select the Save as
   Default check box. To use these colors for the current
   session only, clear the Save as Default
   check box.
8. Click Ok.
