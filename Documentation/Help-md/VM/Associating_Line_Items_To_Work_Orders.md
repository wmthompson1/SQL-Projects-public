# Associating Line Items To Work Orders

You can change or create an association between a
Progress Billing line item and a work order.

1. Highlight the line for
   which you want to create the association.
2. Click the Work order
   button.

The Associate Line Item to Work Order dialog
box appears.

3. Enter a base ID for
   the work order in the Work Order Base ID field, or click the Workorder Base ID button to select from the
   Work Orders table.
4. Enter the correct Lot
   ID and Split ID in the appropriate fields.
5. Click the Ok
   button to save the association.
