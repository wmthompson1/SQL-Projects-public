# Applying CCR Buffer Changes to Work Orders

You can apply updates to any of a CCRs buffers to work orders and
engineering masters. Any CCR buffer can be changed.

To update work orders and engineering masters with new buffers:

1. In the table at the
   top, adjust buffer settings as necessary.
2. To apply the changes
   to all work orders and engineering masters, select Edit,
   Recalculate All Buffer Sizes. To apply
   the changes to particular work order or engineering masters, select
   the work orders and engineering masters in the table. Select Edit, Recalculate Selected
   Work Order Buffer Sizes.
3. Click the Save
   button.
