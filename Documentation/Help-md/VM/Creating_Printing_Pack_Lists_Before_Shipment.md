# Creating (Printing) Pack Lists Before Shipment

The following example details a typical sales order
shipping procedure, with special emphasis on VISUALs creation of
multiple pack lists.

1. Enter a [sales
   order](VMORDENTfrmCustomerOrder.htm) using the Customer Order Entry window. For this example,
   the sales order has three lines, each with a different Shipto
   ID.
2. Using the Shipping Entry
   window, select the order you want to ship and print a pack list
   for it.

3. Click the Yes
   button when VISUAL asks you if the range of pack lists printed
   correctly.
4. If you want to consolidate
   the Pack List ID, ASN, and Invoice at this time, click the Yes button when VISUAL asks you to confirm
   consolidation.
5. Back in the Shipping
   Entry window, click the Order ID button
   and choose the customer order you want to ship.

Because VISUAL created more than one pack
list for the order (remember, VISUAL created a range of pack lists--one
pack list for each Shipto ID), only the order line associated with
the pack list appears in the shipping table.

6. To view a different
   order line for the same order, adjust the Pack List ID.

The order line associated with the Pack
List ID appears in the table.

7. Specify the amount to
   ship or, if you are not shipping piece tracked parts, click the
   Ship All button.

VISUAL ships the line.
