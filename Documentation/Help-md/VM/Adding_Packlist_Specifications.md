# Adding Packlist Specifications

Specifications allow you to add free form text to
packlists. Packlist specifications are specific to an individual packlist
and there can only be one specification per packlist.

1. With the packlist open
   in the Customer Shipments window, select Packlist
   Specifications from the Notes menu.

The Packlist Specifications dialog box appears.

2. Enter the information
   for the packlist.

You can enter an unlimited amount of information.

Use the Paste
button to paste text that you copied from another location into the
box.

Use the Copy button
to copy text in the dialog box to the clipboard.

The entire contents of the dialog box is
copied to the clipboard. If you are pasting information into the dialog
box, the current text of the dialog box is replaced with the incoming
text. If you wish to paste without replacing the entire text, use
the Shift-Insert key combination.

3. Click the Close
   button to close the specification dialog box and return to the
   Customer Shipments window.
