# Creating New Work Orders in the Customer Order Entry

You can create Work Orders automatically for line
items in Customer Order Entry. During order entry, you can select
the New Work Order box on the line item. VISUAL creates a New Work
Order from Engineering Master using the line item Quantity and Desired
Ship Date for Quantity and Want Date. When using Customer Order Entry,
you have the choice of specifying a Firm or Released status for the
new work order. You can also copy any document references from the
engineering master.

This feature provides an easier way to manage make-to-order work
orders. The work order is automatically linked to the customer order.
Inventory receipt and issue is automatic when the order is shipped.

You can also create Work Orders through the Generate Order from
Quote command in Customer Order Entry. When converting a quote to
an order, you can choose to have VISUAL generate work orders. You
can specify Base ID, Lot ID, Release Date, Want Date, and Status for
each order.

For quote line items with quote masters, VISUAL creates a New Work
Order from Quote Master. For inventory line items, VISUAL creates
a New Work Order from Engineering Master. These work orders are also
linked to the customer order line items. See Customer Order Entry
for full details.
