# What is Purchase Order Entry?

Purchase Order Entry allows you to create and maintain
purchase orders for both inventory and non-inventory parts, as well
as for subcontracted services. You can enter all the information necessary
to complete and print purchase orders using the Purchase Order Entry
window.

You can enter a purchase order here, or you can generate a purchase
order using the Manufacturing Window.
