# Entering Vendor Bank Information

Use Vendor Banks to enter specify bank information
for all vendors. Typically, each vendor has only one bank, but you
can add others if necessary.

1. Select Maintain,
   Banks.

2. Specify information
   in the Address section for these fields:

Bank Account ID
- Specify a bank account ID. This is VISUALs identifier for a specific
vendor bank account.

Description -
Specify a description of the bank account.

Name - Specify
the complete name of the bank to which this bank account belongs.

Address - Specify
the address of the bank to which this bank account belongs.

City/State/Zipcode/Country
- Specify the city, state, zip code, and country of the bank to which
this bank account belongs.

Account # - Specify
the bank-issued account number.

IBAN - Specify
the banks IBAN identifier.

SWIFT - Specify
the banks SWIFT identifier.

Branch ID - Specify
the bank-issued sort code or branch ID.

Country ID - Click
the search button and select the country ID from the list.

3. To commit the bank information
   to your database, click the Insert button.

4. If you want to use this
   bank as the default bank for this vendor, click the Use
   as Default button.
5. When you have finished
   entering banks for this vendor, click the Save
   and Close buttons.
