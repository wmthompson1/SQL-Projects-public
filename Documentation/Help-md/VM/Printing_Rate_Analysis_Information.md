# Printing Rate Analysis Information

To print Rate Analysis reports:

1. Open the Rate Analysis
   dialog box and run an analysis for the Year and Period of interest.
2. Click the Category for
   which you want to print a report.
3. Click the Print
   button.
4. The Year and Period
   you selected automatically appears in the Print Dialog box. If
   you want to print a report for different periods, make the appropriate
   changes.
5. The Burden Category
   ID you selected appears selected. If you want to print this report
   for all of your Burden Categories, select the All
   option button.
6. The Summary Report Type
   option appears selected. If you want a more detailed report, select
   the Detailed option button.
7. Click the Destination
   arrow and select how you want to view the report. You can select
   Print, View,
   File, and E-mail.
8. Click the Ok
   button.

VISUAL prints the report to your specifications.
