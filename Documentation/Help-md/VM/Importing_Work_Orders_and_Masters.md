# Importing Work Orders and Masters

Use the Import command to bring in (copy details into
your current window) a Work Order/Engineering Master file that has
been previously exported.

1. If you are importing
   information to a work order/engineering master that currently
   contains details (operations and/or materials), position the cursor
   (focus) on the location where you want the information pasted.
   However, if you are pasting information into a work order/engineering
   master with no details, the focus is unimportant.
2. Select Import
   from the Edit menu.

The Import dialog box appears.

3. Enter the filename .eng
   from which you want to copy information into your current detail
   screen. You can choose from the file names that appear in the
   File name box in the upper corner of the window.
4. If the correct disk
   drive is not specified, you can change it here. If you saved (exported)
   the information with an extension other than .eng, you must enter
   the extension, or use \*.\* to display all files in the directory.
5. Select the file to import
   and click the Ok button.

A message appears, informing you that VISUAL
is pasting the information. When it is completed, the system returns
you to the Manufacturing Window details dialog box.

The information from the file is now pasted
into your current detail window.
