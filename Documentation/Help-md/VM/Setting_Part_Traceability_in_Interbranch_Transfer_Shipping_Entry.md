# Setting Part Traceability in Interbranch Transfer Shipping Entry

VISUAL selects the Trc column
in the IBT Shipping Entry line item table if the part requires traceability.

If the IBT transaction has line items with traceable Part IDs, you
should do the following:

1. Enter all data to complete
   the transaction or transaction line items in the normal way.
2. Click the Part
   Traceability button. The Traceability dialog box appears.

The quantity of the traceable part on the
IBT line is broken down into the maximum number of lots you specified
in Part Trace Profile Maintenance. You can either accept these default
quantity breakdowns or adjust them at this point by moving your cursor
to the Quantity field and modifying the value accordingly

3. Click the Close
   button to close the dialog box when you are done entering part
   trace information.
