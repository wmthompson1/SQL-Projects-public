# Showing Backlog and Shipments

Normally, VISUAL displays all orders of all status
values. There are two other viewing modes for this table, controlled
by the Show radio buttons at the upper right. If you select Backlog,
then only Firm, Released, or On Hold order lines that have some unshipped
quantity are shown.

If you select Shipments, VISUAL displays only orders that have packlists
against them. One line appears for each packlist that contains shipments
of each order line. This may cause the same Order ID/Line # combination
to display more than once, but each line represents a separate packlist
line. In this case, each line represents a single packlist shipment,
and the Shipped Quantity shows the amount shipped on that packlist.
Returns appear as negative Shipped Quantities.
