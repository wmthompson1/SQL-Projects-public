# Printing Tag List Reports

The Tag List report allows you to print tags in a
one-to-a-line report format. This is useful if you want to enter counts
in this way, or for recounting of tags with a discrepancy above a
certain threshold. Count quantities do not appear on this report.
Use the Tag Comparison Report if you want to analyze counted quantities.

To run this report:

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site where you conducted the count. If you are licensed to use
   a single site, this field is unavailable.
2. In the table, select
   the physical inventory count.
3. Select Print
   Tag List from the File menu.

The Print Tag List dialog box appears.

4. Enter the Physical Count
   ID.

You can use this report with any existing
physical count. If a count is in progress, Physical Count ID defaults
to the current count. If you want to compare a different account,
enter the Count ID.

5. Select the Tag Types
   you want to print.

You can print any or all of the following
types of tag:

Uncounted - This
option prints tags that have not had any count entered using Enter
Counts or Review All Counts. This is useful for printing a list of
the remaining uncounted tags for a count that is nearly finished.
You can then use this list to record the remaining tags.

Counted - This
option prints counted tags where the difference between counted and
on-hand quantities or values exceeds a certain threshold. This is
specified in the same way as described previously for the Tag Comparison
Report.

Voided - You can
print voided tags, if necessary.

Check the box for any of these types that
you want printed, and enter the quantity and value change information.

6. Select the Sequence.

You can order the tags by Group Number and
Tag Number, just Tag Number, or by Location ID and Part ID. The first
option is useful for performing a full count, and the later for recounting
a limited number of tags.

Select the sequence radio button for the
appropriate option.

7. Select the output.

Select Print,
View, or File.

8. Click Ok.
9. A standard Windows Print
   dialog box appears.
10. Enter the appropriate
    information and click Ok.

The selected tags print one to a line, with
the following information for each: Tag Number, Part ID, Part Description,
Unit of Measure, Location ID. Additionally, a blank line for Count
and Recount is printed.

If printing by Group Number/Tag Number,
each group is printed on a separate set of pages. At the end of each
group (or the report, if printing in Tag sequence) blank lines are
provided for Counted By, Recounted By, Entered By, and Checked By
identification.
