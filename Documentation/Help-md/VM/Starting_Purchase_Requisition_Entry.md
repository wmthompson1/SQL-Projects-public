# Starting Purchase Requisition Entry

Purchase Requisition Entry is available from the VISUAL main menu.

To start Purchase Requisition Entry:

Select Purchase Requisition Entry from
the Purchasing menu.

The Purchase Order Entry window appears with no information in it.
