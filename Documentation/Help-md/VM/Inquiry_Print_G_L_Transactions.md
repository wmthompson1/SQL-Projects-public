# Inquiry / Print G/L Transactions

Use the Inquiry / Print G/L Transactions window to review all G/L
transactions related to projects. You can view posted and unposted
transactions, and in Inquiry mode you can access the individual transaction
that created the G/L transaction.

To access the Inquiry / Print G/L Transactions window, select Inquiry / Print G/L Transactions from the Info
menu. The Inquiry / Print G/L Transactions window appears.
