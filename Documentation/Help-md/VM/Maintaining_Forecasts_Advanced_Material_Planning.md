# Maintaining Forecasts

You can add and delete planning forecasts in Repetitive
Manufacturing. You can add multiple forecasts for each part.

To add a forecast:

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site to use. If you are licensed to use a single site, this field
   is unavailable.
2. Select Planning
   Forecasts from the File menu.

The Forecasts dialog box appears.

3. If you do not have the
   part for which you want to create the forecast currently in the
   Repetitive Manufacturing window, enter the Part ID for the part
   in the Part ID field or click the Part ID
   button to select a part from the table.
4. Click the Insert
   button.

A new line appears in the Forecasts table.

5. Enter an ID for the
   forecast in the Forecast ID column.
6. Select the status for
   the forecast.

Active - Lists
and Considers the forecast in the Advanced Material Planning window.

Hidden - Considers
the forecast, but is not shown in the Advanced Material Planning window.

Inactive - Does
not consider or show the forecast in the Advanced Material Planning
windows

7. Enter the forecast quantities
   in the appropriate date columns.
8. Click Save
   to save the forecast.

## Deleting a Planning Forecast

To delete an existing Planning Forecast, select that Forecast ID
line and click the Delete button in the
Forecasts table. Click Save to save any
changes to the Forecast dialog box.
