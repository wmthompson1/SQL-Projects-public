# Using the Material Planning Tool

The Material Planning tool has three levels of information:
Part, Warehouse, and Demand/Supply. When you first open the Material
Supply tool, it appears empty giving you the chance to populate it
with a specific range of information.

If you want to view information for a specific time period, click
the calendar buttons and select the Starting and Ending dates you
want to use. These dates restrict the demand and supply with which
VISUAL populates the Material Planning child window. You can use the
following date combinations:

Only an Ending Date If you leave the
Starting Date blank, VISUAL populates the window with all records
up to the Ending Date.

Only a Starting Date If you leave the
Ending Date blank, VISUAL populates the window with all records from
the Starting Date onwards. If you leave the Ending Date blank and
you have entered a quantity of days greater than zero in the Planning
Horizon section of the Search Criteria child window, VISUAL assumes
an ending date of the current date plus the number of horizon days
when filtering the demand and supply information in the Material Planning
child window.

Starting and Ending Dates Blank Leaving
the starting and ending dates blank and clicking the Refresh button
causes VISUAL to populate the Material Planning tool with all available
information.

If you have a Planning Horizon of zero, VISUAL uses the current
date as an Ending Date and no information for the future appears.

To view a list of the warehouses for a part, click the plus icon
to the left of the part in the explorer panel.

 This does not select the part, it only expands it.

If you are licensed to use multiple sites, the table shows material
planning information for all sites.

## Viewing Part Warehouse Information

To view a parts warehouse information select the
part in which you are interested by clicking the part name in the
explorer panel.

All of the warehouses for the part you select appear in the view
panel.

## Viewing Material Planning Information

To view material planning information for parts in
specific warehouses, select the warehouse for the part in which you
are interested in the explorer panel.
