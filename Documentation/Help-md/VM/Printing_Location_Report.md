# Printing Location Report

The Location Report summarizes traceable part information
by warehouse and location.

To print this report:

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site ID to use for the report. If you are licensed to use a single
   site, this field is unavailable.
2. Select Print
   Location Report from the file
   menu.

The Print Trace Locations dialog box appears.

3. Enter the part range
   to print.

You can print the report for a single part,
a range of parts, or all parts. To print for a single part, enter
its Part ID in the Starting Part ID and Ending Part ID fields. You
can specify a closed or open-ended range by entering one or both of
these fields. Starting and Ending Part IDs are included in the range.

4. Choose the print options.

If you want the information for each Part
ID to print on its own group of pages, check the Page Break on Part
ID check box.

Normally, alpha properties, numeric properties,
and comments print with the report. To suppress printing of one or
both, check the corresponding check box.

5. Select the print output.

Select either Print, View, or File.

6. Click Ok.

If you are printing the report, a standard
Windows dialog box appears.

7. Make the appropriate
   selections, then click Ok to print the
   report.

For each traceable Part ID, trace record
information is grouped by Warehouse and Location. One line appears
for each lot number, showing the Part Description, Quantity in the
Location, Expiration Date, and Creation Date of the lot number.
