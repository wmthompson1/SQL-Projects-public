# Arranging Windows

You can also tile the window vertically or horizontally.
From the Window menu, you can select Tile Vertical,
Tile Horizontal, or Cascade,
as you preference for the view display. Select Arrange
Icons to move minimized windows to the bottom of the Order
Management Window.
