# Showing Inventory Transactions

You can view all of the existing inventory transactions
for the current part.

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site ID to use. If you are licensed to use a single site, this
   field is unavailable.
2. Click the Part ID button
   and select the part to view.
3. Select Inventory
   Transactions from the Info menu.

The Inventory Transaction dialog box appears, displaying a table
containing all information for all inventory transactions associated
with the part.

Transactions in the table appear from most recent to the oldest.
To reverse this order, clear the Descending check box. You can also
enter a From Date and a To Date to reduce the list to only those transactions
that were done during a date range.
