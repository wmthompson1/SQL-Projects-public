# Printing Trace Histories

To print a trace history report:

1. While in the Trace History
   window, click the Print button.

The Print Trace History dialog box appears
populated with the current Profile ID.

2. Use the Beginning and
   End Trace ID buttons to select the Trace IDs with which you want
   to start and end the Trace History report.
3. In the Report Type section,
   select the type of report you want to print. You can select either
   Summary or Detail.
4. In the Sequence section,
   select on what you want VISUAL to sort the report. You can select
   either Profile ID or Trace
   ID.
5. Use the View arrow and
   select how you want VISUAL to present the report to you. You can
   select View, Print,
   File, and E-mail.
6. Click the Ok
   button.

Depending on the setting you have made,
VISUAL prints the Trace History report.
