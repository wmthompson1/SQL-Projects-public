# Editing Purchase From Addresses

You can also use the dialog box to add, modify, and
delete alternate addresses for the vendor.

## Adding Purchase From Addresses

1. Click the Insert
   button.

A new line appears in the table.

2. Fill in the address
   information in the fields at the bottom of the dialog box.
3. Click the Save
   button to save the new address.

## Deleting Purchase From Addresses

1. Select the address to
   delete and click the Delete button.
2. Click the Save
   button to permanently delete the address.

## Modifying Purchase From Addresses

1. Select the address to
   modify.
2. Modify the fields that
   appear at the bottom of the dialog box, as necessary.
3. Click the Save
   button to save the modifications.
