# Viewing and Modifying Line and Order Accumulators

VISUAL allows you to modify both line and order accumulators
at the time of order entry, or later during the order fulfillment
process. There are, however, some changes in the way VISUAL calculates
certain values. Use the charts below to reference the manner in which
VISUAL figured values previous to version 6.2.8 and the way it calculates
them now.
