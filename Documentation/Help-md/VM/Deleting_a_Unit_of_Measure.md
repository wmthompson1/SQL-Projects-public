# Deleting a Unit of Measure

1. Select the Unit of Measure
   you want to delete by selecting the left most column in the To
   table.

The line becomes highlighted, indicating
that it is active.

2. Click the Delete
   button.

An X appears beside the row.

3. Click the Save
   button to delete the unit of measure.
