Right-click Calendar Menu Options




## Right-click Calendar Menu Options

Advance One Day Select this
option to advance by one the current day in the date field.

Back One Day Select this option to turn
back by one the current day in the date field.

Advance One Month Select this option to
advance by one the current month in the date field.

Back One Month - Select this option to turn
back by one the current month in the date field.

Advance One Year Select this option to advance
by one the current year in the date field.

Back One Year - Select this option to turn
back by one the current year in the date field.
