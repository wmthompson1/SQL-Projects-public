# Printing EDI Reports

You can print a report by Order ID or by Customer
ID detailing EDI transaction information.

You do not need to have an order open to print an EDI report.

1. From the EDI menu, select
   EDI Reports.

The EDI Reports dialog box appears.

2. Supply VISUAL with parameters
   for the report.

If you want the report to be order-driven,
click the Starting Order ID button to select
the customer order with which you want to begin the report.

Click the Ending Order
ID button to select the customer order with which you want
to end the report.

If you want the report to be customer-driven,
click the Starting Customer ID button to
select the customer with which you want to begin the report.

3. Click the Ending
   Customer ID button to select the customer with which you
   want to end the report.
4. Enter a date range in
   the Starting Date and Ending Date fields.
5. Select the Open
   Orders Only check box if you want to include in the report
   those order that are still open.
6. Select the EDI
   Orders Only check box if you want to include in the report
   those order that have been imported through VMDI and are strictly
   EDI orders. If the customer order is an EDI order, this check
   box is selected by default.
7. Select the CUM
   Discrepancy option button if you want VISUAL to report
   CUM discrepancies for the orders or customers. See Viewing EDI
   Line Accumulators for more information on CUM discrepancies.
8. Select the Session
   Change option button if you want information regarding
   VMDI sessions.
9. Select an output for
   the report:

Print - Select
this option to output the file to a printer. VISUAL sends the file
to the selected default printer. This is the default selection.

View - Select
this option to see what the printed output will look like before you
print it. When the information appears on the screen, you can scroll
through and print it.

File - Select
this option to save the information to a text file. You can then edit
the file outside of VISUAL.

E-Mail - Select
this option to send the report in a Rich Text Format through electronic
mail.

10. Click the Ok
    button to output the information.
