# Viewing and Editing Dispatch Pallet Details

1. To view and edit pallet
   details for a service dispatch, select Pallet
   Details from the Edit menu.

The Edit Pallet Details dialog box appears.

The Dispatch Pack list ID appears in a read-only
field at the top of the dialog box.

2. If applicable, click
   the Insert button to add a new pallet
   line.
3. Modify pallet line information
   if necessary.
4. Click the Save
   button to save any modified pallet lines.
