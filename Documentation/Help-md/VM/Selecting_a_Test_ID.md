# Selecting a Test ID

You can use the tests you set up in Test Definition
to assess the operations in your engineering master. By testing the
results of each operation, you can ensure that the product is correct
before beginning the next operation.

1. From the Operation dialog
   box, click the Other tab.

Test ID and Drawing Number information appear.

2. Click the Test
   ID button to run a test you defined in Test Definition
   Maintenance.

The List of Tests dialog box appears.

3. Select a Test ID and
   click the Ok button.
4. Click the Save
   button to save the new information.

VISUAL runs the test after you enter the
results in Test Definition Maintenance.
