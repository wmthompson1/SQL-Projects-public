# Setting Information on the Allowable Sites Tab

If you are licensed to use multiple sites, use the Allowable Sites
tab to assign sites to users. When you assign a site to a user, the
user can use the site when entering data, depending on the other permissions
you have set up for the user. The user also has access to the Accounting
Entity associated with the site.

To specify a users allowable sites:

1. Verify that the user
   ID you want to use is displayed in the User ID field.
2. Click the Allowable
   Sites tab.
3. Click the Insert
   button.
4. Double-click the Site ID button and select the site to assign
   to the user.
5. Click Save.
