# Setting the Current Calendar

1. Select Set
   Current Calendar from the View menu to set the current
   planning calendar of the display.

The Planning Calendar dialog box appears.

2. In the Planning Calendar
   dialog box, highlight the calendar you want to use, then click
   the Set Current button.
3. Click the Save
   button to set the calendar you selected as the current calendar.
