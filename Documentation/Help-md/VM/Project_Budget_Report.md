# Project Budget Report

The Project Budget report compares the budgeted costs
to the current actual costs.

To run the report:

1. From the Project Summary
   window, click the Site ID arrow and select the site for which
   to print a project budget report.
2. Select File,
   Print Budget Info. The Print Budget
   Info window appears.
3. To view all project
   budgets, leave the project ID fields blank. To view a range of
   projects, enter a project ID in the Starting Project ID field
   and Ending Project ID field. All projects that fall between those
   two IDs (inclusive) will appear in the report.
4. Click Ok.
   The report outputs.
