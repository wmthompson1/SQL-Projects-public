# Using Drag n' Drop to Order WIP/VAS for a Part

You can order WIP/VAS for eligible parts by dragging
a WIP/VAS for a part from the warehouse specified on the order form
into the Order Entry window.

1. In the Order Entry window,
   open the order for which you want to order WIP/VAS. Highlight
   the order line that contains the part for which you want to order
   a WIP/VAS.
2. If not already open,
   open the WIP/VAS child window. Tile or cascade the two windows
   so that you can easily perform a drag n drop operation.
3. In the WIP/VAS window,
   double-click the part icon.

Valid WIP/VAS for the part in the warehouse
appear.

4. Select the WIP/VAS you
   want to order and hold down the left mouse button until the drag
   n drop box appears.
5. Drag the box into the
   open Order Entry child window and release it on the highlighted
   line.

If successful, VISUAL places a check mark
at the beginning of the line.

VISUAL orders this WIP/VAS for the entire
quantity of the part.

6. To verify the ordering
   of the WIP/VAS, in the Order Entry window, click the Assign
   WIP/VAS button on the Customer Order table toolbar or select
   Assign WIP/VAS from the Edit menu.

The Assign WIP/VAS dialog box appears. Check
the information and edit it as necessary. Click the Ok
button.

7. Click the Save
   button on the Customer Order main toolbar to commit the order
   of the WIP/VAS and save the order.

After saving, the price per part of the
WIP/VAS appears in the WIP/VAS Unit price column.
