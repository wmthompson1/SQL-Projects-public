# Using Drag n' Drop to Allocate Available to Customer Supply to Customer Orders

Depending upon a customers Priority Code and Disbursement
Fill Rate (the percentage of the total available quantity a customer
has access to) you can allocate supply to a customer order line using
the Part Availability windows drag n drop function.

To perform this operation, you must have the Current Customer Order
window and Part Availability window open. The part that you want to
allocate to the customer order line must be displaying in the Part
Availability window.

1. In the Part Availability
   window, double-click the icon and highlight Available
   to Customer.
2. Left-click the line
   and hold down the mouse button until the drag n drop box appears.
3. Drag the box into the
   open Current Customer Order Window and release the mouse button,
   thereby "dropping" it.

VISUAL allocates inventory supply from the
appropriate customer warehouse to the customer order line. In the
Current Customer Order window, a plus box appears next to the line
to which you allocated supply. If you click this plus box, the ID
of the Warehouse from which you allocated the supply appears.
