# Creating Activity Types

To create activity types:

1. In an application that
   supports activities, select Edit, Activity Entry.
2. In Activity Entry, select
   Maintain, Activity
   Types.
3. Click Insert.
4. Specify this information:

Activity Type ID
Specify an ID for this activity type.

Description
Specify a description for the ID.

5. Click Save.
