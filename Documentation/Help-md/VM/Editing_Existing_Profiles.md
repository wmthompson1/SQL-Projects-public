# Editing Existing Profiles

To edit an existing profile:

1. Change any of the information
   for any line you wish to change. You CANNOT change the Number
   ID.

When changing Next Number, be aware that
you could change it to a value that has already been used.

2. Click the Save
   button to save your changes.
