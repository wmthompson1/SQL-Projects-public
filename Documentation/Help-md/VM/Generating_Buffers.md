# Generating Buffers

After you set up the free work order buffers and the CCR buffers,
apply them to your existing supply orders. To apply buffers:

1. Select Scheduling,
   DBR Maintenance.
2. If you are licensed
   to use multiple sites, click the Site ID arrow
   and select the site whose buffers you are generating. If you are
   licensed to use a single site, this field is unavailable.
3. Select Edit,
   Generate All Buffers.
