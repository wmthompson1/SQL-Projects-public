# Setting Up Part-related Codes

You can use several basic codes to help define part information.
Before you specify a part in your database, set up the part-related
codes so that they are available for selection when you defined the
part.

Set up these codes in Part Maintenance:

* [Product
  codes](Adding_Product_Codes.md)
* [Commodity
  codes](Adding_Commodity_Codes.md)
* [Price
  groups.](Adding_Price_Groups.md) Use these codes in conjunction with VISUAL Pricebook.
