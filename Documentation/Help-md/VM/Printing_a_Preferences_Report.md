# Printing a Preferences Report

To print a list of your preference settings:

1. Select Admin,
   Preferences Maintenance.
2. Select File,
   Print Preferences.
3. if you are the system
   administrator, click the Starting User ID
   and Ending User ID browse buttons to
   specify a range of user preferences to view in the report. Leave
   the fields blank to view all user preferences. Specify the same
   user ID in the Starting User ID and Ending User ID fields to view
   a single user's preferences.

If you are not the system administrator,
the system inserts your user ID in the Starting User ID and Ending
User ID. You cannot edit these fields.

4. Click the drop-down
   menu to select the output method for the report.
5. Click Ok.
   The system prints a list of your preferences.
