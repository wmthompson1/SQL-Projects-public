# What is the Label Printer Setup Utility?

The capacity to print thermal labels is present throughout
much of Infor VISUAL, with each window allowing you access to a specific
subset of labels. Before attempting to print labels in any of the
supporting windows (a list of which appears below), design a label
complete with a combination of barcodes, text, and graphics using
Loftware, and then link the appropriate VISUAL database fields (columns)
to the corresponding fields using the Label Printer Setup Utility.

Because VISUAL BTS does not incorporate any label design features,
you must design the actual physical label layouts using the third
party label-design application [Loftware](http://www.loftware.com).

As mentioned above, VISUAL supports the printing of labels in the
