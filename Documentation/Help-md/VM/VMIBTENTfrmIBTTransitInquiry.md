# Using IBT Transit Location Inquiry

Use the transit location query feature to monitor
the location of parts while in transit. Base your query on a range
of IBTs, parts, or a date range.

1. From the Info menu,
   select Transit Inquiry.

The IBT Transit Location Inquiry dialog
box appears.

|  |  |
| --- | --- |
| POSTIT.gif | To view all IBTs in the database without configuring a search, click the Refresh button or select Refresh from the File menu. |

2. If you are licensed
   to use multiple sites, select the Sites to include in the search.
   The inter branch transfers you entered in the sites you select
   are displayed in the report. If you are licensed to use a single
   site, this field is unavailable.
3. To view the transit
   status of an individual IBT or a range of IBTs, click the Begin IBT ID and the End
   IBT ID button to specify a range of IBTs.
4. To view the transit
   status of an individual part or a range of parts, click the Begin Part ID and the End
   Part ID button to specify a range of parts.
5. To view the transit
   status of a ship date range, enter date values into the Begin Ship Date and End
   Ship Date fields to form a date range.

Click the Calendar
button to select a date from the drop-down calendar or right-click
the calendar button to choose one of six predefined date options.

5. If you want to view
   only your consignment related IBTs, select the Consignment
   Transfers Only check box.
6. Click the Search
   button to view qualifying data.

VISUAL displays all eligible IBTs in the
upper table, which includes the following columns:

IBT ID - The ID
of the IBT.

Line No - The
number of the line within the IBT.

Part ID - The
part on the IBT line.

Shipped Quantity
- The quantity of the IBT part that has been shipped.

Received Quantity
- The quantity of the IBT part that has been received.

Lines that have not been entirely shipped
or receive are red. Fully shipped and received lines are black.

7. To view shipper/receiver
   detail, select the appropriate line in the upper table. The shippers
   and receivers for the IBT you selected in the upper table appear
   in the lower table.

8. To clear the window
   and begin a new session, click the New
   toolbar button.
