# Editing Information in the Leadtime Process Dialog

To edit process leadtime information:

1. Perform one of these
   steps:
2. Select Sales, Estimating Window.
3. Select Sales, Customer Order Entry.
4. Select Sales, Order Management
   Window.

2. Select the quote or
   order.
3. Select Edit,
   Process Activity.
4. Edit the Actual Days,
   % Complete, Start Date, and End Date as necessary.
5. Click the Save
   button.
