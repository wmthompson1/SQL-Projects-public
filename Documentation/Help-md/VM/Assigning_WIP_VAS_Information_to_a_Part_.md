# Assigning WIP/VAS Information to a Part on a Customer Order Line

|  |  |
| --- | --- |
| POSTIT.gif | Before using WIP/VAS, you must activate it globally.  Start Accounting Entity Maintenance, click the Order Mgt tab and select the WIP/VAS Enabled check box. This activates WIP/VAS globally in the database.  Next, start Warehouse Maintenance, call up the warehouse you want to offer WIP/VAS in, and select the WIP/VAS Supported check box. This activates WIP/VAS at the warehouse level. Finally, specify which parts in the warehouse are eligible for WIP/VAS. See Warehouse Maintenance for more information. |

1. With the customer order
   line to which you wan to assign WIP/VAS information highlighted,
   click the WIP/VAS button located underneath
   the line item table.

The Assign WIP/VAS dialog box appears.

The current Customer Order ID and the line
number of the order to which you are assigning WIP/VAS information
appears in the header section.

2. Click Insert
   to begin adding WIP/VAS information.
3. Double-click <WIP/VAS
   ID> to view the WIP/VAS ID for the part in that warehouse.
   The WIP/VAS for Warehouse Part dialog box appears.

The Warehouse ID and Part ID appear in the
header section.

With the Customers WIP/VAS check box selected,
only the WIP/VAS IDs that have been entered for this customer through
Customer maintenance display in the line item table. Clear this check
box if you want to view all WIP/VAS IDs for the selected Warehouse
ID.

4. Select a line and click
   the Ok button.

The information is transferred to the Assign
WIP/VAS line item table. The total WIP/VAS amount for the C/O line
appears in the header section. For example, for a part quantity of
100 with a WIP/VAS charge of .50 per each part, the WIP/VAS amount
is $50.00.

5. Click the Ok
   button when you are finished entering information.
