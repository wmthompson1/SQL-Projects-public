# Embedding

There are two ways to embed a drawing. The first is
to start from the destination document (where you want the object
to appear). The other way is to start from the source document (where
you are going to get the drawing). Because you most likely already
have drawings created for use in Infor VISUAL, the example below is
from the point of embedding a drawing from inside Infor VISUAL.

1. Select the Manufacturing
   Window object and select Edit from the
   Edit menu.

The dialog box for that object appears.

2. Select Picture/Object
   from the dialog box Edit menu.

The Set Picture/Object window appears.

Because the embedded object appears in a
window, there is no need to select an insertion point (as discussed
in the Windows Users Guide). Infor VISUAL places the object for you.

3. Click the Insert
   Object button.

The Insert Object menu box appears.

The menu listing contains all of the Server
applications that your system has installed. If you have an application
that you think is a Server application but that is not present, refer
to that applications manual for installation information concerning
Embedding and Linking.

4. Select the appropriate
   object type, then click Ok.

The associated application for the object
type you selected opens with a blank editing screen.

5. Create the object you
   want to embed in the destination document.

Some applications allow you to open multiple
windows. If you are embedding from one of these applications, you
can open another document and cut and paste information into the linking
window.

Do not close the embedding window to open
another window. This causes the link between the two applications
to be lost.

6. After entering the information
   to embed, close the source application.

You return to the Manufacturing Window with
the new object displayed in the Set Object/Picture window.

7. Click the Save
   and Exit button to accept the embedded
   object and return to the Manufacturing Window.

The object is now embedded in the Manufacturing
Window object. When you view the Manufacturing Window in graphical
mode, a "p" appears beside the object containing the picture.
