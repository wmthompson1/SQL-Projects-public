# Starting the VISUAL BTS Purchase Receipts Processor

VISUAL BTS Purchase Receipt Entry is not available
from the main Infor VISUAL menu.

From the directory where your executables are installed, double-click
Vmbtsrcv.exe.

Notice the rather simple appearance of the window. Because this
window serves as a transaction processor only, it does not have the
same options as the Purchase Receipt Entry window. Refer to the standard
Purchase Receipts Entry window to verify completed transaction information
and to make any other changes.

[Click here
to view definitions of the columns in both the Pending and Complete
portions of the window.](VISUAL_BTS_Purchase_Receipts_Entry_Table.htm)
