# Editing Customer Forecasts

You can edit customer forecasts or enter new customer
forecasts. Existing customer part forecasts may have been entered
using VISUAL VMDI. See your User's Guide for more information.

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site to use. If you are licensed to use a single site, this field
   is unavailable.
2. From the File menu,
   select Master Production Schedule.

The Master Production Schedule dialog box
appears.

If you have a part loaded in the main Material
Planning window display with an order policy that is not Master Schedule,
VISUAL warns you that the part is not Master Scheduled. Click Ok to
continue.

3. Click the Customer
   Forecasts button located on the bottom right of the dialog
   box.

The Transfer Customer Forecasts to Part
Forecasts dialog box appears.

4. Click the Edit
   Customer Forecasts button.

The Customer Forecasts dialog box appears.

5. Click the Part
   ID button to select a part for the forecast.
6. Click the Customer
   ID button to select a customer.
7. In the Forecast ID field,
   enter an ID for the forecast or select an existing one from the
   list box.
8. To enter new lines,
   click the Insert button and enter Required
   Quantity and Date information.
9. If applicable, enter
   a fill quantity. Auto Fill propagates a specified quantity into
   multiple forecast periods.
10. Specify the period size
    in number of days.

VISUAL creates entries based on a number
of equally sized periods. If you leave this field blank, VISUAL uses
a default period size of 7 days.

11. Specify the horizon
    or select the number of lines to fill.

There are two ways to control the number
of periods generated. By specifying the horizon, you can control how
many periods are generated.

You can also select a number of existing
lines to fill. When you do this, VISUAL ignores the Horizon value.

12. Specify the beginning
    of the week date in the Week Begin field.

Use the Week Begin day to determine the
first date to generate. Select the day of the week from the drop down.
VISUAL uses the next occurrence of that day after the current date
for the first Forecast Date. For example, if today is 01/31/2004 a
Friday, and you specify a week begin of Monday, the first Forecast
Date is 02/07/2004.

13. Click the Save
    button to commit the information.
