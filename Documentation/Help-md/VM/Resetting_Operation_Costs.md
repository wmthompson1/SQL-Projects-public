# Resetting Operation Costs

You can reset operation costs in quotes, engineering
masters, and/or work orders using Reset Operation Costs. This replaces
those costs with the costs currently specified in this resource or
all resources. Do this with caution, as it changes estimated costs.

1. Select Eng/Mfg,
   Shop Resource Maintenance.
2. If you are licensed
   to use multiple sites, click the arrow and select the site that
   contains the resources you want to reset. If you are licensed
   to use a single site, this field is unavailable.
3. Select Edit,
   Reset Operation Costs.
4. To update the current
   shop resource only, click Current Resource
   Only. To update a range of resources, click All
   Resources.
5. Select the check boxes
   for the types of master/work orders whose operations you want
   to update. You can choose any combination of engineering master,
   quote master, and/or work order.
6. If you are resetting
   work order costs, you can reset the costs based on the status
   of each operation or the operation type.

To reset based on status, select the According to Operation Status check box, and then
check each status of operation that you want updated. For example,
you may choose to only update Unreleased and Firmed operations check
these two boxes.

To reset based on operation type, select
the According to Operation Type check box,
then select an operation type from the drop down box.

7. You can also update
   by work order/master creation date. To do this, check this option
   and enter the after and before dates to define a range.
8. Click the Ok
   button.
