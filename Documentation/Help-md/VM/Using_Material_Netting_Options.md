# Using Material Netting Options

Material Netting options control the behavior of the
netting process and are located on the View
menu of the Material Planning Window.

To change any one of the options, select it from the View
menu. If it is checked, it becomes unchecked when you select it, and
vice versa.

For Net by Required Date/Net Using MRP Rules/Net by Release Date,
select the appropriate option. It becomes checked, and the currently
checked option automatically becomes unchecked.
