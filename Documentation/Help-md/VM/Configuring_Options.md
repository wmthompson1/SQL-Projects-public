# Configuring Options

Users can select from the selected filters and configured
options to populate the data in the Firm Orders window.

Save Options

To save options:

1. Click New Options from the File
   menu.

2. Select the required
   filters.

3. Click Save Options from the File menu. The Save
   Option As window is displayed.

4. Specify the name for
   the saved options.

Note:
To set the selected options as default, select the Default check box.

5. Click Save.

Use the Options
drop-down to select a saved options.

Edit Options

To modify the name of an existing saved options or select the default
options:

1. Click Edit. The Edit
   Options window is displayed. You can modify the Name or Default
   column values.
2. Click Save.

Delete Options

To delete a saved option configuration:

1. Select the row, click
   Delete.

2. Click Save.
