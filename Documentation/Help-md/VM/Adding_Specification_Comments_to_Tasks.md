# Adding Specification Comments to Tasks

To add a comment to the Specification column:

1. Select the task to which
   you want to add a comment by either clicking in the Specification
   column or clicking the row header.
2. Click the Task
   Specifications icon on the toolbar.

The Task Specs dialog box appears populated
with any comments anyone has previously entered.

3. Click the Insert
   button.

The Add Comment dialog box appears.

4. Enter the comments you
   want.
5. Click the Save
   button.

A date and time stamp is entered for your
comment and saves it to your database. The date, time, and the beginning
of your comment appear in the Specifications column.
