# Creating Packlist IDs During Shipment

1. Using the Shipping Entry
   window, select the order you want to ship.

VISUAL displays all order lines in the shipping
table.

2. Adjust the Actual Ship
   Date if necessary and then click the Ship All
   button on the main toolbar.
3. Click the Yes
   button to create the pack lists.

If you view pallet details before shipment,
they appear for you to view and modify if necessary.

4. Modify pallet details
   as necessary.

VISUAL ships the order.

As it creates the pack-lists, the IDs appear
in the Pack List ID field, in rapid succession.
