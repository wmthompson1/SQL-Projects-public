# Adding Units of Measure Conversion for Services

1. With a service displaying
   in the Service ID field of the Service Maintenance window, select
   Unit Of Measure Conversion from the
   Edit menu. The Unit of Measure Conversion dialog box appears.
2. Click the Insert
   button to begin adding service unit of measurement conversion
   information. A new line appears in the bottom table.
3. Enter an ID for the
   unit of measure in the Unit of Measure field.

Enter a description for the unit of measure
in the Description field.

Enter a Scale for the unit of measure in
the Scale field.

4. Click the Save
   button to commit the new unit of measure information.

The new line appears in the top table as
well.
