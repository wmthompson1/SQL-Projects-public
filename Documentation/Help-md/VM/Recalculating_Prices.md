# Recalculating Prices

Often, when debugging formulas, you will have both
the Price Testing and Formulas functions open. Additionally, you can
make changes to the contexts of the information within VISUAL Price
Book and refresh the information on the table.

Make changes to formulas or VPB contexts in their various VPB functions.
After you make the changes within the program, you can re-test the
changes on the Price Testing window, without having to re-enter the
information.

To re-calculate prices:

1. If you want to re-calculate
   the price based on a different quantity, you can do so on the
   Price Testing window by changing the value in the Qty column.

If you make changes within other VISUAL
Price Book functions, open the appropriate window and make the changes.
Save each changed record.

2. If you do not have it
   visible, show the Price Testing window.
3. On the VISUAL Price
   Book main toolbar, click the Calculate
   button. VISUAL re-calculates the prices and amounts with the new
   changes to your records.
