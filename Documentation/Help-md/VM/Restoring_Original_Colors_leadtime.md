# Restoring Original Colors

To restore the cards to the original colors used when
VISUAL was delivered:

1. Perform one of these
   steps:
2. Select Sales, Estimating Window.
3. Select Sales, Customer Order Entry.
4. Select Sales, Order Management
   Window.

2. Select the quote or
   order.
3. If the quote or order
   has multiple lines, select the line.
4. Select Info,
   Leadtime Lifecycle.
5. Select Options,
   Color Preference.
6. Click Restore
   Defaults.
