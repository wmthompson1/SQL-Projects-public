# Dispatching Services

You can dispatch parts to outside service vendors
using the Manufacturing Window, but you must have a purchase order
linked to the operation. VISUAL allows you start Purchase Order Entry
from the Manufacturing Window.

1. In the Manufacturing
   Window, select the service operation that you want to dispatch.
2. Select Dispatch
   this Service from the Edit menu.

If there is no service purchase order linked
to this operation, a dialog box appears, asking if you want to purchase
this service.

3. Click the Yes
   button.

The Purchase Service dialog box appears.

The Order Quantity field shows the total
quantity used in the operation.

4. In the Dispatch Quantity
   field, enter the quantity you want to dispatch to the service
   at this time.
5. Click the Order
   button.

The Purchase Order Entry window appears.
The window contains the current information for the service vendor.
The Dispatch Quantity column of the table shows the dispatch quantity
you specified in the Purchase Service dialog box.

6. Make any changes to
   the service vendor information, then click the Save
   button.
