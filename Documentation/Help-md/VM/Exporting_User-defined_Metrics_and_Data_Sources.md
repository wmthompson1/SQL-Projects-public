# Exporting User-defined Metrics and Data Sources

You can export analytics, detail grids, and data sources
to an XML file. You can export both custom content and vendor content.
Vendor content is provided by Infor. You cannot edit vendor content.

1. Select File,
   Dashboard.
2. Select Edit,
   Export Content. A table that lists all
   custom data sources, analytics, and detail grids is displayed.
   Use the Data Source, Analytic, and Detail Grid check boxes to
   filter the type of content displayed in the table. Use the Scope
   options to select the sources of content to include in the table.
3. To select an item for
   export, select the Include check
   box. To select all items, click the Include button.
4. In the File field, specify
   the name of the export file.
5. Click Export.
