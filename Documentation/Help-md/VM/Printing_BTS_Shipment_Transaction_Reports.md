# Printing BTS Shipment Transaction Reports

You can print a report of shipping transactions. Constrain
the range of the report by specifying the inclusion of transaction
types with certain statuses within a given period of time.

1. With the VISUAL BTS
   Shipping Entry window open, select Print
   from the File menu.

The BTS Print Transactions dialog box appears.

2. From the Report Between
   section, select a date range for the report. Click the calendar
   icon or manually enter a date in the Start Date and Thru Date
   fields.

VISUAL BTS uses a long date format. VISUAL
BTS expresses an entry of 1/1/2001 as 2001-01-01-00.-00.-00.

3. Select a report type:

Summary - VISUAL
BTS outputs a summarized version of selected transactions.

Detail - VISUAL
BTS outputs a highly detailed version of selected transactions, including
purchase order, customer, and warehouse information.

4. Select a Transaction
   Status. You can select multiple check boxes.

Pending - Select
this check box if you want BTS to include pending transactions in
the report. A pending transaction is a transaction that has been scanned
but not yet processed.

Completed - Select
this check box if you want BTS to include completed transactions in
the report. A completed transaction is a transaction that has been
scanned and processed.

Errors - Select
this check box if you want BTS to include errors in the report.

In-process - Select
this check box if you want BTS to include in-process transactions
in the report. An in-process transaction is a transaction that you
have scanned and BTS is currently processing.

5. Select a Transaction
   Type:

Select the All of Below
check box to include all the transaction type options in the report.

Select the Shipping
check box to include shipping transactions in the report.

Select the Receiving
check box to include receiving transactions in the report.

Select the Issue to work
order check box to include work order issue transactions in
the report.

Select the Receive from
work order check box to include work order receipt transactions
in the report.

Select the Material Movement
check box to include material movement transactions in the report.

Select the Physical Inventory
Count check box to include physical inventory count transactions
in the report.

6. Specify the output for
   the report. See [Report Output
   Options](Report_Output_Options.md).

7. Click the Ok
   button.
