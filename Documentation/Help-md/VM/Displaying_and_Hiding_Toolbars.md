# Displaying and Hiding Toolbars

If you do not use toolbars, you can hide them.

From the View menu of any application, select the toolbar to clear
the check mark. The toolbar is removed.

Also, you can drag the toolbar to float over the window and close
it using the close button in the upper-right corner.

To reactivate toolbars, from the View menu, select the toolbar.
A check mark is placed next to the menu option and the toolbar is
displayed.
