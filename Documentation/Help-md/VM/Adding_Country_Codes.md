# Adding Country Codes

To add a country code:

1. Select Admin,
   Application Global Maintenance.
2. Select Maintain,
   Table Maintenance, Countries.
3. Click Insert.
4. Specify the ID and description.
   We recommend using two-letter ISO codes for the ID.
5. Click Save.
