# About Entering Contract Information in Customer Order Entry

Enter your contracts, whether they are contracts on which you are
bidding or contracts you have been awarded, using VISUALs Customer
Order Entry window.

VISUAL ties the individual lines of your contracts to their respective
projects using CLINs, Contract Line Item Numbers. By using CLINs,
you can reference each contract line to a specific leg within the
appropriate project. By using a specific work breakdown structure
(WBS) to reference the legs within a project and using the same WBS
codes for the corresponding work orders that belong to the project,
you can set up a system of references that enable you to track labor
and expenditures for your contract.

You can only link a customer order to a project with the status
of released.
