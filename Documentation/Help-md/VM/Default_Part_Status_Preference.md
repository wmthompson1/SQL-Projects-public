# Default Part Status Preference

Use Preferences Maintenance to specify the default status of new parts.

1. Select Admin,
   Preferences Maintenance.
2. Click Insert.
3. Specify this information:

Section Specify
PartMaintenance.

Entry Specify DefaultStatus.

Value Specify the
default status of new parts. To set the status of new parts to Inactive,
specify I. To set the status of new parts to Active, specify A. If you
do not specify a preference, new parts have a default status of Active.

4. Click Save.
