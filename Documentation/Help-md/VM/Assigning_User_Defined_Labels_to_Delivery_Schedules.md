# Assigning User Defined Labels to Delivery Schedules

You can define user defined Layout IDs for a customer
delivery schedules. User defined fields are static fields in the database
you can use to store special information about customer delivery schedules.

To being using user defined labels in customer order delivery schedules:

1. From the main menu,
   select Customer Order Entry from the
   Sales menu.

The Customer Order Entry window appears.

2. If you are creating
   a new order, follow [order
   entry procedures](VMORDENTfrmCustomerOrder.htm).

Click the Our Order ID
button to select the order with the delivery schedule.

3. Highlight the line for
   which you want to create a delivery schedule, and click the Delivery Schedule button or select Delivery
   Schedule from the Edit menu.

The Edit Delivery Schedule (customer order/line
#) dialog box appears.

4. Click the Insert
   button to add a new delivery schedule line.
5. Enter delivery schedule
   line values as you normally would, pausing at the last ten columns.

These are the User Defined columns.

By default they appear as User Defined 1,
User Defined 2, and so on.

6. To change the labels
   of these columns, click the Configure Table
   button.

The Configure Line Item Table dialog box
appears.

7. Scroll down in the table
   until you reach the user defined columns.

They appear as colUSER\_1, colUSER\_2, up
to colUSER\_10.

8. Beginning with the column
   you want to change, place the cursor in the Title field and modify
   the default Title as appropriate. For example, you could change
   a user defined title to Customer Ref# or use these fields for
   note or comments.

To create a hard return in a title field,
place an @ after a word. For example, Customer Order@Ref# will appear
on two lines.

9. Verify that the columns
   are visible by selecting them and clicking the Make
   Visible button.
10. Click the Save
    button.

The fields now appear in the line item table.
