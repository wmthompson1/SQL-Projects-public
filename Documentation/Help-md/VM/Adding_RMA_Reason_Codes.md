# Adding RMA Reason Codes

A Return Material Authorization (RMA) is a document
that controls the return of some part or parts previously sold and
shipped to a customer. Use RMA Reason Codes to specify why a customer
is returning a part.

1. Select Admin,
   Application Global Maintenance.
2. Select Maintain,
   Reasons Maintenance, RMA
   Reason Table.
3. Click Insert.
4. Specify a code and a
   description.
5. Click Save.
