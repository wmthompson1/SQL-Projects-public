# Adding AIC Codes

You can use AIC codes (Alternative Industrial Codes)
as an alternative to SIC code. You can enter and delete AIC codes
from the AIC Table.

1. Select Admin,
   Application Global Maintenance.
2. Select Maintain,
   Table Maintenance, AIC
   Table.
3. Click Insert.
4. Specify an AIC code
   and a description.
5. Click Save.
