# What is Work Order/Operation Input?

Use Work Order/Operation Test Input to assign tests
to engineering masters. This allows work orders generated from masters
to inherit the tests of the master. Work Order/Operation Test Input
also allows you to copy tests from one work order to another work
order for the same part. You can also change the sample size and status
of a test step using Work Order/Operation Test Input.
