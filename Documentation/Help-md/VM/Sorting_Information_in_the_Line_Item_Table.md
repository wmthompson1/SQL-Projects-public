# Sorting Information in the Line Item Table

You can sort the information in the line item table by Ln #, Part
ID, Part Description, or Vendor Part ID. Double-click the triangle
next to the column name by which you would like to sort. The system
sorts the table by the column you choose.

Toggle between ascending and descending order by double-clicking
the triangle. If the triangle points up, then the system sorts the
column in ascending order. If the triangle points down, then the system
sorts the column in descending order.
