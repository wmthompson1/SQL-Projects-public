# Editing Group Captions

To edit the name of groups:

1. Select File,
   Personal Menu from the main VISUAL menu.
2. In the Personal Menu,
   select the name of the group you want to change.
3. Right-click anywhere
   in the Personal Menu and select Customize
   from the right-click menu.

The Customize dialog box appears.

4. Click the Edit
   Current Group button.

The Edit Group dialog box appears.

5. Modify the Group Caption
   name and icon as necessary.
6. Click the OK
   button.
7. Click the OK
   button
