# Searching for Masters and Work Orders

To search for a master/work order:

1. Select Search
   from the File menu, or click the Search button in the Manufacturing Window
   Selection dialog box.

The Work Order/Engineering Master Selection
Table appears.

2. Select the sequencing
   of the result list.

You can sort the result list in six ways:

By Work Order ID By Creation Date

By Customer ID By Want Date

By Customer Name By Release Date

Only Work Order ID and Creation Date apply
to masters; the other options are only useful for work orders. Work
Order ID sequences masters by Base ID, of the listing.

3. Select the appropriate
   No Customer setting.

The No Customer check box controls the display
of the Customer ID column in the search window. Searches and sequences
on the Customer are more time consuming, because VISUAL must also
access the Customer Order Table.

Leave the No Customer box checked (the default)
to greatly speed up your search.

If you choose to sort by Customer ID or
Customer Name, VISUAL automatically clears the No Customer box.

If you wish to search on the Customer ID
field, un-check the No Customer box, and the Customer ID column appears
at the left of the table.

4. Use the Descending
   check box as appropriate.

Click the Descending
check box to reverse the order of the listing.

5. Enter the search strings
   and/or wildcards you want to use for the search criteria.

To search for only a specific type of object,
use the Type column enter M for engineering Master, Q for Quote
master, or W for Work order.

If you pressed Search from the Open dialog,
then any Base ID, Lot ID/Eng ID, or Split ID that you entered, along
with the Type, are carried into the search fields.

6. Click the Search
   button.

The results of the search appear in a table
window. The columns of this window are slightly different than those
in the actual search window the Type, Base ID, Lot ID, and Split
ID are combined into a single column.

At this point you can do one of two things.

Search Again -
It may be easier to perform a search in multiple steps. To perform
a second search on the results of the first, just click Search again.

Open - Open one
of the master/work orders by selecting its line and clicking on Ok, or by double-clicking on the line. You are
now ready to edit the master or work order using the Manufacturing
Window.
