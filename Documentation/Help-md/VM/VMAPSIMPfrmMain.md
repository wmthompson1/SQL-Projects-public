# About the APS Import Utility Window

If you are licensed to use APS functionality, use
the APS Import Utility window to import files that from your advanced
planning and scheduling system into VISUAL.

## Starting the APS Import Utility Window

In the VISUAL executables directory, double-click VMAPSIMP.EXE.

## Tasks

This table shows the tasks that you can complete in the APS Import
Utility window:

| Task | Description |
| [Using the APS Import Utility](Using_the_APS_Import_Utility.md) | Use this procedure to import data from your advanced planning and scheduling system. |

## Fields

Site
ID

The site from which you are exporting data.

Parts

Select this check box to import part masters. In the File Path field,
specify the location of the file.

Shop Resources

Select this check box to import shop resource masters. In the File
Path field, specify the location of the file.

Frozen Operations

Select this check box to import frozen operations. In the File Path
field, specify the location of the file.

Customer Orders

Select this check box to import customer orders. In the File Path
field, specify the location of the file.

Inventory Transactions

Select this check box to import inventory transactions. In the File
Path field, specify the location of the file.

Master Schedule

Select this check box to import the master schedule. In the File
Path field, specify the location of the file.

Work Order

Select this check box to import work orders. In the File Path field,
specify the location of the file.

Purge
First

Select this check box to delete existing work order information.

Purchase Orders

Select this check box to import purchase orders. In the File Path
field, specify the location of the file.

Calendar Exceptions

Select this check box to import calendar exceptions. In the File
Path field, specify the location of the file.

Work Order Sched Priority

Select this check box to import work order priorities. In the File
Path field, specify the location of the file.
