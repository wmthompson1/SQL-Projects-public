# Showing/Hiding the Status Bar

Activate Show Status Bar to display a status bar at
the bottom of the display. When you right click a push button in the
tool bar, a brief description of the function of that button appears
in the status bar. As you drag an operation to a different date and
time, the status bar displays the time and date upon which you are
about to drop the operation.
