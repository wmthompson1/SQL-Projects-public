# Editing a Line Item

1. With the appropriate
   request for quote open in the Vendor RFQ Entry window, move the
   cursor to the appropriate field to edit.
2. Make the appropriate
   changes to the field.

A check mark appears in the row header.

3. Click select Save
   to update the quotation.
