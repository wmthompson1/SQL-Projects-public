# Physical Inventory Count Using VMBTSBI

The standard Infor VISUAL windows provides an excellent
method for locking inventory and printing the necessary paperwork
for a Physical Inventory and/or Cycle Count.

For example:

* Physical Inventory / Cycle
  Count is created within standard VISUAL
* Tags/Sheets and barcodes
  are printed within standard VISUAL
* Material counts are performed
  using barcode hardware
* Counts are verified within
  standard VISUAL
* Material Recounts are performed
  using barcode hardware
