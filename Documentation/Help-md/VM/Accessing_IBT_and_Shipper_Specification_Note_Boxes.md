# Accessing IBT and Shipper Specification/Note Boxes

IBT Shipping Entry allows you to view and modify IBT
Specifications and Shipper Specifications associated with the selected
IBT.

To access IBT Specs for the current IBT, select IBT
Specifications/Notes from the Notes menu, or press Shift+F5.

To access IBT Line Specs for a IBT line, highlight the line and
select IBT Line Specifications/Notes from
the Notes menu, or press Shift+F6.

To access Shipper Specs for a IBT line, highlight the line and select
Shipper Specifications from the Notes menu,
or press Shift+F6.

The notes can remain on the screen for the whole session. Their
contents change as the current IBT and selected line item change.

You can make changes to these specifications. Any changes are saved
when you click Save or Ship
All.

It is important to realize that you are changing specifications
for the IBT itself, not for the packlist.
