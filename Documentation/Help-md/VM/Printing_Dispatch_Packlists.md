# Printing Dispatch Packlists

You can print the current purchase order, the current
dispatch packlist, or packlists for all dispatches that occurred between
certain dates.

1. If you are licensed
   to use multiple sites, click the Site ID arrow and select the
   site that contains the dispatch packlists to print. If you are
   licensed to use a single site, this field is not available.
2. From the File menu,
   select Print Dispatch Packlists.
3. Select what you want
   VISUAL to print:

Print Current Dispatch
Packlist - Select this option if you want VISUAL to print only
the packlist for the current dispatch.

Print Current Purchase
Order - Select this option if you want VISUAL to print only
the current purchase order from which you dispatched parts.

Print All Orders Dispatched
Between - Select this option of you want VISUAL to print all
the orders that you dispatched in a certain period of time.

If you select this option, the Dated From
and Dated Through fields become active. Enter the dates between which
you want VISUAL to print dispatches.

4. Select Print
   Lines Due Through if you want VISUAL to print due dispatch
   line items. This option is available regardless of what you selected
   above. Enter a date through which you want VISUAL to print due
   dispatch lines.
5. Select Do
   Not Print Previously Printed Dispatch Packlists if you
   do not want VISUAL to print dispatch packlists that it already
   printed.
6. Click Ok.

The report appears for you to view before
printing.
