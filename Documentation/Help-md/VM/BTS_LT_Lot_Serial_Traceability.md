# BTS-LT Lot/Serial Traceability

All of the transactions within the VISUAL/BTS-BI module
have the ability to track Lot/Serial Traceability. Any item within
VISUAL Manufacturings system which has been determined to be a Lot/Serial
Traceable Item can be traced within VISUAL/BTS. Only the items that
have Lot/Traceability set, will scanning prompts appear for the appropriate
Lot/Serial properties. Any required Lot/Serial Traceability properties
will be required scanning within VISUAL/BTS to update the necessary
properties.
