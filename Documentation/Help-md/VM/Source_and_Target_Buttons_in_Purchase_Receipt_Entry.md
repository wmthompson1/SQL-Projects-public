# Source and Target Buttons in Purchase Receipt Entry

When you are viewing a transaction, you can use the
source and target toolbar buttons to view related transactions. Use
the source button to view the previous transaction in the lifecycle.
Use the target button to view the next transaction in the lifecycle.

In Purchase Receipt Entry, the Source button is labeled Purchase
Order. Click the button to open the purchase order. The Target button
is labeled Invoice. It is available if you have created an AP invoice
for the purchase receipt. Click the Target button to open the AP invoice.
