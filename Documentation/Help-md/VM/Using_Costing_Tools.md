# Using Costing Tools

The Audit feature tracks and tallies various types
of inventory transactions; either to a work order or to raw materials
inventory (purchase or transfer), applied labor & burden, and/or
indirect labor. All costs are prepared for journal entry through the
Costing Utilities function. It is imperative that the links between
the transaction held in the transaction tables, the related distribution
tables, and the Work-in-process Issue & Detail tables always support
one another and more importantly tie out.

If you are licensed to use multiple sites, click the Site
ID arrow and select the site that contains the costs you are
checking. If you are licensed to use a single site, this field is
unavailable.

Enter your Cost to Check selection in the Cost to Check field. Select
from the following:

* Purchases
* WIP
* Finished Goods
* Shipments
* Adjustments
* Indirect Labor
* WIP Balance

VISUAL performs the necessary check based on your Cost to Check
selection and also notes any discrepancy in the Remarks column. You
can request to make the necessary corrections (Post) to ensure that
the Manufacturing costs are kept in sync with the supporting General
Ledger entries.
