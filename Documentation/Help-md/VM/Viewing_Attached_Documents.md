# Viewing Attached Documents

The Documents button on the toolbar indicates whether documents
are attached to the record.

![](Graphics/tb_documents_no.png)  No documents are attached.

![](Graphics/tb_documents_yes.png)  Documents are attached.

To view an attached document, click the Documents
toolbar button and double-click the document to view it.
