# Deleting Parts, Resources, and Services from Cost Simulations

You can only delete parts, resources, or services
from a simulation if these criteria are met:

* The simulation is not based on an engineering
  master
* The simulation has not been used to update
  costs in the database.

1. Select Inventory,
   Cost Simulation.

2. If you are licensed
   to use multiple sites, click the Site ID
   arrow and select the site to use for the simulation. If you are
   licensed to use a single site, this field is unavailable.
3. Click the Simulation
   ID browse button and select the simulation.
4. Select the row to delete.
5. Click the Delete
   Part Row, Delete Resource Row,
   or Delete Outside Service Row button.
   An X is displayed in the row header to indicate you have marked
   the row for deletion.
6. To delete the marked
   rows, click Save.
