# Viewing Document References in the Material Planning Window

You can view documents that have been attached to
the part in Part Maintenance. You can view existing document attachments
only. You cannot add new document references in the Material Planning
Window.

1. Select Inventory,
   Material Planning Window.
2. If you are licensed
   to use multiple sites, click the Site ID
   arrow and select the Site ID to use. If you are licensed to use
   a single site, this field is unavailable.
3. Click the Part
   ID browse button and select the part to view.
4. Select Info,
   Document Reference.
5. Double-click the document
   ID, or select the row and click Open Doc.
