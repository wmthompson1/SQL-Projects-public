# Adding Outside Service Information

You can use the Operation dialog box to enter outside
service vendor information.

1. In the Operation dialog
   box, click the Service tab.
2. Enter the Vendor ID
   in the Vendor ID field.
3. Enter the Vendor Service
   ID in the Vendor Service ID field.
4. Enter the Service Part
   ID in the Service Part ID field.
5. Click Save
   to save any changes to the Operation dialog box.
