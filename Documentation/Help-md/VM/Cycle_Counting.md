# Cycle Counting

Enter cycle counts and recounts using barcode scanners.
This allows for more accurate counts and part matchups during cycle
counting.

Transactions are inserted into the BTS\_PENDING\_TRANS table by the
reader through the following prompts:

* Employee ID
* Physical Count ID
* Tag No
* Warehouse ID
* Location ID
* \*Trace ID
* \*Alpha Propertys
* \*Numeric Propertys
* Qty

The prompts that are marked with and asterisk(\*) are optional and
are determined by the part traceability of the inventory being scanned.
