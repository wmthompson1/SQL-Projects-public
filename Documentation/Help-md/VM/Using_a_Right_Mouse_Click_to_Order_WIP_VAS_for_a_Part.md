# Using a Right Mouse Click to Order WIP/VAS for a Part

You can order WIP/VAS for a part in a customer order
by right clicking on a WIP/VAS type in the WIP/VAS child window.

1. In the Order Entry window,
   open the order for which you want to order WIP/VAS. Highlight
   the order line containing the part for which you want to order
   a WIP/VAS.
2. Open the WIP/VAS child
   window. Tile or cascade the two windows so that you can see VISUAL
   add the WIP/VAS to the line.
3. In the WIP/VAS window,
   double-click the part icon.

Valid WIP/VAS for the part in the warehouse
appear.

4. Select the WIP/VAS you
   want to order and right-click it.

The Order option appears.

5. Click Order.

VISUAL orders this WIP/VAS for the entire
quantity of the part on the highlighted order line.

If successful, VISUAL places a check mark
at the beginning of the line.

6. To verify the ordering
   of the WIP/VAS, in the Order Entry child window, click the Assign WIP/VAS button on the Customer Order
   table toolbar or select Assign WIP/VAS
   from the Edit menu.

The Assign WIP/VAS dialog box appears. Check
the information and edit it as necessary. Click the Ok
button.

7. Click the Save
   button on the Customer Order main toolbar to commit the order
   of the WIP/VAS and save the order.

After saving, the price per part of the
WIP/VAS appears in the WIP/VAS Unit price column.
