# Monthly EE Count

```sql
USE DataWarehouse
GO

SELECT EMPLOYEE_ID
, RESOURCE_ID
, MONTH(TRANSACTION_DATE)
, YEAR(TRANSACTION_DATE)
--, *
, COUNT(*)
FROM LABOR_TICKET LT
inner join shop_resource sr
on lt.RESOURCE_ID = sr.id
WHERE TRANSACTION_DATE >= '1/1/2018'
AND RESOURCE_ID IS NOT NULL
and user_5 = 'AP1'
GROUP BY EMPLOYEE_ID, RESOURCE_ID, MONTH(TRANSACTION_DATE), YEAR(TRANSACTION_DATE)
```
