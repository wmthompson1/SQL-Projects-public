# System - SSIS Failures and Errors

```sql
USE SSISDB
GO

SELECT TOP 1000 PACKAGE_NAME, SUBCOMPONENT_NAME, PACKAGE_PATH, EXECUTION_PATH, [MESSAGE], MESSAGE_TIME 
FROM [CATALOG].[EVENT_MESSAGES]
WHERE  MESSAGE_TIME > '2021-02-01'
AND    EVENT_NAME IN ('ONERROR', 'ONTASKFAILED')
ORDER BY MESSAGE_TIME DESC

```
