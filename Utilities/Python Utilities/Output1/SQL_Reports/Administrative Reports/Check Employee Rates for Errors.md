# Check Employee Rates for Errors

```sql
SELECT        EMPLOYEE.ID, EMPLOYEE.LAST_NAME, EMPLOYEE.FIRST_NAME, EMPLOYEE.DEPARTMENT_ID,EMPLOYEE.SHIFT_ID, EMPLOYEE.EARNING_CODE_ID, 
                         EMPLOYEE_SITE.BASE_PAY_RATE
FROM            EMPLOYEE INNER JOIN
                         EMPLOYEE_SITE ON EMPLOYEE.ID = EMPLOYEE_SITE.EMPLOYEE_ID
WHERE        (EMPLOYEE.ACTIVE = 'Y')

```
