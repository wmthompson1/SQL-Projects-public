# AP App Invoices without Vouchers

```sql
SELECT P.PNO AS 'PO'
    , F.COMPANYID AS 'VENDOR_ID'
    , F.DOCNO AS 'DOC'
    , F.[FILENAME] AS 'FILE_NAME'
    , F.DDATE AS 'UPLOAD_DATE'
FROM TBLFILES F
LEFT OUTER JOIN TBLPO P
ON P.FILEID = F.ID
WHERE F.SECTION = 'INVOICE'
    AND F.VOUCHER# IS NULL
    -- AND F.ID  >= '3202'
    And F.ID >= '178539'

```
