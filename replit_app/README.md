Replit scaffold for hf-space-inventory-sqlgen

This folder contains a minimal copy of the Hugging Face Space app prepared for Replit.

Run locally:

```bash
python -m venv .venv
.venv\Scripts\Activate.ps1  # Windows PowerShell
pip install -r requirements.txt
python app.py
```

On Replit, configure a run command to execute `python app.py` and set any required env vars.
