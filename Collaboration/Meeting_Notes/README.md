## How to Record Meeting Notes

1. **During or after the meeting**, create a file: `YYYY-MM-DD_meeting-topic.md`
2. **Use the template** from [Templates/meeting-notes-template.md](../Templates/meeting-notes-template.md)
3. **Add to the table above** for easy reference
4. **Tag attendees** by mentioning their GitHub usernames if action is needed

## Meeting Types

- **Team Standup** - Regular team sync meetings
- **Decision Meeting** - Sessions focused on making specific decisions
- **Retrospective** - Reflection and improvement discussions
- **Planning** - Project or sprint planning sessions
- **Ad-hoc** - Unscheduled discussions

## Tips

- Capture decisions and action items clearly
- Link to related documents, proposals, or decisions
- Update the table above so meetings are easy to find
- Follow up on action items in subsequent meetings

---

*For ongoing discussions that span multiple meetings, consider using GitHub Issues instead.*
