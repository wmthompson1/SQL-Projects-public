# CMP COID Issues

```sql
USE DataWarehouse
GO

SELECT CO.ID AS COID
	, REPLACE(CB.[USER_ID], '#', '')
FROM CUSTOMER_ORDER CO
INNER JOIN CUSTOMER_BOOKINGS CB
	ON CB.CUST_ORDER_ID = CO.ID
WHERE CUSTOMER_ID = 'BOE609'
	AND [STATUS] NOT IN ('C', 'X')
	AND CO.ORDER_DATE >= '10/1/2020'
	AND LEN(ID) != 9

```
